# 「力」四节点结构图

> **模块定位**：力学的基础章节。力是贯穿整个高中物理的核心概念，本章不清则后续牛顿定律、功和能全面崩盘。

---

## 一、输入端：这一章研究什么问题？

```
一句话：如何用矢量方法描述物体间的相互作用？

研究对象：力（相互作用的量化描述）
与前章区别：从"运动学"（描述运动）进入"动力学"（解释运动原因）

核心追问：
• 力有哪些种类？如何分类？
• 如何定量描述力的大小和方向？
• 多个力如何合成与分解？
```

## 二、工具箱：核心定理与公式

| 工具 | 公式/内容 | 适用条件 |
|------|----------|----------|
| **重力** | G = mg | 地球表面附近 |
| **弹力（胡克定律）** | F = kx | 弹簧在弹性限度内 |
| **滑动摩擦力** | f = μN | 接触面粗糙，有相对滑动 |
| **静摩擦力** | 0 ≤ f静 ≤ fmax | 有相对运动趋势但未滑动 |
| **力的合成** | 平行四边形法则 / 三角形法则 | 任意两个共点力 |
| **正交分解** | Fx = Fcosθ, Fy = Fsinθ | 多力合成/斜面问题 |
| **共点力平衡** | F合 = 0 | 静止或匀速直线运动 |

### 力的分类（两种维度）
```
按性质分类（本源）：
  重力、弹力、摩擦力、电场力、磁场力、分子力...

按效果分类（表现）：
  向心力、回复力、下滑力、压力、支持力、动力、阻力...

关键区分：
• 性质力是"真实存在"的力
• 效果力是"根据作用效果命名"的力，可由性质力合成或分解得到
```

## 三、输出端：能解决什么问题？

| 问题类型 | 解题方法 | 与其他章节的接口 |
|----------|----------|-----------------|
| **受力分析** | 一重二弹三摩擦，最后再看其他力 | → 牛顿第二定律（F合=ma） |
| **斜面问题** | 正交分解：沿斜面+垂直斜面 | → 运动学（a = gsinθ） |
| **弹簧问题** | 胡克定律 + 形变分析 | → 振动与波（简谐运动） |
| **摩擦力方向判断** | 与相对运动（趋势）方向相反 | → 传送带、板块模型 |
| **力的合成范围** | \|F₁-F₂\| ≤ F合 ≤ F₁+F₂ | → 静力学平衡问题 |

### 接口地图
```
力的分析 ──→ 牛顿第二定律（F合=ma）
    │
    ├──→ 运动学（已知力求加速度，再求运动）
    │
    ├──→ 功和能（W = Fxcosθ，功的计算）
    │
    └──→ 动量（F = Δp/Δt，冲量）
```

## 四、易错点：本章最容易踩的3个坑

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 力的分析三大陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 效果力当作独立力画在受力图上                         │
│     错误：受力图上画"向心力""下滑力"                    │
│     正确：向心力是效果力，是其他力的合力或分力           │
│           下滑力是重力沿斜面的分力                       │
│     受力图上只能画性质力！                               │
├─────────────────────────────────────────────────────────┤
│ E2. 摩擦力方向判断错误                                   │
│     错误：摩擦力方向与运动方向相反                       │
│     正确：滑动摩擦力与相对运动方向相反                   │
│           静摩擦力与相对运动趋势方向相反                 │
│     例：传送带上的物体随传送带加速上升                   │
│         摩擦力方向与运动方向相同（提供动力）！           │
├─────────────────────────────────────────────────────────┤
│ E3. 静摩擦力大小直接套f = μN                             │
│     错误：静摩擦力 = μN                                  │
│     正确：f静由平衡条件或牛顿第二定律求解                │
│           0 ≤ f静 ≤ fmax = μsN                         │
│     f = μN只适用于滑动摩擦力！                           │
├─────────────────────────────────────────────────────────┤
│ E4. 弹力有无的判断遗漏                                   │
│     常用方法：假设法（假设有/无弹力，看是否矛盾）        │
│     注意：接触不一定有弹力，必须有形变                   │
├─────────────────────────────────────────────────────────┤
│ E5. 合力与分力的重复计算                                 │
│     错误：既画重力，又画重力沿斜面的分力                 │
│     正确：要么画重力，要么画分力，不能同时画             │
│     分力是重力的等效替代，不是额外的力                   │
└─────────────────────────────────────────────────────────┘
```
