# 英语三大从句「功能-连接词-辨析」决策树

> **模块定位**：语法填空和短文改错的核心考点。掌握决策树，可在5秒内确定连接词，避免凭语感盲选。

---

## 一、核心决策树

```
遇到从句题
    │
    ├── 先看位置 → 名词后面？
    │       │
    │       是 → 定语从句
    │       │    ├── 缺主/宾/表 → who/whom/that/which
    │       │    ├── 缺定语 → whose
    │       │    ├── 缺状语 → when/where/why/介词+which
    │       │    └── 非限制性？ → which（不用that）
    │       │
    │       否 → 继续判断
    │
    ├── 充当句子成分？（主/宾/表/同位语）
    │       │
    │       是 → 名词性从句
    │       │    ├── 缺成分 → what/who/which/wherever
    │       │    └── 不缺成分 → that/whether/if
    │       │         （that无义，whether/if表"是否"）
    │       │
    │       否 → 继续判断
    │
    └── 修饰动词/形容词/句子？
            │
            是 → 状语从句
                 ├── 时间 → when/while/as/before/after/since/until
                 ├── 条件 → if/unless/as long as
                 ├── 原因 → because/since/as
                 ├── 让步 → although/though/even though
                 ├── 目的 → so that/in order that
                 └── 结果 → so...that/such...that
```

## 二、定语从句速查

### 连接词选择表

| 先行词 | 在从句中充当 | 连接词 | 例句 |
|--------|-------------|--------|------|
| 人 | 主语 | who/that | The man **who** came is my teacher. |
| 人 | 宾语 | whom/who/that/省略 | The man **whom** I met is my teacher. |
| 物 | 主/宾语 | which/that | The book **which** I bought is good. |
| 人/物 | 定语 | whose | The student **whose** book was lost... |
| 时间 | 状语 | when | the day **when** we met |
| 地点 | 状语 | where | the place **where** I was born |
| 原因 | 状语 | why | the reason **why** he left |

### 定语从句易错点
```
⚠️ 只用that不用which的情况：
  ① 先行词是all/everything/nothing等不定代词
  ② 先行词被序数词/最高级修饰
  ③ 先行词既有人又有物
  ④ 主句是以who/which开头的疑问句

⚠️ 介词+which/whom：
  介词前置时不能用that，不能省略
  the house in which I lived = the house which I lived in
```

## 三、名词性从句速查

| 类型 | 功能 | 缺成分？ | 连接词 | 例句 |
|------|------|----------|--------|------|
| 主语从句 | 作主语 | 缺 | what/who | **What** he said is true. |
| 主语从句 | 作主语 | 不缺 | that/whether | **That** he is honest is known. |
| 宾语从句 | 作宾语 | 缺 | what/who | I don't know **what** he wants. |
| 宾语从句 | 作宾语 | 不缺 | that/whether/if | I know **that** he is honest. |
| 表语从句 | 作表语 | — | that/whether/what | The problem is **that** we have no time. |
| 同位语从句 | 解释名词 | 不缺 | that | The news **that** he won excited us. |

### 关键区分：what vs that
```
what = "……的事物/人"（在从句中充当成分，有意义）
that = 无意义，不充当成分，只起连接作用

例：
___ he said is important.
→ 从句缺宾语（said缺宾语）→ 用what

The fact is ___ he is honest.
→ 从句不缺成分 → 用that
```

## 四、状语从句速查

| 类型 | 连接词 | 考点注意 |
|------|--------|----------|
| **时间** | when/while/as/before/after/since/until | while后接延续性动词；since主句用完成时 |
| **条件** | if/unless/as long as/provided that | if虚拟语气；unless=if not |
| **原因** | because/since/as | because语气最强；since/as语气较弱 |
| **让步** | although/though/even though | **不与but连用**！ |
| **目的** | so that/in order that | 从句常用can/could/may/might |
| **结果** | so...that/such...that | so+adj/adv；such+noun |
| **方式** | as/as if/as though | as if常用虚拟语气 |

## 五、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 从句辨析常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 定语从句中what和that混用                             │
│     错误：The book what I bought...                      │
│     正确：The book that/which I bought...                │
│     定语从句中没有what！                                 │
├─────────────────────────────────────────────────────────┤
│ E2. 让步状语从句与but连用                                │
│     错误：Although it rained, but we went out.           │
│     正确：Although it rained, we went out.               │
│     或：It rained, but we went out.                      │
│     英语中although和but不能同时出现！                    │
├─────────────────────────────────────────────────────────┤
│ E3. because与so连用                                      │
│     错误：Because it rained, so we stayed home.          │
│     正确：Because it rained, we stayed home.             │
│     或：It rained, so we stayed home.                    │
│     英语中because和so不能同时出现！                      │
├─────────────────────────────────────────────────────────┤
│ E4. 名词性从句缺成分时误用that                             │
│     错误：I don't know that he bought.                   │
│     正确：I don't know what he bought.                   │
│     （bought缺宾语，that不充当成分）                     │
├─────────────────────────────────────────────────────────┤
│ E5. 非限制性定语从句用that                                 │
│     错误：My mother, that is 50, is a teacher.           │
│     正确：My mother, who is 50, is a teacher.            │
│     非限制性定语从句不能用that，指人用who，指物用which   │
└─────────────────────────────────────────────────────────┘
```
