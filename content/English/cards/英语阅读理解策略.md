# 英语阅读理解「体裁-结构-策略」映射表

> **模块定位**：阅读理解（40分）的核心策略。不同体裁用不同方法，避免用同一套策略应对所有文章。

---

## 一、核心映射表

| 体裁 | 核心结构 | 阅读策略 | 作者观点位置 |
|------|----------|----------|-------------|
| **记叙文** | 情感变化线：开端→发展→高潮→结局→感悟 | 抓情感变化线 | 结尾感悟段 |
| **说明文** | 说明对象+层次：现象→原因→影响→对策 | 抓说明对象+层次 | 首段或尾段 |
| **议论文** | 论点-论据-论证：反驳→立论 | 抓论点-论据-论证 | 首段末或转折后 |
| **应用文** | 信息点：广告→优惠/时间/地点 | 抓信息点 | 直接获取 |

---

## 二、分体裁详解

### 记叙文（Narrative）
```
结构：开端 → 发展 → 高潮 → 结局 → 感悟
        ↑                              ↑
     交代背景                       升华主题

阅读策略：
① 标记时间词和情感词
② 画出人物情感变化曲线
③ 重点关注结尾感悟段（主旨所在）
④ 细节题回原文定位，推断题结合情感线

常见情感词：
• 正向：delighted, grateful, relieved, inspired
• 负向：frustrated, disappointed, embarrassed, anxious
• 转折：however, yet, surprisingly, unexpectedly
```

### 说明文（Expository）
```
结构类型：
类型A：现象 → 原因 → 影响 → 对策
类型B：时间顺序（历史发展类）
类型C：空间顺序（介绍地点类）

阅读策略：
① 首段找说明对象（what）
② 各段首句找层次标志（why/how/result）
③ 关注数字、例子、比较
④ 注意作者态度（客观？担忧？乐观？）

作者观点位置：
• 首段（提出话题时暗示立场）
• 尾段（总结时明确立场）
```

### 议论文（Argumentative）
```
结构类型：
类型A：提出问题 → 分析问题 → 解决问题
类型B：反驳旧观点 → 提出新观点 → 论证新观点
类型C：现象 → 多方观点 → 作者观点

阅读策略：
① 首段末或第二段首找中心论点
② 标记转折词（but/however/yet）后的内容
③ 区分他人观点和作者观点
④ 论据类型：事例/数据/引用/对比

作者观点位置：
• 首段末尾（直接提出）
• 转折词之后（对比引出）
• 尾段总结（重申强化）
```

### 应用文（Practical）
```
常见类型：广告、通知、指南、邮件、菜单

阅读策略：
① 不逐字读，直接定位信息
② 标记关键信息：时间、地点、价格、要求、联系方式
③ 注意小标题和粗体字
④ 题干问什么，找什么

信息点速查：
┌─────────┬─────────────────────────────┐
│ 广告类   │ 优惠内容、有效期、适用条件、联系方式 │
│ 通知类   │ 时间、地点、事件、要求、参与对象     │
│ 指南类   │ 步骤顺序、注意事项、禁忌             │
│ 邮件类   │ 发件人目的、请求事项、截止时间       │
└─────────┴─────────────────────────────┘
```

## 三、题型对应策略

| 题型 | 标志词 | 解题策略 |
|------|--------|----------|
| **主旨大意** | mainly about/best title | 首尾段+各段首句，避免以偏概全 |
| **细节理解** | according to/what/which | 题干关键词回原文定位，比对选项 |
| **推理判断** | infer/suggest/imply | 基于原文，一步推导，不过度推理 |
| **词义猜测** | means/refers to | 上下文语境、同义/反义、构词法 |
| **作者态度** | attitude/tone/opinion | 关注形容词、副词的情感色彩 |

## 四、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 阅读理解常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 以偏概全选主旨                                       │
│     错误：用某段的细节当作全文主旨                       │
│     正确：主旨必须覆盖全文核心内容                       │
│     策略：看首尾段+每段首句，找重复出现的主题词          │
├─────────────────────────────────────────────────────────┤
│ E2. 过度推理                                             │
│     原文："She smiled."                                 │
│     错误选项："She was very happy with her life."       │
│     （smile不能推出"对生活很满意"，过度了）              │
│     正确：推理只能推一步，不能加戏                       │
├─────────────────────────────────────────────────────────┤
│ E3. 偷换概念/张冠李戴                                     │
│     原文：A做了X，B做了Y                                │
│     选项：A做了Y（把B的行为安到A头上）                   │
│     策略：细节题必须回原文逐字核对                       │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视否定词和限定词                                   │
│     原文："not all""seldom""partially"                 │
│     选项去掉否定词或把部分说成全部                       │
│     策略：选项与原文逐词比对                             │
├─────────────────────────────────────────────────────────┤
│ E5. 把他人观点当作作者观点                               │
│     原文："Some experts argue that..."                  │
│     选项："The author believes that..."                 │
│     （some experts ≠ author）                            │
│     策略：标记文中引用的他人观点，与作者观点区分         │
└─────────────────────────────────────────────────────────┘
```

## 五、跨模块接口

```
阅读理解策略
      │
      ├──→ 七选五（同属于阅读技能，都需要抓结构和衔接）
      │
      ├──→ 完形填空（语篇理解能力互通）
      │
      ├──→ 语文阅读（不同语言的阅读策略思维相通）
      │
      └──→ 写作（阅读中积累的好表达可用于写作）
```
