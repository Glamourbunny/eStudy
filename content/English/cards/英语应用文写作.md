# 应用文写作「体裁-格式-语块」速查表

> **模块定位**：高考作文（15分）的保底分数。格式正确 + 语块准确 = 10分以上；内容充实 = 13分以上。

---

## 一、书信类通用框架

```
称呼
Dear Sir/Madam,                    ← 不知姓名
Dear Mr./Ms. + 姓氏,              ← 知道姓名
Dear + 名字,                       ← 熟人

开头（写信目的）
I am writing to...                 ← 万能开头
I am writing to express my...      ← 表达情感
I am writing to apply for...       ← 申请
I am writing to invite you to...   ← 邀请

主体（说明情况/提出请求）
First of all, ...                  ← 第一点
Besides, ... / In addition, ...    ← 补充
As for..., ...                     ← 至于……
It is suggested that...            ← 建议

结尾（期待回复/感谢）
I am looking forward to your reply.
I would appreciate it if you could...
Thank you for your consideration.

落款
Yours sincerely,                   ← 正式
Yours faithfully,                  ← 非常正式
Yours,                              ← 熟人
Li Hua                              ← 名字
```

## 二、分体裁详解

### 建议信

| 部分 | 语块 |
|------|------|
| 开头 | I am writing to offer some suggestions on... |
| 主体 | First, it would be better if... / I suggest that... |
| | Besides, you might consider... / Another suggestion is... |
| | In my opinion, it is a good idea to... |
| 结尾 | I hope these suggestions will be helpful. |

### 申请信

| 部分 | 语块 |
|------|------|
| 开头 | I am writing to apply for the position of... |
| 自我介绍 | I am Li Hua, a student from... |
| 优势 | I believe I am qualified for this position because... |
| | I have experience in... / I am good at... |
| 结尾 | I would greatly appreciate the opportunity to... |

### 邀请信

| 部分 | 语块 |
|------|------|
| 开头 | I am writing to invite you to... |
| 活动信息 | The event will be held at [时间] in [地点]. |
| 内容 | During the activity, we will... |
| 结尾 | I sincerely hope you can join us. |

### 道歉信

| 部分 | 语块 |
|------|------|
| 开头 | I am writing to apologize for... |
| 解释 | I am sorry that I could not... because... |
| 补救 | To make up for it, I would like to... |
| 结尾 | Please accept my sincere apology. |

### 感谢信

| 部分 | 语块 |
|------|------|
| 开头 | I am writing to express my sincere gratitude for... |
| 具体感谢 | It was so kind of you to... |
| | Without your help, I would not have... |
| 结尾 | Thanks again for your kindness. |

## 三、通知（Notice）

```
标题：NOTICE / Notice（居中）

正文：
  A [活动名称] will be held [时间] at [地点].
  All the students are required to...
  The event will include the following activities:
  1. ...
  2. ...
  Those who are interested please contact [人] at [电话].

落款：
  The Student Union
  [日期]

⚠️ 注意：
• 通知不用称呼和Yours sincerely
• 时间、地点、事件、要求、联系方式缺一不可
```

## 四、演讲稿

```
称呼：
Ladies and gentlemen,
Dear judges and fellow students,

开头：
I feel greatly honored to stand here and give a speech.
Today I want to talk about...

主体：
First of all, ...
Besides, ...
Last but not least, ...

结尾（号召）：
Let's take action to...
Together, we can make a difference.
Thank you for listening!
```

## 五、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 应用文写作常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 格式错误                                             │
│     错误：书信没有称呼或落款                             │
│     正确：称呼顶格，落款在右下角                         │
│     通知：标题居中，不用称呼和落款中的Yours              │
├─────────────────────────────────────────────────────────┤
│ E2. 语块用错场景                                         │
│     错误：道歉信里写I am writing to invite you...        │
│     正确：每种体裁有固定开头语块，必须对应               │
├─────────────────────────────────────────────────────────┤
│ E3. 内容空洞，没有具体信息                               │
│     错误："We will have some activities."                │
│     正确："We will visit the science museum and          │
│           attend a lecture on AI."                      │
│     必须有具体的时间、地点、事件细节                     │
├─────────────────────────────────────────────────────────┤
│ E4. 时态错误                                             │
│     已发生的事件 → 过去时                                │
│     计划中的事件 → 将来时（will / be going to）          │
│     常态/事实 → 一般现在时                               │
├─────────────────────────────────────────────────────────┤
│ E5. 中式英语                                             │
│     错误：I very like it. / I write this letter to...    │
│     正确：I like it very much. / I am writing to...      │
│     注意副词位置和时态使用                               │
└─────────────────────────────────────────────────────────┘
```

## 六、跨模块接口

```
应用文写作
      │
      ├──→ 语文应用文（格式思维相通，语块不同）
      │
      ├──→ 阅读理解应用文（在阅读中积累地道表达）
      │
      └──→ 听力（听力中的通知、对话提供真实语料）
```

## 七、自检清单

- [ ] 格式正确（称呼、落款、日期位置）
- [ ] 体裁对应的语块使用正确（申请/邀请/道歉/感谢不混用）
• [ ] 内容有具体信息（时间、地点、事件不空洞）
- [ ] 时态正确（过去/现在/将来对应正确）
- [ ] 无中式英语，表达地道
