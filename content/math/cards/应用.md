# 应用

> 空间向量的应用，就是用向量的语言描述立体几何中的方向和位置关系——让"线面角""二面角"等度量问题变成可计算的方程。

---

## 概念层

### 问题的起源

立体几何的核心问题是空间中的角度和距离：
- 直线与平面所成的角
- 两个平面所成的二面角
- 点到平面的距离

传统几何证明需要巧妙的辅助线，而向量方法提供了一个**程序化的解题框架**：
1. 建立坐标系
2. 求出相关向量
3. 用数量积公式计算

> **一句话：向量应用把立体几何的证明和计算变成了"设坐标→求向量→套公式"的三步程序。**

---

### 本质定义

**向量应用的核心框架：**

```
立体几何问题
    │
Step 1: 建立空间直角坐标系
    │
Step 2: 将几何元素转化为向量
    │  直线 → 方向向量
    │  平面 → 法向量
    │  点 → 位置向量
    │
Step 3: 用向量运算求解
    │  垂直 → 数量积 = 0
    │  平行 → 向量共线
    │  角度 → 数量积公式 cosθ = (a·b)/(|a||b|)
    │
Step 4: 解释结果
```

---

### 核心应用类型

| 应用类型 | 核心公式 | 求解目标 |
|---------|---------|---------|
| **线面角** | sinθ = |(a·n)|/(|a||n|) | 直线与平面的夹角 |
| **面面角** | cosθ = |(n₁·n₂)|/(|n₁||n₂|) | 二面角 |
| **点到面距离** | d = |a·n|/(|n|) | 点到平面的距离 |
| **线线角** | cosθ = |(a·b)|/(|a||b|) | 异面直线所成角 |

> **关键：线面角中，直线方向向量 a 与平面法向量 n 的夹角 θ，直线与平面的夹角是 90°−θ。**

---

### 典型应用流程

**1. 线面角的向量解法**

```
已知直线 l 的方向向量 a⃗，和平面 α 的法向量 n⃗
则：sinθ = |a⃗·n⃗| / (|a⃗||n⃗|)
其中 θ 是直线 l 与平面 α 所成的角
```

**2. 二面角的向量解法**

```
已知平面 α₁ 的法向量 n₁⃗，和平面 α₂ 的法向量 n₂⃗
则：|cosθ| = |n₁⃗·n₂⃗| / (|n₁⃗||n₂⃗|)
其中 θ 是二面角（或它的补角）
```

**3. 点到平面距离**

```
已知点 P 和平面 α:P₀ 点在平面上，n⃗ 是法向量
则：d = |P₀P⃗·n⃗| / |n⃗|
```

### 概念辨析

**辨析 1：线面角 vs 异面直线所成角**

| 对比项 | 线面角 | 异面直线所成角 |
|--------|-------|--------------|
| 对象 | 直线与平面 | 两条异面直线 |
| 向量 | 方向向量与法向量 | 两个方向向量 |
| 公式 | sinθ = |a·n|/(\|a\|\|n\|) | cosθ = \|a·b\|/(\|a\|\|b\|) |
| 范围 | [0°, 90°] | [0°, 90°] |

**辨析 2：面面角 vs 二面角**

两者本质相同：
- 面面角是几何术语
- 二面角是向量术语

都指两个半平面所成的角度。

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 线面角公式 | ✓ | 适用于任何直线与平面 |
| 二面角公式 | ✓ | 需注意取锐角 |
| 点面距离 | ✓ | 必须用平面上一点 |

## 命题层

### 核心定理

**定理 1：直线与平面垂直的向量判定**

直线 l 的方向向量 a⃗ 与平面 α 的法向量 n⃗ 平行：
```
l ⊥ α ⟺ a⃗ // n⃗ ⟺ a⃗ = k·n⃗
```

**定理 2：直线与平面所成角**

```
sinθ = |a⃗·n⃗| / (|a⃗||n⃗|)
其中 θ 是直线 l 与平面 α 的夹角
```

**定理 3：二面角的平面角**

```
|cosθ| = |n₁⃗·n₂⃗| / (|n₁⃗||n₂⃗|)
```

> 注意：n₁⃗ 和 n₂⃗ 是两个平面的法向量，它们夹角的余弦的绝对值等于二面角的余弦。

**定理 4：点到平面的距离公式**

```
d = |P₀P⃗·n̂| = |P₀P⃗·n⃗| / |n⃗|
其中 n̂ 是单位法向量
```

---

### 典型例题

**例题 1：线面角**

```
题目：正方体 ABCD-A'B'C'D' 棱长为 1，求直线 BC' 与平面 ABCD 所成的角。
```

**深层分析：**

```
建立坐标系：A(0,0,0)，B(1,0,0)，C(1,1,0)，C'(1,1,1)

BC'⃗ = (0,1,1)（从 B(1,0,0) 到 C'(1,1,1)）
平面 ABCD 的法向量 n⃗ = (0,0,1)（向上）

cos(90°−θ) = |BC'⃗·n⃗| / (|BC'⃗||n⃗|) = |(0,1,1)·(0,0,1)| / (√2·1) = 1/√2

所以 sinθ = 1/√2，θ = 45°
```

BC' 与平面 ABCD 的夹角是 45°。

> **方法提炼：** 线面角 = 90° 减去方向向量与法向量的夹角。

---

**例题 2：二面角**

```
题目：正方体 ABCD-A'B'C'D' 棱长为 1，求平面 ABC' 与平面 ABCD 所成的二面角。
```

**深层分析：**

```
平面 ABCD 的法向量 n₁⃗ = (0,0,1)
平面 ABC' 的法向量：先求它的两个方向向量
  AC'⃗ = (1,1,1)
  AB⃗ = (1,0,0)
叉积 n₂⃗ = AC'⃗ × AB⃗ = ...（计算得 (1, -1, 0)）

cosθ = |n₁⃗·n₂⃗| / (|n₁⃗||n₂⃗|) = |(0,0,1)·(1,-1,0)| / (1·√2) = 0

所以 θ = 90°，二面角是 90°。
```

> **方法提炼：** 求两平面的法向量，取它们夹角的余弦绝对值。

## 方法层

### 识别信号表

| 题目关键词 | 识别信号 | 首选方法 |
|-----------|---------|---------|
| "线面角""直线与平面所成" | 直线与平面 | sinθ = \|a·n\|/(\|a\|\|n\|) |
| "二面角""两平面所成" | 两平面 | cosθ = \|n₁·n₂\|/(\|n₁\|\|n₂\|) |
| "距离""点到面" | 点到面 | d = \|a·n\|/\|n\| |
| "垂直""线面垂直" | 线面垂直 | a // n |

---

### 方法详解：向量法解立体几何流程

**Step 1: 建立坐标系**

```
选择合适的点作为原点，三个轴尽量垂直于主要平面
```

**Step 2: 求相关向量**

```
直线方向向量：沿线方向的向量差
平面法向量：用叉积（大学）或找平面上两向量求垂直
```

**Step 3: 套用公式**

```
线面角：sinθ = |a·n|/(|a||n|)
面面角：|cosθ| = |n₁·n₂|/(|n₁||n₂|)
```

**Step 4: 解释答案**

```
注意取值范围（线面角和面面角通常取 [0°, 90°]）
```

> **易错提醒：**
> 1. 线面角要用 sin，不是 cos
> 2. 二面角取锐角或用补角
> 3. 法向量要单位化吗？——实际上公式中的模会约掉，不需要单位化

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "线面角用 cos" | 混淆公式 | 应该用 sinθ = \|a·n|/(\|a\|\|n\|) |
| 2 | "二面角就是两法向量夹角" | 不考虑范围 | 二面角可能是法向量夹角或它的补角 |
| 3 | "点到面距离直接除" | 漏掉投影 | 必须用 \|a·n\|/\|n\| |
| 4 | 法向量求错 | 找错方向向量 | 法向量必须垂直于平面上所有向量 |
