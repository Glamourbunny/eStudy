# 数字特征

> 数字特征是用几个数来描述数据的"核心特点"——均值描述平均水平，方差描述分散程度。

---

## 概念层

### 问题的起源

看到一组数据，如何快速了解它们的特点？

"这批学生的平均成绩是 75 分"——用一个数字概括平均水平。

"这两个班的成绩差异大"——方差描述数据的分散程度。

均值和方差是最常用的**数字特征**。

> **一句话：数字特征是用少数几个数来概括大量数据的特征。**

---

### 本质定义

**两个核心数字特征：**

| 特征 | 含义 | 公式 |
|------|------|------|
| **均值（期望）** | 数据的平均水平 | x̄ = (Σxᵢ)/n |
| **方差** | 数据偏离均值的程度 | s² = Σ(xᵢ − x̄)²/n |

**标准差** = √方差，是方差的平方根，与数据单位相同。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **均值** | 数据的"中心位置" |
| **方差** | 数据的"分散程度" |
| **标准差** | 方差的平方根 |

---

### 等价表述

| 表述 | 公式 |
|------|------|
| **均值** | x̄ = (x₁ + x₂ + ... + xₙ)/n |
| **方差** | s² = [(x₁−x̄)² + ... + (xₙ−x̄)²]/n |
| **标准差** | s = √s² |

## 命题层

### 核心定理

**定理：均值的线性性质**

```
E(ax + b) = aE(x) + b
```

**定理：方差的性质**

```
Var(ax + b) = a²Var(x)
```

---

### 典型例题

**例题：计算一组数据的均值和方差**

```
题目：计算 3, 7, 8, 5, 2 的均值和方差。
```

**深层分析：**

```
均值：x̄ = (3+7+8+5+2)/5 = 25/5 = 5

方差：s² = [(3−5)² + (7−5)² + (8−5)² + (5−5)² + (2−5)²]/5
     = [4 + 4 + 9 + 0 + 9]/5
     = 26/5
     = 5.2
```

> **方法提炼：** 先求均值，再求各数据与均值差的平方和。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "均值能完全代表数据" | 误解 | 均值只是平均水平 |
| 2 | "方差单位是原数据单位" | 误解 | 方差单位是原数据单位的平方 |
| 3 | "标准差就是方差的平方根" | 正确 | 标准差与原数据单位相同 |
