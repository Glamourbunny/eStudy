# 空间向量模长

> 空间向量的模长，就是三维空间中向量的"长度"——从平面的 √(x²+y²) 升级到立体的 √(x²+y²+z²)。

---

## 概念层

### 问题的起源

平面向量的模长 |(x, y)| = √(x²+y²)。

空间向量的模长怎么算？

答案是：增加一个维度，公式自然推广为 √(x²+y²+z²)。

从几何上看，这是因为空间向量的长度相当于长方体的对角线长度。

> **一句话：空间向量的模长是勾股定理在三维空间的推广——平方和再开方。**

---

### 原始定义（教材原文）

空间向量 $\vec{a} = (a_x, a_y, a_z)$ 的模（长度）为：

```
|\vec{a}| = √(a_x² + a_y² + a_z²)
```

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 5 页

---

### 本质定义

**模长的本质是向量的"大小"：**

```
向量的模 = 从起点到终点的直线距离
        = √(沿 x 走了 a_x + 沿 y 走了 a_y + 沿 z 走了 a_z 的综合效果)
```

**物理类比：**
从原点到点 P(a_x, a_y, a_z) 的位移向量，位移大小（模）就是走过的路程。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **坐标分量** | a_x, a_y, a_z |
| **平方和** | a_x² + a_y² + a_z² |
| **开方** | 取非负平方根 |

---

### 等价表述

| 表述 | 公式 |
|------|------|
| **坐标形式** | \|a\| = √(a_x² + a_y² + a_z²) |
| **点表示** | A(x₁,y₁,z₁)，O(0,0,0)，则 \|OA\| = √(x₁²+y₁²+z₁²) |
| **向量差** | \|AB\| = √((x_B−x_A)²+(y_B−y_A)²+(z_B−z_A)²) |

### 概念辨析

**辨析：空间向量模长 vs 平面向量模长**

| 对比项 | 平面向量 | 空间向量 |
|--------|---------|---------|
| 公式 | √(x²+y²) | √(x²+y²+z²) |
| 维度 | 二维 | 三维 |
| 几何意义 | 长方形对角线 | 长方体对角线 |
| 关系 | 空间向量模长当 z=0 时退化为平面向量 | |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 模长 ≥ 0 | ✓ | 永远非负 |
| 模长 = 0 | ⟺ a_x = a_y = a_z = 0 | 零向量 |
| 模长计算 | ✓ | 三个分量都不能漏 |

## 命题层

### 核心定理

**定理 1：模长的坐标计算**

```
\vec{a} = (a_x, a_y, a_z)
|\vec{a}| = √(a_x² + a_y² + a_z²)
```

**定理 2：两点间距离（模的特殊情况）**

```
A(x₁,y₁,z₁)，B(x₂,y₂,z₂)
|AB| = √((x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)²)
```

**定理 3：向量加法的模不等式**

```
||\vec{a}| − |\vec{b}|| ≤ |\vec{a} + \vec{b}| ≤ |\vec{a}| + |\vec{b}|
```

---

### 典型例题

**例题：求空间向量的模长**

```
题目：已知 \vec{a} = (3, 4, 0)，求 |\vec{a}|。
```

**深层分析：**

```
|\vec{a}| = √(3² + 4² + 0²) = √(9 + 16) = √25 = 5
```

> **方法提炼：** 模长就是勾股定理，向量分量就是直角三角形两条直角边。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "模长""长度""|a|" | 求向量大小 | √(a_x² + a_y² + a_z²) |
| "距离""AB" | 两点距离 | \|AB\| = √(Δx²+Δy²+Δz²) |
| "单位向量" | 标准化 | a/\|a\| |

---

### 方法详解：模长计算步骤

```
Step 1: 确认向量的三个坐标分量
Step 2: 平方：分别求出 a_x², a_y², a_z²
Step 3: 求和：a_x² + a_y² + a_z²
Step 4: 开方：取非负平方根
```

> **易错提醒：** 模长一定是正数或零，不能是负数。

| 编号 | 常见误区 | 错误根源 | 正确理解 |
