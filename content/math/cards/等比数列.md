# 等比数列

> 等比数列是"每一步都按同样比例放大或缩小"的数列——从第二项起，每一项与前一项的比都是同一个常数。

---

## 概念层

### 问题的起源

现实生活中，很多现象呈现出"等比"（等比例）的特点：
- 细胞分裂：每个细胞分裂成两个，一变二、二变四、四变八...
- 复利增长：本金加利息后，利息计入下期本金继续产生利息
- 放射性衰变：每经过一个半衰期，原子核数量减半

这类"每一步都按固定比例变化"的现象，用数学语言描述就是**等比数列**。

> **一句话：** 等比数列是"指数增长/衰减"的离散模型——每一步乘以固定的比例因子。

---

### 原始定义（教材原文）

如果一个数列从第 2 项起，每一项与它的前一项的比都等于同一个常数，那么这个数列就叫做**等比数列**，这个常数叫做等比数列的**公比**，通常用 q 表示。

> 出处：人教版《普通高中教科书·数学（B版）选择性必修第二册》第 16 页

**定义拆解：**

| 关键词 | 含义 | 示例 |
|--------|------|------|
| 从第2项起 | 第一项没有前一项可比较 | a₂/a₁ = q |
| 同一个常数 | 公比 q 固定不变 | q 可以是正、负（但不能为 0） |
| 公比 q | 衡量"比例"大小 | q > 1 递增，0 < q < 1 衰减，q < 0 变号交替 |

---

### 本质定义（逻辑重构版）

**等比数列的核心特征：**

```
a₂ = a₁ · q
a₃ = a₂ · q = a₁ · q²
a₄ = a₃ · q = a₁ · q³
...
aₙ = a₁ · q^(n−1)
```

**等比数列 = 首项 × 公比的 (项数−1) 次方**

---

### 等比数列的通项公式

```
aₙ = a₁ · q^(n−1)

其中：
a₁ —— 首项
q  —— 公比（q ≠ 0）
n  —— 项数
```

**通项公式的变形：**

```
aₙ = aₘ · q^(n−m)    （已知第 m 项 aₘ，求第 n 项）
q  = ⁿ⁻ᵐ√(aₙ/aₘ)     （由两项求公比，需注意正负号）
```

---

### 公比 q 对数列的影响

| 公比范围 | 数列特征 | 示例 |
|---------|---------|------|
| q > 1 | 指数增长（若 a₁ > 0） | 1, 2, 4, 8, 16, ... |
| q = 1 | 常数列 | 3, 3, 3, 3, ... |
| 0 < q < 1 | 指数衰减（趋向 0） | 1, 1/2, 1/4, 1/8, ... |
| q = 0 | 退化数列（从第二项起全为 0） | 5, 0, 0, 0, ... |
| q < 0 | 符号交替 | 1, −2, 4, −8, ... |

---

### 邻近概念辨析

**辨析：等比数列 vs 等差数列 —— "比例变化" vs "差值变化"**

| 对比项 | 等比数列 | 等差数列 |
|--------|---------|---------|
| 核心特征 | 每一步乘以固定比例 q | 每一步增加固定量 d |
| 关系式 | aₙ₊₁ / aₙ = q（比相同） | aₙ₊₁ − aₙ = d（差相同） |
| 通项公式 | aₙ = a₁·q^(n−1) | aₙ = a₁ + (n−1)d |
| 增长特性 | 指数增长/衰减 | 线性增长/衰减 |
| 求和公式 | 涉及指数函数 | 线性求和 |

**辨析：等比数列 vs 指数函数 —— "离散" vs "连续"**

| 对比项 | 等比数列 | 指数函数 |
|--------|---------|---------|
| 定义域 | 正整数 {1, 2, 3, ...} | 实数 R |
| 表示 | 离散的点 | 连续的曲线 |
| 相同点 | 都涉及指数运算 | 增速/衰减规律相同 |

---

### 适用边界

| 条件 | 说明 |
|------|------|
| q ≠ 0 | 公比不能为零（否则从第二项起全为 0） |
| a₁ ≠ 0 | 首项为零时，数列恒为零 |
| q 的符号 | 影响数列项的符号变化 |

## 命题层

### 核心定理

**定理 1：等比数列的通项公式**

```
aₙ = a₁ · q^(n−1)
```

**定理 2：等比中项**

若 a, b, c 成等比数列，则 b² = ac，b 叫做 a 与 c 的等比中项。

```
等比中项公式：b = ±√(ac)
（注意：等比中项有两个，可能同号也可能异号）
```

**定理 3：等比数列的前 n 项和**

当 q ≠ 1 时：

```
Sₙ = a₁(1 − q^n) / (1 − q)
    = a₁(q^n − 1) / (q − 1)   （等价形式，便于记忆）
```

当 q = 1 时：

```
Sₙ = n·a₁（常数列）
```

**定理 4：等比数列的判定**

数列 {aₙ} 是等比数列 ⟺ aₙ₊₁ / aₙ = 常数（对所有 n 成立，且各项非零）

---

### 典型例题

**例题 1：求等比数列的通项**

```
题目：等比数列满足 a₃ = 12，a₅ = 48，求通项公式和 a₇。
```

**深层分析：**

```
分析：已知两项，可求公比和首项

由通项公式：
a₃ = a₁·q² = 12  ... (1)
a₅ = a₁·q⁴ = 48  ... (2)

(2)/(1)：(a₁·q⁴)/(a₁·q²) = 48/12
       q² = 4
       q = ±2

情况1：q = 2，代入 (1)：a₁·4 = 12 → a₁ = 3
       aₙ = 3·2^(n−1)，a₇ = 3·2⁶ = 192

情况2：q = −2，代入 (1)：a₁·4 = 12 → a₁ = 3
       aₙ = 3·(−2)^(n−1)，a₇ = 3·(−2)⁶ = 192

两种情况 a₇ 相同，但数列为：3, −6, 12, −24, 48, −96, 192, ...

答案：a₇ = 192
```

> **方法提炼：** 已知等比数列任意两项，可通过比值求公比。注意 q 有正负两种可能。

---

**例题 2：等比数列前 n 项和**

```
题目：等比数列 a₁ = 2，q = 3，求前 5 项和。
```

**解题步骤：**

```
S₅ = a₁(1 − q⁵)/(1 − q)
    = 2·(1 − 3⁵)/(1 − 3)
    = 2·(1 − 243)/(−2)
    = 2·(−242)/(−2)
    = 242
```

> **方法提炼：** 使用等比数列求和公式时，注意 q ≠ 1 的条件。

---

**例题 3：等比数列的实际应用**

```
题目：某种细菌每半小时分裂一次，每次每个细菌分裂成 2 个。初始有 1000 个细菌，4 小时后有多少细菌？
```

**深层分析：**

```
分析：每半小时分裂一次，4 小时 = 8 个半小时周期

a₁ = 1000（初始）
q = 2（每个分裂成 2 个，即数量翻倍）
n = 8 + 1 = 9（第 1 次计数是初始状态，第 9 次是 8 次分裂后）

a₉ = a₁·q^(9−1) = 1000·2⁸ = 1000·256 = 256,000

答案：4 小时后有 256,000 个细菌
```

> **方法提炼：** 实际问题中注意确定 n 的值——初始状态算第一项，分裂 k 次后算第 k+1 项。

## 方法层

### 方法 1：等比数列的判断

**核心思想：** 验证"比是否相等"（且各项非零）。

**标准操作流程：**

```
判断 {aₙ} 是否为等比数列：

Step 1: 检查各项是否非零（若有为 0 的项则不是等比数列）
Step 2: 计算 a₂/a₁, a₃/a₂, ..., aₙ/aₙ₋₁
Step 3: 检查这些比是否都相等
Step 4: 若相等则是等比数列，否则不是
```

---

### 方法 2：等比数列求和的推导

**核心思想：** 利用错位相减法。

**推导过程：**

```
Sₙ = a₁ + a₁q + a₁q² + ... + a₁q^(n−1)

两边乘以 q：
q·Sₙ = a₁q + a₁q² + ... + a₁q^(n−1) + a₁q^n

错位相减（第一式减去第二式）：
Sₙ − q·Sₙ = a₁ − a₁q^n
(1−q)·Sₙ = a₁(1 − q^n)

∴ Sₙ = a₁(1 − q^n)/(1 − q)（q ≠ 1）
```

---

### 方法 3：等比中项的应用

**核心思想：** 利用等比中项关系建立方程。

**注意：** 等比中项可能是两个值（正负），需要根据题目条件判断。

**识别信号表：**

| 题干关键词 | 识别信号 | 首选方法 |
|-----------|---------|---------|
| "已知 aₘ 和 aₙ" | 两已知项 | 通项公式建立方程组 |
| "三个数成等比数列" | 等比中项 | 设 a/q, a, aq 或 a, aq, aq² |
| "求等比数列前 n 项和" | 求和 | 公式直接应用 |
| "复利/衰减问题" | 实际问题 | 等比数列模型 |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "等比数列公比不能为负" | 记忆固化 | q 可以是负数，导致数列符号交替 |
| 2 | "等比数列求和公式 Sₙ = a₁(1−q^n)/(1−q)" | 忘写前提 | 该公式仅在 q ≠ 1 时成立 |
| 3 | "q=0 时也是等比数列" | 误解定义 | q=0 时，从第二项起全为 0，教材一般不认为这是等比数列 |
| 4 | "等比中项只有一个" | 混淆等差等比 | 等比中项 b² = ac，所以 b = ±√(ac)，有两个可能 |
| 5 | "等比数列各项同号" | 忽略 q<0 情况 | q<0 时，数列符号交替 |
