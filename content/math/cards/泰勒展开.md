# 泰勒展开

> 泰勒展开是用多项式逼近复杂函数的艺术——把"不会求的"变成"会求的多项式"。

---

## 概念层

### 问题的起源

sin x, eˣ, ln(1+x) 这些函数计算起来很麻烦，但多项式计算简单（只有加、减、乘）。

泰勒提出：能否用多项式近似这些复杂函数？在某一点附近，多项式和原函数有相同的值、相同的导数、相同的二阶导数……

> **一句话：** 泰勒展开是用多项式在一点附近"复制"原函数所有阶导数信息的逼近方法。

---

### 本质定义（逻辑重构版）

**泰勒公式：**

```
f(x) = f(x₀) + f'(x₀)(x−x₀) + f''(x₀)/2! · (x−x₀)² + ... + f⁽ⁿ⁾(x₀)/n! · (x−x₀)ⁿ + Rₙ

其中 Rₙ 是余项，描述近似误差
```

**麦克劳林展开（x₀ = 0）：**

```
eˣ = 1 + x + x²/2! + x³/3! + ... + xⁿ/n! + ...
sin x = x − x³/3! + x⁵/5! − ... + (−1)ⁿx²ⁿ⁺¹/(2n+1)! + ...
cos x = 1 − x²/2! + x⁴/4! − ... + (−1)ⁿx²ⁿ/(2n)! + ...
ln(1+x) = x − x²/2 + x³/3 − ... + (−1)ⁿ⁻¹xⁿ/n + ...
```

---

### 泰勒展开的几何理解

```
用多项式"贴合"原函数：

在 x = x₀ 处：
- 函数值相同：f(x₀) = P(x₀)
- 一阶导数相同：f'(x₀) = P'(x₀)
- 二阶导数相同：f''(x₀) = P''(x₀)
...

多项式 P(x) 在 x₀ 附近非常接近 f(x)
```

---

### 适用边界

| 要点 | 说明 |
|------|------|
| 收敛性 | 泰勒级数不一定收敛，收敛也不一定收敛到原函数 |
| 余项 | Rₙ 描述误差大小，n 越大误差越小 |
| 麦克劳林 | x₀ = 0 的特殊情形，最常用 |

---

## 方法层

### 方法：泰勒展开的应用

**核心思想：** 用多项式近似复杂函数。

**典型应用：**

| 函数 | 泰勒展开（麦克劳林） | 适用场景 |
|------|-------------------|---------|
| eˣ | 1 + x + x²/2! + ... | 近似计算 e 的幂 |
| sin x | x − x³/3! + x⁵/5! − ... | 近似计算正弦值 |
| cos x | 1 − x²/2! + x⁴/4! − ... | 近似计算余弦值 |
| ln(1+x) | x − x²/2 + x³/3 − ... | 近似计算对数 |
