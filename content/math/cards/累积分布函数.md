# 累积分布函数

> 累积分布函数 F(x) = P(X ≤ x)，是"随机变量小于等于 x 的概率"，通过它可以求任意区间的概率。

---

## 概念层

### 问题的起源

已知正态分布 N(100, 15²)，求 P(X ≤ 115)。

这就是累积分布函数 F(x) = P(X ≤ x) 的应用。

F(x) 把"概率分布"变成了"一个函数"，可以通过查表或计算得到任意 x 的累积概率。

> **一句话：累积分布函数 F(x) 是随机变量"小于等于 x"的概率。**

---

### 本质定义

**累积分布函数的定义：**

```
F(x) = P(X ≤ x)
```

**性质：**

| 性质 | 说明 |
|------|------|
| 单调非减 | x₁ < x₂ ⟹ F(x₁) ≤ F(x₂) |
| 范围 | 0 ≤ F(x) ≤ 1 |
| 右连续 | F(x) 右连续 |

---

### 与概率密度函数的关系

```
连续型：F(x) = ∫_{-∞}^{x} f(t)dt
离散型：F(x) = Σ_{xᵢ ≤ x} pᵢ
```

F(x) 是 f(x) 的积分（离散时是求和）。

---

## 命题层

### 核心定理

**定理：概率计算**

```
P(a < X < b) = F(b) - F(a)
```

**定理：标准正态分布的 CDF**

```
Φ(x) = P(Z ≤ x)，Z ~ N(0, 1)
```

---

### 典型例题

**例题：计算累积概率**

```
题目：已知 X ~ N(100, 15²)，求 P(X ≤ 115)。
```

**深层分析：**

```
标准化：Z = (115-100)/15 = 1
P(X ≤ 115) = P(Z ≤ 1) = Φ(1) ≈ 0.8413
```

> **方法提炼：** 标准化后查 Φ(1) 的值。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "F(x) = f(x)" | 混淆 | F 是累积，f 是密度 |
| 2 | "P(X > x) = 1 - F(x)" | 部分正确 | P(X ≥ x) = 1 - F(x)，但需注意连续型时 P(X=x)=0 |
| 3 | "F(x) 总是连续的" | 误解 | 离散型时 F(x) 是阶梯函数 |
