# 平抛运动

> 平抛运动是"水平抛出物体的运动"——它可以分解为水平方向的匀速直线运动和竖直方向的自由落体运动。

---

## 概念层

### 问题的起源

当一个物体被水平抛出（初速度水平，不计空气阻力），它会做什么运动？

答案是：平抛运动。

这是因为：
- 水平方向不受力（忽略空气阻力），初速度 v₀ 保持不变 → 匀速直线运动
- 竖直方向只受重力，从静止开始加速 → 自由落体运动

两个方向的运动**独立进行，互不影响**。

> **一句话：平抛运动 = 水平匀速运动 + 竖直自由落体运动的合成。**

---

### 原始定义（教材原文）

物体以一定的初速度水平抛出，在忽略空气阻力的情况下，物体只在重力作用下做的运动称为**平抛运动**。

> 出处：人教版《普通高中教科书·物理（必修第二册）》

---

### 本质定义

**平抛运动的特点：**

| 方向 | 受力 | 初始速度 | 运动类型 |
|------|------|---------|---------|
| 水平 | 0 | v₀ | 匀速直线运动 |
| 竖直 | mg（重力） | 0 | 自由落体运动 |

**运动方程：**

```
水平：x = v₀t，vx = v₀
竖直：y = (1/2)gt²，vy = gt
```

**速度：**

```
v = √(vx² + vy²) = √(v₀² + g²t²)
方向：tanθ = vy/vx = gt/v₀
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **初速度 v₀** | 水平抛出，速度为 v₀ |
| **重力加速度 g** | g ≈ 9.8 m/s² |
| **运动分解** | 水平匀速 + 竖直自由落体 |

---

### 等价表述

| 表述 | 公式 |
|------|------|
| **轨迹方程** | y = gx²/(2v₀²)（抛物线） |
| **飞行时间** | t = √(2h/g)（只取决于抛出高度 h） |
| **水平射程** | s = v₀√(2h/g)（与 v₀ 成正比） |

### 概念辨析

**辨析：平抛 vs 斜抛**

| 对比项 | 平抛运动 | 斜抛运动 |
|--------|---------|---------|
| 初速度方向 | 水平 | 有角度（斜向上或斜向下） |
| 竖直初速度 | 0 | 不为 0 |
| 轨迹 | 抛物线（对称） | 抛物线（不对称） |
| 飞行时间 | 只取决于高度 | 还取决于初速度的竖直分量 |

## 命题层

### 核心定理

**定理 1：运动分解原理**

```
水平分运动：匀速直线运动
竖直分运动：自由落体运动
两者独立，互不影响。
```

**定理 2：落地速度**

```
v = √(v₀² + (gt)²)
落地角度：tanθ = vy/vx = gt/v₀
```

**定理 3：射程公式**

```
水平射程：s = v₀t = v₀√(2h/g)
结论：射程与 v₀ 成正比，与 h 的平方根成正比
```

---

### 典型例题

**例题：平抛运动的基本计算**

```
题目：一人站在高 h = 20 m 的平台上，以 v₀ = 10 m/s 的速度水平抛出一个小球。
      求：(1) 小球飞行时间；(2) 水平射程；(3) 落地速度。
      （g = 10 m/s²）
```

**深层分析：**

```
(1) 飞行时间：t = √(2h/g) = √(40/10) = √4 = 2 s

(2) 水平射程：s = v₀t = 10 × 2 = 20 m

(3) 落地速度：
vy = gt = 10 × 2 = 20 m/s
v = √(v₀² + vy²) = √(100 + 400) = √500 ≈ 22.4 m/s
落地角度：tanθ = vy/vx = 20/10 = 2
θ ≈ arctan(2) ≈ 63.4°
```

> **方法提炼：** 平抛问题 → 分解为水平和竖直两个方向 → 分别计算 → 合成。

---

### 方法详解：平抛运动问题的处理流程

**Step 1: 分解运动**

```
水平方向：匀速直线运动，v = v₀，x = v₀t
竖直方向：自由落体，v_y = gt，y = (1/2)gt²
```

**Step 2: 找已知量**

```
题目给了什么？h, v₀, s, t, v ...?
```

**Step 3: 选择公式求解**

```
已知 h → 求 t = √(2h/g)
已知 t → 求 y, vy
求 s → s = v₀t
求 v → v = √(v₀² + vy²)
```

> **易错提醒：** 竖直方向的初速度是 0，不是 v₀。
