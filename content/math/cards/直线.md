# 直线

> 直线是解析几何的起点——用方程描述直线，让几何问题变成了代数计算。

---

## 概念层

### 问题的起源

解析几何的核心思想是：用代数方程描述几何曲线。

直线是最简单的曲线。用方程描述直线，可以让：
- 求交点变成解方程组
- 判断平行垂直变成比较斜率
- 求距离变成套公式

> **一句话：直线的方程让"画图看"变成了"算一算"。**

---

### 原始定义（教材原文）

**斜截式：** y = kx + b（k 是斜率，b 是纵截距）

**点斜式：** y − y₀ = k(x − x₀)（已知点 (x₀, y₀) 和斜率 k）

**两点式：** (y − y₁)/(y₂ − y₁) = (x − x₁)/(x₂ − x₁)（已知两点）

**一般式：** Ax + By + C = 0（A、B 不全为零）

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 30-35 页

---

### 本质定义

**直线方程的本质是"一次函数的图像"：**

```
y = kx + b
│ │  │
│ │  └─ 截距（与 y 轴的交点 y = b）
│ └──── 斜率（倾斜程度，k = tanθ）
└────── 一次函数（自变量 x 的最高次数是 1）
```

**斜率 k 的几何意义：**

```
k > 0：向右上方倾斜（递增）
k = 0：水平线
k < 0：向右下方倾斜（递减）
k → +∞：垂直线
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **斜率 k** | 倾斜程度，k = tanθ |
| **截距 b** | 与 y 轴的交点 (0, b) |
| **方向向量** | (1, k)（斜率为 k 的直线的方向向量） |

---

### 等价表述

| 形式 | 方程 | 适用场景 |
|------|------|---------|
| **斜截式** | y = kx + b | 已知斜率和截距 |
| **点斜式** | y − y₀ = k(x − x₀) | 已知一点和斜率 |
| **两点式** | (y − y₁)/(y₂ − y₁) = (x − x₁)/(x₂ − x₁) | 已知两点 |
| **一般式** | Ax + By + C = 0 | 通用形式，垂直线也能表示 |
| **参数式** | x = x₀ + t, y = y₀ + kt | 大学常用 |

---

### 图像表示

**斜率的几何意义：**

```
        y
        │
    b ──┼──
        │╲
        │  ╲ k > 0（递增）
        │   ╲
        │    ╲
        │────╲────────→ x
        │
        │╱
        │╲ k < 0（递减）
```

### 概念辨析

**辨析：直线方程的各种形式**

| 形式 | 方程 | 限制 |
|------|------|------|
| 斜截式 | y = kx + b | 不能表示垂直线（k → ∞） |
| 点斜式 | y − y₀ = k(x − x₀) | 不能表示垂直线 |
| 一般式 | Ax + By + C = 0 | 可以表示所有直线 |

当直线垂直于 x 轴时（x = 常数），斜率无穷大，无法用斜截式，但可以用一般式（如 x = 3 → x − 3 = 0）。

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 斜截式表示任意直线 | ✗（不能表示垂直线） | 垂直线用 x = 常数 |
| 一般式表示任意直线 | ✓ | A、B 不全为零 |
| 平行于 y 轴 | k → ∞，斜截式失效 | 用 x = 常数 |

## 命题层

### 核心定理

**定理 1：两直线平行的判定**

```
l₁: y = k₁x + b₁
l₂: y = k₂x + b₂

l₁ // l₂ ⟺ k₁ = k₂ 且 b₁ ≠ b₂
```

**定理 2：两直线垂直的判定**

```
l₁: y = k₁x + b₁
l₂: y = k₂x + b₂

l₁ ⊥ l₂ ⟺ k₁·k₂ = −1
```

**定理 3：点到直线的距离**

```
点 P(x₀, y₀) 到直线 Ax + By + C = 0 的距离：
d = |Ax₀ + By₀ + C| / √(A² + B²)
```

---

### 典型例题

**例题：判断直线位置关系**

```
题目：判断以下两组直线的位置关系：
(1) l₁: y = 2x + 1, l₂: y = 2x − 3
(2) l₁: y = 2x + 1, l₂: y = −(1/2)x + 3
```

**深层分析：**

(1) k₁ = k₂ = 2，b₁ ≠ b₂，所以 l₁ // l₂

(2) k₁·k₂ = 2 × (−1/2) = −1，所以 l₁ ⊥ l₂

> **方法提炼：** 斜率相同则平行，斜率乘积为 −1 则垂直。

---

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "平行""//" | 斜率相等 | k₁ = k₂ |
| "垂直""⊥" | 斜率乘积 −1 | k₁·k₂ = −1 |
| "求距离" | 点到线 | d = \|Ax₀+By₀+C\|/√(A²+B²) |
| "交点""解方程组" | 求交点 | 解方程组 |

---

### 方法详解：直线方程选择指南

**已知条件** → **首选形式**：

| 已知 | 首选 | 说明 |
|------|------|------|
| 斜率 k 和截距 b | 斜截式 y = kx + b | 最直接 |
| 一点和斜率 | 点斜式 y − y₀ = k(x − x₀) | 变形即得 |
| 两点 | 两点式，然后化简 | 先算斜率 k = (y₂−y₁)/(x₂−x₁) |
| 一般描述 | 一般式 Ax + By + C = 0 | 最通用 |

> **易错提醒：** 垂直于 x 轴的直线（x = 常数）不能用斜截式，要用一般式。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "垂直就是 k₁·k₂ = 1" | 记反 | 是 −1 |
| 2 | "垂直线斜率无穷大" | 误解 | 垂直线没有有限斜率 |
| 3 | "平行就是 b 相等" | 混淆 | 平行要求 k 相等且 b 不等 |
| 4 | "两点式直接写" | 忘检查分母 | 当 x₁ = x₂ 时两点式不适用 |
