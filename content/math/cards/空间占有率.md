# 空间占有率

> 空间占有率是晶体中原子所占体积与晶体总体积的比值——它衡量原子堆积的紧密程度。

---

## 概念层

### 问题的起源

原子堆在一起，有多少空间被原子"实打实地占据"？

这是一个关于"效率"的问题。

- 体心立方（BCC）：原子占据了 68% 的空间
- 面心立方（FCC）和六方最密（HCP）：原子占据了 74% 的空间

这个比例叫做**空间占有率**或**致密度**。

> **一句话：空间占有率是原子体积占晶体总体积的比例，衡量原子堆积的紧密程度。**

---

### 原始定义（教材原文）

空间占有率（致密度）是指晶胞中原子所占体积与晶胞体积之比。

> 出处：人教版《普通高中教科书·化学（选择性必修）》

---

### 本质定义

**定义公式：**

```
空间占有率 = n × (4/3)πr³ / a³
```

其中：
- n 是每个晶胞中的原子数
- r 是原子半径
- a 是晶胞边长

**常见数值：**

| 堆积类型 | 空间占有率 |
|---------|-----------|
| 体心立方（BCC） | 68% |
| 面心立方（FCC） | 74% |
| 六方最密（HCP） | 74% |
| 六方简单堆积 | 60% |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **原子体积** | n × (4/3)πr³ |
| **晶胞体积** | a³ |
| **比例** | 原子体积 / 晶胞体积 |

---

### 等价表述

| 表述 | 说明 |
|------|------|
| **致密度** | 同一个概念的不同名称 |
| **堆积因子** | Packing Factor |
| **利用率** | 空间被利用的程度 |

---

### 典型堆积的致密度

**体心立方（BCC）：**

```
致密度 = 2 × (4/3)πr³ / a³
其中 a = 4r/√3（几何关系）
致密度 = √3π/8 ≈ 68%
```

**面心立方/六方最密：**

```
致密度 = 4 × (4/3)πr³ / a³
其中 a = 2√2r（FCC）或 a = 2r（HCP 的底面边长）
致密度 = √2π/3 ≈ 74%
```

### 概念辨析

**辨析：堆积密度 vs 配位数**

| 对比项 | 堆积密度（空间占有率） | 配位数 |
|--------|---------------------|--------|
| 衡量 | 体积效率 | 接触紧密度 |
| 数值范围 | 0 ~ 100% | 无限（但实际有限） |
| 最大值 | 74%（最密堆积） | 12 |

两者都衡量堆积的紧密程度，但角度不同。

## 命题层

### 核心定理

**定理 1：空间占有率公式**

```
空间占有率 = nV_atom / V_cell
            = n × (4/3)πr³ / a³
```

**定理 2：最密堆积的极限**

```
最密堆积的空间占有率 = √2π/3 ≈ 74%
这是球堆积能达到的最高密度
```

---

### 典型例题

**例题：计算面心立方的空间占有率**

```
题目：铜晶体为面心立方结构，原子半径 r = 1.28 Å。
      计算铜晶体的空间占有率。
```

**深层分析：**

```
FCC 晶胞：
- 原子数 n = 4
- 原子半径 r = 1.28 Å
- 晶胞边长 a = 2√2r = 2.828 × 1.28 ≈ 3.62 Å

原子总体积 = 4 × (4/3)π(1.28)³
晶胞体积 = (3.62)³

空间占有率 = 4 × (4/3)π(1.28)³ / (3.62)³
           ≈ 74%
```

> **方法提炼：** FCC 的空间占有率理论上就是 √2π/3 ≈ 74%。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "空间占有率""致密度" | 堆积密度 | nV_atom / V_cell |
| "最密堆积" | 74% | FCC 或 HCP |
| "体心立方" | 68% | BCC |

---

### 方法详解：空间占有率的计算流程

**Step 1: 确定晶胞类型**

```
FCC → n = 4，a = 2√2r
BCC → n = 2，a = 4r/√3
```

**Step 2: 计算体积**

```
原子总体积 = n × (4/3)πr³
晶胞体积 = a³
```

**Step 3: 求比值**

```
空间占有率 = 原子总体积 / 晶胞体积
```
