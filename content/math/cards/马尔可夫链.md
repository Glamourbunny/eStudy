# 马尔可夫链

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 马尔可夫链在高考中以"状态转移"题出现——一个系统有几个状态，每步按固定概率跳转，且"只看当前，不看历史"。你的任务是认出这道题属于马尔可夫链，然后画转移图，写好转移矩阵。

---

## 视角层（像什么）

### 核心隐喻：失忆的醉汉——下一步只取决于这一步，不记得之前走过的路

**马尔可夫链 = 一个失忆的人在城市里逛——他每次选择"下一步去哪"，只取决于"他现在站在哪"，完全不记得自己从哪来的。**

```
  时间 t=0：醉汉在广场 A
  时间 t=1：他走到 B（概率 0.6）或留在 A（概率 0.4）
  时间 t=2：他站在 B，基于 B 选择下一步去哪

  关键：时间 t=2 时，他从 B 出发的选择，
  和他之前"从 A 走到 B"的历史毫无关系！
  
  如果醉汉有记忆："我上次从 A 走到了 B，这次我换个方向"
  → 这就不是马尔可夫链了
```

**马尔可夫链的"失忆法则"（马尔可夫性质）：**

| 要素 | 直觉理解 | 数学含义 |
|------|---------|---------|
| **状态空间** | 城市的几个地点（A, B, C…） | 所有可能的状态集合 S |
| **转移概率** | 从某地 A 出发，去 B 的概率 | P(X_{n+1}=B \| X_n=A)，与 n 无关 |
| **无记忆性** | 醉汉不记得怎么来的，只看现在站在哪 | P(X_{n+1} \| X_n, X_{n-1}, …) = P(X_{n+1} \| X_n) |
| **转移矩阵** | 一张"从哪到哪的概率表" | P_{ij} = P(状态i → 状态j) |

> **一句话直觉：** 马尔可夫链是一个"只认当前位置"的随机游走。未来只和现在有关，和历史无关。高考题中，只要题目说"每次选择只取决于当前状态"或画了一个状态转移图——这就是马尔可夫链。

### 动态想象：天气的"健忘"模型

```
  最简单的马尔可夫链——天气预测：
  
  状态：{晴天, 雨天}
  
  转移规则（只依赖今天）：
  • 今天晴天 → 明天晴天 0.8，明天下雨 0.2
  • 今天下雨 → 明天晴天 0.3，明天下雨 0.7
  
  转移矩阵 P：
       明天晴  明天雨
  今天晴 [0.8    0.2]
  今天雨 [0.3    0.7]
  
  问题：如果今天晴天，后天晴天的概率？
  
  拆解：
  后天 = 两步之后的状态
  路径1：晴 → 晴 → 晴：0.8 × 0.8 = 0.64
  路径2：晴 → 雨 → 晴：0.2 × 0.3 = 0.06
  总概率 = 0.64 + 0.06 = 0.70
  
  n 步之后 → 用 P^n（转移矩阵的 n 次方）
```

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **以为有记忆** | "之前都是晴天，明天应该也是晴天" | 马尔可夫链只看今天！即使连续晴了 30 天，明天还是 P(晴\|晴)=0.8 |
| **稳态=停止** | "极限分布是系统停下来了" | 稳态（平稳分布）只是分布不再变，状态本身还在跳！好比人的体温维持在 37°C，但分子还在疯狂运动 |
| **一步转移 ≠ 多步转移** | "转移矩阵用一次就够了" | n 步转移 = P^n，不是 n×P |
| **吸收态不会出** | "到了吸收态还有概率跳出来" | 吸收态意味着 P(自己→自己) = 1，一旦进入永远出不来 |

## 联结层（连什么）

### 跨学科锚点

**1. 计算机：PageRank（Google 搜索算法）**

```
  网页之间的跳转 = 马尔可夫链：
  状态 = 每个网页
  转移概率 = 从一个网页点击链接跳到另一个的概率
  
  随机浏览者的稳态分布 = 网页的 PageRank 值
  ↓ Google 用它来排名搜索结果
  ↓ 这就是马尔可夫链稳态分布最有名的应用
```

**2. 物理：布朗运动**

```
  花粉颗粒在水中被水分子撞击，做无规则运动
  下一次的位置只取决于当前位置 + 随机撞击
  ↓ 连续状态 + 连续时间 = 马尔可夫过程的物理原型
  ↓ 爱因斯坦 1905 年用这个解释了布朗运动，证明了原子存在
```

**3. 生物：种群遗传（Hardy-Weinberg 之前的直觉模型）**

```
  基因型频率的世代传递 = 马尔可夫链：
  状态 = {AA, Aa, aa}
  转移 = 随机交配后的下一代基因型分布
  ↓ 经过几代后趋近稳态（Hardy-Weinberg 平衡）
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 画好状态转移图，求 n 步后某状态概率 | 这是马尔可夫链 n 步转移 | 状态转移矩阵 P，求 P^n 的对应位置 |
| 求稳态（极限）分布 | 设 π = πP，解方程 | πP = π, Σπ_i = 1 |
| 是否有吸收态 | 找 P_{ii}=1 的状态 | 吸收马尔可夫链分析 |
| 递推数列与转移矩阵 | 概率递推 a_{n+1} = p·a_n + q | 转移矩阵 P 的特征根 |

**首选工具**：状态转移矩阵 P（从转移图画）
**次选工具**：递推公式（单个状态的概率递推）

### 历史/哲学视角：马尔可夫的"语言链"

```
1906-1913 年（马尔可夫的研究）：
  安德雷·马尔可夫，俄国数学家
  他不是在研究随机过程——他在研究普希金《尤金·奥涅金》里的元音和辅音序列
  
  他发现：一个字母是元音还是辅音的概率，只与前一个字母有关
  ↓ 这就是"马尔可夫链"的起源——不是为了物理，而是为了语言学

  马尔可夫用这个模型证明了一个重要的统计学定理（大数定律的扩展）
  ↓ 但他的"链"变成了 20 世纪最重要的数学工具之一
  ↓ 从气体分子运动到股票价格、从语音识别到 DNA 序列分析
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | "每次选择只取决于当前状态""系统在几个状态间切换""状态转移图如下" | 不知道应构建转移矩阵，试图用古典概型 + 树状图逐层算 | 3 秒画出转移图，写出 P，n 步后概率 = P^n |
| 🔀 **嵌套** | 两次马尔可夫链叠加："先按规则 A 转移，再按规则 B 转移"；或状态又是另一层马尔可夫链 | 两套转移概率混合，算不出 | 识别两层各自独立的转移矩阵，乘积（顺序注意） |
| 🆕 **新定义** | "定义随机过程 {Xn} 满足……""若 P(X_{n+1}\|X_n, …) = P(X_{n+1}\|X_n)，则称…" | 对新定义的无记忆条件不理解，不知是马尔可夫链 | 识别"只看当前"这一核心条件 → 这就是马尔可夫性质 → 套标准框架 |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"状态转移""与历史无关""只取决于当前"？
    → 这就是马尔可夫链
  • 画状态转移图：哪些状态？箭头从哪到哪？概率多少？

Step 2：剥外壳找骨骼（4 秒）
  • 用转移图写出转移矩阵 P（每行和 = 1）
  • 明确初始状态分布（向量 π₀）
  • 看题目要求：一步？n 步？极限分布？

Step 3：锁定 Formal 着陆点（3 秒）
  • n 步后 → P^n
  • 极限：πP = π + Σπ_i = 1（解线性方程组）
  • 吸收概率：基本矩阵 (I-Q)⁻¹
  • 写下：拆包→【状态转移矩阵】，立即切换
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 转移图已画好，P 矩阵已写出 | 用矩阵乘法求 P^n，或列递推方程组 |
| 嵌套题 | 两层马尔可夫链各自独立 | 先独立解每层的转移，再按顺序组合（注意状态空间是否变化） |
| 新定义题 | 确认"无记忆"条件 + 状态空间 | 按定义写转移矩阵，套标准公式 |

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **忘记检查行和为 1** | 转移矩阵某行和 ≠ 1（漏了可能转移） | 每行代表"从该状态出发的所有可能去向"，概率和必须 = 1 |
| **P^n ≠ (P_{ij})^n** | 误把每个元素 n 次方当作 n 步转移概率 | n 步转移是矩阵的 n 次幂，不是元素的 n 次方 |
| **稳态 = 不动** | 认为极限分布意味着状态不再切换 | 稳态是分布不动，状态仍在跳。投硬币无限次，正面概率趋近 0.5，但每次结果还在变 |
| **吸收态存在却忽略了** | 有吸收态时还求"所有状态的极限分布" | 吸收态出现后极限分布退化为某些状态概率=1 |

## 一句话总结

> **马尔可夫链是一个"失忆"的随机游走——下一步只看这一步，不管历史。高考里它以状态转移图和概率箭头登场，你的任务是 3 秒画出转移图、写出转移矩阵 P，然后根据题目要求：一步直接用 P，n 步用 P^n，稳态解 πP=π。**
