# 古典概型

> 古典概型是"等可能"的概率——当每个基本事件发生概率相等时，P(A) = A 中包含的基本事件数 / 所有基本事件数。

---

## 概念层

### 问题的起源

抛硬币，出现正面和反面的概率是多少？

直觉告诉我们：各占 1/2。

为什么？

因为"正面"和"反面"这两个基本事件是**等可能**的——每种结果发生的概率相同。

这就是**古典概型**。

> **一句话：古典概型是每个基本事件等可能发生的概率模型。**

---

### 原始定义（教材原文）

在一次试验中，所有可能出现的结果是有限的，并且每种结果出现的可能性都相等的概率模型称为**古典概型**。

对于古典概型，事件 A 的概率：

```
P(A) = A 中包含的基本事件数 / 所有基本事件数
```

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 10 章

---

### 本质定义

**古典概型的两个条件：**

| 条件 | 含义 |
|------|------|
| **有限性** | 所有可能结果是有限个 |
| **等可能性** | 每个结果发生的可能性相同 |

**概率公式：**

```
P(A) = m(A) / m(Ω) = A中基本事件数 / 基本事件总数
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **基本事件** | 不可再分的最简单事件 |
| **等可能性** | 每种结果概率相同 |
| **有限性** | 基本事件数有限 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **P(A)** | 事件 A 的概率 |
| **P(A) = m/n** | m 是 A 中基本事件数，n 是总基本事件数 |
| **0 ≤ P(A) ≤ 1** | 概率的范围 |

## 命题层

### 核心定理

**定理：古典概型的概率公式**

```
P(A) = m/n
其中 m 是 A 包含的基本事件数
     n 是基本事件总数
```

**推论：**

```
P(Ω) = 1
P(∅) = 0
对任意事件 A：0 ≤ P(A) ≤ 1
```

---

### 典型例题

**例题：掷骰子**

```
题目：掷一枚均匀骰子，求出现偶数点的概率。
```

**深层分析：**

```
基本事件总数：6（点 1, 2, 3, 4, 5, 6）
偶数点事件：3 个（2, 4, 6）

P(偶数点) = 3/6 = 1/2
```

> **方法提炼：** 数基本事件数 → P(A) = 有利数/总数。
