# 差分

> 差分是"离散版本的微分"——Δy = y(x+1) - y(x)，它是研究离散数据变化率的工具。

---

## 概念层

### 问题的起源

微分适用于连续函数，但现实中的数据往往是离散的：
- 每天的股票价格
- 每年的 GDP
- 每隔 5 分钟测量的温度

对于这些离散数据，用**差分**代替微分：
- Δy = y(x+h) - y(x) 代替 dy/dx

> **一句话：差分是离散版本的微分，用于处理离散数据。**

---

### 本质定义

**差分的定义：**

```
一阶差分：Δy = y(x+1) - y(x)
二阶差分：Δ²y = Δ(Δy) = y(x+2) - 2y(x+1) + y(x)
```

**与微分对应的概念：**

| 连续（微分） | 离散（差分） |
|-------------|-------------|
| dy/dx | Δy/Δx |
| 导数 | 差商 |
| 微分方程 | 差分方程 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **一阶差分** | Δy = y(x+1) - y(x) |
| **二阶差分** | Δ²y = y(x+2) - 2y(x+1) + y(x) |
| **差商** | Δy/Δx = (y(x+1)-y(x))/1 |

---

## 命题层

### 核心定理

**定理：差分与数列**

```
数列 aₙ 的差分：Δaₙ = aₙ₊₁ - aₙ
```

**定理：差分方程**

```
Δ²y = 0 ⟹ y是一次式
Δ²y = c ⟹ y是二次式
```
