# 组合数

> 组合数是"从 n 个东西里选出 k 个"的方法数——记作 Cₙᵏ，等于从 n 个不同元素中取出 k 个的组合数。

---

## 概念层

### 问题的起源

从 5 个人中选 2 个人去比赛，有多少种选法？

不是排列（因为选出来的人不需要排队），而是组合。

5 个人选 2 个的选法数 = 5×4/2 = 10 种。

这就是组合数 C₅² = 10。

> **一句话：组合数 Cₙᵏ 是从 n 个不同元素中选出 k 个，不考虑顺序的方法数。**

---

### 原始定义（教材原文）

从 n 个不同元素中，任取 k（k ≤ n）个元素并成一组，叫做从 n 个不同元素中取出 k 个元素的**组合**。

组合的个数叫做**组合数**，记作 Cₙᵏ 或 (ₙᵏ)。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修三》第 2-5 页

---

### 本质定义

**组合数的公式：**

```
Cₙᵏ = n! / (k!(n−k)!)
    = n(n−1)...(n−k+1) / k!
```

**组合 vs 排列：**

| 对比项 | 组合（无序） | 排列（有序） |
|--------|-------------|-------------|
| 是否考虑顺序 | 不考虑 | 考虑 |
| 公式 | Cₙᵏ = n!/(k!(n−k)!) | Pₙᵏ = n!/(n−k)! |
| 例子 | 5人中选2人 | 5人排成2队 |

Cₙᵏ 只关心"选谁"，Pₙᵏ 关心"谁排前面"。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **n 个元素** | 所有候选元素 |
| **选 k 个** | 不放回抽取 |
| **不考虑顺序** | 只要选出来的组合 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **符号** | Cₙᵏ 或 (ₙᵏ) |
| **阶乘形式** | n!/(k!(n−k)!) |
| **乘积形式** | n(n−1)...(n−k+1)/k! |

---

### 性质

**性质 1：对称性**

```
Cₙᵏ = Cₙⁿ⁻ᵏ
```

**性质 2：递推关系**

```
Cₙᵏ = Cₙ₋₁ᵏ⁻¹ + Cₙ₋₁ᵏ
```

**性质 3：加和**

```
Cₙ⁰ + Cₙ¹ + ... + Cₙⁿ = 2ⁿ
```

### 概念辨析

**辨析：组合 vs 排列**

| 对比项 | 组合 Cₙᵏ | 排列 Pₙᵏ |
|--------|----------|----------|
| 考虑顺序 | ✗ | ✓ |
| 公式 | n!/(k!(n−k)!) | n!/(n−k)! |
| 关系 | Pₙᵏ = Cₙᵏ × k! | Cₙᵏ = Pₙᵏ / k! |

## 命题层

### 核心定理

**定理 1：组合数公式**

```
Cₙᵏ = n! / (k!(n−k)!)
    = n(n−1)...(n−k+1) / k!
```

**定理 2：帕斯卡法则**

```
Cₙᵏ = Cₙ₋₁ᵏ⁻¹ + Cₙ₋₁ᵏ
```

**定理 3：二项式定理中的系数**

```
Cₙ⁰, Cₙ¹, ..., Cₙⁿ 是二项式 (a+b)ⁿ 展开式的系数
```

---

### 典型例题

**例题：计算 C₅²**

```
题目：计算 C₅² = ?
```

**深层分析：**

```
C₅² = 5! / (2!·3!) = 120 / (2·6) = 10
```

或者用乘积形式：
C₅² = (5×4) / (2×1) = 10

> **方法提炼：** Cₙᵏ = n!/[k!(n−k)!]，或直接用乘积形式。

| "对称""Cₙᵏ=Cₙⁿ⁻ᵏ" | 简化计算 | 用对称性 |

---

### 方法详解：组合数计算

**Step 1: 判断 n 和 k**

**Step 2: 用公式计算**

```
Cₙᵏ = n!/(k!(n−k)!)
```

**Step 3: 用对称性简化**

```
如果 k > n/2，用 Cₙᵏ = Cₙⁿ⁻ᵏ 简化
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "C₅² = 5×4 = 20" | 忘记除以 2! | C₅² = 5×4/2 = 10 |
| 2 | "组合和排列混淆" | 混淆 | 组合无序，排列有序 |
| 3 | "Cₙᵏ = Cₙᵏ⁻¹" | 计算错误 | Cₙᵏ = Cₙ₋₁ᵏ⁻¹ + Cₙ₋₁ᵏ |
