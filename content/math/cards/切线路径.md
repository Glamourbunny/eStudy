# 切线路径

> 切线是"与曲线仅有一个公共点的直线"——对于圆锥曲线，切线的代数定义是"代入方程后 Δ = 0"。

---

## 概念层

### 问题的起源

切线是研究曲线性质的重要工具：

- 物理中，瞬时速度的方向是切线方向
- 切线斜率是导数的几何意义
- 圆锥曲线的切线在光学、力学中有重要应用

传统几何中，切线是"与曲线仅接触一次的直线"。解析几何中，切线方程可以通过代数方法精确求出。

> **一句话：切线是"刚好碰到曲线一次就离开"的直线。**

---

### 原始定义（教材原文）

**圆的切线：** 与圆只有一个公共点的直线。

**圆锥曲线的切线：** 将曲线方程 F(x, y) = 0 代入直线方程 y = kx + m，整理成一元二次方程，当 Δ = 0 时直线与曲线相切。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 45-50 页

---

### 本质定义

**切线的代数本质：**

```
曲线 F(x, y) = 0
直线 y = kx + m
代入 → 变成 x 的一元二次方程
Δ = 0 → 只有一个解 → 相切
```

**切线的几何本质：**

```
切线 T 在 P 点"刚好接触"曲线 C
T 与 C 的交点是 P（重根，P 是二重交点）
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **切点** | 切线与曲线的接触点 |
| **唯一交点** | 代数上 Δ = 0，几何上只有一个公共点 |
| **切线斜率** | k = tan(切线与 x 轴夹角) |

---

### 等价表述

| 曲线 | 切线方程 | 说明 |
|------|---------|------|
| **圆** | 圆上一点 P(x₀, y₀) 的切线：(x₀−a)(x−a)+(y₀−b)(y−b)=r² | 半径垂直于切线 |
| **椭圆** | P(x₀, y₀) 的切线：xx₀/a² + yy₀/b² = 1 | 弦线公式法 |
| **抛物线** | y² = 2px，P(y₀, 2px₀/p) 的切线：yy₀ = p(x+x₀) | 代入法 |
| **一般二次曲线** | F(x,y)=0，P(x₀,y₀) 的切线：F_x(x₀,y₀)(x−x₀)+F_y(x₀,y₀)(y−y₀)=0 | 偏导数法 |

## 命题层

### 核心定理

**定理 1：圆的切线方程**

```
圆 (x−a)² + (y−b)² = r² 上一点 P(x₀, y₀)：
(x₀−a)(x−a) + (y₀−b)(y−b) = r²
```

**定理 2：椭圆切线方程**

```
椭圆 x²/a² + y²/b² = 1 上一点 P(x₀, y₀)：
xx₀/a² + yy₀/b² = 1
```

**定理 3：抛物线切线方程**

```
抛物线 y² = 2px 上一点 P(x₀, y₀)：
yy₀ = p(x + x₀)
```

---

### 典型例题

**例题：求椭圆切线**

```
题目：求椭圆 x²/9 + y²/4 = 1 上一点 P(2, 4/3) 的切线方程。
```

**深层分析：**

```
椭圆：x²/9 + y²/4 = 1
P(2, 4/3) 满足：4/9 + 16/36 = 4/9 + 4/9 = 8/9 ≠ 1

实际上 P(2, 4/3)：4/9 + 16/36 = 4/9 + 4/9 = 8/9 ≠ 1

重新计算，P 的 y 坐标可能不是 4/3...

假设 P(x₀, y₀) 在椭圆上
切线：xx₀/9 + yy₀/4 = 1

代入 x₀ = 2（如果椭圆上）：
2²/9 + y₀²/4 = 1
y₀²/4 = 5/9
y₀ = ±(2√5)/3

这里取 y₀ = 2√5/3 ≈ 1.49
切线方程：2x/9 + (2√5/3)y/4 = 1
```

> **方法提炼：** 椭圆切线方程用 xx₀/a² + yy₀/b² = 1。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |

| "切线""切点" | 切线问题 | 判断曲线类型，用对应公式 |
| "Δ=0" | 代数法 | 代入后判别式为零 |
| "求斜率" | 导数法 | 用导数求切线斜率 |

---

### 方法详解：切线方程求法

**方法 1：代入法（通用）**

```
Step 1: 设切线 y = kx + m
Step 2: 代入曲线方程，整理成 x 的方程
Step 3: 令 Δ = 0
Step 4: 由切点条件求 k, m
```

**方法 2：公式法（针对特定曲线）**

```
圆：直接用公式
椭圆/双曲线：xx₀/a² ± yy₀/b² = 1
抛物线：yy₀ = p(x + x₀)
```

> **易错提醒：** 切线方程中的 (x₀, y₀) 必须是切点坐标。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "Δ > 0 是切线" | 混淆 | Δ = 0 才是切线 |
| 2 | "圆上一点切线斜率不存在" | 误解 | 水平切线时斜率才不存在 |
| 3 | "忘记判断点在曲线上" | 粗心 | 切点必须在曲线上 |
