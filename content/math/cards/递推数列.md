# 递推数列

> 递推数列是"用前几项来定义后一项"的数列——通过递推关系 aₙ = f(aₙ₋₁, ...) 可以求出通项公式。

---

## 概念层

### 问题的起源

斐波那契数列：1, 1, 2, 3, 5, 8, ...

这个数列不是用公式定义的，而是用递推关系：
- a₁ = 1, a₂ = 1
- aₙ = aₙ₋₁ + aₙ₋₂（n ≥ 3）

这就是**递推数列**。

> **一句话：递推数列通过 aₙ 与 aₙ₋₁ 等的关系来定义。**

---

### 本质定义

**递推数列的形式：**

```
a₁, a₂, ..., aₙ 的初始值
aₙ = f(aₙ₋₁)      （一阶递推）
aₙ = f(aₙ₋₁, aₙ₋₂)（二阶递推）
```

---

### 常见类型

| 类型 | 递推关系 | 通项特征 |
|------|---------|---------|
| 等差数列 | aₙ = aₙ₋₁ + d | 线性 |
| 等比数列 | aₙ = q·aₙ₋₁ | 指数 |
| 斐波那契 | aₙ = aₙ₋₁ + aₙ₋₂ | 黄金比例近似 |

---

## 命题层

### 核心定理

**定理：特征根法（线性齐次递推）**

```
对于 aₙ = p₁aₙ₋₁ + p₂aₙ₋₂
特征方程：x² = p₁x + p₂
解特征根 λ₁, λ₂
通项：aₙ = A·λ₁ⁿ + B·λ₂ⁿ
```
