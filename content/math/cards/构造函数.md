# 构造函数

> 构造函数是"为了解决问题而人为制造一个函数"——它是数学证明和求解中的重要技巧。

---

## 概念层

### 问题的起源

有些数学问题直接求解很难，但如果你能"构造"一个合适的辅助函数，问题就变得简单了。

例如：证明不等式 f(x) > 0，如果能构造 F(x) = eˣf(x)，则只需证明 F(x) > 0。

> **一句话：构造函数是把问题化难为易的技巧——找一个好用的"桥梁函数"。**

---

### 本质定义

**构造函数的核心思想：**

```
原问题：A（难）
构造辅助函数：F（让 A 变得简单）
解决：通过 F 解决 A
```

**常见构造模式：**

| 目的 | 构造方式 |
|------|---------|
| 证明不等式 | 移项构造 F(x) = f(x) - g(x) |
| 证明等式 | 构造 F(x) = 左边 - 右边 |
| 解方程 | 构造 F(x) = 方程左边 |
| 证明恒成立 | 构造 F(x) 并分析单调性 |

---

### 方法详解：构造函数步骤

**Step 1: 明确目标**

```
要证明/求解什么？
```

**Step 2: 分析原问题的结构**

```
把问题变形，找到需要简单化的形式
```

**Step 3: 构造辅助函数**

```
常见策略：
- 移项：F = f - g
- 指数化：F = eˣf
- 对数化：F = ln f
- 复合化：F = f(g(x))
```

**Step 4: 解决新问题**

```
研究 F 的性质
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "构造很随意" | 误解 | 需要有目的地构造 |

| 3 | "构造唯一的标准方法" | 误解 | 同一问题可能有多种构造方式 |
