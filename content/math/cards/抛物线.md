# 抛物线

> 抛物线是"到一个焦点和一条准线距离相等的点的轨迹"——它是投掷物体在重力作用下的运动轨迹，也是手电筒反射面的数学基础。

---

## 概念层

### 问题的起源

抛物线最初来自物理学的抛体运动：在忽略空气阻力的情况下，抛出的物体（如炮弹、篮球）的运动轨迹就是抛物线。

数学上的抛物线定义更加一般化：
- 有一个焦点 F
- 有一条准线 l
- 抛物线上的点到焦点 F 的距离 = 到准线 l 的距离

这个定义与物理轨迹的区别是：物理轨迹有起点，数学抛物线是完整的曲线。

**应用：**
- 手电筒和汽车前灯的反光面（旋转抛物面）使光平行反射
- 雷达天线（旋转抛物面）使接收信号聚焦
- 卫星天线（旋转抛物面）同样原理

> **一句话：抛物线是"到焦点和到准线等距"的点的轨迹。**

---

### 原始定义（教材原文）

平面内到一个定点 F 和一条定直线 l（l 不经过 F）距离相等的点的轨迹称为**抛物线**。

定点 F 称为**焦点**，定直线 l 称为**准线**，焦点 F 到准线 l 的距离称为**焦距**，记作 p。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 66-70 页

---

### 本质定义

**抛物线的标准方程（焦点在 x 轴上，p > 0）：**

```
y² = 2px（开口向右）
y² = −2px（开口向左）
x² = 2py（开口向上）
x² = −2py（开口向下）
```

**焦点性质：**

|PF| = |Pl|（P 到焦点 F 的距离 = P 到准线 l 的距离）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **焦点 F** | 定点 |
| **准线 l** | 不过焦点的定直线 |
| **焦距 p** | 焦点到准线的距离（p > 0） |
| **离心率** | e = 1（抛物线的定义） |

---

### 等价表述

| 表述 | 方程 | 焦点 | 准线 | 开口 |
|------|------|------|------|------|
| **标准式1** | y² = 2px | (p/2, 0) | x = −p/2 | 右 |
| **标准式2** | y² = −2px | (−p/2, 0) | x = p/2 | 左 |
| **标准式3** | x² = 2py | (0, p/2) | y = −p/2 | 上 |
| **标准式4** | x² = −2py | (0, −p/2) | y = p/2 | 下 |
| **定义式** | \|PF\| = \|Pl\| | — | — | — |

---

**y² = 2px（开口向右）：**

```

            │      ╲
```

### 邻近概念网络

```
              圆锥曲线
                  │
      ┌───────────┼───────────┐
      │           │           │
     椭圆        双曲线       抛物线
      │           │           │
 |PF₁|+|PF₂|=2a ||PF₁|−|PF₂|=2a |PF|=|Pl|（e=1）
```

**与椭圆、双曲线的对比：**

| 曲线 | 离心率 e | 定义特征 |
|------|---------|---------|
| 椭圆 | 0 < e < 1 | 到两焦点距离之和 |
| 抛物线 | e = 1 | 到焦点和准线等距 |
| 双曲线 | e > 1 | 到两焦点距离之差 |

---

### 概念辨析

**辨析 1：抛物线 vs 椭圆 vs 双曲线**

| 对比项 | 抛物线 | 椭圆 | 双曲线 |
|--------|--------|------|------|
| 定义 | \|PF\| = \|Pl\| | \|PF₁\| + \|PF₂\| = 2a | \|\|PF₁\| − \|PF₂\|\| = 2a |
| 离心率 | e = 1 | 0 < e < 1 | e > 1 |
| 焦点数 | 1 | 2 | 2 |
| 准线 | 有 | 无 | 无 |
| 形状 | 一支 | 封闭曲线 | 两支 |

**辨析 2：开口方向与方程的关系**

| 方程 | 焦点 | 准线 | 开口方向 |
|------|------|------|---------|
| y² = 2px | (p/2, 0) | x = −p/2 | 向右（p > 0） |
| y² = −2px | (−p/2, 0) | x = p/2 | 向左 |
| x² = 2py | (0, p/2) | y = −p/2 | 向上 |
| x² = −2py | (0, −p/2) | y = p/2 | 向下 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| p > 0 | ✓ | 焦距 p 永远是正数 |
| 抛物线上点 P | \|PF\| = \|Pl\| | 定义 |
| 离心率 | e = 1 | 恒等于 1 |

## 命题层

### 核心定理

**定理 1：抛物线的标准方程**

```
y² = 2px（开口向右）
焦点 (p/2, 0)，准线 x = −p/2
```

**定理 2：抛物线的切线方程**

```
抛物线 y² = 2px 上一点 P(x₀, y₀)：
yy₀ = p(x + x₀)
```

**定理 3：抛物线的光学性质**

抛物线的切线具有**反射性质**：
- 从焦点发出的光线，经抛物线反射后平行于对称轴
- 反过来，平行光经抛物面反射后汇聚于焦点

这应用于手电筒、卫星天线、雷达等。

---

### 典型例题

**例题 1：求抛物线方程**

```
题目：已知抛物线的焦点在 (3, 0)，求它的标准方程。
```

**深层分析：**

```
焦点 (3, 0) 在 x 轴上，所以是 y² = 2px 形式
焦点 = (p/2, 0) = (3, 0)
p/2 = 3 ⟹ p = 6

方程：y² = 2×6×x = 12x
```

> **方法提炼：** 焦点在 x 轴 → y² = 2px，p = 2 × 焦点 x 坐标。

---

**例题 2：判断开口方向**

```
题目：判断 y² = −4x 的开口方向、焦点和准线。
```

**深层分析：**

```
y² = −4x
这是 y² = −2px 形式
−2p = −4 ⟹ p = 2

开口向左
焦点： (−p/2, 0) = (−1, 0)
准线：x = p/2 = 1
```

> **方法提炼：** 系数为负 → 开口向左；y² = 2px 的 p 由系数除以 2 得到。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "焦点在 x 轴" | y² = 2px | 确定 p |
| "焦点在 y 轴" | x² = 2py | 确定 p |
| "求方程" | 需确定 p | 焦点或准线位置求 p |
| "开口方向" | 看 x 或 y 系数符号 | 系数正数开向正方向 |

---

### 方法详解：抛物线方程求法流程

**Step 1: 确定开口方向**

```
开口向右 → y² = 2px
开口向左 → y² = −2px
开口向上 → x² = 2py
开口向下 → x² = −2py
```

**Step 2: 确定 p**

```
焦点 (p/2, 0) 或准线 x = −p/2
由已知条件求出 p
```

**Step 3: 写出方程**

```
代入 p 值
```

> **易错提醒：** p 永远是正数，开口方向由方程形式决定，不是由 p 决定。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "y² = 2px 的 p 是焦距" | 混淆 | p 是焦点到准线的距离，焦点在 (p/2, 0) |
| 2 | "抛物线离心率 e 变化" | 记错 | 抛物线 e = 1，恒定 |
| 3 | "开口向下时 y² 还是正" | 混淆 | 向下时是 x² = −2py，与 y² 无关 |
| 4 | "抛物线没有焦点" | 误解 | 抛物线有且仅有一个焦点 |
