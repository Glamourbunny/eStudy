# 物理轨迹

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 物理轨迹题是"几何+时间"的混血儿——它长着几何的身体（轨迹是曲线），但血液里流着物理的时间（位置随时间变化）。这张卡片让你认出：轨迹方程就是"消去时间 t 后的 x-y 关系"。

---

## 视角层（像什么）

### 核心隐喻：篮球出手后的飞行弧线

**物理轨迹 = 你投篮时，篮球在空中划出的那条抛物线**

```
  你站在罚球线，篮球出手：
  
  水平方向：匀速向前（忽略空气阻力），x = v₀·t·cosθ
  竖直方向：先上升后下降，y = v₀·t·sinθ - (1/2)gt²
  
  两个方向同时运动，t 是隐藏的"导演"——它同时控制 x 和 y
  消去 t，得到 y 关于 x 的表达式 → 一条抛物线！
```

**轨迹的三层结构：**

| 层次 | 投篮场景 | 数学语言 |
|------|---------|---------|
| **参数方程** | 球在时刻 t 的位置 (x(t), y(t)) | x = f(t), y = g(t) |
| **轨迹方程** | 消去时间，只看球的路线形状 | F(x, y) = 0（不出现 t） |
| **速度信息** | 球在每个位置飞多快、往哪飞 | 导数：v_x = dx/dt, v_y = dy/dt |

> **一句话直觉：** 物理轨迹 = 你在球场上投出的那个球——水平匀速、竖直有加速度，合起来就是抛物线。t 是隐身导演，消掉它你就看到路线的形状。

### 动态想象：画沙画的沙漏

```
  一个沙漏在平面上移动，同时漏沙
  
  x 方向：沙漏沿 x 轴以速度 a 移动 → x(t) = at
  y 方向：沙漏沿 y 轴以速度 b 移动 → y(t) = bt
  
  沙漏漏下的沙子形成一条直线 → y = (b/a)x
  
  但如果 y 方向是加速的：y(t) = (1/2)bt²
  沙子就形成一条抛物线 → y = (b/(2a²))x²
```

**高考里的翻版：**
- "质点从原点出发，水平方向匀速、竖直方向匀加速，求轨迹方程"——消去 t 即得抛物线
- "两质点同时出发，求它们最近距离"——距离 = f(t)，求导得最小值
- "质点在磁场中做圆周运动，求轨迹"——洛伦兹力 = 向心力，轨迹是圆弧

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **参数方程和轨迹方程混为一谈** | "x=2t, y=t² 就是轨迹方程" | 含 t 的是参数方程，消去 t 之后的 y 与 x 的关系才是轨迹方程。两者信息量不同（参数方程包含了"时间信息"） |
| **水平匀速+竖直匀加速 = 抛物线，不是任何情况** | "有加速度就是抛物线" | 要水平没有加速度、竖直有恒定加速度，且两个方向独立——这是平抛/斜抛的条件。如果 x 方向也有加速度，轨迹就不是抛物线了 |
| **轨迹方程只能描述路径形状** | "有了轨迹方程就知道一切" | 轨迹方程丢失了"时间信息"——你知道球的路线长什么样，但不知道它什么时候到哪。要求最短距离、相遇时间，必须回到参数方程 |

## 联结层（连什么）

### 跨学科锚点

**1. 体育：跳远的空中姿态**

```
  跳远运动员起跳后：
  
  重心轨迹 = 抛物线（平抛运动，水平匀速+竖直自由落体）
  
  手的摆动、腿的折叠——影响的是姿态，不影响重心轨迹！
  重心一旦离地，轨迹就由初速度和重力决定了（动量守恒+外力=重力）
```

**2. 计算机：游戏中的子弹弹道**

```python
# FPS 游戏中子弹的轨迹
bullet_x = start_x + v * t * cos(angle)
bullet_y = start_y + v * t * sin(angle) - 0.5 * g * t**2

# 每帧更新 t，计算新的 (x, y)
# 物理引擎每秒钟运算 60 次（60 FPS）
```

**3. 物理：带电粒子在匀强磁场中的圆周运动**

```
  电子以速度 v 垂直射入匀强磁场 B
  
  洛伦兹力 F = qvB（始终垂直于速度方向）
  → 向心力 = mv²/R = qvB
  → R = mv/(qB)（轨迹是半径为 R 的圆）
  
  轨迹方程：x²+(y-R)² = R²（圆心在 (0, R)）
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 平抛/斜抛轨迹 | 水平匀速 + 竖直匀加速 → 消 t 得抛物线 | 运动分解 + 消元法 |
| 磁场圆周运动 | 洛伦兹力=向心力 → 圆弧 | R = mv/(qB) |
| 两质点最近距离 | 距离 d(t) = 函数 → 求最小值 | 求导：d'(t)=0 |
| 相遇问题 | 两质点的参数方程在同一时刻重合 | x₁(t)=x₂(t) 且 y₁(t)=y₂(t) |

**首选工具**：`运动分解法`——把运动拆成两个独立方向，分别写参数方程，再消 t
**次选工具**：能量法——不需要知道时间细节时，用动能定理/机械能守恒

### 历史/哲学视角

**伽利略是第一个把"轨迹"用数学描述的人。** 他提出"抛体运动可以分解为水平的匀速运动（惯性）和竖直的匀加速运动（重力）"。这两个运动独立进行、互不干扰——这就是"运动的独立性原理"。

这个洞察是革命性的：在此之前，人们觉得抛出去的东西是先直线上升再直线下落。伽利略用数学证明了它是一条抛物线。**从此，人类学会了把复杂的物理运动"拆成"几个简单的独立分量来研究。**

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | 物理题问"轨迹方程""轨道形状""运动路径" | 用物理公式硬推，卡在时间参数上 | 认出"轨迹=消去 t 的 x-y 关系"，先写参数方程再消 t |
| 🔀 **嵌套** | "质点先平抛再进入磁场做圆周运动" | 两种运动模式切换混乱 | 分段处理：平抛段→抛物线，入磁场→圆弧，衔接点满足连续性 |
| 🆕 **新定义** | 较少以新定义出现 | — | — |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"运动轨迹""轨道""从 A 运动到 B" → 物理轨迹模式
  • 问：有哪几个方向的运动？各是什么性质（匀速/匀加速/变加速）？

Step 2：剥外壳找骨骼（4 秒）
  • 写出每个方向的参数方程（用 t 表示 x 和 y）
  • 如果要轨迹方程 → 消去 t，得 y = f(x) 或 F(x,y) = 0
  • 如果要求距离/相遇 → 保留 t，用参数方程计算

Step 3：锁定 Formal 着陆点（3 秒）
  • 写下：拆包→【运动分解+消元/参数方程求导】
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 运动的分解完成，参数方程写出 | 消去 t 得轨迹方程，或联立参数方程求交点 |
| 嵌套题 | 各段运动模式已识别 | 分段写参数方程，衔接点用位置和速度连续条件 |

**违反此铁律的后果**：运动分解时方向搞反（如竖直方向忘记重力加速度向下是负值），导致轨迹方程符号全错。

---

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **忘记轨迹方程丢失时间信息** | 用轨迹方程求最短距离时找不到时间 | 最短距离（动态问题）→必须用参数方程，距离 d(t) 求导 |
| **水平竖直混淆** | 把 y 方向的加速度写到 x 方向 | 先画受力分析图，确认加速度的方向，再分配参数方程 |
| **参数方程消 t 时代数出错** | 复杂的 sin/cos 消元时搞错 | 用 sin²+cos²=1 消元是最常用技巧 |

---

## 一句话总结

> **物理轨迹 = 篮球划出的抛物线——运动的两个方向同时进行，时间 t 是"隐身导演"。轨迹方程就是"辞退导演 t 后，x 和 y 自己的直接对话"。遇到轨迹题，先写参数方程（含 t），再根据问题决定消不消 t。**
