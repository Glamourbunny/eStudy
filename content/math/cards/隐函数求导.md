# 隐函数求导

> 隐函数求导是在不显式解出 y 的情况下，直接从方程 F(x, y) = 0 中求 dy/dx 的技巧。

---

## 概念层

### 问题的起源

有些函数关系难以或无法写成 y = f(x) 的显式形式：
- 圆：x² + y² = r²（y = ±√(r² − x²)，分成上下两支）
- 心形线：x² + y² = (x² + y² − ax)³ − a²y² = 0

强行显式化可能很复杂或不可能。隐函数求导允许我们直接对原方程求导。

> **一句话：** 隐函数求导是"戴着镣铐跳舞"——在 F(x, y) = 0 的约束下，直接对 x 求导，然后把 dy/dx 当作未知数解出来。

---

### 本质定义（逻辑重构版）

**隐函数定理（简单版）：**

若方程 F(x, y) = 0 在某点附近能确定一个可导函数 y = f(x)，则：

```
∂F/∂x + (∂F/∂y) · (dy/dx) = 0

即：dy/dx = −(∂F/∂x) / (∂F/∂y)
```

**实际操作——对方程两边同时对 x 求导：**

```
F(x, y) = 0
    ↓
对方程两边关于 x 求导
    ↓
遇到 y 的项：使用链式法则 d/dx [f(y)] = f'(y) · (dy/dx)
    ↓
解出 dy/dx
```

---

### 隐函数求导的步骤

```
Example: x² + y² = r² 求 dy/dx

Step 1: 对方程两边同时对 x 求导
        d/dx (x² + y²) = d/dx (r²)
        2x + 2y · (dy/dx) = 0

Step 2: 解出 dy/dx
        2y · (dy/dx) = −2x
        dy/dx = −x/y
```

---

### 邻近概念辨析

**辨析：隐函数求导 vs 显函数求导 —— "约束下求导" vs "直接求导"**

| 对比项 | 隐函数求导 | 显函数求导 |
|--------|-----------|-----------|
| 函数形式 | F(x, y) = 0（y 未单独解出） | y = f(x)（y 已单独解出） |
| 求导方法 | 方程两边同时对 x 求导 | 直接套用求导公式 |
| 链式法则 | 需要处理 y 项的 dy/dx | 无需处理 |
| 典型应用 | 圆、椭圆等曲线 | 常见初等函数 |

## 方法层

### 方法：隐函数求导标准流程

**核心思想：** 在约束条件下求导。

**标准操作流程：**

```
Step 1: 确定方程 F(x, y) = 0
Step 2: 对方程两边关于 x 求导
        - x 的项正常求导
        - y 的项用链式法则：d/dx [f(y)] = f'(y) · y'
        - 常数项导数为 0
Step 3: 收集含有 y'（即 dy/dx）的项
Step 4: 解出 y' = dy/dx
Step 5: （可选）用原方程代入，简化结果
```

---

### 方法：对数隐函数求导（取对数法）

**核心思想：** 对难处理的乘积、幂、指数形式，取对数简化。

```
Example: y = xˣ 求 dy/dx

Step 1: 取对数 ln y = x ln x
Step 2: 两边对 x 求导：(1/y) · y' = ln x + 1
Step 3: 解出：y' = y(ln x + 1) = xˣ(ln x + 1)
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "对 y 求导时忘乘 dy/dx" | 链式法则应用不完整 | d/dx [y²] = 2y · (dy/dx) |
| 2 | "把 y 当作 x 一样求导" | 混淆变量角色 | y 是 x 的函数，必须用链式法则 |
| 3 | "隐函数求导结果要用 y 表示 x" | 误解简化目标 | 结果可以含有 y，这是正常的 |
