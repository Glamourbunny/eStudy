# 万有引力

> 万有引力是宇宙中所有物体之间都存在的相互吸引的力——它让苹果落地，也束缚地球绕太阳公转。

---

## 概念层

### 问题的起源

牛顿思考的问题：
- 为什么苹果会落地？
- 为什么月亮绕地球转而不飞走？
- 行星为什么绕太阳运动？

牛顿的伟大发现是：**苹果落地和月亮公转是同一种力**——万有引力。

这个力的大小与质量成正比，与距离平方成反比：

```
F = G·M·m/r²
```

> **一句话：万有引力是任何两个有质量的物体之间都存在的吸引力。**

---

### 原始定义（教材原文）

任意两个物体之间的万有引力大小为：

```
F = G·M·m/r²
```

其中 G 是万有引力常量（6.67×10⁻¹¹ N·m²/kg²），M 和 m 是两物体的质量，r 是两物体质心间的距离。

> 出处：人教版《普通高中教科书·物理（必修第二册）》第十章

---

### 本质定义

**万有引力的本质是质量对空间的"扭曲"（广义相对论视角）：**

在经典力学中，我们把它理解为两个质量之间的超距作用力。

**核心特点：**
- 与质量成正比：质量越大，引力越大
- 与距离平方成反比：距离越远，引力衰减越快
- 是相互的：M 吸引 m，m 也吸引 M（大小相等方向相反）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **质量 M, m** | 引力与质量成正比 |
| **距离 r** | 引力与距离平方成反比 |
| **万有引力常量 G** | G = 6.67×10⁻¹¹ N·m²/kg² |

---

### 等价表述

| 表述 | 公式 |
|------|------|
| **标准形式** | F = G·M·m/r² |
| **重力近似** | mg = G·M·m/R²（地面附近） |
| **黄金代换** | GM = gR² |

### 概念辨析

**辨析：万有引力 vs 重力**

| 对比项 | 万有引力 | 重力 |
|--------|---------|------|
| 公式 | F = G·M·m/r² | mg |
| 方向 | 指向地心 | 竖直向下（近似指向地心） |
| 适用范围 | 宇宙任何地方 | 地球表面附近 |
| 大小变化 | 随高度变化 | 近似恒定 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 地面附近 | mg ≈ G·M·m/R² | 重力是万有引力的近似 |
| 卫星轨道 | F = mv²/r | 万有引力提供向心力 |
| 逃逸速度 | v = √(2GM/r) | 引力势能 |

## 命题层

### 核心定理

**定理 1：万有引力定律**

```
F = G·M·m/r²
方向：沿两物体连线，相互吸引
```

**定理 2：黄金代换**

```
GM = gR²（地球表面）
其中 g ≈ 9.8 m/s²，R ≈ 6400 km
GM ≈ 4×10¹⁴ m³/s²
```

**定理 3：天体质量计算**

```
由 F = G·M·m/r² 和 F = mg
得 g = GM/R²
M = gR²/G
```

---

### 方法详解：万有引力问题解题流程

**Step 1: 识别是哪种力**

```
万有引力/重力/向心力？
```

**Step 2: 选择合适的公式**

```
万有引力 → F = G·M·m/r²
地面重力 → mg
环绕运动 → F = mv²/r
```

**Step 3: 注意单位**

```
G = 6.67×10⁻¹¹
质量用 kg，距离用 m
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "地球表面的重力就是万有引力" | 近似混淆 | 重力是万有引力的近似，但随高度变化 |
| 2 | "距离 r 是地球半径" | 忽略高度 | r = R + h（高度） |
| 3 | "G 和 g 混淆" | 记混 | G 是常数，g 是加速度 |
