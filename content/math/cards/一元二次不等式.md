# 一元二次不等式

> 一元二次不等式是"二次式子大于或小于零"的问题——它的解集由抛物线与 x 轴的位置关系决定。

---

## 概念层

### 问题的起源

解方程 x² − 3x + 2 = 0 可以用公式。
但解不等式 x² − 3x + 2 > 0 呢？

这需要理解二次函数的图像：抛物线 y = x² − 3x + 2。

不等式 x² − 3x + 2 > 0 意味着 y > 0（图像在 x 轴上方）。

> **一句话：一元二次不等式的解集，由对应抛物线与 x 轴的交点位置决定。**

---

### 原始定义（教材原文）

只含有一个未知数，且未知数的最高次数是 2 的不等式称为**一元二次不等式**。

一般形式：ax² + bx + c > 0（或 < 0、≥ 0、≤ 0），其中 a ≠ 0。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 32-35 页

---

### 本质定义

**图像解法的本质：**

```
抛物线 y = ax² + bx + c 与 x 轴的关系：
           y
           │
           │       > 0 的区域
           │     ╱     ╲
           │   ╱         ╲
    ───────┼────────────────────→ x
           │   (交点 x₁, x₂)
           │
           │       < 0 的区域
```

a > 0 时，抛物线开口向上
a < 0 时，抛物线开口向下

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **二次项系数 a** | 决定开口方向 |
| **判别式 Δ = b² − 4ac** | 决定与 x 轴的交点个数 |
| **不等号** | 决定取哪部分区间 |

---

### 等价表述

| 形式 | Δ > 0 | Δ = 0 | Δ < 0 |
|------|-------|-------|-------|
| **a > 0，> 0** | x < x₁ 或 x > x₂ | x ≠ x₀ | x ∈ R |
| **a > 0，< 0** | x₁ < x < x₂ | 无解 | 无解 |
| **a < 0，> 0** | x₁ < x < x₂ | x ∈ R | x ∈ R |
| **a < 0，< 0** | x < x₁ 或 x > x₂ | 无解 | 无解 |

---

### 概念辨析

**辨析：解方程 vs 解不等式**

| 对比项 | 解方程 x² − 3x + 2 = 0 | 解不等式 x² − 3x + 2 > 0 |
|--------|----------------------|------------------------|
| 解 | x = 1, 2 | x < 1 或 x > 2 |
| 本质 | 求交点 | 求 y > 0 的 x 范围 |
| 结果形式 | 离散点 | 区间 |

## 命题层

### 核心定理

**定理：解的判别**

设 ax² + bx + c = 0 的两根为 x₁ ≤ x₂：

| 条件 | ax² + bx + c > 0 | ax² + bx + c < 0 |
|------|------------------|------------------|
| a > 0, Δ > 0 | x < x₁ 或 x > x₂ | x₁ < x < x₂ |
| a > 0, Δ = 0 | x ≠ x₀ | 无解 |
| a > 0, Δ < 0 | x ∈ R | 无解 |
| a < 0, Δ > 0 | x₁ < x < x₂ | x < x₁ 或 x > x₂ |
| a < 0, Δ = 0 | 无解 | x ≠ x₀ |
| a < 0, Δ < 0 | 无解 | x ∈ R |

---

### 典型例题

**例题：解不等式 x² − 3x + 2 > 0**

```
题目：解不等式 x² − 3x + 2 > 0
```

**深层分析：**

```
Step 1: 求方程 x² − 3x + 2 = 0 的根
(x − 1)(x − 2) = 0
x₁ = 1, x₂ = 2

Step 2: 判断 a = 1 > 0，开口向上

Step 3: 写解集
x² − 3x + 2 > 0 的解集：x < 1 或 x > 2
```

> **方法提炼：** 大于零取两边，小于零取中间（当 a > 0）。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "二次不等式""Δ" | 解不等式 | 先求根，再判断区间 |
| "> 0" | 大于零 | a > 0 时取两边，a < 0 时取中间 |
| "< 0" | 小于零 | a > 0 时取中间，a < 0 时取两边 |

---

### 方法详解：解二次不等式步骤

**Step 1: 判断 a 的符号**

```
a > 0：开口向上
a < 0：开口向下
```

**Step 2: 求 Δ 和根**

```
Δ = b² − 4ac
求 x₁, x₂
```

**Step 3: 根据 a 和不等号写解集**

```
开口向上 + > 0：x < x₁ 或 x > x₂
开口向上 + < 0：x₁ < x < x₂
开口向下：反过来
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "Δ < 0 时不等式无解" | 记错 | 开口向上 + > 0 实际上恒成立 |
| 2 | "大于零取中间" | 混淆 | a > 0 时大于零取两边 |
| 3 | "忘记写 Δ = 0 的情况" | 粗心 | Δ = 0 时根重合，解集要排除那个点或无限制 |
