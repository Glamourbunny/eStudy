# 半衰期

> 半衰期是放射性物质或一级反应中，浓度/数量减少到原来一半所需的时间。

---

## 概念层

### 问题的起源

放射性元素的衰变、一级化学反应，都是指数衰减过程。描述这类过程时，"减少到一半需要多久"是一个直观又有用的量，这就是**半衰期**。

应用场景：
- 考古学：用碳-14半衰期推算年代
- 医学：药物半衰期决定给药频率
- 核物理：核废料半衰期决定存储时间

> **一句话：** 半衰期是指数衰减过程中，数值减少到原来一半所需的时间。

---

### 本质定义（逻辑重构版）

**半衰期的定义：**

```
设初始量为 N₀，经过时间 t₁/₂ 后，剩余量为 N₀/2

由 N(t) = N₀·e^(−kt)：
N₀/2 = N₀·e^(−kt₁/₂)
1/2 = e^(−kt₁/₂)
−kt₁/₂ = ln(1/2) = −ln2
t₁/₂ = ln2/k
```

**半衰期公式：**

```
t₁/₂ = ln2/k

特点：半衰期与初始量无关！无论初始有多少，一级反应的半衰期都是固定的。
```

---

### 半衰期 vs 反应速率常数

| 量 | 符号 | 关系 | 单位 |
|------|------|------|------|
| 半衰期 | t₁/₂ | t₁/₂ = ln2/k | 时间单位 |
| 速率常数 | k | k = ln2/t₁/₂ | 1/时间 |

---

### 多次半衰期

```
经过 n 个半衰期后：
N = N₀ · (1/2)ⁿ
```

| 半衰期个数 | 剩余比例 |
|-----------|---------|
| 1 | 1/2 |
| 2 | 1/4 |
| 3 | 1/8 |
| 10 | 1/1024 |

## 命题层

### 典型例题

**例题：碳-14 dating**

```
题目：碳-14的半衰期约为5730年。考古学家发现一块骨头中碳-14含量
是原始的 1/8。请问这块骨头大约有多少年历史？
```

**深层分析：**

```
1/8 = (1/2)³

所以经过了 3 个半衰期

年龄 = 3 × 5730 = 17190 年
```

---

**例题：药物半衰期计算**

```
题目：某药物的半衰期为 6 小时。某人上午 8 点服用 100mg。
(1) 下午 2 点体内还剩多少？
(2) 多久后药物浓度降到 10mg 以下？
```

**深层分析：**

```
(1) 从 8 点到下午 2 点经过了 6 小时 = 1 个半衰期
    剩余：100 × (1/2) = 50mg

(2) 需要降到 10mg：
    10 = 100 × (1/2)ⁿ
    (1/2)ⁿ = 0.1
    n = log₂(10) ≈ 3.32 个半衰期
    时间 = 3.32 × 6 ≈ 19.9 小时
```

## 方法层

### 方法：半衰期问题求解流程

**核心思想：** 利用半衰期的固定性和指数衰减公式。

**标准操作流程：**

```
Step 1: 确定初始量 N₀ 和半衰期 t₁/₂
Step 2: 计算速率常数 k = ln2/t₁/₂
Step 3: 代入指数衰减公式 N(t) = N₀·e^(−kt)
Step 4: 或者利用半衰期倍数：N/N₀ = (1/2)^n，其中 n = t/t₁/₂
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "半衰期与初始量有关" | 混淆一级反应特性 | 一级反应的半衰期与初始量无关 |
| 2 | "半衰期是一个固定值，不会变" | 忽略条件 | 仅对一级反应成立 |
| 3 | "1/8 是 1.5 个半衰期" | 计算错误 | 1/8 = (1/2)³，是 3 个半衰期 |
