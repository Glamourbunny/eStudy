# 正弦定理

> 正弦定理是"边与对角的正弦成正比"——它建立了三角形中边长与对角正弦之间的比例关系，是解三角形的两大支柱之一。

---

## 概念层

### 问题的起源

在三角形中，边长与对角之间是否存在一个简洁的数量关系？

观察几个特殊三角形：
- 直角三角形（30°-60°-90°）：边长比例 1:√3:2，对应正弦值 1/2:√3/2:1
- 等边三角形（60°-60°-60°）：边长相等，正弦值也相等

这暗示了一个规律：**三角形的边长与对角的正弦值成正比**。正弦定理就是这个规律的精确表述。

> **一句话：正弦定理建立了"边"与"对角的正弦"之间的比例关系。**

---

### 原始定义（教材原文）

在△ABC 中，三条边 a、b、c 分别对应的内角为 A、B、C，则有：

```
a/sinA = b/sinB = c/sinC = 2R
```

其中 R 是三角形外接圆的半径。

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 12 页

---

### 本质定义

正弦定理的本质是**比例关系**：

```
a : b : c = sinA : sinB : sinC
```

即：**大边对大角，大角对大边**。

**比例常数 2R 的几何含义：**

R 是外接圆半径。2R 是直径，所以 a/sinA = 2R 意味着"边长除以对角正弦等于直径"。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **边（a, b, c）** | 三角形的对边 |
| **角（A, B, C）** | 边的对角 |
| **比例常数 2R** | 外接圆直径 |

---

### 等价表述

| 表述形式 | 公式 |
|----------|------|
| **标准形式** | a/sinA = b/sinB = c/sinC |
| **比例形式** | a:b:c = sinA:sinB:sinC |
| **边表示形式** | a = 2R·sinA |
| **求 R 形式** | R = a/(2sinA) |

| 定理 | 公式 | 核心关系 | 适用条件 |
|------|------|---------|---------|
| **正弦定理** | a/sinA = b/sinB = c/sinC | 一次（比例） | 已知两角一边或两边及对角 |
| **余弦定理** | c² = a² + b² − 2ab·cosC | 二次（平方） | 已知两边及夹角或三边 |

### 概念辨析

**辨析 1：正弦定理 vs 余弦定理**

详见"余弦定理"卡片。

**辨析 2："大边对大角"的几何直觉**

```
         C（大角）
        ╱│╲
       ╱ │ ╲
      ╱  │  ╲
   b/    │    \a
     ╱   │   ╲
   A─────┴────B
        c（小边）
```

大边 c（AB）对应的对角 C 最大。
反过来，大角 C 的对边 c 也最大。

> **一个口诀：** "正弦定理把边角连，大边对大角是必然。"

**辨析 3：SSA 情况与正弦定理的关系**

已知两边及较小边的对角（如 a, b, A，其中 a 是较小边）时，用正弦定理可能得到两个解。

原因：sinB = b·sinA/a，当 b·sinA/a < 1 时，arcsin 有两个候选角（锐角和钝角），但需满足 A + B < 180° 才有效。

---

### 适用边界

| 条件 | 正弦定理是否适用 | 说明 |
|------|----------------|------|
| 已知两角及一边（A, B, c） | ✓ | 直接用 |
| 已知两边及对角（a, b, A） | ✓（注意双解） | 求 sinB |
| 已知两边及夹角（a, b, C） | ✗（用余弦定理） | 不能直接用正弦定理 |
| 已知三边 | ✗（用余弦定理） | 求角时用余弦定理 |

## 命题层

### 核心定理

**定理：正弦定理**

```
a/sinA = b/sinB = c/sinC = 2R
```

**证明思路 1（面积法）：**

```
S△ABC = (1/2)bc·sinA = (1/2)ac·sinB = (1/2)ab·sinC
两边除以 (1/2)abc：
sinA/a = sinB/b = sinC/c
取倒数即得正弦定理
```

**证明思路 2（外接圆几何）：**

```
∠A 对应的弧长 = (圆心角 2A/360°) × 2πR = 2R·sinA
而弧长也等于边长 a（圆周角定理）
∴ a = 2R·sinA，即 a/sinA = 2R
```

---

### 典型例题的深层分析

**例题 1：已知两角及一边（A, B, c）**

```
题目：在△ABC 中，A = 45°，B = 60°，c = 10，求 a 和 b。
```

**深层分析：**

```
Step 1: 求 C
C = 180° − 45° − 60° = 75°

Step 2: 用正弦定理求 a
a/sinA = c/sinC
a = c·sinA/sinC = 10·sin45°/sin75°
    = 10·(√2/2)/( (√6+√2)/4 )
    = 10√2 / ((√6+√2)/2)
    = 20√2/(√6+√2)
    ≈ 7.32
```

> **方法提炼：** 两角一边 → 先求第三角 → 直接代入正弦定理。

---

**例题 2：已知两边及对角（SSA）**

```
题目：在△ABC 中，a = 5，b = 7，A = 30°，求 B。
```

**深层分析：**

已知 a, b, A，用正弦定理求 B：

```
sinB / b = sinA / a
sinB / 7 = sin30° / 5
sinB = 7 × (1/2) / 5 = 7/10 = 0.7
```

因为 sinB = 0.7，有两种可能：
- B₁ = arcsin(0.7) ≈ 44.4°（锐角）
- B₂ = 180° − 44.4° = 135.6°（钝角）

检验 A + B₂ = 30° + 135.6° = 165.6° < 180°，所以 B₂ 也是有效解。

因此有两解：B₁ ≈ 44.4° 或 B₂ ≈ 135.6°。

> **方法提炼：** SSA 情况 → sinB < 1 → 考虑双解 → 检验 A+B < 180°。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | SSA 认为只有一解 | 习惯思维 | b > a > b·sinA 时有两解 |
| 2 | arcsin 得到一个角 | 三角函数性质 | arcsin(x) 有两个候选：θ 和 180°−θ |
| 3 | 忽略检验 A+B+C=180° | 粗心 | 必须检验三内角和为 180° |
| 4 | 边角写错 | 不熟悉对应关系 | 边 a 对角 A（对边对角） |
