# 余弦定理

> 余弦定理是勾股定理的"升级版"——当夹角不是 90° 时，就需要用余弦来修正勾股定理的结果。

---

## 概念层

### 问题的起源

勾股定理告诉我们：直角三角形的两条直角边平方和等于斜边平方。

但如果三角形不是直角呢？比如，夹角是 60° 或 120° 时，第三边与前两边的关系是什么？

余弦定理回答了这个问题：**当夹角不是 90° 时，需要用 cosθ 来"修正"勾股定理**。

> **一句话：余弦定理是勾股定理的一般化——它不要求夹角是 90°。**

---

### 原始定义（教材原文）

在△ABC 中，三条边 a、b、c 分别对应的内角为 A、B、C，则有：

```
c² = a² + b² − 2ab·cosC
b² = a² + c² − 2ac·cosB
a² = b² + c² − 2bc·cosA
```

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 12 页

---

### 本质定义

余弦定理的本质是**广义的勾股定理**：

当 C = 90° 时，cosC = 0，代入得 c² = a² + b²——这正是勾股定理！

**当 C > 90° 时（钝角）：**

```
C = 120°，cosC = −1/2
c² = a² + b² − 2ab·(−1/2) = a² + b² + ab
```
比 a² + b² **更大**——钝角对的边更长。

**当 C < 90° 时（锐角）：**

```
C = 60°，cosC = 1/2
c² = a² + b² − 2ab·(1/2) = a² + b² − ab
```
比 a² + b² **更小**——锐角对的边更短。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **两边（a, b）** | 夹角的两条边 |
| **夹角（C）** | 两边之间的角 |
| **cosC** | 夹角的余弦（修正系数） |

---

### 等价表述

| 表述形式 | 公式 |
|----------|------|
| **标准形式** | c² = a² + b² − 2ab·cosC |
| **求边形式** | c = √(a² + b² − 2ab·cosC) |
| **求角形式** | cosC = (a² + b² − c²) / (2ab) |
| **勾股退化** | C = 90° ⟹ c² = a² + b² |

---

### 图像表示

```
        C（夹角 C）
       /│\
      / │ \
     /  │  \
  b/    │    \a
    ╱   │   ╲
  A─────┴────B
      c（第三边）
```

**余弦定理的物理类比：**

力的平行四边形对角线长度：
- 两个力 a 和 b 夹角为 C
- 合力（对角线）的大小 c 满足 c² = a² + b² + 2ab·cos(180°−C)

不对——这里有点混淆。让我们用向量点积来理解。

**用向量点积推导（大学知识，高中生可作拓展）：**

```
|c|² = c·c = (a − b)·(a − b) = a·a + b·b − 2a·b
     = |a|² + |b|² − 2|a||b|cosθ
```

              解三角形

**与正弦定理的对比：**

| 定理 | 公式 | 核心关系 | 适用条件 |
|------|------|---------|---------|
| **正弦定理** | a/sinA = b/sinB = c/sinC | 一次（比例） | 已知两角一边或两边及对角 |
| **余弦定理** | c² = a² + b² − 2ab·cosC | 二次（平方） | 已知两边及夹角或三边 |

---

### 概念辨析

**辨析 1：余弦定理 vs 勾股定理**

| 对比项 | 勾股定理 | 余弦定理 |
|--------|---------|---------|
| **适用条件** | 直角三角形（90°） | 任意三角形 |
| **公式** | c² = a² + b² | c² = a² + b² − 2ab·cosC |
| **关系** | 余弦定理的特殊情况 | 勾股定理的一般化 |

**辨析 2：余弦定理 vs 正弦定理**

| 对比项 | 余弦定理 | 正弦定理 |
|--------|---------|---------|
| **公式结构** | 二次（平方） | 一次（比例） |
| **主要用途** | 已知两边及夹角 → 求第三边 | 已知两角一边 → 求边 |
| **求角能力** | 能直接求角（cosC 反解） | 需用 sin，再讨论双解 |
| **解的个数** | 通常唯一解 | SSA 情况可能双解 |

**什么情况用余弦定理：**

| 已知条件 | 工具 | 原因 |
|---------|------|------|
| 两边及夹角（a, b, C） | 余弦定理 | 直接求第三边 |
| 三边（a, b, c） | 余弦定理 | 求角（用反余弦） |
| 两边及对角（a, b, A） | 正弦定理 | 用余弦定理要解二次方程，不方便 |

---

### 适用边界

| 条件 | 余弦定理是否适用 | 说明 |
|------|----------------|------|
| 已知两边及夹角（a, b, C） | ✓ | 直接代公式求第三边 c |
| 已知三边（a, b, c） | ✓ | 求角：用 cosC = (a²+b²−c²)/(2ab) |
| 已知两边及对角（a, b, A） | △（可用但不便） | 求 cosB 需要解方程，不如正弦定理方便 |
| 纯角度已知，无边 | ✗ | 需要至少一边长 |

**求角时的符号判断：**

| cosC 值 | C 的范围 | 三角形类型 |
|---------|---------|-----------|
| cosC > 0 | 0° < C < 90° | 锐角 |
| cosC = 0 | C = 90° | 直角 |
| cosC < 0 | 90° < C < 180° | 钝角 |

## 命题层

### 核心定理

**定理：余弦定理**

```
c² = a² + b² − 2ab·cosC
```

**证明思路（坐标法）：**

将△ABC 放在坐标系中：
- A 在原点 (0, 0)
- B 在 x 轴上 (c, 0)
- C 的位置由 a, b 和角 C 决定

设 C 点坐标为 (x, y)，则：
```
|AC| = b ⇒ x² + y² = b²
|BC| = a ⇒ (x−c)² + y² = a²
两式相减得：x = (b² + c² − a²) / (2c)
代入得 y² = b² − x²
然后求 |BC|²，整理可得余弦定理
```

---

### 典型例题的深层分析

**例题 1：已知两边及夹角（SAS）**

```
题目：在△ABC 中，a = 5，b = 7，C = 60°，求 c。
```

**深层分析：**

```
c² = a² + b² − 2ab·cosC
    = 5² + 7² − 2×5×7×cos60°
    = 25 + 49 − 70×(1/2)
    = 74 − 35
    = 39
c = √39 ≈ 6.245
```

> **方法提炼：** SAS → 直接代入余弦定理。

---

**例题 2：已知三边（SSS）**

```
题目：已知三角形三边 a=7, b=8, c=5，求角 C。
```

**深层分析：**

用余弦定理求角：
```
cosC = (a² + b² − c²) / (2ab)
     = (49 + 64 − 25) / (2×7×8)
     = 88 / 112
     = 11/14 ≈ 0.7857
C = arccos(11/14) ≈ 38.2°
```

> **方法提炼：** SSS → 用 cosC = (a²+b²−c²)/(2ab) 求角。

---

**例题 3：判断三角形形状**

```
题目：已知 a=4, b=5, c=6，判断这是什么类型的三角形。
```

**深层分析：**

求最大角（最大边 c=6 所对的角 C）：
```
cosC = (a² + b² − c²) / (2ab)
     = (16 + 25 − 36) / (2×4×5)
     = 5 / 40
     = 1/8 = 0.125 > 0
```

cosC > 0，所以 C < 90°，是锐角。

检查其他角：
```
cosA = (b² + c² − a²) / (2bc) = (25+36−16)/(2×5×6) = 45/60 = 0.75 > 0
cosB = (a² + c² − b²) / (2ac) = (16+36−25)/(2×4×6) = 27/48 = 0.5625 > 0
```

所有 cos 都为正，所以**所有角都是锐角**，是**锐角三角形**。

> **方法提炼：** 判断形状 → 求最大角 → 看余弦符号。

---

**例题 4：余弦定理与向量数量积的联系**

```
题目：已知向量 a 和 b 满足 |a|=3, |b|=5, a·b=12，求|a−b|。
```

**深层分析：**

|a−b|² = (a−b)·(a−b) = a·a + b·b − 2a·b = |a|² + |b|² − 2a·b

代入已知：
```
|a−b|² = 3² + 5² − 2×12 = 9 + 25 − 24 = 10
|a−b| = √10
```

这个计算**本质上是余弦定理**！

因为 a·b = |a||b|cosθ = 15cosθ，所以 12 = 15cosθ，cosθ = 4/5。

验证：
```
|a−b|² = 3² + 5² − 2×3×5×(4/5) = 9+25−24 = 10
```

> **方法提炼：** 向量的模长关系可以用数量积表达，这与余弦定理完全一致。

### 方法详解：余弦定理的两大用法

**用法 1：求边（SAS 情况）**

```
已知：a, b, C
求：c

Step 1: 直接代入公式 c² = a² + b² − 2ab·cosC
Step 2: 计算数值
Step 3: 开方取正值（边长为正）
```

**用法 2：求角（SSS 情况）**

```
已知：a, b, c
求：C

Step 1: 用公式 cosC = (a² + b² − c²) / (2ab)
Step 2: 计算 cosC 的值
Step 3: 用反余弦 arccos 求 C（或判断锐角/钝角）
```

> **易错提醒：**
> 1. cosC 可能为负，角可能是钝角
> 2. 求边时开方取正值
> 3. 公式中的边角对应关系：c 对应 C，a 对应 A

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "cosC 为负则无解" | 误解 | cosC 为负只说明 C > 90°，三角形仍存在 |
| 2 | "余弦定理求角只有一个" | 三角函数性质 | 对于 0°<C<180°，cosC 确定则 C 唯一（不是双解） |
| 3 | 公式记错 | 符号错误 | 记住是 −2ab·cosC，不是 +2ab·cosC |
| 4 | 边角对应关系错 | 记混 | 角 C 是边 c 的对角，公式 c² = ... 对应 cosC |
| 5 | 开方忘取正值 | 粗心 | 边长 c = √(a²+b²−2ab·cosC) ≥ 0 |
