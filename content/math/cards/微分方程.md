# 微分方程

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 微分方程听起来像大学内容，但它在高中的外衣下到处潜伏——放射性衰变、冷却定律、种群增长，都是微分方程的"卧底"。

---

## 视角层（像什么）

### 核心隐喻：连锁反应——每一步的变化决定了下一步的状态

**微分方程 = 多米诺骨牌的连锁反应：这张牌倒下的速度，取决于此刻已经倒了多少张牌**

```
  初始状态                    方程描述                      后续状态
┌────────────┐       ┌─────────────────────┐       ┌────────────┐
│  此刻水温   │       │ "水温下降的速度     │       │  下一时刻  │
│  T₀=80°C   │  ───→ │  正比于水温与环境   │  ───→ │  的水温    │
│            │       │  的温差"            │       │  T₁       │
└────────────┘       └─────────────────────┘       └────────────┘

  微分方程的核心：变化率（导数）与当前状态（函数值）之间的关系
  用符号表达：y' = f(y)  或  y' = f(t,y)
```

**连锁反应思维的三条铁律：**

| 铁律 | 多米诺场景 | 违反的后果 |
|------|-----------|-----------|
| **变化率驱动**：状态的变化速度由当前状态决定 | 水温的冷却速度取决于"当前水温跟室温差多少" | 把变化率当常数，用线性公式计算 |
| **反馈循环**：状态改变→变化率改变→状态继续改变 | 人口越多→增长越快→人口更多→增长更快 | 以为变化是匀速的 |
| **分离变量**：把 y' = dy/dt 拆开，y 归一边，t 归一边 | dy/y = k dt，然后两边各自积分 | 试图用代数方法硬解微分关系 |

> **一句话直觉：** 微分方程不是找"y 等于几"，而是找"什么样的函数 y(t) 能使得它的导数 y'(t) 和 y(t) 本身满足题设的关系"——状态和变化率互为因果，这就是微分方程。

### 动态想象：牛顿冷却定律

```
  一杯热水放在 20°C 的房间里：

  时刻 t=0：水温 80°C，温差 60°C → 冷却很快
  时刻 t=1：水温 70°C，温差 50°C → 冷却变慢
  时刻 t=2：水温 63°C，温差 43°C → 冷却更慢
  ...
  时刻 t→∞：水温 → 20°C（趋近室温但不低于室温）

  规律：冷却速率 dT/dt 正比于温差 (T - 20)
  → dT/dt = -k(T - 20)
  → 分离变量：dT/(T-20) = -k dt
  → 积分：ln|T-20| = -kt + C
  → 解：T = 20 + 60e^(-kt)（指数衰减）
```

### 反直觉警示

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **变化率恒定** | "冷却速度应该是恒定的" | 温差越大冷却越快——变化率跟当前状态有关 |
| **分离变量是万能药** | 所有微分方程都能分离变量 | 只有特定形式（可分离/一阶线性）能用分离变量 |
| **微分方程只存在于大学** | "高中不考微分方程" | 指数增长、指数衰减、物理简单运动——本质都是微分方程，只是包装成了其他词 |

## 联结层（连什么）

### 跨学科锚点

**1. 生物学：种群增长（马尔萨斯模型）**
```
人口增长率 dP/dt = kP
→ 人口越多，增长越快
→ 解：P(t) = P₀ e^(kt)（指数增长）
```

**2. 化学：一级反应速率**
```
反应物浓度 [A] 的减少速率正比于当前浓度
d[A]/dt = -k[A]
→ [A](t) = [A]₀ e^(-kt)（指数衰减）
→ 这就是半衰期的来源！
```

**3. 物理：弹簧振动（简谐运动）**
```
加速度 d²x/dt² = -ω²x
→ 二阶微分方程
→ 解：x(t) = A sin(ωt + φ)（正弦振动）
```

### 向 Formal 的桥梁

| 微分方程类型 | 识别信号 | Formal 着陆点 |
|-------------|---------|--------------|
| 指数型：y' = ky | 变化率正比于当前值 | 一阶ODE.md, 指数函数.md |
| 冷却型：y' = -k(y-a) | 变化率正比于与某常数的差 | 一阶ODE.md, 构造函数.md |
| 简谐型：y'' = -ω²y | 加速度正比于位移（反向） | 二阶ODE.md, 三角恒等变换.md |

### 概念网络坐标

```
  物理/生物/化学现象
      │
      ▼
  变化率与状态的关系（"…的速度取决于…"）
      │
      ▼
  微分方程（描述变化规律的等式）
      │
      ├── 分离变量法 → 一阶ODE
      ├── 特征根法   → 二阶ODE
      └── 定性分析   → 方向场/斜率场
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 拆包收益 |
|:---|:---|:---|
| 🎭 **包装** | "某量的变化率与…成正比"，穿了物理/生物的外衣 | 识别出 y'=ky 或 y'=k(y-a)，直接用指数模型 |
| 🔀 **嵌套** | 变化率+参数+方程求函数的组合 | 先写出微分关系，分离变量，再积分求解 |
| 🆕 **新定义** | "定义增长率 r(t)=f'(t)/f(t)…" | 识别出这是相对变化率，可能通向微分方程 |

### 拆包方向标（10 秒流程）

```
Step 1：找"变化率"关键词（3 秒）
  • "…的速度""…的速率""…的变化率"→ 导数 y'
  • "…正比于…"→ y' = k×(某个量)

Step 2：写出微分关系（4 秒）
  • 把文字翻译成：y' = f(y) 或 y' = f(t,y)
  • 判断是否可分离变量

Step 3：锁定 Formal 着陆点（3 秒）
  • 可分离 → 分离变量后积分
  • 一阶线性 → 用积分因子法
  • 超出高中范围 → 用定性分析（单调性、方向）
```

---

### 直觉陷阱清单

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **不分变量就积分** | dy/dx = f(x,y) 时直接对 x 积分 | 必须先把 y 和 x 分别移到等号两边 |
| **忘记积分常数** | 积分后漏了 +C | C 必须保留，用初始条件确定 |
| **忽视定义域** | 解出 y 的表达式后忘了检查定义域 | 检查是否有分母为零、对数定义域等问题 |

## 一句话总结

> **微分方程 = 状态和变化率的因果循环——此刻的状态决定下一刻的变化速度，变化速度又改变下一时刻的状态。高中虽不直接考查求解复杂的微分方程，但指数增长/衰减、冷却定律、简谐运动的底层全是微分方程。认出它、写出关系、分离变量、两端积分——这就是高中版的微分方程解法。**
