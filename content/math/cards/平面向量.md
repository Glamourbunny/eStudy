# 平面向量

> 平面向量是有大小和方向的量——它不仅告诉我们"有多长"，还告诉我们"指向哪"。

---

## 概念层

### 问题的起源

物理学中，力、速度、加速度这些量既有大小又有方向，代数无法完全描述它们。数学家引入了**向量**的概念。

平面向量让我们能够：
- 用统一方式处理几何和物理问题
- 将"形"的问题转化为"数"的问题
- 简化复杂的几何证明

> **一句话：** 平面向量是既有大小又有方向的量，是描述力、速度、位移等物理量的数学工具。

---

### 本质定义（逻辑重构版）

**向量的定义：**

```
向量 = 大小 + 方向

记作：AB（A 是起点，B 是终点）
或：小写字母 a, b, c

模（大小）：|a| 或 |AB|
方向：向量指向的方向
```

**向量的表示：**

| 表示方式 | 形式 | 说明 |
|---------|------|------|
| 几何表示 | 有向线段 | 箭头表示方向 |
| 坐标表示 | (x, y) | 在坐标系中 |
| 字母表示 | a, b, c | 粗体或带箭头 |

---

### 向量的特征

```
┌─────────────────────────────────┐
│           向量 a                 │
│                                 │
│      ·B                        │
│     ╱                           │
│    ╱                            │
│   A──────→                      │
│                                 │
│  大小：线段 AB 的长度            │
│  方向：从 A 到 B 的指向          │
└─────────────────────────────────┘
```

---

### 邻近概念辨析

**辨析：向量 vs 标量 —— "有向" vs "无向"**

| 对比项 | 向量 | 标量 |
|--------|------|------|
| 特征 | 有大小和方向 | 只有大小 |
| 例子 | 速度、力、位移 | 长度、质量、温度 |
| 运算 | 遵循向量运算法则 | 遵循代数运算法则 |
| 大小表示 | |a| | a（普通字母） |

---

### 适用边界

| 要点 | 说明 |
|------|------|
| 向量可以平移 | 大小方向相同的向量是相等的 |
| 零向量 | 大小为 0，方向任意 |
| 单位向量 | 大小为 1 的向量 |

---

## 命题层

### 核心定理

**定理：向量相等的条件**

两个向量相等 ⟺ 大小相等且方向相同（可以平移后重合）

**定理：向量的模**

```
|a| = √(a₁² + a₂²)

向量的模是非负数，|a| = 0 ⟺ a = 0
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "向量 (1,2) 和 (2,1) 相等" | 混淆坐标 | 向量相等要求方向和大小都相同 |
| 2 | "向量有固定位置" | 忽视向量的可平移性 | 向量可以自由平移，相等向量等效 |
| 3 | "零向量没有方向" | 误解零向量 | 零向量方向任意，但数学上认为"无方向" |
