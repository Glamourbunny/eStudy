# 独立性检验

> 独立性检验是"判断两个分类变量是否相关"的统计方法——通过 χ² 统计量检验两个变量是否独立。

---

## 概念层

### 问题的起源

吸烟和肺癌有关吗？性别和学习成绩有关吗？

这些都是**分类变量**之间的关系问题。

独立性检验通过样本数据判断两个变量是否相关。

> **一句话：独立性检验通过 χ² 统计量判断两个分类变量是否独立。**

---

### 本质定义

**卡方检验的基本思想：**

```
如果两个变量独立，则观察频数与期望频数应该接近
χ² = Σ (观察频数 - 期望频数)² / 期望频数
χ² 越大，说明观察与期望差异越大，两变量越可能相关
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **观察频数** | 样本中各类别的实际计数 |
| **期望频数** | 如果两变量独立，应该有的频数 |
| **χ² 统计量** | 衡量观察与期望的差异 |

---

## 命题层

### 核心定理

**定理：χ² 统计量**

```
χ² = Σ (Oᵢ - Eᵢ)² / Eᵢ
其中 Oᵢ 是观察频数，Eᵢ 是期望频数
```

**定理：自由度**

```
自由度 = (行数 - 1) × (列数 - 1)
```
