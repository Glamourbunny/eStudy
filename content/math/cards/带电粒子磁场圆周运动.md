# 带电粒子磁场圆周运动

> 带电粒子垂直进入匀强磁场时，会在洛伦兹力的作用下做匀速圆周运动——半径与速度成正比，周期与速度无关。

---

## 概念层

### 问题的起源

带电粒子在磁场中会受到洛伦兹力 F = qvBsinθ。

当带电粒子**垂直**进入匀强磁场时：
- v ⊥ B，所以 θ = 90°，sinθ = 1
- F = qvB，方向始终垂直于速度 v
- 这正好提供向心力，带电粒子做**匀速圆周运动**

这个现象是制造粒子加速器、质谱仪的基础。

> **一句话：带电粒子垂直进入匀强磁场时，洛伦兹力提供向心力，使其做匀速圆周运动。**

---

### 本质定义

**运动本质：**

```
向心力 = 洛伦兹力
mv²/r = qvB
        ↓
r = mv/(qB)
```

**核心物理量：**

| 量 | 公式 | 说明 |
|------|------|------|
| **半径 r** | r = mv/(qB) | v 大则 r 大 |
| **周期 T** | T = 2πm/(qB) | 与 v 无关 |
| **频率 f** | f = 1/T = qB/(2πm) | 与 v 无关 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **垂直入射** | v ⊥ B 是做圆周运动的关键条件 |
| **洛伦兹力** | F = qvB 始终垂直于 v |
| **向心力平衡** | mv²/r = qvB |

---

### 概念辨析

**辨析：不同入射角的运动**

| 入射角 | 运动轨迹 | 特点 |
|--------|---------|------|
| v ⊥ B | 匀速圆周运动 | 半径 r = mv/(qB)，周期 T = 2πm/(qB) |
| v 与 B 斜交 | 螺旋线运动 | 螺旋半径 r⊥ = mv⊥/(qB)，螺距 h = v∥T |
| v // B | 匀速直线运动 | 不受洛伦兹力 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| v ⊥ B | ✓ | 做匀速圆周运动 |
| v // B | ✓ | 做匀速直线运动 |
| v 与 B 斜交 | ✓ | 做螺旋线运动 |

## 命题层

### 核心定理

**定理 1：半径公式**

```
r = mv/(qB)
```

速度 v 越大，半径 r 越大；B 越大，半径 r 越小。

**定理 2：周期公式**

```
T = 2πm/(qB)
```

周期 T 与速度 v 和半径 r 无关，只由 m/(qB) 决定。

**定理 3：回旋加速器原理**

利用 T 与 v 无关的特性，用固定频率的交变电场多次加速粒子。

---

### 典型例题

**例题：质谱仪**

```
题目：质谱仪中，带电离子（q = 1.6×10⁻¹⁹ C）垂直射入 B = 0.5 T 的匀强磁场。
      离子质量 m = 1.67×10⁻²⁷ kg，速度 v = 1×10⁵ m/s。
      求：(1) 轨迹半径 r；(2) 运动周期 T。
```

**深层分析：**

```
(1) r = mv/(qB)
    = (1.67×10⁻²⁷ × 1×10⁵) / (1.6×10⁻¹⁹ × 0.5)
    = 1.67×10⁻²² / (8×10⁻²⁰)
    ≈ 2.1×10⁻³ m = 2.1 mm

(2) T = 2πm/(qB)
    = 2π × 1.67×10⁻²⁷ / (1.6×10⁻¹⁹ × 0.5)
    = 3.35π×10⁻²⁷ / (8×10⁻²⁰)
    ≈ 2.6×10⁻⁷ s
```

> **方法提炼：** 垂直入射 → 直接用公式 r 和 T。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "垂直""v ⊥ B" | 圆周运动 | r = mv/(qB)，T = 2πm/(qB) |
| "螺旋线""斜交" | 螺旋运动 | 分解 v 为 v⊥ 和 v∥ |
| "质谱仪""加速器" | 应用题 | 多次加速或质量分离 |

---

### 方法详解：带电粒子圆周运动问题的处理

**Step 1: 确认 v ⊥ B**

```
只有垂直入射才做匀速圆周运动
```

**Step 2: 用向心力公式求半径**

```
qvB = mv²/r ⟹ r = mv/(qB)
```

**Step 3: 求周期**

```
T = 2πr/v = 2πm/(qB)
```

> **易错提醒：** 公式中的 m 是离子质量，q 是离子电量，不要用错了。
