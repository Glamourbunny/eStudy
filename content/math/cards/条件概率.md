# 条件概率

> 条件概率是"在已知 B 发生的条件下，A 发生的概率"——记作 P(A|B)。

---

## 概念层

### 问题的起源

一副扑克牌，已知抽到的是红桃，那么它是 Q 的概率是多少？

这就是条件概率——**在已知某个信息（B 发生）的条件下**，求另一个事件的概率。

红桃 Q 是 1 张，红桃共 13 张，所以 P(Q|红桃) = 1/13。

> **一句话：条件概率 P(A|B) 是在已知 B 发生的条件下，A 发生的概率。**

---

### 原始定义（教材原文）

设 A、B 是两个事件，且 P(B) > 0，在事件 B 已经发生的条件下，事件 A 发生的概率称为**条件概率**，记作 P(A|B)。

```
P(A|B) = P(AB) / P(B)
```

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 10 章

---

### 本质定义

**条件概率的本质是"缩减样本空间"：**

```
原来的样本空间 Ω（全部52张牌）
在已知 B 发生后（抽到红桃），样本空间缩减为 B（红桃13张）

P(A|B) = 在缩减后的样本空间中 A 发生的概率
```

**公式：**

```
P(A|B) = P(AB) / P(B) = P(B|A) × P(A) / P(B)
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **条件 B** | 已知发生的事件 |
| **事件 A** | 要求概率的事件 |
| **公式** | P(A|B) = P(AB)/P(B) |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **P(A\|B)** | 在 B 条件下 A 的概率 |
| **乘法公式** | P(AB) = P(A\|B) × P(B) = P(B\|A) × P(A) |
| **缩减样本空间** | 以 B 为新的样本空间 |

### 概念辨析

**辨析：P(A|B) vs P(A)**

| 对比项 | P(A\|B) | P(A) |
|--------|---------|------|
| 条件 | 已知 B 发生 | 没有任何条件 |
| 样本空间 | 缩减为 B | 整个 Ω |
| 信息 | 已知 B 的信息 | 无额外信息 |

## 命题层

### 核心定理

**定理 1：条件概率公式**

```
P(A|B) = P(AB) / P(B)
```

**定理 2：乘法公式**

```
P(AB) = P(A|B) × P(B) = P(B|A) × P(A)
```

**定理 3：贝叶斯公式**

```
P(A|B) = P(B|A) × P(A) / P(B)
```

---

### 典型例题

**例题：条件概率计算**

```
题目：一个家庭有两个孩子，已知其中一个是女孩，求另一个也是女孩的概率。
（假设生男生女概率相同）
```

**深层分析：**

```
样本空间：{男男, 男女, 女男, 女女}（等可能）
B = "至少有一个女孩" = {男女, 女男, 女女}
A = "两个都是女孩" = {女女}

P(A|B) = P(AB)/P(B) = P({女女})/P({男女, 女男, 女女})
       = (1/4)/(3/4) = 1/3
```

> **方法提炼：** 用公式 P(A|B) = P(AB)/P(B)。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "P(A\|B) = P(A)" | 忽略条件 | 条件概率可能改变概率 |
| 2 | "P(A\|B) 和 P(B\|A) 混淆" | 混淆 | 两者不同，用贝叶斯公式转换 |
| 3 | "条件概率是 A 的概率除以 B 的概率" | 错误类比 | 是 P(AB)/P(B)，不是 P(A)/P(B) |
