# 导数定义

> 导数是"变化率的极限"——当自变量的增量趋于零时，函数值的增量与自变量增量之比的极限。

---

## 概念层

### 问题的起源

在物理中，我们用"速度"描述物体运动的快慢。但当物体做变速运动时，"某一点"的速度如何定义？

古希腊人用"穷竭法"处理曲线下的面积问题，牛顿和莱布尼茨在17世纪独立发明了微积分。核心思想是：**用"瞬时变化率"代替"平均变化率"，让增量趋于零**。

> **一句话：** 导数是用极限工具定义的"瞬时变化率"，回答"此时此地变化得多快"的问题。

---

### 原始定义（教材原文）

设函数 y = f(x) 在点 x₀ 处的某邻域内有定义，当自变量 x 在 x₀ 处取得增量 Δx（点 x₀+Δx 在该邻域内）时，相应地函数取得增量 Δy = f(x₀+Δx) − f(x₀)。如果 Δy 与 Δx 之比当 Δx→0 时的极限存在，那么这个极限叫做函数 f(x) 在点 x₀ 处的导数。

> 出处：人教版《普通高中教科书·数学（B版）选择性必修第二册》第 98 页

---

### 本质定义（逻辑重构版）

**导数的几何意义：**

```
f'(x₀) = lim(Δx→0) [f(x₀+Δx) − f(x₀)] / Δx
       = 曲线在 x₀ 处切线的斜率
```

**导数的物理意义（瞬时速度）：**

```
v(t₀) = lim(Δt→0) [s(t₀+Δt) − s(t₀)] / Δt
      = 时刻 t₀ 的瞬时速度
```

---

### 导数与导函数

| 概念 | 说明 | 记号 |
|------|------|------|
| 点导数 | 在特定点 x₀ 处的导数 | f'(x₀) |
| 导函数 | 在任意点 x 处的导数 | f'(x) = lim(Δx→0) [f(x+Δx)−f(x)]/Δx |

**可导与连续的关系：**

```
可导 ⟹ 连续（但连续不一定可导）
```

---

### 邻近概念辨析

**辨析：导数 vs 微分 —— "变化率" vs "变化量"**

| 对比项 | 导数 | 微分 |
|--------|------|------|
| 本质 | 比值的极限（变化率） | 变化量的线性主部 |
| 关系 | f'(x) = dy/dx | dy = f'(x)·dx |
| 几何意义 | 切线斜率 | 切线的纵增量 |

## 命题层

### 核心定理

**定理：可导的充要条件**

f(x) 在 x₀ 处可导 ⟺ f(x) 在 x₀ 处左右导数都存在且相等。

**定理：基本求导公式**

| 函数 | 导数 |
|------|------|
| C（常数） | 0 |
| xⁿ | n·xⁿ⁻¹ |
| sin x | cos x |
| cos x | −sin x |
| eˣ | eˣ |
| aˣ | aˣ·ln a |
| ln x | 1/x |

## 方法层

### 方法：导数定义的应用

**核心思想：** 用定义求导本质上就是计算极限。

**标准操作流程：**

```
求 f'(x₀)：

Step 1: 写增量：Δy = f(x₀+Δx) − f(x₀)
Step 2: 写比值：Δy/Δx
Step 3: 取极限：lim(Δx→0) Δy/Δx
Step 4: 化简计算结果
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "连续一定可导" | 混淆充分必要条件 | 连续是可导的必要条件，不是充分条件 |
| 2 | "导数存在就是极限存在" | 忽略单侧极限 | 必须左右导数都存在且相等 |
| 3 | "f'(x₀) = (f(x₀+Δx) − f(x₀))/Δx" | 漏掉极限过程 | 必须取 Δx→0 的极限 |
