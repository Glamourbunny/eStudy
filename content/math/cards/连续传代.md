# 连续传代

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 连续传代在高考中以"经过 n 代后""逐代计算""建立递推关系求通项"出现——它是把 Hardy-Weinberg 或选择/突变的效果拉长到时间轴上，看基因频率如何在代际之间演化。你的任务是：看清单代变化规则，然后"复制粘贴"n 次，找通项或递推极限。

---

## 视角层（像什么）

### 核心隐喻：传话游戏——每一代把"消息"传给下一代，但消息每传一次就变一点点

**连续传代 = 一群人排成一列玩传话游戏。第 0 代对着第 1 代的耳朵说了一个词（初始基因频率 p₀），第 1 代传给第 2 代时可能会"听错"（选择/突变），第 n 代人听到的词和第 0 代已经不一样了。你问的就是：第 n 代听到的是什么？**

```
  传话链条（基因频率的世代递推）：

  第 0 代（初始）：p₀
     ↓ 经历一次"选择/突变/交配"
  第 1 代：p₁ = f(p₀)     ← 规则 f 是单代变换
     ↓ 同样的规则再执行一次
  第 2 代：p₂ = f(p₁) = f(f(p₀))
     ↓ …重复 n 次…
  第 n 代：p_n = f^n(p₀)   ← n 代之后的频率
```

> **一句话直觉：** 连续传代就是把单代规则（HW 平衡、选择淘汰、突变修正）反复执行 n 次。核心在于找到"从这一代到下一代"的递推公式 a_{n+1} = f(a_n)，然后要么求通项 a_n，要么求极限 a_∞。

### 动态想象：复印机的退化

```
  把基因频率想象成一张纸在复印机上反复复印：

  原件 p₀（初始频率）→ 复印 → p₁ → 复印 → p₂ → ……
  
  如果复印机是完美的（Hardy-Weinberg）：
  每一张复印件都和原件一模一样 → pₙ = p₀（不变）
  
  如果复印机有偏差（自然选择）：
  每次复印都会往某个方向略微偏移 → pₙ 逐渐漂移
  
  如果复印机有"自动纠正"（极限/平衡点）：
  不管 p₀ 是多少，复印 n 次后都会趋近同一个 p*
  → 这就是稳态（平衡频率），满足 p* = f(p*)
```

**考场翻版：** 题目给出"初始基因频率 p₀"，然后说"每代经历选择后随机交配，求第 n 代的基因频率"或"问 n 代后频率趋近于多少"。

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **逐代迭代 vs 通项公式** | "一代一代算就行" | 如果 n=1000 呢？必须找通项。递推 a_{n+1} = r·a_n + c 的通项是 a_n = r^n·a₀ + c(1-r^n)/(1-r) |
| **极限存在 ≠ 一代到达** | "平衡就是下一代不变了" | 趋近平衡可能需要很多代。即使极限是 0.75，第 10 代可能才到 0.749 |
| **不同初始值 → 同一极限** | "从哪开始不一样" | 有些递推（收缩映射）会让所有初始值都收敛到同一个不动点 |
| **单调 ≠ 收敛** | "一直在变，没有平衡" | 如果递推有不动点且每一代都在缩小距离（|a_n - p*| 递减），最终必然收敛 |

## 联结层（连什么）

### 跨学科锚点

**1. 化学：反应动力学——连续反应**

```
  A → B → C → 终产物（连续反应）

  每个时间步（对应每一代），一部分 A 变成 B，一部分 B 变成 C
  ↓ 递推方程：
  [A]_{t+1} = (1-k₁)·[A]_t
  [B]_{t+1} = k₁·[A]_t + (1-k₂)·[B]_t
  ↓ 和基因频率代际递推是同一结构：x_{n+1} = M·x_n
```

**2. 金融：复利计算**

```
  本金 p₀，年利率 r
  每年末：p_{n+1} = (1+r)·p_n
  n 年后：p_n = p₀·(1+r)^n
  
  对应：基因频率在没有反向压力时的"指数衰减/增长"
  ↓ 若每代 a 频率减少 20%：q_{n+1} = 0.8·q_n
  ↓ q_n = q₀·(0.8)^n → 趋近于 0（但永远不会真正到达 0）
```

**3. 控制论：反馈系统**

```
  恒温器的温度调节：
  T_{n+1} = T_n + k·(T_目标 - T_n)
  
  每次测量后，加热/制冷系统"拉"温度向目标靠近
  ↓ 不动点：T_目标 = T_n 时不再变化
  ↓ 收敛速度取决于 k（增益系数）
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 已知初始频率 + 每代变化规则，求第 n 代频率 | 递推 a_{n+1} = f(a_n)，找通项 | 递推数列通项公式（等差/等比/一阶线性） |
| 判断频率是否收敛 | 找不动点 p* = f(p*) | 不动点迭代 + 收敛性判定 |
| 多代选择后的 Hardy-Weinberg | 选择改变频率 → 每代套 HW | 哈迪-温伯格定律：每次新频率下 p²:2pq:q² |
| 求第 n 代基因型频率 | 先算第 n 代基因频率，再套 HW | p_n → (p_n)² : 2·p_n·q_n : (q_n)² |

**首选工具**：递推公式 + 通项（找规律）
**次选工具**：不动点迭代（求极限时）

**拆包调用优先级：**
1. 写出单代变换规则 f（从 p_n 到 p_{n+1}）
2. 识别递推类型：等差？等比？一阶线性？
3. 求通项或极限

### 历史/哲学视角：时间箭头在遗传学中

```
19 世纪（孟德尔）：
  孟德尔只研究了"一代"的交配结果
  P 代 → F1 代 → F2 代
  但没有考虑"无限代之后会怎样"
  
20 世纪初（费希尔、霍尔丹、赖特）：
  三人独立将代际变化数学化
  关键问题：给定单代规则，多代之后的频率演化的走向？
  
  ↓ 这就是递推和不动点的数学
  ↓ 费希尔证明了"自然选择的基本定理"：
     群体的平均适合度随时间单调增加（在特定条件下）
  
  哲学意义：
  连续传代把"静态遗传"变成了"动态演化"
  原来基因频率是时间轴上的一个动力系统——和行星轨道、化学反应的动力学
  是同一个数学框架。生物学从此从"描述"走向了"预测"。
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | "经过 n 代随机交配后""每一代都经历选择后再交配""求第 n 代的频率/基因型分布""长期演化趋势" | 一代一代硬算（n 大时手工不可能） | 3 秒建立递推 a_{n+1}=f(a_n)，求通项或直接写极限 |
| 🔀 **嵌套** | 每代有多重变化规则："先选择、再突变、再迁移，然后交配"，而且每代的顺序固定 | 不知道几个规则的先后顺序影响结果 | 拆成多层叠加：先应用全部外力 → 得到新频率 → 最后交配 |
| 🆕 **新定义** | "定义世代变换 T(p) = …""若满足 p_{n+1} = T(p_n)，则称…" | 对新定义的 T 函数不理解，不知道这是递推 | 识别三要素：输入 p_n → 变换 T → 输出 p_{n+1} = 递推关系 |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"n 代""世代""长期趋势""趋近"？
    → 连续传代题，需要递推或极限
  • 确定：单代规则是什么？初值是多少？

Step 2：剥外壳找骨骼（4 秒）
  • 写出递推公式：p_{n+1} = f(p_n)
  • 给前几代赋值检验递推是否正确
  • 判断：是线性递推（可找通项）？还是需要不动点？

Step 3：锁定 Formal 着陆点（3 秒）
  • 线性递推 → 求通项公式
  • 不动点 → 解 p* = f(p*)，判断收敛性
  • 极限后的基因型 → 用 p* 套 HW
  • 写下：拆包→【递推数列/Hardy-Weinberg】，立即切换
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 递推公式 p_{n+1} = f(p_n) 已写出 | 求通项：p_n = r^n·p₀ + c(1-r^n)/(1-r)；或求极限 |
| 嵌套题 | 每代多层规则的叠加顺序已确定 | 逐层应用：选择→突变→迁移→交配，顺序不可打乱 |
| 新定义题 | 变换 T 的输入/输出/规则已理解 | 按 T 的定义建立递推，再用标准递推方法处理 |

---

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **一代一代算** | 当 n 很大时暴算 | 找通项！线性递推 a_{n+1} = r·a_n + c 的通项公式记牢 |
| **以为每次变化后都是 HW** | 选择压力下每代基因频率都在变，不是平衡态 | 先判断是否有选择/突变——有的话不是 HW，需要递推 |
| **忘记最后一轮交配** | 递推算完 p_n 就直接当基因型频率 | 最后一代的基因型频率 = 用 p_n 套 HW：(p_n)²:2·p_n·q_n:(q_n)² |
| **极限存在但收敛太慢** | 递推 r=0.999，理论上收敛但实际几乎不动 | 看 r：|r| 越接近 1，收敛越慢。可以估算 "半衰代数" |

## 一句话总结

> **连续传代是把单代规则"复制粘贴"n 次——从 p₀ 出发，每代执行同样的变换 f，第 n 代得到 fⁿ(p₀)。高考里它以"n 代后""长期趋势"出现，你的任务是用递推公式 a_{n+1} = f(a_n) 建模，然后要么求通项，要么找不动点。极限分析时不要忘记最后一轮交配套 HW。**
