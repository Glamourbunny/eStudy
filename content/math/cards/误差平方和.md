# 误差平方和

> 误差平方和 SSE 是"实际值与预测值之差的平方和"——它是衡量回归模型拟合效果的核心指标。

---

## 概念层

### 问题的起源

回归直线 ŷ = a + bx 与实际数据点有多接近？

衡量标准：所有点到直线的"距离平方和"——这就是**误差平方和**（Sum of Squares Error，SSE）。

SSE 越小，直线拟合越好。

> **一句话：误差平方和 SSE 是实际值与预测值差的平方和，衡量拟合效果。**

---

### 本质定义

**误差平方和的定义：**

```
SSE = Σ(yᵢ - ŷᵢ)²
    = Σ(yᵢ - a - bxᵢ)²
```

**与总平方和的关系：**

```
SST = Σ(yᵢ - ȳ)²（总变异）
SSR = Σ(ŷᵢ - ȳ)²（回归解释的变异）
SSE = Σ(yᵢ - ŷᵢ)²（未被解释的变异）

SST = SSR + SSE
```

**决定系数：**

```
R² = SSR/SST = 1 - SSE/SST
```
