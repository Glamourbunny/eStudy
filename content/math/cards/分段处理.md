# 分段处理

> 分段处理是"把复杂问题分成几段来处理"——当问题在不同区间有不同规律时，需要分段研究。

---

## 概念层

### 问题的起源

现实中的很多量不是平滑变化的，而是在不同区间有不同规律：

- **出租车收费**：3公里内一个价，超过3公里另一个价
- **个人所得税**：不同收入区间税率不同
- **快递计费**：重量在不同区间收费标准不同

数学上用**分段函数**来描述这种"不同区间不同规则"的现象。

> **一句话：分段处理是根据不同区间的不同规律分别处理问题的思维方式。**

---

### 本质定义

**分段函数的定义：**

```
f(x) = { 表达式1,  x ∈ 区间1
       { 表达式2,  x ∈ 区间2
       { 表达式3,  x ∈ 区间3
```

**分段处理的核心：**

```
整体复杂 → 分成若干段 → 每段分别处理 → 合并结果
```

---

### 分段函数的要素

| 要素 | 说明 | 示例 |
|------|------|------|
| **分段点** | 分段的边界点 | x = 0, x = 2 |
| **各段定义域** | 每一段的 x 范围 | x < 0, 0 ≤ x < 2, x ≥ 2 |
| **各段表达式** | 该区间内的函数式 | x², 2x+1, 3 |
| **区间并集** | 各段定义域的并 = 整体定义域 | (-∞, +∞) |

---

### 与分类讨论的关系

| 概念 | 说明 | 联系 |
|------|------|------|
| **分段处理** | 按自变量区间分 | 自变量取值不同则函数式不同 |
| **分类讨论** | 按参数取值分 | 参数不同则问题性质不同 |
| **联系** | 都是"化整为零"的策略 | 分段处理是分类讨论在函数上的体现 |

## 命题层

### 核心定理

**定理：分段函数的连续性**

```
分段函数在分段点 x₀ 处连续的条件：
lim(x→x₀⁻) f(x) = lim(x→x₀⁺) f(x) = f(x₀)
```

**定理：分段函数的求导**

```
在非分段点：直接求导
在分段点：求左导数和右导数
f'(x₀) 存在 ⟺ f'_-(x₀) = f'_+(x₀)
```

**定理：分段函数的积分**

```
∫f(x)dx = 各段积分结果 + 常数
（在分段点处拼接）
```

### 分段处理的常见类型

| 类型 | 示例 | 处理要点 |
|------|------|---------|
| 分段函数求值 | f(x) = \|x\| | 按 x 的正负分 |
| 分段函数作图 | 取绝对值函数 | 找分段点，画各段图像 |
| 分段函数求导 | 含有绝对值的函数 | 去绝对值后分段求导 |
| 分段不等式 | 解 f(x) > 0 | 各段分别解，再合并 |
| 分段最值 | 分段函数的最大值 | 各段求最值，再比较 |

### 方法详解：分段函数的解题步骤

**Step 1: 找出分段点**

```
分段点通常是：
- 绝对值内部为零的点（|x - a| → x = a）
- 实际问题的关键节点（3公里、起征点）
- 题目明确给出的分界点
```

**Step 2: 明确各段定义域**

```
用区间表示每一段的范围
确保各段不重不漏
```

**Step 3: 分别处理各段**

```
在每段定义域上用对应的表达式
按照普通函数的方法处理
```

**Step 4: 检验分段点**

```
分段点往往需要单独检验：
- 连续性检验
- 可导性检验
- 是否满足方程或不等式
```

**Step 5: 合并结果**

```
各段的结果合并
注意区间端点的取舍
```

---

### 方法详解：绝对值问题的分段处理

**核心思想：** 绝对值内部符号变化导致分段。

```
|f(x)| 的分段点：f(x) = 0 的根

|f(x)| = { f(x),  f(x) ≥ 0
         { -f(x), f(x) < 0
```

**步骤：**

```
1. 令 f(x) = 0，求出所有根
2. 以根为分点，划分区间
3. 在每个区间上判断 f(x) 的正负
4. 去掉绝对值，化为分段函数
5. 分别求解
```

---

### 方法详解：分段不等式的解法

**步骤：**

```
1. 找出所有关键点（分段点、零点）
2. 划分区间
3. 在每个区间内，去掉分段符号（原不等式或绝对值）
4. 求出该区间内的解
5. 与该区间取交集
6. 合并所有区间的解
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "分段点漏解" | 粗心 | 分段点必须单独检验 |
| 2 | "区间重叠或遗漏" | 粗心 | 画数轴，标注所有关键点 |
| 3 | "端点取舍混乱" | 粗心 | 区间开闭要严格判断 |
| 4 | "各段结果直接拼接" | 误解 | 要与各段定义域取交集 |
| 5 | "忽略分段点的特殊性质" | 误解 | 分段点往往左右极限不同 |
