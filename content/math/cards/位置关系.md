# 位置关系

> 空间中的位置关系，用向量的语言来描述，就是"平行""垂直""共线""异面"——这些都可以用数量积和向量比例来精确判定。

---

## 概念层

### 问题的起源

平面几何中，点线面的位置关系相对简单。但进入三维空间后，情况变得复杂：

- 两条直线可以平行、相交、异面
- 直线与平面可以平行、相交、垂直、在平面内
- 两个平面可以平行、相交、垂直

传统几何证明需要空间想象，而向量方法提供了一套**程序化的判定工具**。

> **一句话：空间的位置关系，用向量来判定就是"比例"和"数量积"的游戏。**

---

### 原始定义（教材原文）

**直线与直线的位置关系：**
- 平行（包含重合）
- 相交
- 异面

**直线与平面的位置关系：**
- 直线在平面内
- 直线与平面平行
- 直线与平面相交（包含垂直）

**平面与平面的位置关系：**
- 平行（包含重合）
- 相交

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 12-18 页

---

### 本质定义

**用向量判定位置关系：**

| 关系 | 向量条件 | 说明 |
|------|---------|------|
| **直线//直线** | 方向向量共线：a = k·b | 两直线平行或重合 |
| **直线⊥直线** | 方向向量垂直：a·b = 0 | 两直线垂直（可能异面） |
| **直线//平面** | 直线的方向向量垂直于平面的法向量：a·n = 0 | 直线与平面平行 |
| **直线⊥平面** | 直线的方向向量平行于平面的法向量：a // n | 直线垂直于平面 |
| **平面//平面** | 两平面的法向量共线：n₁ = k·n₂ | 两平面平行 |
| **平面⊥平面** | 两平面的法向量垂直：n₁·n₂ = 0 | 两平面垂直 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **方向向量** | 直线的方向向量 |
| **法向量** | 垂直于平面的向量 |
| **垂直/平行** | 核心判定关系 |

---

### 等价表述

| 表述 | 向量条件 |
|------|---------|
| **直线与直线平行** | a//b ⟺ a = k·b |
| **直线与直线垂直** | a⊥b ⟺ a·b = 0 |
| **直线与平面平行** | a⊥n ⟺ a·n = 0 |
| **直线与平面垂直** | a//n ⟺ a = k·n |
| **平面与平面平行** | n₁//n₂ ⟺ n₁ = k·n₂ |
| **平面与平面垂直** | n₁⊥n₂ ⟺ n₁·n₂ = 0 |

### 概念辨析

**辨析 1：直线与直线垂直的不同情况**

| 情况 | 图形特征 | 向量条件 |
|------|---------|---------|
| 相交垂直 | 两条直线交于一点，且成 90° | a·b = 0，且交于一点 |
| 异面垂直 | 两条直线异面，但方向垂直 | a·b = 0，但不相交 |

异面垂直：两条异面直线的最短距离线段与两直线都垂直，而这条线段的方向向量与两直线的方向向量垂直（数量积为 0）。

**辨析 2：线面平行 vs 线面垂直**

| 关系 | 方向向量 a 与法向量 n 的关系 | 几何特征 |
|------|---------------------------|---------|
| 线面平行 | a ⊥ n | 直线与平面没有交点 |
| 线面垂直 | a // n | 直线垂直于平面内的所有直线 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| a = k·b | 平行（包括重合） | 需要 k 存在 |
| a·b = 0 | 垂直 | 不管是否相交 |
| a·n = 0 | 线//面 | n 是平面的法向量 |

## 命题层

### 核心定理

**定理 1：线面平行的向量判定**

```
直线 l 的方向向量 a⃗，平面 α 的法向量 n⃗
l // α ⟺ a⃗ ⊥ n⃗ ⟺ a⃗·n⃗ = 0
```

**定理 2：线面垂直的向量判定**

```
l ⊥ α ⟺ a⃗ // n⃗ ⟺ a⃗ = k·n⃗
```

**定理 3：面面平行的向量判定**

```
α // β ⟺ n₁⃗ // n₂⃗ ⟺ n₁⃗ = k·n₂⃗
```

**定理 4：面面垂直的向量判定**

```
α ⊥ β ⟺ n₁⃗ ⊥ n₂⃗ ⟺ n₁⃗·n₂⃗ = 0
```

---

### 典型例题

**例题：判断线面垂直**

```
题目：正方体 ABCD-A'B'C'D' 棱长为 1。
      判断直线 BC' 与平面 ABCD 是否垂直。
```

**深层分析：**

```
建立坐标系：A(0,0,0)

BC'⃗ = (0,1,1)
平面 ABCD 的法向量 n⃗ = (0,0,1)

检查：BC'⃗ // n⃗？
BC'⃗ = (0,1,1) = k·(0,0,1)？
k = 0 时横向分量为 0，但 BC'⃗ 的 y 分量是 1，所以不平行

再检查：BC'⃗ ⊥ n⃗？
BC'⃗·n⃗ = (0,1,1)·(0,0,1) = 1 ≠ 0

所以 BC'⃗ 与 n⃗ 既不平行也不垂直，但可以用夹角公式判断线面角
```

> **方法提炼：** 线面垂直的判定是方向向量与法向量平行，不平行则不垂直。

## 方法层

### 识别信号表

| 题目关键词 | 识别信号 | 方法 |
|-----------|---------|------|
| "平行""共线" | 方向关系 | 向量共线：a = k·b |
| "垂直""正交" | 垂直关系 | 数量积为 0 |
| "线面"关系 | 方向向量与法向量 | a·n 关系 |
| "面面"关系 | 两平面的法向量 | n₁·n₂ 关系 |

---

### 方法详解：位置关系判定流程

**Step 1: 识别要判断什么关系**

```
线线？线面？面面？
```

**Step 2: 找出相关向量**

```
线线 → 两直线的方向向量
线面 → 直线的方向向量 + 平面的法向量
面面 → 两平面的法向量
```

**Step 3: 判定**

```
平行 → 向量共线 (a = k·b)
垂直 → 数量积为 0 (a·b = 0)
```

> **易错提醒：** "//" 表示平行或重合，"⊥" 表示垂直但可能异面。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "a·b = 0 则两直线相交" | 忽略异面情况 | a·b = 0 只保证方向垂直，可能异面 |
| 2 | "线面平行就是直线在平面内" | 混淆 | 线面平行意味着没有交点，不是在内 |
| 3 | "面面平行就是两平面无交点" | 混淆 | 也可能完全重合 |
