# 状态转移矩阵

> 状态转移矩阵描述了系统状态之间的转移概率——用它可以预测系统经过多次转移后的长期概率分布。

---

## 概念层

### 问题的起源

股票价格有涨、跌、持平三种状态。今天的涨跌影响明天...

如何用数学描述这种"状态随时间变化"的系统？

用**状态转移矩阵** P，其中 Pᵢⱼ 表示从状态 i 转移到状态 j 的概率。

> **一句话：状态转移矩阵 P 描述了系统从一个状态转移到另一个状态的概率。**

---

### 本质定义

**状态转移矩阵的定义：**

```
P = [pᵢⱼ]，其中 pᵢⱼ = P(从状态 i 转移到状态 j)
```

每行和为 1（从每个状态出发的转移概率之和为 1）。

**n 步转移：**

```
Pⁿ = P × P × ... × P（n 个矩阵相乘）
```

---

### 典型应用

**马尔可夫链：** 未来只与当前状态有关，与历史无关。

---

## 命题层

### 核心定理

**定理：n 步转移概率**

```
第 n 步从状态 i 到状态 j 的概率 = Pⁿ 的 (i,j) 元素
```

**定理：稳态分布**

```
若存在极限分布 π，满足 πP = π
则 π 是平稳分布
```
