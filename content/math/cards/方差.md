# 方差

> 方差是随机变量"偏离均值的程度"——Var(X) = E[(X−E(X))²]，越大说明数据越分散。

---

## 概念层

### 问题的起源

两个随机变量的均值可能相同，但分散程度不同。

- X：取值 49, 50, 51（均值 50）
- Y：取值 0, 50, 100（均值 50）

X 集中在均值附近，Y 则更分散。

如何量化这个"分散程度"？

用 (X − E(X))² 的期望——这就是方差。

> **一句话：方差是随机变量取值偏离均值的平方期望，衡量数据的分散程度。**

---

### 本质定义

**方差的定义：**

```
Var(X) = E[(X − E(X))²]
       = Σ (xᵢ − E(X))² × pᵢ
```

**标准差** = √Var(X)，与 X 单位相同。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **偏离** | X − E(X)，偏离均值 |
| **平方** | (X − E(X))²，消除负号 |
| **期望** | E[(X − E(X))²]，加权平均 |

---

### 计算公式

| 公式 | 说明 |
|------|------|
| Var(X) = E(X²) − [E(X)]² | 常用计算公式 |

---

### 性质

| 性质 | 公式 |
|------|------|
| 常数倍 | Var(aX + b) = a²Var(X) |
| 独立可加 | Var(X + Y) = Var(X) + Var(Y)（独立时） |

---

## 命题层

### 核心定理

**定理：常用分布的方差**

```
① X ~ B(n,p)：Var(X) = np(1-p)
② X ~ 超几何：Var(X) = n × K/N × (N-K)/N × (N-n)/(N-1)
```

**定理：方差与期望的关系**

```
Var(X) = E(X²) − [E(X)]²
```

---

### 典型例题

**例题：计算方差**

```
题目：掷骰子，X 为点数，求 Var(X)。
```

**深层分析：**

```
E(X) = 3.5（如前一例）

E(X²) = 1²×1/6 + 2²×1/6 + ... + 6²×1/6
      = (1+4+9+16+25+36)/6
      = 91/6

Var(X) = E(X²) − [E(X)]²
       = 91/6 − (3.5)²
       = 91/6 − 12.25
       = 35/12 ≈ 2.92
```

> **方法提炼：** Var(X) = E(X²) − [E(X)]²。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "方差单位是原数据单位" | 误解 | 方差单位是原数据单位的平方 |
| 2 | "Var(X+Y) = Var(X)+Var(Y)" | 忽略条件 | 独立时才成立 |
| 3 | "方差越大越差" | 误解 | 方差大只是说明分散，方差本身无所谓好坏 |
