# 突变率

> 突变率是"每个配子或每个世代中，基因发生突变的概率"——它是群体遗传变异的最终来源。

---

## 概念层

### 问题的起源

基因突变是进化的原材料。

突变率 μ 表示每个配子或每个世代中，基因 A 突变为 a（或反之）的概率。

典型值：μ ≈ 10⁻⁵ ~ 10⁻⁸（每代每基因的突变率很低）

> **一句话：突变率是基因发生突变的概率，是遗传变异的来源。**

---

### 本质定义

**突变率的定义：**

```
基因 A → a 的突变率：μ₁
基因 a → A 的回复突变率：μ₂
```

**平衡条件：**

```
当 μ₁p = μ₂q 时，基因频率达到平衡
p（A频率）和 q（a频率）不再变化
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **突变率 μ** | 每代每个配子的突变概率 |
| **突变方向** | A→a（正向）或 a→A（回复突变） |
| **平衡频率** | μ₁p = μ₂q 时的 p, q |

---

## 命题层

### 核心定理

**定理：突变平衡**

```
设 A → a 的突变率为 μ
a → A 的回复突变率为 ν
平衡时：μp = νq
即 p = ν/(μ+ν)，q = μ/(μ+ν)
```
