# 二项式定理

> 二项式定理把 (a+b)ⁿ 展开成一项一项的和——系数是组合数 Cₙᵏ。

---

## 概念层

### 问题的起源

(a + b)¹ = a + b
(a + b)² = a² + 2ab + b²
(a + b)³ = a³ + 3a²b + 3ab² + b³

规律是什么？

系数的规律：1, 2, 1 → 3, 3, 1 → ...
这是组合数 Cₙᵏ！

**二项式定理：**
(a + b)ⁿ = Σ Cₙᵏ aⁿ⁻ᵏ bᵏ = Cₙ⁰aⁿ + Cₙ¹aⁿ⁻¹b + ... + Cₙⁿbⁿ

> **一句话：(a+b)ⁿ 的展开系数就是组合数 Cₙᵏ。**

---

### 原始定义（教材原文）

```
(a + b)ⁿ = Cₙ⁰aⁿ + Cₙ¹aⁿ⁻¹b + Cₙ²aⁿ⁻²b² + ... + Cₙⁿbⁿ
```

其中 Cₙᵏ = n!/[k!(n−k)!] 是组合数。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修三》第 6-10 页

---

### 本质定义

**二项式展开式：**

```
(a + b)ⁿ = Σ(k=0 to n) Cₙᵏ aⁿ⁻ᵏ bᵏ
```

**第 k+1 项（通项）：**

```
Tₖ₊₁ = Cₙᵏ aⁿ⁻ᵏ bᵏ
```

**杨辉三角：**

```
     C₅⁰ = 1
    C₅¹  C₅⁰ = 1
   C₅² C₅¹  C₅⁰ = 1
  C₅³ C₅² C₅¹  C₅⁰ = 1
 C₅⁴ C₅³ C₅² C₅¹  C₅⁰ = 1
       1
      1 1
     1 2 1
    1 3 3 1
   1 4 6 4 1
```

## 命题层

### 核心定理

**定理 1：二项式展开**

```
(a + b)ⁿ = Σ(k=0 to n) Cₙᵏ aⁿ⁻ᵏ bᵏ
```

**定理 2：通项公式**

```
Tₖ₊₁ = Cₙᵏ aⁿ⁻ᵏ bᵏ（注意是第 k+1 项）
```

**定理 3：二项式系数的性质**

```
Cₙ⁰ + Cₙ¹ + ... + Cₙⁿ = 2ⁿ
Cₙ⁰ + Cₙ² + ... = Cₙ¹ + Cₙ³ + ... = 2ⁿ⁻¹
```

---

### 典型例题

**例题：求 (x + 1/x)⁶ 展开式中的常数项**

```
题目：求 (x + 1/x)⁶ 展开式中的常数项。
```

**深层分析：**

```
通项：Tₖ₊₁ = C₆ᵏ x⁶⁻ᵏ · (1/x)ᵏ = C₆ᵏ x⁶⁻²ᵏ

常数项要求指数为 0：6 − 2k = 0 ⟹ k = 3

常数项 = C₆³ = 20
```

> **方法提炼：** 常数项 → 指数为 0 → 求 k → 求 Cₙᵏ。

### 方法详解：二项式展开问题

**Step 1: 识别 a, b, n**

```
(a + b)ⁿ，其中 a = ..., b = ..., n = ...
```

**Step 2: 写出通项**

```

```

**Step 3: 根据要求找特定项**

```
常数项 → 指数为 0
x³ 项 → 指数为 3
```

| 编号 | 常见误区 | 错误根源 | 正确理解 |
