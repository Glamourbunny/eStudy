# 离散型随机变量

> 离散型随机变量是"取值可以一一列举"的随机变量——它通过分布列来完整描述。

---

## 概念层

### 问题的起源

随机变量有两种：
- **离散型**：取值可以一一列举（0, 1, 2, ...）
- **连续型**：取值充满一个区间，无法一一列举

掷骰子的点数是离散型随机变量（1, 2, 3, 4, 5, 6）。

测量身高是连续型随机变量（可以是任何实数值）。

> **一句话：离散型随机变量是取值有限的或可列的随机变量。**

---

### 本质定义

**离散型随机变量的特征：**

| 特征 | 说明 |
|------|------|
| 可列取值 | x₁, x₂, ..., xₙ（有限个或无限可列个） |
| 每个取值有概率 | P(X = xᵢ) = pᵢ |
| 完整描述 | 通过分布列 {xᵢ, pᵢ} 完整描述 |

---

### 与连续型随机变量的对比

| 对比项 | 离散型随机变量 | 连续型随机变量 |
|--------|---------------|---------------|
| 取值 | 可列举 | 区间内任意值 |
| 概率描述 | 分布列 P(X = xᵢ) | 概率密度 f(x) |
| 概率计算 | P(X = a) | P(a ≤ X ≤ b) = ∫f(x)dx |
| 例子 | 掷骰子点数 | 身高测量值 |

---

## 命题层

### 核心定理

**定理：分布列的性质**

```
① pᵢ ≥ 0（概率非负）
② Σpᵢ = 1（概率总和为 1）
```

**定理：期望和方差的计算**

```
E(X) = Σ xᵢ pᵢ
Var(X) = Σ (xᵢ − E(X))² pᵢ
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "离散型随机变量不能取无穷多值" | 误解 | 无限可列个也可以（如正整数） |
| 2 | "P(X = 2.5) 对离散型有意义" | 误解 | 如果 X 不取 2.5，则概率为 0 |
| 3 | "离散型没有密度函数" | 正确 | 连续型才有密度函数 |
