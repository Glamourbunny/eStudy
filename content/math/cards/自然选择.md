# 自然选择

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 自然选择在高考遗传计算中以"适合度/致死基因/频率逐代变化"出现——它是 Hardy-Weinberg 的"破坏者"：原本稳定的基因频率，因为某些基因型生存或繁殖能力不同，开始定向漂移。你的任务是：找出"谁在筛选"，建立递推。

---

## 视角层（像什么）

### 核心隐喻：筛子筛选——网眼大小决定谁能通过

**自然选择 = 一个筛子悬在群体上方。每繁殖一代之前，筛子先过一遍——适合度低的基因型"漏不下去"（被淘汰），只有适合度高的才能进入下一代基因池。**

```
  初始群体（繁殖前）：
  
  AA: 25%,  Aa: 50%,  aa: 25%
  
  ↓ 筛子一（环境选择）：
  
  如果 aa 的适合度为 0（致死）：
  
  AA: 25%  ──→ 全部通过 ✓
  Aa: 50%  ──→ 全部通过 ✓
  aa: 25%  ──→ 全部淘汰 ✗（致死基因，活不到繁殖年龄）
  
  繁殖池中的有效群体：
  AA: 0.25/(0.25+0.5) = 1/3
  Aa: 0.5/(0.25+0.5) = 2/3
  aa: 0
  
  新的基因频率：
  p' = P(AA有效) + ½P(Aa有效) = 1/3 + ½(2/3) = 2/3
  q' = 1 - p' = 1/3
  
  对比：选择前 p=0.5, q=0.5 → 选择后 p'=0.667, q'=0.333
  ↓ a 的频率从 0.5 降到了 0.333 ← 自然选择在改变频率
```

**筛子的三档网眼大小（适合度分级）：**

| 筛选强度 | 直觉理解 | 频率变化速度 |
|---------|---------|-------------|
| **致死（s=1）** | aa 完全不能存活 → 网眼 = 0，全部拦截 | 最快——一代淘汰所有 aa，但隐性 a 藏在 Aa 里 |
| **部分选择（0<s<1）** | aa 只有部分能存活 → 网眼中等 | 速度中等 |
| **杂合子优势（Aa 最适合）** | 中间型最适合 → 筛子反而放过中等大小 | 维持多态性，两种基因都不会被完全淘汰 |

> **一句话直觉：** 自然选择是在繁殖前摆一道筛子，不同基因型通过筛子的概率不同。适合度低 = 筛子拦下，基因频率下降；适合度高 = 筛子放行，基因频率上升。高考题就是算"筛子过完一轮后，基因频率变到了多少"。

### 动态想象：温度计在锅里的变化

```
  把基因频率想象成一根温度计：

  有选择压力时：
  • 选择压（筛子网眼）→ 温度计往一个方向稳定变化
  • a 有害（被筛掉）→ a 频率下降 → "温度"持续降低
  • a 有利（被保留）→ a 频率上升 → "温度"持续升高
  
  没有选择压力时（Hardy-Weinberg）：
  • 温度计的读数不动
  • 但分子（个体）还在交换配对
```

**考场翻版：** 题目说"aa 个体在出生前死亡""AA 适合度为 0.8，Aa 适合度为 1""隐性纯合致死"——这就是筛子在起作用。你需要根据每种基因型的"通过率"重新计算繁殖池里的有效比例。

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **隐性致死，a 很快消失** | "aa 全死了，a 应该一代灭绝" | 隐性 a 藏在 Aa 杂合子里！即使 aa 全部淘汰，Aa 携带的 a 仍然在基因池里。消除隐性有害基因极慢 |
| **杂合子也有害** | "Aa 和 AA 表型一样（显性），所以适合度一样" | 有时杂合子有隐藏代价（如镰刀型贫血的携带者），适合度可能介于两者之间 |
| **杀死 = 频率立刻变化** | "筛子过后立刻看到效果" | 一代只改变一点点频数，要很多代才能看到显著变化 |
| **选择 = 增加好的** | "有利基因越多越好" | 有时杂合子比纯合子更适应（杂合优势），选择不追求"纯化"而追求"最优混合" |

## 联结层（连什么）

### 跨学科锚点

**1. 生态：抗生素耐药性**

```
  细菌群体 + 抗生素（筛子）：
  • 敏感型细菌 → 被杀灭（适合度 = 0）
  • 耐药型细菌 → 存活繁殖（适合度 = 1）
  
  几代之后：耐药基因频率急剧上升
  ↓ "超级细菌"就是自然选择在人类眼皮底下发生的演化
  ↓ 时间尺度：细菌几十分钟一代，一年内能看到显著变化
```

**2. 工程：质量控制筛选**

```
  工厂流水线：
  • 产品通过质检（筛子）才能出厂
  • 不良品被拒收（致死）
  ↓ "出厂品"的不良率取决于筛子严格度和上一批次遗留的不良率
  
  对应遗传学：
  • "出厂品" = 进入基因池的有效繁殖个体
  • "不良品" = 被选择淘汰的基因型
```

**3. 化学：蒸馏提纯**

```
  混合液体（初始群体）→ 加热蒸馏（选择性筛子）
  • 沸点低 → 先蒸发出来（高适合度）
  • 沸点高 → 留在瓶底（低适合度）
  
  多轮蒸馏 → 纯度越来越高（对应：多代选择 → 有利基因频率趋近 1）
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 隐性纯合致死：aa 在繁殖前淘汰 | 筛选后有效比例变化 | 有效 p' = P(AA幸存)/(幸存总体), 新 q' = ½P(Aa幸存)/(幸存总体) |
| 部分选择：每种基因型有不同适合度 w | 加权有效比例 | 有效 p' = (w_AA·p² + ½w_Aa·2pq) / w̄（平均适合度） |
| 多代选择趋势 | 频率递推公式 | q_{n+1} = f(q_n)，分析收敛性 |

**首选工具**：适合度加权重新计算有效频率
**次选工具**：递推公式（多代预测）

### 概念网络坐标

```
              ┌─────────────────┐
              │   Hardy-Weinberg │
              │     平衡状态     │
              └────────┬────────┘
                       │
        ┌──────────────┼──────────────┐
        │              │              │
        ▼              ▼              ▼
  ┌─────────┐   ┌───────────┐  ┌──────────┐
  │ 自然选择 │   │   突变    │  │   漂变   │
  │ 筛子筛选 │   │ A↔a 互换  │  │ 随机波动 │
  └────┬────┘   └───────────┘  └──────────┘
       │
       ▼
  ┌──────────────────────────┐
  │  适合度 w：通过筛子的概率 │
  │  致死 s=1，中性 s=0      │
  │  w = 1 − s              │
  └──────────────────────────┘
```

**拆包调用优先级：**
1. 找出每种基因型的"适合度"或"存活率"
2. 用初始基因型频率 × 适合度 = 有效比例
3. 归一化有效比例（除以 w̄）
4. 从有效比例重新算基因频率

### 历史/哲学视角：从神创到自然选择

```
1859 年（达尔文《物种起源》）：
  "自然选择"这个词诞生——但达尔文没有数学工具
  他不知道基因为何物（孟德尔的论文就在他书房里，但没读过）
  
19 世纪末-20 世纪初：
  孟德尔遗传学 + 达尔文自然选择 = 群体遗传学
  费希尔、霍尔丹、赖特三人各自将自然选择数学化
  ↓ 适合度 w、选择系数 s 进入方程
  ↓ 演化生物学从此有了定量语言

  赖特的"适应度景观"：把基因频率变化想象成爬山
  选择驱动力 = 山上的坡度，推动群体往适应度更高的方向
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | "aa 个体在出生前全部死亡""AA 的适合度为 0.8，Aa 为 1.0""选择系数 s=0.2" | 不知道"适合度""选择系数"就是"通过筛子的概率"，硬用原始频率算 | 3 秒认出筛子逻辑：有效频率 = 原始频率 × 适合度 / 平均适合度 |
| 🔀 **嵌套** | 先有选择改变频率，再随机交配一代 | 选择后的频率没算完就交配，两步混淆 | 拆两层：选择层（加权求有效频率）→ 交配层（用新频率套 HW） |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"致死""适合度""存活率""选择系数"？
    → 自然选择在筛，基因频率会变
  • 确定：完全致死（s=1）还是部分选择（0<s<1）？

Step 2：剥外壳找骨骼（4 秒）
  • 列出每种基因型的适合度 w
  • 计算平均适合度 w̄ = Σ(基因型频率 × w)
  • 有效频率 = 原始频率 × w / w̄

Step 3：锁定 Formal 着陆点（3 秒）
  • 从有效频率重新算 p' 和 q'
  • 如果题目说"选择后随机交配" → 再用新频率套 HW
  • 写下：拆包→【致死基因/适合度计算】，立即切换
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 筛子逻辑已理解，各基因型的适合度/存活率已提取 | 写出 p' = (w_AA·p² + ½·w_Aa·2pq) / w̄ |
| 嵌套题 | 两层已分开：选择层 + 交配层 | 先算选择后的新 p/q，再代入 p²:2pq:q² |

---

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **忘记平均适合度的归一化** | 算了有效比例但没除以 w̄（总和不等于 1） | 有效比例必须归一化：每个有效比例除以总和 |
| **隐性淘汰极慢** | 以为 aa 致死一代后 a 就没了 | a 藏在 Aa 杂合子里！算一下：初始 q=0.5，一代选择后 q'=1/3，还远不为 0 |
| **适合度 > 1** | 以为适合度只能是 [0,1] | 相对适合度可以 > 1——通常是最高适合度设为 1，其他小于 1 |
| **忽略杂合子的适合度** | 只考虑纯合子的选择忽略 Aa | Aa 也可能有选择效应（杂合劣势或杂合优势） |

---

## 一句话总结

> **自然选择是"基因池上的筛子"——每种基因型通过筛子的概率不同。适合度低的被拦下，基因频率下降；适合度高的被放行，基因频率上升。高考里它以"致死/适合度/选择系数"出现，你的任务是算出筛子过后各基因型的有效比例，然后重新算基因频率。**
