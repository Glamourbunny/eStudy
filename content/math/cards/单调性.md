# 单调性

> 单调性描述的是：当 x 增大时，y 的变化趋势——变大还是变小？

---

## 概念层

### 问题的起源

"当 x 增大时，y 会怎么变？"

这个问题在现实中无处不在：
- 物价上涨时，需求如何变化？（需求曲线通常是递减的）
- 气温随海拔升高如何变化？（通常气温递减）
- 投资收益随时间如何增长？（通常收益递增）

为了精确描述这种"同增同减"的趋势，数学家引入了**单调性**概念。

> **一句话：** 单调性描述的是自变量变化时，函数值的整体变化趋势。

---

### 原始定义（教材原文）

设函数 f(x) 的定义域为 I，区间 D ⊆ I。如果对于区间 D 上的任意两个点 x₁, x₂，当 x₁ < x₂ 时，都有：

- f(x₁) < f(x₂)，那么就说函数 f(x) 在区间 D 上是**增函数**（递增）
- f(x₁) > f(x₂)，那么就说函数 f(x) 在区间 D 上是**减函数**（递减）

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 72 页

**定义拆解：**

| 关键词 | 含义 | 注意事项 |
|--------|------|---------|
| 区间 D | 单调性是**区间性质**，必须指定区间 | 不能脱离区间谈单调 |
| "任意" | 对区间内任意两点都成立，不能有反例 | 必须严格的量化表述 |
| "当 x₁ < x₂ 时" | 比较的基础是 x 的大小关系 | 不能搞反顺序 |

---

### 本质定义（逻辑重构版）

**增量形式（本质）：**

```
f(x) 在 D 上递增 ⟺ ∀ x₁, x₂ ∈ D, x₁ < x₂ ⇒ f(x₂) − f(x₁) > 0
f(x) 在 D 上递减 ⟺ ∀ x₁, x₂ ∈ D, x₁ < x₂ ⇒ f(x₂) − f(x₁) < 0
```

**差商形式（等价表述）：**

```
f(x) 在 D 上递增 ⟺ ∀ x₁, x₂ ∈ D, x₁ ≠ x₂, (f(x₂) − f(x₁))/(x₂ − x₁) > 0
f(x) 在 D 上递减 ⟺ ∀ x₁, x₂ ∈ D, x₁ ≠ x₂, (f(x₂) − f(x₁))/(x₂ − x₁) < 0
```

> 差商形式的几何意义：连接曲线上任意两点的割线斜率恒正（递增）或恒负（递减）。

### 单调性与函数图像

```
        y                         y
        │                        │
    ────┼───────→ x          ────┼───────→ x
        │                        │
   递增函数                    递减函数
   （上升型）                 （下降型）
        │                        │
        │     ╱                  │  ╲
        │   ╱                    │    ╲
        │ ╱                      │      ╲
```

**图像特征：**
- 增函数：图像从左到右**上升**
- 减函数：图像从左到右**下降**

---

### 邻近概念辨析

**辨析 1：单调性 vs 奇偶性 —— "走势" vs "对称"**

| 对比项 | 单调性 | 奇偶性 |
|--------|--------|--------|
| 描述内容 | x 变化时 y 如何变 | x 变号时 y 如何变 |
| 图像特征 | 上升/下降趋势 | 关于 y 轴或原点对称 |
| 成立条件 | 区间内任取两点 | 定义域关于原点对称 |
| 组合类型 | 增+增、减+减、增+减、减+增 | 奇+奇、偶+偶 |

**辨析 2：单调性 vs 有界性 —— "趋势" vs "范围"**

| 对比项 | 单调性 | 有界性 |
|--------|--------|--------|
| 描述对象 | y 随 x 的变化趋势 | y 能取到的数值范围 |
| 典型函数 | y = 2x（无界递增） | y = sin x（有界） |
| 两者关系 | 独立性质，可同时成立 | 可独立存在 |

---

### 适用边界

| 成立条件 | 不成立情形 | 典型反例 |
|---------|-----------|---------|
| 必须在区间上讨论 | 单点没有单调性 | f(x)=x² 在 x=0 处不讨论单调 |
| 整个区间内无反例 | 存在 x₁<x₂ 但 f(x₁)≥f(x₂) | f(x)=x² 在 R 上不单调 |
| 与最值不同 | 有最值未必单调 | f(x)=-x² 在 [0,1] 有最大值 |

## 命题层

### 核心定理

**定理 1：单调性的运算性质**

设 f(x) 和 g(x) 在区间 D 上同单调（都是增或都是减）：

| 运算 | 条件 | 结果 |
|------|------|------|
| f + g | 同为增（或同为减） | 在 D 上同号单调 |
| f − g | f 增，g 减 → 增 | f 减，g 增 → 减 |
| f · g | f,g ≥ 0，且同增或同减 | 在 D 上递增 |
| 1/f | f > 0 且递增 → 递减 | f < 0 且递减 → 递增 |

**定理 2：复合函数的单调性（同增异减）**

设 y = f(g(x))，若 f 和 g 都可单调：

```
外层 f（增减）× 内层 g（增减）= 整体单调性
     增      ×     增      =     增
     增      ×     减      =     减
     减      ×     增      =     减
     减      ×     减      =     增
```

> **记忆口诀：** 同增异减——内外同向则增，反向则减。

**定理 3：导数与单调性的关系（高二内容）**

```
f'(x) > 0（在区间 D 上） ⟺ f(x) 在 D 上递增
f'(x) < 0（在区间 D 上） ⟺ f(x) 在 D 上递减
f'(x) = 0 的点可能是极值点，也可能只是平点
```

### 典型例题

**例题：单调性的判断与证明**

```
题目：判断并证明 f(x) = −2x + 3 在 R 上的单调性。
```

**深层分析：**

这是一道证明题，需要严格按照单调性定义进行证明。

**证明：**

```
任取 x₁, x₂ ∈ R，且 x₁ < x₂

则 f(x₁) − f(x₂) = (−2x₁ + 3) − (−2x₂ + 3)
                  = −2x₁ + 3 + 2x₂ − 3
                  = −2(x₁ − x₂)
                  = 2(x₂ − x₁)

∵ x₁ < x₂  ∴ x₂ − x₁ > 0
∴ f(x₁) − f(x₂) = 2(x₂ − x₁) > 0
即 f(x₁) > f(x₂)

∴ f(x) 在 R 上是减函数
```

> **方法提炼：** 证明单调性的标准流程是"任取 → 作差 → 变形 → 判断符号 → 结论"。关键是因式分解后能判断每个因子的正负。

---

**例题：利用单调性解不等式**

```
题目：已知 f(x) 是 R 上的增函数，解不等式 f(3x − 1) > f(2)。
```

**深层分析：**

这道题的关键是利用"增函数：自变量大 → 函数值大"的等价关系。

**解题步骤：**

```
f(3x − 1) > f(2)
    ↓
∵ f(x) 是增函数
    ↓
3x − 1 > 2          （函数值大 ⟺ 自变量大）
    ↓
3x > 3
    ↓
x > 1
```

> **方法提炼：**
> - 增函数：f(a) > f(b) ⟺ a > b（不等号方向**不变**）
> - 减函数：f(a) > f(b) ⟺ a < b（不等号方向**改变**）
> - 前提：必须确认函数在相关区间上的单调性！

---

**例题：含参函数的单调性讨论**

```
题目：讨论 f(x) = ax − x³ (a > 0) 在 x > 0 上的单调性。
```

**深层分析：**

需要用导数工具：f'(x) = a − 3x²。

```
令 f'(x) = 0，得 a − 3x² = 0
解得 x = √(a/3)

当 0 < x < √(a/3) 时：f'(x) > 0，f(x) 递增
当 x > √(a/3) 时：f'(x) < 0，f(x) 递减
```

> **方法提炼：** 含参单调性问题的关键是根据导数的零点划分区间，分别讨论各区间的符号。

## 方法层

### 方法 1：定义法证明单调性

**核心思想：** 严格按照"任取 → 作差 → 变形 → 判断符号 → 结论"的流程。

**标准操作流程：**

```
Step 1: 任取 x₁, x₂ ∈ D，且 x₁ < x₂
Step 2: 作差：f(x₁) − f(x₂)（或 f(x₂) − f(x₁)）
Step 3: 变形：因式分解，化为乘积形式
Step 4: 判断符号：根据已知条件（通常是 x₁ < x₂ 得出的大小关系）
Step 5: 结论：根据符号判断单调性
```

**易错提醒：**
- 不能假设要证明的结论
- 变形要彻底，能看清符号
- 最后一步要明确写出"增"或"减"

---

### 方法 2：图像法判断单调性

**核心思想：** 直观观察图像的升降趋势。

**判断技巧：**

| 图像走势 | 单调性 |
|---------|--------|
| 从左到右上升 | 增函数 |
| 从左到右下降 | 减函数 |
| 先升后降 | 先增后减（可能有极值点） |

---

### 方法 3：导数法判断单调性（高二+）

**核心思想：** 通过导数符号判断原函数的增减。

**决策树：**

```
求 f'(x)
    │
    ├─ f'(x) > 0（整个区间）→ 增函数
    │
    ├─ f'(x) < 0（整个区间）→ 减函数
    │
    └─ f'(x) 变号 → 分段讨论
        │
        ├─ f'(x) > 0 → 该区间递增
        └─ f'(x) < 0 → 该区间递减
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "f(x) 在 x=1 处递增" | 混淆点与区间 | 单调性是**区间性质**，必须说在哪个区间 |
| 2 | "f'(x₀)=0 则 x₀ 是极值点" | 混淆必要条件与充要条件 | f(x)=x³ 在 x=0 处导数为 0 但不是极值点 |
| 3 | "增函数就是一直变大" | 忽略区间限制 | 必须在指定区间内考虑 |
| 4 | "单调递增和严格递增一样" | 混淆包含关系 | 非严格递增允许相等，严格递增不允许 |
| 5 | "f'(x)>0 则函数递增" | 忽略区间条件 | 必须在整个区间上 f'(x)>0 |
