# 线性回归

> 线性回归是"用一条直线近似表示两个变量之间的关系"——找到一条直线，使得所有点到直线的"距离平方和最小"。

---

## 概念层

### 问题的起源

两个变量 x 和 y 有某种关系，但数据有随机误差。如何找到它们之间的关系？

答案是找到一条直线 ŷ = a + bx，使得预测值 ŷ 与实际值 y 的偏差最小。

这就是**最小二乘线性回归**。

> **一句话：线性回归是找到一条直线，使得误差平方和最小。**

---

### 本质定义

**线性回归模型：**

```
ŷ = a + bx
其中 b = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
      a = ȳ - bx̄
```

**最小二乘原则：**

```
最小化 Σ(yᵢ - ŷᵢ)²
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **回归直线** | ŷ = a + bx |
| **斜率 b** | 反映 x 对 y 的影响 |
| **截距 a** | 直线在 y 轴的截距 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **回归方程** | ŷ = a + bx |
| **最小二乘法** | 最小化 Σ(yᵢ - ŷᵢ)² |
| **预测** | 已知 x 预测 ŷ |

---

## 命题层

### 核心定理

**定理：回归系数公式**

```
b = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
a = ȳ - bx̄
```

**定理：决定系数 R²**

```
R² = 1 - Σ(yᵢ - ŷᵢ)² / Σ(yᵢ - ȳ)²
```

R² 越接近 1，直线拟合越好。

---

### 典型例题

**例题：求线性回归方程**

```
题目：给定数据点 (1,2), (2,4), (3,5)，求回归直线。
```

**深层分析：**

```
计算：
x̄ = 2, ȳ = 11/3

b = [(1-2)(2-11/3) + (2-2)(4-11/3) + (3-2)(5-11/3)] / [(1-2)²+(2-2)²+(3-2)²]
  = ...

a = ȳ - bx̄

最终得到 ŷ = a + bx
```

> **方法提炼：** 用公式计算 b 和 a。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "回归直线过所有点" | 误解 | 只使误差平方和最小 |
| 2 | "x 和 y 有因果关系" | 误解 | 回归只描述相关性 |
| 3 | "R² = 1 就完美" | 误解 | 可能是过拟合 |
