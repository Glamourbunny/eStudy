# 正态分布

> 正态分布是自然界最常见的分布——呈钟形曲线，用 N(μ, σ²) 表示，其概率密度曲线关于 μ 对称。

---

## 概念层

### 问题的起源

人的身高、测量误差、考试成绩……这些数据往往呈"中间多、两头少"的分布。

这种分布在数学上被精确描述为**正态分布**——呈钟形曲线，关于均值对称。

> **一句话：正态分布是"中间多、两头少"的钟形分布。**

---

### 本质定义

**正态分布的概率密度函数：**

```
f(x) = (1/√(2πσ²)) × e^[−(x−μ)²/(2σ²)]
```

**参数：**

| 参数 | 含义 |
|------|------|
| μ | 均值，也是对称中心 |
| σ² | 方差，决定曲线"胖瘦" |

**记号：** X ~ N(μ, σ²)

---

### 图像特征

```
          f(x)
           │
           │         ╱╲
           │        ╱  ╲
           │       ╱    ╲
           │      ╱      ╲
           │     ╱        ╲
           │    ╱          ╲
           │   ╱            ╲
           │  ╱              ╲
           └────────────────────→ x
              μ-3σ  μ-σ   μ  μ+σ μ+3σ
```

- 关于 x = μ 对称
- 68-95-99.7 法则：约 68% 在 μ±σ 内，约 95% 在 μ±2σ 内，约 99.7% 在 μ±3σ 内

---

### 与其他分布的关系

| 对比项 | 正态分布 | 二项分布 |
|--------|---------|-----------|
| 类型 | 连续型 | 离散型 |
| 参数 | μ, σ² | n, p |
| 近似 | — | np > 5 且 np(1-p) > 5 时可近似 |

## 命题层

### 核心定理

**定理：标准化变换**

```
若 X ~ N(μ, σ²)，则 Z = (X-μ)/σ ~ N(0, 1)
Z 服从标准正态分布
```

**定理：正态分布的概率计算**

```
P(a < X < b) = Φ((b-μ)/σ) - Φ((a-μ)/σ)
其中 Φ 是标准正态分布函数
```

---

### 典型例题

**例题：正态分布概率计算**

```
题目：已知 X ~ N(100, 15²)，求 P(85 < X < 115)。
```

**深层分析：**

```
标准化：Z = (X-100)/15
P(85 < X < 115) = P((85-100)/15 < Z < (115-100)/15)
                = P(-1 < Z < 1)
                ≈ 0.6826（68% 法则）
```

> **方法提炼：** 标准化后查标准正态分布表。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "正态分布是离散分布" | 混淆 | 正态分布是连续分布 |
| 2 | "P(X = μ) = 0" | 误解 | 连续型随机变量取单点概率为 0 |
| 3 | "所有数据都服从正态分布" | 误解 | 只有"中间多两头少"的数据才近似正态 |
