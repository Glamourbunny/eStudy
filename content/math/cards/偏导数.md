# 偏导数

> 偏导数是"多变量函数中，只让一个变量变化、其他变量固定时的导数"——∂f/∂x 衡量 f 对 x 的局部变化率。

---

## 概念层

### 问题的起源

二元函数 f(x, y) = x² + xy + y²

当 x 变化时 f 如何变化？但 y 也会变化...

怎么办？——分别研究每个变量的影响。

固定 y，对 x 求导，得到**偏导数** ∂f/∂x。

> **一句话：偏导数 ∂f/∂x 是让其他变量固定，只让 x 变化时 f 的变化率。**

---

### 本质定义

**偏导数的定义：**

```
∂f/∂x = lim_{Δx→0} [f(x+Δx, y) - f(x, y)] / Δx
```

（这里 y 被当作常数）

类似地：
```
∂f/∂y = lim_{Δy→0} [f(x, y+Δy) - f(x, y)] / Δy
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **多变量** | 函数有多个自变量 |
| **固定其他** | 求偏导时其他变量视为常数 |
| **偏导数符号** | ∂f/∂x 或 fₓ |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "∂f/∂x 和 df/dx 一样" | 混淆 | ∂ 表示多变量中只对 x 求导 |
| 2 | "偏导数不考虑其他变量的影响" | 误解 | 偏导数反映的是局部变化率 |
| 3 | "∂²f/∂x∂y = ∂²f/∂y∂x" | 需要可微条件 | 当函数二阶偏导连续时相等 |
