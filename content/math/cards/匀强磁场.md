# 匀强磁场

> 匀强磁场是磁场强度和方向都恒定不变的磁场——它的磁感线是均匀分布的平行直线。

---

## 概念层

### 问题的起源

磁场有两种：
- **非匀强磁场**：磁铁、通电导线周围的磁场，强度随位置变化
- **匀强磁场**：磁场强度 B 和方向在空间各处都相同

匀强磁场在电磁学中特别重要，因为：
- 带电粒子在其中运动轨迹是圆形或螺旋线（可预测）
- 许多实际应用（质谱仪、回旋加速器）都基于匀强磁场

> **一句话：匀强磁场是"大小和方向都不变"的磁场，带电粒子在其中做匀速圆周运动。**

---

### 原始定义（教材原文）

磁感应强度 B 处处相同、方向也处处相同的磁场称为**匀强磁场**。

匀强磁场的磁感线是相互平行的、疏密程度相同的直线。

> 出处：人教版《普通高中教科书·物理（选择性必修）》

---

### 本质定义

**匀强磁场的特点：**

| 特点 | 说明 |
|------|------|
| **B 大小** | 空间各点相等 |
| **B 方向** | 空间各点相同 |
| **磁感线** | 等间距平行直线 |
| **产生方式** | 平行载流导线或螺线管内部 |

**常见产生方式：**
- 两个平行放置的磁铁（N 极与 S 极相对，中间区域近似匀强）
- 长直螺线管内部（远离两端处）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **磁感应强度 B** | B = F/(IL)，单位 Tesla（T） |
| **方向** | 磁感线切线方向 |
| **大小恒定** | B 不随位置变化 |

## 命题层

### 核心定理

**定理 1：洛伦兹力公式**

```
F = qvBsinθ
其中 θ 是 v 与 B 的夹角
```

当 v ⊥ B（θ = 90°）时，F = qvB，这是带电粒子做匀速圆周运动的向心力。

**定理 2：半径公式**

```
qvB = mv²/r
⟹ r = mv/(qB)
```

半径 r 与速度 v 成正比，与 B 和 q 成反比。

**定理 3：周期公式**

```
T = 2πr/v = 2πm/(qB)
```

周期 T 与速度 v 无关，只与 m/(qB) 有关。

---

### 典型例题

**例题：带电粒子在匀强磁场中的运动半径**

```
题目：一个电子（q = 1.6×10⁻¹⁹ C，m = 9.1×10⁻³¹ kg）
      以 v = 1×10⁶ m/s 的速度垂直射入 B = 0.1 T 的匀强磁场。
      求电子运动的轨迹半径。
```

**深层分析：**

```
r = mv/(qB)
  = (9.1×10⁻³¹ × 1×10⁶) / (1.6×10⁻¹⁹ × 0.1)
  = 9.1×10⁻²⁵ / (1.6×10⁻²⁰)
  = 9.1/1.6 × 10⁻⁵
  ≈ 5.7×10⁻⁵ m = 0.057 mm
```

> **方法提炼：** v ⊥ B 时，向心力 = 洛伦兹力，求半径。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "匀强磁场""B=" | 磁场强度 | B = F/(IL) |
| "半径""r=" | 圆周运动 | r = mv/(qB) |
| "周期""T=" | 运动周期 | T = 2πm/(qB) |

---

### 方法详解：带电粒子在匀强磁场中运动问题的处理

**Step 1: 判断运动类型**

```
v ⊥ B → 匀速圆周运动
v 与 B 斜交 → 螺旋线运动
```

**Step 2: v ⊥ B 情况**

```
向心力：qvB = mv²/r
求 r = mv/(qB)
求 T = 2πm/(qB)
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "匀强磁场磁感线会相交" | 误解 | 磁感线永不相交 |
| 2 | "速度越大半径越大" | 正确 | r = mv/(qB)，v 大则 r 大 |
| 3 | "周期与速度有关" | 误解 | T = 2πm/(qB)，与 v 无关 |
