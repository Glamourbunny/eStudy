# Random Walk

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 随机游走在高考中以"酒鬼走路""粒子运动""股票波动"等面貌出现——每一步随机选择方向，走 n 步后位置不确定，但位置的"分布"有规律。你的任务是认出这是随机游走，然后建立递推关系。

---

## 视角层（像什么）

### 核心隐喻：醉汉在路灯下晃悠——每一步都随机，但大量步数后位置会扩散

**Random Walk = 一个醉汉站在路灯下，每秒钟抛一次硬币——正面朝左走一步，反面朝右走一步。n 步之后，他在哪？不知道（不确定），但他在每个位置的概率可以用二项分布算出来。**

```
  一维随机游走（醉汉在一条直线上）：
  
  起点 0（路灯）
  t=0: ○
  
  每一步：50% 左走(-1)、50% 右走(+1)
  
  t=1:    ○    概率：左 0.5，右 0.5
  t=2:  ○   ○  概率：-2(0.25), 0(0.5), +2(0.25)
  t=3: ○ ○ ○ ○ 概率：-3(0.125), -1(0.375), +1(0.375), +3(0.125)
  
  规律：
  • 期望位置 = 0（向左向右的概率一样，期望不动）
  • 方差随步数增长：Var(X_n) = n·σ² ← 扩散！
  • 步数越多，醉汉"可能"离起点越远（虽然期望在原点）
```

> **一句话直觉：** Random Walk 是"每一步都随机扔硬币决定方向"的游走——你不知道 n 步后他在哪，但你知道他"平均在原位"，而"扩散范围"随步数增加。关键不是预测他走到哪，而是算他"在某个位置的概率"。

### 动态想象：花粉在水中的布朗运动

```
  花粉颗粒在水中的运动 = 三维随机游走

  水分子从四面八方随机撞击花粉
  每次撞击 = 一小步随机位移
  累积 n 次微小撞击 = 花粉的随机游走
  
  爱因斯坦 1905 年的发现：
  花粉的位移平方的平均值 = 2D·t
  （D 是扩散系数，t 是时间）
  ↓ 扩散距离 ∝ √t（不是 ∝ t！）
  
  重要：无论水分子从哪个方向来，
  花粉"最可能"回到原点还是在原点附近？
  → 一维和二维：永远会回来的（常返性）
  → 三维及以上：可能永远不回来了
```

**考场翻版：** 题目描述一个对象"每次以概率 p 向右移动一格，以概率 1-p 向左移动一格"——这是一维随机游走。问题通常是：n 步后在某位置的概率？首次到达某位置的概率？回到原点的概率？

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **期望在原点 ≠ 喜欢回原点** | "期望在 0，所以最可能在 0" | 偶数步后最可能在 0，但方差很大！n=100 时有 95% 概率在 [-20, 20] 之间——扩散范围很广 |
| **扩散 ∝ t（线性）** | "走 100 步后误差约 100" | 标准差 ∝ √n，而不是 n！100 步后标准差约 10，不是 100 |
| **"平均回来"不一定回来** | "概率对称，平均会回来" | 在三维空间，随机游走有正概率永远不回来！维度越高，"迷路"越容易 |
| **赌徒谬误** | "已经连续向左 5 步了，下一步应该是向右" | 每次独立！之前的历史不影响下一步。这就构成了马尔可夫链 |

## 联结层（连什么）

### 跨学科锚点

**1. 物理：布朗运动与扩散定律**

```
  爱因斯坦 1905 年的布朗运动论文：
  花粉的随机游走 = 大量独立随机撞击的累积
  
  扩散定律：<x²> = 2Dt
  ↓ 均方位移正比于时间
  ↓ 标准差 ∝ √t ← 对高考递推题极其重要
```

**2. 金融：股票价格的随机游走假说**

```
  股票价格的变化 = 随机游走（几何布朗运动）
  推论：明天的股价 = 今天的股价 + 随机扰动
  ↓ 你无法通过历史价格预测明天的涨跌
  ↓ 这就是"有效市场假说"的数学基础
  ↓ （当然现实中不完全成立，但模型是起点）
```

**3. 生物：动物觅食路径**

```
  许多动物的觅食路径是随机游走（Levy flight）
  在找不到食物痕迹时，随机搜索是最优策略
  ↓ 细菌趋化性：随机游走 + 浓度梯度偏差 → 偏向食物方向
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| n 步后在位置 k 的概率 | 二项分布 | P(S_n = k) = C(n, (n+k)/2)·p^{(n+k)/2}·(1-p)^{(n-k)/2} |
| 期望和方差 | E[S_n] = n(2p-1)，Var(S_n) = 4np(1-p) | 递推数列 + 期望/方差线性 |
| 首次到达某位置的概率 | 用递推或反射原理 | 递推数列 a_{n+1} = p·a_{n+2} + (1-p)·a_n |
| 吸收边界（赌徒输光问题） | 有两个吸收壁的随机游走 | 递推 + 边界条件 = 解线性方程 |

**首选工具**：递推数列（a_n = 前一步 + 随机转移）
**次选工具**：二项分布直接计算（对称 p=0.5 时最方便）

**拆包调用优先级：**
1. 问题要求"n 步后在位置 k 的概率" → 二项分布直接算
2. 问题要求"首次到达"或"被吸收" → 递推数列 + 边界条件
3. 问题要求"平均位移"或"扩散程度" → 期望/方差公式

### 历史/哲学视角：从花粉到物理学革命

```
1827 年（布朗的观察）：
  植物学家罗伯特·布朗在显微镜下看到花粉颗粒在水中"跳舞"
  他认为是"生命力"在驱动 → 错

1905 年（爱因斯坦的奇迹年）：
  爱因斯坦用"随机游走"模型解释了布朗运动
  他推导出扩散系数 D，把布朗运动和原子分子运动论联系起来
  ↓ "你看不到原子，但你可以看到原子对花粉的撞击效果"
  ↓ 这是原子存在的最直接证据之一

  随机游走从此不只是一个有趣的思想实验
  它是连接微观世界（分子）和宏观世界（花粉）的数学桥梁

  今天：随机游走是金融数学、机器学习、量子力学的核心模型
  一个简单的"扔硬币→走路"的游戏，解释了大半个宇宙的扩散现象
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | "每次随机向左或向右""每一步以概率 p…""粒子在一维/二维空间随机运动" | 看不出是随机游走，试图枚举每条路径（n 大了就炸了） | 3 秒锁定：递推 a_n 或二项分布直接算概率 |
| 🔀 **嵌套** | 随机游走在多个维度叠加："在 x 方向走一步，再在 y 方向走一步" | 两个方向的游走耦合，不会分离 | 随机游走在正交维度上独立！各维度分别算 |
| 🆕 **新定义** | "定义 S_n = X_1+X_2+…+X_n，每次 X_i 独立同分布…" | 对新定义的累积过程不理解，当成一般数列处理 | 识别：这是随机游走的标准定义 → S_n 是部分和过程 |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"随机 + 步/位移/位置"？
    → 这是随机游走
  • 确定：一维还是多维？对称还是不对称？有吸收边界吗？

Step 2：剥外壳找骨骼（4 秒）
  • 对称 p=0.5 → 无漂移 → 期望位置 = 0
  • 不对称 p≠0.5 → 有漂移 → 期望位置 = n(2p-1)
  • 画状态转移图：哪些位置可达？有无吸收壁？

Step 3：锁定 Formal 着陆点（3 秒）
  • 求位置的概率 → 二项分布或递推
  • 求期望/方差 → 公式直接
  • 求首次到达/吸收概率 → 递推 + 边界条件
  • 写下：拆包→【递推数列/二项分布】，立即切换
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 确认是随机游走，明确了维度、对称性、边界 | 建立递推：a_{n+1} = p·a_n(右来) + (1-p)·a_n(左来) |
| 嵌套题 | 各维度独立已确认 | 各维度独立建立递推，概率相乘（独立性） |
| 新定义题 | 确认 S_n = ΣX_i 是随机游走 | 用独立同分布的性质：E[S_n]=nE[X], Var(S_n)=nVar(X) |

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **扩散 ∝ n（线性）** | 认为走 100 步后位置约 ±100 | 标准差 ∝ √n。100 步后 95% 在 ±20 内，不是 ±100 |
| **赌徒谬误** | "连续向左，下一步该向右了" | 每次独立！仅 p 的依赖，历史不影响 p |
| **三维不返回** | "随机游走总会回来" | 一维/二维常返，三维及以上有正概率永不返回 |
| **忘记录不同方向概率** | p≠0.5 时还用对称公式 | 不对称随机游走的漂移 = n(2p-1)，严重偏离原点 |

## 一句话总结

> **Random Walk 是"醉汉扔硬币走路"——每次独立决定方向，n 步后的位置由二项分布描述。高考里它以"随机移动""粒子运动"登场，你的任务是 3 秒认出递推结构，用 a_{n+1} = p·a_{n-1} + (1-p)·a_{n+1}（或二项分布）算概率。**
