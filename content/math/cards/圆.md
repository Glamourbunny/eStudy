# 圆

> 圆是"到定点的距离等于定长的点的轨迹"——用方程描述圆，让圆的性质变成了代数计算。

---

## 概念层

### 问题的起源

圆是最完美的曲线。生活中的圆无处不在：
- 车轮的轮廓
- 水面的波纹
- 行星的轨道（近似）

如何用数学语言精确描述圆？解析几何的思路是：用方程描述"到定点距离为定值"这个几何条件。

> **一句话：圆是"到定点等距离"的点的轨迹，方程把这个条件翻译成了代数。**

---

### 原始定义（教材原文）

**标准方程：** (x − a)² + (y − b)² = r²（圆心 (a, b)，半径 r）

**一般方程：** x² + y² + Dx + Ey + F = 0（需满足 D² + E² − 4F > 0）

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 38-40 页

---

### 本质定义

**圆的标准方程本质是"距离公式"：**

```
(x − a)² + (y − b)² = r²
│      │      │    │
│      │      │    └─ 半径的平方
│      │      └─────── 圆心 y 坐标
│      └──────────────── 圆心 x 坐标
└─────────────────────── 距离差（勾股定理的结构）
```

**距离公式验证：**

点 P(x, y) 到圆心 C(a, b) 的距离平方：
|PC|² = (x−a)² + (y−b)²

当 |PC|² = r² 时，P 在圆上。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **圆心 (a, b)** | 圆的位置 |
| **半径 r** | 圆的大小（r > 0） |
| **距离条件** | 到圆心的距离 = r |

---

### 等价表述

| 形式 | 方程 | 说明 |
|------|------|------|
| **标准方程** | (x − a)² + (y − b)² = r² | 清晰给出圆心和半径 |
| **一般方程** | x² + y² + Dx + Ey + F = 0 | 可配方化为标准式 |
| **参数方程** | x = a + r·cosθ, y = b + r·sinθ | 用角度参数 |

## 命题层

### 核心定理

**定理 1：圆与直线的位置关系**

```
设圆 C: (x−a)² + (y−b)² = r²，直线 l: Ax + By + C = 0
圆心到直线的距离：d = |Aa + Bb + C₀| / √(A²+B²)

位置关系：
- d > r：直线在圆外（相离）
- d = r：直线与圆相切
- d < r：直线与圆相交（割线）
```

**定理 2：圆的切线方程**

```
圆 (x−a)² + (y−b)² = r² 上一点 P(x₀, y₀) 的切线：
(x₀−a)(x−a) + (y₀−b)(y−b) = r²
```

**定理 3：两圆的位置关系**

```
两圆圆心距 d = √((a₁−a₂)² + (b₁−b₂)²)
半径 r₁, r₂：

d > r₁ + r₂：外离
d = r₁ + r₂：外切
|r₁ − r₂| < d < r₁ + r₂：相交
d = |r₁ − r₂|：内切
d < |r₁ − r₂|：内含
```

---

### 典型例题

**例题：求圆的切线**

```
题目：求圆 x² + y² = 25 上一点 P(3, 4) 的切线方程。
```

**深层分析：**

```
OP⃗ = (3, 4)，这是半径，也是切线的法向量

切线方程：3x + 4y + C = 0

P 在切线上：3×3 + 4×4 + C = 0
9 + 16 + C = 0
C = −25

切线方程：3x + 4y − 25 = 0
```

> **方法提炼：** 圆上一点的切线，垂直于该点与圆心连线的半径。

---

### 方法详解：圆的问题解题流程

**Step 1: 识别问题类型**

```
求方程？位置关系？切线？
```

**Step 2: 确定圆心和半径**

```
标准方程直接读出
一般方程配方：(x + D/2)² + (y + E/2)² = (D²+E²−4F)/4
```

**Step 3: 转化为代数问题**

```
切线 → 垂直关系
相交 → 方程组
距离 → 距离公式
```

> **易错提醒：** 圆心到直线的距离公式中的分子有绝对值，别忘了。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "圆的一般方程一定表示圆" | 忽略判别式 | 需要 D²+E²−4F > 0 |
| 2 | "切线垂直于半径" | 记反 | 切线垂直于半径（径向） |
| 3 | "忘记配方" | 粗心 | 一般方程需配方才知圆心半径 |
