# 空间直角坐标系

> 空间直角坐标系是平面直角坐标系的"升级版"——增加了一根 z 轴，让三维空间中的每个点都能用三个数字 (x, y, z) 精确描述。

---

## 概念层

### 问题的起源

平面上的一点可以用 (x, y) 描述。但空间中的点呢？

想象房间里的一个位置：
- 沿着房间长度方向的距离（x）
- 沿着房间宽度方向的距离（y）
- 距离地面的高度（z）

只有同时知道这三个数，才能精确定位空间中的点。

**空间直角坐标系**就是为此而建立的——用三条两两垂直的数轴，组成一个三维坐标系。

> **一句话：空间直角坐标系用 (x, y, z) 三个数来定位空间中的任一点。**

---

### 原始定义（教材原文）

在空间中，取三条两两垂直的数轴 x 轴、y 轴、z 轴，它们的公共原点 O 称为坐标原点，这就构成了一个**空间直角坐标系**，记作 Oxyz。

通常取 x 轴、y 轴、z 轴满足**右手系**规则：
- 右手大拇指指向 x 轴正方向
- 食指指向 y 轴正方向
- 中指指向 z 轴正方向

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 3 页

---

### 本质定义

**空间直角坐标系的本质是"用三个数定位一个点"：**

```
空间中的点 P
    │
    ├── x 坐标：沿 x 轴的位移（正右负左）
    ├── y 坐标：沿 y 轴的位移（正前负后）
    └── z 坐标：沿 z 轴的位移（正上负下）
```

**坐标面的名称：**

| 坐标面 | 包含的轴 | 特点 |
|--------|---------|------|
| **xOy 平面** | x 轴、y 轴 | z = 0 |
| **yOz 平面** | y 轴、z 轴 | x = 0 |
| **zOx 平面** | z 轴、x 轴 | y = 0 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **原点 O** | 三轴交点，坐标 (0, 0, 0) |
| **坐标轴** | x 轴、y 轴、z 轴，两两垂直 |
| **右手系** | 三个轴的方向满足右手定则 |

---

### 等价表述

| 表述 | 形式 | 说明 |
|------|------|------|
| **坐标表示** | P(x, y, z) | 三维坐标 |
| **向量表示** | OP⃗ = (x, y, z) | 位置向量 |
| **坐标轴** | Oxyz | 标准记法 |

### 邻近概念网络

```
          坐标系体系
               │
    ┌──────────┼──────────┐
    │          │          │
数轴(一维)  平面坐标系  空间坐标系
   R       (二维 Oxy)   (三维 Oxyz)
               │          │
           解析几何    空间向量
```

**与平面坐标系的关系：**
- 空间坐标系 = 平面坐标系 + 一根 z 轴
- 平面坐标系是空间坐标系在 z = 0 时的截面
- 许多空间问题可以转化为平面问题来解决

---

### 概念辨析

**辨析 1：左手系 vs 右手系**

| 对比项 | 右手系 | 左手系 |
|--------|-------|--------|
| 坐标系 | x 右、y 上、z 前 | x 右、y 上、z 后 |
| 叉积方向 | 右手定则 | 左手定则 |
| 应用 | 数学、物理（通用） | 某些计算机图形学 |

高中统一使用右手系。

**辨析 2：坐标 vs 向量**

| 对比项 | 坐标 | 向量 |
|--------|------|------|
| 表示 | (x, y, z) | (x, y, z) |
| 起点 | 原点 | 任意 |
| 位置 | 固定 | 可平移 |
| 相等 | 不存在 | 起点不同但分量相同则相等 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 原点选择 | 可任选 | 通常选在几何中心或特殊点 |
| 轴的方向 | 可调整 | 但通常 x 右、y 前、z 上 |
| 右手系定则 | 必须满足 | 高中统一规定 |

## 命题层

### 核心定理

**定理 1：两点间距离公式**

```
A(x₁, y₁, z₁)，B(x₂, y₂, z₂)
|AB| = √((x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)²)
```

**定理 2：中点坐标公式**

```
M 是 A 和 B 的中点
M = ((x₁+x₂)/2, (y₁+y₂)/2, (z₁+z₂)/2)
```

**定理 3：向量与坐标的关系**

```
AB⃗ = (x₂−x₁, y₂−y₁, z₂−z₁)
```

---

### 典型例题

**例题 1：求空间中两点距离**

```
题目：已知 A(1, 2, 3)，B(4, 6, 3)，求 |AB|。
```

**深层分析：**

```
|AB| = √((4−1)² + (6−2)² + (3−3)²)
     = √(3² + 4² + 0²)
     = √(9 + 16)
     = √25
     = 5
```

> **方法提炼：** 三维空间中距离公式是二维的推广。

---

**例题 2：坐标变换**

```
题目：正方体 ABCD-A'B'C'D' 棱长为 2，顶点 A 在原点，
      AB 在 x 轴正方向，AD 在 y 轴正方向，AA' 在 z 轴正方向。
      求各顶点坐标。
```

**深层分析：**

```
A = (0, 0, 0)
B = (2, 0, 0)
C = (2, 2, 0)
D = (0, 2, 0)
A' = (0, 0, 2)
B' = (2, 0, 2)
C' = (2, 2, 2)
D' = (0, 2, 2)
```

> **方法提炼：** 建立坐标系时尽量让几何体与坐标轴对齐。

---

### 方法详解：建立空间直角坐标系的原则

**原则 1: 原点选在特殊点**

```
几何中心、对称中心、或某个顶点
```

**原则 2: 轴尽量与几何体对齐**

```
让平面与坐标平面平行
让线段与坐标轴平行或重合
```

**原则 3: 右手系**

```
x × y = z（右手定则）
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "左手系和右手系是一样的" | 混淆 | 它们是镜像关系，叉积方向相反 |
| 2 | "距离公式少一个分量" | 二维思维 | 三维距离公式 √(Δx²+Δy²+Δz²) |
| 3 | "y 和 z 轴方向搞反" | 记错 | 通常 y 轴向前，z 轴向上 |
