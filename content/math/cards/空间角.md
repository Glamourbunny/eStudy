# 空间角

> 空间角是立体几何的核心度量——用向量的夹角公式，把线面角、二面角、异面直线所成角都变成可计算的数字。

---

## 概念层

### 问题的起源

立体几何中有三种核心角度：

1. **异面直线所成角**：两条异面直线的方向所成的角
2. **直线与平面所成角**：直线与它在平面上的投影所成的角
3. **二面角**：两个半平面所成的角

这三种角都是空间中"方向差异"的度量，都可以用向量方法来计算。

> **一句话：空间角是"方向差异"的量化表达，都可以用向量夹角公式计算。**

---

### 原始定义（教材原文）

**异面直线所成角：** 将两条异面直线平移到同一点，它们所成的角（[0°, 90°]）。

**直线与平面所成角：** 直线与它在平面上的投影所成的角（[0°, 90°]）。

**二面角：** 过棱上一点，在两个半平面内分别作垂直于棱的射线，这两条射线所成的角（[0°, 180°]）。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 22-28 页

---

### 本质定义

**三种空间角的向量计算：**

| 角 | 向量 | 公式 |
|----|------|------|
| **异面直线所成角** | 两直线的方向向量 a, b | cosθ = \|a·b\|/(\|a\|\|b\|) |
| **线面角** | 直线方向向量 a 与平面法向量 n | sinθ = \|a·n\|/(\|a\|\|n\|) |
| **二面角** | 两平面的法向量 n₁, n₂ | \|cosθ\| = \|n₁·n₂\|/(\|n₁\|\|n₂\|) |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **对象** | 线线、线面、面面 |
| **向量** | 方向向量或法向量 |
| **公式** | 夹角公式 |

---

### 等价表述

| 空间角 | 几何定义 | 向量公式 |
|--------|---------|---------|
| **异面直线所成角 θ** | 方向向量的夹角 | cosθ = \|a·b\|/(\|a\|\|b\|) |
| **线面角 θ** | 直线与平面夹角的余角 | sinθ = \|a·n\|/(\|a\|\|n\|) |
| **二面角 θ** | 法向量夹角或其补角 | \|cosθ\| = \|n₁·n₂\|/(\|n₁\|\|n₂\|) |

---

### 图像表示

**异面直线所成角：**

```
      l₁
       ╲
        ╲ θ
         ╲
          ╲________l₂
```

平移 l₂ 到与 l₁ 同一点，测量夹角 θ。

**线面角：**

```
            l
           ╱
          ╱θ
         ╱
        ╱
       ╱
      α──────→ a⃗（方向向量）
      ╲
       ╲
        ↘ n⃗（法向量）
```

线面角 θ = 90° 减去 a⃗ 与 n⃗ 的夹角。

**二面角：**

```
        α₁
       ╱
      ╱ θ
     ╱
    α₁╱________α₂
```

法向量 n₁ 和 n₂ 的夹角 θ（或其补角）就是二面角。

### 概念辨析

**辨析 1：线面角 vs 异面直线所成角**

| 对比项 | 线面角 | 异面直线所成角 |
|--------|-------|---------------|
| 对象 | 直线与平面 | 两条直线 |
| 向量 | 方向向量与法向量 | 两个方向向量 |
| 公式 | sinθ = \|a·n\|/(\|a\|\|n\|) | cosθ = \|a·b\|/(\|a\|\|b\|) |
| 结果 | θ = 90° 减去 a·n 形成的角 | 直接是方向向量的夹角 |

**辨析 2：二面角的两个候选角**

两个平面的法向量夹角 θ 有两个候选：
- θ 本身
- 180° − θ

这两个角中，通常取 [0°, 180°] 范围内的较小者，或根据题意判断取哪个。

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 异面直线所成角 | θ ∈ [0°, 90°] | 规定取锐角 |
| 线面角 | θ ∈ [0°, 90°] | 不能大于 90° |
| 二面角 | θ ∈ [0°, 180°] | 可以是钝角 |

## 命题层

### 核心定理

**定理 1：异面直线所成角公式**

```
cosθ = \frac{|\vec{a}·\vec{b}|}{|\vec{a}||\vec{b}|}
```
θ ∈ [0°, 90°]（取锐角）

**定理 2：线面角公式**

```
sinθ = \frac{|\vec{a}·\vec{n}|}{|\vec{a}||\vec{n}|}
```
θ ∈ [0°, 90°]

**定理 3：二面角公式**

```
|cosθ| = \frac{|\vec{n_1}·\vec{n_2}|}{|\vec{n_1}||\vec{n_2}|}
```

> 注意：二面角可能是 θ 或 180°−θ，需根据实际图形判断。

---

### 典型例题

**例题：异面直线所成角**

```
题目：正方体 ABCD-A'B'C'D' 棱长为 1。
      求直线 BC' 与 AD' 所成的角。
```

**深层分析：**

```
BC'⃗ = (0,1,1)
AD'⃗ = (0,1,1) (从 A(0,0,0) 到 D'(0,1,1))

cosθ = |BC'⃗·AD'⃗| / (|BC'⃗||AD'⃗|)
     = |(0,1,1)·(0,1,1)| / (√2·√2)
     = |0+1+1| / 2
     = 1

所以 θ = 0°，两直线平行（实际都是垂直于 x 轴方向）
```

> **方法提炼：** 异面直线所成角先平移，再求方向向量夹角。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "异面直线所成角" | 两条异面直线 | cosθ = \|a·b\|/(\|a\|\|b\|) |
| "线面角" | 直线与平面 | sinθ = \|a·n\|/(\|a\|\|n\|) |
| "二面角" | 两个半平面 | \|cosθ\| = \|n₁·n₂\|/(\|n₁\|\|n₂\|) |

---

### 方法详解：空间角计算流程

**Step 1: 判断是什么类型的角**

```
线线？线面？面面？
```

**Step 2: 找出相关向量**

```
异面直线 → 两个方向向量
线面角 → 方向向量 + 法向量
二面角 → 两个法向量
```

**Step 3: 代入公式**

**Step 4: 取合适的范围**

```
线线角、线面角 → [0°, 90°]
二面角 → [0°, 180°]，根据图形判断
```

> **易错提醒：**
> 1. 线面角公式用 sin，不是 cos
> 2. 异面直线所成角取锐角（绝对值）
> 3. 二面角要注意是 θ 还是 180°−θ

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "线面角用 cos" | 记混公式 | 应该用 sin |
| 2 | "异面角直接是向量夹角" | 忘记绝对值 | 需要 \|cosθ\| 取锐角 |
| 3 | "二面角只有一种可能" | 忽略补角 | 有 θ 和 180°−θ 两个候选 |
| 4 | "线面角可以大于 90°" | 误解 | 线面角范围 [0°, 90°] |
