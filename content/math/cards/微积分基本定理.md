# 微积分基本定理

> 微积分基本定理是连接微分与积分的桥梁——揭示了求导与积分互为逆运算的深刻事实。

---

## 概念层

### 问题的起源

17世纪，牛顿和莱布尼茨各自独立发现了微分与积分之间的联系：
- **微分**：研究曲线的切线和瞬时变化率
- **积分**：研究曲线下的面积

两者看似毫无关系，但微积分基本定理揭示：**求曲线下面积的原函数，恰好是对应的导函数**。

这意味着：求面积只需要找到一个原函数即可，而不需要用复杂的穷竭法。

> **一句话：** 微积分基本定理告诉我们，定积分的计算可以通过求原函数来实现。

---

### 原始定义（教材原文）

如果 F(x) 是连续函数 f(x) 在区间 [a, b] 上的一个原函数，那么：

```
∫ₐᵇ f(x)dx = F(b) − F(a)
```

这个公式叫做**牛顿-莱布尼茨公式**。

> 出处：人教版《普通高中教科书·数学（B版）选择性必修第二册》第 124 页

---

### 本质定义（逻辑重构版）

**定理的完整表述：**

```
设 f(x) 在 [a, b] 上连续

定义 F(x) = ∫ₐˣ f(t)dt

则 F'(x) = f(x)  （即 F 是 f 的原函数）

于是：∫ₐᵇ f(x)dx = F(b) − F(a)
```

**核心含义：**

```
∫ₐᵇ f(x)dx = F(b) − F(a)
     ↓           ↓     ↓
   积分      原函数在   原函数在
  运算      终点值    起点值
```

**换句话说：** 求曲线 y = f(x) 在 [a, b] 区间下的面积，等于 f(x) 的任一原函数在 b 和 a 点的函数值之差。

---

### 定理的几何意义

```
         y = f(x)
         │
    f(x) │███████████████
         │███████████████
    ─────┼───────────────→ x
         a              b

面积 A = ∫ₐᵇ f(x)dx = F(b) − F(a)
       = [原函数的面积值]
```

---

### 邻近概念辨析

**辨析：不定积分 vs 定积分 —— "原函数集合" vs "具体数值"**

| 对比项 | 不定积分 | 定积分 |
|--------|---------|--------|
| 结果 | 原函数的一般表达式 + C | 一个具体的数值 |
| 形式 | ∫f(x)dx = F(x) + C | ∫ₐᵇ f(x)dx = F(b) − F(a) |
| 几何意义 | 所有的原函数 | 曲线下的有向面积 |
| 变量 | 无上下限 | 有上下限 |

## 命题层

### 核心定理

**定理：微积分基本定理（牛顿-莱布尼茨公式）**

```
若 F'(x) = f(x)，则：
∫ₐᵇ f(x)dx = F(b) − F(a)
```

**定理：原函数存在定理**

连续函数一定有原函数（即定积分可以计算）。

---

### 典型例题

**例题：计算定积分**

```
题目：计算 ∫₀¹ x² dx
```

**深层分析：**

```
分析：找 x² 的原函数

因为 (x³/3)' = x²

所以 F(x) = x³/3

∫₀¹ x² dx = F(1) − F(0)
           = 1³/3 − 0³/3
           = 1/3
```

> **方法提炼：** 定积分计算 = 找原函数 → 代上、下限 → 相减

---

**例题：利用定理证明导数公式**

```
题目：设 F(x) = ∫₀ˣ sin(t²)dt，求 F'(x)。
```

**深层分析：**

```
由微积分基本定理的逆过程：
若 F(x) = ∫ₐᵖˣ f(t)dt，则 F'(x) = f(x)

这里 a = 0，被积函数 f(t) = sin(t²)

∴ F'(x) = sin(x²)
```

## 方法层

### 方法：定积分的计算步骤

**核心思想：** 利用牛顿-莱布尼茨公式。

**标准操作流程：**

```
Step 1: 找出 f(x) 的一个原函数 F(x)
Step 2: 计算 F(b) − F(a)
Step 3: 写出答案
```

**常用原函数公式：**

| f(x) | F(x)（一个原函数） |
|------|-------------------|
| xⁿ | xⁿ⁺¹/(n+1)（n ≠ −1） |
| 1/x | ln|x| |
| eˣ | eˣ |
| sin x | −cos x |
| cos x | sin x |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "∫ₐᵇ f(x)dx = F(b) − F(a) 忘写负号" | 记错公式 | 就是 F(b) − F(a)，不是 F(a) − F(b) |
| 2 | "不定积分和定积分混淆" | 概念不清 | 不定积分结果加 C，定积分结果是数 |
| 3 | "原函数找错" | 导数公式不熟 | 求导检验是最保险的方法 |
