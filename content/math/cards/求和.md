# 求和

> 求和是计算数列所有项累加的过程——将离散的数量通过加法汇总为总量。

---

## 概念层

### 问题的起源

在日常生活中，我们经常需要"加起来"：
- 计算一学期各科成绩总和
- 统计一年12个月的收入总额
- 计算复利中本金加各期利息的总和

这些"把所有项加在一起"的操作，就是**求和**。对于等差和等比这种有规律可循的数列，我们需要专门的公式来高效求解。

> **一句话：** 求和是把数列各项按顺序累加，得到总量 Sₙ 的过程。

---

### 本质定义（逻辑重构版）

**前 n 项和的定义：**

```
Sₙ = a₁ + a₂ + a₃ + ... + aₙ

或者用求和符号：
Sₙ = Σ(k=1 to n) aₖ
```

**求和与通项的关系：**

```
aₙ = Sₙ − Sₙ₋₁    （第 n 项 = 前 n 项和 − 前 n−1 项和）
```

> 这个关系在已知 Sₙ 表达式反求通项时特别有用。

---

### 等差数列求和公式

**公式推导（倒序求和法）：**

```
Sₙ = a₁ + (a₁+d) + (a₁+2d) + ... + [a₁+(n-1)d]

写成倒序形式：
Sₙ = [a₁+(n-1)d] + [a₁+(n-2)d] + ... + a₁

两式相加：
2Sₙ = n(a₁ + [a₁+(n-1)d])
    = n(a₁ + aₙ)

∴ Sₙ = n(a₁ + aₙ)/2          （形式一）
    = n[2a₁ + (n-1)d]/2       （形式二）
```

---

### 等比数列求和公式

**公式推导（错位相减法）：**

```
Sₙ = a₁ + a₁q + a₁q² + ... + a₁q^(n-1)  ... (1)

两边乘以 q：
qSₙ = a₁q + a₁q² + ... + a₁q^(n-1) + a₁q^n  ... (2)

(1) − (2)：
Sₙ − qSₙ = a₁ − a₁q^n
(1−q)Sₙ = a₁(1 − q^n)

∴ Sₙ = a₁(1 − q^n)/(1 − q)（q ≠ 1 时）  ... 形式一
    = a₁(q^n − 1)/(q − 1)               ... 形式二（便于记忆）
```

**特殊情况：**

| 条件 | 公式 | 说明 |
|------|------|------|
| q = 1 | Sₙ = n·a₁ | 常数列直接相加 |
| q → ∞ 且 |q| < 1 | S∞ = a₁/(1 − q) | 无穷等比数列求和 |

---

### 邻近概念辨析

**辨析：倒序求和 vs 错位相减 —— "配对" vs "消项"**

| 对比项 | 倒序求和法 | 错位相减法 |
|--------|-----------|-----------|
| 适用类型 | 等差数列 | 等比数列×等差数列的混合 |
| 核心思想 | 首尾配对相加都等于 a₁ + aₙ | 通过错位相减消去中间项 |
| 典型结构 | a + (a+d) + ... + [a+(n-1)d] | 等比数列前 n 项和推导 |

---

### 适用边界

| 公式 | 限制条件 | 注意事项 |
|------|---------|---------|
| 等差求和 | 无特殊限制 | 结果是关于 n 的二次函数 |
| 等比求和 | q ≠ 1 时公式成立 | q = 1 时用 na₁ |
| 无穷等比求和 | |q| < 1 | 发散时无穷和不存在 |

## 命题层

### 核心定理

**定理：等差数列求和公式**

```
Sₙ = n(a₁ + aₙ)/2 = n[2a₁ + (n−1)d]/2
```

**定理：等比数列求和公式**

```
Sₙ = a₁(1 − q^n)/(1 − q)（q ≠ 1）
Sₙ = na₁（q = 1）
```

**定理：常见数列求和公式**

| 数列类型 | 通项 | 前 n 项和 |
|---------|------|---------|
| 自然数平方 | n² | n(n+1)(2n+1)/6 |
| 自然数立方 | n³ | [n(n+1)/2]² |

---

### 典型例题

**例题 1：等差数列求和**

```
题目：等差数列 a₁ = 1，d = 2，求前 20 项和。
```

**解题步骤：**

```
S₂₀ = 20[2×1 + (20−1)×2]/2
    = 20[2 + 38]/2
    = 20×40/2
    = 400
```

---

**例题 2：等比数列求和**

```
题目：等比数列 a₁ = 3，q = 2，求前 10 项和。
```

**解题步骤：**

```
S₁₀ = 3(1 − 2^10)/(1 − 2)
    = 3(1 − 1024)/(−1)
    = 3×(−1023)/(−1)
    = 3069
```

---

**例题 3：已知前 n 项和求通项**

```
题目：已知数列前 n 项和 Sₙ = n² + 1，求通项公式。
```

**深层分析：**

```
aₙ = Sₙ − Sₙ₋₁（n ≥ 2）
    = (n² + 1) − [(n−1)² + 1]
    = n² − (n−1)²
    = n² − (n² − 2n + 1)
    = 2n − 1

验证 a₁：a₁ = S₁ = 1² + 1 = 2
由公式 a₁ = 2×1 − 1 = 1（不吻合！）

原因：a₁ = S₁ = 2，但公式只在 n ≥ 2 时适用

所以：
a₁ = 2
aₙ = 2n − 1（n ≥ 2）
```

> **方法提炼：** 由 Sₙ 求 aₙ 时，先用 aₙ = Sₙ − Sₙ₋₁ 求出一般项，再单独处理 a₁。

## 方法层

### 方法：数列求和的策略选择

**核心思想：** 根据数列类型选择合适的求和方法。

**决策树：**

```
数列类型是什么？
    │
    ├─ 等差数列 → 直接使用等差求和公式
    │
    ├─ 等比数列 → 检查 q 是否为 1
    │             ├─ q=1 → Sₙ = na₁
    │             └─ q≠1 → Sₙ = a₁(1−q^n)/(1−q)
    │
    ├─ 等差×等比（如 n·2ⁿ）→ 错位相减法
    │
    ├─ 分数形式 → 裂项求和（如 1/[n(n+1)] = 1/n − 1/(n+1)）
    │
    └─ 其他规律 → 尝试化简或分组
```

---

### 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "等比求和忘分 q=1 和 q≠1 情况" | 记公式不完整 | 必须先判断 q 是否为 1 |
| 2 | "由 Sₙ 求 aₙ 时忘了单独处理 a₁" | 公式记忆不准确 | a₁ = S₁，需要单独验证 |
| 3 | "等差求和 Sₙ = n(a₁ + aₙ) 忘除以 2" | 梯形面积公式记错 | 必须除以 2 |
| 4 | "等比求和公式分子分母写反" | 记忆不牢 | Sₙ = a₁(1−q^n)/(1−q)，不是倒过来 |
