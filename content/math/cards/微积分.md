# 微积分

> 微积分是研究"变化"的数学——它包括微分（研究瞬时变化率）和积分（研究累积量），两者由基本定理联系起来。

---

## 概念层

### 问题的起源

微积分要解决两类问题：
1. **微分**：已知路程求速度——求瞬时变化率
2. **积分**：已知速度求路程——求累积量

牛顿和莱布尼茨发现：这两个问题是互逆的！

> **一句话：微积分是研究变化的数学，核心是微分与积分的互逆关系。**

---

### 本质定义

**微分学研究：**

```
导数：f'(x) = dy/dx = lim Δx→0 (f(x+Δx)-f(x))/Δx
几何意义：切线斜率
物理意义：瞬时变化率
```

**积分学研究：**

```
定积分：∫f(x)dx = F(b) - F(a)
几何意义：曲线下的面积
物理意义：累积量
```

---

### 核心定理

**微积分基本定理（牛顿-莱布尼茨公式）：**

```
∫f(x)dx = F(b) - F(a)
其中 F'(x) = f(x)
```

---

## 命题层

### 核心定理

**定理：导数的计算**

```
① (xⁿ)' = nxⁿ⁻¹
② (sin x)' = cos x
③ (cos x)' = -sin x
④ (eˣ)' = eˣ
⑤ (ln x)' = 1/x
```

**定理：积分的计算**

```
∫xⁿdx = xⁿ⁺¹/(n+1) + C
∫eˣdx = eˣ + C
∫sin x dx = -cos x + C
```
