# 投影

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 投影从不出现在题干里大喊"我是投影"——它总是藏在向量分解的背后、躲在物理量的计算里。这张卡片只做一件事：让你在考场 30 秒内认出"这是在问投影"，拆掉它的外壳。

---

## 视角层（像什么）

### 核心隐喻：路灯下的影子

**投影 = 路灯照在人身上，地上出现的影子**

```
       路灯（光源）
         │
         │ 光线
         ▼
    人 ═══╦═══════════════ 竖直站立（原向量 a）
          ║
          ║
══════════╩═══════════════ 地面（方向 b）
     ◄─── 影子的长度 ───►
         （投影的长度）
```

**影子的三条规律：**

| 规律 | 路灯场景 | 向量语言 |
|------|---------|---------|
| **影子比人短** | 除非太阳在正上方（人垂直于地面），否则影子总是比人短 | \|投影\| ≤ \|原向量\| |
| **影子方向由地面决定** | 影子的方向永远是"地面方向"，不管人往哪站 | 投影的方向 = b 的方向 |
| **影子长度取决于角度** | 太阳越低，影子越长；太阳越高，影子越短 | \|投影\| = \|a\|·\|cosθ\|

> **一句话直觉：** 投影就是"看一个向量在另一个方向上贡献了多少"——相当于问"你在 b 方向上的分量有多大"。

### 动态想象：推箱子

把投影想象成推箱子：

```
  人推箱子的力 F
        ╱
       ╱  ← 斜着推
      ╱
     ╱  ↙ 有效的水平推力 = F 在水平方向的投影
    ╱
═══╪═════════════════════  地面方向
   箱子 →
```

斜着推箱子时，只有"沿着地面方向的力"真正推动了箱子。"垂直于地面的力"白白浪费了。**投影就是在问：沿着 b 方向看，a 起多大作用？**

**高考里的翻版：**
- "求力 F 在位移方向上的投影"——问的是"沿着位移方向，力有多大贡献"
- "求 a 在 b 上的投影"——问的是"沿着 b 的方向看，a 有多长"
- 所有含参数的内积题，本质上都在问你"在某某方向上翻了多少倍的投影"

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **投影比原向量长** | "影子怎么可能比人长？" | **投影是带符号的长度！** 如果 a 反向于 b，投影是负的；投影的绝对值也可能大于 \|a\| 吗？不，但投影长度可能等于 \|a\|（当 a∥b 时） |
| **投影为零** | "怎么可能没影子？" | 当 a⊥b 时，你在 b 方向上的分量确实是零——就像你完全垂直于地面躺着，在地面上的影子宽度为零 |
| **投影方向** | "影子方向肯定是原向量的方向" | **错！** 投影的方向永远是 b 的方向，不是 a 的方向。影子归影子，人归人 |

## 联结层（连什么）

### 跨学科锚点

**1. 物理：力的正交分解**

```
拉力 F = 50N，与水平夹角 30°

   F 在水平方向的投影 = 50·cos30° = 43.3N（有效拉力）
   F 在竖直方向的投影 = 50·sin30° = 25N（被地面抵消）
```

物理老师每一章都在用"投影"，只是不叫这个名字——他们叫"分解"。**投影就是分解的工具。**

**2. 生活：工资条的"税后收入"**

```
税前工资 ——投影到→ 到手收入

   社保基数方向（扣款）：扣除 20%
   公积金方向（扣款）：扣除 12%
   ...

   你关心的是"到手多少"，即税前工资在"到手收入方向"上的投影
```

**3. 计算机：图像压缩（PCA 降维）**

```
高维图片数据 ——投影到→ 主成分方向（保留下最重要信息）
                           ↓
                    降维后的压缩图像
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 求 a 在 b 方向的投影 | 识别出是"路灯照人"模型 | 投影公式：\|a\|·cosθ，即 (a·b)/\|b\| |
| 向量分解题 | 拆成正交分解的投影 | 正交分解定理 |
| 含参内积题 | 拆出"投影"本质 | 数量积的几何意义 |
| 力×位移算功 | 拆出"合力在位移方向的投影" | W = F·s = \|F\|\|s\|cosθ |

**首选工具**：`投影公式`——`(a·b)/|b|`，直接等于投影长度（带符号）
**次选工具**：余弦公式回代——用 cosθ = (a·b)/(\|a\|\|b\|)，再乘 \|a\|

### 概念网络坐标

```
              ┌─────────────────┐
              │    平面向量     │  ← 舞台：定义了向量和方向
              └────────┬────────┘
                       │
                       ▼
              ┌─────────────────┐
              │      投影       │  ← 柱子：在"某个方向"上拆解向量
              │              │
              └──────┬──────────┘
                     │
         ┌───────────┼───────────┐
         │           │           │
         ▼           ▼           ▼
   ┌─────────┐ ┌─────────┐ ┌─────────┐
   │ 数量积  │ │正交分解 │ │ 物理：功│
   │a·b公式  │ │坐标运算 │ │ W=F·s  │
   └─────────┘ └─────────┘ └─────────┘
```

**拆包调用优先级：**
1. 题目出现"在…方向上的分量/投影" → 直接用路灯模型，落回投影公式
2. 题目给力+位移，问功 → 识别出是"力的投影×位移"，落回数量积
3. 题目给坐标，问"沿某方向的距离" → 正交分解，落回坐标运算

### 历史/哲学视角

**投影的概念比向量早了上千年。** 阿基米德用杠杆原理时，已经在用"力在力臂方向的投影"；中国古代的日晷，本质上就是"太阳光线在晷面上的投影"。

数学家直到 19 世纪才把"投影"从物理直觉中剥离出来，变成一个纯数学概念。**人类的直觉路径是：先有影子→再有分解力→最后才有向量投影。** 你在考场上的思维也应该遵循这个路径——先想"影子"，再写公式。

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | "在…方向上的分量""沿…方向的有效值" | 被物理词汇唬住，忘了底层就是投影公式 | 3 秒识别：分量=投影，直接落回公式 |
| 🔀 **嵌套** | 多层向量关系：力→位移→功→功率 | 层次混乱，用错方向 | 画投影链条：每层只关心"在当前方向的投影" |
| 🆕 **新定义** | （投影较少以新定义出现，更多是包装和嵌套） | — | — |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"分量""有效值""沿…方向" → 拆"物理包装"
  • 看到力/速度/加速度 + 角度 → 拆"向量嵌套"

Step 2：剥外壳找骨骼（4 秒）
  • 找出"被投影的向量"（路灯下的那个人）
  • 找出"投影方向"（地面）
  • 投影长度 = |a|·cosθ（影子的长度）

Step 3：锁定 Formal 着陆点（3 秒）
  • 写下：拆包→【投影公式/数量积】，立即切换符号语言
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 认出"分量=投影" | 用投影公式 prj_b(a) = (a·b)/\|b\| 严格书写 |
| 嵌套题 | 确认每层投影方向和长度 | 逐层计算投影，用符号代入 |

**违反此铁律的后果**：用"影子类比"直接估计答案，忽略正负号，导致计算结果反号。

---

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **忽略投影的符号** | 影子长度永远是正的，但投影向量可以是负的 | 投影数 = (a·b)/\|b\|，正负由 a·b 决定 |
| **搞混投影方向和原方向** | 误以为投影跟原向量同向 | 投影方向=b 的方向，不是 a 的方向 |
| **忘记除以 |b|** | 很多学生算完 a·b 就停，忘了投影数是 (a·b)/\|b\|，不是 a·b | 每次算投影先问："我除以 b 的长度了吗？" |

---

## 一句话总结

> **投影就是"路灯下的影子"——告诉你一个向量在另一个方向上贡献了多少。高考里它从不说"投影"，而是藏在"分量""有效值""内在某方向"这些外壳里。拆掉外壳，影子公式就一句话：a·b 除以 |b|。**
