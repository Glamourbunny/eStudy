# 椭圆

> 椭圆是"到两定点距离之和为常数的点的轨迹"——它是一种"压扁的圆"，在行星轨道、光学、工程中有广泛应用。

---

## 概念层

### 问题的起源

椭圆最初来自开普勒第一定律：行星绕太阳运动的轨道是椭圆。

数学上，椭圆是一个比圆更"丰富"的曲线：
- 圆是"到圆心等距"，椭圆是"到两焦点等距"
- 圆只有1个中心，椭圆有2个焦点
- 圆是椭圆的特例（当两焦点重合时）

> **一句话：椭圆是"到两点距离之和恒定"的点的轨迹。**

---

### 原始定义（教材原文）

平面内到两个定点 F₁、F₂ 距离之和等于常数 2a（2a > |F₁F₂|）的点的轨迹称为**椭圆**。

两个定点 F₁、F₂ 称为**焦点**。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 52-56 页

---

### 本质定义

**椭圆的标准方程：**

```
焦点在 x 轴上：x²/a² + y²/b² = 1（a > b > 0）
焦点在 y 轴上：y²/a² + x²/b² = 1（a > b > 0）
```

其中 a 是**长半轴**，b 是**短半轴**，c = √(a²−b²) 是**半焦距**。

**焦点性质：**

|F₁P| + |F₂P| = 2a（对椭圆上任意点 P）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **焦点 F₁, F₂** | 两定点，距离之和为 2a |
| **长半轴 a** | a 是长轴的一半（a > b） |
| **短半轴 b** | 与 a, c 满足 a² = b² + c² |
| **半焦距 c** | c = √(a²−b²)，焦点到中心的距离 |

---

### 等价表述

| 表述 | 方程 | 说明 |
|------|------|------|
| **标准方程（横椭圆）** | x²/a² + y²/b² = 1 | a > b，焦点在 x 轴 |
| **标准方程（纵椭圆）** | y²/a² + x²/b² = 1 | a > b，焦点在 y 轴 |
| **参数方程** | x = a·cosθ, y = b·sinθ | θ 是参数 |
| **定义式** | |PF₁| + |PF₂| = 2a | 距离和定义 |
| **焦点式** | x²/a² + y²/b² = 1，c² = a² − b² | 焦点在 (±c, 0) |

**与圆的关系：**
- 圆是椭圆的特例（a = b）
- 圆是"一个焦点的距离相等"，椭圆是"两个焦点的距离之和相等"

### 概念辨析

**辨析 1：椭圆 vs 圆**

| 对比项 | 椭圆 | 圆 |
|--------|------|------|
| 定义 | 到两焦点距离之和 = 2a | 到圆心距离 = r |
| 方程 | x²/a² + y²/b² = 1 | x² + y² = r² |
| 对称中心 | 1 个（原点） | 1 个（圆心） |
| 焦点 | 2 个（a > b 时） | 1 个（焦点在圆心处，重合） |
| 离心率 | 0 < e < 1 | e = 0 |

**辨析 2：焦点位置与 a, b 的关系**

| 类型 | 焦点位置 | 条件 | 焦点坐标 |
|------|---------|------|---------|
| 横椭圆 | 在 x 轴上 | a > b | (±c, 0) |
| 纵椭圆 | 在 y 轴上 | a > b | (0, ±c) |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 椭圆上点 P | \|PF₁\| + \|PF₂\| = 2a | 定义 |
| 焦点在椭圆内 | c < a | c² = a² − b² |
| 离心率 | e = c/a ∈ (0, 1) | 越扁越大 |

## 命题层

### 核心定理

**定理 1：椭圆的标准方程**

```
焦点在 x 轴：x²/a² + y²/b² = 1
焦点在 y 轴：y²/a² + x²/b² = 1
其中 a > b > 0，c² = a² − b²
```

**定理 2：椭圆的离心率**

```
e = c/a = √(1 − b²/a²)
0 < e < 1
e 越大，椭圆越扁；e = 0 时为圆
```

**定理 3：椭圆的切线方程**

```
椭圆 x²/a² + y²/b² = 1 上一点 P(x₀, y₀)：
xx₀/a² + yy₀/b² = 1
```

---

### 典型例题

**例题 1：求椭圆标准方程**

```
题目：已知椭圆焦点在 x 轴上，焦距为 8，长轴长为 10，求椭圆方程。
```

**深层分析：**

```
焦距 = 2c = 8 ⟹ c = 4
长轴 = 2a = 10 ⟹ a = 5
b² = a² − c² = 25 − 16 = 9 ⟹ b = 3

焦点在 x 轴，所以方程：
x²/25 + y²/9 = 1
```

> **方法提炼：** 已知焦距和长轴 → 求 c, a → 求 b → 写方程。

---

**例题 2：焦点性质的应用**

```
题目：已知椭圆 x²/25 + y²/9 = 1，求其焦距、长轴、短轴、离心率。
```

**深层分析：**

```
a² = 25, b² = 9
c² = a² − b² = 25 − 9 = 16
c = 4

焦距 = 2c = 8
长轴 = 2a = 10
短轴 = 2b = 6
离心率 e = c/a = 4/5 = 0.8
```

> **方法提炼：** 从方程中读出 a², b²，直接计算其他参数。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "到两焦点距离之和" | 椭圆定义 | 用 \|PF₁\| + \|PF₂\| = 2a |
| "焦点在 x 轴" | 横椭圆 | x²/a² + y²/b² = 1 |
| "离心率" | e = c/a | 计算 c = √(a²−b²) |
| "求椭圆方程" | 需确定 a, b | 已知条件求 a², b² |

---

### 方法详解：椭圆方程求法流程

**Step 1: 确定焦点位置**

```
焦点在 x 轴 → 横椭圆
焦点在 y 轴 → 纵椭圆
```

**Step 2: 求 a, b, c**

```
已知条件：
- 焦距 → 2c
- 长轴 → 2a
- 短轴 → 2b
- 关系：a² = b² + c²
```

**Step 3: 写出方程**

```
横椭圆：x²/a² + y²/b² = 1
纵椭圆：y²/a² + x²/b² = 1
```

> **易错提醒：** a 是长半轴，b 是短半轴，不能混淆。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "a 是长轴" | 混淆 | a 是长半轴，长轴 = 2a |
| 2 | "焦点在 x 轴就是横椭圆" | 因果倒置 | 横椭圆时焦点在 x 轴 |
| 3 | "b > a" | 记反 | 椭圆定义要求 a > b |
| 4 | "椭圆离心率可以大于 1" | 记错 | e = c/a < 1 |
