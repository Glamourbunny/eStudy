# 自感电动势

> 自感电动势是当电流变化时，导体自身产生的感应电动势——它抵抗电流的变化。

---

## 概念层

### 问题的起源

当电路中电流变化时，线圈会产生感应电动势来抵抗这种变化——这是楞次定律的体现。

自感电动势公式：
```
E = −L·(ΔI/Δt)
```

其中 L 是自感系数，ΔI/Δt 是电流变化率。

> **一句话：** 自感电动势与电流变化率成正比，方向是抵抗电流变化。

---

### 本质定义

**公式：**

```
E = −L · (dI/dt)

其中：
E —— 自感电动势（V）
L —— 自感系数（H），与线圈匝数、铁芯等有关
dI/dt —— 电流变化率（A/s）
负号 —— 表示抵抗电流变化（楞次定律）
```

---

## 命题层

### 典型例题

**例题：线圈自感电动势**

```
题目：线圈自感系数 L = 0.5H，当电流在 0.1s 内从 0 增加到 5A 时，
求自感电动势的大小。
```

**解题步骤：**

```
E = −L·(ΔI/Δt) = −0.5 × (5/0.1) = −25V

自感电动势大小为 25V，负号表示方向
```
