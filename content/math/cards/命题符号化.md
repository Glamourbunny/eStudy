# 命题符号化

> 命题符号化是用字母 p, q, r 表示命题，用 ∧, ∨, ¬ 等表示逻辑关系，把日常语言翻译成数学语言。

---

## 概念层

### 问题的起源

日常语言有时模糊，数学需要精确。

"如果天气好且不下雨，那么我们去爬山"——这句话怎么用数学表达？

把"天气好"记作 p，"不下雨"记作 q，"去爬山"记作 r：

"如果 p 且 q，那么 r" → (p ∧ q) ⇒ r

这样推理就清晰了。

> **一句话：命题符号化把自然语言翻译成精确的逻辑表达式。**

---

### 原始定义（教材原文）

用小写字母 p, q, r 等表示命题，用符号化的语言表示命题关系。

常见联结词对应关系：
- "且" → ∧
- "或" → ∨
- "非" → ¬
- "如果...那么..." → ⇒
- "当且仅当" → ⇔

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 22-25 页

---

### 本质定义

**符号系统：**

| 日常语言 | 符号 | 含义 |
|---------|------|------|
| "且" | ∧ | 合取 |
| "或" | ∨ | 析取 |
| "非" | ¬ | 否定 |
| "如果...那么..." | ⇒ | 蕴含 |
| "当且仅当" | ⇔ | 等价 |

---

### 等价表述

| 日常语言 | 符号化 |
|---------|-------|
| p 且 q | p ∧ q |
| p 或 q | p ∨ q |
| 非 p | ¬p |
| 如果 p 那么 q | p ⇒ q |
| p 当且仅当 q | p ⇔ q |

---

### 典型例题

**例题：将命题符号化**

```
题目：将以下命题符号化：
(1) 如果 x > 0 且 x ≠ 0，则 x² > 0
(2) x = 0 或 x = 1 当且仅当 x(x-1) = 0
```

**深层分析：**

(1) 令 p: x > 0，q: x ≠ 0，r: x² > 0
    → (p ∧ q) ⇒ r

(2) 令 p: x = 0 或 x = 1，q: x(x-1) = 0
    → p ⇔ q

> **方法提炼：** 先识别原子命题，用字母表示，再用联结词连接。

---

## 方法层

### 方法详解：命题符号化的步骤

**Step 1: 识别原子命题**

```
找出句子中最基本的、不可分割的命题
```

**Step 2: 用字母表示原子命题**

```
p: ...
q: ...
```

**Step 3: 用逻辑联结词连接**

```
(p ∧ q) ⇒ r
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "p ⇒ q 和 p ⇔ q 混淆" | 记错 | ⇒ 是单向蕴含，⇔ 是双向等价 |
| 2 | "自然语言的'或者'对应 ∧" | 混淆 | 自然语言'或者'通常对应 ∨ |
| 3 | "省略联结词导致歧义" | 粗心 | 需要时必须补全所有联结词 |
