# 瞬时速度与加速度

> 瞬时速度是位置的瞬时变化率，加速度是速度的瞬时变化率——它们都是导数在物理中的应用。

---

## 概念层

### 问题的起源

在变速运动中，"平均速度"不能描述某一时刻的真实运动状态。我们需要知道"此刻速度是多少"。

如果物体的位置随时间变化关系是 s = s(t)，那么：
- 瞬时速度 v(t) = s'(t)
- 瞬时加速度 a(t) = v'(t) = s''(t)

这就是导数在运动学中的核心应用。

> **一句话：** 瞬时速度是位置对时间的导数，瞬时加速度是速度对时间的导数。

---

### 本质定义（逻辑重构版）

**基本公式：**

```
位置：s = s(t)

瞬时速度：v(t) = ds/dt = s'(t)

瞬时加速度：a(t) = dv/dt = d²s/dt² = s''(t)
```

**物理意义的对应：**

| 数学 | 物理 | 单位 |
|------|------|------|
| s'(t) | 速度 | m/s |
| s''(t) | 加速度 | m/s² |
| 速度的积分 | 位移 | ∫v(t)dt = s(t) + C |
| 加速度的积分 | 速度 | ∫a(t)dt = v(t) + C |

---

### 典型运动模型

**匀加速直线运动：**

```
a(t) = a₀（常数）

v(t) = v₀ + a₀·t
s(t) = s₀ + v₀·t + (1/2)·a₀·t²
```

**自由落体（a = g）：**

```
v(t) = g·t
s(t) = (1/2)·g·t²
```

## 命题层

### 典型例题

**例题：求瞬时速度和加速度**

```
题目：物体做直线运动，位置 s(t) = t³ − 6t² + 9t + 1（单位：m）。求：
(1) 瞬时速度 v(t)
(2) 瞬时加速度 a(t)
(3) t = 2s 时的速度和加速度
```

**深层分析：**

```
(1) v(t) = s'(t) = 3t² − 12t + 9
(2) a(t) = v'(t) = s''(t) = 6t − 12
(3) t = 2 时：
    v(2) = 3·4 − 12·2 + 9 = 12 − 24 + 9 = −3 m/s
    a(2) = 6·2 − 12 = 0 m/s²
```

## 方法层

### 方法：运动学问题求解流程

**核心思想：** 利用导数建立数学模型。

**标准操作流程：**

```
Step 1: 建立位置函数 s(t)
Step 2: 求一阶导得速度 v(t) = s'(t)
Step 3: 求二阶导得加速度 a(t) = s''(t)
Step 4: 代入具体时刻求值
Step 5: 如需位移，积分 s(t)
```
