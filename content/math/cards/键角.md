# 键角

> 键角是三个成键原子所形成的角度——它决定了分子的几何形状，是分子结构的核心参数。

---

## 概念层

### 问题的起源

分子不是"一团乱麻"，而是有确定形状的。

水分子的形状是"弯曲的"（键角约 104.5°），而不是直的。
甲烷是正四面体（键角 109.5°）。
二氧化碳是直线形（键角 180°）。

是什么决定了分子的形状？——**中心原子的价层电子对互斥**。

键角是描述分子几何形状的核心参数。

> **一句话：键角是三个成键原子的原子核连线所成的角，它决定了分子的空间形状。**

---

### 原始定义（教材原文）

分子中同一原子形成的两个化学键之间的夹角称为**键角**。

> 出处：人教版《普通高中教科书·化学（选择性必修）》

---

### 本质定义

**键角的本质是原子轨道重叠方向的"平衡角度"：**

```
       B
       ↑
      ╱  ← 键角 θ
     ╱
    A ——— C
```

A 是中心原子，B 和 C 是配位原子，∠BAC = θ 就是键角。

**常见分子的键角与形状：**

| 分子 | 键角 | 几何形状 |
|------|------|---------|
| CO₂ | 180° | 直线形 |
| BF₃ | 120° | 平面三角形 |
| CH₄ | 109.5° | 正四面体形 |
| H₂O | 104.5° | V 形（折线形） |
| NH₃ | 107° | 三角锥形 |

---

## 命题层

### 核心定理

**定理 1：理想键角与杂化类型**

```
sp 杂化：180°（直线形）
sp² 杂化：120°（平面三角形）
sp³ 杂化：109.5°（正四面体）
```

**定理 2：孤对电子的压缩效应**

```
孤对电子越多，键角越小：
NH₃（107°）< CH₄（109.5°）
H₂O（104.5°）< NH₃（107°）
```

**定理 3：VSEPR 模型**

```
价层电子对互斥 → 分子采取使电子对距离最大的几何构型
→ 决定了键角
```

---

### 典型例题

**例题：预测 H₂O 的键角**

```
题目：用 VSEPR 理论预测水分子的键角。
```

**深层分析：**

```
Step 1: 确定中心原子 O 的价电子数
O 有 6 个价电子

Step 2: 计算价层电子对数
H₂O 中，O 与两个 H 形成两个键，还剩两对孤对电子
总电子对数 = 4（2 键 + 2 孤对）

Step 3: VSEPR 构型
4 对电子 → 四面体形

Step 4: 考虑孤对电子的压缩
理想 sp³ 键角 109.5°，但两对孤对电子的排斥力更大，
把 H-O-H 键角压缩到 104.5°

所以 H₂O 的键角约为 104.5°
```

> **方法提炼：** VSEPR 模型 → 电子对排布 → 孤对压缩 → 最终键角。

---

### 方法详解：预测键角的方法

**Step 1: 确定中心原子的价层电子对数**

```
价层电子对数 = 键数 + 孤对电子数
```

**Step 2: 判断 VSEPR 构型**

```
2 对 → 直线形
3 对 → 平面三角形
4 对 → 四面体
```

**Step 3: 考虑孤对电子的压缩效应**

```
孤对电子越多，实际键角越小于理想键角
```
