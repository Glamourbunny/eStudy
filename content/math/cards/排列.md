# 排列

> 排列是"从 n 个东西里选 k 个排成一排"——记作 Pₙᵏ，等于 n!/(n−k)!。

---

## 概念层

### 问题的起源

从 5 个人中选 2 个人排成一排，有多少种排法？

这与组合不同——排列需要考虑**顺序**。

5 个人选 2 人排成一排：
- 第1个位置：5 个人都可以
- 第2个位置：剩下 4 个人

所以 5 × 4 = 20 种。

这就是排列数 P₅² = 5!/(5−2)! = 5!/3! = 20。

> **一句话：排列是"有顺序"的选取——Pₙᵏ = n!/(n−k)!。**

---

### 原始定义（教材原文）

从 n 个不同元素中取出 k（k ≤ n）个元素，按照**一定的顺序**排成一排，叫做从 n 个不同元素中取出 k 个元素的**排列**，记作 Pₙᵏ 或 Aₙᵏ。

排列数公式：
```
Pₙᵏ = n(n−1)(n−2)...(n−k+1) = n!/(n−k)!
```

> 出处：人教版《普通高中教科书·数学（A版）选择性必修三》第 2-4 页

---

### 本质定义

**排列 vs 组合的区别：**

| 对比项 | 排列（有序） | 组合（无序） |
|--------|-------------|-------------|
| 是否考虑顺序 | 考虑 | 不考虑 |
| 公式 | Pₙᵏ = n!/(n−k)! | Cₙᵏ = n!/(k!(n−k)!) |
| 例子 | 5人排2队 | 5人选2人 |

**关系：** Pₙᵏ = Cₙᵏ × k!（先选出来，再排序）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **n 个元素** | 不同的候选元素 |
| **选 k 个** | 不放回抽取 |
| **排序** | 考虑顺序 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **符号** | Pₙᵏ 或 Aₙᵏ 或 ₙPₖ |
| **阶乘形式** | n!/(n−k)! |
| **连乘形式** | n(n−1)...(n−k+1) |

---

### 典型例子

**例：P₅² = 5 × 4 = 20**

5 个不同元素选 2 个排成一排，排列数是 20。

**例：全排列 Pₙⁿ = n!**

n 个元素全部排成一排，有 n! 种。

## 命题层

### 核心定理

**定理 1：排列数公式**

```
Pₙᵏ = n!/(n−k)! = n(n−1)...(n−k+1)
```

**定理 2：全排列**

```
Pₙⁿ = n!（n 个元素全排列）
```

**定理 3：排列与组合的关系**

```
Pₙᵏ = Cₙᵏ × k!
```

---

### 典型例题

**例题：计算 P₅³**

```
题目：计算 P₅³ = ?
```

**深层分析：**

```
P₅³ = 5!/(5−3)! = 5!/2! = 120/2 = 60

或者：P₅³ = 5 × 4 × 3 = 60
```

> **方法提炼：** Pₙᵏ = n!/(n−k)!，或直接连乘。

---

### 方法详解：排列问题解题流程

**Step 1: 判断是排列还是组合**

```
"排成一排""排队""排序" → 排列
"选出来""分组" → 组合
```

**Step 2: 确定 n 和 k**

**Step 3: 用公式 Pₙᵏ = n!/(n−k)! 计算**

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "P₅² = 5×4/2 = 10" | 与组合混淆 | P₅² = 5×4 = 20，不除以 2! |
| 2 | "排列和组合混淆" | 混淆 | 排列有序，组合无序 |
| 3 | "Pₙ⁰ = 1" | 正确 | 0 个排列是 1 种（空排列） |
