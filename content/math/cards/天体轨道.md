# 天体轨道

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 天体运动题的难点不是公式复杂，而是场景抽象——你脑子里没有一颗"会动"的行星。这张卡片帮你建立天体运动的动态直觉。

---

## 视角层（像什么）

### 核心隐喻：宇宙圆舞曲——每颗行星都在引力场中跳着一支永不停止的舞蹈

**天体轨道 = 两个人手拉手在天上旋转——有拉力（引力）、有惯性（切向速度）、有固定的舞步（轨道方程）**

```
          太阳（S）                    行星（P）
             ●                          ○
            /|\  ═══ 引力 F ═══       /|\
           / | \                      / | \
              ↕                          ↕
          质量 M（大）                质量 m（小）


  行星的运动 = 两种力量的平衡：

  1. 引力（向心力）：太阳用力把行星拉向自己
     F = GMm/r²  ← 距离越近，拉力越大

  2. 惯性（切向速度）：行星想沿着切线方向飞出去
     如果没有太阳拉，行星就会直线飞走

  两者的平衡 → 椭（(椭)圆轨道
  引力 > 所需向心力 → 行星"掉"向太阳，轨道变小
  引力 = 所需向心力 → 完美圆形轨道
  引力 < 所需向心力 → 行星"甩"出去，轨道变大
```

**圆舞曲思维的三条铁律：**

| 铁律 | 圆舞曲场景 | 违反的后果 |
|------|-----------|-----------|
| **引力提供向心力**：别把两个力分着算 | 舞伴之间的拉力 = 旋转所需的向心力 | 把引力和向心力当成两个独立力，列式中重复计算 |
| **轨道形状由能量决定**：总能量决定椭圆/抛物线/双曲线 | 舞伴用力一甩就飞出椭圆变抛物线 | 以为轨道总是椭圆 |
| **角动量守恒 ≠ 速度守恒**：靠近太阳时转得快，远离时转得慢 | 舞伴靠近时旋转加快，远离时变慢 | 以为行星做匀速圆周运动 |

> **一句话直觉：** 天体轨道是引力和惯性的拔河——太阳拉着行星往里拽，行星的惯性要往外飞。两者的平衡态是椭圆轨道，拉开力大于所需向心力就飞出去。

### 动态想象：开普勒第二定律——面积定律

```
  太阳在椭圆的一个焦点上，行星绕椭圆运动。

  行星靠近太阳（近日点）：距离近、引力大 → 转得快
  行星远离太阳（远日点）：距离远、引力小 → 转得慢

  开普勒第二定律：
  行星与太阳的连线在相同时间内扫过相同面积

  近日点扫过的扇形：弧长×半径/2，半径小但弧长大（速度快）
  远日点扫过的扇形：弧长×半径/2，半径大但弧长小（速度慢）
  → 面积相等！

  直觉：行星像花样滑冰选手——
  手臂收回（离太阳近）→ 转得快
  手臂展开（离太阳远）→ 转得慢
```

### 反直觉警示

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **轨道是圆的** | "行星绕太阳转，应该是圆" | 开普勒第一定律：轨道是椭圆，太阳在焦点上，不是中心 |
| **速度和距离无关** | "同一个行星的速度应该恒定" | 近日点速度最大，远日点速度最小——越近越快 |
| **周期和半径成比例** | "轨道大当然周期长，成比例的" | 开普勒第三定律：T²/a³=常数——不是线性关系，是 3/2 次方 |

## 联结层（连什么）

### 跨学科锚点

**1. 体育运动：链球运动员**
```
链球运动员抓着链子的一端，链球在另一端旋转
→ 如果松手，球沿切线方向飞出（惯性）
→ 如果拉紧，球继续圆周运动（向心力）
→ 运动员 = 太阳，链球 = 行星，链子 = 引力
```

**2. 计算机图形学：粒子系统模拟轨道**
```
游戏引擎中模拟行星轨道：
  每帧计算：位置 → 引力 → 加速度 → 速度 → 新位置
  这不是"算出椭圆方程再画"，而是"每帧更新一次状态"
  这种逐帧模拟就是数值仿真！
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 拆包收益 |
|:---|:---|:---|
| 🎭 **包装** | "某天体绕另一天体运动"穿上了太空探索/卫星的外衣 | 去掉情境，只留质量和距离，用万有引力公式 |
| 🔀 **嵌套** | 椭圆轨道+速度+能量+周期多变量 | 画椭圆标焦点+远近点，逐条列公式 |
| 🆕 **新定义** | "定义轨道的偏心率为…" | 翻译新术语：离心率 e=0 是圆，0<e<1 是椭圆 |

### 拆包方向标（10 秒流程）

```
Step 1：确定模型（3 秒）
  • 圆轨道 → 引力=向心力，等号联立
  • 椭圆轨道 → 开普勒三定律
  • 变轨问题 → 能量守恒+角动量守恒

Step 2：画轨道示意图（4 秒）
  • 标出太阳位置（焦点，不是中心！）
  • 标近日点和远日点
  • 在要求的点上画受力图

Step 3：锁定 Formal 着陆点（3 秒）
  • 选公式：引力提供向心力 / 开普勒定律 / 机械能守恒
```

### 直觉陷阱清单

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **混淆焦点和中心** | 算距离时代入"椭圆中心到行星的距离" | 太阳在焦点！距离用椭圆定义或直接给定 |
| **引力不等于向心力** | 椭圆轨道非匀速时错误使用 GMm/r²=mv²/r | 椭圆轨道此式只在近日点和远日点成立 |
| **忘开普勒第三定律** | 比较不同行星的周期时手动算 | T²∝a³，记住这个比例关系 |

## 一句话总结

> **天体轨道 = 引力（往里拉）和惯性（往外飞）的拔河。圆轨道用"引力=向心力"，椭圆轨道用开普勒三定律。关键图示：太阳在焦点不是中心，近日点快远日点慢，面积速率恒定。**
