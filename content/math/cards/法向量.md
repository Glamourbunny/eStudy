# 法向量

> 法向量是垂直于一个平面的向量——有了它，直线与平面的夹角、点到平面的距离，都变成了套公式的计算题。

---

## 概念层

### 问题的起源

在空间中，如何判断一条直线是否与平面垂直？如何求点到平面的距离？

传统方法需要复杂的几何证明，而**法向量**提供了一个简单有力的工具：

法向量 n⃗ 垂直于平面 α 内的所有直线，因此：
- 直线垂直于平面 ⟺ 直线的方向向量与平面的法向量平行
- 点到平面的距离 ⟺ 点到平面的有向距离在法向量方向上的投影

> **一句话：法向量是平面的"指向标"——垂直于平面内所有方向。**

---

### 原始定义（教材原文）

垂直于平面 α 的向量称为平面 α 的**法向量**。若 n⃗ 是平面 α 的法向量，则 n⃗ ⊥ α，即 n⃗ 与平面 α 内的任意直线都垂直。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 15 页

---

### 本质定义

**法向量的本质是"垂直特性的人格化"：**

```
平面 α
   │
   │ 无数条直线（方向各异）
   │
   │ n⃗（法向量）垂直于所有这些直线
   │
   ▼
n⃗ 是平面的"法线"——代表平面的朝向
```

**法向量不唯一：** 任何一个垂直于平面的向量都是法向量，它们互为数乘关系。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **垂直性** | n⃗ ⊥ α（与平面内所有直线垂直） |
| **方向性** | n⃗ 指向平面"哪一边" |
| **不唯一性** | k·n⃗（k≠0）仍是同一平面的法向量 |

---

### 等价表述

| 表述 | 说明 |
|------|------|
| **定义** | 垂直于平面的向量 |
| **性质** | n⃗·a⃗ = 0（a⃗ 是平面内任意向量） |
| **计算** | 用平面内两不共线向量的叉积得到 |

### 概念辨析

**辨析：法向量 vs 方向向量**

| 对比项 | 法向量 | 方向向量 |
|--------|-------|---------|
| 定义 | 垂直于平面的向量 | 沿直线方向的向量 |
| 作用对象 | 平面 | 直线 |
| 记号 | n⃗ | a⃗ |
| 应用 | 线面角、点面距离、面面角 | 线线角 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| n⃗ 是 α 的法向量 | ✓ | n⃗ ⊥ α |
| k·n⃗（k≠0） | ✓ | 仍是 α 的法向量 |
| 单位法向量 n̂ = n/|n| | ✓ | 模为 1 的法向量 |

## 命题层

### 核心定理

**定理：法向量的求法**

已知平面 α 上两点 A、B 和它们的位置向量，以及平面外一点 C，则：

**方法 1：利用平面内两向量垂直**

```
在平面内取两个不共线的向量 AB⃗ 和 AC⃗
则法向量 n⃗ 满足：n⃗·AB⃗ = 0 且 n⃗·AC⃗ = 0
求解得到 n⃗ = AB⃗ × AC⃗（叉积）
```

**方法 2：待定系数法**

```
设 n⃗ = (a, b, c)
利用 n⃗·AB⃗ = 0 和 n⃗·AC⃗ = 0 列方程
解得一个方向（如取 a = 1，解得 b, c）
```

---

### 典型例题

**例题：求平面 ABC 的法向量**

```
题目：已知 A(1,0,0)，B(0,1,0)，C(0,0,1)，求平面 ABC 的法向量。
```

**深层分析：**

```
AB⃗ = (−1,1,0)
AC⃗ = (−1,0,1)

设 n⃗ = (a, b, c)
n⃗·AB⃗ = −a + b = 0  ⟺  b = a
n⃗·AC⃗ = −a + c = 0  ⟺  c = a

取 a = 1，则 b = 1，c = 1
所以 n⃗ = (1, 1, 1)

验证：n⃗·AB⃗ = 1×(−1)+1×1+1×0 = 0 ✓
      n⃗·AC⃗ = 1×(−1)+1×0+1×1 = 0 ✓
```

> **方法提炼：** 设方程组，解得一个非零解，即为法向量。

| "线面角""二面角" | 需用法向量 | 套公式 |
| "点面距离" | 需用法向量 | d = \|n·PQ\|/\|n\| |

---

### 方法详解：法向量求法流程

**Step 1: 在平面内找两个不共线的向量**

```
如 AB⃗ 和 AC⃗
```

**Step 2: 设 n⃗ = (a, b, c)**

**Step 3: 列方程组**

```
n⃗·AB⃗ = 0
n⃗·AC⃗ = 0
```

**Step 4: 解方程组，令一个未知数为 1（或某个方便的值），求另外两个**

> **易错提醒：** 求出的法向量可以任意缩放，k·n⃗（k≠0）都是法向量。

| 编号 | 常见误区 | 错误根源 | 正确理解 |
