# 牛顿迭代

> 牛顿迭代是用切线逼近求方程根的算法——每一步用切线"切"一下，步步逼近真解。

---

## 概念层

### 问题的起源

很多方程没有求根公式，如 f(x) = x³ − 2x − 5 = 0。我们需要数值方法求近似根。

牛顿提出：用切线来逼近零点。每次迭代时，在当前点做切线，切线与 x 轴的交点作为下一个近似点。

> **一句话：** 牛顿迭代的核心思想是"以直代曲"——用切线近似曲线，找切线与 x 轴的交点作为新的近似。

---

### 本质定义（逻辑重构版）

**迭代公式：**

```
xₙ₊₁ = xₙ − f(xₙ)/f'(xₙ)
```

**几何解释：**

```
在点 (xₙ, f(xₙ)) 做曲线 y = f(x) 的切线
切线方程：y − f(xₙ) = f'(xₙ)(x − xₙ)
令 y = 0（x 轴），解得 x = xₙ − f(xₙ)/f'(xₙ)
这个 x 就是下一个迭代点 xₙ₊₁
```

---

### 迭代过程示意

```
         y
         │
    f(x₁)│●
         │ ╲
         │   ╲ ←切线
    f(x₀)│    ╲●
         │      ╲
      ───┼───────╲────────→ x
         x₀      x₁
```

---

## 方法层

### 方法：牛顿迭代标准流程

**标准操作流程：**

```
Step 1: 选取初始值 x₀
Step 2: 计算 f(xₙ) 和 f'(xₙ)
Step 3: 用公式 xₙ₊₁ = xₙ − f(xₙ)/f'(xₙ) 计算下一个近似
Step 4: 判断收敛性（|xₙ₊₁ − xₙ| < ε）
Step 5: 重复直到满足精度要求
```

---

### 典型例题

**例题：用牛顿迭代求 √2**

```
题目：用牛顿迭代法求 √2 的近似值。
```

**深层分析：**

```
√2 是方程 x² − 2 = 0 的正根
所以 f(x) = x² − 2，f'(x) = 2x

迭代公式：xₙ₊₁ = xₙ − (xₙ² − 2)/(2xₙ) = (xₙ + 2/xₙ)/2

取 x₀ = 1：
x₁ = (1 + 2)/2 = 1.5
x₂ = (1.5 + 2/1.5)/2 = (1.5 + 1.333)/2 = 1.4167
x₃ = (1.4167 + 2/1.4167)/2 ≈ 1.4142

答案：√2 ≈ 1.4142（已精确到 4 位小数）
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "初始值选不好导致不收敛" | 初始值选择不当 | 需要选在根附近且 f'(x) ≠ 0 |
| 2 | "迭代公式记错" | 粗心 | 正确公式是 xₙ₊₁ = xₙ − f(xₙ)/f'(xₙ) |
| 3 | "不知道什么时候停止" | 迭代终止条件 | 设定精度 ε，当 |xₙ₊₁ − xₙ| < ε 时停止 |
