# 哈迪-温伯格定律

> 哈迪-温伯格定律描述了理想群体中基因频率和基因型频率的关系——p² : 2pq : q² 是遗传平衡时的比例。

---

## 概念层

### 问题的起源

一个群体的基因频率会世代变化吗？

哈迪和温伯格发现：如果满足某些条件（足够大的种群、随机交配、无选择、无突变、无迁移），基因频率和基因型频率会**代代保持不变**。

这就是**遗传平衡**。

> **一句话：哈迪-温伯格定律指出，在理想条件下群体基因频率代代保持平衡。**

---

### 本质定义

**哈迪-温伯格定律的数学表述：**

```
基因频率：A = p, a = q
基因型频率：AA = p², Aa = 2pq, aa = q²
```

**平衡条件：**
1. 大种群（避免遗传漂变）
2. 随机交配（避免选择交配）
3. 无选择（各种基因型生存繁殖能力相同）
4. 无突变（基因不变）
5. 无迁移（无个体流入流出）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **基因频率 p, q** | p + q = 1 |
| **基因型频率** | p² + 2pq + q² = 1 |
| **平衡条件** | 五个理想条件 |

---

## 命题层

### 核心定理

**定理：哈迪-温伯格定律**

```
(p + q)² = p² + 2pq + q² = 1
```

**定理：平衡世代的基因型频率**

```
若 p 和 q 是上一代的基因频率
则在平衡群体中：
AA 频率 = p²
Aa 频率 = 2pq
aa 频率 = q²
```
