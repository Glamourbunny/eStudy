# 基本不等式

> 基本不等式是"平方和大于等于两倍积"—— a² + b² ≥ 2ab，当且仅当 a = b 时取等。

---

## 概念层

### 问题的起源

(a − b)² ≥ 0 是显然的（任何实数的平方非负）。

展开：(a − b)² = a² − 2ab + b² ≥ 0

移项：a² + b² ≥ 2ab

这就是**基本不等式**。

它告诉我们：两个实数的平方和至少是两倍积。

> **一句话：a² + b² ≥ 2ab，是从 (a − b)² ≥ 0 直接推导出来的。**

---

### 原始定义（教材原文）

对于任意实数 a, b，有 a² + b² ≥ 2ab，当且仅当 a = b 时，等号成立。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册）第 28-30 页

---

### 本质定义

**基本不等式的几何意义：**

```
(a − b)² ≥ 0
   ↓
a² − 2ab + b² ≥ 0
   ↓
a² + b² ≥ 2ab
```

等号成立当且仅当 a − b = 0，即 a = b。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **a, b 为实数** | 不等式对任意实数成立 |
| **不等号** | ≥ |
| **取等条件** | a = b |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **标准形式** | a² + b² ≥ 2ab |
| **等价形式** | (a − b)² ≥ 0 |
| **变式** | ab ≤ (a² + b²)/2 |

---

### 典型应用

**求 ab 的最大值：**

当 a + b 固定时，ab ≤ (a² + b²)/2 = (a + b)²/4

所以当 a = b 时，ab 取最大值。

---

### 概念辨析

**辨析：基本不等式 vs 均值不等式**

| 对比项 | 基本不等式 | 均值不等式 |
|--------|-----------|-----------|
| 形式 | a² + b² ≥ 2ab | (a + b)/2 ≥ √(ab) |
| 联系 | 两者等价 | 基本不等式的变形 |
| 适用范围 | a, b ∈ R | a, b ≥ 0 |

基本不等式是均值不等式的更一般形式。

---

## 命题层

### 核心定理

**定理：基本不等式**

```
a² + b² ≥ 2ab
等号成立 ⟺ a = b
```

---

### 典型例题

**例题：求 ab 的最大值**

```
题目：已知 a + b = 10，a ≥ 0, b ≥ 0，求 ab 的最大值。
```

**深层分析：**

```
ab ≤ (a² + b²)/2
    = ((a + b)² − 2ab)/2
    = (100 − 2ab)/2
    = 50 − ab

⟹ ab ≤ 50 − ab
⟹ 2ab ≤ 50
⟹ ab ≤ 25
```

当 a = b = 5 时取等。

> **方法提炼：** 直接用 a² + b² ≥ 2ab，或用均值不等式。

| 编号 | 常见误区 | 错误根源 | 正确理解 |
