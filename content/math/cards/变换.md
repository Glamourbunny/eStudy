# 变换

> 变换描述的是函数图像随参数变化而发生的几何变化——平移、伸缩、翻折。

---

## 概念层

### 问题的起源

当我们研究函数 y = sin x 时，它的图像是标准正弦曲线。但如果把它改成 y = sin 2x 或 y = sin x + 1，图像会怎么变？

这引出一个核心问题：**函数解析式的变化，如何反映在图像的几何变化上？**

理解变换规律，可以：
1. 从已知函数的图像，快速画出相关函数的图像
2. 理解复杂函数是如何由简单函数"改造"而来的
3. 在物理中理解振动、波动、信号等实际现象

> **一句话：** 变换揭示"解析式变，图像怎么变"的对应规律。

---

### 本质定义（逻辑重构版）

**变换的基本类型：**

```
变换类型：
│
├─ 平移变换：图像位置整体移动
│   ├─ 水平平移：y = f(x − a)  ← 右移 a 个单位
│   └─ 垂直平移：y = f(x) + b  ← 上移 b 个单位
│
├─ 伸缩变换：图像形状拉伸或压缩
│   ├─ 水平伸缩：y = f(kx)    ← 横坐标缩为 1/k
│   └─ 垂直伸缩：y = Af(x)    ← 纵坐标扩为 A 倍
│
└─ 对称变换：图像关于某轴/点对称
    ├─ 关于 x 轴：y = −f(x)
    ├─ 关于 y 轴：y = f(−x)
    └─ 关于原点：y = −f(−x)
```

---

### 变换速查表

| 变换类型 | 变换方式 | 对图像的影响 | 操作顺序 |
|---------|---------|-------------|---------|
| 左平移 a | y = f(x + a) | 整个图像左移 a | 先 |
| 右平移 a | y = f(x − a) | 整个图像右移 a | 先 |
| 上移 b | y = f(x) + b | 整个图像上移 b | 后 |
| 下移 b | y = f(x) − b | 整个图像下移 b | 后 |
| 横坐标缩 k 倍 | y = f(kx) | 图像压缩为 1/k | 在平移之后 |
| 纵坐标扩 A 倍 | y = Af(x) | 图像拉神为 A 倍 | 在平移之前 |

---

### 邻近概念辨析

**辨析：平移变换 vs 伸缩变换 —— "位置" vs "形状"**

| 对比项 | 平移变换 | 伸缩变换 |
|--------|---------|---------|
| 变化内容 | 图像位置 | 图像形状（大小） |
| 参数位置 | 加减（在 x 上加减，或结果加减） | 乘除（乘以系数） |
| 图像效果 | 形状不变，位置移动 | 形状改变（压扁或拉长） |

---

### 变换的复合顺序

**重要原则：先伸缩后平移 vs 先平移后伸缩，结果不同！**

```
y = f(2x + 1) 的变换顺序（先平移再伸缩错误！）

正确理解：
y = f(2x + 1) = f[2(x + 1/2)]

分解为：
1. 先把 x 变成 x + 1/2：左移 1/2
2. 再把 x 变成 2x：横坐标缩为 1/2

所以：y = f(2x + 1) 是先将图像左移 1/2，再横坐标压缩一半
```

> **记忆口诀：** "有伸缩，先伸缩；有平移，看清楚是 x 的平移还是整体的平移"。

---

### 适用边界

| 要点 | 说明 |
|------|------|
| 变换顺序很重要 | y = f(ax + b) 要先提取系数 |
| 多个变换叠加 | 从最内层往外依次变换 |
| 定义域变化 | 伸缩/平移可能改变定义域范围 |

## 命题层

### 典型例题

**例题 1：y = sin x 到 y = sin 2x 的变换**

```
题目：函数 y = sin 2x 的图像如何由 y = sin x 变换得到？
```

**深层分析：**

```
分析：y = sin 2x = f(2x)，其中 f(x) = sin x

变换：横坐标压缩为原来的 1/2
（因为当 f(x) 的图像上点 (x₀, y₀) 变成 f(2x) 时，
需要 x = x₀/2 才能得到相同的 y 值）

答案：将 y = sin x 的图像上每点的横坐标缩短为原来的 1/2
```

> **方法提炼：** y = f(kx)（k > 1）将横坐标压缩为 1/k；y = f(x/k)（k > 1）将横坐标扩大 k 倍。

---

**例题 2：y = sin x 到 y = sin(x − π/3) + 1 的变换**

```
题目：如何由 y = sin x 得到 y = sin(x − π/3) + 1？
```

**解题步骤：**

```
分析：
y = sin(x − π/3) + 1 = f(x − π/3) + 1，其中 f(x) = sin x

变换顺序：
1. 水平平移：右移 π/3 → y = sin(x − π/3)
2. 垂直平移：上移 1 → y = sin(x − π/3) + 1

答案：先将图像右移 π/3，再上移 1
```

> **方法提炼：** 平移变换中，x − a 表示右移 a，x + a 表示左移 a；结果 + b 表示上移 b，−b 表示下移 b。

---

**例题 3：综合变换**

```
题目：由 y = f(x) 的图像，如何得到 y = 2f(x − 1) + 3 的图像？
```

**深层分析：**

```
分析：y = 2f(x − 1) + 3

分解步骤：
1. f(x − 1)：右移 1
2. 2f(x − 1)：纵坐标扩大 2 倍（图像上各点纵坐标乘以 2）
3. 2f(x − 1) + 3：上移 3

答案：先将图像右移 1，再纵坐标扩大 2 倍，最后上移 3
```

> **方法提炼：** 多个变换叠加时，按从内到外的顺序执行：先 x 的变换（平移/伸缩），再外面的系数，最后加减常数。

## 方法层

### 方法 1：变换分析法

**核心思想：** 从解析式结构出发，识别变换类型和顺序。

**标准操作流程：**

```
Step 1: 将解析式写成标准形式 f(ax + b) + c
Step 2: 提取内层变换：令 u = ax + b → x = (u − b)/a
Step 3: 按顺序写出变换步骤（从 x 开始）：
        - 先处理 ax（伸缩）
        - 再处理 +b/−b（平移）
Step 4: 最后处理外面的 +c/−c（平移）
```

---

### 方法 2：图像变换法

**核心思想：** 在图像上找到关键点，观察变换后的位置变化。

**关键点选取：**

| 函数类型 | 选取的关键点 |
|---------|--------------|
| 三角函数 | 零点、最大点、最小点 |
| 幂函数 | 与坐标轴交点、顶点 |
| 指数/对数 | 与坐标轴交点、渐近线交点 |

### 识别信号表

| 题干关键词 | 识别信号 | 首选方法 |
|-----------|---------|---------|
| "由 f(x) 得到 f(2x)" | 伸缩变换 | 横坐标缩短一半 |
| "左移 2 单位" | 平移变换 | x 替换为 x + 2 |
| "纵坐标扩大 3 倍" | 伸缩变换 | 整个函数乘以 3 |
| "关于 x 轴对称" | 对称变换 | y 替换为 −y |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "y = f(x − 1) 是左移 1" | 记反方向 | x − 正数 = 右移；x + 正数 = 左移 |
| 2 | "y = 2f(x) 是横坐标扩大 2 倍" | 混淆纵横 | 系数乘在 f 上影响纵坐标 |
| 3 | "多个变换顺序无所谓" | 不理解变换本质 | 顺序不同结果可能不同 |
| 4 | "y = f(2x + 1) 直接横坐标压缩 2 倍" | 忽略平移 | 要先把 +1 提取出来 |
