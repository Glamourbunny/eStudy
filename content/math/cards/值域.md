# 值域

> 值域是函数三要素的"输出端"——回答"经过规则后落到哪里"的问题。

---

## 概念层

### 问题的起源

知道了"谁来"（定义域）和"怎么变"（对应法则），还需要问："最终落在哪？"

- y = x² 中，x 可以是任意实数，但 y 永远是非负数
- y = sin x 中，x 任意，y 永远在 [−1, 1] 之间

值域描述的是：**在定义域上经过对应法则后，y 所有可能取值的集合**。

> **一句话：** 值域是由定义域和对应法则共同决定的输出范围。

---

### 原始定义（教材原文）

一般地，设 A 是函数 y = f(x) 的定义域，对于集合 A 中的任意一个 x，通过规则 f，在集合 B 中都有唯一确定的 y 与之对应，那么 y 的取值范围称为**函数的值域**，记作 f(A)。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 63 页

**关键理解：**
- 值域是**结果**，不是起点
- 值域 ⊆ B（B 是陪域），但可能只是 B 的一个子集
- 求值域的本质是：**在定义域约束下，y 能取哪些值**

---

### 本质定义（逻辑重构版）

```
值域 = { y ∈ R | ∃ x ∈ 定义域，使得 f(x) = y }

"存在性"是值域的核心：
- y 在值域中 ⟺ 至少有一个 x 在定义域内，使 f(x) = y
- y 不在值域中 ⟺ 定义域内没有 x 能产生这个 y
```

**函数三要素的关系：**

```
    定义域（输入集合 A）
          │
          ▼ 对应法则 f
          │
    值域（输出集合 f(A)）⊆ B
```

> 两函数相等：定义域相同 **且** 对应法则相同。值域由前两者决定，不能单独比较。

---

### 定义域与值域的对比

| 对比项 | 定义域 | 值域 |
|--------|--------|------|
| 本质 | 输入 x 的范围 | 输出 y 的范围 |
| 决定因素 | 解析式本身的限制 | 定义域 + 对应法则共同决定 |
| 求法思路 | 解不等式（禁区取补） | 反解、配方、分析上界/下界 |
| 关系 | 自变量独立范围 | 因变量受约束范围 |
| 特殊情形 | 自然/指定/实际 | 可能等于、包含或被包含于 B |

---

### 邻近概念辨析

**值域 vs 定义域 —— "舞台" vs "观众席"**

| 对比项 | 值域 | 定义域 |
|--------|------|--------|
| 位置 | 输出端（终点） | 输入端（起点） |
| 角色 | 规则的"产出" | 规则的"原料" |
| 计算 | 由 f 和定义域共同决定 | 解析式本身的限制 |
| 举例（y=x²） | y ≥ 0 | x ∈ R |
| 举例（y=sin x） | y ∈ [−1,1] | x ∈ R |

---

### 适用边界

| 情形 | 值域特征 | 示例 |
|------|---------|------|
| 有界函数 | 值域有确定的上界或下界 | y=sin x：值域 [−1,1] |
| 无界函数 | 值域可无限大 | y=x²：值域 [0,+∞) |
| 周期函数 | 值域在有限区间 | y=sin x：总是 [−1,1] |
| 分段函数 | 各段值域取并集 | f(x)={x² (x≥0), -x (x<0)}，值域 [0,+∞)∪(0,+∞) |

## 命题层

### 核心定理

**定理：常见函数的值域公式**

| 函数类型 | 定义域 | 值域求法 | 值域结果 |
|---------|--------|---------|---------|
| y = ax² + bx + c | R | 配方：y = a(x−h)² + k | a>0：y≥k；a<0：y≤k |
| y = (ax+b)/(cx+d) | x≠−d/c | 反解法：x = (y−b)/(a−cy) | y≠a/c |
| y = √(ax+b) | ax+b≥0 | 直接观察 | y≥0 |
| y = aˣ | R | 指数函数性质 | y>0 |
| y = logₐx | x>0 | 对数函数性质 | y∈R |

**典型例题：**

求 y = x² − 4x + 6 的值域。

```
分析：二次函数，配方求最值。

y = x² − 4x + 6
  = (x² − 4x + 4) + 2
  = (x − 2)² + 2

∵ (x − 2)² ≥ 0
∴ y ≥ 2

答案：值域为 [2, +∞)
```

> **方法提炼：** 二次函数值域的关键是**配方成 (x−h)² + k 形式**，由平方的非负性确定 y 的范围。

## 方法层

### 方法 1：反解法（适合分式函数）

**核心思想：** 把 x 用 y 表示，解出 y 的限制条件。

**标准操作流程：**

```
f(x) = (ax + b) / (cx + d)
    ↓
y(cx + d) = ax + b
    ↓
cyx + yd = ax + b
    ↓
整理得：x = (b − yd) / (cy − a)
    ↓
要求分母不为零：cy − a ≠ 0
    ↓
即 y ≠ a/c

答案：值域为 { y ∈ R | y ≠ a/c }
```

---

### 方法 2：配方法（适合二次函数）

**核心思想：** 通过配方将二次式转化为完全平方加常数，利用平方的非负性求范围。

**标准操作流程：**

```
y = ax² + bx + c
    ↓
提取 a：y = a[(x + b/2a)² − (b/2a)²] + c
    ↓
配方：y = a(x + b/2a)² + (4ac − b²)/4a
    ↓
由 (x + b/2a)² ≥ 0
判断 a 的符号，确定 y 的最值
```

---

### 方法 3：换元法（适合复合函数）

**核心思想：** 设 u = g(x)，将复合函数转化为关于 u 的简单函数。

```
已知 f(u) 的值域为 V，g(x) 的值域为 U
则 f(g(x)) 的值域 ⊆ V
需要进一步确认 u 是否能取得 V 中的所有值
```

---

### 方法 4：图像法（适合已知图像的函数）

**核心思想：** 通过函数图像直接观察 y 的取值范围。

**判断技巧：**

| 图像特征 | 值域特征 |
|---------|---------|
| 有最高点 | 有上界（最大值） |
| 有最低点 | 有下界（最小值） |
| 向右无限延伸 | 无上界或下界 |
| 震荡范围有限 | 有界（上下界都存在） |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "定义域相同则值域相同" | 混淆两要素 | 值域还取决于对应法则 |
| 2 | "二次函数值域一定是 [k, +∞)" | 忽略 a 的符号 | a<0 时值域是 (−∞, k] |
| 3 | "分段函数值域是各段值域的并集" | 未检查连续性 | 并集没错，但要检查是否"断档" |
| 4 | "反解法得到的 y 就是值域" | 未排除分母为零情况 | y = a/c 必须排除 |
| 5 | "值域一定等于整个 B 集" | 混淆值域与陪域 | 值域可能只是 B 的真子集 |
| 6 | "配方时符号搞错" | 粗心 | 配方后一定要根据 a 的正负判断 |
