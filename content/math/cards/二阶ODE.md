# 二阶ODE

> 二阶ODE是包含二阶导数的方程——描述"加速度由位置和速度决定"的规律，是弹簧振动、电路振荡等物理现象的数学模型。

---

## 概念层

### 问题的起源

二阶ODE在物理中无处不在：
- 弹簧振动：m·d²x/dt² + kx = 0（牛顿第二定律）
- RLC电路：L·d²q/dt² + R·dq/dt + q/C = 0（电磁感应）
- 摆动：小角度近似下 θ'' + (g/L)θ = 0

这些问题的共同特征：**加速度（二阶导数）由位置和速度决定**。

> **一句话：** 二阶ODE是描述"加速度如何由当前状态决定"的方程，是物理世界最常见的微分方程类型。

---

### 本质定义（逻辑重构版）

**二阶齐次线性ODE的一般形式：**

```
y'' + p·y' + q·y = 0
```

**解法：特征根法**

假设解的形式为 y = e^(rt)，代入方程得特征方程：

```
r² + pr + q = 0
```

根据特征根的不同情况：

| 特征根情况 | 通解形式 |
|-----------|---------|
| 两个不等实根 r₁, r₂ | y = C₁·e^(r₁x) + C₂·e^(r₂x) |
| 两个相等实根 r₀ | y = (C₁ + C₂·x)·e^(r₀x) |
| 共轭复根 α±βi | y = e^(αx)(C₁·cosβx + C₂·sinβx) |

---

### 物理模型示例：弹簧振动

```
m·x'' + kx = 0

特征方程：mr² + k = 0
r² = −k/m
r = ±i·√(k/m)

解：x(t) = C₁·cos(ωt) + C₂·sin(ωt)，其中 ω = √(k/m)

这正是简谐运动！
```

## 方法层

### 方法：二阶齐次线性ODE求解流程

**标准操作流程：**

```
Step 1: 将方程化为标准形式 y'' + py' + qy = 0
Step 2: 写特征方程 r² + pr + q = 0
Step 3: 解特征方程，判断根的情况
Step 4: 根据根的情况写通解
Step 5: 用初始条件求 C₁, C₂
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "特征方程写错符号" | 粗心 | 必须是 r² + pr + q = 0，不是 r² − pr − q = 0 |
| 2 | "重根情况忘乘 x" | 记忆不完整 | 通解是 (C₁ + C₂·x)·e^(r₀x) |
| 3 | "复根情况写错形式" | 记混 | 复根时是三角函数形式，不是指数形式 |
