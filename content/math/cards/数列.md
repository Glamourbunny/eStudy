# 数列

> 数列是定义域为正整数集的离散函数——从第一项开始，按规则排列的一串数。

---

## 概念层

### 问题的起源

在日常生活中，我们经常遇到按顺序排列的数：
- 每个月存入银行的本金：a₁, a₂, a₃, ...
- 每周股票的价格：p₁, p₂, p₃, ...
- 细胞分裂的个数：1, 2, 4, 8, ...

这些"按一定顺序排列的数"就是**数列**。与连续函数不同，数列的定义域是正整数，描述的是**离散型变化**。

> **一句话：** 数列是"离散版的函数"——定义域是正整数，描述的是依次出现的一串数。

---

### 原始定义（教材原文）

按照一定顺序排列的一列数称为**数列**，数列中的每一个数叫做这个数列的**项**，第 n 项记作 aₙ。

> 出处：人教版《普通高中教科书·数学（B版）选择性必修第二册》第 4 页

**定义拆解：**

| 关键词 | 含义 | 示例 |
|--------|------|------|
| 顺序 | 每个数都有确定的位置 | a₁ 是首项，a₂ 是第二项，... |
| 项 | 数列中的每个数 | a₁, a₂, ..., aₙ, ... |
| 通项 | 第 n 项的表达式 | aₙ = f(n) |

---

### 本质定义（逻辑重构版）

**数列是函数：**

```
数列：a₁, a₂, a₃, ..., aₙ, ...
      ↑
函数：f(1), f(2), f(3), ..., f(n), ...
```

数列的本质是定义域为 {1, 2, 3, ..., n}（或正整数集）的函数 aₙ = f(n)。

**数列的表示：**

| 表示方法 | 形式 | 示例 |
|---------|------|------|
| 通项公式 | aₙ = 表达式 | aₙ = 2n + 1 |
| 递推公式 | aₙ = f(aₙ₋₁) | aₙ = 2aₙ₋₁, a₁ = 1 |
| 图像 | 离散的点 |  |

---

### 数列的分类

| 分类标准 | 类型 | 特征 |
|---------|------|------|
| 项数有限/无限 | 有限数列 | 有最后一项 |
| | 无限数列 | 没有最后一项 |
| 单调性 | 递增数列 | aₙ₊₁ > aₙ |
| | 递减数列 | aₙ₊₁ < aₙ |
| | 常数列 | aₙ₊₁ = aₙ |
| 有界性 | 有界数列 | 存在 M 使 |aₙ| ≤ M |
| | 无界数列 | 不存在这样的 M |

---

### 邻近概念辨析

**辨析：数列 vs 函数 —— "离散" vs "连续"**

| 对比项 | 数列 aₙ | 函数 f(x) |
|--------|---------|-----------|
| 定义域 | 正整数集 {1,2,3,...} | 实数集 R 或区间 |
| 自变量 | n（离散） | x（连续） |
| 图像 | 离散的点 | 连续的曲线 |
| 研究方法 | 递推、通项 | 解析式、连续分析 |
| 例子 | aₙ = 2n | f(x) = 2x |

**辨析：数列 vs 数集 —— "有序" vs "无序"**

| 对比项 | 数列 | 数集 |
|--------|------|------|
| 元素顺序 | 有顺序，不能打乱 | 无顺序 |
| 重复性 | 可以有重复项 | 不能有重复元素 |
| 记号 | a₁, a₂, a₃, ... | {a, b, c, ...} |

---

### 适用边界

| 要点 | 说明 |
|------|------|
| 数列下标从1开始 | 也有从0开始的约定 |
| 数列可以收敛 | 如 1/n 收敛于 0 |
| 递推公式是数列的重要表示 | 不是所有数列都有通项公式 |

## 命题层

### 核心定理

**定理：数列极限的存在条件**

单调有界数列必定收敛（极限存在）。

- 递增且有上界 → 收敛
- 递减且有下界 → 收敛

**定理：数列极限的四则运算**

若 lim aₙ = A，lim bₙ = B，则：
- lim (aₙ ± bₙ) = A ± B
- lim (aₙ · bₙ) = A · B
- lim (aₙ / bₙ) = A / B（B ≠ 0）

---

### 典型例题

**例题 1：已知通项公式求特定项**

```
题目：已知数列 aₙ = n²/(n+1)，求 a₁₀。
```

**解题步骤：**

```
a₁₀ = 10²/(10+1) = 100/11
```

---

**例题 2：由递推公式求通项**

```
题目：已知 a₁ = 1，aₙ₊₁ = 2aₙ，求通项公式。
```

**深层分析：**

```
分析：这是等比数列型递推

a₁ = 1
a₂ = 2a₁ = 2
a₃ = 2a₂ = 4
a₄ = 2a₃ = 8
...

观察规律：aₙ = 2^(n-1)

验证：a₁ = 2^0 = 1 ✓
      a₂ = 2^1 = 2 ✓

答案：aₙ = 2^(n-1)
```

> **方法提炼：** 形如 aₙ₊₁ = q·aₙ 的递推，数列是公比为 q 的等比数列，通项 aₙ = a₁·q^(n-1)。

---

**例题 3：数列的单调性判断**

```
题目：判断数列 aₙ = n/(n+1) 的单调性。
```

**深层分析：**

```
aₙ₊₁ − aₙ = (n+1)/(n+2) − n/(n+1)
           = [(n+1)² − n(n+2)] / [(n+2)(n+1)]
           = (n²+2n+1 − n²−2n) / [(n+2)(n+1)]
           = 1/[(n+2)(n+1)] > 0

∴ aₙ₊₁ > aₙ，数列递增
```

> **方法提炼：** 判断数列单调性常用方法：比较 aₙ₊₁ 与 aₙ 的大小。

## 方法层

### 方法 1：求数列通项公式

**核心思想：** 识别数列模式，找到 aₙ 与 n 的关系。

**常见模式：**

| 递推类型 | 通项公式 | 方法 |
|---------|---------|------|
| aₙ₊₁ − aₙ = d | aₙ = a₁ + (n−1)d | 等差数列 |
| aₙ₊₁ / aₙ = q | aₙ = a₁·q^(n−1) | 等比数列 |
| aₙ₊₁ = aₙ + f(n) | 累加法 | 叠加 |
| aₙ₊₁ = aₙ · f(n) | 累乘法 | 叠乘 |

---

### 方法 2：求数列前 n 项和

**核心思想：** 根据数列类型选择求和公式。

**公式：**

```
等差数列前 n 项和：Sₙ = n(a₁ + aₙ)/2 = n(2a₁ + (n-1)d)/2
等比数列前 n 项和：Sₙ = a₁(1 − q^n)/(1 − q)（q ≠ 1）
                  Sₙ = na₁（q = 1）
```

---

### 方法 3：数列极限的计算

**核心思想：** 识别类型，应用公式。

| 类型 | 方法 | 示例 |
|------|------|------|
| 多项式/指数 | 抓大头 | n→∞ 时，n/(n+1) → 1 |
| 0/0 型 | 分子分母同除以最高次幂 | (n²+1)/(2n²+3) → 1/2 |
| ∞/∞ 型 | 无穷多项和 | 等比数列求和公式 |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "数列下标可以从0开始" | 混淆约定 | 教材默认从1开始，但也有从0开始的约定 |
| 2 | "等比数列公比不能为1" | 混淆公比 | q=1 时是常数列，等比数列公比可为1 |
| 3 | "求和公式 Sₙ = na₁ 对所有等比数列适用" | 记错条件 | q≠1 时才用 (1−q^n)/(1−q)，q=1 时才是 na₁ |
| 4 | "数列收敛一定有极限" | 术语混淆 | 收敛和极限是同一含义的不同说法 |
