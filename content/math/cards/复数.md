# 复数

> 复数是数学家为解决"负数不能开平方"而创造的人工数系——它在后来被发现是描述旋转和波动的天然语言。

---

## 概念层

### 问题的起源

**问题：** 方程 x² + 1 = 0 有解吗？

在实数范围内，x² = −1 无解，因为任何实数的平方都 ≥ 0。

16 世纪的数学家开始思考：如果允许负数开平方，会发生什么？意大利数学家卡尔丹（Cardano）引入了 √(-1) 的概念，开始了复数的历史。

**为什么需要复数？**

| 问题类型 | 实数能否解决 | 需要复数的原因 |
|---------|-------------|--------------|
| 方程 x² = 4 | ✓（x = ±2） | 实数范围内已有解 |
| 方程 x² = −4 | ✗ | 需要引入 √(-1) |
| 方程 x² + 1 = 0 | ✗ | 需要 i = √(-1) |
| 旋转描述 | 有限制 | 需要复数的乘法来描述旋转 |

> **一句话：复数最初是为了让负数也能开平方而引入的，最终却发现它是描述旋转和波动的完美语言。**

---

### 原始定义（教材原文）

形如 a + bi（a, b ∈ R）的数称为**复数**，其中 i 称为**虚数单位**，满足 i² = −1。

- a 称为复数的**实部**，记作 Re(z) = a
- b 称为复数的**虚部**，记作 Im(z) = b

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 18-20 页

---

### 本质定义

复数的本质是**二维数**——用一个有序数对 (a, b) 表示一个数：

```
z = a + bi  ⟺  z = (a, b)
```

**复数的两种表示：**

| 表示形式 | 写法 | 说明 |
|----------|------|------|
| **代数形式** | z = a + bi | a 是实部，b 是虚部 |
| **坐标形式** | z = (a, b) | 对应复平面上的点 |

**复数的分类：**

| 类型 | 条件 | 示例 |
|------|------|------|
| **实数** | b = 0 | z = 3（虚部为 0） |
| **纯虚数** | a = 0 且 b ≠ 0 | z = 2i |
| **非纯虚数** | a ≠ 0 且 b ≠ 0 | z = 1 + 2i |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **实部 a** | 复数的"横坐标" |
| **虚部 b** | 复数的"纵坐标" |
| **虚数单位 i** | i² = −1，复数的"生成元" |

---

### 等价表述

| 表述形式 | 公式 | 适用场景 |
|----------|------|---------|
| **代数形式** | z = a + bi | 四则运算 |
| **坐标形式** | z = (a, b) | 几何意义 |
| **三角形式** | z = r(cosθ + i·sinθ) | 乘法/乘方 |
| **指数形式** | z = re^(iθ) | 大学数学，复利增长 |

### 图像表示

**复平面：**

```
                    Im(z) 轴
                    │
                    │
                    │• P(a,b)
                    │ ╱
                    │╱
                    │╱ r
                    O────────────────→ Re(z) 轴
                    │
                    │
```

复平面上的点 P(a, b) 对应复数 z = a + bi。

- 实部 a 是 P 到虚轴的水平距离
- 虚部 b 是 P 到实轴的垂直距离
- 模 r = |z| = √(a² + b²) 是 OP 的长度
- 辐角 θ 是 OP 与正实轴的夹角

### 邻近概念网络

```
                    数系扩充
                        │
    ┌────────┌─────────┼─────────┌────────┐
    │        │         │         │        │
  自然数    整数     有理数     实数     复数
   N        Z         Q         R        C
                            (b=0)  (a+bi)
```

**与实数的关系：**
- 实数是复数的子集（b = 0 时）
- 复数包含所有实数和虚数
- 复数域 C 是封闭的（任何多项式方程都有解）

**与向量的联系：**
- 复数 z = a + bi 对应平面向量 (a, b)
- 复数的加减对应向量的加减
- 复数的乘法对应向量的旋转和伸缩

### 概念辨析

**辨析 1：复数 vs 实数**

| 对比项 | 实数 | 复数 |
|--------|------|------|
| **形式** | a | a + bi |
| **几何** | 数轴上的点 | 复平面上的点 |
| **开平方** | 正数有两个，负数无 | 任何数都有两个 |
| **大小比较** | 能比较（> , <） | 不能直接比较大小 |
| **例子** | 3, −2, π, √2 | 1 + 2i, 3i, −4 |

> **一句话：** 复数不能比大小，因为它是二维的，点与点之间没有"谁更大"。

**辨析 2：复数的相等**

```
z₁ = a₁ + b₁i，z₂ = a₂ + b₂i
z₁ = z₂ ⟺ a₁ = a₂ 且 b₁ = b₂
```

两个复数相等，当且仅当它们的实部和虚部分别相等。

> **注意：** 不能比较大小，但可以比较是否相等。

**辨析 3：i 的本质**

i 不是"某个具体的数"，而是满足 i² = −1 的**抽象单位**。

类似地，"1"是满足 1×1 = 1 的单位。i² = −1 是它的"负平方"版本。

```
i¹ = i
i² = −1
i³ = −i
i⁴ = 1
i⁵ = i
...（周期性）
```

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 复数加减乘除 | ✓ | 结果仍是复数 |
| 复数开平方 | ✓ | 任何复数都有两个平方根 |
| 复数比较大小 | ✗ | 无法定义 < 或 > |
| i² = −1 | ✓ | 虚数单位的定义 |
| 复数为 0 | a = 0 且 b = 0 | 实部和虚部都为 0 |

**复数的模的性质：**

| 性质 | 公式 |
|------|------|
| 非负性 | |z| ≥ 0 |
| 定正性 | |z| = 0 ⟺ z = 0 |
| 对称性 | |z| = |z̄| |
| 乘法 | \|z₁z₂\| = \|z₁\|·\|z₂\| |

## 命题层

### 核心定理

**定理 1：复数的四则运算**

```
加法：(a + bi) + (c + di) = (a + c) + (b + d)i
减法：(a + bi) − (c + di) = (a − c) + (b − d)i
乘法：(a + bi)(c + di) = (ac − bd) + (ad + bc)i
除法：(a + bi)/(c + di) = (ac + bd)/(c² + d²) + (bc − ad)/(c² + d²)i
```

> **记忆口诀：** 复数乘法"穿针引线"——实部乘实部减虚部乘虚部，虚部是内外分别相乘再相加。

**定理 2：复数的模**

```
|z| = √(a² + b²)
|z₁z₂| = |z₁|·|z₂|
|z₁/z₂| = |z₁|/|z₂|
```

**定理 3：复数的共轭**

z = a + bi 的共轭复数是 z̄ = a − bi。

```
z·z̄ = |z|² = a² + b²
```

**定理 4：代数基本定理**

n 次方程在复数域中有且仅有 n 个根（重根按重数计数）。

这是复数域的"封闭性"保证——在复数域内，任何多项式方程都能分解成一次因式。

---

### 典型例题的深层分析

**例题 1：复数的四则运算**

```
题目：计算 (1 + 2i)(3 − i)。
```

**深层分析：**

```
(1 + 2i)(3 − i) = 1×3 + 1×(−i) + 2i×3 + 2i×(−i)
                = 3 − i + 6i − 2i²
                = 3 + 5i − 2(−1)
                = 3 + 5i + 2
                = 5 + 5i
```

> **方法提炼：** 复数乘法按多项式乘法展开，记得 i² = −1。

---

**例题 2：复数的模**

```
题目：已知 |z − 3 + 4i| = 5，表示什么图形？
```

**深层分析：**

|z − (3 − 4i)| = 5 表示：**到定点 3 − 4i 的距离等于 5 的点的轨迹**。

在复平面上，这是以 (3, −4) 为圆心，半径为 5 的圆。

> **方法提炼：** |z − z₀| = r 在复平面上表示圆。

---

**例题 3：复数方程**

```
题目：解方程 z² + 2z + 5 = 0。
```

**深层分析：**

```
z² + 2z + 5 = 0
Δ = 4 − 20 = −16
z = (−2 ± √(−16)) / 2 = (−2 ± 4i) / 2 = −1 ± 2i
```

解是 z₁ = −1 + 2i，z₂ = −1 − 2i（互为共轭）。

> **方法提炼：** 复数域中，负数也能开平方，√(−16) = 4i。

---

**例题 4：复数的几何意义**

```
题目：证明复数乘法 z₁·z₂ 的模等于模的乘积，辐角等于辐角的和。
```

**深层分析：**

设 z₁ = r₁(cosθ₁ + i·sinθ₁)，z₂ = r₂(cosθ₂ + i·sinθ₂)。

则：
```
z₁z₂ = r₁r₂[(cosθ₁cosθ₂ − sinθ₁sinθ₂) + i(sinθ₁cosθ₂ + cosθ₁sinθ₂)]
     = r₁r₂[cos(θ₁+θ₂) + i·sin(θ₁+θ₂)]
```

所以 |z₁z₂| = r₁r₂ = |z₁|·|z₂|，辐角为 θ₁ + θ₂。

> **方法提炼：** 复数乘法的几何意义是**旋转+伸缩**——模相乘，辐角相加。

### 方法详解：复数运算通法

**1. 加减乘法：直接展开**

```
Step 1: 按多项式乘法展开
Step 2: 合并实部（i² 换成 −1）
Step 3: 合并虚部
```

**2. 除法：分母实数化**

```
Step 1: 分子分母同乘分母的共轭
Step 2: 分母变成 |z|²（实数）
Step 3: 分子按乘法展开
Step 4: 分离实部和虚部
```

**3. 几何问题：转化为距离**

```
|z − z₀| = r → 圆（圆心 z₀，半径 r）
|z − z₁| = |z − z₂| → 垂直平分线
|z − z₁| + |z − z₂| = 常数 → 椭圆
```

> **易错提醒：**
> 1. i² = −1，不是 i² = 1
> 2. 复数不能比较大小，但模可以
> 3. 除法时分母实数化后要分离实部虚部
