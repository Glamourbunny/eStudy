# 组合

> 组合是"从 n 个东西里选 k 个，不排顺序"——记作 Cₙᵏ，等于 n!/[k!(n−k)!]。

---

## 概念层

### 问题的起源

从 5 个人中选 3 个人组成篮球队，有多少种选法？

这和排列不同——选出来的人**不需要排队**，只要把人挑出来就行。

5 选 3 的选法数 = C₅³ = 5!/(3!2!) = 10 种。

这就是**组合数**。

> **一句话：组合是"不考虑顺序"的选取——Cₙᵏ = n!/[k!(n−k)!]。**

---

### 本质定义

**组合的本质是"无序选取"：**

```
组合：只管"谁被选中"
排列：还要管"谁排前面谁排后面"
```

**组合与排列的关系：**

```
Pₙᵏ = Cₙᵏ × k!
```

先选出 k 个人（组合），再给这 k 个人排序（排列）。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **n 个元素** | 不同候选元素 |
| **选 k 个** | 不放回抽取 |
| **无序** | 不考虑顺序 |

---

### 与排列的关系

| 对比项 | 组合 Cₙᵏ | 排列 Pₙᵏ |
|--------|----------|----------|
| 是否考虑顺序 | ✗ | ✓ |
| 公式 | n!/[k!(n−k)!] | n!/(n−k)! |
| 关系 | Pₙᵏ = Cₙᵏ × k! | Cₙᵏ = Pₙᵏ/k! |

---

## 命题层

### 核心定理

**定理 1：组合数公式**

```
Cₙᵏ = n!/[k!(n−k)!]
```

**定理 2：对称性**

```
Cₙᵏ = Cₙⁿ⁻ᵏ
```

**定理 3：帕斯卡法则**

```
Cₙᵏ = Cₙ₋₁ᵏ⁻¹ + Cₙ₋₁ᵏ
```

---

### 典型例题

**例题：从 5 人中选 3 人，有多少种选法？**

```
题目：从 5 人中选 3 人组成篮球队，不考虑位置，有多少种选法？
```

**深层分析：**

```
C₅³ = 5!/(3!2!) = 120/(6×2) = 10
```

> **方法提炼：** 组合问题用 Cₙᵏ，不考虑顺序。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "C₅² = 5×4 = 10" | 与排列混淆 | C₅² = 5×4/2 = 10，正确 |
| 2 | "排列和组合混淆" | 混淆 | 排列有序，组合无序 |
| 3 | "Cₙᵏ = Cₙᵏ⁻¹" | 计算错误 | Cₙᵏ = Cₙ₋₁ᵏ⁻¹ + Cₙ₋₁ᵏ |
