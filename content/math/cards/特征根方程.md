# 特征根方程

> 特征根方程是求解二阶线性递推数列的代数工具——把复杂的递推关系转化为清晰的通项公式。

---

## 概念层

### 问题的起源

等差数列和等比数列有明确的通项公式，但更一般的递推数列如何求通项？

例如：斐波那契数列 1, 1, 2, 3, 5, 8, 13, ... 满足 aₙ₊₂ = aₙ₊₁ + aₙ

这类**二阶线性递推数列**的通项如何求解？特征根方程提供了系统的方法。

> **一句话：** 特征根方程通过代数手段，把递推关系"翻译"成通项公式。

---

### 本质定义（逻辑重构版）

**二阶齐次线性递推的一般形式：**

```
aₙ₊₂ = p·aₙ₊₁ + q·aₙ    （p, q 为常数，q ≠ 0）
```

**特征根方程的构造：**

假设解的形式为 aₙ = rⁿ（指数型猜想），代入递推式：

```
rⁿ⁺² = p·rⁿ⁺¹ + q·rⁿ
r² = p·r + q
r² − p·r − q = 0

这个关于 r 的二次方程，就是特征根方程
r 称为特征根
```

---

### 特征根方程的解法

**情况一：两个不相等实根 r₁, r₂**

```
特征方程：r² − pr − q = 0 有两个不相等实根 r₁, r₂

通项公式：aₙ = C₁·r₁ⁿ + C₂·r₂ⁿ

其中 C₁, C₂ 由初始条件 a₁, a₂ 确定
```

**情况二：两个相等实根 r₀**

```
特征方程的判别式 Δ = 0，两个相等实根 r₁ = r₂ = r₀

通项公式：aₙ = (C₁ + C₂·n)·r₀ⁿ
```

**情况三：共轭复根 r = α ± βi**

```
通项公式：aₙ = rⁿ 可转化为三角形式
aₙ = ρⁿ(A·cos nθ + B·sin nθ)

其中 ρ = √(α² + β²)，θ = arctan(β/α)
```

---

### 斐波那契数列的求解示例

```
递推关系：aₙ₊₂ = aₙ₊₁ + aₙ，初始 a₁ = 1, a₂ = 1

特征方程：r² − r − 1 = 0
解得：r₁ = (1 + √5)/2 ≈ 1.618，r₂ = (1 − √5)/2 ≈ −0.618

通项公式：aₙ = C₁[(1+√5)/2]ⁿ + C₂[(1−√5)/2]ⁿ

由初始条件：
a₁ = C₁·r₁ + C₂·r₂ = 1
a₂ = C₁·r₁² + C₂·r₂² = 1

解得：C₁ = 1/√5，C₂ = −1/√5

∴ aₙ = [(1+√5)/2]ⁿ − [(1−√5)/2]ⁿ / √5
```

### 邻近概念辨析

**辨析：特征根方程 vs 待定系数法 —— "系统解法" vs "特殊技巧"**

| 对比项 | 特征根方程 | 待定系数法 |
|--------|----------|----------|
| 适用类型 | 二阶齐次线性递推 | 低阶特殊递推 |
| 核心思想 | 解特征方程 | 设特定形式的解 |
| 通用性 | 通用公式 | 针对特定形式 |
| 复杂度 | 需解二次方程 | 相对简单 |

---

### 适用边界

| 条件 | 说明 |
|------|------|
| 二阶齐次线性递推 | 形式必须是 aₙ₊₂ = p·aₙ₊₁ + q·aₙ |
| 非齐次情况 | 非齐次递推需要先求齐次解再加特解 |
| 复根情况 | 会出现三角函数形式的通项 |

## 命题层

### 典型例题

**例题 1：两个不等实根**

```
题目：已知 a₁ = 1, a₂ = 3，递推关系 aₙ₊₂ = 3aₙ₊₁ − 2aₙ，求通项。
```

**深层分析：**

```
特征方程：r² − 3r + 2 = 0
解得：r₁ = 1, r₂ = 2

通项形式：aₙ = C₁·1ⁿ + C₂·2ⁿ = C₁ + C₂·2ⁿ

由初始条件：
a₁ = C₁ + 2C₂ = 1
a₂ = C₁ + 4C₂ = 3

解得：C₂ = 1, C₁ = −1

∴ aₙ = 2ⁿ − 1
```

> **方法提炼：** 代入初始条件求 C₁, C₂ 是标准步骤。

---

**例题 2：两个相等实根**

```
题目：已知 a₁ = 1, a₂ = 4，递推关系 aₙ₊₂ = 2aₙ₊₁ − aₙ，求通项。
```

**深层分析：**

```
特征方程：r² − 2r + 1 = 0
判别式 Δ = 4 − 4 = 0，重根 r = 1

通项形式：aₙ = (C₁ + C₂·n)·1ⁿ = C₁ + C₂·n

由初始条件：
a₁ = C₁ + C₂ = 1
a₂ = C₁ + 2C₂ = 4

解得：C₁ = −2, C₂ = 3

∴ aₙ = 3n − 2
```

> **方法提炼：** 重根情况下通项有 (C₁ + C₂·n)·rⁿ 的形式。

## 方法层

### 方法：特征根方程求解流程

**标准操作流程：**

```
Step 1: 将递推式化为标准形式 aₙ₊₂ − p·aₙ₊₁ − q·aₙ = 0
Step 2: 构造特征方程 r² − pr − q = 0
Step 3: 解特征方程，判断根的情况
Step 4: 根据根的情况写出通项形式
Step 5: 用初始条件求 C₁, C₂
Step 6: 写出完整通项公式
```

---

### 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "特征方程写错符号" | 粗心 | 必须是 r² − pr − q = 0 |
| 2 | "重根情况忘记乘以 n" | 记忆不完整 | 通项是 (C₁ + C₂·n)·rⁿ，不是简单的 C·rⁿ |
| 3 | "复根情况不会处理" | 不熟悉复数运算 | 复根会导致三角函数形式的通项 |
