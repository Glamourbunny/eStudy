# 瞬时变化率

> 瞬时变化率是"在某一时刻/某一点的变化剧烈程度"——不是"一段距离走了多久"，而是"此刻正在多快"。

---

## 概念层

### 问题的起源

平均速度回答"这段路平均多快"，但瞬时速度回答"此刻速度表指向多少"。

生活中我们需要瞬时概念：
- 汽车在某一时刻的速度（不是全程平均）
- 某一时刻的人口增长率（不是十年平均）
- 某时刻的电流大小（不是平均电流）

瞬时变化率就是导数的物理意义——把"平均"缩到"瞬间"。

> **一句话：** 瞬时变化率是把时间间隔 Δt 无限趋近于零时的平均变化率的极限。

---

### 本质定义（逻辑重构版）

**数学定义：**

```
瞬时变化率 = lim(Δx→0) [f(x₀+Δx) − f(x₀)] / Δx
           = f'(x₀)
```

**物理含义（以速度为例）：**

```
物体运动 s(t)，时刻 t₀ 的瞬时速度：
v(t₀) = lim(Δt→0) [s(t₀+Δt) − s(t₀)] / Δt = s'(t₀)
```

---

### 瞬时 vs 平均

| 对比项 | 瞬时变化率 | 平均变化率 |
|--------|-----------|-----------|
| 公式 | f'(x₀) = lim(Δx→0) Δy/Δx | Δy/Δx |
| 本质 | 某一点的变化率 | 一段区间内的变化率 |
| 几何意义 | 切线斜率 | 割线斜率 |
| 应用场景 | 精确描述"此刻" | 粗略描述"整体" |

---

## 方法层

### 方法：瞬时变化率的求法

**核心思想：** 用导数定义求极限。

**标准操作流程：**

```
Step 1: 写出平均变化率 Δy/Δx = [f(x₀+Δx) − f(x₀)] / Δx
Step 2: 化简这个比值
Step 3: 取 Δx→0 的极限
Step 4: 结果就是瞬时变化率
```
