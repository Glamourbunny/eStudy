# 三角恒等变换

> 三角恒等变换是利用三角函数之间的内在关系，对三角表达式进行化简或求值的技巧。

---

## 概念层

### 问题的起源

在数学运算中，经常需要：
- 已知 sin α，求 cos α、tan α（同角关系）
- 已知 sin(α±β)，展开成 sin 和 cos 的组合（和差公式）
- 已知 sin α，求 sin 2α（倍角公式）

这些变换不是随意进行的，而是基于三角函数之间客观存在的数学关系。三角恒等变换就是这些关系的系统化总结。

> **一句话：** 三角恒等变换是利用三角函数之间的内在规律，对表达式进行"等价变形"的技巧。

---

### 原始定义（教材原文）

三角恒等变换包括同角三角函数的基本关系、诱导公式、两角和差的三角函数公式、二倍角公式等。

### 同角三角函数基本关系

**平方关系：**

```
sin²α + cos²α = 1

变形：
sin²α = 1 − cos²α
cos²α = 1 − sin²α
```

**商数关系：**

```
tanα = sinα / cosα（cosα ≠ 0）

变形：
sinα = tanα · cosα
```

**倒数关系：**

```
cotα = 1 / tanα = cosα / sinα
```

---

### 和差公式

**正弦和差：**

```
sin(α + β) = sinα cosβ + cosα sinβ
sin(α − β) = sinα cosβ − cosα sinβ
```

**余弦和差：**

```
cos(α + β) = cosα cosβ − sinα sinβ
cos(α − β) = cosα cosβ + sinα sinβ
```

**正切和差：**

```
tan(α + β) = (tanα + tanβ) / (1 − tanα tanβ)
tan(α − β) = (tanα − tanβ) / (1 + tanα tanβ)
```

---

### 倍角公式

**基本倍角：**

```
sin2α = 2sinα cosα
cos2α = cos²α − sin²α
```

**余弦倍角的变形：**

```
cos2α = 2cos²α − 1 = 1 − 2sin²α

变形（降次公式）：
cos²α = (1 + cos2α) / 2
sin²α = (1 − cos2α) / 2
```

**正切倍角：**

```
tan2α = 2tanα / (1 − tan²α)
```

---

### 邻近概念辨析

**辨析：诱导公式 vs 三角恒等变换 —— "转角" vs "变形"**

| 对比项 | 诱导公式 | 三角恒等变换 |
|--------|---------|-------------|
| 研究对象 | 不同角的三角函数关系（差 π、π/2 等） | 任意角之间的和差倍关系 |
| 公式数量 | 6组基本公式 | 更多（和差、倍角、半角等） |
| 核心思想 | 利用圆的对称性 | 利用代数变形 |

---

### 适用边界

| 公式 | 使用条件 | 注意事项 |
|------|---------|---------|
| tanα = sinα/cosα | cosα ≠ 0 | tan 定义域限制 |
| 和差公式 | α, β, α±β 都有意义 | 无特殊限制 |
| 二倍角公式 | 2α 有意义 | α 任意实数 |

## 命题层

### 核心定理

**定理：同角三角函数关系组**

```
sin²α + cos²α = 1
tanα = sinα / cosα（α ≠ π/2 + kπ）
```

这三个关系让"知一求三"成为可能。

**定理：和差化积与积化和差**

```
积化和差：
sinα cosβ = [sin(α+β) + sin(α−β)] / 2
cosα sinβ = [sin(α+β) − sin(α−β)] / 2
cosα cosβ = [cos(α+β) + cos(α−β)] / 2
sinα sinβ = [cos(α−β) − cos(α+β)] / 2

和差化积：
sinα + sinβ = 2 sin[(α+β)/2] cos[(α−β)/2]
sinα − sinβ = 2 cos[(α+β)/2] sin[(α−β)/2]
cosα + cosβ = 2 cos[(α+β)/2] cos[(α−β)/2]
cosα − cosβ = −2 sin[(α+β)/2] sin[(α−β)/2]
```

---

### 典型例题

**例题 1：知 sinα 求 cosα 和 tanα**

```
题目：已知 sinα = 3/5，α ∈ (π/2, π)，求 cosα 和 tanα。
```

**深层分析：**

```
分析：α 在第二象限，cosα < 0，tanα < 0

cosα = −√(1 − sin²α) = −√(1 − 9/25) = −√(16/25) = −4/5

tanα = sinα/cosα = (3/5) / (−4/5) = −3/4
```

> **方法提炼：** 已知 sin²α + cos²α = 1，求 cos 时注意符号由象限决定。

---

**例题 2：和差公式应用**

```
题目：求 sin75° 的值。
```

**深层分析：**

```
sin75° = sin(45° + 30°)
       = sin45° cos30° + cos45° sin30°
       = (√2/2)(√3/2) + (√2/2)(1/2)
       = (√6 + √2)/4
```

> **方法提炼：** 非特殊角的和差，可以拆成已知特殊角的和差，再使用和差公式。

---

**例题 3：二倍角公式应用**

```
题目：已知 cosα = 4/5，α ∈ (0, π/2)，求 sin2α 和 cos2α。
```

**深层分析：**

```
sin2α = 2 sinα cosα

先求 sinα：
sinα = √(1 − cos²α) = √(1 − 16/25) = 3/5

sin2α = 2 × (3/5) × (4/5) = 24/25

cos2α = cos²α − sin²α = 16/25 − 9/25 = 7/25
```

> **方法提炼：** 二倍角公式中，注意先求 sinα 或用 cos2α 的变形公式。

## 方法层

### 方法 1：知一求三（同角关系应用）

**核心思想：** 用同角三角函数关系，由已知求未知。

**标准操作流程：**

```
已知 sinα = k，求 cosα, tanα：

Step 1: 判断 α 所在象限，确定三角函数的符号
Step 2: 用 sin²α + cos²α = 1 求 cosα = ±√(1 − k²)
Step 3: tanα = sinα/cosα

已知 tanα = k，求 sinα, cosα：

Step 1: 设 sinα = k·cosα
Step 2: 代入 sin²α + cos²α = 1
Step 3: 解方程求 cosα，再求 sinα
```

---

### 方法 2：和差公式使用策略

**核心思想：** 拆角思想——把复杂角化为简单角的和差。

**核心技巧：**
- 看到非特殊角，想想能否拆成特殊角的和差
- 看到 sinα ± sinβ，想法和差化积
- 看到 sinα cosβ，想到积化和差

---

### 方法 3：二倍角的多种形式

**核心思想：** 根据已知条件选择合适的二倍角公式。

**选择策略：**

| 已知 | 适合公式 | 原因 |
|------|---------|------|
| sinα, cosα | sin2α = 2sinαcosα | 直接代入 |
| cos²α | cos2α = 2cos²α − 1 | 降次 |
| sin²α | cos2α = 1 − 2sin²α | 降次 |
| tanα | tan2α = 2tanα/(1−tan²α) | 用 tan |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "cos2α = cos²α − sin²α 有两个值" | 公式本身只有一个值 | 公式确定，两个变形也都等于它 |
| 2 | "sin²α = (1−cos2α)/2 可以直接开方" | 忘考虑符号 | sinα = ±√[(1−cos2α)/2]，符号由 α 决定 |
| 3 | "tanα = sinα/cosα 中 cosα 可以为 0" | 忽略定义域 | tanα 要求 cosα ≠ 0 |
| 4 | "和差公式记反符号" | 记忆不牢 | sin 相减时第二项为负；cos 相减时第二项为正 |
