# 柯西不等式

> 柯西不等式是"平方和的乘积大于等于乘积和的平方"——它把两个向量的数量积与它们的模长联系起来。

---

## 概念层

### 问题的起源

向量 a = (a₁, a₂, ..., aₙ) 和 b = (b₁, b₂, ..., bₙ) 的数量积：

a·b = a₁b₁ + a₂b₂ + ... + aₙbₙ

它们的模长：
|a| = √(a₁² + a₂² + ... + aₙ²)
|b| = √(b₁² + b₂² + ... + bₙ²)

有什么关系？

柯西发现：|a·b| ≤ |a|·|b|

等号成立当且仅当 a 和 b 共线。

> **一句话：柯西不等式是 |Σaᵢbᵢ| ≤ √(Σaᵢ²)·√(Σbᵢ²)。**

---

### 原始定义（教材原文）

对于任意实数 a₁, a₂, ..., aₙ 和 b₁, b₂, ..., bₙ，有：

```
(a₁² + a₂² + ... + aₙ²)(b₁² + b₂² + ... + bₙ²) ≥ (a₁b₁ + a₂b₂ + ... + aₙbₙ)²
```

等号成立当且仅当 a₁/b₁ = a₂/b₂ = ... = aₙ/bₙ（或 bᵢ = 0）。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第二册》

---

### 本质定义

**柯西不等式的向量形式：**

设向量 a = (a₁, a₂, ..., aₙ)，b = (b₁, b₂, ..., bₙ)，则：

```
|a·b| ≤ |a|·|b|
```

即：

```
|Σaᵢbᵢ| ≤ √(Σaᵢ²)·√(Σbᵢ²)
```

平方形式：
```
(Σaᵢ²)(Σbᵢ²) ≥ (Σaᵢbᵢ)²
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **两个向量** | a = (a₁, ..., aₙ)，b = (b₁, ..., bₙ) |
| **数量积** | Σaᵢbᵢ |
| **模长** | √(Σaᵢ²)，√(Σbᵢ²) |

---

### 典型形式

**二维情形（n = 2）：**

```
(a₁² + a₂²)(b₁² + b₂²) ≥ (a₁b₁ + a₂b₂)²
```

**三维情形（n = 3）：**

```
(a₁² + a₂² + a₃²)(b₁² + b₂² + b₃²) ≥ (a₁b₁ + a₂b₂ + a₃b₃)²
```

## 命题层

### 核心定理

**定理：柯西不等式**

```
(Σaᵢ²)(Σbᵢ²) ≥ (Σaᵢbᵢ)²
等号成立 ⟺ a 和 b 共线（存在 k 使 aᵢ = k·bᵢ）
```

**证明思路：**
考虑 |a − λb|² ≥ 0（向量的模长非负），展开得到关于 λ 的二次函数，判别式非正，即得柯西不等式。

---

### 典型例题

**例题：证明 (a² + b²)(c² + d²) ≥ (ac + bd)²**

```
题目：证明柯西不等式在 n = 2 时的情形。
```

**深层分析：**

```
左边 − 右边
= (a² + b²)(c² + d²) − (ac + bd)²
= a²c² + a²d² + b²c² + b²d² − a²c² − 2abcd − b²d²
= a²d² + b²c² − 2abcd
= (ad − bc)² ≥ 0

所以 (a² + b²)(c² + d²) ≥ (ac + bd)²
```

等号当 ad = bc，即 ad = bc 时成立。

> **方法提炼：** 直接展开化简为完全平方。
