# 面心立方

> 面心立方是原子在三维空间中最密集的堆积方式之一——它的堆积效率高达 74%，铜、银、金等金属都采用这种结构。

---

## 概念层

### 问题的起源

金属原子倾向于紧密堆积以降低能量。面心立方（FCC，Face-Centered Cubic）是一种最密堆积方式。

面心立方晶胞的特点：
-立方体的 8 个顶点各有 1 个原子
- 立方体的 6 个面心各有 1 个原子
- 堆积方式：ABCABC...（三层一循环）

> **一句话：面心立方是 ABCABC 型三层循环堆积，堆积效率 74%。**

---

### 原始定义（教材原文）

面心立方堆积的堆积方式为 ABCABC 型，即三层原子交错排列，第四层与第一层重合。

晶胞为立方晶胞。

> 出处：人教版《普通高中教科书·化学（选择性必修）》

### 本质定义

**晶胞结构：**

| 位置 | 原子数 | 说明 |
|------|--------|------|
| 顶点 | 8 × 1/8 = 1 | 每个顶点原子属于 8 个晶胞 |
| 面心 | 6 × 1/2 = 3 | 每个面心原子属于 2 个晶胞 |
| **合计** | **4** | 每个晶胞含有 4 个原子 |

**堆积方式：**

| 层 | A 位 | B 位 | C 位 |
|----|------|------|------|

| 4 | A | | |

## 命题层

### 核心定理

**定理 1：原子半径与晶胞边长的关系**

```
对于面心立方：a = 2√2r
其中 a 是晶胞边长，r 是原子半径
```

**定理 2：空间占有率**

```
空间占有率 = √2π/6 ≈ 74%
```

**定理 3：晶胞原子数**

```
每个 FCC 晶胞含有 4 个原子
```

---

### 典型例题

**例题：铜的面心立方结构**

```
题目：铜晶体为面心立方结构，铜原子半径 r = 1.28 Å。
      求铜晶胞的边长 a 和密度。
      （Cu 原子量 63.5，阿伏伽德罗常数 6.02×10²³）
```

**深层分析：**

```
(1) a = 2√2r = 2 × 1.414 × 1.28 ≈ 3.62 Å

(2) 每个晶胞有 4 个 Cu 原子
晶胞体积 V = a³ = (3.62×10⁻¹⁰ m)³ ≈ 4.75×10⁻²⁹ m³
质量 m = 4 × 63.5 / (6.02×10²³) ≈ 4.22×10⁻²⁵ kg
密度 ρ = m/V ≈ 8.89×10³ kg/m³
```

> **方法提炼：** FCC 晶胞 a = 2√2r 是关键关系。

### 方法详解：面心立方晶胞的计算

**Step 1: 确认是 FCC**

```
ABCABC 堆积 → FCC
```

**Step 2: 求边长**

```
a = 2√2r
```

**Step 3: 计算密度**

```
ρ = (原子数 × 原子量) / (阿伏伽德罗常数 × 晶胞体积)
```
