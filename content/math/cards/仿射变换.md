# 仿射变换

> 仿射变换是"把直线变成直线的几何变换"——它保持直线平行关系不变，但不保持角度和距离。

---

## 概念层

### 问题的起源

几何图形可以经过各种变换：
- 旋转：保持形状和大小
- 位似：保持形状，改变大小
- 仿射变换：保持"直线还是直线"、"平行线还是平行"

仿射变换是最一般的"线性几何变换"。

> **一句话：仿射变换把直线变成直线，把平行线变成平行线，但不保角不保距。**

---

### 本质定义

**仿射变换的一般形式：**

```
[x']   [a b] [x]   [e]
[y'] = [c d] [y] + [f]
```

用矩阵表示：x' = Ax + b

其中 A 是 2×2 可逆矩阵，b 是平移向量。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **直线→直线** | 直线经过变换后仍是直线 |
| **平行线保持** | 平行线变换后仍平行 |
| **不保持的** | 角度、距离、面积比例（某些不变量除外） |

---

### 不变量

| 不变量 | 说明 |
|--------|------|
| **直线平直性** | 直线还是直线 |
| **平行性** | 平行线仍是平行线 |
| **共线点比例** | 共线上三点的简单比例不变 |
| **直线与直线的相交关系** | 交点变换后仍是交点 |

---

## 命题层

### 核心定理

**定理：仿射变换的表示**

```
仿射变换 = 线性变换 + 平移
        = A·x + b
其中 A 是 2×2 可逆矩阵
```

**定理：行列式与面积比**

```
|A| = 变换后面积 / 变换前面积
面积比不变（比例不变）
```
