# 参数方程导数

> 参数方程导数是用参数 t 同时表达 x 和 y，通过"以 t 为中介"来求 dy/dx 的技巧。

---

## 概念层

### 问题的起源

有些曲线难以用 y = f(x) 直接表示，但可以用参数方程表示：
- 摆线：x = r(t − sin t), y = r(1 − cos t)
- 星形线：x = a cos³t, y = a sin³t
- 圆的参数方程：x = r cos t, y = r sin t

参数方程求导让我们能够在不消去参数的情况下，求曲线的切线斜率。

> **一句话：** 参数方程导数利用链式法则 dy/dx = (dy/dt) / (dx/dt)，以参数 t 为桥梁求导。

---

### 本质定义（逻辑重构版）

**参数方程求导公式：**

```
参数方程：x = φ(t), y = ψ(t)

则：dy/dx = (dy/dt) / (dx/dt) = ψ'(t) / φ'(t)    （前提：φ'(t) ≠ 0）
```

**几何理解：**

```
dy/dx = (dy/dt) / (dx/dt)
      = (dy/dt 的变化率) / (dx/dt 的变化率)
      = y 随 t 的变化快慢 / x 随 t 的变化快慢
      = 消去 t 后的 y 随 x 的变化快慢
```

---

### 二阶导数（参数方程形式）

```
d²y/dx² = d/dx (dy/dx) = d/dt (dy/dx) / (dx/dt)
        = [d/dt (ψ'(t)/φ'(t))] / φ'(t)
```

---

## 命题层

### 典型例题

**例题：求圆参数方程的切线斜率**

```
题目：圆 x = r cos t, y = r sin t，求 dy/dx。
```

**深层分析：**

```
dx/dt = −r sin t
dy/dt = r cos t

dy/dx = (dy/dt) / (dx/dt) = (r cos t) / (−r sin t) = −cot t

这正是圆的切线斜率
当 t = π/4 时，dy/dx = −cot(π/4) = −1
```

---

## 方法层

### 方法：参数方程求导流程

**标准操作流程：**

```
Step 1: 确认参数方程 x = φ(t), y = ψ(t)
Step 2: 分别求 dx/dt 和 dy/dt
Step 3: 计算 dy/dx = (dy/dt) / (dx/dt)
Step 4: （如需二阶导）再对 dy/dx 关于 t 求导，除以 dx/dt
Step 5: 需要时可代入 t 的值求具体斜率
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "dy/dx = (dx/dt) / (dy/dt)" | 搞反分子分母 | 是 (dy/dt) / (dx/dt)，顺序不能错 |
| 2 | "二阶导直接 d²y/dx² = ψ''(t)/φ''(t)" | 混淆公式 | d²y/dx² = [d/dt(dy/dx)] / (dx/dt)，需要再除一次 dx/dt |
