# 二项分布

> 二项分布是"n 次独立重复试验中成功次数"的分布——X ~ B(n, p)，P(X=k) = C(n,k)pᵏ(1-p)ⁿ⁻ᵏ。

---

## 概念层

### 问题的起源

抛硬币 10 次，每次正面概率 p = 0.5，求恰好 k 次正面的概率。

这是 n 次独立重复试验（每次试验只有两种结果：成功/失败）。

n 次独立重复试验中"成功次数"服从二项分布 B(n, p)。

> **一句话：二项分布是 n 次独立重复试验中成功次数的分布。**

---

### 本质定义

**二项分布的定义：**

```
X ~ B(n, p)
P(X=k) = C(n,k) × pᵏ × (1-p)ⁿ⁻ᵏ
其中 k = 0, 1, 2, ..., n
```

**参数：**

| 参数 | 含义 |
|------|------|
| n | 试验次数 |
| p | 每次试验成功的概率 |

---

### 与超几何分布的对比

| 对比项 | 二项分布 | 超几何分布 |
|--------|---------|-----------|
| 试验 | 独立重复（放回） | 不放回抽样 |
| 概率 | 不变 | 变化（抽一次少一个） |
| 近似 | 总体很大时，超几何近似二项 | — |

---

## 命题层

### 核心定理

**定理：二项分布的期望和方差**

```
X ~ B(n, p)
E(X) = np
Var(X) = np(1-p)
```

**定理：二项分布的可加性**

```
若 X ~ B(n₁, p)，Y ~ B(n₂, p)，且 X, Y 独立
则 X + Y ~ B(n₁ + n₂, p)
```

---

### 典型例题

**例题：二项分布概率计算**

```
题目：投掷一枚均匀硬币 10 次，求恰好 3 次正面的概率。
```

**深层分析：**

```
X ~ B(10, 0.5)
P(X=3) = C(10,3) × 0.5³ × 0.5⁷
        = 120 × 0.5¹⁰
        = 120/1024
        = 15/128 ≈ 0.117
```

> **方法提炼：** 直接代入公式 P(X=k) = C(n,k)pᵏ(1-p)ⁿ⁻ᵏ。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "二项分布和超几何分布一样" | 混淆 | 二项是放回，超几何是不放回 |
| 2 | "p 每次都变" | 误解 | 独立重复试验中 p 恒定 |
| 3 | "E(X) = np 对所有分布成立" | 混淆 | 只对二项分布成立 |
