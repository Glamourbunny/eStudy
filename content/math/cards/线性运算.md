# 线性运算

> 向量的线性运算，是"行走"的数学抽象——从 A 到 B，再从 B 到 C，等价于直接从 A 到 C。

---

## 概念层

### 问题的起源

在物理和日常生活中，"位移"是无法用单个数字描述的量。

一只蚂蚁从点 O 爬到点 A，位移是 **OA**；再从 A 爬到 B，位移是 **AB**。两次位移叠加，等价于直接从 O 到 C 的位移 **OC**。

如何用数学语言描述这种"叠加"？——这就需要向量的线性运算。

> **一句话：线性运算描述的是"先后行走"的叠加规则。**

---

### 原始定义（教材原文）

向量是既有**大小**又有**方向**的量。向量的加法、减法以及数乘运算，统称为向量的**线性运算**。

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 6-8 页

---

### 本质定义

线性运算的本质是向量的"行走叠加"：

```
位移 a：从 A 到 B
位移 b：从 B 到 C
──────────────
a + b：从 A 到 C（合成位移）
```

**三种基本操作：**

| 操作 | 物理意义 | 几何操作 |
|------|---------|---------|
| **加法** a + b | 两次位移的合成 | 首位相接，画出 a 的终点到 b 的终点的向量 |
| **减法** a − b | 从 a 中"去掉" b 的效果 | 转化为 a + (−b)，即 a 加上方向相反的 b |
| **数乘** k·a | 将位移"拉伸"或"压缩"并反向 | 同向 k>1 伸长，反向 k<0 缩短 |

---

### 定义的三要素

| 要素 | 加法 a + b | 减法 a − b | 数乘 k·a |
|------|-----------|-----------|---------|
| **对象** | 两个向量 | 两个向量 | 向量与实数 |
| **规则** | 首位相接 | a − b = a + (−b) | 同向延长/反向缩短 |
| **结果** | 仍为向量 | 仍为向量 | 仍为向量 |

---

### 等价表述

| 表述方式 | 加法 | 减法 | 数乘 |
|----------|------|------|------|
| **几何语言** | 平行四边形法则 / 三角形法则 | 转移向量 | 伸缩变换 |
| **坐标语言** | (x₁,y₁)+(x₂,y₂) = (x₁+x₂, y₁+y₂) | (x₁,y₁)−(x₂,y₂) = (x₁−x₂, y₁−y₂) | k(x,y) = (kx, ky) |
| **物理语言** | 合力等于各分力的矢量和 | 两速度之差 | 放大/缩小 |


### 概念辨析

**辨析 1：向量加法 vs 实数加法**

| 对比项 | 向量加法 a + b | 实数加法 a + b |
|--------|---------------|---------------|
| **比喻** | "走路"：先走 a，再走 b | "称重量"：把两份合在一起 |
| **结果** | 仍为向量（带方向） | 仍为标量（只有大小） |
| **交换律** | a + b = b + a（交换路线，结果相同） | a + b = b + a |
| **结合律** | (a + b) + c = a + (b + c) | (a + b) + c = a + (b + c) |
| **几何意义** | 两次位移的合成 | 两次增量的合并 |

**辨析 2：向量减法 vs 实数减法**

向量减法的本质是"找差距"：
- 已知 A 到 B 的向量 a 和 A 到 C 的向量 c
- 求 B 到 C 的向量

```
B ────a───→ A ───c───→ C
        ↓           ↑
         ───b───→   
```

B 到 C 的向量 b = c − a

> **一句话：** a − b 是"从 a 的终点看 b 的起点"的向量。

---

### 适用边界

| 条件 | 是否线性运算 | 说明 |
|------|------------|------|
| 对象都是向量 | ✓ | 向量 + 向量 = 向量 |
| 数乘实数 k | ✓ | k·向量 = 向量 |
| 向量 + 标量 | ✗ | 不能直接相加，单位不同 |
| 向量 · 向量（数量积） | ✗（结果不是线性运算） | 数量积是另一种运算 |

## 命题层

### 核心定理

**定理 1：向量加法的交换律与结合律**

```
a + b = b + a
(a + b) + c = a + (b + c)
```

**证明思路：** 由三角形法则的和平行四边形法则，几何意义清晰。

**定理 2：数乘的分配律**

```
(k + l)a = ka + la
k(a + b) = ka + kb
```

**定理 3：线性运算坐标化**

若 a = (x₁, y₁)，b = (x₂, y₂)，则：
```
a + b = (x₁ + x₂, y₁ + y₂)
a − b = (x₁ − x₂, y₁ − y₂)
k·a = (kx₁, ky₁)
```

---

### 典型例题

**例题 1：向量加法的几何应用**

```
题目：在平行四边形 ABCD 中，AB = a，AD = b。
      用 a, b 表示 AC 和 BD。
```

**深层分析：**

AC 是从 A 到 C 的对角线，相当于先走 AB 再走 BC，即 a + b。

BD 需要考虑方向：B 到 D = B 到 A + A 到 D = −a + b = b − a。

**答案：** AC = a + b；BD = b − a

---

**例题 2：数乘的线性组合**

```
题目：已知 a = (2, 1)，b = (1, 3)，求 3a − 2b。
```

**深层分析：**

按坐标运算：
3a = 3(2,1) = (6,3)
2b = 2(1,3) = (2,6)
3a − 2b = (6−2, 3−6) = (4, −3)

> **方法提炼：** 线性运算的坐标形式与实数运算完全兼容，直接按分量计算。

### 方法详解：向量线性运算的两大路径

**路径 1：几何法（三角形/平行四边形）**

适用场景：题目只给定了向量的起点和终点，没有坐标。

```
Step 1: 画出各向量，标清方向
Step 2: 根据法则是"首尾相接"（三角形）还是"共起点"（平行四边形）
Step 3: 找出待求向量的起点和终点
Step 4: 用已有向量表示（加减关系）
```

**路径 2：坐标法**

适用场景：已知向量的坐标表示。

```
Step 1: 将各向量写为坐标形式 (x, y)
Step 2: 按分量进行加减乘运算
Step 3: 写出结果的坐标形式
```

> **易错提醒：** 方向相反的向量相减容易出错，转化为加法（加上相反向量）更稳妥。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "向量加法和实数加法一样" | 忽视向量的方向性 | 向量加法结果仍带方向 |
| 2 | a − b = b − a | 照搬实数减法交换律 | 向量减法**不满足**交换律 |
| 3 | k 为负时不会处理 | 只会伸长不会缩 | 负数乘使向量反向 |
| 4 | 平行四边形法则画错 | 不清楚哪条是对角线 | 对角线是 a + b，不是 a − b |
| 5 | 分量运算时丢符号 | 粗心 | 减法 = 对应分量相减 |
