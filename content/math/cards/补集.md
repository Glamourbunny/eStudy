# 补集

> 补集是"在全集里，不是 A 的东西"——A 的补集是在全集 U 中不属于 A 的所有元素。

---

## 概念层

### 问题的起源

在一个确定的范围内（全集），"不是 A"的东西是什么？

比如在实数集 R 中，{x | x > 0} 的补集是 {x | x ≤ 0}。

补集让我们能够从"正面"思考转向"反面"思考。

> **一句话：补集 A∁ 是在全集 U 中不属于 A 的所有元素。**

---

### 原始定义（教材原文）

对于一个集合 A，在全集中所有不属于 A 的元素组成的集合称为 A 在全集 U 下的**补集**，记作 A∁。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 8 页

---

### 本质定义

**补集的本质是"排除"：**

```
A∁ = {x ∈ U | x ∉ A}
```

**文氏图表示：**

```
        全集 U
       ╱╲
      ╱  ╲
     ╱    ╲
    ╱──────╲
   ╱  A     ╲
  ╱ ██████   ╲
  ██████████
    ─────
    A 的补集（阴影部分）
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **全集 U** | 讨论的范围 |
| **条件** | x ∈ U 且 x ∉ A |
| **符号** | A∁ 或 C_U(A) |

### 概念辨析

**辨析：补集 vs 余集**

| 对比项 | 补集 | 余集 |
|--------|------|------|
| 相对性 | 相对于全集 U | 同一个概念 |
| 记号 | A∁ | C_U(A) |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| A ∪ A∁ = U | ✓ | 并集是全集 |
| A ∩ A∁ = ∅ | ✓ | 交集是空集 |
| (A∁)∁ = A | ✓ | 补两次回到自身 |

---

## 命题层

### 核心定理

**定理 1：德摩根定律**

```
(A ∩ B)∁ = A∁ ∪ B∁
(A ∪ B)∁ = A∁ ∩ B∁
```

**定理 2：补集的基本性质**

```
A ∪ A∁ = U
A ∩ A∁ = ∅
(A∁)∁ = A
```

---

### 典型例题

**例题：求补集**

```
题目：已知全集 U = {1, 2, 3, 4, 5}，A = {1, 2, 3}，求 A∁。
```

**深层分析：**

```
A∁ = {4, 5}
```

> **方法提炼：** 在全集中去掉 A 的元素。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "A∁ 与 A 无交集" | 正确 | A ∩ A∁ = ∅ |
| 2 | "(A∁)∁ = ∅" | 计算错误 | (A∁)∁ = A |
| 3 | "补集是空集" | 特殊混淆 | 只有 A = U 时 A∁ 才是空集 |
