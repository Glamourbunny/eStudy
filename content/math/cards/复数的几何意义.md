# 复数的几何意义

> 复数的几何意义是"复数对应复平面上的点或向量"——复数的加减乘除都有明确的几何解释。

---

## 概念层

### 问题的起源

复数 z = a + bi 在复平面上对应点 Z(a, b)，也对应向量 $\vec{OZ}$。

更重要的是，复数乘法对应**旋转和伸缩**：
- 乘以模为 1 的复数 e^(iθ) = cosθ + i·sinθ，实现旋转
- 乘以模为 r 的复数，实现伸缩

> **一句话：复数的几何意义是把代数运算翻译成几何变换（平移、旋转、伸缩）。**

---

### 本质定义

**复数运算的几何意义：**

| 运算 | 几何意义 |
|------|---------|
| 加减 | 向量平移（平行四边形） |
| 乘以实数 | 伸缩（同向或反向） |
| 乘以 e^(iθ) | 旋转（逆时针 θ） |
| 乘以 re^(iθ) | 旋转 + 伸缩 |
| 除法 | 旋转 + 收缩（乘以倒数的几何意义） |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **复平面点** | Z(a, b) |
| **向量表示** | $\vec{OZ}$ = (a, b) |
| **三角形式** | z = r(cosθ + i·sinθ) = re^(iθ) |

## 命题层

### 核心定理

**定理：复数乘法的几何意义**

```
z₁ = r₁e^(iθ₁)，z₂ = r₂e^(iθ₂)
z₁z₂ = r₁r₂e^(i(θ₁+θ₂))
模相乘，辐角相加
```

---

### 典型例题

**例题：用复数实现旋转**

```
题目：将点 P(1, 1) 绕原点逆时针旋转 90°。
```

**深层分析：**

```
旋转 90° = 乘以 e^(iπ/2) = i
z_P = 1 + i
z_P' = (1 + i) × i = i - 1 = -1 + i
对应点 P'(-1, 1)
```
