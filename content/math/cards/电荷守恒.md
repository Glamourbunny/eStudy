# 电荷守恒

> 电荷守恒是"化学反应中电荷不会凭空产生或消失"——它让配平的氧化还原方程式有了数学基础。

---

## 概念层

### 问题的起源

化学反应中，溶液往往含有离子。

例如：Fe³⁺ + S²⁻ → Fe₂S₃

反应前后的电荷如何？

Fe³⁺ 带 3 个正电荷，S²⁻ 带 2 个负电荷。
产物 Fe₂S₃ 中：2 个 Fe³⁺ 共 +6，3 个 S²⁻ 共 −6。

总电荷：+6 + (−6) = 0 —— 反应前后电荷总数相等！

这就是**电荷守恒**。

> **一句话：电荷守恒是反应前后溶液中正负电荷总数相等。**

---

### 本质定义

**电荷守恒的本质是"电荷守恒律"：**

```
反应前电荷总数 = 反应后电荷总数
Σ(正电荷) = Σ(负电荷)
```

**溶液中的电荷守恒：**

```
溶液中：Σ(阳离子所带正电荷) = Σ(阴离子所带负电荷)
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **正电荷** | 阳离子所带正电荷 |
| **负电荷** | 阴离子所带负电荷 |
| **守恒** | 反应前后电荷总数不变 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **电荷守恒** | 反应前后电荷总数相等 |
| **离子反应** | 溶液中阴阳离子电荷相等 |
| **氧化还原** | 氧化剂得电子数 = 还原剂失电子数 |

---

### 典型应用

**1. 配平氧化还原方程式：**

```
Cu + 2HNO₃(浓) → Cu(NO₃)₂ + NO₂↑ + H₂O
```

用氧化还原反应中电子转移数目相等来配平。

**2. 离子方程式电荷守恒：**

```
Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
左右电荷：−2 + 14×(+1) = 12
右边：2×(+3) = +6
不对——等等，应该是配平后相等...
```

### 概念辨析

**辨析：电荷守恒 vs 物料守恒**

| 对比项 | 电荷守恒 | 物料守恒 |
|--------|---------|---------|
| 守恒的是什么 | 电荷数 | 原子数 |
| 应用范围 | 离子反应、氧化还原 | 所有化学反应 |
| 表达式 | 电荷代数和为零 | 各原子数相等 |

## 命题层

### 核心定理

**定理：电荷守恒定律**

```
在任何化学反应中，反应前后电荷的代数和保持不变。
```

**推论：氧化还原反应的电子转移相等**

```
氧化剂得电子总数 = 还原剂失电子总数
```

---

### 典型例题

**例题：利用电荷守恒配平**

```
题目：配平以下氧化还原反应：
KMnO₄ + HCl → KCl + MnCl₂ + Cl₂ + H₂O
```

**深层分析：**

```
这是典型的氧化还原反应：
Mn: +7 → +2（得到5个电子）
Cl: -1 → 0（失去1个电子）

从 Mn 来看，需要 5 个 Cl 失去电子来平衡
所以方程系数初步为：
2KMnO₄ + 10HCl → ...
```

然后用电荷守恒和物料守恒继续配平。

> **方法提炼：** 电荷守恒用于确定氧化还原反应的系数。
