# 一级反应速率

> 一级反应速率是反应物的消耗速率与当前浓度成正比的化学反应模型。

---

## 概念层

### 问题的起源

在化学反应中，反应物浓度随时间变化。一级反应的特征是：反应速率与反应物浓度的一次方成正比。

一级反应广泛存在：
- 放射性衰变
- 药物代谢
- 酶促反应（米氏动力学近似）

> **一句话：** 一级反应的微分方程是 d[N]/dt = −k[N]，其中 k 是反应速率常数。

---

### 本质定义（逻辑重构版）

**一级反应微分方程：**

```
d[N]/dt = −k[N]

其中：
[N] —— 反应物浓度
k —— 反应速率常数（k > 0）
负号 —— 表示反应物在减少
```

**积分形式（解）：**

```
[N](t) = [N]₀ · e^(−kt)

即：浓度随时间指数衰减
```

---

### 一级反应的特征

| 特征 | 表达式 | 说明 |
|------|--------|------|
| 微分形式 | d[N]/dt = −k[N] | 速率与浓度成正比 |
| 积分形式 | [N] = [N]₀·e^(−kt) | 指数衰减 |
| 半衰期 | t₁/₂ = ln2/k | 与初始浓度无关 |
| 线性化 | ln[N] = ln[N]₀ − kt | ln[N] 与 t 成线性关系 |

---

## 命题层

### 典型例题

**例题：一级反应计算**

```
题目：某一级反应，k = 0.1 min⁻¹，初始浓度 [A]₀ = 1.0 mol/L。
求 (1) 10分钟后浓度 (2) 半衰期
```

**深层分析：**

```
(1) [A](10) = [A]₀ · e^(−kt) = 1.0 × e^(−0.1×10) = e^(−1) ≈ 0.368 mol/L

(2) t₁/₂ = ln2/k = 0.693/0.1 ≈ 6.93 min
```

---

## 方法层

### 方法：一级反应分析流程

**核心思想：** 利用指数衰减模型。

**标准操作流程：**

```
Step 1: 确认是一级反应（速率与浓度成正比）
Step 2: 写出微分方程 d[N]/dt = −k[N]
Step 3: 积分得到 [N](t) = [N]₀·e^(−kt)
Step 4: 代入已知条件求 k 或浓度
Step 5: 如需半衰期，用 t₁/₂ = ln2/k
```
