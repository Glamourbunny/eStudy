# 均值不等式

> 均值不等式是"算术平均大于等于几何平均"—— (a + b)/2 ≥ √(ab)，当 a = b 时取等。

---

## 概念层

### 问题的起源

两个正数 a 和 b：
- 算术平均：(a + b)/2
- 几何平均：√(ab)

哪个更大？

直觉上，(a + b)/2 是"平均分"，√(ab) 是"等面积替换"——算术平均应该更大。

数学上确实如此：(a + b)/2 ≥ √(ab)，等号当且仅当 a = b 时成立。

> **一句话：均值不等式说的是"算术平均 ≥ 几何平均"。**

---

### 原始定义（教材原文）

对于任意正数 a, b，有：

```
(a + b)/2 ≥ √(ab)
```

当且仅当 a = b 时，等号成立。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 30-32 页

---

### 本质定义

**均值不等式的变形链：**

```
(a + b)/2 ≥ √(ab)
       ↓
  (a + b)² ≥ 4ab
       ↓
  a² + 2ab + b² ≥ 4ab
       ↓
  a² − 2ab + b² ≥ 0
       ↓
  (a − b)² ≥ 0（显然成立）
```

所以均值不等式本质上是基本不等式的变形。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **a, b > 0** | 要求正数（几何平均需要平方根） |
| **算术平均** | (a + b)/2 |
| **几何平均** | √(ab) |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **标准形式** | (a + b)/2 ≥ √(ab) |
| **等价形式** | a + b ≥ 2√(ab) |
| **变形** | ab ≤ (a + b)²/4 |

---

### 典型应用

**求 ab 的最大值：**

当 a + b 固定为 S 时：
ab ≤ (a + b)²/4 = S²/4

等号当 a = b = S/2 时成立。

---

### 邻近概念网络

```
            不等式体系
                │
    ┌───────────┼───────────┐
    │           │           │
  基本不等式  均值不等式   柯西不等式
  a²+b²≥2ab  (a+b)/2≥√ab   (a₁²+...)(b₁²+...)≥(Σaᵢbᵢ)²
```

---

### 概念辨析

**辨析：均值不等式 vs 基本不等式**

| 对比项 | 均值不等式 | 基本不等式 |
|--------|-----------|-----------|
| 形式 | (a + b)/2 ≥ √(ab) | a² + b² ≥ 2ab |
| 限制 | a, b > 0 | a, b ∈ R |
| 联系 | 两者等价 | 可以互相推导 |

## 命题层

### 核心定理

**定理：均值不等式**

```
(a + b)/2 ≥ √(ab)，a, b > 0
等号成立 ⟺ a = b
```

**推广到 n 个数：**

```
(a₁ + a₂ + ... + aₙ)/n ≥ ⁿ√(a₁a₂...aₙ)
等号成立 ⟺ a₁ = a₂ = ... = aₙ
```

---

### 典型例题

**例题：利用均值不等式求最值**

```
题目：已知 a + b = 10，a > 0，b > 0，求 ab 的最大值。
```

**深层分析：**

```
ab ≤ ((a + b)/2)² = (10/2)² = 25
```

等号当 a = b = 5 时成立，所以 ab 的最大值为 25。

> **方法提炼：** 利用 ab ≤ ((a + b)/2)² 来求 ab 的最大值。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "均值不等式""(a+b)/2" | 均值不等式 | (a + b)/2 ≥ √(ab) |
| "ab 最大值" | 积的最大值 | ab ≤ ((a+b)/2)² |
| "a = b" | 取等条件 | 当 a = b 时取等 |

---

### 方法详解：均值不等式的应用

**Step 1: 判断 a, b 是否都是正数**

```
均值不等式要求 a > 0, b > 0
```

**Step 2: 凑出 (a + b) 或 ab**

```
已知和 → 求积的最大值
已知积 → 求和的最小值
```

**Step 3: 验证取等条件**

```
当 a = b 时取等
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "a, b 可以是负数" | 忽略条件 | 必须是正数 |
| 2 | "等号在 a = b 时成立" | 正确 | 验证是否满足 |
| 3 | "ab ≤ (a + b)²/4 和 ab ≤ (a² + b²)/2 混淆" | 记混 | 两个不等式不同但相关 |
