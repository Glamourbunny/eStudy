# 并集

> 并集是"加在一起"——A ∪ B 是属于 A 或者属于 B 的所有元素。

---

## 概念层

### 问题的起源

两个集合合在一起，会有什么元素？

{1, 2, 3} 和 {2, 3, 4} 合在一起，会得到 {1, 2, 3, 4}。

注意：重复的 {2, 3} 只会出现一次（集合的互异性）。

> **一句话：并集 A ∪ B 是属于 A 或者属于 B 的所有元素。**

---

### 原始定义（教材原文）

由属于集合 A 或属于集合 B 的所有元素组成的集合称为 A 与 B 的**并集**，记作 A ∪ B。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 7 页

---

### 本质定义

**并集的本质是"全部"：**

```
A ∪ B = {x | x ∈ A 或 x ∈ B}
```

### 概念辨析

**辨析：并集 vs 交集**

| 对比项 | 并集 A ∪ B | 交集 A ∩ B |
|--------|-----------|-----------|
| 定义 | x ∈ A 或 x ∈ B | x ∈ A 且 x ∈ B |
| 元素特征 | 在 A 或在 B 或在两者 | 既在 A 又在 B |
| 结果大小 | 通常较大 | 通常较小 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| A ∪ ∅ = A | ✓ | 与空集的并集是自身 |
| A ∪ A = A | ✓ | 与自身的并集是自身 |
| A ∪ B = B ∪ A | ✓ | 交换律 |

## 命题层

### 核心定理

**定理 1：并集的交换律**

```
A ∪ B = B ∪ A
```

**定理 2：并集的结合律**

```
(A ∪ B) ∪ C = A ∪ (B ∪ C)
```

**定理 3：并集与子集**

```
A ⊆ A ∪ B，B ⊆ A ∪ B
```

**定理 4：分配律**

```
A ∩ (B ∪ C) = (A ∩ B) ∪ (A ∩ C)
A ∪ (B ∩ C) = (A ∪ B) ∩ (A ∪ C)
```

---

### 典型例题

**例题：求并集**

```
题目：已知 A = {1, 2, 3, 4}，B = {3, 4, 5, 6}，求 A ∪ B。
```

**深层分析：**

```
A ∪ B = {1, 2, 3, 4, 5, 6}
```

注意 {3, 4} 只出现一次。

> **方法提炼：** 将两个集合的所有元素合并（去重）。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "并集""∪""所有" | 并集运算 | 合并元素，去重 |
| "A ∪ B" | 并集 | 将 A 和 B 的元素合在一起 |

---

### 方法详解：并集运算步骤

**Step 1: 列出两个集合的元素**

**Step 2: 合并所有元素**

**Step 3: 去掉重复，书写成集合形式**
