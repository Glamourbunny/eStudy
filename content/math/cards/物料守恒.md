# 物料守恒

> 物料守恒是"化学反应中原子种类和数量不变"——它是最基本的守恒律，也是配平化学方程式的根本。

---

## 概念层

### 问题的起源

化学反应不过是原子的重新组合。

2H₂ + O₂ → 2H₂O

反应前有 4 个 H 和 2 个 O，反应后还是 4 个 H 和 2 个 O。

原子不会凭空产生，也不会凭空消失。

这就是**物料守恒**（也称原子守恒、质量守恒）。

> **一句话：物料守恒是化学反应前后原子的种类和数量保持不变。**

---

### 本质定义

**物料守恒的本质是"原子守恒"：**

```
反应前原子总数 = 反应后原子总数
每一种原子的数量反应前后相等
```

**化学方程式配平就是基于这个原理：**

```
2H₂ + O₂ → 2H₂O
左边：H: 4, O: 2
右边：H: 4, O: 2
配平了！
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **原子种类** | 反应前后原子类型不变 |
| **原子数量** | 每种原子数量不变 |
| **守恒** | 方程式配平的基础 |

---

## 命题层

### 核心定理

**定理：物料守恒定律**

```
在任何化学反应中，反应前后原子的种类和数量保持不变。
```

**推论：化学方程式必须配平**

```
方程式配平的本质就是满足物料守恒
各原子数：左边 = 右边
```

---

### 典型例题

**例题：检验方程式是否配平**

```
题目：检验以下方程式是否配平：
2H₂ + O₂ → 2H₂O
```

**深层分析：**

```
左边原子数：
H: 2×2 = 4
O: 2

右边原子数：
H: 2×2 = 4
O: 2×1 = 2

左边 = 右边，配平了！
```

> **方法提炼：** 数原子数，检查是否相等。
