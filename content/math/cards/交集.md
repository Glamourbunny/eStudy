# 交集

> 交集是"共同的"——A ∩ B 就是同时属于 A 和属于 B 的元素。

---

## 概念层

### 问题的起源

两个集合放在一起，它们共同拥有的元素是什么？

{1, 2, 3} 和 {2, 3, 4} 放在一起，它们共同的是 {2, 3}。

这在生活中很常见：
- 既是物理学家又是数学家的人 → 两个领域的交集
- 会唱歌又会跳舞的人 → 两种技能的交集

> **一句话：交集 A ∩ B 是同时属于 A 和属于 B 的元素。**

---

### 原始定义（教材原文）

由属于集合 A 且属于集合 B 的所有元素组成的集合称为 A 与 B 的**交集**，记作 A ∩ B。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 7 页

---

### 本质定义

**交集的本质是"共同部分"：**

```
A ∩ B = {x | x ∈ A 且 x ∈ B}
```

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **条件** | x ∈ A 且 x ∈ B |
| **符号** | ∩ |
| **结果** | 还是一个集合 |

### 等价表述

| 表述 | 形式 |
|------|------|
| **符号** | A ∩ B |
| **描述** | A 与 B 的公共元素 |
| **性质** | A ∩ B ⊆ A，A ∩ B ⊆ B |

### 概念辨析

**辨析：交集 vs 并集**

| 对比项 | 交集 A ∩ B | 并集 A ∪ B |
|--------|-----------|-----------|
| 定义 | x ∈ A 且 x ∈ B | x ∈ A 或 x ∈ B |
| 元素特征 | 既在 A 又在 B | 在 A 或在 B 或在两者 |
| 结果大小 | 通常较小 | 通常较大 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| A ∩ ∅ = ∅ | ✓ | 与空集的交集是空集 |
| A ∩ A = A | ✓ | 与自身的交集是自身 |
| A ∩ B = B ∩ A | ✓ | 交换律 |

## 命题层

### 核心定理

**定理 1：交集的交换律**

```
A ∩ B = B ∩ A
```

**定理 2：交集的结合律**

```
(A ∩ B) ∩ C = A ∩ (B ∩ C)
```

**定理 3：交集与子集**

```
A ∩ B ⊆ A，A ∩ B ⊆ B
```

**定理 4：德摩根定律**

```
(A ∩ B)∁ = A∁ ∪ B∁
(A ∪ B)∁ = A∁ ∩ B∁
```

---

### 典型例题

**例题：求交集**

```
题目：已知 A = {1, 2, 3, 4}，B = {3, 4, 5, 6}，求 A ∩ B。
```

**深层分析：**

```
A ∩ B = {3, 4}
```

> **方法提炼：** 找两个集合的公共元素。
