# 分布列

> 分布列是"随机变量 X 取每个值时的概率清单"——它完整描述了离散型随机变量的概率分布。

---

## 概念层

### 问题的起源

抛硬币 3 次，设 X 为正面出现的次数。X 可以取 0, 1, 2, 3，对应的概率：

- P(X=0) = 1/8
- P(X=1) = 3/8
- P(X=2) = 3/8
- P(X=3) = 1/8

这就是 X 的**分布列**——把随机变量所有可能的取值和对应概率列出来。

> **一句话：分布列是随机变量 X 的"概率清单"，列出每个取值和它的概率。**

---

### 本质定义

**分布列的形式：**

```
X       x₁    x₂    ...    xₙ
P      p₁    p₂    ...    pₙ
```

其中 pᵢ = P(X = xᵢ)，且 Σpᵢ = 1。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **随机变量 X** | 取值随机的变量 |
| **可能取值** | x₁, x₂, ..., xₙ |
| **概率** | 每个取值的概率 pᵢ |

---

### 典型分布

| 分布 | 分布列 |
|------|--------|
| **二项分布 B(n,p)** | P(X=k) = C(n,k)pᵏ(1-p)ⁿ⁻ᵏ |
| **超几何分布** | P(X=k) = C(M,k)C(N-M,n-k)/C(N,n) |

---

## 命题层

### 核心定理

**定理：分布列的基本性质**

```
① pᵢ ≥ 0（概率非负）
② Σpᵢ = 1（概率总和为 1）
```

**定理：已知分布列求概率**

```
P(a ≤ X ≤ b) = Σ pᵢ（对所有 a ≤ xᵢ ≤ b 求和）
```

---

### 典型例题

**例题：写出二项分布的分布列**

```
题目：抛硬币 3 次，写出正面次数 X 的分布列。
```

**深层分析：**

```
X ~ B(3, 0.5)
P(X=k) = C(3,k) × 0.5ᵏ × 0.5³⁻ᵏ = C(3,k)/8

P(X=0) = C(3,0)/8 = 1/8
P(X=1) = C(3,1)/8 = 3/8
P(X=2) = C(3,2)/8 = 3/8
P(X=3) = C(3,3)/8 = 1/8

分布列：
X       0     1     2     3
P      1/8   3/8   3/8   1/8
```
