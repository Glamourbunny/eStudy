# 空间向量夹角

> 空间向量的夹角，就是三维空间中两个向量的"方向差异"——用数量积公式 cosθ = (a·b)/(|a||b|) 来计算。

---

## 概念层

### 问题的起源

在平面上，我们可以用数量积求两向量的夹角。在空间中，这个方法同样有效。

两个空间向量的夹角定义为：将两个向量平移到同一起点，它们所形成的 [0°, 180°] 范围内的角。

> **一句话：空间向量的夹角与平面向量完全相同——都是用数量积公式 cosθ = (a·b)/(|a||b|)。**

---

### 原始定义（教材原文）

已知空间向量 $\vec{a}$ 和 $\vec{b}$，它们的夹角 θ 满足：

```
cosθ = \frac{\vec{a}·\vec{b}}{|\vec{a}||\vec{b}|}
= \frac{a_x b_x + a_y b_y + a_z b_z}{\sqrt{a_x^2+a_y^2+a_z^2} \sqrt{b_x^2+b_y^2+b_z^2}}
```

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 10 页

---

### 本质定义

**夹角的本质是方向匹配程度：**

| cosθ 值 | θ 角 | 含义 |
|---------|------|------|
| cosθ = 1 | θ = 0° | 完全同向 |
| 0 < cosθ < 1 | 0° < θ < 90° | 锐角（有一定同向性） |
| cosθ = 0 | θ = 90° | 垂直 |
| −1 < cosθ < 0 | 90° < θ < 180° | 钝角（有一定反向性） |
| cosθ = −1 | θ = 180° | 完全反向 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **向量 a** | 第一个向量 |
| **向量 b** | 第二个向量 |
| **数量积** | a·b = a_xb_x + a_yb_y + a_zb_z |

---

### 等价表述

| 表述 | 公式 |
|------|------|
| **数量积形式** | cosθ = (a·b)/(\|a\|\|b\|) |
| **坐标形式** | cosθ = (a_xb_x + a_yb_y + a_zb_z) / (√(a_x²+a_y²+a_z²)·√(b_x²+b_y²+b_z²)) |
| **特殊情况** | a·b = 0 ⟺ a ⊥ b |

**辨析：空间向量夹角 vs 平面向量夹角**

| 对比项 | 平面向量夹角 | 空间向量夹角 |
|--------|-------------|-------------|
| 公式 | cosθ = (a·b)/(\|a\|\|b\|) | 完全相同 |
| 维度 | 二维 | 三维 |
| 计算 | 2 个分量 | 3 个分量 |

| 条件 | 是否成立 | 说明 |

| 两向量夹角范围 | [0°, 180°] | 余弦在这个范围单调递减 |
| a·b = 0 | ⟺ a ⊥ b（当 a,b ≠ 0） | 垂直判定 |

## 命题层

### 核心定理

**定理：夹角余弦公式**

```
cosθ = \frac{\vec{a}·\vec{b}}{|\vec{a}||\vec{b}|}
```

**特例：垂直判定**
```
\vec{a}·\vec{b} = 0 ⟺ \vec{a} ⊥ \vec{b}（当 \vec{a},\vec{b} ≠ \vec{0}）
```

---

### 典型例题

**例题：求两向量夹角**

```
题目：已知 \vec{a} = (1, 2, 2)，\vec{b} = (2, 3, 1)，求它们的夹角。
```

**深层分析：**

```
Step 1: 计算数量积
a·b = 1×2 + 2×3 + 2×1 = 2 + 6 + 2 = 10

Step 2: 计算模长
|a| = √(1+4+4) = √9 = 3
|b| = √(4+9+1) = √14

Step 3: 计算 cosθ
cosθ = 10/(3×√14) = 10/(3√14)

Step 4: 求 θ
θ = arccos(10/(3√14)) ≈ ...
```

> **方法提炼：** 三个步骤：数量积→模长→cosθ→arccos。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "夹角""求θ" | 角度问题 | cosθ = (a·b)/(\|a\|\|b\|) |
| "垂直""正交" | 垂直判定 | a·b = 0 |
| "锐角""钝角" | 角的范围 | 看 cosθ 的符号 |

---

### 方法详解：求夹角的步骤

```
Step 1: 计算 a·b = a_xb_x + a_yb_y + a_zb_z
Step 2: 计算 |a| = √(a_x²+a_y²+a_z²)
Step 3: 计算 |b| = √(b_x²+b_y²+b_z²)
Step 4: 计算 cosθ = (a·b)/(|a||b|)
Step 5: θ = arccos(cosθ)（注意取值范围 [0°, 180°]）
```

> **易错提醒：** 夹角范围是 [0°, 180°]，不是 [0°, 360°]。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "cosθ 为负则无解" | 误解 | cosθ 为负只说明 θ > 90°，解仍然存在 |
| 2 | "夹角范围是 0°~360°" | 二维思维 | 空间向量夹角范围是 [0°, 180°] |
| 3 | "用 sin 代替 cos" | 记混 | 必须用 cosθ 公式 |
