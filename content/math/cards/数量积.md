# 数量积

> 数量积不是"乘法"，而是"做功"的数学抽象——力在位移方向上做了多少功，看的不是力有多大，而是力在推动方向上"真正出了多少力"。

---

## 概念层

### 问题的起源

物理中有一个经典问题：一个人用斜向下的力拉车，力的大小是 F，位移是 S，如何计算这个力做了多少功？

显然，不能简单用 F × S。如果力完全沿位移方向，功 = FS；如果力完全垂直于位移方向，力根本不做功。

这说明：**功的大小不仅取决于力和位移的大小，还取决于它们之间的夹角**。

数量积（也称点积、内积）正是为了描述这种"方向匹配程度"而诞生的数学工具。

> **一句话：数量积度量的是两个向量在"同向程度"上的乘积。**

---

### 原始定义（教材原文）

已知两个非零向量 a 和 b，它们的夹角为 θ，则 a 与 b 的**数量积**（或**点积**）定义为：

```
a · b = |a| · |b| · cosθ
```

其中 |a|、|b| 是向量的模，θ 是夹角（0° ≤ θ ≤ 180°）。

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 9 页

---

### 本质定义

数量积的本质是"投影相乘"：

```
a · b = (|a| cosθ) · |b|
      = a 在 b 方向上的投影 × |b|
      = b 在 a 方向上的投影 × |a|
```

**数量积的几何含义：**

| 视角 | 含义 |
|------|------|
| 投影视角 | a 在 b 方向上的"影子长度"乘以 b 的长度 |
| 做功视角 | 力在位移方向上实际做功的大小 |
| 夹角视角 | 衡量两个向量的"同向程度"（cosθ 越大越同向） |

---

### 定义的三要素

| 要素 | 说明 | 核心问题 |
|------|------|---------|
| **向量模长** | |a| 和 |b| 的大小 | "两个向量有多长？" |
| **夹角 θ** | 两个向量的夹角 | "它们有多'合作'？" |
| **cosθ** | 夹角的余弦 | "同向为正，垂直为0，反向为负" |

**cosθ 的含义：**

| θ | cosθ | 数量积符号 | 几何含义 |
|---|------|-----------|---------|
| θ = 0° | 1 | 正最大 | 完全同向 |
| 0° < θ < 90° | > 0 | 正 | 锐角，有正向贡献 |
| θ = 90° | 0 | 零 | 垂直，互相不做功 |
| 90° < θ < 180° | < 0 | 负 | 钝角，有反向贡献 |
| θ = 180° | −1 | 负最大 | 完全反向 |

---

### 等价表述

| 表述方式 | 公式 | 适用场景 |
|----------|------|---------|
| **几何定义** | a · b = |a||b|cosθ | 已知夹角和模长 |
| **坐标形式** | (x₁,y₁)·(x₂,y₂) = x₁x₂ + y₁y₂ | 已知坐标 |
| **物理意义** | F·S = |F||S|cosθ（功） | 力学中的功 |
| **投影形式** | a·b = (|a|cosθ)·|b| | 几何分析 |

**坐标形式的推导：**

设 a = (x₁, y₁)，b = (x₂, y₂)，则：
```
a · b = x₁x₂ + y₁y₂
```

这是高中阶段最常用的计算形式。

### 邻近概念网络

```
                   向量运算
                       │
         ┌─────────────┼─────────────┐
         │             │             │
     线性运算       数量积         向量积
    (加减、数乘)    (点积)         (叉积)
                     │
            ┌────────┴────────┐
            ▼                 ▼
        a·b = |a||b|cosθ  坐标：(x₁y₁)·(x₂y₂)=x₁x₂+y₁y₂
```

**与线性运算的关系：**
- 线性运算结果仍是向量
- 数量积结果是一个**标量**（数）
- 数量积不满足结合律：(a·b)·c ≠ a·(b·c)

---

### 概念辨析

**辨析 1：数量积 vs 线性乘法**

| 对比项 | 数量积 a·b | 小学乘法的 a×b |
|--------|-----------|----------------|
| **对象** | 两个向量 | 两个数（标量） |
| **结果类型** | 标量 | 标量 |
| **几何依赖** | 依赖夹角 | 与方向无关 |
| **交换律** | a·b = b·a ✓ | a×b = b×a ✓ |
| **结合律** | (a·b)·c ≠ a·(b·c) ✗ | (a×b)×c = a×(b×c) ✓ |

> **一句话：** 数量积和普通乘法**结果都是数**，但数量积与向量的方向有关，普通乘法与方向无关。

**辨析 2：数量积 vs 向量积（叉积）**

| 对比项 | 数量积（点积） | 向量积（叉积） |
|--------|--------------|--------------|
| **符号** | a · b | a × b |
| **结果** | 标量（数） | 向量（既有大小又有方向） |
| **几何意义** | 投影相乘，度量"同向程度" | 垂直于原两向量，度量"平行四边形面积" |
| **物理对应** | 功 | 力矩 |
| **交换律** | a·b = b·a | a×b = −b×a |

> **高中不要求叉积，仅作拓展了解。**

**辨析 3：数量积的正负**

数量积的正负完全由夹角决定：

| 数量积符号 | 夹角范围 | 含义 |
|-----------|---------|------|
| a·b > 0 | θ ∈ [0°, 90°) | 两向量夹角为锐角，互相"帮助" |
| a·b = 0 | θ = 90° | 两向量垂直，互不影响 |
| a·b < 0 | θ ∈ (90°, 180°] | 两向量夹角为钝角，互相"阻碍" |

> **一句口诀：** "正数帮忙，零是无关，负数帮倒忙。"

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 两向量都是非零向量 | ✓ | 零向量的数量积定义为 0 |
| 夹角 0° ≤ θ ≤ 180° | ✓ | 任意两向量都有夹角 |
| 一向量为零向量 | ✓，结果为 0 | 零向量与任何向量的数量积都是 0 |
| 数量积为 0 | 可能夹角 90°，也可能某向量为零 | 需分类讨论 |

**常见陷阱：**

| 陷阱 | 错误理解 | 正确理解 |
|------|---------|---------|
| a·b=0 ⇒ a⊥b | 忽略零向量情况 | 必须说明 a≠0 且 b≠0 |
| cosθ=1 ⇒ a=b | 方向相同，但模长可能不同 | cosθ=1 只保证同向，|a|=|b| 才保证相等 |
| 数量积满足结合律 | 照搬实数乘法 | (a·b)·c 是标量乘向量，与 a·(b·c) 不同 |

## 命题层

### 核心定理

**定理 1：数量积的坐标公式**

若 a = (x₁, y₁)，b = (x₂, y₂)，则：
```
a · b = x₁x₂ + y₁y₂
```

> 这是高中计算数量积的**主要工具**。

**证明思路：** 将向量写成坐标形式，利用向量坐标与模长、夹角的关系推导。

**定理 2：向量的垂直判定**

a ⊥ b ⟺ a · b = 0（当 a ≠ 0, b ≠ 0 时）

> 逆定理也成立，是判断垂直的重要方法。

**定理 3：数量积的运算律**

```
① 交换律：a · b = b · a
② 数乘结合律：(k·a) · b = k(a · b) = a · (k·b)
③ 分配律：(a + b) · c = a · c + b · c
```

> 注意：**不满足结合律**！(a·b)·c ≠ a·(b·c)

**定理链：**

```
向量坐标表示
       ↓
数量积坐标公式：a·b = x₁x₂ + y₁y₂
       ↓
模长公式：|a| = √(x₁² + y₁²)
       ↓
夹角公式：cosθ = (x₁x₂ + y₁y₂) / (|a||b|)
       ↓
垂直判定：a·b = 0 ⟺ a ⊥ b
       ↓
平行判定：a//b ⟺ θ = 0° 或 180° ⟺ cosθ = ±1
```

---

### 典型例题的深层分析

**例题 1：利用数量积求模长**

```
题目：已知 a = (3, 4)，求 |a|。
```

**深层分析：**

|a|² = a · a = (3,4)·(3,4) = 3² + 4² = 9 + 16 = 25

∴ |a| = √25 = 5

> **方法提炼：** 模长的本质是自身与自身的数量积的平方根。这是求复杂向量模长的通法。

---

**例题 2：求两向量夹角**

```
题目：已知 a = (1, 2)，b = (−2, 1)，求 a 与 b 的夹角。
```

**深层分析：**

先算 a·b = 1×(−2) + 2×1 = 0

发现 a·b = 0，所以 cosθ = 0，θ = 90°

> **快速判断：** 坐标分量相乘后相加为 0，说明两向量垂直。

如果 a·b ≠ 0，则：
```
cosθ = (a·b) / (|a||b|)
     = (x₁x₂ + y₁y₂) / (√(x₁²+y₁²) · √(x₂²+y₂²))
```

---

**例题 3：利用数量积判断平行与垂直**

```
题目：判断以下向量是否平行或垂直：
(1) a = (2, 3)，b = (4, 6)
(2) c = (1, −1)，d = (3, 3)
```

**深层分析：**

**(1) 平行判断：**

计算 a·b = 2×4 + 3×6 = 8 + 18 = 26

|a| = √(4+9) = √13
|b| = √(16+36) = √52 = 2√13

cosθ = 26/(√13·2√13) = 26/(2·13) = 1

∴ θ = 0°，a 与 b **平行**（实际上是同向的，因为 cosθ = 1）

**(2) 垂直判断：**

c·d = 1×3 + (−1)×3 = 3 − 3 = 0

∴ c ⊥ d（两向量**垂直**）

> **方法提炼：**
> - 垂直：检查 a·b = 0
> - 平行：检查 cosθ = ±1，或检查 x₁y₂ = x₂y₁（即分量成比例）

---

**例题 4：数量积在物理中的应用**

```
题目：一个人用 30N 的力斜向下拉车，力的方向与水平方向成 30° 角，
      车水平位移了 10m。求拉力做的功。
```

**深层分析：**

这是数量积的物理原型——功 = F · S = |F||S|cosθ

```
W = F · S = 30 × 10 × cos30°
     = 300 × (√3/2)
     = 150√3 焦耳
     ≈ 259.8 焦耳
```

> **方法提炼：** 物理中的"功"就是力和位移的数量积。一切涉及"一个矢量在另一个方向上做多少贡献"的问题，都是数量积。

### 方法详解：数量积计算三步法

**Step 1：判断已知条件类型**

| 类型 | 处理方式 |
|------|---------|
| 已知两向量坐标 | 用坐标公式 a·b = x₁x₂ + y₁y₂ |
| 已知模长和夹角 | 用定义 a·b = \|a\|\|b\|cosθ |
| 求向量模长 | 用 \|a\|² = a·a |

**Step 2：选择公式计算**

- 垂直判定：检查 x₁x₂ + y₁y₂ 是否为 0
- 夹角公式：cosθ = (x₁x₂ + y₁y₂) / (√(x₁²+y₁²)·√(x₂²+y₂²))

**Step 3：解释几何意义**

- 根据正负判断是锐角/钝角/垂直
- 根据 cosθ 的具体值判断同向/反向程度

> **易错提醒：**
> 1. 数量积是**标量**，不是向量
> 2. a·b=0 不能直接推出 a⊥b，需排除零向量情况
> 3. 夹角范围是 [0°, 180°]，cos 在这个范围内单调递减

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "a·b=0 则 a⊥b" | 忽略零向量 | 需 a≠0 且 b≠0 |
| 2 | "数量积满足结合律" | 照搬实数乘法 | (a·b)·c 是数乘向量，不是数量积 |
| 3 | "cosθ 越大夹角越大" | 误以为 cos 是单调增 | cos 在 [0°, 180°] 单调递减 |
| 4 | "a·b = \|a\|\|b\| 则 a=b" | 忽略方向 | 只保证夹角为 0°，不保证模长相等 |
| 5 | 求模长直接开方 | 忘检验正负 | \|a\| = √(a·a) ≥ 0 |
