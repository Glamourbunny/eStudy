# 期望

> 期望是随机变量的"平均值"——E(X) = Σ xᵢ pᵢ，是所有可能值的加权平均。

---

## 概念层

### 问题的起源

抛硬币 3 次，正面次数 X 的分布列：
- X=0, 1, 2, 3
- 概率 1/8, 3/8, 3/8, 1/8

平均正面次数是多少？

不是 (0+1+2+3)/4 = 1.5，而是考虑概率加权的：
E(X) = 0×1/8 + 1×3/8 + 2×3/8 + 3×1/8 = 1.5

这正好是二项分布的期望 np = 3×0.5 = 1.5。

> **一句话：期望是随机变量所有可能值的概率加权平均。**

---

### 本质定义

**期望的本质是"概率加权平均"：**

```
E(X) = Σ 所有取值 × 该取值的概率
     = x₁p₁ + x₂p₂ + ... + xₙpₙ
```

**物理类比：** 把概率当作"质量"，期望就是"质心位置"。

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **取值 xᵢ** | 随机变量的所有可能取值 |
| **概率 pᵢ** | 每个取值对应的概率 |
| **加权求和** | Σ xᵢ pᵢ |

---

### 性质

| 性质 | 公式 |
|------|------|
| 线性 | E(aX + b) = aE(X) + b |
| 再生性（独立） | E(X + Y) = E(X) + E(Y) |

## 命题层

### 核心定理

**定理：期望公式**

```
E(X) = Σ xᵢ pᵢ
```

**定理：常用分布的期望**

```
① X ~ B(n,p)：E(X) = np
② X ~ 超几何：E(X) = n × K/N
③ X ~ 几何分布：E(X) = 1/p
```

---

### 典型例题

**例题：计算期望**

```
题目：掷骰子，X 为点数，求 E(X)。
```

**深层分析：**

```
X 的分布列：
X       1     2     3     4     5     6
P      1/6   1/6   1/6   1/6   1/6   1/6

E(X) = 1×1/6 + 2×1/6 + ... + 6×1/6
     = (1+2+3+4+5+6)/6
     = 21/6 = 3.5
```

> **方法提炼：** E(X) = Σ xᵢ pᵢ。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "期望一定是样本平均值" | 混淆 | 期望是理论值，不一定等于样本均值 |
| 2 | "E(X+Y) = E(X)+E(Y) 永远成立" | 误解 | 独立时才成立 |
| 3 | "期望一定在取值范围内" | 正确 | E(X) 通常在 min 和 max 之间 |
