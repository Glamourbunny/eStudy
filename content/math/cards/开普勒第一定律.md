# 开普勒第一定律

> 开普勒第一定律指出：行星绕太阳运动的轨道是椭圆——而不是圆。这一发现颠覆了人们认为天体运动是"完美圆形"的千年执念。

---

## 概念层

### 问题的起源

在开普勒之前，人们普遍相信：
- 托勒密体系：地心说，圆轨道
- 哥白尼体系：日心说，但仍是圆轨道

然而观测数据与圆轨道预测不符。开普勒分析了第谷的观测数据10年后发现：

**行星的轨道不是圆，而是椭圆。太阳不在圆心，而在椭圆的一个焦点上。**

> **一句话：开普勒第一定律是"行星绕太阳的轨道是椭圆，太阳在椭圆的一个焦点上"。**

---

### 原始定义（教材原文）

开普勒第一定律（椭圆定律）：
行星绕太阳运动的轨道是椭圆，太阳位于椭圆的一个焦点上。

> 出处：人教版《普通高中教科书·物理（必修第二册）》第十章

---

### 本质定义

**椭圆轨道的特点：**

```
              远日点
             ╱
            ╱
           ╱
    ●S    ╱    ●P（行星）
          ╲
           ╲
            ╲
             ╲
              近日点
```

- 太阳在焦点 S，不是椭圆中心
- 行星在近日点时离太阳最近，速度最快
- 行星在远日点时离太阳最远，速度最慢

**开普勒三定律的整体：**

| 定律 | 内容 | 数学表达 |
|------|------|---------|
| 第一定律 | 椭圆轨道 | 焦点在太阳 |
| 第二定律 | 面积速度相等 | 相同时间扫过相同面积 |
| 第三定律 | 周期与半长轴关系 | T²/a³ = 常数 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **椭圆轨道** | 行星轨迹是椭圆，不是圆 |
| **太阳在焦点** | 太阳不在椭圆中心，在焦点上 |
| **近日点/远日点** | 距离最近/最远的点 |

---

### 等价表述

| 表述 | 说明 |
|------|------|
| **椭圆轨道** | r₁ + r₂ = 2a = 常数 |
| **近日点距离** | r_min = a − c |
| **远日点距离** | r_max = a + c |

其中 a 是长半轴，c 是半焦距，a² = b² + c²。

| 对比项 | 椭圆轨道 | 圆轨道 |
|--------|---------|--------|

## 命题层

### 核心定理

**定理 1：开普勒第一定律（椭圆定律）**

```
行星轨道是椭圆，太阳在椭圆的一个焦点上。
```

**定理 2：近日点/远日点公式**

```
近日点距离：r_min = a − c
远日点距离：r_max = a + c
其中 a 是长半轴，c 是半焦距，c = ae（e 是离心率）
```

**定理 3：开普勒第三定律的椭圆形式**

```
T²/a³ = 4π²/GM = 常数（对同一中心天体）
```

---

### 典型例题

**例题：地球轨道的近日点和远日点**

```
题目：地球绕太阳的轨道可近似为圆，但实际是椭圆。
      已知地球的近日点距离约为 1.47×10¹¹ m，远日点距离约为 1.52×10¹¹ m。
      求地球轨道的半长轴 a 和半焦距 c。
```

**深层分析：**

```
r_min = a − c = 1.47×10¹¹
r_max = a + c = 1.52×10¹¹

两式相加：2a = 3.99×10¹¹ ⟹ a ≈ 1.50×10¹¹ m
两式相减：2c = 0.05×10¹¹ ⟹ c ≈ 0.025×10¹¹ = 2.5×10⁹ m

离心率 e = c/a ≈ 2.5×10⁹/1.50×10¹¹ ≈ 0.017
（地球轨道非常接近圆）
```

> **方法提炼：** 近日点+远日点 → 求 a, c。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "椭圆轨道""焦点" | 开普勒第一定律 | 用椭圆性质 |
| "近日点""远日点" | 距离问题 | r_min = a − c，r_max = a + c |
| "面积速度" | 开普勒第二定律 | 等面积原理 |

---

### 方法详解：椭圆轨道问题的处理流程

**Step 1: 判断考的是哪个开普勒定律**

```
椭圆轨道 → 第一定律
面积速度 → 第二定律
周期关系 → 第三定律
```

**Step 2: 第一定律问题转化为椭圆参数**

```
近日点/远日点 → 求 a, c
已知 r_min, r_max：
a = (r_min + r_max)/2
c = (r_max − r_min)/2
```
