# 定义域

> 定义域是函数三要素的"输入端"——回答"谁可以进来"的问题。

---

## 概念层

### 问题的起源

有了函数概念"输入→规则→输出"，紧接着要问：**x 能取哪些值？**

现实问题天然有限制：
- 物理：行程时间 t ≥ 0
- 经济：价格不能为负
- 数学运算：分母不能为 0，负数不能开平方

这些"x 能取哪些值"的限制条件，就是**定义域**存在的理由。

> **一句话：** 定义域是使函数表达式有意义的所有 x 的集合。

---

### 原始定义（教材原文）

设 A、B 是非空的数集，如果按照某种确定的对应关系 f，使对于集合 A 中的任意一个数 x，在集合 B 中都有唯一确定的数 f(x) 和它对应，那么就称 f：A → B 为从集合 A 到集合 B 的函数。

其中**集合 A 称为函数的定义域**。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 62 页

**定义拆解：**

| 关键词 | 含义 | 常见陷阱 |
|--------|------|---------|
| 非空数集 | 定义域不能是空集 | ∅ 不是合法定义域 |
| "任意一个" | 每个 x 都必须有对应 | 漏掉某些 x 就违反任意性 |
| "唯一确定" | 一个 x 不能对应多个 y | 违反则不是函数 |

---

### 本质定义（逻辑重构版）

```
定义域 = { x ∈ R | 表达式 f(x) 有意义 }

"有意义"的判断标准：
  ├─ 分式：分母 ≠ 0
  ├─ 偶次根式：被开方数 ≥ 0
  ├─ 对数：真数 > 0，底数 > 0 且 ≠ 1
  ├─ 零次幂：底数 ≠ 0
  └─ 正切：角度 ≠ π/2 + kπ
```

---

### 定义域的类型

| 类型 | 描述 | 示例 |
|------|------|------|
| **自然定义域** | 使解析式本身有意义的所有 x | f(x) = √(1−x) → x ≤ 1 |
| **指定定义域** | 题目额外限制的范围 | f(x) = x²，x ∈ [0, 1] |
| **实际定义域** | 由实际问题的意义决定 | 人数 x ≥ 0 |

> **关键：** 先求自然定义域，再叠加实际限制。

---

### 邻近概念辨析

**定义域 vs 值域 —— "入口" vs "出口"**

| 对比项 | 定义域 | 值域 |
|--------|--------|------|
| 角色 | 输入端（谁可以进来） | 输出端（最终落在哪） |
| 决定因素 | 解析式本身的限制 | 定义域 + 对应法则共同决定 |
| 求法 | 解不等式组 | 往往需要反解或配方法 |
| 常见问题 | 漏条件、忘负号 | 不知道值域受定义域限制 |

> **记忆口诀：** 定义域是"白名单"——只有符合条件才能进入；值域是"结果区"——经过规则后实际输出的范围。

---

### 适用边界

| 条件 | 是否存在问题 | 典型错误 |
|------|-------------|---------|
| 分式定义域 | 排除分母 = 0 的点 | f(x)=1/(x−2) 忘了 x≠2 |
| 复合函数定义域 | 内层值域要作为外层定义域 | f(x)=ln(sin x)，先求 sin x>0 |
| 分段函数定义域 | 各段定义域取并集 | 分段讨论时忘记叠加 |
| 实际应用题定义域 | 不仅要数学有意义，还要实际合理 | 人数不能为负，分数不能超100 |

## 命题层

### 核心定理

**定理：复合函数的定义域**

若函数 f(x) 的定义域为 A，g(x) 的值域为 B，则复合函数 f(g(x)) 的定义域为：

```
D = { x ∈ A | g(x) ∈ B }
```

即：内层函数 g(x) 的输出必须落在外层函数 f 的定义域内。

**典型例题：**

已知 f(x) 的定义域为 [0, 1]，求 f(2x−1) 的定义域。

**分析：** f(2x−1) 相当于内层 g(x)=2x−1 代入外层 f。

```
设 u = 2x − 1
由 f(u) 的定义域为 [0, 1]
即 0 ≤ u ≤ 1
即 0 ≤ 2x − 1 ≤ 1
解得：1/2 ≤ x ≤ 1
```

> **方法提炼：** "代换法"——设 u = g(x)，解 0 ≤ u ≤ 1，得 x 的范围。

## 方法层

### 方法：求定义域——"列禁区，取交集"

**核心思想：** 定义域是"安全 x 的集合"，求定义域就是**找出所有不安全的禁区，然后取补集**。

**标准操作流程：**

```
Step 1: 扫描解析式，列出所有禁区
         ├─ 分式：分母 = 0
         ├─ 偶次根式：被开方数 < 0
         ├─ 对数：真数 ≤ 0，底数 ≤ 0 或 = 1
         ├─ 零次幂：底数 = 0
         └─ 正切：角度 = π/2 + kπ

Step 2: 解不等式，确定每类禁区的范围

Step 3: 取所有禁区的并集 S（所有不安全的地方）

Step 4: 定义域 = R \ S（即取补集）
```

**决策树：**

```
表达式含有什么？
    │
    ├─ 分式 f(x)/g(x) → 令 g(x) = 0，得 x₀ → 排除 x₀
    │
    ├─ 偶次根式 √f(x) → 令 f(x) < 0 → 解得 x 范围
    │
    ├─ 对数 log_a f(x) → 令 f(x) ≤ 0；再验证 a > 0 且 a ≠ 1
    │
    ├─ 零次幂 [f(x)]⁰ → 令 f(x) = 0 → 排除该 x
    │
    └─ 复合函数 → 先求内层值域，再作为外层输入限制
```

**例题应用：**

求 f(x) = ln(x−1) / √(3−x) 的定义域。

```
禁区 1：对数 → x − 1 > 0  →  x > 1
禁区 2：根号 → 3 − x ≥ 0  →  x ≤ 3
禁区 3：分式 → 分母 √(3−x) ≠ 0  →  3 − x ≠ 0  →  x ≠ 3

综合：x > 1 且 x ≤ 3 且 x ≠ 3

答案：定义域为 (1, 3)
```

> **易错提醒：** 多个限制条件取**交集**，不是并集！对数和根号同时出现时，要分别解不等式再取交集。

### 常见题型模板

| 题型 | 识别信号 | 操作步骤 |
|------|---------|---------|
| 简单解析式 | 单一运算（分式/根式/对数） | 直接列不等式 |
| 复合函数 | f(g(x)) 形式 | 换元：令 u=g(x)，解 0≤u≤D_f |
| 分段函数 | 不同区间不同表达式 | 各段定义域取并集 |
| 应用题 | 有实际背景 | 数学定义域 ∩ 实际合理范围 |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "定义域就是 x 的范围，不用管 y" | 忽略分式分母不为零 | 分式定义域必须排除分母=0 |
| 2 | "ln(x²−1) 的定义域是 x²−1>0" | 对真数理解不完整 | 真数大于零，不是大于等于零 |
| 3 | "复合函数定义域就是内层定义域" | 混淆了内层输出和外层输入 | 要看内层值域是否在外层定义域内 |
| 4 | "化简后定义域会变大" | 忽略化简可能产生额外限制 | 求定义域必须用**原始解析式** |
| 5 | "忽略底数的限制" | 忘对对数底数分类 | 底数需满足 a>0 且 a≠1 |
