# 双曲线

> 双曲线是"到两焦点距离之差的绝对值为常数的点的轨迹"——它有两条分开的曲线分支，在光学和工程中有独特应用。

---

## 概念层

### 问题的起源

双曲线与椭圆一样，来自对圆锥截线的研究。当用一个平面斜切圆锥时，可能得到双曲线。

双曲线的特点：
- 有**两支**分开的曲线
- 椭圆是距离"之和"为常数，双曲线是距离"之差"的绝对值为常数
- 焦点在**两支曲线的内部**

工程应用：
- 双曲线冷却塔的建筑形状（基于双曲线的力学稳定性）
- 光学中的双曲面反射镜（焦点特性）

> **一句话：双曲线是"到两焦点距离之差的绝对值恒定"的点的轨迹。**

---

### 原始定义（教材原文）

平面内到两个定点 F₁、F₂ 距离之差的绝对值等于常数 2a（2a < |F₁F₂|）的点的轨迹称为**双曲线**。

两个定点 F₁、F₂ 称为**焦点**。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 60-65 页

---

### 本质定义

**双曲线的标准方程：**

```
焦点在 x 轴上：x²/a² − y²/b² = 1
焦点在 y 轴上：y²/a² − x²/b² = 1
```

其中 a 是**实半轴**，b 是**虚半轴**，c = √(a² + b²) 是**半焦距**。

**焦点性质：**

||PF₁| − |PF₂|| = 2a（对双曲线上任意点 P）

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **焦点 F₁, F₂** | 两定点，距离之差的绝对值为 2a |
| **实半轴 a** | a 是实轴的一半（与椭圆不同，此处 a 可以小于 b） |
| **虚半轴 b** | 满足 c² = a² + b² |
| **半焦距 c** | c = √(a² + b²)，焦点到中心的距离 |

---

### 等价表述

| 表述 | 方程 | 说明 |
|------|------|------|
| **标准方程（横双曲线）** | x²/a² − y²/b² = 1 | 焦点在 x 轴 |
| **标准方程（纵双曲线）** | y²/a² − x²/b² = 1 | 焦点在 y 轴 |
| **定义式** | \|\|PF₁\| − \|PF₂\|\| = 2a | 距离差定义 |
| **渐近线** | y = ±(b/a)x | 双曲线逼近但不相交的直线 |

**与椭圆的关系：**
- 椭圆：距离之和为常数
- 双曲线：距离之差的绝对值为常数
- 两者合称"有心二次曲线"

---

### 概念辨析

**辨析 1：双曲线 vs 椭圆**

| 对比项 | 双曲线 | 椭圆 |
|--------|--------|------|
| 定义 | \|\|PF₁\| − \|PF₂\|\| = 2a | \|PF₁\| + \|PF₂\| = 2a |
| 方程符号 | x²/a² − y²/b² = 1 | x²/a² + y²/b² = 1 |
| 离心率 | e = c/a > 1 | e = c/a < 1 |
| 焦点位置 | 在两支曲线"内部" | 在椭圆内部 |
| 渐近线 | 有渐近线 y = ±(b/a)x | 无渐近线 |

**辨析 2：焦点位置与 a, b 的关系**

| 类型 | 焦点位置 | 方程 | 条件 |
|------|---------|------|------|
| 横双曲线 | 在 x 轴上 | x²/a² − y²/b² = 1 | 焦点在两支之间 |
| 纵双曲线 | 在 y 轴上 | y²/a² − x²/b² = 1 | 焦点在两支之间 |

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 双曲线上点 P | \|\|PF₁\| − \|PF₂\|\| = 2a | 定义 |
| 焦点在双曲线外 | c > a | c² = a² + b² |
| 离心率 | e = c/a > 1 | 必须大于 1 |

## 命题层

### 核心定理

**定理 1：双曲线的标准方程**

```
焦点在 x 轴：x²/a² − y²/b² = 1
焦点在 y 轴：y²/a² − x²/b² = 1
其中 a > 0, b > 0，c² = a² + b²
```

**定理 2：双曲线的离心率**

```
e = c/a = √(1 + b²/a²) > 1
e 越大，双曲线越"张开"
```

**定理 3：双曲线的渐近线**

```
横双曲线 x²/a² − y²/b² = 1 的渐近线：
y = ±(b/a)x
```

**定理 4：双曲线的切线方程**

```
双曲线 x²/a² − y²/b² = 1 上一点 P(x₀, y₀)：
xx₀/a² − yy₀/b² = 1
```

---

### 典型例题

**例题 1：求双曲线标准方程**

```
题目：已知双曲线焦点在 x 轴上，焦距为 10，实轴长为 6，求双曲线方程。
```

**深层分析：**

```
焦距 = 2c = 10 ⟹ c = 5
实轴 = 2a = 6 ⟹ a = 3
b² = c² − a² = 25 − 9 = 16 ⟹ b = 4

焦点在 x 轴，所以方程：
x²/9 − y²/16 = 1
```

> **方法提炼：** 双曲线中 b² = c² − a²（与椭圆 b² = a² − c² 相反）。

---

**例题 2：求双曲线的渐近线**

```
题目：求双曲线 x²/9 − y²/4 = 1 的渐近线方程。
```

**深层分析：**

```
渐近线：y = ±(b/a)x
a² = 9, b² = 4
a = 3, b = 2

渐近线方程：y = ±(2/3)x
```

> **方法提炼：** 横双曲线的渐近线 y = ±(b/a)x。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "到两焦点距离之差" | 双曲线定义 | 用 \|\|PF₁\| − \|PF₂\|\| = 2a |
| "渐近线" | 渐近线问题 | y = ±(b/a)x |
| "离心率 e > 1" | 双曲线判定 | e = c/a > 1 |
| "求双曲线方程" | 需确定 a, b | 已知条件求 a², b² |

---

### 方法详解：双曲线方程求法流程

**Step 1: 确定焦点位置**

```
焦点在 x 轴 → 横双曲线
焦点在 y 轴 → 纵双曲线
```

**Step 2: 求 a, b, c**

```
已知条件：
- 焦距 → 2c
- 实轴 → 2a
- 关系：c² = a² + b²（注意是加号！）
```

**Step 3: 写出方程**

```
横双曲线：x²/a² − y²/b² = 1
纵双曲线：y²/a² − x²/b² = 1
```

> **易错提醒：** 双曲线中 b² = c² − a²，与椭圆 b² = a² − c² 正好相反！

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "双曲线 b² = a² − c²" | 与椭圆混淆 | 双曲线是 c² − a² |
| 2 | "双曲线离心率 < 1" | 记错 | e = c/a > 1 |
| 3 | "双曲线有两条渐近线" | 正确 | y = ±(b/a)x |
| 4 | "椭圆和双曲线渐近线相同" | 混淆 | 只有双曲线有渐近线 |
