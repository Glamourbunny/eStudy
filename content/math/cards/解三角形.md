# 解三角形

> 解三角形，就是已知三角形的部分"零件"，求出它的全部"零件"——边和角。

---

## 概念层

### 问题的起源

生活中经常遇到这样的问题：

- 两点之间的距离无法直接测量，怎么办？
- 测量山的高度，不用爬到山顶，如何知道？
- 航行中，已知航向和航速，如何计算到达位置？

这些问题的共同特点是：**只知道三角形的部分边角，却要求出其他的边角**。

这就是"解三角形"问题——从已知出发，求解未知。

> **一句话：解三角形是根据已知边角推断未知边角的过程。**

---

### 原始定义（教材原文）

在△ABC 中，三条边 a、b、c 分别对应三个角 A、B、C。已知三角形的某些边和角，求解其他边和角的过程称为**解三角形**。

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 11-15 页

---

### 本质定义

解三角形的本质是建立**边角关系**，利用正弦定理和余弦定理作为主要工具：

**已知条件类型与求解策略：**

| 已知条件 | 求解目标 | 首选工具 |
|---------|---------|---------|
| 两角及一边（A, B, c） | 求另一角及两边 | 正弦定理 |
| 两边及夹角（a, b, C） | 求第三边及余角 | 余弦定理 |
| 三边（a, b, c） | 求三角 | 余弦定理 |
| 两边及一边的对角（a, b, A） | 可能有双解 | 正弦定理 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **边（a, b, c）** | 三角形的三条边 |
| **角（A, B, C）** | 三角形的三个内角 |
| **关系定理** | 正弦定理、余弦定理 |

---

### 等价表述

| 表述方式 | 内容 |
|----------|------|
| **正弦定理** | a/sinA = b/sinB = c/sinC = 2R |
| **余弦定理** | c² = a² + b² − 2ab·cosC |
| **角度和公式** | A + B + C = 180° |
| **边长关系** | 任意两边之和大于第三边 |

### 概念辨析

**辨析 1：正弦定理 vs 余弦定理**

| 对比项 | 正弦定理 | 余弦定理 |
|--------|---------|---------|
| **公式** | a/sinA = b/sinB = c/sinC = 2R | c² = a² + b² − 2ab·cosC |
| **核心关系** | 边与对角的正弦成正比 | 边与夹角余弦的关系 |
| **已知条件** | 两角一边或两边及对角 | 两边及其夹角或三边 |
| **解决的问题** | 求边或求角 | 求第三边或判断三角形形状 |
| **易错点** | 角对边的双解问题 | 不能直接求角 |

**辨析 2："边边角"（SSA）情况——可能的歧义**

已知两边及一边对角（如已知 a, b, A），用正弦定理求 B 时，可能出现：

| 情况 | 图形 | 解的个数 |
|------|------|---------|
| a < b 且 a = b·sinA | 唯一解（直角） | 1 |
| a > b | 唯一解（大边对大角） | 1 |
| a = b·sinA < b | 唯一解 | 1 |
| b > a > b·sinA | 两解（如右图） | 2 |
| a = b·sinA | 一解 | 1 |
| a < b·sinA | 无解（构不成三角形） | 0 |

```
    C                    C
   /│\                  /│\
  / │ \                / │ \
 /  │  \              /  │  \
A───B                   A───B
  唯一解                  两解（△ABC₁ 和 △ABC₂）
```

> **一句话：** SSA 情况下，已知 a < b 时可能有两解，因为 A 可以是锐角或钝角（但 B + A < 180° 的限制最终会使某些解无效）。

**辨析 3：解三角形 vs 解方程**

| 对比项 | 解三角形 | 解方程 |
|--------|---------|--------|
| **目标** | 求出边长和角度 | 求出未知数的值 |
| **约束** | 几何条件（边角关系） | 代数条件（等式成立） |
| **工具** | 正弦定理、余弦定理 | 代数运算 |
| **解的个数** | 可能无解、一解或两解 | 通常有限个解 |

---

### 适用边界

| 条件 | 解的情况 | 说明 |
|------|---------|------|
| 已知两角及一边 | 唯一解 | 能确定唯一三角形 |
| 已知两边及夹角 | 唯一解 | 用余弦定理求第三边 |
| 已知三边 | 唯一解或无解 | 用余弦定理判断 cosC 的符号 |
| 已知两边及大边对大角 | 唯一解 | 大边对大角，唯一确定 |
| 已知两边及小边对小角 | 可能两解 | SSA 情况需讨论 |

**无解情况：**
- 三边不满足三角不等式
- SSA 情况下 a < b·sinA（够不着）

## 命题层

### 核心定理

**定理 1：正弦定理**

在△ABC 中：
```
a/sinA = b/sinB = c/sinC = 2R
```
其中 R 是三角形外接圆的半径。

**证明思路：** 利用三角形的高和面积关系，或者利用外接圆的几何关系。

**变形公式：**
```
a = 2R·sinA
sinA = a/(2R)
```

**定理 2：余弦定理**

在△ABC 中：
```
c² = a² + b² − 2ab·cosC
b² = a² + c² − 2ac·cosB
a² = b² + c² − 2bc·cosA
```

**推论：**
```
cosC = (a² + b² − c²) / (2ab)
```

**定理 3：三角形面积公式**

```
S = (1/2)ab·sinC = (1/2)bc·sinA = (1/2)ca·sinB
```

> 这个公式是正弦定理的另一个重要应用，将边长与夹角的正弦联系起来。

**定理链：**

```
正弦定理 + 余弦定理（边角关系的两大支柱）
         │
    ┌────┴────┐
    ▼         ▼
 求边神器    求角神器
            │
    ┌───────┴───────┐
    ▼               ▼
 判断三角形形状    判断能否构成三角形
  (a²+b²?c²)        (三边关系)
```

### 典型例题的深层分析

**例题 1：已知两边及夹角，求第三边**

```
题目：在△ABC 中，a = 3，b = 4，C = 60°，求 c。
```

**深层分析：**

这是典型的"SAS"情况，使用余弦定理：

```
c² = a² + b² − 2ab·cosC
    = 3² + 4² − 2×3×4×cos60°
    = 9 + 16 − 24×(1/2)
    = 25 − 12
    = 13
c = √13
```

> **方法提炼：** SAS → 余弦定理 → 直接代公式求第三边。

---

**例题 2：已知三边，判断三角形形状**

```
题目：已知三角形三边 a=7, b=4, c=5，判断这是什么类型的三角形。
```

**深层分析：**

用余弦定理求最大角 C 的余弦：

```
cosC = (a² + b² − c²) / (2ab)
     = (49 + 16 − 25) / (2×7×4)
     = 40 / 56
     = 5/7 > 0
```

因为 cosC > 0，所以 C 是锐角。但这不是最终结论——需要判断最大角：

a 是最大边，所以 A 是最大角。计算 cosA：

```
cosA = (b² + c² − a²) / (2bc)
     = (16 + 25 − 49) / (2×4×5)
     = (−8) / 40
     = −0.2 < 0
```

cosA < 0，所以 A 是钝角。三边形有一个钝角，是**钝角三角形**。

> **方法提炼：** 判断三角形形状 → 求最大角（最大边所对的角）→ 看余弦符号。

---

**例题 3：SSA 情况的讨论**

```
题目：已知 a=3, b=4, A=30°，解此三角形。
```

**深层分析：**

这是 SSA 情况。用正弦定理求 B：

```
sinB / b = sinA / a
sinB / 4 = sin30° / 3
sinB = 4 × (1/2) / 3 = 2/3
```

sinB = 2/3 ≈ 0.667。

因为 sinB > 0，B 可能是锐角或钝角：
- B₁ = arcsin(2/3) ≈ 41.8°（锐角）
- B₂ = 180° − 41.8° = 138.2°（钝角）

检验 A + B₂ = 30° + 138.2° = 168.2° < 180°，所以 B₂ 也是有效解。

因此有两解：
- 解1：A=30°, B≈41.8°, C≈108.2°
- 解2：A=30°, B≈138.2°, C≈11.8°

> **方法提炼：** SSA 情况 → 求 sinB → 考虑 B₁ 和 B₂ 两个候选 → 检验 A+B 是否 < 180°。

---

**例题 4：测量问题（解三角形的实际应用）**

```
题目：为了测量一条不能到达的河流两岸 A、B 之间的距离，
      我们在对岸选一点 C，测得 CA = 200m，CB = 150m，
      ∠ACB = 60°，求 AB 的长度。
```

**深层分析：**

这是典型的"SAS"情况，已知两边及夹角，求第三边 AB（即 c）。

```
c² = a² + b² − 2ab·cosC
    = 200² + 150² − 2×200×150×cos60°
    = 40000 + 22500 − 60000×(1/2)
    = 62500 − 30000
    = 32500
c = √32500 = 100√3.25 ≈ 180.28m
```

> **方法提炼：** 测量问题 → 转化为三角形已知条件 → 选用正弦/余弦定理 → 计算。

## 方法层

### 识别信号表

| 已知条件 | 识别信号 | 首选方法 |
|---------|---------|---------|
| 两角及一边 | "已知 A, B, a" | 正弦定理 |
| 两边及夹角 | "已知 a, b, C" | 余弦定理 |
| 三边 | "已知三边长" | 余弦定理（先求角） |
| 两边及对角 | "已知 a, b, A" | 正弦定理（注意双解） |
| 求面积 | "已知两边及其夹角" | 面积公式 S = (1/2)ab·sinC |

### 决策树

```
已知三角形的某些边角
            │
    ┌───────┼───────┐
    │       │       │
  AAS/ASA  SAS/SSS  SSA(两边及对角)
    │       │       │
    │       │       └────→ 正弦定理（考虑双解）
    │       │
    │       └────→ 余弦定理
    │
    └────→ 正弦定理
            │
     ┌──────┴──────┐
     │              │
   求角            求边
     │              │
   余弦定理        正弦定理
```

### 方法详解：解三角形标准流程

**Step 1：整理已知条件**

画图，标出已知边角，判断已知条件类型。

**Step 2：选择定理**

| 判断方法 | 选择 |
|---------|------|
| 已知两角一边 | 正弦定理 |
| 已知两边及夹角 | 余弦定理 |
| 已知三边 | 余弦定理（先求角） |
| 已知两边及对角 | 正弦定理（注意双解） |
| 求面积 | 面积公式 |

**Step 3：代入计算**

- 用正弦定理：边/角 = 边/角
- 用余弦定理：c² = a² + b² − 2ab·cosC

**Step 4：检验和讨论**

- 检查是否符合三角形条件（A + B + C = 180°）
- SSA 情况注意双解检验

> **易错提醒：**
> 1. 余弦定理求角时，用反余弦函数，注意范围
> 2. 正弦定理求角时，考虑双解情况
> 3. 最终检查三边是否满足三角不等式

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "SSA 有唯一解" | 忽视双解情况 | a < b 时可能有两解，需检验 A+B<180° |
| 2 | "余弦定理求角直接除" | 忘记是 cos | cosC 可能为负，角可能是钝角 |
| 3 | "三角形有两个解" | SSA 必有两个解 | 只有满足 b > a > b·sinA 才有两解 |
| 4 | "acosA" 写法错误 | 记错公式 | 应该是 a·cosA，不是 acoA |
| 5 | 忽略角度单位 | 弧度制/角度制混淆 | 高中通常用角度制 |
| 6 | 三个角加起来不等于 180° | 计算或读数错误 | A + B + C = 180° 是必须满足的条件 |
