# 功

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** "功"在物理课本里是 W = F·s·cosθ，在数学课本里它就是向量数量积。这张卡片让你看到：功 = 数量积的物理肉身。

---

## 视角层（像什么）

### 核心隐喻：行李箱的拉杆

**功 = 你斜着拉拉杆箱，真正有效的力只有"沿着前进方向的那部分"**

```
  你拉行李箱：
  
  拉力 F = 50N，与水平面成 30° 角
  行李箱水平移动了 s = 10m
  
       Fₓ     ← 水平分量（有效拉力，做了功）
      ↗ 30°
     ↗
    ↗ Fᵧ    ← 竖直分量（提箱子的力，不做功——箱子没有竖直移动！）
  
  只有 F 在位移方向上的投影 × 位移 = 功
  W = F·s = |F||s|cos30° = 50×10×0.866 = 433 J
```

**拉杆箱的三条做功法则：**

| 法则 | 拉杆箱场景 | 数量积语言 |
|------|-----------|-----------|
| **力与位移同向时做功最大** | 平着拉箱子（力与位移平行）→ 全部力都在做功 | cos0°=1，W=|F||s| |
| **力与位移垂直时不做功** | 提箱子竖直向上走（力与位移垂直）→ 没做功？不，是"水平方向不做功" | cos90°=0，W=0 |
| **力与位移反向时做负功** | 摩擦力：与位移方向相反 → 阻碍运动 | cos180°=-1，W=-|F||s| |

> **一句话直觉：** 功 = 力在位移方向上的投影 × 位移 —— 也就是"你的努力中，有多少是朝着目标方向使的劲"。数学上就是向量 F 和位移 s 的数量积。

### 动态想象：推购物车

```
  购物车重 20kg，你在后面水平推它移动 5m
  
  推力水平向前 → 与位移同向 → W = F×5（正功，给系统输入能量）
  摩擦力水平向后 → 与位移反向 → W_friction = -f×5（负功，耗散能量）
  
  合力做的总功 = 推力正功 + 摩擦力负功 = 动能增加量（动能定理）
```

**高考里的翻版：**
- "力 F=(2,3) 作用于质点，使其从 (1,1) 移动到 (4,5)，求做的功"——直接 F·Δs = 2×3+3×4 = 18 J
- "物体沿斜面下滑，求重力做的功"——重力×沿斜面方向的位移投影，或重力×竖直高度差（mgh）
- "变力做功，F-x 图线下面积"——积分的物理意义

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **"用力了就有功"** | "我花了力气所以做了功" | 把箱子水平提着走：手臂很累（肌肉在做功），但从物理学的角度看，你对箱子做的功为零（力垂直位移）。肌肉酸痛是因为肌肉内部消耗了能量 |
| **"负功就是反向做功"** | "负功=往回缩" | 负功表示力在"消耗"物体的动能，不是在"往反方向做功"。摩擦力的方向没有反过来，它只是跟位移反向 |
| **功的正负不代表方向** | "正功往前负功往后" | 功是标量，没有方向！正负表示能量是增加还是减少 |

## 联结层（连什么）

### 跨学科锚点

**1. 数学：向量数量积的物理原型**

```
  数学：a·b = |a||b|cosθ = x₁x₂+y₁y₂
  
  物理：W = F·s = |F||s|cosθ = Fx·sx + Fy·sy
  
  完全一样！功就是力向量和位移向量的数量积
```

**2. 生活：搬家具上楼梯**

```
  沿着斜面推箱子：一部分力克服重力、一部分力克服摩擦力
  
  推力的功 = (重力分量+摩擦力) × 斜面长度
  等效于：直接竖直抬上去（克服重力做的功）+ 克服摩擦力水平移动
  
  注意：克服重力做的功只取决于高度差，与路径无关（保守力！）
```

**3. 化学：气体膨胀做功**

```
  气体推动活塞向外膨胀
  
  功 = 压力 × 活塞移动距离 × 活塞面积 / 面积
     = P × A × Δx
     = P × ΔV（压强×体积变化）
  
  这也是"力在位移方向的投影"——只是改头换面成了压强和体积
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 恒力做功 | 力在位移方向的投影×位移 | W = F·s = |F||s|cosθ |
| 变力做功 | 力沿路径的积累 | 积分：W = ∫F·ds（或 F-x 图面积） |
| 合力做功 = 动能变化 | 功-能桥梁 | 动能定理：W_合 = ΔE_k |
| 保守力做功 = 势能减小 | 与路径无关 | W_保 = -ΔE_p |

**首选工具**：`数量积公式`——W = F·s = Fx·sx+Fy·sy，适用于恒力直线运动
**次选工具**：动能定理——当力或路径复杂时，从能量变化反推功

### 历史/哲学视角

**"功"是人类发明的第一个"能量转化度量"。** 19 世纪初，蒸汽机工程师需要比较不同机器的"功效"，但当时没有统一标准。科里奥利（Coriolis）在 1829 年定义"功 = 力 × 位移"时，可能没想到他同时定义了向量数量积的物理原型。

**哲学意义**：功是"跨越"——它是力在空间中的积累。力只是一个瞬时的概念，而功是力在空间上的积分。"做功"就是把能量从一个地方"搬运"到另一个地方。

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | 物理题问"做功"，给了一对坐标形式的力和位移 | 在物理和数学之间来回切换，不知道可以用数量积 | 认出"功 = F·s"，直接用向量坐标运算 |
| 🔀 **嵌套** | "力 F₁ 做正功、摩擦力 f 做负功、重力 mg 做负功…求总功" | 一项一项算，正负号搞乱 | 画图标注每个力的方向和位移的方向，力与位移夹角决定正负 |
| 🆕 **新定义** | 较少以新定义出现 | — | — |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 看到"功""力做的功" → 数量积模式
  • 问：力和位移各是什么？（大小+方向）

Step 2：剥外壳找骨骼（4 秒）
  • 找到力 F 向量和位移 s 向量
  • 功 = |F||s|cosθ = F·s（数量积）
  • θ = 力与位移的夹角（不是力与水平面的夹角！）

Step 3：锁定 Formal 着陆点（3 秒）
  • 写下：拆包→【数量积公式/动能定理】
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 力和位移向量已确定 | W = F·s = |F||s|cosθ，或直接算分量乘积分量 |
| 嵌套题 | 每个力的做功路径清楚 | 逐力计算 W_i，代数和 = W_总 |

**违反此铁律的后果**：夹角 θ 找错（把力和位移的夹角当成了力和水平面的夹角），导致 cosθ 算反。

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **力和位移夹角 ≠ 力与水平面夹角** | 在斜面上时，位移沿斜面，力也有角度 | 画图：标出 F 的方向箭头和 s 的方向箭头，两箭头的夹角才是 θ |
| **正负号乱** | 摩擦力做负功 = W_f<0 | 摩擦力与位移反向 → cos180°=-1 → W_f = -f·s < 0 |
| **变力做功用恒力公式** | 当力随位置变化时，W ≠ F·s | 变力→ 积分或 F-x 图面积 |

## 一句话总结

> **功 = 行李箱的拉杆——你斜着拉箱子，只有"沿着前进方向的那部分力"真正做了功。数学上就是力向量和位移向量的数量积：W = F·s = |F||s|cosθ。物理和数学在这里握手——向量内积不是空穴来风，它是"功"的化身。**
