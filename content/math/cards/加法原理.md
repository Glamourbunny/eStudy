# 加法原理

> 加法原理是"分类相加"——完成一件事有 n 类办法，每类办法分别有 a₁, a₂, ..., aₙ 种，则总共有 a₁ + a₂ + ... + aₙ 种。

---

## 概念层

### 问题的起源

从北京去上海，可以坐飞机（3种班次）或坐火车（5种班次）或坐长途汽车（2种班次）。

问：从北京去上海有多少种走法？

这是不同**类别的选择**，每类内部互斥（不能同时坐飞机又坐火车）。

答案是 3 + 5 + 2 = 10 种。

这就是**加法原理**。

> **一句话：加法原理是"分类计数"——各类办法数相加。**

---

### 原始定义（教材原文）

完成一件事有 n 类办法，这 n 类办法是**互斥的**（每一类办法都能独立完成这件事）。若第 1 类办法有 a₁ 种，第 2 类办法有 a₂ 种，...，第 n 类办法有 aₙ 种，则完成这件事共有 **a₁ + a₂ + ... + aₙ** 种不同的方法。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修三》第 1 页

---

### 本质定义

**加法原理的本质是"分类"：**

```
完成一件事
    │
    ├─ 第1类（a₁种方法）→ 独立完成
    ├─ 第2类（a₂种方法）→ 独立完成
    └─ ...
    └─ 第n类（aₙ种方法）→ 独立完成

总方法数 = a₁ + a₂ + ... + aₙ
```

**关键特征：**
- 各类办法**互斥**（选了这类就不能选那类）
- 每类办法都能**独立完成**这件事

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **分类** | n 类办法，互斥 |
| **每类办法数** | a₁, a₂, ..., aₙ |
| **加法** | 总数 = 各层相加 |

---

### 等价表述

| 表述 | 形式 |
|------|------|
| **分类相加** | 各类方法数相加 |
| **互斥事件** | 每类独立完成 |
| **加法原理** | 总数 = Σ 各层方法数 |

---

### 与乘法原理的对比

| 对比项 | 加法原理 | 乘法原理 |
|--------|---------|---------|
| 核心 | 分类 | 分步 |
| 关系 | 各类互斥 | 各步依存 |
| 公式 | 总数 = a₁ + a₂ + ... | 总数 = a₁ × a₂ × ... |
| 例子 | 去上海：飞机或火车或汽车 | 穿衣服：先选上衣再选裤子 |

## 命题层

### 核心定理

**定理：加法原理**

```
完成一件事有 n 类互斥办法
第 i 类有 aᵢ 种办法（i = 1, 2, ..., n）
则总办法数 = Σaᵢ = a₁ + a₂ + ... + aₙ
```

---

### 典型例题

**例题：分类计数**

```
题目：从 1, 2, 3, 4, 5 中取一个数，取到奇数有多少种？
```

**深层分析：**

```
1, 2, 3, 4, 5 中的奇数有：1, 3, 5

取到奇数 = 取到 1 或取到 3 或取到 5
共 1 + 1 + 1 = 3 种
```

> **方法提炼：** 分类相加。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "或""或者" | 分类 | 用加法原理 |
| "第1类""第2类" | 分类 | 各类相加 |
| "互斥" | 分类特征 | 各类不重叠 |

---

### 方法详解：判断用加法还是乘法

**Step 1: 判断是"分类"还是"分步"**

```
"或" → 加法（分类）
"然后""再" → 乘法（分步）
```

**Step 2: 分类时，每类是否互斥**

```
每类都能独立完成这件事
```

**Step 3: 相加或相乘**

```
加法原理 → 各类相加
乘法原理 → 各步相乘
```

| 编号 | 常见误区 | 错误根源 | 正确理解 |
