# 最小二乘法

> 最小二乘法是"使误差平方和最小"的优化方法——在拟合直线时，让所有点到直线的竖直距离平方和最小。

---

## 概念层

### 问题的起源

拟合直线时，如何衡量"好坏"？

用点到直线的"距离"来衡量。但"距离"有正负，直接求和会抵消。

怎么办？——用**平方**消除负号，用**求和**累加所有误差。

最小二乘法就是找到使 Σ(yᵢ - ŷᵢ)² 最小的参数。

> **一句话：最小二乘法是最小化误差平方和的优化方法。**

---

### 本质定义

**最小二乘法的目标：**

```
最小化 Q = Σ(yᵢ - ŷᵢ)²
       = Σ(yᵢ - a - bxᵢ)²

通过 ∂Q/∂a = 0 和 ∂Q/∂b = 0 求出 a, b
```

---

### 与线性回归的关系

```
线性回归 ⟹ 使用最小二乘法拟合直线
最小二乘法 ⟹ 一种通用的优化准则
```

最小二乘法是线性回归的**方法论基础**。

---

## 命题层

### 核心定理

**定理：最小二乘法的正规方程**

```
∂Q/∂a = 0 ⟹ Σ(yᵢ - a - bxᵢ) = 0
∂Q/∂b = 0 ⟹ Σxᵢ(yᵢ - a - bxᵢ) = 0

解得：
b = Σ(xᵢ - x̄)(yᵢ - ȳ) / Σ(xᵢ - x̄)²
a = ȳ - bx̄
```
