# 全概率公式

> 全概率公式是把样本空间分割成若干部分，分别计算各部分中 B 发生的概率，然后求和。

---

## 概念层

### 问题的起源

直接计算 P(B) 很难，但 B 是由若干互斥事件 A₁, A₂, ..., Aₙ 之一发生时组成的。

利用**条件概率**和**完备事件组**的思想，可以把 P(B) 拆分成若干容易计算的项。

这就是**全概率公式**。

> **一句话：全概率公式把复杂事件的概率分解为若干互斥情况下概率的加权和。**

---

### 原始定义（教材原文）

设 A₁, A₂, ..., Aₙ 是一组互斥事件，且 ⋃Aᵢ = Ω（完备事件组），则：

```
P(B) = Σ P(Aᵢ) × P(B|Aᵢ)
```

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》

---

### 本质定义

**全概率公式的本质是"分类求和"：**

```
样本空间 Ω 被分割为 A₁, A₂, ..., Aₙ（互斥且完备）
B 在每种 Aᵢ 情况下都可能发生

P(B) = Σ P(Aᵢ 且 B)
      = Σ P(Aᵢ) × P(B|Aᵢ)
```

**几何理解：**

```
    Ω（全部样本）
    ├──── A₁（概率 P(A₁)）→ B 发生的条件概率 P(B|A₁)
    ├──── A₂（概率 P(A₂)）→ B 发生的条件概率 P(B|A₂)
    └──── ...
    └──── Aₙ（概率 P(Aₙ)）→ B 发生的条件概率 P(B|Aₙ)

P(B) = Σ P(Aᵢ)P(B|Aᵢ)
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **完备事件组** | A₁~Aₙ 互斥且覆盖全空间 |
| **条件概率** | P(B|Aᵢ) |
| **加权和** | Σ P(Aᵢ)×P(B\|Aᵢ) |

---

### 与贝叶斯公式的关系

```
全概率公式：P(B) = Σ P(Aᵢ)P(B|Aᵢ)
贝叶斯公式：P(Aⱼ|B) = P(Aⱼ)P(B|Aⱼ) / Σ P(Aᵢ)P(B|Aᵢ)
```

贝叶斯公式是用全概率公式的分母。

## 命题层

### 核心定理

**定理：全概率公式**

```
若 A₁, A₂, ..., Aₙ 互斥且 ⋃Aᵢ = Ω，则：
P(B) = Σᵢ P(Aᵢ) × P(B|Aᵢ)
```

**定理：贝叶斯公式**

```
P(Aⱼ|B) = P(Aⱼ)P(B|Aⱼ) / Σᵢ P(Aᵢ)P(B|Aᵢ)
```

---

### 典型例题

**例题：全概率公式应用**

```
题目：有两个工厂生产同一零件。工厂甲占 60%，次品率 1%；工厂乙占 40%，次品率 2%。
      求从所有零件中任取一件是次品的概率。
```

**深层分析：**

```
设 A₁ = "零件来自甲厂"，A₂ = "零件来自乙厂"
A₁ 和 A₂ 互斥，且 P(A₁)+P(A₂) = 1

P(次品) = P(A₁)P(次品|A₁) + P(A₂)P(次品|A₂)
        = 0.6×0.01 + 0.4×0.02
        = 0.006 + 0.008
        = 0.014
```

> **方法提炼：** 先找完备事件组，再用全概率公式。

| "贝叶斯""逆概率" | 贝叶斯公式 | 用全概率公式作为分母 |

---

### 方法详解：全概率公式解题步骤

**Step 1: 找出完备事件组 A₁~Aₙ**

```
互斥 + 完全覆盖
```

**Step 2: 计算 P(Aᵢ) 和 P(B|Aᵢ)**

**Step 3: 代入公式**

```
P(B) = Σ P(Aᵢ)P(B|Aᵢ)
```
