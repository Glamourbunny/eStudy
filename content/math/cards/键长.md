# 键长

> 键长是成键原子之间的核间距——它决定了化学键的强弱，也影响分子的形状和性质。

---

## 概念层

### 问题的起源

化学键是原子之间的"粘合剂"。键长就是两个成键原子核之间的距离。

键长越短，通常意味着键越强：
- 键长 ~ 1.54 Å（C-C 单键）
- 键长 ~ 1.34 Å（C=C 双键）
- 键长 ~ 1.20 Å（C≡C 三键）

这说明**键级越高，键长越短，键越强**。

> **一句话：键长是成键原子核间的距离，键长越短，键越强。**

---

### 原始定义（教材原文）

分子中两个成键原子的核间距离称为**键长**。

键长是描述化学键强度的重要参数之一。

> 出处：人教版《普通高中教科书·化学（选择性必修）》

---

### 本质定义

**键长的本质是原子核对电子云重叠的"平衡距离"：**

```
核间距
 ←→ r ←→
 H——Cl
 ↑     ↑
H核    Cl核
```

电子云重叠太近 → 排斥力增大
电子云重叠太远 → 吸引力减小
平衡位置 → 键长

**键长与键能的关系：**

通常键长越短，键能越大。

| 键类型 | 键长（Å） | 键能（kJ/mol） |
|--------|----------|---------------|
| C-C | 1.54 | 347 |
| C=C | 1.34 | 614 |
| C≡C | 1.20 | 839 |
| C-H | 1.09 | 413 |

## 命题层

### 核心定理

**定理 1：键长与键级的反比关系**

```
键级越高 → 键长越短 → 键越强
```

**定理 2：原子半径与键长的关系**

```
共价键长 ≈ 两个原子的共价半径之和
r(C-C) ≈ r(C) + r(C) = 1.54/2 + 1.54/2 = 1.54 Å
```

**定理 3：多重键的键长缩短规律**

```
C-C（单键） > C=C（双键） > C≡C（三键）
1.54 Å    > 1.34 Å    > 1.20 Å
```

---

### 典型例题

**例题：从键长数据判断键的类型**

```
题目：某碳碳键的键长为 1.34 Å，判断这是什么键。
```

**深层分析：**

```
已知数据：
C-C 单键：1.54 Å
C=C 双键：1.34 Å
C≡C 三键：1.20 Å

键长 1.34 Å 与 C=C 双键的键长完全吻合，
所以这是碳碳双键。
```

> **方法提炼：** 键长的本质是"标准数据对照"，熟记常见键长数据。

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |

| "键长" | 判断键的类型 | 对照标准数据 |
| "原子半径" | 估算键长 | 键长 ≈ 两原子半径之和 |
| "判断单双三键" | 键级 | 1.54/1.34/1.20 |

---

### 方法详解：键长的应用

**应用 1: 判断键的类型**

```
对比未知键长与标准数据：
1.54 Å → 单键
1.34 Å → 双键
1.20 Å → 三键
```

**应用 2: 估算共价半径**

```
共价半径 r(A) ≈ 已知键长 / 2（对于相同原子形成的键）
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "键长越短键越弱" | 记反 | 键长越短，键通常越强 |
| 2 | "键长是原子的接触距离" | 误解 | 键长是核间距，不是原子表面接触距离 |
| 3 | "双键键长是单键的一半" | 计算错误 | 双键键长只是比单键短，不是减半 |
