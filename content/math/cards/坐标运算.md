# 坐标运算

> 空间向量的坐标运算，就是把向量的加减乘除分解到 x、y、z 三个方向上独立进行——三个维度互不干扰，平行处理。

---

## 概念层

### 问题的起源

有了空间向量的坐标表示，如何进行运算？

答案是：**把运算分解到三个坐标轴方向，分别独立计算**。

这就像把一个三维的运动分解为前后、左右、上下三个方向的分运动——物理中的"运动的独立性原理"在数学上叫"坐标运算"。

> **一句话：坐标运算的核心是"分而治之"——把三维运算分解为三个一维运算。**

---

### 原始定义（教材原文）

设空间向量 $\vec{a} = (a_x, a_y, a_z)$，$\vec{b} = (b_x, b_y, b_z)$，则：

```
加法：\vec{a} + \vec{b} = (a_x+b_x, a_y+b_y, a_z+b_z)
减法：\vec{a} - \vec{b} = (a_x-b_x, a_y-b_y, a_z-b_z)
数乘：k·\vec{a} = (k·a_x, k·a_y, k·a_z)
数量积：\vec{a}·\vec{b} = a_x·b_x + a_y·b_y + a_z·b_z
```

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 5-8 页

---

### 本质定义

**坐标运算的本质是"分量独立运算"：**

```
三维向量运算 → 三个一维向量运算
\x方向: a_x 与 b_x 单独运算
\y方向: a_y 与 b_y 单独运算
\z方向: a_z 与 b_z 单独运算
最终结果 = (三个分量运算结果的有序三元组)
```

**为什么这样有效？**
因为坐标轴是相互垂直的（正交），x 方向、y 方向、z 方向互不干扰。

---

### 定义的三要素

| 运算 | 核心规则 | 结果 |
|------|---------|------|
| **加法/减法** | 对应分量相加减 | 仍是向量 |
| **数乘** | 每个分量乘以 k | 仍是向量 |
| **数量积** | 对应分量相乘后求和 | 标量（数） |

---

### 等价表述

| 运算 | 坐标形式 | 几何意义 |
|------|---------|---------|
| **加法** | (a_x+b_x, a_y+b_y, a_z+b_z) | 平行六面体对角线 |
| **减法** | (a_x−b_x, a_y−b_y, a_z−b_z) | 向量差 |
| **数乘** | (k·a_x, k·a_y, k·a_z) | 同向伸缩 |
| **数量积** | a_x·b_x + a_y·b_y + a_z·b_z | 投影相乘 |

### 概念辨析

**辨析：坐标运算 vs 向量的几何运算**

| 对比项 | 坐标运算 | 几何运算 |
|--------|---------|---------|
| 工具 | 数字计算器 | 平行四边形/三角形法则 |
| 优势 | 程序化，不易出错 | 直观，几何意义清晰 |
| 适用场景 | 复杂计算题 | 简单问题、证明 |

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 对应分量都存在 | ✓ | 三个分量缺一不可 |
| 数乘实数 k 任意 | ✓ | k 可正可负 |
| 数量积结果为标量 | ✓ | 不是向量 |

## 命题层

### 核心定理

**定理 1：坐标运算律**

```
交换律：\vec{a} + \vec{b} = \vec{b} + \vec{a}
结合律：(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})
数乘分配律：k·(\vec{a} + \vec{b}) = k·\vec{a} + k·\vec{b}
```

**定理 2：向量的坐标与起点终点**

已知 A(x₁,y₁,z₁)，B(x₂,y₂,z₂)，则：
```
AB⃗ = (x₂−x₁, y₂−y₁, z₂−z₁)
BA⃗ = −AB⃗ = (x₁−x₂, y₁−y₂, z₁−z₂)
```

**定理 3：中点坐标公式**

```
M 是 A(x₁,y₁,z₁) 和 B(x₂,y₂,z₂) 的中点
M 的坐标 = ((x₁+x₂)/2, (y₁+y₂)/2, (z₁+z₂)/2)
```

### 典型例题

**例题 1：向量运算**

```
题目：已知 A(1,2,3)，B(2,4,1)，C(3,1,5)，
      求 AB⃗ + AC⃗ 的坐标。
```

**深层分析：**

```
AB⃗ = (1,2,-2)
AC⃗ = (2,-1,2)
AB⃗ + AC⃗ = (1+2, 2+(-1), (-2)+2) = (3,1,0)
```

> **方法提炼：** 加法就是分量直接相加。

---

**例题 2：数乘与减法**

```
题目：已知 A(1,0,2)，B(2,3,1)，求 2AB⃗ 的坐标。
```

**深层分析：**

```
AB⃗ = (1,3,-1)
2AB⃗ = (2,6,-2)
```

> **方法提炼：** 数乘就是每个分量乘以 k。

---

**例题 3：空间一点求中点**

```
题目：已知长方体 ABCD-A'B'C'D' 的顶点坐标：
      A(0,0,0)，B(4,0,0)，D(0,3,0)，C'(4,3,2)，
      求该长方体的中心（对角线交点）坐标。
```

**深层分析：**

长方体的对角线交点 = 任意一条对角线的中点。

一条对角线是 A 到 C'：
```
中点 = ((0+4)/2, (0+3)/2, (0+2)/2) = (2, 1.5, 1)
```

> **方法提炼：** 对角线交点 = 任一对角线中点 = 六个顶点坐标的平均值。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
