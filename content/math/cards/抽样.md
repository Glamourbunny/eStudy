# 抽样

> 抽样是用"一部分"推断"全体"的方法——从总体中抽取样本，用样本估计总体的特征。

---

## 概念层

### 问题的起源

不可能把全国所有人都调查一遍再得出结论。

怎么办？抽取一部分人作为**样本**，用样本的特征去估计总体的特征。

这就是**抽样调查**——用局部推断整体。

> **一句话：抽样是从总体中抽取样本，用样本估计总体的统计方法。**

---

### 本质定义

**抽样的核心问题：**

| 问题 | 说明 |
|------|------|
| **总体** | 想要研究的全部对象 |
| **样本** | 从总体中抽取的部分对象 |
| **样本容量** | 抽取的个数 n |
| **抽样方法** | 如何抽取才能保证代表性 |

---

### 常见抽样方法

| 方法 | 原理 | 适用场景 |
|------|------|---------|
| **简单随机抽样** | 每个个体被抽到的概率相等 | 总体差异不大 |
| **系统抽样** | 等间隔抽取 | 总体数量大 |
| **分层抽样** | 按比例从各层抽取 | 总体分层明显 |

### 概念辨析

**辨析：抽样 vs 普查**

| 对比项 | 抽样 | 普查 |
|--------|------|------|
| 调查范围 | 部分 | 全部 |
| 成本 | 低 | 高 |
| 精度 | 有误差 | 无抽样误差 |
| 适用 | 总体太大 | 总体不大 |

---

## 命题层

### 核心定理

**定理：简单随机抽样的代表性**

```
每个个体被抽到的概率相等
样本均值的期望等于总体均值
```

**定理：样本容量与精度**

```
样本容量越大，样本估计越精确
但边际效用递减
```

## 方法层

### 识别信号表

| 关键词 | 信号 | 方法 |
|--------|------|------|
| "抽样""样本" | 抽样问题 | 选合适抽样方法 |
| "随机抽样" | 简单随机 | 每个概率相等 |
| "分层抽样" | 分层抽样 | 按比例从各层抽取 |

---

### 方法详解：抽样方法的选择

**Step 1: 分析总体的特点**

```
总体差异大 → 分层抽样
总体差异小 → 简单随机抽样
```

**Step 2: 确定样本容量**

```
精度要求高 → 大样本
成本限制 → 小样本
```

**Step 3: 执行抽样**

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "抽样调查一定不准确" | 误解 | 抽样误差可控制 |
| 2 | "样本越大越好" | 误解 | 边际效用递减，成本增加 |
| 3 | "随机抽样就是随便抽" | 误解 | 随机抽样有严格的数学定义 |
