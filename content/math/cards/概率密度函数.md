# 概率密度函数

> 概率密度函数 f(x) 是连续型随机变量的"概率分布描述"——它本身不是概率，但通过积分给出概率。

---

## 概念层

### 问题的起源

连续型随机变量（如身高）无法一一列举每个取值的概率。

如何描述？
- 用**概率密度函数** f(x)
- P(a < X < b) = ∫_a^b f(x)dx

注意：f(x) 本身不是概率，而"曲线下的面积"才是概率。

> **一句话：概率密度函数 f(x) 的积分给出概率。**

---

### 本质定义

**概率密度函数的性质：**

| 性质 | 公式 |
|------|------|
| 非负性 | f(x) ≥ 0 |
| 归一性 | ∫_{-∞}^{+∞} f(x)dx = 1 |
| 概率计算 | P(a < X < b) = ∫_a^b f(x)dx |

---

### 与离散型分布列的对比

| 对比项 | 离散型分布列 | 连续型概率密度 |
|--------|-------------|---------------|
| 描述工具 | P(X = xᵢ) | f(x) |
| 概率形式 | 单点概率 | 区间积分 |
| 图像 | 散点图 | 连续曲线 |
| 总概率 | Σpᵢ = 1 | ∫f(x)dx = 1 |

---

## 命题层

### 核心定理

**定理：正态分布的概率密度**

```
f(x) = (1/√(2πσ²)) × e^[−(x−μ)²/(2σ²)]
```

**定理：均匀分布的概率密度**

```
f(x) = 1/(b-a)，a ≤ x ≤ b
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "f(x) 就是概率" | 误解 | f(x) 是密度，要积分才是概率 |
| 2 | "P(X = a) = f(a)" | 误解 | 连续型中 P(X=a) = 0 |
| 3 | "f(x) 可以大于 1" | 误解 | f(x) 可以大于 1，只要积分等于 1 |
