# 等差数列

> 等差数列是"每一步都迈同样大小"的数列——从第二项起，每一项与前一项的差都是同一个常数。

---

## 概念层

### 问题的起源

现实生活中，很多现象呈现出"等差"的特点：
- 每个月存 1000 元定期存款：本金逐月增加相同数额
- 等速行驶的汽车：每秒钟经过的距离相同
- 剧场座位：每排比前一排多固定数量的座位

这类"每一步变化量相同"的现象，用数学语言描述就是**等差数列**。

> **一句话：** 等差数列是"匀速变化"的数列——每一步迈出的步长（公差）固定。

---

### 原始定义（教材原文）

如果一个数列从第 2 项起，每一项与它的前一项的差都等于同一个常数，那么这个数列就叫做**等差数列**，这个常数叫做等差数列的**公差**，通常用 d 表示。

> 出处：人教版《普通高中教科书·数学（B版）选择性必修第二册》第 8 页

**定义拆解：**

| 关键词 | 含义 | 示例 |
|--------|------|------|
| 从第2项起 | 第一项没有前一项 | a₂ − a₁ = d |
| 同一个常数 | 公差 d 固定不变 | d 可以是正、负或零 |
| 公差 d | 衡量"步长"大小 | d > 0 递增，d < 0 递减，d = 0 常数列 |

---

### 本质定义（逻辑重构版）

**等差数列的核心特征：**

```
a₂ = a₁ + d
a₃ = a₂ + d = a₁ + 2d
a₄ = a₃ + d = a₁ + 3d
...
aₙ = a₁ + (n−1)d
```

**等差数列 = 首项 + (项数−1) × 公差**

---

### 等差数列的通项公式

```
aₙ = a₁ + (n−1)d

其中：
a₁ —— 首项
d  —— 公差
n  —— 项数
```

**通项公式的变形：**

```
aₙ = aₘ + (n−m)d    （已知第 m 项 aₘ，求第 n 项）
d  = (aₙ − aₘ)/(n−m)  （由两项求公差）
```

---

### 邻近概念辨析

**辨析：等差数列 vs 等比数列 —— "匀速" vs "变速（匀率）"**

| 对比项 | 等差数列 | 等比数列 |
|--------|---------|---------|
| 核心特征 | 每一步增加固定量 | 每一步乘以固定比例 |
| 关系式 | aₙ₊₁ − aₙ = d（差相同） | aₙ₊₁ / aₙ = q（比相同） |
| 公差/公比作用 | 衡量步长大小 | 衡量增速/缩速比例 |
| 通项公式 | aₙ = a₁ + (n−1)d | aₙ = a₁·q^(n−1) |
| 典型例子 | 1, 3, 5, 7, 9, ... | 1, 2, 4, 8, 16, ... |

---

### 适用边界

| 情形 | 公差 d 的影响 |
|------|-------------|
| d > 0 | 递增数列 |
| d = 0 | 常数列（所有项相同） |
| d < 0 | 递减数列 |
| d 任意实数 | 等差数列（可正可负） |

## 命题层

### 核心定理

**定理 1：等差数列的通项公式**

```
aₙ = a₁ + (n−1)d
```

**定理 2：等差中项**

若 a, b, c 成等差数列，则 b = (a + c)/2，b 叫做 a 与 c 的等差中项。

```
等差中项公式：b = (a + c)/2
```

**定理 3：等差数列的前 n 项和**

```
Sₙ = n(a₁ + aₙ)/2 = n[2a₁ + (n−1)d]/2
```

> **几何意义：** 前 n 项和相当于梯形面积 = (上底 + 下底) × 高 / 2

**定理 4：等差数列的判定**

数列 {aₙ} 是等差数列 ⟺ aₙ₊₁ − aₙ = 常数（对所有 n 成立）

---

### 典型例题

**例题 1：求等差数列的通项**

```
题目：等差数列满足 a₃ = 7，a₇ = 15，求通项公式和 a₂₀。
```

**深层分析：**

```
分析：已知两项，可用通项公式求公差

a₃ = a₁ + 2d = 7
a₇ = a₁ + 6d = 15

两式相减：(a₁+6d) − (a₁+2d) = 15 − 7
         4d = 8
         d = 2

代入 a₃ = a₁ + 2d = 7：
a₁ + 4 = 7
a₁ = 3

∴ aₙ = 3 + (n−1)·2 = 2n + 1

a₂₀ = 2×20 + 1 = 41
```

> **方法提炼：** 已知等差数列任意两项，可通过方程组求 a₁ 和 d。

---

**例题 2：等差数列前 n 项和的最值**

```
题目：等差数列 {aₙ} 满足 a₁ = 25，d = −2，求前多少项和最大？
```

**深层分析：**

```
分析：递增数列（d > 0 时）前 n 项和随 n 增大而增大；
      递减数列（d < 0 时）需要找"转折点"

找最后一个正项：
aₙ = 25 + (n−1)(−2) = 27 − 2n
令 aₙ ≥ 0：27 − 2n ≥ 0 → n ≤ 13.5

所以第 13 项为最后一个正项（a₁₃ = 27 − 26 = 1 > 0）
第 14 项开始为负（a₁₄ = 27 − 28 = −1）

∴ 前 13 项和最大

S₁₃ = 13×(2×25 + 12×(−2))/2 = 13×(50−24)/2 = 13×13 = 169
```

> **方法提炼：** d < 0 时，找最后一个非负项是求最大前 n 项和的关键。

---

**例题 3：等差中项的应用**

```
题目：已知三个数成等差数列，和为 15，平方和为 83，求这三个数。
```

**深层分析：**

```
设这三个数为：a−d, a, a+d（等差数列的对称表示）

则 (a−d) + a + (a+d) = 15 → 3a = 15 → a = 5

平方和：(5−d)² + 25 + (5+d)² = 83
      = (25−10d+d²) + 25 + (25+10d+d²) = 83
      = 75 + 2d² = 83
      2d² = 8
      d² = 4 → d = ±2

∴ 这三个数为：3, 5, 7 或 7, 5, 3
```

> **方法提炼：** 利用等差中项的对称表示（a−d, a, a+d）可以简化计算。

## 方法层

### 方法 1：等差数列的判断

**核心思想：** 验证"差是否相等"。

**标准操作流程：**

```
判断 {aₙ} 是否为等差数列：

Step 1: 计算 a₂ − a₁, a₃ − a₂, ..., aₙ − aₙ₋₁
Step 2: 检查这些差是否都相等
Step 3: 若相等则是等差数列，否则不是
```

---

### 方法 2：等差数列求和

**核心思想：** 用配对法简化计算。

**核心思想推导：**

```
Sₙ = a₁ + a₂ + ... + aₙ
    = a₁ + (a₁+d) + ... + [a₁+(n−1)d]

倒写 Sₙ：
Sₙ = aₙ + (aₙ−d) + ... + [aₙ−(n−1)d]

两式相加：
2Sₙ = n(a₁ + aₙ)
∴ Sₙ = n(a₁ + aₙ)/2
```

---

### 方法 3：等差数列通项与求和的综合

**核心思想：** 建立方程组求解未知量。

**识别信号表：**

| 题干关键词 | 识别信号 | 首选方法 |
|-----------|---------|---------|
| "已知 aₘ 和 aₙ" | 两已知项 | 通项公式建立方程组 |
| "求前 n 项和最大" | 递减数列+最值 | 找最后一个正项 |
| "三个数成等差数列" | 等差中项 | 对称表示法 |

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "等差数列公差一定为正" | 记忆固化 | d 可以是负数（递减数列） |
| 2 | "等差数列不能有重复项" | 忽视 d=0 情况 | d=0 时是常数列，各项相同 |
| 3 | "前 n 项和 Sₙ = n(a₁ + aₙ)/2 忘除以 2" | 粗心 | 梯形面积公式，必须除以 2 |
| 4 | "等差中项只有一个" | 理解错误 | a 与 c 的等差中项是 b = (a+c)/2，是唯一的 |
