# 先验概率

> **Intuitive 卡片不教你"怎么算"，而是教你"算什么"。** 先验概率在高考贝叶斯题中总是以"已知患病率 0.1%""盒子里有 3 红 2 白"的形式出现——它就是你在"看到新证据之前"手里握着的那个概率。你的任务是别把它和新证据混在一起。

---

## 视角层（像什么）

### 核心隐喻：出发前的天气预报——手里唯一的旧情报

**先验概率 = 出门前看的天气预报"今天降雨概率 30%"——在你亲眼看到窗外乌云之前，你对"下雨"这件事的全部知识。**

```
  早上 8:00：看天气预报 → 降雨概率 30%  ← 先验概率 P(雨)
  
  早上 9:00：走到阳台，看到满天乌云
  
  你的大脑自动更新：
  旧情报（30%）+ 新证据（乌云）→ 新判断（90%）
  
  这个"旧情报"就是先验概率——新证据到来前的基准线。
```

**先验概率的三种"身份"：**

| 身份 | 来源 | 例子 |
|------|------|------|
| **历史数据** | 过去大量统计的结果 | "该地区历史降雨概率 30%" |
| **抽样比例** | 不完整但已知的信息 | "盒子里有 3 个红球、2 个白球" |
| **默认假设** | 缺乏信息时的合理起点 | "假设硬币均匀，正面概率 50%" |

> **一句话直觉：** 先验概率是"新证据出现之前你对这件事的把握"。它不是猜的，它是基于已有信息的理性判断。贝叶斯做的事就是：拿新证据去修正这个先验。

### 动态想象：棋盘上的起始位置

```
  把概率更新想象成走棋：

  起始位置（先验概率）→ 第1步（新证据1）→ 第2步（新证据2）→ ……
  
  例：疾病诊断
  起始位置 P(病) = 0.1%            ← 先验概率
  走第1步：检测阳性（证据1）
    新位置 P(病|阳性) = ？        ← 后验概率
  
  而 P(阳性|病) = 99%（检测准确率）是"似然"——另一个概念
  
  三个概念不要混淆：
  • 先验 P(病) —— 你没做检测前就知道的
  • 似然 P(阳性|病) —— 检测给你多少"新信息"
  • 后验 P(病|阳性) —— 综合新旧信息后的答案
```

### 反直觉警示：大脑会在哪里翻车

| 翻车点 | 直觉告诉你 | 实际真相 |
|--------|-----------|---------|
| **忽略先验（基础概率谬误）** | "检测 99% 准确，阳性就是有病" | 如果先验 P(病) = 0.1%，一万个人里 10 个真病人中 10 个阳性，9990 个健康人中 100 个假阳性。阳性→有病概率 = 10/(10+100) ≈ 9%，不是 99% |
| **先验和新证据混在一起** | "我把已知的都扔进公式算" | 先验和新证据是乘法关系：后验 ∝ 先验 × 似然。把两者"加在一起"会严重错误 |
| **先验≠50%的默认值** | "不知道概率就当 50%" | "不知道"不是"等可能"！患病率不因为你不知道而变成 50% |
| **先验被过度信任** | "先验概率来自权威数据，永远对" | 先验可能过时、有偏、基于不同人群。新证据够强时，后验可能和先验差很远 |

## 联结层（连什么）

### 跨学科锚点

**1. 医学诊断：基础患病率**

```
  某种罕见病在人群中的患病率 = 0.01%（先验概率）
  
  医生不能因为"检测阳性"就直接下诊断
  必须把先验（0.01%）和检测准确率（99%）结合
  ↓ 贝叶斯公式给出的后验概率可能仍然很低
  ↓ 这就是为什么罕见病筛查经常需要二次确认
```

**2. 法律：无罪推定**

```
  法庭上："在证明有罪之前，被告是无罪的"
  
  先验 P(有罪) ≈ 0（无罪推定原则）
  检方提供证据 = 新信息（似然）
  陪审团更新判断 = 后验概率
  
  ↓ 刑事标准：后验必须 > "超出合理怀疑"（~99%）
  ↓ 民事标准：后验只需 > "优势证据"（~51%）
```

**3. 生活：面试第一印象**

```
  看简历时：先验 = 学校和 GPA（约 30% 的把握这人不错）
  面试后：新证据 = 谈吐、反应、思维（似然比）
  最终评估 = 先验 × 新证据 → 这才是理性的招聘决策
```

### 向 Formal 的桥梁

**拆包后，落回哪条 formal 定理？**

| 高考场景 | 拆包结论 | Formal 着陆点 |
|---------|---------|--------------|
| 贝叶斯公式题目中给出的"基础概率" | 这就是先验 P(A) | 全概率公式：P(B) = Σ P(B|A_i)·P(A_i) |
| "盒子里有 3 红 2 白"在第一问 | 这是先验分布 | 古典概型 + 全概率公式 |
| 多阶段概率题第一步概率 | 这是先验 | 贝叶斯公式：P(A|B) = P(B|A)·P(A)/P(B) |

**首选工具**：先列先验 P(A_i)，再列似然 P(B|A_i)，用全概率公式算分母
**次选工具**：画概率树，标注先验在根节点

### 概念网络坐标

```
              ┌─────────────────┐
              │    概率更新      │
              └────────┬────────┘
                       │
     ┌─────────────────┼─────────────────┐
     │                 │                 │
     ▼                 ▼                 ▼
┌─────────┐     ┌───────────┐     ┌─────────┐
│ 先验概率 │ ──→ │   新证据   │ ──→ │后验概率  │
│ P(因)   │  ×  │ P(果|因)  │  =  │ P(因|果) │
└─────────┘     └───────────┘     └─────────┘
     │                               │
     └─────── 全概率公式 ────────────┘
              （连接先验和全体的桥梁）
```

**拆包调用优先级：**
1. 题干第一句话的概率数值 → 几乎总是先验
2. 有多个"原因"且概率已知 → 先验分布
3. 题目问"在 B 发生下 A 的概率" → 目标后验，需要先验作为输入

### 历史/哲学视角：人类的"概率偏见"

```
18 世纪（贝叶斯牧师）：
  贝叶斯 1763 年发表论文，提出"逆概率"
  问题：已知"果"，如何反推"因"的概率？
  ↓ 在此之前，概率论只研究"从因到果"（正概率）
     贝叶斯问了一个反向问题：从结果反推原因

  他的答案是：先有一个"初始信念"（先验），然后用新证据修正
  ↓ 这在当时非常激进——概率竟然可以是"主观的"信念

20 世纪（频率学派 vs 贝叶斯学派）：
  频率学派：概率必须来自大量重复实验，"先验"是不科学的
  贝叶斯学派：先验是你对世界已有的合理知识，不用假装自己一无所知
  
  ↓ 这场争论到今天还在继续
  ↓ 但高考不选边站——它考的是"给定先验 + 新证据 → 后验"的计算过程
```

## 迁移层（何时调用）

### 三类场景判定表

| 场景 | 识别信号 | 不拆包的代价 | 拆包收益 |
|:---|:---|:---|:---|
| 🎭 **包装** | 题目在贝叶斯题中称先验为"基础概率""初始概率""背景发生率" | 没认出这是先验，把它和似然混在一起算 | 3 秒识别"这是 P(A)"，明确定位在全概率公式分母的哪一项 |
| 🔀 **嵌套** | 多阶段贝叶斯题：先算先验分布 P(A_i)，再算似然 P(B\|A_i)，最后用贝叶斯 | 先验被套在"第二步"需要反推回去，层次混淆 | 画因果链：先验→似然→全概率分母→贝叶斯后验，逐层清晰 |

### 拆包方向标（10 秒流程）

```
Step 1：识别结构类型（3 秒）
  • 贝叶斯题中"一开始就知道的概率"？
    → 这是先验，标为 P(A)
  • 多个"原因"各有概率？
    → 这是先验分布，用全概率公式

Step 2：剥外壳找骨骼（4 秒）
  • 画因果箭头：因（先验 P(A)）→ 果（B）
  • 标注似然：果|因 的箭头（P(B|A)）
  • 确认问题是：已知果，问因的概率 ← 贝叶斯方向

Step 3：锁定 Formal 着陆点（3 秒）
  • 先验 P(A) 放贝叶斯公式分子
  • 分母用全概率公式 = Σ P(B|A_i)·P(A_i)
  • 写下：拆包→【全概率/贝叶斯】，立即切换
```

### 向 Formal 切换铁律

> **拆包完成后，30 秒内必须切换为 formal 语言。**

| 场景 | 拆包结束的标志 | 切换后的 formal 动作 |
|:---|:---|:---|
| 包装题 | 识别出哪个数字是 P(A) | 列出：先验 P(A)、似然 P(B\|A)、目标 P(A\|B) |
| 嵌套题 | 因果链图画完 | 从先验出发，逐层用全概率和贝叶斯公式 |

---

### 直觉陷阱清单（仅拆包场景）

| 陷阱 | 何时失效 | 纠错方式 |
|------|---------|---------|
| **先验被忽略** | "阳性→有病"跳过先验 | 永远先写 P(病)，再写 P(阳性|病)，最后求 P(病|阳性) |
| **先验混淆** | 多个"原因 A_i"时不知道哪个先验最重要 | 所有 P(A_i) 列出，和为 1，选目标那个 |
| **新证据当先验** | 误把 P(B|A) 当 P(A) | 问自己：这个概率是在"还不知道 B"时就知道的吗？是→先验，否→不是 |

---

## 一句话总结

> **先验概率是"新证据到来之前你的全部底牌"。高考贝叶斯题里它是第一个数字、第一句话给你的概率。你的任务是在计算中把它放在"原因"的位置上，不让它被"新证据"淹没。**
