# 质子守恒

> 质子守恒是水溶液中"质子转移的平衡"——它与电荷守恒、物料守恒并称为溶液离子平衡的三大守恒。

---

## 概念层

### 问题的起源

水的电离：H₂O ⇌ H⁺ + OH⁻

在溶液中，H⁺ 的来源有多种：
- 水的电离
- 酸的电离

质子守恒考虑的是溶液中 H⁺ 的来源和去向的平衡。

对于弱酸HA溶液：
```
HA ⇌ H⁺ + A⁻
H₂O ⇌ H⁺ + OH⁻
```

质子守恒式：c(H⁺) = c(A⁻) + c(OH⁻)

这联系了溶液中各种含 H⁺ 的离子。

> **一句话：质子守恒是溶液中 H⁺ 来源和去向的平衡关系。**

---

### 本质定义

**质子守恒的本质是"质子转移平衡"：**

```
溶液中质子（H⁺）的转移有两条途径：
① 来自酸（如 HA → H⁺ + A⁻）
② 来自水（H₂O → H⁺ + OH⁻）

而溶液中的 H⁺ 会：
① 与碱结合（如 H⁺ + A⁻ → HA）
② 与 OH⁻ 结合（H⁺ + OH⁻ → H₂O）

质子守恒：得 H⁺ 的物质 = 失 H⁺ 的物质
```

---

### 三大守恒的关系

| 守恒 | 核心关系 | 适用对象 |
|------|---------|---------|
| **物料守恒** | 原子数量不变 | 所有反应 |
| **电荷守恒** | 电荷数量不变 | 离子反应 |
| **质子守恒** | 质子转移平衡 | 水溶液 |

### 典型例题

**例题：写出 NH₃·H₂O 溶液的质子守恒式**

```
题目：写出氨水（NH₃·H₂O）的质子守恒式。
```

**深层分析：**

```
NH₃·H₂O 是弱碱，电离：
NH₃·H₂O ⇌ NH₄⁺ + OH⁻

水的电离：
H₂O ⇌ H⁺ + OH⁻

质子守恒：得 H⁺ 的物质 = 失 H⁺ 的物质

得 H⁺ 的物质：NH₄⁺（NH₄⁺ + H⁺ ← NH₃·H₂O）
失 H⁺ 的物质：OH⁻（OH⁻ = H⁺ + OH⁻，来自水的电离，但实际是失 H⁺）

更准确的推导：
以 H₂O 和 NH₃·H₂O 为参考

得质子产物：H₃O⁺（即 H⁺）、NH₄⁺
失质子产物：OH⁻

质子守恒式：c(NH₄⁺) + c(H⁺) = c(OH⁻)
```

> **方法提炼：** 以溶剂和溶质为参考，分析质子转移。

### 识别信号表

| "三大守恒""水溶液" | 溶液离子平衡 | 三种守恒都要满足 |
| "得 H⁺""失 H⁺" | 质子转移 | 得失质子数相等 |

---

### 方法详解：质子守恒的写法

**Step 1: 确定参考物质（溶剂和溶质）**

```
通常以 H₂O 和溶质本身为参考
```

**Step 2: 列出得 H⁺ 的产物**

```
得 H⁺：酸失去 H⁺ 后的共轭碱
```

**Step 3: 列出失 H⁺ 的产物**

```
失 H⁺：碱得到 H⁺ 后的共轭酸
```

**Step 4: 写出守恒式**

```
得 H⁺ 的物质浓度之和 = 失 H⁺ 的物质浓度之和
```
