# 不等式

> 不等式是"比大小"的数学表达——它描述的是大小关系，而非相等。

---

## 概念层

### 问题的起源

数学起源于"多少"的问题。

- 3 > 2（3 比 2 大）
- x + 1 < 5（x 加 1 小于 5）
- a² + b² ≥ 2ab（平方和至少是两倍积）

不等式把"大""小""等"用符号精确表达。

> **一句话：不等式是用符号表示大小关系的数学表达式。**

---

### 原始定义（教材原文）

用符号 >、<、≥、≤、≠ 表示大小关系的式子称为**不等式**。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 26-30 页

---

### 本质定义

**五种不等号：**

| 符号 | 含义 | 读作 |
|------|------|------|
| > | 大于 | 严格大于 |
| < | 小于 | 严格小于 |
| ≥ | 大于或等于 | 不小于 |
| ≤ | 小于或等于 | 不大于 |
| ≠ | 不等于 | 不等于 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **左式** | 不等号左边的表达式 |
| **不等号** | >, <, ≥, ≤, ≠ |
| **右式** | 不等号右边的表达式 |

---

### 基本性质

| 性质 | 公式 |
|------|------|
| 对称性 | a > b ⟺ b < a |
| 传递性 | a > b 且 b > c ⟹ a > c |
| 加法保序 | a > b ⟹ a + c > b + c |
| 乘法保序（正数） | a > b 且 c > 0 ⟹ ac > bc |
| 乘法变序（负数） | a > b 且 c < 0 ⟹ ac < bc |

### 概念辨析

**辨析：> vs ≥**

| 对比项 | > | ≥ |
|--------|---|---|
| 名称 | 严格不等式 | 非严格不等式 |
| 等号是否包含 | 不包含 | 包含 |
| 示例 | a > 0（正数） | a ≥ 0（非负数） |

## 命题层

### 核心定理

**定理 1：不等式的基本性质**

```
① 传递性：a > b, b > c ⟹ a > c
② 加法：a > b ⟹ a + c > b + c
③ 正数乘法：a > b, c > 0 ⟹ ac > bc
④ 负数乘法：a > b, c < 0 ⟹ ac < bc
⑤ 倒数：a > b > 0 ⟹ 1/a < 1/b
```

**定理 2：解不等式的基本原则**

```
不等式两边同时加减一个数，不等号方向不变
不等式两边同时乘除一个正数，不等号方向不变
不等式两边同时乘除一个负数，不等号方向改变
```

---

### 典型例题

**例题：解不等式 2x + 3 > 7**

```
题目：解不等式 2x + 3 > 7
```

**深层分析：**

```
2x + 3 > 7
    ↓
2x > 7 − 3（移项）
    ↓
2x > 4
    ↓
x > 2（两边除以 2，正数不等号不变）
```

> **方法提炼：** 移项 → 合并 → 两边除以系数（注意正负）。

### 方法详解：解一次不等式步骤

**Step 1: 移项**

```
把含 x 的项移到一边，常数项移到另一边
```

**Step 2: 合并同类项**

**Step 3: 两边除以 x 的系数**

```
正系数：不等号方向不变
负系数：不等号方向改变
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "两边乘负数不等号不变" | 记错 | 乘负数要变方向 |
| 2 | "a > b 则 1/a < 1/b" | 忽略符号 | 只有 a, b 同号且为正时才成立 |
| 3 | "移项不改符号" | 粗心 | 移项相当于加减，要变号 |
