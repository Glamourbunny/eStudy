# 超几何分布

> 超几何分布是"不放回抽样"时的分布——从 N 件产品中抽出 n 件，其中 K 件次品，求恰好 k 件次品的概率。

---

## 概念层

### 问题的起源

工厂生产了 N 件产品，其中有 K 件次品。不放回地抽出 n 件，恰好抽到 k 件次品的概率是多少？

这就是超几何分布的场景——**不放回抽样**。

> **一句话：超几何分布描述的是不放回抽样中"成功次数"的分布。**

---

### 本质定义

**超几何分布的定义：**

```
X ~ H(N, K, n)
P(X = k) = C(K, k) × C(N-K, n-k) / C(N, n)
其中 max(0, n-(N-K)) ≤ k ≤ min(K, n)
```

**参数意义：**

| 参数 | 含义 |
|------|------|
| N | 总体产品数 |
| K | 次品总数 |
| n | 抽样数量 |

---

### 与二项分布的关系

| 对比项 | 超几何分布 | 二项分布 |
|--------|-----------|---------|
| 抽样方式 | 不放回 | 放回 |
| 概率计算 | 组合公式 | 独立重复 |
| 近似关系 | N 很大时近似二项分布 | n 次独立试验 |

当 N 很大、n 相对较小时，超几何分布近似二项分布 B(n, K/N)。

---

## 命题层

### 核心定理

**定理：超几何分布的分布列**

```
P(X = k) = C(K, k) × C(N-K, n-k) / C(N, n)
```

**定理：期望和方差**

```
E(X) = n × K/N
Var(X) = n × K/N × (1-K/N) × (N-n)/(N-1)
```

---

### 典型例题

**例题：超几何分布概率计算**

```
题目：盒中有 10 个零件，其中 3 个次品。不放回抽取 4 个，求恰好 2 个次品的概率。
```

**深层分析：**

```
N = 10, K = 3, n = 4, k = 2

P(X=2) = C(3,2) × C(7,2) / C(10,4)
        = 3 × 21 / 210
        = 63/210
        = 3/10
```

> **方法提炼：** 直接代入公式。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "超几何分布是有放回抽样" | 混淆 | 正好相反，是不放回 |
