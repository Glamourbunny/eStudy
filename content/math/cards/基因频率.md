# 基因频率

> 基因频率是"群体中某基因占所有等位基因的比例"——它是群体遗传学的核心概念。

---

## 概念层

### 问题的起源

一个种群中，基因 A 和基因 a 的频率是多少？

如果 A 频率是 p，a 频率是 q = 1-p，那么基因型频率：
- AA: p²
- Aa: 2pq
- aa: q²

这就是**哈迪-温伯格定律**。

> **一句话：基因频率 p 和 q 是群体中等位基因的比例。**

---

### 本质定义

**基因频率的定义：**

```
基因 A 的频率 = AA 个体数×2 + Aa 个体数 / 总等位基因数×2
```

**哈迪-温伯格平衡：**

```
在大种群、随机交配、无选择、无突变、无迁移条件下
基因频率代代保持不变
```

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **等位基因** | 同一基因座的不同形式 |
| **基因频率** | p（A频率）+ q（a频率）= 1 |
| **基因型频率** | AA: p², Aa: 2pq, aa: q² |

---

## 命题层

### 核心定理

**定理：哈迪-温伯格定律**

```
基因频率 p, q
基因型频率：AA = p², Aa = 2pq, aa = q²
```

**定理：遗传平衡条件**

```
① 大种群
② 随机交配
③ 无选择
④ 无突变
⑤ 无迁移
```
