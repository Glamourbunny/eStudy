# 一阶ODE

> 一阶ODE是包含一阶导数的方程——描述"变化率由当前状态决定"的规律。

---

## 概念层

### 问题的起源

在物理、生物、经济等领域，很多规律可以用"变化率等于某个函数"来描述：
- 放射性衰变：dN/dt = −λN（衰变率与当前数量成正比）
- 人口增长：dp/dt = kp（增长率与当前人口成正比）
- 冷却问题：dT/dt = −k(T−T₀)（温差的变化率与温差成正比）

这类问题中，未知的是函数，方程中出现导数，这就是**微分方程**。

> **一句话：** 一阶ODE是"变化的规律"——告诉我们在任何时刻，变化率如何由当前状态决定。

---

### 本质定义（逻辑重构版）

**一阶常微分方程的标准形式：**

```
dy/dx = f(x, y)

或者等价地：
M(x)dx + N(y)dy = 0
```

**解微分方程就是找满足方程的函数 y = f(x)**

---

### 一阶ODE的主要类型

| 类型 | 标准形式 | 解法 |
|------|---------|------|
| 可分离变量 | dy/dx = f(x)·g(y) | 分离变量后两边积分 |
| 齐次方程 | dy/dx = f(y/x) | 设 u = y/x |
| 一阶线性 | dy/dx + P(x)y = Q(x) | 用积分因子 |

---

### 分离变量法

**核心思想：** 把 x 和 y 分离到等式两边，分别积分。

```
dy/dx = f(x)·g(y)
    ↓
dy/g(y) = f(x)dx
    ↓
∫ dy/g(y) = ∫ f(x)dx
    ↓
解出 y = ...
```

---

## 命题层

### 典型例题

**例题：分离变量法**

```
题目：求解 dy/dx = y，y(0) = 1
```

**深层分析：**

```
dy/dx = y
    ↓
dy/y = dx（分离变量）
    ↓
∫ dy/y = ∫ dx
    ↓
ln|y| = x + C
    ↓
|y| = e^(x+C) = e^C · e^x
    ↓
y = ±Ae^x

由初始条件 y(0) = 1：
1 = ±A·e^0
∴ A = 1，且取正号

答案：y = e^x
```

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "积分后忘加常数 C" | 粗心 | 解微分方程必须加 C |
| 2 | "分离变量时除以 g(y) 忘讨论零解" | 不完整 | g(y) = 0 可能是特解 |
| 3 | "初始条件代错时机" | 时机不清 | 先求通解，再代初始条件求 C |
