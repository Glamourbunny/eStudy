# 四则运算

> 复数的四则运算，是用实数和虚数单位 i 做加减乘除——规则与实数相同，只需要记得 i² = −1。

---

## 概念层

### 问题的起源

引入复数后，首先要解决：复数如何做加减乘除？

复数的四则运算必须满足：
1. 兼容实数运算（当虚部为 0 时，结果与实数运算一致）
2. 封闭性（任意两个复数运算结果还是复数）
3. 运算律（交换律、结合律、分配律）

> **一句话：复数四则运算是实数运算的"升级版"，只需增加一条规则——i² = −1。**

---

### 原始定义（教材原文）

复数的四则运算法则：

**加法：** (a + bi) + (c + di) = (a + c) + (b + d)i

**减法：** (a + bi) − (c + di) = (a − c) + (b − d)i

**乘法：** (a + bi)(c + di) = (ac − bd) + (ad + bc)i

**除法：** (a + bi) ÷ (c + di) = (ac + bd)/(c² + d²) + (bc − ad)/(c² + d²)i（其中 c + di ≠ 0）

> 出处：人教版《普通高中教科书·数学（A版）必修第二册》第 21-23 页

---

### 本质定义

**加法/减法：** 对应虚部，实部虚部分别相加减

**乘法：** 按多项式乘法展开，i² 替换为 −1

**除法：** 分子分母同乘分母的共轭复数，分母实数化

```
记忆口诀：乘法"穿针引线"
(a + bi)(c + di)
= ac + adi + bci + bdi²
= ac + (ad + bc)i + bd·(−1)
= (ac − bd) + (ad + bc)i
```

---

### 定义的三要素

| 运算 | 核心规则 | 结果 |
|------|---------|------|
| **加法/减法** | 实部虚部分别加减 | 还是复数 |
| **乘法** | 多项式展开 + i² = −1 | 还是复数 |
| **除法** | 同乘共轭 + 分母实数化 | 还是复数 |

---

### 等价表述

| 运算 | 代数形式 | 复平面视角 |
|------|---------|-----------|
| **加法** | (a+bi)+(c+di) | 向量加法（平行四边形） |
| **减法** | (a+bi)−(c+di) | 向量减法 |
| **乘法** | (a+bi)(c+di) | 模长相乘，辐角相加 |
| **除法** | (a+bi)/(c+di) | 模长相除，辐角相减 |

### 图像表示

**复数加法的几何意义：**

```
Im(z)
  │
  │• P(a+bi,c+di)
  │ ╱
  │╱
  │╱ (a+bi)
  O────────────────→ Re(z)
  │• Q
  │• R
```

复数加法对应平面向量的加法（平行四边形法则）。

**复数乘法的几何意义：**

```
|z₁z₂| = |z₁|·|z₂|（模相乘）
arg(z₁z₂) = arg(z₁) + arg(z₂)（辐角相加）
```

旋转和伸缩的复合。

### 概念辨析

**辨析 1：复数乘法 vs 实数乘法**

| 对比项 | 实数乘法 | 复数乘法 |
|--------|---------|---------|
| **结果** | 仍是实数 | 仍是复数 |
| **几何意义** | 放大/缩小 | 旋转+伸缩 |
| **交换律** | ✓ | ✓ |
| **结合律** | ✓ | ✓ |
| **分配律** | ✓ | ✓ |

**辨析 2：复数除法 vs 复数乘法**

除法是乘法的逆运算：
```
(a + bi) ÷ (c + di) = (a + bi) × (c + di)⁻¹
```

求 (c + di)⁻¹ 的方法：
```
(c + di)⁻¹ = 1/(c + di) = (c − di)/(c² + d²)
```

> **一句话：** 除以一个复数，等于乘以它的倒数；求倒数的方法是同乘共轭复数。

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 复数 + 复数 | ✓ | 结果仍是复数 |
| 复数 × 复数 | ✓ | 封闭性 |
| 复数 ÷ 复数 | ✓（分母 ≠ 0） | 除数不能为 0 |
| i² | = −1 | 不是 +1 |

## 命题层

### 核心定理

**定理 1：四则运算律**

```
交换律：z₁ + z₂ = z₂ + z₁，z₁·z₂ = z₂·z₁
结合律：(z₁ + z₂) + z₃ = z₁ + (z₂ + z₃)，(z₁·z₂)·z₃ = z₁·(z₂·z₃)
分配律：z₁(z₂ + z₃) = z₁z₂ + z₁z₃
```

**定理 2：共轭复数的运算性质**

```
z + z̄ = 2Re(z)（实数）
z − z̄ = 2i·Im(z)（纯虚数或 0）
z·z̄ = |z|²（实数）
```

**定理 3：模的运算**

```
|z₁z₂| = |z₁|·|z₂|
|z₁/z₂| = |z₁|/|z₂|
|z₁ + z₂| ≤ |z₁| + |z₂|（三角不等式）
```

---

### 典型例题

**例题 1：乘法计算**

```
题目：计算 (1 + i)²
```

**深层分析：**

```
(1 + i)² = 1 + 2i + i² = 1 + 2i − 1 = 2i
```

> **方法提炼：** (1 + i)² = 2i，这是一个常用的结论。

---

**例题 2：除法计算**

```
题目：计算 (1 + i)/(1 − i)
```

**深层分析：**

```
(1 + i)/(1 − i) = (1 + i)(1 + i)/(1 − i)(1 + i)
                = (1 + 2i + i²)/(1 + 1)
                = (1 + 2i − 1)/2
                = 2i/2
                = i
```

> **方法提炼：** 分母实数化——同乘分母的共轭。

---

**例题 3：纯计算**

```
题目：已知 z = (1 + 2i)/(3 − i)，求 z 的实部和虚部。
```

**深层分析：**

```
z = (1 + 2i)/(3 − i)
  = (1 + 2i)(3 + i)/(9 + 1)
  = (3 + i + 6i + 2i²)/10
  = (3 + 7i − 2)/10
  = (1 + 7i)/10
  = 1/10 + 7/10 i
```

实部 = 1/10，虚部 = 7/10。

> **方法提炼：** 除法结果分离实部虚部即可。

## 方法层

### 识别信号表

| 题目特征 | 识别信号 | 方法 |
|---------|---------|------|
| "计算 (a+bi)±(c+di)" | 加减法 | 实部虚部分别运算 |
| "计算 (a+bi)(c+di)" | 乘法 | 多项式展开 + i²=−1 |
| "计算 (a+bi)/(c+di)" | 除法 | 同乘共轭复数 |
| "求实部/虚部/模" | 运算后处理 | 先化简再提取 |

---

### 方法详解：除法标准流程

**Step 1: 判断是否需要分母实数化**

只有除法需要，步骤：

```
Step 1: 分子分母同乘分母的共轭 (c − di)
Step 2: 分母变成 c² + d²（实数）
Step 3: 分子按乘法展开
Step 4: 分离实部和虚部
```

> **易错提醒：** i² = −1 不是 i² = 1。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | "i² = 1" | 记反 | i² = −1 |
| 2 | 除法忘记同乘共轭 | 漏步骤 | 必须分子分母同乘分母的共轭 |
| 3 | i³ 计算错误 | 不记得周期 | i³ = i²·i = −i |
| 4 | (a+bi)/(c+di) 直接分项 | 错误 | 必须同乘共轭实数化 |
