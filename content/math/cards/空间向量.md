# 空间向量

> 空间向量是平面向量的"升级版"——从二维平面跃迁到三维空间，它让立体几何的证明和计算变得像解析几何一样程序化。

---

## 概念层

### 问题的起源

平面几何能解决的问题有限。当研究对象进入三维空间时：

- 三棱锥的线面角如何计算？
- 空间的平面与直线如何描述？
- 两个平面之间的夹角怎么度量？

平面向量只解决"平面内"的问题。空间向量将向量概念推广到三维空间，使得：
- 立体几何的证明可以用向量运算完成
- 空间的方向关系可以用坐标精确描述
- 线面角、二面角等度量问题可以系统化求解

> **一句话：空间向量让三维空间中的方向和长度问题变得可以计算。**

---

### 原始定义（教材原文）

空间中具有大小和方向的量称为**空间向量**，记作 $\vec{a}$、$\vec{AB}$ 等。

空间向量用有序三元组表示：$\vec{a} = (a_x, a_y, a_z)$，其中 $a_x, a_y, a_z$ 是向量在 x、y、z 轴上的分量。

> 出处：人教版《普通高中教科书·数学（A版）选择性必修第一册》第 4-6 页

---

### 本质定义

**空间向量的本质是三维有向线段：**

```
从 A(x₁,y₁,z₁) 到 B(x₂,y₂,z₂) 的向量：
AB⃗ = (x₂−x₁, y₂−y₁, z₂−z₁)
```

**与平面向量的区别：**

| 维度 | 平面向量 | 空间向量 |
|------|---------|---------|
| 坐标分量 | (x, y) | (x, y, z) |
| 分量个数 | 2 | 3 |
| 运算 | 二维 | 三维 |
| 应用场景 | 平面几何 | 立体几何 |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **大小（模长）** | |\vec{a}| = √($a_x^2 + a_y^2 + a_z^2$) |
| **方向** | 由三个坐标分量决定 |
| **表示** | 有序三元组 (x, y, z) |

---

### 等价表述

| 表述形式 | 公式 | 适用场景 |
|----------|------|---------|
| **坐标形式** | \vec{a} = (a_x, a_y, a_z) | 一般运算 |
| **起点终点形式** | \vec{AB} = (x_B−x_A, y_B−y_A, z_B−z_A) | 几何问题 |
| **单位向量形式** | \vec{a} = |\vec{a}|·\vec{e_a} | 方向分析 |

### 邻近概念网络

```
              向量体系
                  │
      ┌───────────┼───────────┐
      │           │           │
   平面向量    空间向量     向量运算
   (二维)     (三维)      (加减数乘)
      │           │           │
   数量积      数量积      应用
   线性运算    向量积      (线面角等)
```

**与平面向量的关系：**
- 平面向量是空间向量的特例（z 分量为 0）
- 空间向量的运算律与平面向量完全一致
- 空间向量的几何应用比平面向量更丰富

---

### 概念辨析

**辨析 1：空间向量 vs 平面向量**

| 对比项 | 平面向量 | 空间向量 |
|--------|---------|---------|
| 坐标分量 | 两个 (x, y) | 三个 (x, y, z) |
| 应用范围 | 平面几何 | 立体几何 |
| 能表示的方向 | 平面内的任意方向 | 空间的任意方向 |
| 夹角范围 | 0°~180° | 0°~180° |

**辨析 2：向量 vs 坐标**

向量 $\vec{AB} = (x_B−x_A, y_B−y_A, z_B−z_A)$ 是有向线段，描述的是"从哪到哪"和"方向+长度"。

点 (x, y, z) 描述的是一个位置，没有方向。

```
点 A → 向量 OA⃗ = (x_A, y_A, z_A)
向量 OA⃗ 的起点是原点，终点是 A
```

---

### 适用边界

| 条件 | 是否成立 | 说明 |
|------|---------|------|
| 向量加减 | ✓ | 分量分别加减 |
| 数乘 | ✓ | 每个分量乘以实数 |
| 数量积 | ✓ | 三个分量对应相乘相加 |
| 向量积（叉积） | 高中不要求 | 大学内容 |

## 命题层

### 核心定理

**定理 1：空间向量的坐标运算**

```
\vec{a} = (a_x, a_y, a_z)，\vec{b} = (b_x, b_y, b_z)
加减：\vec{a} ± \vec{b} = (a_x±b_x, a_y±b_y, a_z±b_z)
数乘：k·\vec{a} = (k·a_x, k·a_y, k·a_z)
```

**定理 2：空间向量的模长**

```
|\vec{a}| = √(a_x² + a_y² + a_z²)
```

**定理 3：空间向量的数量积**

```
\vec{a}·\vec{b} = a_x·b_x + a_y·b_y + a_z·b_z
```

**定理 4：空间两点间距离**

```
A(x₁,y₁,z₁)，B(x₂,y₂,z₂)
|AB| = √((x₂−x₁)² + (y₂−y₁)² + (z₂−z₁)²)
```

---

### 典型例题

**例题 1：空间向量的坐标运算**

```
题目：已知 A(1,2,3)，B(2,4,1)，求 AB⃗ 的坐标和模长。
```

**深层分析：**

```
AB⃗ = (2−1, 4−2, 1−3) = (1, 2, −2)
|AB⃗| = √(1² + 2² + (−2)²) = √(1+4+4) = √9 = 3
```

> **方法提炼：** 向量 = 终点坐标 − 起点坐标，模长用公式。

---

**例题 2：空间向量平行判定**

```
题目：判断 AB⃗ = (1,2,3) 与 CD⃗ = (2,4,6) 是否平行。
```

**深层分析：**

CD⃗ = 2×(1,2,3) = 2·AB⃗，所以两向量平行（实际上是同向的）。

一般判断：$\vec{a} // \vec{b}$ ⟺ $\vec{a} = k·\vec{b}$（分量成比例）

> **方法提炼：** 分量对应成比例则平行。

## 方法层

### 识别信号表

| 关键词 | 识别信号 | 方法 |
|--------|---------|------|
| "空间向量""三维" | 向量问题 | 坐标运算 |
| "求模长""长度" | 模长计算 | √(x²+y²+z²) |
| "平行""共线" | 方向关系 | 分量成比例 |
| "垂直""90°" | 垂直关系 | 数量积为 0 |

---

### 方法详解：空间向量解题流程

**Step 1: 建立空间直角坐标系**

```
选择合适的点作为原点，标清 x, y, z 轴的方向
```

**Step 2: 将几何条件转化为向量坐标**

```
点 → 坐标
直线 → 方向向量
平面 → 法向量
```

**Step 3: 用向量运算求解**

```
长度问题 → 模长公式
角度问题 → 数量积公式
垂直/平行 → 数量积/比例关系
```

> **易错提醒：** 空间向量的第三个分量 z 很容易漏掉。

---

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |
|------|---------|---------|---------|
| 1 | 漏掉 z 分量 | 二维思维惯性 | 空间向量有 x, y, z 三个分量 |
| 2 | 向量与坐标混淆 | 不理解向量本质 | 向量是有方向的有向线段，坐标只是它的表示 |
| 3 | 模长计算少平方 | 计算错误 | \|a\| = √(a_x² + a_y² + a_z²) |
| 4 | 平行判定只比较两个分量 | 不完整 | 三个分量都要成比例 |
