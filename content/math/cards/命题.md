# 命题

> 命题是"可以判断真假"的陈述句——能判断对错的就是命题，不能判断对错的就不是。

---

## 概念层

### 问题的起源

数学需要严谨的逻辑。什么是"可以讨论对错"的？

- "3 是奇数" → 真命题
- "2 + 2 = 5" → 假命题
- "x > 3" → 不能判断真假（除非知道 x 的值）

只有第一种和第二种这样的陈述句才是命题。

> **一句话：命题是有真假之分的陈述句。**

---

### 原始定义（教材原文）

可以判断真假的陈述句叫做**命题**。

判断为真的命题叫做**真命题**，判断为假的命题叫做**假命题**。

> 出处：人教版《普通高中教科书·数学（A版）必修第一册》第 10-12 页

---

### 本质定义

**命题的本质是"有真假判定"的语句：**

| 语句类型 | 示例 | 是否命题 |
|---------|------|---------|
| 陈述句，有确定真假 | "今天下雨" | ✓ |
| 数学陈述句 | "1 + 1 = 2" | ✓（真） |
| 数学陈述句 | "0.5 是整数" | ✓（假） |
| 无法判断真假 | "x > 3" | ✗（x 未定） |
| 感叹句 | "哇！" | ✗ |
| 疑问句 | "你好吗？" | ✗ |
| 命令句 | "请开门" | ✗ |

---

### 定义的三要素

| 要素 | 说明 |
|------|------|
| **陈述句** | 必须是陈述句（不是疑问句、感叹句等） |
| **确定性** | 能判断真假 |
| **真假值** | 真或假 |

---

### 等价表述

| 表述 | 说明 |
|------|------|
| **真命题** | 判断为真的命题 |
| **假命题** | 判断为假的命题 |
| **真值** | 真（True）或假（False） |

### 适用边界

| 条件 | 是否是命题 | 说明 |
|------|-----------|------|
| 陈述句，能判断真假 | ✓ | 标准命题 |
| 疑问句/感叹句/命令句 | ✗ | 不是命题 |
| 含有变量，无法判断真假 | ✗ | 不是命题 |

## 命题层

### 核心定理

**定理 1：命题的真假判定**

```
命题 p 为真 ⟺ p 正确
命题 p 为假 ⟺ p 错误
```

**定理 2：复合命题的真值表**

| p | q | p ∧ q | p ∨ q | ¬p |
|---|---|-------|-------|----|
| 真 | 真 | 真 | 真 | 假 |
| 真 | 假 | 假 | 真 | 假 |
| 假 | 真 | 假 | 真 | 真 |
| 假 | 假 | 假 | 假 | 真 |

---

### 典型例题

**例题：判断是否是命题**

```
题目：判断以下是否是命题：
(1) 北京是中国的首都
(2) x > 5
(3) 这道题真难！
(4) 2 + 3 = 6
```

**深层分析：**

(1) ✓ 真命题
(2) ✗ 命题含有变量 x，无法判断真假
(3) ✗ 感叹句，不是陈述句
(4) ✓ 假命题（2+3=5）

> **方法提炼：** 能判断真假 + 是陈述句 = 命题。

## 常见误区与纠正

| 编号 | 常见误区 | 错误根源 | 正确理解 |

| 1 | "x > 3 是假命题" | 误解 | 含有变量，无法判断真假，不是命题 |
