# 几何与度量链 — 学生疑问

## 关于「平面向量」的问题

**Q1: [向量到底是有方向的线段，还是一个数——为什么向量不能像数字一样直接比大小]**
向量有大小和方向，但我总搞不清楚：两个向量到底能不能比较大小？比如向量 (1,2) 和 (2,1) 哪个更大？如果不能比大小，那我们说的"向量运算"和普通代数运算有什么区别？
> 产生这个疑问的原因：向量既像线段又像坐标，但数字可以直接比大小，向量却不行，感觉很不一致。

**Q2: [向量相等是什么意思——明明位置不同，为什么说它们相等]**
向量 (1,2) 从原点出发和从 (3,4) 出发，为什么都叫向量 (1,2)？这不是两个不同的有向线段吗？为什么说它们相等？
> 产生这个疑问的原因：日常概念里，"相等"意味着同一个东西，但向量的起点可以不同，位置不同却说相等，不太理解。

**Q3: [零向量到底有没有方向——为什么说方向任意又说不确定]**
教材说零向量大小为 0，方向任意。但"方向任意"是什么意思？是所有方向都可以，还是没有方向？如果没有方向，为什么还要说"方向任意"？
> 产生这个疑问的原因：零向量既不像其他向量有明确方向，又不说没有方向，搞不清楚它的定位。

## 关于「正弦定理」的问题

**Q4: [正弦定理里为什么要用 sin——sin 是怎么和边长挂上钩的]**
正弦定理说 a/sinA = b/sinB = c/sinC，但我一直在想：为什么偏偏是 sin，不是 cos 或 tan？这个公式是怎么推导出来的？sin 和三角形边长之间有什么内在联系？
> 产生这个疑问的原因：记住了公式会做题，但不理解公式背后的几何意义或代数推导。

**Q5: [SSA 情况为什么会产生双解——两个解都 valid 吗]**
已知 a = 5, b = 7, A = 30°，用正弦定理求 sinB = 7×sin30°/5 = 0.7，得到 B₁ ≈ 44.4° 和 B₂ ≈ 135.6°。两个解都符合三角形内角和 180° 吗？考试时两个都要写吗？
> 产生这个疑问的原因：知道 SSA 可能双解，但不知道什么条件下两个解都 valid，什么条件下要舍去。

**Q6: [正弦定理和余弦定理到底怎么选择——什么时候用哪个]**
拿到一道解三角形题目，我怎么判断该用正弦定理还是余弦定理？有没有简单的判断标准？
> 产生这个疑问的原因：教材讲了两个定理的适用条件，但实际做题时经常不知道该选哪个。

**Q7: [大边对大角是什么意思——这是显然的吗]**
教材说"大边对大角"，我凭直觉觉得对，但为什么？这个结论的几何意义是什么？
> 产生这个疑问的原因：知道这个结论很有用，但不理解为什么成立。

## 关于「余弦定理」的问题

**Q8: [余弦定理和勾股定理是什么关系——为什么 C = 90° 时就变成勾股定理]**
教材说余弦定理是勾股定理的"升级版"，当 C = 90° 时，cosC = 0，代入得 c² = a² + b²，这正是勾股定理。但我不理解：为什么余弦定理恰好包含了勾股定理这个特殊情形？这背后有什么更深的联系？
> 产生这个疑问的原因：知道两者的关系，但不理解为什么余弦定理恰好是勾股定理的一般化。

**Q9: [余弦定理中符号为什么是负的——为什么是减不是加]**
余弦定理 c² = a² + b² - 2ab·cosC，其中有一项是"-2ab·cosC"。我老是记不住符号，或者记成"+2ab·cosC"。这个负号是从哪来的？有没有直观理解的方法？
> 产生这个疑问的原因：记公式时符号最容易出错，不知道为什么偏偏是减号。

**Q10: [用余弦定理求角时，为什么 C 不一定是锐角]**
我用余弦定理求 cosC = (a² + b² - c²)/(2ab)，如果算出来 cosC < 0，那 C 是钝角。但我一直想：钝角的余弦是负数，这我理解，但为什么 cosC < 0 就能断定 C > 90°？不会有两个可能的角吗（比如一个是锐角一个是钝角但余弦值相同）？
> 产生这个疑问的原因：知道 cos 在 0°-180° 范围内是单调递减的，但不确定这个结论是否总是成立。

## 关于「直线」的问题

**Q11: [斜率 k = tanθ 中的 θ 为什么一定要是锐角——tan 在钝角范围怎么理解]**
斜率 k = tanθ，当 θ 是钝角时，比如 θ = 120°，tan120° = -√3，那 k 就是负的。但我不理解：直线倾斜角不是应该在 0°-180° 之间吗？钝角的 tan 怎么理解？
> 产生这个疑问的原因：几何上"倾斜角"是直线与 x 轴正方向的夹角，应该是 0°-180°，但 tan 在这个范围内不是单调的。

**Q12: [垂直的两条直线为什么 k₁·k₂ = -1——这个公式怎么来的]**
我知道斜率 k₁ 和 k₂ 的两条直线垂直时，k₁·k₂ = -1，但我不知道这个公式怎么来的。另外，当直线垂直于 x 轴时，斜率无穷大，这个公式还能用吗？
> 产生这个疑问的原因：知道这个结论但不会推导，而且垂直线的情况好像会让公式失效。

**Q13: [直线方程的各种形式之间怎么转换——什么时候用哪种形式]**
直线方程有斜截式 y = kx + b、点斜式 y - y₀ = k(x - x₀)、两点式、一般式 Ax + By + C = 0 等等。拿到一道题，我不知道该用哪种形式，有没有选择标准？
> 产生这个疑问的原因：学了很多形式，但不知道什么情况下该选哪种，经常混用或选错。

## 关于「圆」的问题

**Q14: [圆的一般方程 x² + y² + Dx + Ey + F = 0 怎么判断它是不是真的表示圆]**
我把一般方程配方后发现需要 D² + E² - 4F > 0 才是圆，但我不理解：为什么这个条件能判断是否表示圆？如果这个条件不满足，那代表什么图形？
> 产生这个疑问的原因：知道要用这个条件判断，但不理解它的几何意义。

**Q15: [圆的切线为什么垂直于半径——这是规定还是推导出来的]**
教材说"圆上一点的切线垂直于该点与圆心连线的半径"，我不理解：这是圆的特殊性质，还是可以从更基本的定义推导出来？如果我画一条线垂直于半径，它一定就是切线吗？
> 产生这个疑问的原因：知道切线垂直于半径，但不知道这是圆特有的性质还是更一般的规律。

## 关于「椭圆」的问题

**Q16: [椭圆的离心率 e 为什么必须小于 1——e > 1 会怎样]**
教材说椭圆离心率 0 < e < 1，双曲线 e > 1，抛物线 e = 1。但我不理解：为什么椭圆离心率必须小于 1？如果 e = 1 或 e > 1，椭圆会变成什么形状？
> 产生这个疑问的原因：知道离心率的取值范围，但不知道这个限制的物理或几何含义。

**Q17: [椭圆标准方程 x²/a² + y²/b² = 1 中，a 和 b 哪个是长轴哪个是短轴]**
我老是记不住：方程 x²/a² + y²/b² = 1 中，到底是 a > b 还是 a < b？如果我把 a 和 b 换位置，方程代表同一个椭圆吗？
> 产生这个疑问的原因：教材说 a > b，但实际解题时经常搞混 a 和 b 的关系。

**Q18: [椭圆、双曲线、抛物线都是圆锥曲线——它们到底有什么区别和联系]**
我知道它们都是用平面切圆锥得到的，但我不理解：一个平面切圆锥，怎么有时候得到椭圆，有时候得到双曲线或抛物线？什么条件决定得到哪种曲线？
> 产生这个疑问的原因：知道三种曲线统称"圆锥曲线"，但不理解它们在几何上的本质区别。

## 关于「双曲线」的问题

**Q19: [双曲线为什么有两条渐近线——渐近线是什么意思]**
教材说双曲线有渐近线 y = ±(b/a)x，但我不理解："渐近"是什么意思？双曲线和渐近线永远不会相交吗？为什么要研究渐近线？
> 产生这个疑问的原因：知道渐近线的方程，但不理解"渐近"这个概念本身。

**Q20: [双曲线中 b² = c² - a² 和椭圆的 b² = a² - c² 为什么符号不同]**
我老是记混双曲线和椭圆的 b² 计算公式。双曲线是 b² = c² - a²，椭圆是 b² = a² - c²，这两个公式就差一个符号。为什么不能统一？
> 产生这个疑问的原因：两个公式太像了，考试时经常记错，不知道怎么区分。

## 关于「抛物线」的问题

**Q21: [抛物线的离心率为什么恒等于 1——1 从哪来的]**
椭圆 e < 1，双曲线 e > 1，抛物线 e = 1。我理解抛物线的定义是"到焦点和准线距离相等"，但这个定义怎么推出 e = 1？
> 产生这个疑问的原因：知道抛物线的定义，但不理解这个定义为什么等价于 e = 1。

**Q22: [抛物线标准方程 y² = 2px 中的 p 到底是焦距还是焦点到准线的距离]**
教材说 p 是"焦点到准线的距离"，但有些教材说 p 是"焦距"。我搞不清楚这两个说法是不是同一个东西。另外，焦点在 (p/2, 0)，这和 p 是什么关系？
> 产生这个疑问的原因：教材中 p 的定义有时说是焦距，有时说是焦点到准线的距离，不知道该信哪个。

## 关于「解三角形」的问题

**Q23: [解三角形和普通代数方程有什么区别——为什么要有专门的定理]**
解三角形问题看起来就是已知一些边和角，求其他边和角，和解方程差不多。但教材专门讲了正弦定理和余弦定理。我就想：为什么不直接用代数运算，要用这些定理？这些定理能解决的问题，普通方程解不了吗？
> 产生这个疑问的原因：知道正弦定理和余弦定理有用，但不理解为什么不能用普通代数方程代替。

**Q24: [三角形内角和为什么一定是 180°——不是 180° 会怎样]**
教材说三角形内角和是 180°，这是公理还是定理？如果不是 180°，比如内角和是 200°，会出现什么问题？
> 产生这个疑问的原因：从小就知道三角形内角和 180°，但没想过为什么一定是 180°，以及如果不是会怎样。

## 关于「空间向量」的问题

**Q25: [空间向量和平面向量到底有什么区别——z 坐标只是多了一个分量吗]**
空间向量就是比平面向量多了一个 z 坐标吗？比如向量 (1,2) 和 (1,2,0) 本质上是不是同一个东西？三维向量的运算比二维复杂很多吗？
> 产生这个疑问的原因：感觉空间向量只是平面向量的简单扩展，不理解为什么要单独学。

**Q26: [空间向量的数量积和平面向量的数量积是一回事吗——公式看起来不一样]**
平面向量数量积：a·b = a_x·b_x + a_y·b_y
空间向量数量积：a·b = a_x·b_x + a_y·b_y + a_z·b_z
两个公式形式不一样，是同一个运算吗？
> 产生这个疑问的原因：两个公式一个只有两项，一个有三项，不知道是不是同一个运算在不同维度的推广。

**Q27: [向量积（叉积）高中生要不要学——它和数量积有什么区别]**
教材说向量积（叉积）是大学内容，但我不理解：为什么有了数量积还要学向量积？向量积到底解决什么问题数量积解决不了？
> 产生这个疑问的原因：数量积已经能求夹角了，向量积还有什么独特的作用吗？
