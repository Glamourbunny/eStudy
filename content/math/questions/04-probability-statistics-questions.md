# 概率与统计链 — 学生疑问

## 关于「古典概型」的问题

**Q1: [古典概型为什么要求"等可能性"——如果不等可能会怎样]**
古典概型要求每个基本事件"等可能"发生。但我不理解：如果不等可能，比如抛一枚不均匀的硬币，正面概率是 0.6，反面是 0.4，那该怎么计算概率？"等可能性"是必须的吗？如果不满足怎么办？
> 产生这个疑问的原因：知道古典概型的两个条件（有限性和等可能性），但不理解为什么需要"等可能性"这个条件。

**Q2: [基本事件到底怎么数——什么时候用排列什么时候用组合]**
古典概型中数基本事件数，我经常搞不清楚什么时候用排列 C(n,k)，什么时候用组合 A(n,k)。比如抛两枚硬币，基本事件是"正正、正反、反正、反反"4 个，这是组合问题还是排列问题？
> 产生这个疑问的原因：知道排列和组合的区别，但实际数基本事件时不知道该用哪个。

**Q3: [P(A) + P(B) = 1 什么时候成立——我经常误用在所有情况]**
我经常看到有同学做概率题时直接写 P(A) + P(B) = 1，但我记得这好像只在 A 和 B 是对立事件时才成立。什么时候 P(A) + P(B) = 1？什么时候不成立？
> 产生这个疑问的原因：知道不是所有情况都有 P(A) + P(B) = 1，但记不住具体条件。

## 关于「条件概率」的问题

**Q4: [P(A|B) 和 P(B|A) 到底有什么区别——我老是搞混]**
条件概率 P(A|B) 是在 B 发生的条件下 A 的概率，P(B|A) 是在 A 发生的条件下 B 的概率。但我经常搞混这两个东西，不理解为什么它们不一样。
> 产生这个疑问的原因：知道两个符号的含义，但实际做题时不知道该用哪个。

**Q5: [条件概率为什么要 P(A|B) = P(AB)/P(B) ——这个公式怎么来的]**
条件概率的定义是 P(A|B) = P(AB)/P(B)，但我不理解：为什么偏偏是这个公式？P(AB)/P(B) 为什么能表示"在 B 发生的条件下 A 的概率"？
> 产生这个疑问的原因：知道公式但不理解它的来源。

**Q6: [缩减样本空间是什么意思——为什么已知 B 发生后样本空间就变小了]**
教材说条件概率是"缩减样本空间"。已知 B 发生，样本空间从 Ω 缩减为 B。但我不理解：为什么已知 B 发生后，样本空间就变小了？这个"缩减"是字面意思还是比喻？
> 产生这个疑问的原因：知道缩减样本空间这个概念，但不理解它的具体含义。

**Q7: [乘法公式 P(AB) = P(A|B)·P(B) 怎么记——什么时候该用]**
我知道乘法公式，但不知道什么时候该用它。什么情况下我需要把 P(AB) 拆成 P(A|B)·P(B) 而不是直接算 P(AB)？
> 产生这个疑问的原因：知道公式但不知道什么时候该用。

## 关于「分布列」的问题

**Q8: [分布列中为什么所有概率加起来必须等于 1——少一点或多一点不行吗]**
分布列要求 Σpᵢ = 1，所有概率之和必须等于 1。但我不理解：如果我算出来的概率之和是 0.98 或者 1.05，这意味着什么？是算错了还是有其他解释？
> 产生这个疑问的原因：知道概率之和要为 1，但不理解为什么必须严格等于 1。

**Q9: [随机变量 X 到底是什么——它和普通变量 x 有什么区别]**
教材说随机变量 X 是"取值随机的变量"，但我不理解：普通变量 x 也可以取不同的值啊，比如解方程时 x = 2 或 x = 3。X 和 x 到底有什么区别？
> 产生这个疑问的原因：知道随机变量 X 和普通变量 x 不同，但不理解本质区别在哪里。

**Q10: [已知分布列怎么求 P(1 ≤ X ≤ 3)——这个区间包含哪些值]**
我有分布列后，求 P(1 ≤ X ≤ 3) 时，到底是只加 X = 1, 2, 3 的概率，还是也包含其他值？边界上的值怎么处理？
> 产生这个疑问的原因：不确定"≤"和"<"在概率计算中有没有区别。

## 关于「期望」的问题

**Q11: [期望 E(X) 和平均值有什么区别——E(X) 不就是把所有值加起来除以个数吗]**
期望的公式 E(X) = Σxᵢpᵢ，看起来像是把所有取值加权和再除以总数。但我不理解：如果我做 100 次实验，这 100 次结果的平均值会不会恰好等于 E(X)？期望是理论值，平均值是实验值，它们什么关系？
> 产生这个疑问的原因：以为期望就是加权平均，和普通平均值差不多。

**Q12: [E(X + Y) = E(X) + E(Y) 什么时候成立——为什么不是永远成立]**
我以为期望一定有 E(X + Y) = E(X) + E(Y)，但教材说独立时才成立。X 和 Y 什么时候独立？"不独立"的情况是什么样？
> 产生这个疑问的原因：记不住独立条件，不知道什么情况下不能用这个公式。

**Q13: [如果 E(X) 不在 X 的取值范围内会怎样——这可能吗]**
我算了某个随机变量的期望 E(X) = 3.5，但 X 只能取整数值 1, 2, 3, 4, 5。3.5 不在取值范围内啊！这是怎么回事？E(X) 必须在取值范围内吗？
> 产生这个疑问的原因：直觉上觉得平均值应该是一个实际能取到的值，但期望可能不是。

## 关于「方差」的问题

**Q14: [方差为什么是平方——为什么要 (X - E(X))² 而不是直接 |X - E(X)|]**
方差的定义 Var(X) = E[(X - E(X))²]，我理解它衡量数据分散程度，但为什么不直接用 E[|X - E(X)|]？绝对值不是更直观吗？平方有什么好处？
> 产生这个疑问的原因：知道方差能衡量离散程度，但不理解为什么用平方而不是绝对值。

**Q15: [标准差 σ 和方差 Var(X) 是什么关系——为什么要有两个概念]**
标准差 σ = √Var(X)，这两个概念都能衡量离散程度。但我不理解：既然有了方差，为什么还要定义标准差？标准差有什么独特的用处？
> 产生这个疑问的原因：知道两个概念的关系，但不理解为什么要两个都用。

**Q16: [Var(X + Y) = Var(X) + Var(Y) 什么时候成立——为什么不是永远成立]**
和期望一样，方差也有"独立时才可加"的性质。但我不理解：如果 X 和 Y 不独立，Var(X + Y) 会发生什么变化？
> 产生这个疑问的原因：记不住独立条件，不知道什么情况下不能用这个公式。

## 关于「二项分布」的问题

**Q17: [二项分布和超几何分布到底怎么区分——放回和不放回有那么大区别吗]**
二项分布是 n 次独立重复试验（放回抽样），超几何分布是不放回抽样。但我不理解：总体很大时，不放回抽样和放回抽样差别应该很小吧？那什么时候该用二项分布，什么时候该用超几何分布？
> 产生这个疑问的原因：知道两个分布的区别，但不确定什么时候该用哪个。

**Q18: [二项分布公式 P(X=k) = C(n,k)pᵏ(1-p)ⁿ⁻ᵏ 怎么来的——C(n,k) 是什么意思]**
我知道二项分布公式，但 C(n,k) 是什么意思？为什么组合数会出现在这里？n 次试验中恰好 k 次成功，为什么要用 C(n,k)？
> 产生这个疑问的原因：知道公式会套用，但不理解组合数为什么恰好是 C(n,k)。

**Q19: [二项分布的期望 E(X) = np，为什么 np 就是期望——这个公式怎么来的]**
二项分布 B(n, p) 的期望是 np，但我不理解：为什么 n 次试验，每次成功概率 p，期望恰好是 np？这个公式能推导出来吗？
> 产生这个疑问的原因：知道公式但不会推导，不理解为什么恰好是 np。

## 关于「独立性检验」的问题

**Q20: [χ² 统计量为什么能判断两个变量是否相关——这个数字大或小说明什么]**
独立性检验用 χ² = Σ(Oᵢ - Eᵢ)²/Eᵢ 来计算。但我不理解：这个值到底代表什么？为什么 χ² 越大说明两个变量越可能相关？有没有一个具体的临界值来判断？
> 产生这个疑问的原因：知道怎么算 χ²，但不理解它为什么能衡量两个变量的相关性。

**Q21: [自由度 (r-1)(c-1) 怎么来的——为什么要用行数和列数来算]**
教材说列联表的自由度是 (行数 - 1) × (列数 - 1)。但我不理解：为什么是这种形式？行数和列数减去 1 后相乘，这个公式背后的道理是什么？
> 产生这个疑问的原因：记住公式了但不知道它怎么来的。

**Q22: [观察频数和期望频数有什么区别——我不会区分什么时候该用哪个]**
χ² 统计量中，Oᵢ 是观察频数（实际样本数据），Eᵢ 是期望频数（如果两变量独立应该有的频数）。但我不理解：我怎么知道哪个是观察的，哪个是期望的？两者怎么计算？
> 产生这个疑问的原因：知道两个概念的名字，但实际计算时不知道该用什么数据。
