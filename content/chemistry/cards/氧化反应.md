# 氧化反应（Oxidation）

> **反应本质**：失电子、加氧或去氢的过程。化合价升高，还原剂被氧化。

---

## 一、反应本质与判断标志

```
判断标志（满足其一即可）：
✓ 元素化合价升高
✓ 加氧原子（与O₂反应、加氧原子）
✓ 去氢原子（脱氢）
✓ 失电子
```

### 氧化数变化通式
```
还原剂 —ne⁻ → 氧化产物
（化合价升高，被氧化）
```

---

## 二、无机代表反应

| 反应 | 方程式 | 氧化特征 | 化合价变化 |
|------|--------|----------|-----------|
| **金属与O₂** | `2Mg + O₂ → 2MgO` | Mg被氧化 | Mg: 0→+2 |
| **非金属与O₂** | `S + O₂ → SO₂` | S被氧化 | S: 0→+4 |
| **置换反应** | `Zn + 2HCl → ZnCl₂ + H₂↑` | Zn被氧化 | Zn: 0→+2 |
| **Fe²⁺被氧化** | `2FeCl₂ + Cl₂ → 2FeCl₃` | Fe²⁺被氧化 | Fe: +2→+3 |

## 三、有机代表反应

### 醇的氧化序列
```
          氧化           氧化
CH₃CH₂OH ───→ CH₃CHO ───→ CH₃COOH
  乙醇        乙醛         乙酸
 (-OH在端碳)  (醛基)       (羧基)
 
条件：
醇→醛：Cu/Ag催化剂，加热（去氢氧化）
醛→酸：银氨溶液/新制Cu(OH)₂（加氧氧化）
```

### 燃烧反应（完全氧化）
```
CH₄ + 2O₂ → CO₂ + 2H₂O          （所有有机物燃烧都是氧化）
C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O
```

### 醛的特征氧化
```
银镜反应：
CH₃CHO + 2[Ag(NH₃)₂]OH → CH₃COONH₄ + 2Ag↓ + 3NH₃ + H₂O
（醛基被氧化为羧基，Ag⁺被还原为Ag）

与新制Cu(OH)₂反应：
CH₃CHO + 2Cu(OH)₂ + NaOH → CH₃COONa + Cu₂O↓(砖红) + 3H₂O
```

## 四、规律与原理

### 无机视角 vs 有机视角

| 角度 | 无机化学 | 有机化学 |
|------|----------|----------|
| **判断依据** | 化合价升高 | 加氧/去氢 |
| **氧化剂** | O₂、Cl₂、KMnO₄等 | O₂、CuO、银氨溶液等 |
| **还原剂** | 金属、低价态离子 | 醇、醛、烯烃等 |

### 常见氧化剂强度顺序
```
强氧化剂：KMnO₄(H⁺) > K₂Cr₂O₇(H⁺) > Cl₂ > Br₂ > Fe³⁺
中强氧化剂：O₂(点燃) > 浓H₂SO₄ > 浓HNO₃
弱氧化剂：Ag⁺(银氨溶液) > Cu²⁺(Cu(OH)₂) > I₂
```

### 醇氧化的结构要求
```
RCH₂OH（伯醇）—[O]→ RCHO（醛）—[O]→ RCOOH（酸）
       ↑
R₂CHOH（仲醇）—[O]→ R₂C=O（酮）
       ↑
R₃COH（叔醇）—[O]→ 难以氧化
```

## 五、纵向贯通——后续扩展

```
氧化反应知识链：

金属氧化 ──→ 金属氧化物 ──→ 碱/盐
     │
     ↓
醇的氧化 ──→ 醛 ──→ 酸 ──→ 酯
     │
     ↓
烯烃氧化 ──→ 环氧/二醇 ──→ 断裂氧化
     │
     ↓
醛的氧化（银镜反应）──→ 鉴别醛基特征反应
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 氧化反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆氧化反应与燃烧反应的关系                          │
│     燃烧是剧烈的氧化反应，但氧化≠燃烧                      │
│     缓慢氧化：铁生锈、食物腐败（不发光）                   │
├─────────────────────────────────────────────────────────┤
│ E2. 醛的氧化反应方程式配平错误                            │
│     银镜反应：1mol —CHO ~ 2mol Ag                        │
│     与Cu(OH)₂：1mol —CHO ~ 1mol Cu₂O                    │
│     甲醛特殊：1mol HCHO ~ 4mol Ag（两个醛基等效）        │
├─────────────────────────────────────────────────────────┤
│ E3. 忽视银氨溶液的配制要求                                │
│     必须：向AgNO₃中滴加稀氨水，至沉淀恰好溶解为止         │
│     错误：氨水过量或不足，或直接用AgNO₃溶液               │
├─────────────────────────────────────────────────────────┤
│ E4. 醇氧化产物判断错误                                    │
│     —CH₂OH → —CHO（醛）                                  │
│     —CHOH— → C=O（酮）                                   │
│     —COH<  （叔醇）→ 不氧化                              │
├─────────────────────────────────────────────────────────┤
│ E5. 氧化还原概念混淆                                      │
│     "被氧化" = 失电子 = 化合价升高 = 作还原剂            │
│     "被还原" = 得电子 = 化合价降低 = 作氧化剂            │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        氧化反应
       /    |    \
      ↓     ↓     ↓
   燃烧   催化氧化  缓慢氧化
    |       |        |
   完全    部分     长期
   
氧化反应 ↔ 还原反应（同时发生，电子转移守恒）
氧化反应 → 取代反应（部分氧化伴随取代）
```
