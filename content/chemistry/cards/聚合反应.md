# 聚合反应（Polymerization）

> **反应本质**：小分子（单体）通过化学键连接形成高分子化合物（聚合物）的过程。是多次加成或缩合的宏观表现。

---

## 一、反应本质与判断标志

```
判断标志：
✓ 产物分子量远大于反应物
✓ 反应物为小分子（单体），产物为高分子
✓ 链节重复出现
```

### 两大类聚合反应
```
聚合反应
├── 加聚反应（加成聚合）：不饱和键打开，相互连接
│     例：乙烯 → 聚乙烯
│
└── 缩聚反应（缩合聚合）：单体间脱去小分子（H₂O等）
      例：对苯二甲酸 + 乙二醇 → 聚酯 + H₂O
```

---

## 二、加聚反应代表

### 乙烯类加聚
```
n CH₂=CH₂ → [CH₂-CH₂]ₙ          （聚乙烯，PE）
n CH₂=CHCl → [CH₂-CHCl]ₙ        （聚氯乙烯，PVC）
n CH₂=CH-CN → [CH₂-CH(CN)]ₙ    （聚丙烯腈，腈纶）
```

### 二烯烃加聚
```
n CH₂=CH-CH=CH₂ → [CH₂-CH=CH-CH₂]ₙ   （顺丁橡胶）
```

### 共聚反应
```
n CH₂=CH₂ + n CH₂=CH-CH₃ → 乙烯-丙烯共聚物
（两种或多种单体共同聚合）
```

---

## 三、缩聚反应代表

### 聚酯类
```
n HOOC-C₆H₄-COOH + n HOCH₂CH₂OH
  → [OC-C₆H₄-COOCH₂CH₂O]ₙ + 2n H₂O
  （对苯二甲酸 + 乙二醇 → 聚酯/PET）
```

### 聚酰胺类
```
n H₂N-(CH₂)₆-NH₂ + n HOOC-(CH₂)₄-COOH
  → [NH-(CH₂)₆-NHCO-(CH₂)₄-CO]ₙ + 2n H₂O
  （己二胺 + 己二酸 → 尼龙-66）
```

### 酚醛树脂
```
n C₆H₅OH + n HCHO → 酚醛树脂 + n H₂O
（苯酚 + 甲醛，缩聚）
```

---

## 四、规律与原理

### 加聚 vs 缩聚 对比

| 对比项 | 加聚反应 | 缩聚反应 |
|--------|----------|----------|
| **单体特征** | 含不饱和键（C=C、C≡C、C=O） | 含双官能团（—OH、—COOH、—NH₂） |
| **产物特征** | 只有聚合物 | 聚合物 + 小分子（H₂O、HCl等） |
| **键的变化** | 不饱和键打开，相互连接 | 官能团间反应，脱去小分子 |
| **典型代表** | 聚乙烯、聚丙烯、聚氯乙烯 | 聚酯、聚酰胺、酚醛树脂 |

### 单体判断方法
```
从聚合物结构反推单体：

加聚产物：
[CH₂-CH₂]ₙ  →  单体：CH₂=CH₂
[CH₂-CH=CH-CH₂]ₙ → 单体：CH₂=CH-CH=CH₂

缩聚产物：
[OC-C₆H₄-COOCH₂CH₂O]ₙ
  → 单体：HOOC-C₆H₄-COOH + HOCH₂CH₂OH
```

## 五、纵向贯通——后续扩展

```
聚合反应知识链：

小分子单体 ──→ 加聚/缩聚 ──→ 高分子材料
     │                        │
     ↓                        ↓
乙烯 ──→ 聚乙烯 ──→ 塑料薄膜  │
丙烯 ──→ 聚丙烯 ──→ 塑料容器  │
氯乙烯 ──→ PVC ──→ 管道建材   │
苯乙烯 ──→ 聚苯乙烯 ──→ 泡沫塑料
二烯烃 ──→ 橡胶 ──→ 轮胎     │
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 聚合反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 加聚反应方程式写法错误                                │
│     正确：n CH₂=CH₂ → [CH₂-CH₂]ₙ                         │
│     错误：不写n或链节写错                                 │
│     链节：聚合物中重复的最小结构单元                      │
├─────────────────────────────────────────────────────────┤
│ E2. 缩聚反应漏写小分子                                    │
│     缩聚一定有小分子生成（H₂O、HCl、NH₃等）              │
│     如果只写聚合物，方程式不完整                          │
├─────────────────────────────────────────────────────────┤
│ E3. 单体判断错误                                          │
│     加聚：找链节中两个碳，恢复双键                        │
│     缩聚：找酯基/酰胺键，断开恢复官能团                   │
├─────────────────────────────────────────────────────────┤
│ E4. 混淆聚合度与链节                                      │
│     链节：重复的结构单元                                  │
│     聚合度(n)：链节重复的次数                             │
│     高分子分子量 = 链节式量 × 聚合度                      │
├─────────────────────────────────────────────────────────┤
│ E5. 认为所有聚合都需要催化剂和加热                        │
│     大部分需要，但自由基聚合可在光/热引发下进行           │
│     加聚常用催化剂：Ziegler-Natta催化剂                  │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        聚合反应
       /        \
      ↓          ↓
   加聚反应    缩聚反应
      |          |
   多次加成    多次缩合
      |          |
   不饱和键    双官能团
   
加聚反应 ← 加成反应（多次加成聚合）
缩聚反应 ← 取代反应（脱水/脱HCl等）
```

---

## 核心记忆口诀

> **小分子变大分子，加聚缩聚两路分**
> **加聚不饱和键开，缩聚官能团脱水成**
> **链节重复n次连，单体判断逆推明**
