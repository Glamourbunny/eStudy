# 氢（H）元素卡片

> **元素档案**：第一周期，IA族，原子序数1，最轻的元素，双原子分子H₂，无色无味气体

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第1周期
- **族**：IA族（碱金属，但性质特殊）
- **原子序数**：1
- **核外电子排布**：(+1) 1

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 下 | He（氦） | 惰性气体，非金属性极弱 |
| 右 | H的相邻元素较少（唯一） | — |

### 推断题身份证 ⭐
- ⭐ "最轻的气体，密度最小" → H₂
- ⭐ "在空气中燃烧火焰呈淡蓝色" → H₂
- ⭐ "与O₂混合点火爆炸" → H₂
- ⭐ "可燃烧生成水，产物无污染" → H₂（清洁能源）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+1
- 电子层数：1层
- 最外层电子数：1e⁻（独生子）

### 【第2层】电子行为
- **易失**：失去1e⁻ → H⁺（质子，无电子）
- **易得**：获得1e⁻ → H⁻（氢负离子，类卤素）
- **共用**：形成共价键（H₂、H₂O等）

### 【第3层】化学键型
- **H₂分子**：非极性共价键（H-H键）
- **与金属**：形成离子氢化物 → NaH（Na⁺H⁻）
- **与非金属**：形成极性共价键 → H₂O、HCl

### 【第4层】晶体类型
- **H₂单质**：分子晶体（熔点-259℃，沸点-253℃）
- 解释：分子量小，分子间作用力极弱→熔沸点极低

> **结构解释性质**：H原子半径最小→电子最难失去→非金属性很强（但无中子）

---

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色状态 | 无色无味气体 | 双原子分子 |
| 密度 | 0.089g/L（最轻） | 充气球、放空 |
| 溶解性 | 难溶于水 | 可用排水法收集 |

### 化学性质

#### 还原性（H₂被氧化）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与O₂ | 2H₂ + O₂ =点燃= 2H₂O | 淡蓝色火焰，爆炸极限4%-75% |
| 与CuO | H₂ + CuO =△= Cu + H₂O | 黑色变红色，有水珠 |
| 与Fe₂O₃ | 3H₂ + Fe₂O₃ =△= 2Fe + 3H₂O | 冶炼金属 |

🔴 **反例/边界**：H₂还原CuO需加热，实验中需先通H₂排空气，再加热，结束时先停止加热再停H₂

#### 可燃性（与O₂反应）🔴
| 类型 | 方程式 | 说明 |
|------|--------|------|
| 完全燃烧 | 2H₂ + O₂ =点燃= 2H₂O | 无污染，清洁能源 |
| 不完全燃烧 | 2H₂ + O₂ =点燃/不完全= 2H₂O（部分燃烧时可能生成H₂O） | 实际无不完全燃烧产物 |

🔴 **反例/边界**：H₂燃烧不是爆炸，爆炸是H₂与O₂混合达到爆炸极限

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：H₂ ↔ H₂O】

    H₂（无色气体）
     │
     ├──[点燃O₂]──→ H₂O（清洁燃烧）
     │    2H₂ + O₂ =点燃= 2H₂O
     │    现象：淡蓝色火焰，烧杯壁有水珠
     │
     ├──[还原金属氧化物]──→ 金属 + H₂O
     │    H₂ + CuO =△= Cu + H₂O
     │    现象：黑色CuO变红，生成水珠
     │
     ├──[与N₂]──→ NH₃（工业合成氨）
     │    N₂ + 3H₂ ⇌高温高压催化剂= 2NH₃
     │    🔵 哈伯法，重要工业原料
     │
     └──[电解]──→ H₂↑ + O₂↑
                  2H₂O =电解= 2H₂↑ + O₂↑

                              【H⁺的转化】

    H⁺（酸中）
     │
     ├──[与活泼金属]──→ 金属 + H₂↑
     │    Zn + 2HCl = ZnCl₂ + H₂↑
     │    现象：气泡产生
     │
     ├──[与碱]──→ 盐 + H₂O
     │    HCl + NaOH = NaCl + H₂O
     │
     └──[与金属氧化物]──→ 盐 + H₂O
         Fe₂O₃ + 6HCl = 2FeCl₃ + 3H₂O

                              【H⁻的转化（氢负离子）】

    H⁻（氢负离子）
     │
     └──[与H₂O]──→ H₂↑ + OH⁻（剧烈反应）
         NaH + H₂O = NaOH + H₂↑
         氢负离子是强还原剂
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | H₂还原金属氧化物后有水生成 | 正确，但实验操作顺序重要：先通H₂排空气，再加热，结束时先停止加热再停H₂ |
| **E2** | H₂燃烧爆炸很危险 | 爆炸需要H₂与O₂混合比例达到4%-75%，纯净H₂安静燃烧 |
| **E3** | H原子与H₂分子性质相同 | 不同！H原子极活泼（还原剂），H₂分子稳定（可作燃料） |
| **E4** | H₂能还原所有金属氧化物 | 不能还原Au、Ag等不活泼金属的氧化物（这些氧化物本身不稳定） |
| **E5** | 所有酸中都含H⁺ | 是，但有些酸（如有机酸）提供H⁺能力弱，是弱酸 |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **O** | 反应伴侣 | H₂ + O₂ =点燃= H₂O（清洁燃烧） |
| **N** | 工业联合 | N₂ + 3H₂ ⇌ 2NH₃（合成氨） |
| **C** | 能源联合 | 2H₂ + O₂ = 2H₂O（氢能源）；CH₄ + 2O₂ = CO₂ + 2H₂O（天然气燃烧） |
| **He** | 同周期 | H和He在第一周期，He是惰性气体，H很活泼 |
| **Na** | 与金属氢化物 | NaH + H₂O = NaOH + H₂↑（H⁻是强还原剂） |
