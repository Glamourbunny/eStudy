# 镁（Mg）元素卡片

> **元素档案**：第三周期，IIA族，原子序数12，银白色金属，密度1.74g/cm³，轻金属，燃烧发出耀眼白光

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：IIA族（碱土金属）
- **原子序数**：12
- **核外电子排布**：(+12) 2)8)2

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Be（铍） | 金属性Mg > Be（原子半径增大） |
| 下 | Ca（钙） | 金属性Mg < Ca（原子半径增大） |
| 左 | Na（钠） | 金属性Na > Mg（同一周期左>右） |
| 右 | Al（铝） | 金属性Mg > Al（但Al有两性） |

### 推断题身份证 ⭐
- ⭐ "燃烧发出耀眼白光" → Mg
- ⭐ "在CO₂中能燃烧" → Mg（2Mg + CO₂ =点燃= 2MgO + C）
- ⭐ "与热水反应生成无色气体" → Mg + 2H₂O =△= Mg(OH)₂ + H₂↑
- ⭐ "焰色反应呈白色" → Mg（实际不明显）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+12
- 电子层数：3层
- 最外层电子数：2e⁻（易失）

### 【第2层】电子行为
- **易失**：失去最外层2e⁻ → Mg²⁺（10电子稳定结构，类似Ne）
- 失电子能力较强，但比Na弱

### 【第3层】化学键型
- **与氧**：形成MgO（离子化合物）
- **与卤素**：形成MgCl₂、MgBr₂（离子化合物）
- **与酸**：生成Mg²⁺盐 + H₂↑

### 【第4层】晶体类型
- **Mg单质**：金属晶体（密排六方结构）
- **MgO**：离子晶体（熔点2852℃，高）
- 解释：Mg²⁺半径小、电荷大→离子键极强→氧化物熔点极高

> **结构解释性质**：Mg原子比Na小但电荷大→MgO晶格能大→MgO比Na₂O更稳定（耐火材料）

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色，有金属光泽 | 金属晶体 |
| 密度 | 1.74g/cm³（轻金属） | 比Al（2.7）更轻 |
| 熔点 | 648.8℃ | 中等 |
| 延展性 | 好（能压成薄片、拉成丝） | 金属键 |

### 化学性质

#### 还原性（被氧化）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与O₂ | 2Mg + O₂ =点燃= 2MgO | 耀眼白光，生成白色MgO |
| 与N₂ | 3Mg + N₂ =点燃= Mg₃N₂ | 氮化镁（灰绿色固体） |
| 与CO₂ | 2Mg + CO₂ =点燃= 2MgO + C | 镁条在CO₂中燃烧，继续发出白光 |
| 与H₂O（热） | Mg + 2H₂O =△= Mg(OH)₂ + H₂↑ | 热水，生成白色沉淀 |

🔴 **反例/边界**：Mg在冷水中反应缓慢（不如Na），需加热才明显

#### 与酸反应 🔴
| 反应 | 方程式 | 现象 |
|------|--------|------|
| 与稀盐酸 | Mg + 2HCl = MgCl₂ + H₂↑ | 气泡，溶液无色 |
| 与稀硫酸 | Mg + H₂SO₄ = MgSO₄ + H₂↑ | 气泡 |

🔴 **反例/边界**：Mg与浓H₂SO₄反应复杂，可能生成SO₂（氧化性酸）

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：MgO / Mg(OH)₂】

    Mg（银白色金属）
     │
     ├──[燃烧O₂]──→ MgO（白色固体）
     │    2Mg + O₂ =点燃= 2MgO
     │    现象：耀眼白光
     │
     ├──[燃烧CO₂]──→ MgO + C
     │    2Mg + CO₂ =点燃= 2MgO + C
     │    🔵 镁燃烧不能用CO₂灭火器
     │
     ├──[与N₂]──→ Mg₃N₂（氮化镁）
     │    3Mg + N₂ =点燃= Mg₃N₂
     │
     ├──[与热水]──→ Mg(OH)₂ + H₂↑
     │    Mg + 2H₂O =△= Mg(OH)₂ + H₂↑
     │    白色沉淀
     │
     └──[与稀酸]──→ MgCl₂/MgSO₄ + H₂↑
         Mg + 2HCl = MgCl₂ + H₂↑

                              【MgO的转化】

    MgO + H₂O ⇌ Mg(OH)₂（缓慢）
     │
     └── MgO + 2HCl = MgCl₂ + H₂O（酸碱中和）

                              【Mg(OH)₂的转化】

    Mg(OH)₂ =△= MgO + H₂O（煅烧分解）
     │
    Mg(OH)₂ + 2HCl = MgCl₂ + 2H₂O

                              【重要化合物】

    🔵 菱镁矿：MgCO₃ =△= MgO + CO₂↑
       （高温分解，用于制备MgO）

    🔵 光卤石：KCl·MgCl₂·6H₂O
       （可电解制Mg）

    🔵 Mg在合金中应用：
       镁铝合金（轻质合金）
       航空材料：Mg合金用于减轻重量
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Mg在CO₂中不能燃烧 | Mg在CO₂中**能燃烧**！2Mg + CO₂ =点燃= 2MgO + C，因此不能用CO₂灭火器灭镁火 |
| **E2** | Mg与水反应像Na一样剧烈 | Mg与冷水反应**很慢**，需加热才明显；Mg(OH)₂微溶，覆盖表面阻止反应 |
| **E3** | Mg燃烧只生成MgO | 还可能生成Mg₃N₂（空气中N₂参与），纯净O₂中只生成MgO |
| **E4** | MgO是酸性氧化物 | MgO是**碱性氧化物**，可与酸反应：MgO + 2HCl = MgCl₂ + H₂O |
| **E5** | 焰色反应Mg呈绿色 | Mg的焰色反应**不明显**，通常观察不到明显颜色 |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Na** | 同周期左邻 | 金属性Na > Mg（Na更活泼，与水反应更剧烈） |
| **Al** | 同周期右邻 | 金属性Mg > Al，但Al有两性；Mg不显两性 |
| **Ca** | 同族下 | 金属性Mg < Ca（Ca更活泼） |
| **Be** | 同族上 | 金属性Mg > Be（对角线规则） |
| **Fe** | 联合出题 | Mg可在炼钢中作脱硫剂：Mg + FeS = MgS + Fe |
| **O** | 反应伴侣 | 2Mg + O₂ =点燃= 2MgO（耀眼白光） |
