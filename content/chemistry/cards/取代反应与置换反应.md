# 取代反应 vs 置换反应

> **底线地位**：两者名字相近但本质完全不同。混淆会导致反应类型判断和方程式书写错误。

---

## 一、概念定义对比

| 维度 | 取代反应 | 置换反应 |
|------|----------|----------|
| **反应本质** | 原子或原子团被替代 | 单质与化合物反应生成新单质和新化合物 |
| **反应物** | 化合物 + 化合物/单质 | 单质 + 化合物 |
| **产物** | 两种化合物（或化合物+单质） | 新单质 + 新化合物 |
| **是否有单质** | 不一定有 | 一定有单质参与和生成 |
| **反应类型归属** | 有机/无机反应类型 | 氧化还原反应的细分（四大基本反应之一） |
| **电子转移** | 不一定有 | 一定有（属于氧化还原） |

---

## 二、核心区别

```
取代反应：
  R—X + Y → R—Y + X
  特点：不一定有单质，旧基团被新基团替代
  归属：可以是有机反应，也可以是无机反应

置换反应：
  A + BC → AC + B
  特点：单质+化合物→新单质+新化合物，一定有电子转移
  归属：四大基本反应类型之一，一定是氧化还原反应

记忆：
取代 → "取而代之"（基团替换）
置换 → "置换位置"（单质换单质）
```

---

## 三、判断方法

```
第一步：看是否有单质参与和生成
  有单质参与且有单质生成 → 可能是置换反应
  没有单质或没有单质生成 → 取代反应

第二步：看是否是氧化还原
  置换反应一定是氧化还原
  取代反应不一定（如酯化不是氧化还原）

第三步：看反应场景
  有机化学中的"某基团被替代" → 取代反应
  金属活动性顺序中的反应 → 置换反应
```

## 四、典型例子

### 取代反应（有机为主）
```
烷烃卤代：
CH₄ + Cl₂ —光→ CH₃Cl + HCl
（Cl取代H，产物都是化合物）

苯的硝化：
C₆H₆ + HNO₃ —浓H₂SO₄→ C₆H₅NO₂ + H₂O
（—NO₂取代—H）

酯化反应：
CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O
（—OH被—OC₂H₅取代）

酯的水解：
CH₃COOC₂H₅ + H₂O ⇌ CH₃COOH + C₂H₅OH
（逆过程，也是取代）

无机中的取代（离子交换）：
AgNO₃ + NaCl → AgCl↓ + NaNO₃
（Ag⁺取代Na⁺，Cl⁻取代NO₃⁻）
```

### 置换反应（无机）
```
金属+酸：
Zn + H₂SO₄ → ZnSO₄ + H₂↑
（单质Zn置换出单质H₂）

金属+盐：
Fe + CuSO₄ → FeSO₄ + Cu
（单质Fe置换出单质Cu）

非金属置换：
Cl₂ + 2NaBr → 2NaCl + Br₂
（单质Cl₂置换出单质Br₂）

H₂还原金属氧化物：
H₂ + CuO —△→ Cu + H₂O
（单质H₂置换出单质Cu）
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 取代vs置换常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 将CH₄+Cl₂当作置换反应                                 │
│     错误：CH₄+Cl₂→CH₃Cl+HCl是置换反应                   │
│     正确：是取代反应（产物中无单质，HCl是化合物）        │
│     虽然Cl₂是单质，但产物中没有单质生成                  │
├─────────────────────────────────────────────────────────┤
│ E2. 将有单质参与就认为是置换反应                          │
│     反例：Cl₂ + CH₄ → CH₃Cl + HCl                       │
│     有单质(Cl₂)参与，但不是置换反应                      │
│     判断关键：是否有单质生成 + 是否符合单质换单质模式    │
├─────────────────────────────────────────────────────────┤
│ E3. 混淆取代反应与复分解反应                              │
│     取代反应：AB + C → AC + B（一对一替换）              │
│     复分解反应：AB + CD → AD + CB（离子交换）            │
│     很多无机取代本质上是复分解                           │
├─────────────────────────────────────────────────────────┤
│ E4. 认为取代反应不是氧化还原                              │
│     不一定！                                              │
│     甲烷氯代：不是氧化还原（无化合价变化）               │
│     但某些取代伴随氧化还原                               │
│     置换反应：一定是氧化还原                             │
├─────────────────────────────────────────────────────────┤
│ E5. 有机反应类型判断时忽视"取代"的本质                    │
│     取代的本质：饱和键上的原子/基团被替代                │
│     与加成（不饱和键打开）是竞争关系                     │
│     烯烃与Cl₂：加成（双键打开）                          │
│     烷烃与Cl₂：取代（单键上H被Cl替代）                   │
└─────────────────────────────────────────────────────────┘
```

## 六、关联概念网络

```
        化学反应类型
       /            \
      ↓              ↓
  取代反应        置换反应
  (基团替换)     (单质换单质)
      |              |
  不一定氧化      一定氧化
  还原            还原
      \          /
       ↓        ↓
      可能有交集
      （某些反应既是取代又是氧化还原）
```

---

## 核心记忆口诀

> **取代反应：基团被替代，产物两化合物**
> **置换反应：单质换单质，氧化还原定**
> **看到单质生成，先想是不是置换**
> **有机里面取代多，无机置换常见到**
