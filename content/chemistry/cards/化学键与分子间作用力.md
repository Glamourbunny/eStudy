# 化学键 vs 分子间作用力

> **底线地位**：若分不清两者，化学性质与物理性质的所有解释都会错乱。这是高中化学最基础也最容易被忽视的辨析。

---

## 一、概念定义对比

| 维度 | 化学键 | 分子间作用力 |
|------|--------|-------------|
| **本质** | 原子/离子之间的强烈相互作用 | 分子之间的弱相互作用 |
| **强度** | 强（100-600 kJ/mol） | 弱（2-50 kJ/mol） |
| **影响性质** | **化学性质**（稳定性、反应活性） | **物理性质**（熔沸点、溶解性） |
| **类型** | 离子键、共价键、金属键 | 范德华力（色散/诱导/取向）、氢键 |
| **作用范围** | 原子/离子级别 | 分子级别 |

---

## 二、核心区别

```
化学键 ──→ 决定物质"是什么"（化学身份）
分子间力 ──→ 决定物质"怎么样"（物理状态）

例：H₂O
化学键：O-H共价键（决定H₂O能分解成H₂和O₂）
分子间力：氢键（决定H₂O沸点100℃，比H₂S高）
```

### 对性质的影响对比

| 性质 | 由什么决定 | 举例 |
|------|-----------|------|
| 热稳定性 | 化学键强弱 | HI < HBr < HCl < HF（键能递增） |
| 熔沸点 | 分子间力强弱 | H₂O（氢键）> H₂S（范德华力） |
| 溶解性 | 两者共同作用 | "相似相溶"：分子间力匹配 |
| 硬度 | 化学键类型 | 金刚石（共价键）> NaCl（离子键）> 干冰（分子间力） |

---

## 三、判断方法

```
看到问题问"熔沸点"→ 想分子间作用力
看到问题问"稳定性/能否分解"→ 想化学键

例外提醒：
• 原子晶体（金刚石、SiO₂）的熔沸点由共价键决定
  （因为整个晶体就是一个大分子，没有独立分子）
• 离子晶体的熔沸点由离子键强弱决定
```

---

## 四、典型例子

### 例1：为什么H₂O沸点比H₂S高？
```
不是因为O-H键比S-H键强
而是因为H₂O分子间有氢键，H₂S只有范德华力
→ 破坏分子间力所需能量：H₂O > H₂S
→ 沸点：H₂O(100℃) > H₂S(-60℃)
```

### 例2：为什么HF热稳定性比HCl差？
```
H-F键能(568 kJ/mol) > H-Cl键能(431 kJ/mol)
但HF更稳定？不！
实际：热稳定性 HF > HCl > HBr > HI
因为键能越大，化学键越强，越难断裂
→ 这是化学键决定的化学性质
```

### 例3：碘升华 vs 碘分解
```
I₂升华（固体→气体）：只克服分子间力，I-I键未断
I₂分解（I₂→I原子）：I-I共价键断裂，化学变化
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 概念混淆常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 将原子晶体的熔沸点归因于分子间力                      │
│     错误：金刚石熔点高是因为分子间力强                    │
│     正确：金刚石是原子晶体，熔点高是因为C-C共价键极强      │
│     （原子晶体中不存在独立分子，没有分子间力）            │
├─────────────────────────────────────────────────────────┤
│ E2. 认为化学键断裂一定是化学变化                          │
│     正确：NaCl溶于水，离子键断裂，但是物理变化            │
│     （没有新物质生成，只是电离）                          │
│     区分：是否有新物质生成                                │
├─────────────────────────────────────────────────────────┤
│ E3. 用键能解释熔沸点                                      │
│     错误：HF沸点高是因为H-F键能大                         │
│     正确：HF沸点高是因为分子间氢键                        │
│     （沸点是物理变化，与化学键无关）                      │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视氢键的特殊性                                      │
│     氢键不是化学键！是比范德华力强、但比化学键弱的         │
│     分子间作用力                                          │
│     有氢键的物质：H₂O、HF、NH₃、醇、羧酸                  │
└─────────────────────────────────────────────────────────┘
```
