## 二、四层递进结构

### 第一层：原子结构
```
原子序数 53 → 核电荷数 +53
质子数 = 53，中子数 = 74（常见同位素 127I）
最外层电子数 = 7 → **极易得1电子**，氧化性弱于Cl、Br
```

### 第二层：电子层排布
```
K层：2电子（饱和）
L层：8电子（饱和）
M层：18电子（饱和）
N层：18电子（饱和）
O层：7电子（**未饱和，得1电子达饱和**）
```

### 第三层：化学键类型
```
📌 主要成键方式：
• 离子键：NaI、KI（与金属形成离子化合物）
• 共价键：I-Cl、I-Br（卤素互化物）
• 配位键：I⁻作为配体与金属离子结合
📌 价态规律：
  -1价：I⁻（最稳定，**还原性最强**）
   0价：I₂（单质，紫色蒸气）
  +1价：IO⁻（次碘酸根，不稳定）
  +5价：IO₃⁻（碘酸根，稳定）
  +7价：IO₄⁻（高碘酸根）
```

### 第四层：晶体类型
```
📍 单质碘：分子晶体（I₂分子）
📍 碘化物：离子晶体（如KI、NaI晶体）
📍 分子间作用力：范德华力（**较弱，升华原因**）
📍 特性：**微溶于水，易溶于有机溶剂（CS₂、CCl₄）**
```

## 三、三合一转化网

### 3.1 碘的化合物转化关系

```
            还原剂（如S²⁻、I⁻）
      I₂ ←───────────────────── I⁻
       ↓                      ↑
       → I⁻（歧化）←          │
       ↗                      │
      IO⁻                   氧化剂（如Cl₂）
       ↓
      IO₃⁻
```

### 3.2 重要反应方程式

| 反应类型 | 条件 | 化学方程式 | 现象 |
|---------|------|-----------|------|
| **碘与氢气** | 持续加热 | `I₂ + H₂ ⇌ 2HI` | 反应最难，可逆程度最大 |
| **碘与金属** | 加热 | `2Na + I₂ → 2NaI` | 白色晶体生成 |
| **碘与淀粉** 🟢 | — | `I₂ + 淀粉 → 蓝色络合物` | **蓝色（特征检验）** |
| **碘的歧化** | 碱液 | `3I₂ + 6NaOH → 5NaI + NaIO₃ + 3H₂O` | 溶液褪色 |
| **碘与硫代硫酸钠** | — | `I₂ + 2Na₂S₂O₃ → Na₂S₄O₆ + 2NaI` | 蓝色消失（滴定反应） |
| **氯气氧化碘离子** | — | `Cl₂ + 2KI → 2KCl + I₂` | 溶液变蓝（加淀粉） |
| **碘的升华** | 加热 | `I₂(s) → I₂(g)` | 紫色蒸气，冷却重新凝华 |

### 3.3 碘的提取（海带提碘）🔵

```
步骤1：灼烧海带
       海带灰化，得到含碘化物灰烬

步骤2：水浸
       热水浸取，I⁻进入溶液

步骤3：氧化
       通入Cl₂或加入H₂O₂氧化I⁻
       2I⁻ + Cl₂ → I₂ + 2Cl⁻
       或：2I⁻ + H₂O₂ + 2H⁺ → I₂ + 2H₂O

步骤4：萃取
       用CCl₄或苯萃取I₂
       振荡后静置，**有机层呈紫红色**
```

## 四、🔴 红框反例边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 高频失分陷阱 E1-E5                                    │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆碘的升华和挥发                                    │
│     升华：固态→气态（I₂熔点113.7℃，**不经过液态**）        │
│     挥发：液态→气态（如Br₂）                              │
│     ⚠️ 碘加热时直接变成紫色蒸气，这是**升华**不是挥发       │
├─────────────────────────────────────────────────────────┤
│ E2. 误认为淀粉检验所有卤素                                │
│     **只有I₂与淀粉显蓝色，Cl₂、Br₂都不能**                │
│     Cl₂氧化淀粉会使其**褪色/降解**                        │
│     检验I₂的正确方法：淀粉溶液 → 蓝色                     │
├─────────────────────────────────────────────────────────┤
│ E3. 混淆氧化性顺序                                        │
│     氧化性：F₂ > Cl₂ > Br₂ > **I₂**                      │
│     还原性：F⁻ < Cl⁻ < Br⁻ < **I⁻**（最强）              │
│     ⚠️ I₂氧化性弱，但I⁻还原性强                          │
│     例：2I⁻ + Cl₂ → I₂ + 2Cl⁻（反应可进行）              │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视碘与维生素C的反应                                 │
│     维生素C（还原剂）可还原I₂                             │
│     I₂ + C₆H₁₂O₆ → 2I⁻ + C₆H₁₂O₆（氧化产物）            │
│     ⚠️ 解释"碘酒可以美白"：I₂被维生素C还原为I⁻            │
├─────────────────────────────────────────────────────────┤
│ E5. 混淆HI的还原性和HIO₃的氧化性                          │
│     HI（含I⁻）：**强还原性**，可被氧化剂氧化              │
│     HIO₃（含I⁺⁵）：**强氧化性**，可被还原剂还原           │
│     ⚠️ 同一元素不同价态，性质完全相反                     │
└─────────────────────────────────────────────────────────┘
```

## 五、跨元素关联

### 5.1 卤素单质物理性质递变（VIIA族）

| 性质 | F₂ | Cl₂ | Br₂ | I₂ |
|------|-----|------|------|-----|
| 状态 | 气体 | 气体 | 液体 | **固体** |
| 颜色 | 淡黄 | 黄绿 | 红棕 | **紫黑** |
| 熔沸点 | 最低 | 较低 | 较高 | **最高** |
| 升华 | — | — | — | **是** |
| 溶解性 | 反应 | 微溶 | 微溶 | **难溶** |

### 5.2 碘的含氧酸递变规律

```
稳定性：HIO₄ > HIO₃ > HIO₂ > HIO（次碘酸最不稳定）

氧化性：HIO > HIO₃（与稳定性相反）
        次碘酸不稳定，氧化性强

碘的价态转化：
I⁻ → I₂ → IO₃⁻（氧化）
I₂ → I⁻ → IO₃⁻（还原）
```

### 5.3 卤素互化物与拟卤素

```
卤素互化物：
• ICl（氯化碘）：红棕色液体
• IBr（溴化碘）：暗蓝色固体
• BrCl（氯化溴）：黄绿色气体

拟卤素（行为类似卤素）：
• (CN)₂：氰气
• SCN⁻：硫氰根
```

```

       /
  Cl ←  Br ←  I ← Te（VIA）
   \    |    /
    卤化物/互化物
```
