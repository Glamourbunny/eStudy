# 加成反应（Addition）

> **反应本质**：不饱和键（双键/三键）打开，原子或原子团直接加合到不饱和碳原子上。不饱和度降低。

---

## 一、反应本质与判断标志

```
判断标志：
✓ 反应物含 C=C、C≡C、C=O 等不饱和键
✓ 产物只有一种（加成产物）
✓ 不饱和度降低（双键变单键，三键变双键或单键）
```

### 核心通式
```
R₁R₂C=CR₃R₄ + A-B → R₁R₂C(A)-C(B)R₃R₄
（不饱和键打开，A和B分别加在两个碳上）
```

---

## 二、无机代表反应

| 反应 | 方程式 | 条件 | 说明 |
|------|--------|------|------|
| **乙烯+Br₂** | `CH₂=CH₂ + Br₂ → CH₂Br-CH₂Br` | 常温 | 使溴水/溴的CCl₄溶液褪色 |
| **乙烯+HCl** | `CH₂=CH₂ + HCl → CH₃CH₂Cl` | 催化剂 | 不对称加成遵循马氏规则 |
| **乙烯+H₂O** | `CH₂=CH₂ + H₂O → CH₃CH₂OH` | 浓H₂SO₄/△ | 工业制乙醇 |
| **乙炔+HCl** | `CH≡CH + HCl → CH₂=CHCl` | 催化剂 | 制氯乙烯（塑料原料） |
| **乙烯+H₂** | `CH₂=CH₂ + H₂ → CH₃CH₃` | Ni/△ | 催化加氢（还原反应） |

---

## 三、有机代表反应

### 烯烃加成类型总览
```
CH₂=CH₂ + Br₂ → CH₂Br-CH₂Br        （加卤素，鉴别不饱和键）
CH₂=CH₂ + HBr → CH₃CH₂Br           （加卤化氢）
CH₂=CH₂ + H₂O → CH₃CH₂OH           （加水，制醇）
CH₂=CH₂ + H₂ → CH₃CH₃              （加氢，还原）
```

### 炔烃加成（分步进行）
```
CH≡CH + H₂ —Ni/△→ CH₂=CH₂ —H₂/Ni→ CH₃CH₃
        （部分加氢）        （完全加氢）

CH≡CH + HCl —催化剂→ CH₂=CHCl    （制氯乙烯）
CH≡CH + 2Cl₂ → CHCl₂-CHCl₂      （完全加成）
```

### 醛酮的加成
```
CH₃CHO + H₂ —Ni/△→ CH₃CH₂OH      （醛还原为醇）
CH₃COCH₃ + HCN → 氰醇            （增长碳链方法）
```

## 四、规律与原理

### 马氏规则（Markovnikov规则）
```
不对称烯烃与不对称试剂加成时：
氢原子加在含氢较多的碳上

例：CH₃CH=CH₂ + HBr → CH₃CHBrCH₃（主要产物）
    不是 CH₃CH₂CH₂Br（次要产物）

原因：中间体碳正离子稳定性：
      3° > 2° > 1°
```

### 加成反应活性顺序
```
烯烃加成活性：
CH₂=CH₂ > RCH=CH₂ > R₂C=CH₂ > R₂C=CR₂
（位阻越小，活性越高）

试剂活性：
F₂ > Cl₂ > Br₂ > I₂（卤素加成）
HCl > HBr > HI（卤化氢加成）
```

## 五、纵向贯通——后续扩展

```
加成反应知识链：

烯烃加成 ──→ 卤代烃 ──→ 醇/醚/醛/酸
     │
     ↓
炔烃加成 ──→ 烯烃/烷烃 ──→ 聚合反应基础
     │
     ↓
醛酮加成 ──→ 增长碳链 ──→ 有机合成
     │
     ↓
催化加氢 ──→ 油脂硬化 ──→ 人造奶油
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 加成反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆"使溴水褪色"的原因                                │
│     加成褪色：烯烃、炔烃（化学变化）                       │
│     萃取褪色：苯、CCl₄（物理变化，分层）                   │
│     区分方法：加成后不分层，萃取后分层                      │
├─────────────────────────────────────────────────────────┤
│ E2. 马氏规则使用条件不清                                    │
│     只适用于：不对称烯烃 + 不对称试剂（HX、H₂O）           │
│     不适用：对称烯烃、Br₂、H₂等对称试剂                    │
├─────────────────────────────────────────────────────────┤
│ E3. 炔烃加成计量比写错                                    │
│     1mol炔烃最多消耗：2mol Br₂ 或 2mol H₂                │
│     （三键含2个π键）                                      │
│     1mol烯烃最多消耗：1mol Br₂ 或 1mol H₂                │
├─────────────────────────────────────────────────────────┤
│ E4. 醛酮加成只记C=O，忽略其他不饱和键                     │
│     醛基C=O可加氢还原，羧基C=O不可直接加氢                 │
│     （羧酸需用LiAlH₄等强还原剂）                          │
├─────────────────────────────────────────────────────────┤
│ E5. 忽视聚合反应是加成反应的延伸                          │
│     n CH₂=CH₂ → [CH₂-CH₂]ₙ 是多次加成聚合                │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        加成反应
       /    |    \
      ↓     ↓     ↓
   加卤素  加HX   加H₂
    |      |      |
   鉴别   制卤代烃  还原

加成反应 → 聚合反应（多次加成）
加成反应 + H₂ → 还原反应（催化加氢）
加成反应与取代反应是竞争关系
```

## 核心记忆口诀

> **不饱和键可加成，双键三键能打开**
> **溴水褪色是标志，马氏规则记心间**
> **氢多加氢主产物，烯烃炔烃要分辨**
