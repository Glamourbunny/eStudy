# 铬（Cr）元素卡片

> **元素档案**：第四周期，VIB族，原子序数24，银白色金属，密度7.19g/cm³，变价元素（+2/+3/+6），高熔点金属

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第4周期
- **族**：VIB族（铬副族）
- **原子序数**：24
- **核外电子排布**：(+24) 2)8)13)1

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Mo（钼） | 金属性Cr > Mo（但Mo熔点更高） |
| 下 | W（钨） | 金属性Cr > W（W熔点最高） |
| 左 | V（钒） | 金属性V > Cr（V更活泼） |
| 右 | Mn（锰） | 金属性Cr < Mn（Cr更稳定） |

### 推断题身份证 ⭐
- ⭐ "高熔点金属，熔点约1900℃" → Cr（W>Cr>Mn）
- ⭐ "变价+2/+3/+6，+3价最稳定" → Cr（类似Fe但Cr³⁺更稳定）
- ⭐ "能形成配合物" → [Cr(H₂O)₆]³⁺（紫色）、[Cr(NH₃)₆]³⁺（黄色）
- ⭐ "用于镀铬防腐" → Cr镀层（银白色，有光泽）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+24
- 电子层数：4层
- 最外层电子数：1e⁻
- **特殊**：d轨道有5e⁻（半满），但最外层只有1e⁻

### 【第2层】电子行为
- **可变价**：可以失去1e⁻（Cr³⁺）、2e⁻（Cr²⁺）、或全部d电子（Cr⁶⁺）
- **Cr³⁺最稳定**：d³构型（类似Mn²⁺），因为d轨道半满
- Cr⁶⁺形成CrO₃、CrO₄²⁻、Cr₂O₇²⁻（类似S、Mn的含氧酸根）

### 【第3层】化学键型
- **与氧**：形成CrO（碱性）、Cr₂O₃（两性）、CrO₃（酸性）
- **与酸**：生成Cr³⁺盐（或Cr²⁺）
- **与碱**：Cr(OH)₃显两性，可溶于NaOH
- **配合物**：易形成[Cr(NH₃)₆]³⁺、[Cr(CN)₆]³⁻等

### 【第4层】晶体类型
- **Cr单质**：金属晶体（体心立方）
- **Cr₂O₃**：离子晶体（刚玉结构，硬度大）
- **PbCrO₄**：离子晶体（铬黄颜料）

> **结构解释性质**：d轨道半满→Cr³⁺稳定（+3价Cr最常见）；高熔点源于金属键强

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色，有光泽 | 金属晶体 |
| 密度 | 7.19g/cm³（重金属） | 与Fe、Mn相近 |
| 熔点 | 1907℃ | 极高（VIB族特征） |
| 硬度 | 极硬（9，仅次于钻石） | 金属间键极强 |

### 化学性质

#### 还原性（+2/+3价Cr）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与酸（稀） | Cr + 2HCl = CrCl₂ + H₂↑ | 生成Cr²⁺（蓝色） |
| 与氧化剂 | 2Cr + 3Cl₂ = 2CrCl₃ | 生成Cr³⁺（绿色/紫色） |
| 与O₂ | 4Cr + 3O₂ = 2Cr₂O₃ | 表面生成致密氧化膜 |

🔴 **反例/边界**：Cr在空气中表面形成Cr₂O₃保护膜，耐腐蚀（用于镀铬）

#### 氧化物性质 🔴
| 氧化物 | 性质 | 反应 |
|--------|------|------|
| CrO | 碱性 | CrO + 2HCl = CrCl₂ + H₂O |
| Cr₂O₃ | 两性 | Cr₂O₃ + 6HCl = 2CrCl₃ + 3H₂O；Cr₂O₃ + 2NaOH = 2NaCrO₂ + H₂O |
| CrO₃ | 酸性 | CrO₃ + H₂O = H₂CrO₄（铬酸） |

🔴 **反例/边界**：Cr₂O₃不溶于水，但可用NaOH共熔转化为NaCrO₂

#### 铬酸盐/重铬酸盐 🔴
| 转化 | 方程式 | 说明 |
|------|--------|------|
| CrO₄²⁻ ⇌ Cr₂O₇²⁻ | 2CrO₄²⁻ + 2H⁺ ⇌ Cr₂O₇²⁻ + H₂O | 碱性黄色，酸性橙色 |
| 加碱 | Cr₂O₇²⁻ + 2OH⁻ → 2CrO₄²⁻ + H₂O | 黄色 |

🔴 **反例/边界**：CrO₄²⁻和Cr₂O₇²⁻颜色随pH变化，pH小时Cr₂O₇²⁻为主（橙色）

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Cr / Cr³⁺ / Cr₂O₃】

    Cr（银白色高熔点金属）
     │
     ├──[氧化]──→ Cr₂O₃（绿色）
     │    4Cr + 3O₂ = 2Cr₂O₃
     │    表面致密氧化膜，耐腐蚀
     │
     ├──[与酸]──→ Cr³⁺盐（绿色/紫色）
     │    2Cr + 6HCl = 2CrCl₃ + 3H₂↑
     │    Cr³⁺ + 6H₂O = [Cr(H₂O)₆]³⁺（紫色）
     │
     ├──[与碱]──→ CrO₂⁻（亚铬酸根，绿色）
     │    Cr₂O₃ + 2NaOH = 2NaCrO₂ + H₂O
     │
     └──[与氧化剂]──→ CrO₄²⁻/Cr₂O₇²⁻
          2CrCl₃ + 3H₂O₂ + 2NaOH = 2Na₂CrO₄ + 6HCl
          现象：溶液变黄色

                              【Cr³⁺的转化】

    Cr³⁺ + 3OH⁻ → Cr(OH)₃↓（灰绿色）
     │
    Cr(OH)₃ + OH⁻ → CrO₂⁻ + 2H₂O（两性溶解）
     │
    Cr(OH)₃ + 3HCl = CrCl₃ + 3H₂O
     │
    2Cr³⁺ + 3S²⁻ → Cr₂S₃↓（黑色）

                              【Cr(VI)的转化】

    Cr₂O₇²⁻（橙）+ H₂O ⇌ 2CrO₄²⁻（黄）+ 2H⁺
     │
    CrO₄²⁻ + Ba²⁺ → BaCrO₄↓（黄色，铬黄）
     │
    Cr₂O₇²⁻ + 2H⁺ → CrO₃（红色，铬酐）
     │
    🟢 检验：CrO₄²⁻ + Ba²⁺ → BaCrO₄↓（黄色沉淀，不溶于HAc）

                              【重要化合物】

    🔵 铬绿（Cr₂O₃）：颜料、抛光粉、绿色玻璃
       2Cr + 3O₂ = 2Cr₂O₃

    🔵 铬酸钾/钠（K₂CrO₄/Na₂CrO₄）：黄色，染色
       CrO₃ + 2KOH = K₂CrO₄ + H₂O

    🔵 重铬酸钾（K₂Cr₂O₇）：橙红色，氧化剂
       🔵 实验室制氯气：K₂Cr₂O₇ + 4HCl = 2KCl + 2CrCl₃ + Cl₂↑ + 2H₂O

    🔵 铬黄（PbCrO₄）：黄色颜料，有毒

    🔵 镀铬：Cr作镀层（银白光亮，耐腐蚀）

    🔵 铬铁合金：炼钢（不锈钢含有Cr）
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Cr的+3价不稳定 | Cr³⁺**非常稳定**（d³构型），是铬最常见的价态；比Fe³⁺更稳定 |
| **E2** | Cr₂O₃是酸性氧化物 | Cr₂O₃是**两性氧化物**，既溶于酸：Cr₂O₃ + 6HCl = 2CrCl₃ + 3H₂O；又溶于碱：Cr₂O₃ + 2NaOH = 2NaCrO₂ + H₂O |
| **E3** | CrO₄²⁻和Cr₂O₇²⁻颜色相同 | 两者颜色不同：CrO₄²⁻**黄色**，Cr₂O₇²⁻**橙红色**，可通过pH相互转化 |
| **E4** | Cr与酸反应生成Cr³⁺和H₂ | Cr与**非氧化性酸**反应可生成Cr²⁺（蓝色，不稳定易被氧化为Cr³⁺） |
| **E5** | Cr的熔点比W高 | W熔点3410℃，Cr熔点1907℃，**W的熔点更高**（元素中最高） |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Mn** | 同周期右邻 | Cr与Mn都有变价；Cr³⁺稳定，Mn²⁺稳定 |
| **Fe** | 同周期下下 | Fe-Cr合金（不锈钢），Cr防锈 |
| **Mo** | 同族上 | Mo熔点更高；两者都是高熔点金属 |
| **W** | 同族下下 | W熔点最高（3410℃）；Cr-W合金用于切削工具 |
| **S** | 相似性 | Cr₂(SO₄)₃与Fe₂(SO₄)₃性质相似（矾类） |
| **Al** | 相似性 | Cr₂O₃与Al₂O₃都是两性氧化物，且都是离子晶体（刚玉结构） |
| **V** | 同周期左邻 | V也是高熔点金属（V₂O₅是催化剂） |
