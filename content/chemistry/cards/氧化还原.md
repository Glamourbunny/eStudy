# 氧化还原反应（Redox）

> **反应本质**：电子转移（得失或偏移）的反应。特征是反应前后元素化合价发生变化。氧化还原反应是化学反应中最重要的反应类型之一。

---

## 一、反应本质与判断标志

```
判断标志（任一满足即可）：
✓ 有元素化合价变化
✓ 有电子得失或偏移
✓ 有单质参与或生成（不一定！有例外）
```

### 核心关系图
```
        氧化还原反应
       /            \
      ↓              ↓
  氧化反应        还原反应
  （失电子）      （得电子）
      ↑              ↑
   还原剂         氧化剂
  （被氧化）     （被还原）
```

---

## 二、无机代表反应

### 置换反应（一定是氧化还原）
```
Zn + H₂SO₄ → ZnSO₄ + H₂↑
Zn: 0→+2（失电子，被氧化，还原剂）
H: +1→0（得电子，被还原，氧化剂）

Cl₂ + 2NaBr → 2NaCl + Br₂
Cl: 0→-1（得电子，被还原，氧化剂）
Br: -1→0（失电子，被氧化，还原剂）
```

### 化合反应（部分是）
```
2Na + Cl₂ → 2NaCl          ✓ 是氧化还原
CaO + H₂O → Ca(OH)₂        ✗ 不是氧化还原
```

### 分解反应（部分是）
```
2KClO₃ —MnO₂/△→ 2KCl + 3O₂↑   ✓ 是氧化还原
CaCO₃ —△→ CaO + CO₂↑          ✗ 不是氧化还原
```

### 歧化反应
```
Cl₂ + 2NaOH → NaCl + NaClO + H₂O
Cl: 0→-1（被还原）和 0→+1（被氧化）
同一元素既升又降
```

---

## 三、有机代表反应

### 燃烧反应
```
CH₄ + 2O₂ → CO₂ + 2H₂O
C: -4→+4（被氧化）
O: 0→-2（被还原）
```

### 催化氧化
```
2CH₃CH₂OH + O₂ —Cu/△→ 2CH₃CHO + 2H₂O
乙醇中的C被氧化（去氢），O₂被还原
```

## 四、规律与原理

### 氧化还原反应与四大基本反应的关系
```
              化学反应
                 │
      ┌──────────┴──────────┐
      ↓                     ↓
 氧化还原反应           非氧化还原反应
      │                     │
   ┌──┼──┐               ┌──┼──┐
   ↓  ↓  ↓               ↓  ↓  ↓
 置换 部分化合 部分分解    复分解 部分化合 部分分解
```

### 氧化性/还原性强弱判断
```
氧化性：氧化剂 > 氧化产物
还原性：还原剂 > 还原产物

金属活动性顺序（还原性）：
K > Ca > Na > Mg > Al > Zn > Fe > Sn > Pb > (H) > Cu > Hg > Ag > Pt > Au

对应阳离子氧化性顺序（相反）：
K⁺ < Ca²⁺ < Na⁺ < Mg²⁺ < Al³⁺ < Zn²⁺ < Fe²⁺ < Sn²⁺ < Pb²⁺ < H⁺ < Cu²⁺ < Hg²⁺ < Ag⁺
```

### 电子守恒计算
```
氧化剂得电子总数 = 还原剂失电子总数

计算方法：
① 标出变价元素的化合价
② 计算每个原子得失电子数
③ 乘以原子个数，令得失相等
④ 配平其他原子

例：2KMnO₄ + 16HCl → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O
Mn: +7→+2，每个Mn得5e⁻，2个Mn共得10e⁻
Cl: -1→0，每个Cl失1e⁻，10个Cl共失10e⁻
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 氧化还原常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 有单质参加就一定是氧化还原？                          │
│     不一定！同素异形体转化不是氧化还原：                   │
│     3O₂ ⇌ 2O₃（化合价均为0，无变化）                     │
│     金刚石→石墨（化合价均为0）                            │
├─────────────────────────────────────────────────────────┤
│ E2. 氧化还原方程式配平错误                                │
│     必须同时满足：原子守恒 + 电子守恒 + 电荷守恒           │
│     酸性条件用H⁺和H₂O配平，碱性条件用OH⁻和H₂O配平        │
├─────────────────────────────────────────────────────────┤
│ E3. 歧化反应 vs 归中反应混淆                              │
│     歧化：同一元素，0价→高价+低价                         │
│       例：Cl₂ + 2NaOH → NaCl + NaClO + H₂O              │
│     归中：同一元素，高价+低价→中间价                      │
│       例：2H₂S + SO₂ → 3S↓ + 2H₂O                       │
├─────────────────────────────────────────────────────────┤
│ E4. 氧化产物/还原产物判断错误                             │
│     还原剂 → 被氧化 → 氧化产物                            │
│     氧化剂 → 被还原 → 还原产物                            │
│     记忆口诀："升失氧还氧，降得还氧还"                    │
├─────────────────────────────────────────────────────────┤
│ E5. 过量问题中的氧化还原顺序                              │
│     多种还原剂与氧化剂反应时，还原性强的先反应             │
│     例：FeBr₂溶液中通入Cl₂，Fe²⁺先被氧化，Br⁻后被氧化     │
│     （还原性：Fe²⁺ > Br⁻）                                │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        氧化还原反应
       /    |    |    \
      ↓     ↓    ↓     ↓
   置换   化合   分解   电化学
   反应   (部分) (部分) 反应
    |      |      |      |
  一定    部分   部分   原电池
  是      是     是     电解池

氧化还原 + 离子反应 → 有电子转移的离子反应
氧化还原 + 有机反应 → 醇→醛→酸的氧化链
```

## 核心记忆口诀

> **升失氧还氧，降得还氧还**
> **（升价失电子被氧化还原剂氧化产物，**
> ** 降价得电子被还原氧化剂还原产物）**
> **置换一定是氧化还原，复分解一定不是**
> **电子守恒配平方，原子电荷不能忘**
