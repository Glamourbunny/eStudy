# 取代反应（Substitution）

> **反应本质**：旧化学键断裂 + 新化学键形成，原子或原子团被替代。饱和键特征反应。

---

## 一、反应本质与判断标志

```
判断标志：
✓ 反应物中存在可被替代的基团
✓ 产物中生成新的单键化合物
✓ 不饱和度不变（饱和键特征）
```

### 核心通式
```
R—X + Y → R—Y + X
（X被Y取代）
```

---

## 二、无机代表反应

| 反应 | 方程式 | 条件 | 取代特征 |
|------|--------|------|----------|
| **酸碱中和** | `HCl + NaOH → NaCl + H₂O` | 常温 | H⁺被Na⁺取代（离子层面） |
| **复分解反应** | `AgNO₃ + NaCl → AgCl↓ + NaNO₃` | 常温 | Cl⁻被NO₃⁻取代 |
| **甲烷氯代** | `CH₄ + Cl₂ → CH₃Cl + HCl` | 光照 | H被Cl取代 |
| **酯化反应** | `CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O` | 浓H₂SO₄/加热 | —OH被—OR取代 |

---

## 三、有机代表反应

### 烷烃卤代
```
CH₄ + Cl₂ —光→ CH₃Cl + HCl
CH₃Cl + Cl₂ —光→ CH₂Cl₂ + HCl
（可继续取代至CCl₄）
```

### 苯的取代
```
苯 + Br₂ —FeBr₃→ 溴苯 + HBr        （卤代）
苯 + HNO₃ —浓H₂SO₄→ 硝基苯 + H₂O   （硝化）
苯 + H₂SO₄ —△→ 苯磺酸 + H₂O        （磺化）
```

### 酯化与水解（互为逆过程）
```
羧酸 + 醇 ⇌ 酯 + 水
酯 + H₂O ⇌ 羧酸 + 醇
⚠️ 酯化用浓H₂SO₄催化，水解用酸或碱催化
```

---

## 四、规律与原理

### 取代反应 vs 加成反应 关键区别

| 对比项 | 取代反应 | 加成反应 |
|--------|----------|----------|
| **反应物不饱和度** | 饱和（单键） | 不饱和（双/三键） |
| **产物数量** | 两种产物 | 一种产物 |
| **键的变化** | 旧键断，新键成 | 不饱和键打开 |
| **典型代表** | 烷烃、苯 | 烯烃、炔烃 |

### 取代活性规律
```
烷烃卤代活性：F₂ > Cl₂ > Br₂ > I₂
             （太剧烈）  （常用）   （较慢）  （难反应）
苯环取代定位：邻对位定位基（—OH、—CH₃）
             间位定位基（—NO₂、—SO₃H）
```

## 五、纵向贯通——后续扩展

```
取代反应知识链：

卤代烃水解 ──→ 醇的生成
     │
     ↓
酯化反应 ──→ 酯的生成 ──→ 酯的水解（皂化）
     │
     ↓
苯的取代 ──→ 硝化/磺化 ──→ 染料/药物合成基础
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 取代反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆取代反应与置换反应                                │
│     取代：化合物+化合物/单质 → 新化合物+新化合物/单质       │
│     置换：单质+化合物 → 新单质+新化合物                     │
│     例：CH₄+Cl₂→CH₃Cl+HCl 是取代（不是置换！）             │
├─────────────────────────────────────────────────────────┤
│ E2. 认为苯能与溴水反应                                    │
│     苯与液溴在FeBr₃催化下取代，与溴水不反应（萃取褪色）     │
├─────────────────────────────────────────────────────────┤
│ E3. 酯化反应条件写错                                      │
│     正确：浓H₂SO₄ + 加热                                  │
│     错误：稀H₂SO₄或只写"催化剂"                           │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视酯水解的可逆性                                    │
│     酸性水解：可逆                                        │
│     碱性水解（皂化）：不可逆（产物羧酸盐不逆反应）          │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        取代反应
       /    |    \
      ↓     ↓     ↓
   酯化   水解   卤代
    ↑      ↑      ↑
   酸+醇  酯+H₂O 烷烃+X₂
    
取代反应 ↔ 加成反应（竞争关系）
取代反应 → 氧化反应（部分取代伴随氧化）
```
