# 钙（Ca）元素卡片

> **元素档案**：第三周期，IIA族，原子序数20，银白色金属，密度1.54g/cm³，金属性活泼，与Na同周期右邻，金属性Ca>Mg

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：IIA族（碱土金属）
- **原子序数**：20
- **核外电子排布**：(+20) 2)8)8)2

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Mg（镁） | 金属性Ca > Mg（原子半径增大，失电子能力增强） |
| 下 | Sr（锶） | 金属性Ca < Sr（同族向下金属性增强） |
| 左 | K（钾） | 金属性K > Ca（同一周期左>右） |
| 右 | Sc（钪） | 金属性Ca > Sc（但Sc是过渡金属） |

### 推断题身份证 ⭐
- ⭐ "与水反应生成无色气体，反应剧烈" → Ca + 2H₂O = 2Ca(OH)₂ + H₂↑（比Mg剧烈）
- ⭐ "在空气中易氧化成白色粉末" → CaO（氧化钙）+ Ca(OH)₂
- ⭐ "焰色反应呈砖红色" → Ca
- ⭐ "可作补钙剂" → CaCO₃（碳酸钙）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+20
- 电子层数：4层
- 最外层电子数：2e⁻（易失）

### 【第2层】电子行为
- **易失**：失去最外层2e⁻ → Ca²⁺（18电子稳定结构，类似Ar）
- 失电子能力很强，比Mg更容易失去电子

### 【第3层】化学键型
- **与氧**：形成CaO（离子化合物）
- **与卤素**：形成CaCl₂、CaBr₂（离子化合物）
- **与酸**：生成Ca²⁺盐 + H₂↑
- **与水**：剧烈反应生成Ca(OH)₂ + H₂↑

### 【第4层】晶体类型
- **Ca单质**：金属晶体（面心立方结构）
- **CaO**：离子晶体（熔点2572℃）
- **Ca(OH)₂**：离子晶体（层状结构，微溶）
- **CaCO₃**：离子晶体（三方晶系，难溶）

> **结构解释性质**：Ca原子比Mg大→失电子更容易→金属性更强→与水反应更剧烈

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色（新鲜断面） | 金属晶体 |
| 密度 | 1.54g/cm³（轻金属） | 比Mg（1.74）更轻 |
| 熔点 | 842℃ | 较低（比Mg低） |
| 硬度 | 较软（可用刀切） | 金属键较弱 |

### 化学性质

#### 还原性（被氧化）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与O₂ | 2Ca + O₂ = 2CaO | 常温缓慢氧化，点燃剧烈燃烧 |
| 与N₂ | 3Ca + N₂ =△= Ca₃N₂ | 氮化钙（灰绿色固体） |
| 与H₂O | Ca + 2H₂O = 2Ca(OH)₂ + H₂↑ | 冷水即剧烈反应 |
| 与酸 | Ca + 2HCl = CaCl₂ + H₂↑ | 气泡，反应剧烈 |

🔴 **反例/边界**：Ca与冷水反应比Mg剧烈得多，但生成的Ca(OH)₂微溶，覆盖表面使反应减慢

#### 特殊反应 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与CO₂ | Ca + CO₂ =△= CaO + C | 高温下反应（不常用） |
| 与氧化物 | Ca + CuO =△= CaO + Cu | 还原性体现 |

🔴 **反例/边界**：Ca不能像Mg一样在CO₂中燃烧（Ca的还原性不如Mg强）

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Ca²⁺ / Ca(OH)₂】

    Ca（银白色金属）
     │
     ├──[氧化]──→ CaO（白色固体）
     │    2Ca + O₂ = 2CaO
     │    现象：银白色变白色粉末
     │
     ├──[与水]──→ Ca(OH)₂ + H₂↑
     │    Ca + 2H₂O = 2Ca(OH)₂ + H₂↑
     │    剧烈反应，溶液变浑浊
     │
     ├──[与酸]──→ CaCl₂/CaSO₄ + H₂↑
     │    Ca + 2HCl = CaCl₂ + H₂↑
     │
     └──[与CO₂]──→ CaCO₃（碳酸钙沉淀）
          Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O
          现象：澄清石灰水变浑浊

                              【CaO的转化】

    CaO + H₂O = Ca(OH)₂（放热）
     │
    CaO + 2HCl = CaCl₂ + H₂O
     │
    CaO + CO₂ = CaCO₃↓（慢慢吸收CO₂）

                              【Ca(OH)₂的转化】

    Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O
     │
    Ca(OH)₂ + 2HCl = CaCl₂ + 2H₂O
     │
    Ca(OH)₂ =△= CaO + H₂O（煅烧分解）

                              【重要化合物】

    🔵 生石灰：CaO（建筑材料、干燥剂）
       Ca(OH)₂ + CO₂ = CaCO₃ + H₂O（石灰浆固化）

    🔵 熟石灰：Ca(OH)₂（石灰水、砌砖）
       检验CO₂：Ca(OH)₂ + CO₂ = CaCO₃↓ + H₂O

    🔵 大理石/石灰石：CaCO₃
       CaCO₃ =△= CaO + CO₂↑（高温分解）

    🔵 补钙剂：CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑

    🔵 硬水软化：用Na₆₂CO₃去除Ca²⁺、Mg²⁺
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Ca与水反应与Na一样剧烈 | Ca与冷水反应剧烈但不如Na（Na密度小于水浮在水面，反应放热可点燃H₂） |
| **E2** | CaO是酸性氧化物 | CaO是**碱性氧化物**，与酸反应：CaO + 2HCl = CaCl₂ + H₂O |
| **E3** | Ca(OH)₂可任意溶解 | Ca(OH)₂**微溶**（约0.16g/100mL），饱和溶液称石灰水 |
| **E4** | Ca在空气中稳定 | Ca在空气中**易氧化**，需密封保存；表面生成CaO和Ca(OH)₂ |
| **E5** | CaCO₃加热分解只需加热 | CaCO₃高温分解（煅烧石灰窑，约900℃）；普通加热难以分解 |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Mg** | 同族上 | 金属性Ca > Mg（Ca与水反应更剧烈） |
| **Sr** | 同族下 | 金属性Ca < Sr（锶更活泼） |
| **Na** | 同周期右邻 | 金属性Na > Ca（同一周期左>右） |
| **K** | 同周期左邻 | 金属性K > Na > Ca |
| **Ba** | 同族下下 | 金属性Ba > Ca，BaSO₄难溶（医学上用于消化道X射线检查） |
| **Fe** | 联合出题 | 炼钢时Ca可脱氧脱硫：Ca + O = CaO，Ca + S = CaS |
