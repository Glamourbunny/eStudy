# 锌（Zn）元素卡片

> **元素档案**：第四周期，IIB族，原子序数30，蓝白色金属，密度7.14g/cm³，两性金属但主要显碱性，与Fe同族

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第4周期
- **族**：IIB族（锌副族）
- **原子序数**：30
- **核外电子排布**：(+30) 2)8)18)2

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Cd（镉） | 金属性Zn > Cd（同族向上金属性减弱） |
| 下 | Hg（汞） | 金属性Zn > Hg（汞为液体） |
| 左 | Cu（铜） | 金属性Zn > Cu（Zn更活泼） |
| 右 | Ga（镓） | 金属性Zn > Ga（Ga是半导体元素） |

### 推断题身份证 ⭐
- ⭐ "两性金属，能溶于NaOH" → Zn + 2NaOH = Na₂ZnO₂ + H₂↑（生成锌酸钠）
- ⭐ "与酸反应生成无色气体" → Zn + 2HCl = ZnCl₂ + H₂↑
- ⭐ "在空气中表面生成致密氧化膜" → ZnO（灰白色，保护内部）
- ⭐ "可用于镀锌防锈" → 牺牲阳极保护法

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+30
- 电子层数：4层
- 最外层电子数：2e⁻（d轨道全满10e⁻，稳定）

### 【第2层】电子行为
- **易失**：失去最外层2e⁻ → Zn²⁺（18电子稳定结构）
- **特殊**：d轨道全满，化学性质相对稳定（比Fe、Cu稳定）
- 不易失去第二个电子（没有+4价）

### 【第3层】化学键型
- **与氧**：形成ZnO（两性氧化物）
- **与酸**：生成Zn²⁺盐 + H₂↑
- **与碱**：生成[Zn(OH)₄]²⁻或[ZnO₂]²⁻（锌酸根）
- **与氨水**：形成[Zn(NH₃)₄]²⁺（氨配合物）

### 【第4层】晶体类型
- **Zn单质**：金属晶体（六方密排结构）
- **ZnO**：离子晶体（纤锌矿结构）
- **ZnS**：离子晶体（闪锌矿/纤锌矿）

> **结构解释性质**：d轨道全满→结构稳定→ Zn在空气中不易被腐蚀（比Fe耐蚀）

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 蓝白色，有金属光泽 | 金属晶体 |
| 密度 | 7.14g/cm³（重金属） | 比Fe（7.87）略轻 |
| 熔点 | 419.5℃ | 较低（IIB族特点） |
| 延展性 | 好（能压成薄片） | 金属键 |

### 化学性质

#### 还原性（被氧化）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与O₂ | 2Zn + O₂ = 2ZnO | 加热，在表面生成灰白色ZnO |
| 与酸（非氧化性） | Zn + 2HCl = ZnCl₂ + H₂↑ | 气泡，溶解 |
| 与碱 | Zn + 2NaOH = Na₂ZnO₂ + H₂↑ | 溶解，生成氢气 |
| 与氨水 | Zn + 4NH₃·H₂O = [Zn(NH₃)₄]²⁺ + H₂↑ + 4H₂O | 溶解形成无色配合物 |

🔴 **反例/边界**：Zn与浓H₂SO₄、浓HNO₃反应生成SO₂、NO₂等（氧化性酸还原产物）

#### 两性性质 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与NaOH | Zn + 2NaOH = Na₂ZnO₂ + H₂↑ | 生成锌酸钠 |
| ZnO + NaOH | ZnO + 2NaOH = Na₂ZnO₂ + H₂O | 两性氧化物 |
| Zn(OH)₂ + NaOH | Zn(OH)₂ + 2NaOH = Na₂[Zn(OH)₄] | 氢氧化锌的两性 |

🔴 **反例/边界**：Zn的两性不如Al典型，主要显碱性；ZnO是弱两性

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Zn / Zn²⁺ / ZnO】

    Zn（蓝白色金属）
     │
     ├──[氧化]──→ ZnO（灰白色）
     │    2Zn + O₂ = 2ZnO
     │    表面形成致密保护膜
     │
     ├──[与酸]──→ ZnCl₂/ZnSO₄ + H₂↑
     │    Zn + 2HCl = ZnCl₂ + H₂↑
     │    🟢 检验：加NaOH生成白色沉淀→溶解于过量NaOH
     │
     ├──[与碱]──→ Na₂ZnO₂ + H₂↑
     │    Zn + 2NaOH = Na₂ZnO₂ + H₂↑
     │    两性金属特征反应
     │
     └──[与氨水]──→ [Zn(NH₃)₄]²⁺
          Zn + 4NH₃·H₂O = [Zn(NH₃)₄](OH)₂ + H₂↑ + 2H₂O
          现象：溶解形成无色溶液

                              【Zn²⁺的转化】

    Zn²⁺ + 2OH⁻ → Zn(OH)₂↓（白色）
     │
    Zn(OH)₂ + 2OH⁻ → [Zn(OH)₄]²⁻（溶解）
     │
    ZnO + 2HCl = ZnCl₂ + H₂O
     │
    ZnO + NaOH = Na₂ZnO₂ + H₂O

                              【Zn(OH)₂的转化】

    Zn(OH)₂ =△= ZnO + H₂O（加热分解）
     │
    Zn(OH)₂ + 2HCl = ZnCl₂ + 2H₂O
     │
    Zn(OH)₂ + 2NaOH = Na₂[Zn(OH)₄]（两性溶解）

                              【重要化合物】

    🔵 锌白（ZnO）：白色颜料、 sunscreen
       ZnO + H₂SO₄ = ZnSO₄ + H₂O

    🔵 皓矾（ZnSO₄·7H₂O）：收敛剂、媒染剂

    🔵 菱锌矿（ZnCO₃）：主要锌矿石
       ZnCO₃ =△= ZnO + CO₂↑

    🔵 镀锌：Zn作牺牲阳极，保护Fe（电化学防腐）

    🔵 干电池：Zn作负极（外壳）
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Zn是两性金属所以显酸性 | Zn主要显**碱性**，两性是次要特征；与酸反应更剧烈 |
| **E2** | Zn与酸反应不如Fe剧烈 | Zn的金属性比Fe强，与酸反应**更剧烈**（但表面氧化膜需先去除） |
| **E3** | ZnO是酸性氧化物 | ZnO是**两性氧化物**，既能与酸反应：ZnO + 2HCl = ZnCl₂ + H₂O；也能与碱反应：ZnO + 2NaOH = Na₂ZnO₂ + H₂O |
| **E4** | Zn与NaOH反应是酸性氧化物性质 | Zn与NaOH反应体现的是Zn的**两性金属**性质，不是酸性氧化物性质 |
| **E5** | Zn比Fe更容易生锈 | 实际上Zn在空气中生成致密ZnO保护膜，**比Fe更难腐蚀**（镀锌防锈就是这个原理） |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Fe** | 同族（VIIIB） | Zn与Fe同族，但Zn更稳定；Zn可镀在Fe表面防腐 |
| **Cu** | 同周期右邻 | 金属性Zn > Cu；Zn可从CuSO₄溶液中置换Cu |
| **Cd** | 同族下 | 金属性Zn > Cd（同族向上金属性增强） |
| **Hg** | 同族下下 | Zn > Hg（汞是液态金属） |
| **Mg** | 联合出题 | Zn-Mn干电池；Mg-Zn合金 |
| **Al** | 相似性 | 两者都有两性，但Al的两性更典型 |
| **S** | 反应伴侣 | Zn + S =△= ZnS（白色固体） |
