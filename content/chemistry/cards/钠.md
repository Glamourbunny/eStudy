# 钠（Na）元素卡片

> **元素档案**：第三周期，IA族，原子序数11，银白色金属，质软如蜡，密度0.97g/cm³（比水轻）

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：IA族（碱金属）
- **原子序数**：11
- **核外电子排布**：(+11) 2)8)1

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Li（锂） | 金属性更强，熔点更高 |
| 下 | K（钾） | 金属性更强，熔点更低 |
| 右 | Mg（镁） | 失电子能力Na>K，金属性Na更强 |

### 推断题身份证 ⭐
- ⭐ "投入水中浮在水面，熔成小球，四处游动" → Na
- ⭐ "保存于煤油中的银白色金属" → Na（与K区分：K密度更小需密封）
- ⭐ "焰色反应呈黄色" → Na（可检测Na⁺）
- ⭐ "与水反应生成无色无味气体" → H₂（可点燃）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+11
- 电子层数：3层
- 最外层电子数：1e⁻（易失电子）

### 【第2层】电子行为
- **易失**：失去最外层1e⁻ → Na⁺（10电子稳定结构）
- 失去1个电子后，从IA族→变成稳定的惰性气体电子层结构

### 【第3层】化学键型
- **与非金属**：形成离子键 → NaCl、Na₂O（离子化合物）
- **与水反应**：离子反应，Na失电子，水中H⁺得电子

### 【第4层】晶体类型
- **金属钠**：金属晶体（自由电子海）
- 解释：熔点97.8℃，低（金属键较弱）；导电导热性好（自由电子）

> **结构解释性质**：Na原子半径比Li大 → 最外层电子更易失去 → Na比Li更活泼

---

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色，质软（可用刀切） | 金属晶体，金属键较弱 |
| 密度 | 0.97g/cm³（比水轻） | 投入水中浮在水面 |
| 熔点 | 97.8℃（低） | 钠与水反应放热可使其熔化 |
| 导热导电 | 优良 | 自由电子在金属晶体中可移动 |

### 化学性质

#### 氧化性（被还原）🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂ | 4Na + O₂ = 2Na₂O | 常温生成Na₂O（白色） |
| 与O₂（点燃） | 2Na + O₂ =点燃= Na₂O₂ | 燃烧生成Na₂O₂（淡黄色） |
| 与S | 2Na + S = Na₂S | 研磨混合可能爆炸 |

🔴 **反例/边界**：钠露置空气中先变暗（Na₂O），不会直接生成Na₂O₂（需点燃条件）

#### 还原性（还原H₂O）🔴
| 反应 | 方程式 | 现象 |
|------|--------|------|
| 与H₂O | 2Na + 2H₂O = 2NaOH + H₂↑ | 浮、熔、游、响、红 |
| 与酸 | 2Na + 2HCl = 2NaCl + H₂↑ | 剧烈反应 |

🔴 **反例/边界**：钠与CuSO₄溶液反应，不是简单的置换
- 2Na + 2H₂O = 2NaOH + H₂↑
- 2NaOH + CuSO₄ = Cu(OH)₂↓（蓝色沉淀）+ Na₂SO₄
- 先看到蓝色沉淀，后有少量H₂逸出

#### 与盐溶液反应顺序
1. Na与水反应生成NaOH
2. NaOH再与盐反应
3. **不能**直接置换出金属（除K、Ca、Na顺序的金属）

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Na₂O / NaOH】

         H₂O
    Na₂O ──────────→ NaOH  （Na₂O + H₂O = 2NaOH）
         可逆，剧烈放热

    Na ──[常温]──→ Na₂O（白色） ──[点燃]──→ Na₂O₂（淡黄色）
         4Na + O₂ = 2Na₂O              2Na + O₂ =点燃= 2Na₂O₂

    Na₂O₂ ──[与CO₂]──→ Na₂CO₃ + O₂    （呼吸面具用途）
             2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂

    NaOH ──[与CO₂]──→ Na₂CO₃ ──[继续通CO₂]──→ NaHCO₃
             2NaOH + CO₂ = Na₂CO₃ + H₂O
                                 Na₂CO₃ + H₂O + CO₂ = 2NaHCO₃

    Na ──[与Cl₂]──→ NaCl（点燃，黄光）
             2Na + Cl₂ =点燃= 2NaCl

    NaOH + HCl = NaCl + H₂O     （中和反应）

         【实验室与工业】

    🔵 电解饱和NaCl溶液：2NaCl + 2H₂O =电解= 2NaOH + Cl₂↑ + H₂↑
         （氯碱工业的核心反应）

    🔵 侯氏制碱法：NaCl + NH₃ + CO₂ + H₂O = NaHCO₃↓ + NH₄Cl
         （先得NaHCO₃，再煅烧得Na₂CO₃）
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | 钠与CuSO₄溶液反应置换出Cu | 先生成NaOH，再与CuSO₄生成Cu(OH)₂蓝色沉淀，无金属Cu生成 |
| **E2** | 钠保存在水中 | 钠与水剧烈反应，必须保存在**煤油**中（密度煤油>Na>水） |
| **E3** | 钠与盐酸反应先与HCl反应 | Na先与H₂O反应（即使有酸存在，水更多），本质上都是还原H⁺ |
| **E4** | Na₂O₂是碱性氧化物 | Na₂O₂是过氧化物，与水反应生成NaOH**和H₂O₂**，不是简单碱性氧化物 |
| **E5** | 焰色反应呈黄色可以确定样品中含有Na | 焰色反应是**检验**手段，但Na⁺在火焰中呈现黄色只是物理现象，不能说"含Na即黄色" |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **K** | 上下族 | 同 IA 族，性质相似，Na < K（更活泼） |
| **Mg** | 右邻 | 第三周期金属性Na > Mg（失去电子能力） |
| **Al** | 右邻 | 两性，Na比Al更活泼（Na能直接与水反应，Al不能） |
| **Cl** | 反应伴侣 | 2Na + Cl₂ = 2NaCl（工业制盐） |
| **O** | 反应伴侣 | 4Na + O₂ = 2Na₂O；2Na + O₂ = Na₂O₂ |
| **Fe** | 联合出题 | 钠保存在钢铁容器中（防氧化） |

### 联合推断规律
- Na ──联合推断──→ 与Fe、Cu联合（钠先与水反应，不能直接置换不活泼金属）
- Na ──工业流程──→ 与Cl、S联合（氯碱工业、制Na₂S）
- Na ──实验探究──→ 与K联合（焰色反应：Na黄、K紫）
