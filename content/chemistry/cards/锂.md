# 锂（Li）元素卡片

> **元素档案**：第二周期，IA族，原子序数3，银白色金属，密度0.534g/cm³，最轻的金属，质地软

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第2周期
- **族**：IA族（碱金属）
- **原子序数**：3
- **核外电子排布**：(+3) 2)1

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | 无（第1周期无IA族） | — |
| 下 | Na（钠） | 金属性Li < Na（失电子能力） |
| 右 | Be（铍） | 金属性Li > Be（Li更活泼） |

### 推断题身份证 ⭐
- ⭐ "最轻的金属，密度比水还小" → Li（0.534g/cm³）
- ⭐ "保存在石蜡中" → Li（Na、K保存在煤油中，但Li密度太小需石蜡）
- ⭐ "焰色反应呈红色" → Li（深红色）
- ⭐ "与水反应不如Na剧烈" → Li（Li比Na活泼性弱）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+3
- 电子层数：2层
- 最外层电子数：1e⁻（易失电子）

### 【第2层】电子行为
- **易失**：失去最外层1e⁻ → Li⁺（2电子稳定结构，类似He）
- 失去1个电子后变成稳定的惰性气体电子层结构

### 【第3层】化学键型
- **与氧**：形成Li₂O（离子化合物，过氧化物Li₂O₂较少见）
- **与氮**：形成Li₃N（氮化锂，Li是唯一能直接与N₂反应的碱金属）
- **与卤素**：形成LiX（LiCl、LiBr等）

### 【第4层】晶体类型
- **Li单质**：金属晶体
- **Li₂O**：离子晶体（熔点1700℃，高）
- 解释：离子键强→氧化物熔点高

> **结构解释性质**：Li原子半径比Na小→失电子能力比Na弱→Li不如Na活泼

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色，质软 | 金属晶体，金属键较弱 |
| 密度 | 0.534g/cm³（最轻的金属） | 能浮在煤油上（但石蜡密度更小） |
| 熔点 | 180.5℃ | 比Na（97.8℃）高 |
| 硬度 | 低，可用刀切 | 金属键较弱 |

### 化学性质

#### 还原性（被氧化）🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂ | 4Li + O₂ = 2Li₂O | 常温缓慢，生成Li₂O（白色） |
| 与N₂ | 6Li + N₂ = 2Li₃N | Li是唯一能在常温与N₂反应的碱金属 |
| 与H₂O | 2Li + 2H₂O = 2LiOH + H₂↑ | 反应剧烈但不如Na（Li密度小浮在水面） |
| 与酸 | 2Li + 2HCl = 2LiCl + H₂↑ | 剧烈 |

🔴 **反例/边界**：Li与N₂反应生成Li₃N（氮化锂），其他碱金属无此性质

#### 与氧族反应 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂ | 4Li + O₂ = 2Li₂O | 生成过氧化物Li₂O₂较难（比Na特殊） |
| 与S | 2Li + S = Li₂S | 加热或研磨 |

🔴 **反例/边界**：Li不形成稳定的过氧化物（Li₂O₂），而Na可形成Na₂O₂

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Li₂O / LiOH】

    Li（银白色金属）
     │
     ├──[与O₂]──→ Li₂O（白色）
     │    4Li + O₂ = 2Li₂O
     │
     ├──[与N₂]──→ Li₃N（氮化锂，红色）
     │    6Li + N₂ = 2Li₃N
     │    🟢 Li是唯一能在常温与N₂反应的金属
     │
     ├──[与H₂O]──→ LiOH + H₂↑
     │    2Li + 2H₂O = 2LiOH + H₂↑
     │    现象：Li浮在水面，快速反应
     │
     └──[与酸]──→ LiCl + H₂↑
         2Li + 2HCl = 2LiCl + H₂↑

                              【Li₂O的转化】

    Li₂O + H₂O = 2LiOH（可逆）
     │
     └── 2LiOH =△= Li₂O + H₂O（煅烧分解）

                              【LiOH的转化】

    LiOH + HCl = LiCl + H₂O
     │
     ├── 热稳定性好（熔点445℃）
     │
     └── 可作电解质（锂电池电解液前身）

                              【锂电池核心】

    🔵 锂离子电池：
       正极：LiCoO₂（钴酸锂）
       负极：C（石墨）
       反应：Li⁺在两极间嵌入脱出
       优点：能量密度高、重量轻
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Li保存在煤油中 | Li密度太小（0.534），煤油密度约0.8，Li会浮在煤油上无法隔绝空气，需保存在**石蜡**中 |
| **E2** | Li与Na化学性质完全相同 | 不同！Li与N₂反应生成Li₃N，而Na无此反应；Li不形成稳定的过氧化物 |
| **E3** | LiOH是强碱 | LiOH是**中等强度的碱**，碱性弱于NaOH（因Li⁺半径小，水解程度大） |
| **E4** | 焰色反应Li呈黄色 | Li焰色反应呈**红色（深红色）**，不是黄色（Na是黄色） |
| **E5** | Li比Na更活泼 | 事实相反：Na比Li更活泼（Na原子半径更大，最外层电子更容易失去） |
