# 吸热反应 vs 放热反应

> **底线地位**：取决于反应物总键能 vs 生成物总键能，与反应条件（是否加热）无关。这是能量变化辨析的核心。

---

## 一、概念定义对比

| 维度 | 吸热反应 | 放热反应 |
|------|----------|----------|
| **能量变化** | 吸收热量（ΔH > 0） | 放出热量（ΔH < 0） |
| **键能关系** | 断键吸热 > 成键放热 | 断键吸热 < 成键放热 |
| **能量图像** | 生成物能量 > 反应物能量 | 生成物能量 < 反应物能量 |
| **常见类型** | 大多数分解、C+CO₂、铵盐溶解 | 大多数化合、燃烧、中和、金属+酸 |

---

## 二、核心区别：键能与条件无关

```
热量变化的本质：
断键吸收的能量  vs  成键放出的能量

断键吸热 > 成键放热 → 净吸热 → 吸热反应
断键吸热 < 成键放热 → 净放热 → 放热反应

⚠️ 关键提醒：
反应是吸热还是放热，与反应条件（是否加热）无关！
加热只是提供活化能，不是判断依据。
```

### 能量变化图示

```
  │       /             │      / \

```

## 三、判断方法

```
方法一：记忆常见反应类型
  放热：燃烧、中和、金属+酸/水、大多数化合
  吸热：大多数分解、C+CO₂→2CO、Ba(OH)₂·8H₂O+NH₄Cl

方法二：比较键能（本质方法）
  反应物总键能 < 生成物总键能 → 放热
  反应物总键能 > 生成物总键能 → 吸热

方法三：实验现象（粗略）
  温度升高 → 放热
  温度降低 → 吸热
```

## 四、典型例子

### 例1：条件≠吸放热
```
C + O₂ → CO₂      点燃，但放热反应
N₂ + O₂ → 2NO     高温/放电，但吸热反应
CaCO₃ → CaO + CO₂  高温，但吸热反应

结论：加热≠吸热，点燃≠放热
     条件只影响反应速率，不改变反应热效应
```

### 例2：溶解过程的吸放热
```
放热溶解：NaOH固体溶于水、浓H₂SO₄稀释
吸热溶解：NH₄NO₃固体溶于水（温度明显下降）
微热溶解：NaCl固体溶于水（温度变化不明显）

注意：溶解是物理变化，不是化学反应
     但伴随的能量变化与"水合热"和"晶格能"有关
```

### 例3：键能计算
```
H₂ + Cl₂ → 2HCl
断键吸热：H-H(436) + Cl-Cl(243) = 679 kJ/mol
成键放热：2×H-Cl(431) = 862 kJ/mol
净：862 - 679 = 183 kJ/mol > 0 → 放热反应
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 吸放热反应常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 用反应条件判断吸放热                                  │
│     错误：需要加热就是吸热反应                           │
│     正确：C燃烧需要点燃，但是放热反应                    │
│     反例：Ba(OH)₂·8H₂O+NH₄Cl 常温进行，但是吸热        │
├─────────────────────────────────────────────────────────┤
│ E2. 混淆反应热与活化能                                    │
│     活化能：反应开始所需最低能量（翻越能垒）              │
│     反应热：反应物与生成物的能量差（始终态差）            │
│     加热提供的是活化能，不是反应热                        │
├─────────────────────────────────────────────────────────┤
│ E3. 认为吸热反应不能自发进行                              │
│     错误：吸热反应一定需要持续加热                       │
│     正确：NH₄Cl与Ba(OH)₂在常温下就能自发吸热反应        │
│     自发性由ΔG=ΔH-TΔS决定，不只是ΔH                    │
├─────────────────────────────────────────────────────────┤
│ E4. 燃烧反应方程式中ΔH符号写反                            │
│     放热反应：ΔH < 0（负值）                            │
│     吸热反应：ΔH > 0（正值）                            │
│     常见错误：燃烧放热写成ΔH=+Q                          │
├─────────────────────────────────────────────────────────┤
│ E5. 混淆溶解热与反应热                                    │
│     NaOH溶解放热 ≠ NaOH与酸反应放热                      │
│     前者是物理变化（水合过程），后者是化学变化             │
└─────────────────────────────────────────────────────────┘
```
