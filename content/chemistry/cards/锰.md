# 锰（Mn）元素卡片

> **元素档案**：第四周期，VIIB族，原子序数25，银白色金属，密度7.44g/cm³，变价元素（+2/+4/+7），与Fe同周期

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第4周期
- **族**：VIIB族（锰副族）
- **原子序数**：25
- **核外电子排布**：(+25) 2)8)13)2

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Tc（锝） | 金属性Mn > Tc（人工放射性元素） |
| 下 | Re（铼） | 金属性Mn < Re（但Re是稀贵金属） |
| 左 | Cr（铬） | 金属性Mn < Cr（Cr更稳定、熔点更高） |
| 右 | Fe（铁） | 金属性Mn > Fe（Mn更活泼） |

### 推断题身份证 ⭐
- ⭐ "变价元素，常见+2/+4/+7价" → Mn
- ⭐ "紫红色溶液" → MnO₄⁻（高锰酸根，紫色）
- ⭐ "黑色粉末" → MnO₂（二氧化锰）
- ⭐ "与Fe形成合金" → 锰铁合金（炼钢脱氧剂、脱硫剂）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+25
- 电子层数：4层
- 最外层电子数：2e⁻
- **特殊**：d轨道有5e⁻（半满），可以失去不同数目电子

### 【第2层】电子行为
- **可变价**：可以失去2、4、7个电子形成不同价态
- Mn²⁺：失去最外层2e⁻（d⁵，半满稳定）
- Mn⁴⁺：失去外层2e⁻+3d电子（较少见）
- Mn⁷⁺：失去所有外层电子（形成MnO₄⁻，类似卤素）

### 【第3层】化学键型
- **与氧**：形成MnO、MnO₂、Mn₂O₇（不同价态）
- **与酸**：生成Mn²⁺盐（+2价最稳定）
- **与强氧化剂**：生成MnO₄⁻（+7价，高锰酸根）

### 【第4层】晶体类型
- **Mn单质**：金属晶体（多种同素异形体）
- **MnO₂**：原子晶体（黑色粉末，VIB族特征）
- **KMnO₄**：离子晶体（紫红色）

> **结构解释性质**：d轨道半满→+2价稳定（类似Cr³⁺）；+7价Mn形成类似卤素的含氧酸根

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色（纯锰） | 金属晶体 |
| 密度 | 7.44g/cm³（重金属） | 与Fe相近 |
| 熔点 | 1246℃ | 较高（过渡金属） |
| 脆性 | 硬而脆 | 金属间化合物性质 |

### 化学性质

#### 还原性（+2价Mn）🔴
| 反应 | 方程式 | 条件/现象 |
|------|--------|-----------|
| 与酸 | Mn + 2HCl = MnCl₂ + H₂↑ | 缓慢（表面氧化膜） |
| 与非氧化性酸 | Mn + H₂SO₄ = MnSO₄ + H₂↑ | 缓慢 |
| 与氧化性酸 | 3Mn + 8HNO₃ = 3Mn(NO₃)₂ + 2NO↑ + 4H₂O | 需加热 |

🔴 **反例/边界**：Mn在空气中表面易生成氧化膜，与酸反应比Fe、Zn缓慢

#### 氧化物两性/酸性 🔴
| 氧化物 | 性质 | 反应 |
|--------|------|------|
| MnO | 碱性 | MnO + 2HCl = MnCl₂ + H₂O |
| MnO₂ | 两性偏酸 | MnO₂ + 4HCl = MnCl₂ + Cl₂↑ + 2H₂O |
| Mn₂O₇ | 酸性 | 遇水生成HMnO₄（高锰酸） |

🔴 **反例/边界**：MnO₂是两性偏酸性氧化物，可与浓盐酸反应生成Cl₂

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Mn / Mn²⁺ / MnO₂】

    Mn（银白色金属）
     │
     ├──[氧化]──→ MnO（绿色）
     │    2Mn + O₂ = 2MnO
     │    碱性氧化物
     │
     ├──[弱氧化]──→ MnO₂（黑色）
     │    Mn + O₂ =△= MnO₂
     │    🔵 干电池正极材料
     │
     └──[强氧化]──→ MnO₄⁻（紫色）
          2Mn + O₂ =高压= Mn₂O₇
          🔵 高锰酸钾（KMnO₄）氧化剂

                              【Mn²⁺的转化】

    Mn²⁺ + 2OH⁻ → Mn(OH)₂↓（白色）
     │
    Mn(OH)₂ + O₂ + H₂O → MnO(OH)₂（棕色）
     │  空气中被氧化
     │
    MnO₂ + 4HCl = MnCl₂ + Cl₂↑ + 2H₂O
     │  🔵 实验室制氯气

                              【KMnO₄的转化】

    2KMnO₄ =△= K₂MnO₄ + MnO₂ + O₂↑
     │  受热分解
     │
    2KMnO₄ + 3H₂S + H₂O = 2MnO₂↓ + 2KOH + 3S↓
     │  🟢 紫红色退去，棕色沉淀
     │  🔵 检验还原性物质

                              【重要化合物】

    🔵 软锰矿（MnO₂）：干电池正极、催化荆
       MnO₂ + 4HCl = MnCl₂ + Cl₂↑ + 2H₂O（实验室制Cl₂）

    🔵 高锰酸钾（KMnO₄）：强氧化剂、消毒剂
       2KMnO₄ =△= K₂MnO₄ + MnO₂ + O₂↑
       🟢 紫红色溶液褪色（检验氧化性）

    🔵 锰酸钾（K₂MnO₄）：绿色（MnO₄²⁻）
       3MnO₂ + 6KOH + KClO₃ = 3K₂MnO₄ + KCl + 3H₂O

    🔵 锰铁：炼钢脱氧剂（Fe-Mn合金）
       Mn + S = MnS，Mn + O = MnO（脱硫脱氧）

    🔵 锰铜合金：电阻材料
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Mn只有+2价 | Mn有+2、+4、+7等多种价态；MnO₄⁻中Mn为+7价（高锰酸根） |
| **E2** | MnO₂是碱性氧化物 | MnO₂是**两性偏酸性**氧化物，可与浓盐酸反应生成Cl₂ |
| **E3** | KMnO₄与H₂S反应生成MnS | KMnO₄氧化H₂S→S↓，本身被还原为MnO₂棕色沉淀，不是MnS |
| **E4** | Mn与酸反应剧烈 | Mn表面有氧化膜，与酸反应**比Fe、Zn缓慢**（需除去氧化膜） |
| **E5** | MnO₄⁻是绿色的 | MnO₄⁻（高锰酸根）呈**紫红色**，MnO₄²⁻（锰酸根）才是绿色 |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Fe** | 同周期（4周期VIIIB） | Mn > Fe金属性；Mn-Fe合金是重要钢材料 |
| **Cr** | 同周期左邻 | Cr熔点更高、更稳定；两者都有变价 |
| **S** | 反应伴侣 | Mn + S =△= MnS（Mn²⁺硫化物） |
| **Cl** | 相似性 | MnO₄⁻（+7）与ClO₄⁻有相似氧化性 |
| **K** | 化合物 | KMnO₄（高锰酸钾）是最重要锰化合物 |
| **O** | 反应伴侣 | 2Mn + O₂ = 2MnO；Mn + O₂ =△= MnO₂ |
| **C** | 合金 | Fe-Mn-C三元系（珠光体钢） |
