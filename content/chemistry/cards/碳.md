## 二、四层递进结构：原子→电子→化学键→晶体

### 第一层：原子结构
```
6C
├── 核电荷数：6
├── 电子层：K(2) L(4)
├── 价电子排布：2s² 2p²
└── sp杂化轨道类型：sp/sp²/sp³
```

### 第二层：电子层与成键
```
价层电子对互斥：4对（正四面体）
├── 成键电子对：2-4个
├── 孤电子对：0-2个
└── 杂化类型判断：
    ├── sp³：4个σ键（如CH₄）
    ├── sp²：3个σ键+1个π键（如C₂H₄）
    └── sp：2个σ键+2个π键（如C₂H₂）
```

### 第三层：化学键类型
| 键型 | 存在形式 | 键能特征 |
|------|----------|----------|
| C-C单键 | 金刚石、烷烃 | 键能348 kJ/mol |
| C=C双键 | 烯烃 | 键能614 kJ/mol |
| C≡C三键 | 炔烃 | 键能839 kJ/mol |
| C=O双键 | 醛酮、CO₂ | 键能745 kJ/mol |

## 三、三合一转化网 🔵🟢

### 碳的转化关系图
```
                    点燃
            C + O₂ ────→ CO₂
              ↑           │
              │       高温
         炼焦/干馏       ↓
              │      C + 2CuO → 2Cu + CO₂↑
         ┌────┴────┐      │
      CO₂◄────C     │      │ 高温
         │         │      ↓
      工业           │   CO + CuO → Cu + CO₂
    🔵煅烧石灰石     │      ↑       (置换反应)
      CaCO₃→CO₂    │      │
                    │   🔵水煤气
         高温       │   C + H₂O ⇌ CO + H₂
         ↓          │      ↑
      CO₂ + C ⇌ 2CO │      │
    (可逆反应🔴)     │   🔵碳的化学性质：
                    │   - 还原性（冶金）
         高温       │   - 可燃性（燃料）
         ↓          │   - 吸附性（活性炭）
      CO₂ + NaOH→   │      ↓
       Na₂CO₃        │   有机物转化
                    │   C → CH₄ → C₂H₂ → ...
```

### 核心反应方程式
| 反应 | 条件 | 方程式 | 现象 |
|------|------|--------|------|
| 碳的燃烧 | 点燃 | C + O₂ → CO₂ | 发光放热 |
| 碳还原氧化铜 | 高温 | C + 2CuO → 2Cu + CO₂↑ | 黑色变红色 |
| 碳还原CO₂ | 高温 | CO₂ + C ⇌ 2CO | — |
| 水煤气反应 | 🔵高温 | C + H₂O ⇌ CO + H₂ | — |
| 碳与浓硫酸 | 加热 | C + 2H₂SO₄ → CO₂↑ + 2SO₂↑ + 2H₂O | 刺激性气体 |
| 碳与浓硝酸 | 加热 | C + 4HNO₃ → CO₂↑ + 4NO₂↑ + 2H₂O | 红棕色气体 |

### 🟢特征检验
```
碳的检验方法：
├─ 燃烧检验：产生CO₂ → 使澄清石灰水变浑浊
├─ 活性炭吸附：吸附有色分子（如品红溶液褪色）
└─ 定性分析：高温下还原金属氧化物生成CO₂
```

## 四、高频陷阱 E1-E5

### 🔴 E1：金刚石和石墨的误区
> **陷阱**：认为金刚石和石墨是同一种物质
>
> **正确**：二者是碳的同素异形体，物理性质差异显著
> - 金刚石：不导电、硬度最大
> - 石墨：导电、质软

### 🔴 E2：CO₂与澄清石灰水的反应
> **陷阱**：CO₂使澄清石灰水变浑浊后继续通入会变澄清
>
> **正确**：CaCO₃ + CO₂ + H₂O → Ca(HCO₃)₂（可溶）
> - 继续通入CO₂：先浑浊后变澄清
> - 加热：Ca(HCO₃)₂ → CaCO₃↓ + CO₂↑ + H₂O（重新浑浊）

### 🔴 E3：碳的氧化态判断
> **陷阱**：认为碳总是显+4价
>
> **正确**：碳的氧化态范围-4到+4
> - CO中碳为+2价
> - CH₄中碳为-4价
> - 有机物中碳的氧化态需根据化合物类型判断

### 🔴 E4：碳的同素异形体混淆
> **陷阱**：认为碳的同素异形体只有金刚石和石墨
>
> **正确**：还有C₆₀、C₇₀、富勒烯、碳纳米管、石墨烯等
> - C₆₀：分子晶体，60个碳原子
> - 石墨烯：单层石墨，sp²杂化，导电性优异

### 🔴 E5：碳热还原法的条件
> **陷阱**：认为任何碳都能还原金属氧化物
>
> **正确**：碳还原金属氧化物的条件
> - 需高温条件
> - 适用于不如碳活泼的金属（如Cu、Zn、Fe等）
> - 活泼金属（Na、Mg、Al等）不能用电解法以外的碳热还原

## 五、跨元素关联

### 垂直关系（同族）
```
IVA族碳族元素：
C（6）→ Si（14）→ Ge（32）→ Sn（50）→ Pb（82）

递变规律：
├─ 原子半径：增大
├─ 非金属性：减弱，金属性：增强
├─ 碳：非金属，Si：准金属，Ge/Sn/Pb：金属
└─ 最高价氧化物：
   - CO₂（酸性氧化物）
   - SiO₂（酸性氧化物）
   - GeO₂（两性）
   - SnO₂/PbO₂（两性偏碱性）
```

### 水平关系（同周期）
```
第二周期元素：
Li → Be → B → **C** → N → O → F → Ne

碳的特殊性：
├─ 能形成sp/sp²/sp³多种杂化
├─ 能形成碳碳键（长链、环状）
├─ 是有机物的基础
└─ 与氧形成多重键（CO、CO₂）
```

### 对角线关系
```
与氮的相似性（第二周期）：
├─ 都能形成p-p π键（如N₂、CO₂）
├─ 都能形成三键（乙炔C₂H₂、氮气N₂）
└─ 氢化物都能形成氢键（CH₄无氢键，但CH₃OH有）
```
