# 铝（Al）元素卡片

> **元素档案**：第三周期，IIIA族，原子序数13，银白色金属，密度2.70g/cm³，两性金属，既能与非氧化性酸反应又能与强碱反应

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：IIIA族
- **原子序数**：13
- **核外电子排布**：(+13) 2)8)3

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | B（硼） | 金属性Al > B（Al是金属，B是准金属） |
| 下 | Ga（镓） | 金属性Al < Ga |
| 左 | Mg（镁） | 金属性Mg > Al（Mg更活泼） |
| 右 | Si（硅） | Si是非金属，Al是金属 |

### 推断题身份证 ⭐
- ⭐ "两性金属，既能溶于酸又能溶于强碱" → Al
- ⭐ "溶于NaOH溶液生成无色气体和白色沉淀" → 2Al + 2NaOH + 2H₂O = 2NaAlO₂ + 3H₂↑
- ⭐ "与稀盐酸反应生成无色气体" → 2Al + 6HCl = 2AlCl₃ + 3H₂↑
- ⭐ "表面致密氧化膜，耐腐蚀" → Al（钝化）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+13
- 电子层数：3层
- 最外层电子数：3e⁻（可失可赢）

### 【第2层】电子行为
- **可失**：失去最外层3e⁻ → Al³⁺（10电子稳定结构）
- **可赢**：有时可形成Al³⁺的配位化合物
- 两性：既可失电子（显还原性）又可在特定条件下得电子（显氧化性）

### 【第3层】化学键型
- **与氧**：形成Al₂O₃（离子晶体，氧化铝）
- **与酸**：形成Al³⁺盐（AlCl₃、Al₂(SO₄)₃）
- **与碱**：形成AlO₂⁻（偏铝酸盐，如NaAlO₂）

### 【第4层】晶体类型
- **Al单质**：金属晶体
- **Al₂O₃**：离子晶体（熔点2054℃，高）
- 解释：Al₂O₃晶格极稳定→耐高温、耐腐蚀；两性Al₂O₃既能与酸反应又能与碱反应

> **结构解释性质**：Al₂O₃氧化膜致密→阻止内部Al继续反应→钝化现象

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色，有金属光泽 | 金属晶体 |
| 密度 | 2.70g/cm³ | 轻金属（飞机材料） |
| 熔点 | 660.3℃ | 中等 |
| 延展性 | 好（可压成铝箔、拉成铝丝） | 金属键 |

### 化学性质

#### 与酸反应（显还原性）🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与稀盐酸 | 2Al + 6HCl = 2AlCl₃ + 3H₂↑ | 气泡，产生Al³⁺ |
| 与稀硫酸 | 2Al + 3H₂SO₄ = Al₂(SO₄)₃ + 3H₂↑ | 气泡 |

🔴 **反例/边界**：冷浓H₂SO₄/HNO₃中Al**钝化**（表面氧化膜阻止反应），加热才反应

#### 与强碱反应（两性）🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与NaOH | 2Al + 2NaOH + 6H₂O = 2NaAl(OH)₄ + 3H₂↑ | 偏铝酸钠 |
| 简化 | 2Al + 2NaOH + 2H₂O = 2NaAlO₂ + 3H₂↑ | 实际上NaAlO₂是脱水产物 |

🔴 **反例/边界**：铝与碱反应需水参与，水是氧化剂，Al还原H₂O中的H⁺

#### 与氧反应 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂（点燃） | 4Al + 3O₂ = 2Al₂O₃ | 放热，发出耀眼白光 |
| 钝化 | Al + O₂ → Al₂O₃（常温） | 表面氧化膜，阻止进一步氧化 |

🔴 **反例/边界**：Al在空气中表面生成致密Al₂O₃薄膜，保护内部不被腐蚀

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Al ↔ Al³⁺ ↔ AlO₂⁻】

    Al（银白色金属）
     │
     ├──[与酸]──→ Al³⁺（如AlCl₃）
     │    2Al + 6HCl = 2AlCl₃ + 3H₂↑
     │
     ├──[与强碱]──→ AlO₂⁻（如NaAlO₂）
     │    2Al + 2NaOH + 6H₂O = 2NaAl(OH)₄
     │    = 2NaAlO₂ + 3H₂↑ + 4H₂O
     │
     └──[与O₂]──→ Al₂O₃（氧化膜）
         4Al + 3O₂ = 2Al₂O₃

                              【Al³⁺的转化】

    Al³⁺（无色溶液）
     │
     ├──[加NaOH少]──→ Al(OH)₃↓（白色凝胶）
     │    Al³⁺ + 3OH⁻ = Al(OH)₃↓
     │
     ├──[加NaOH过]──→ [Al(OH)₄]⁻（可溶）
     │    Al(OH)₃ + OH⁻ = [Al(OH)₄]⁻
     │    🔵 两性：Al(OH)₃既能跟酸反应又能跟碱反应
     │
     └──[煅烧]──→ Al₂O₃（白色固体）
         2Al(OH)₃ =△= Al₂O₃ + 3H₂O

                              【AlO₂⁻的转化】

    AlO₂⁻（偏铝酸根）
     │
     ├──[加酸少]──→ Al(OH)₃↓
     │    AlO₂⁻ + H⁺ + H₂O = Al(OH)₃↓
     │
     └──[加酸过]──→ Al³⁺
         AlO₂⁻ + 4H⁺ = Al³⁺ + 2H₂O

                              【重要工业应用】

    🔵 铝合金：
       硬铝（Al-Cu-Mg-Mn）
       航空航天、汽车轻量化

    🔵 铝热反应：
       2Al + Fe₂O₃ =点燃= 2Fe + Al₂O₃
       焊接铁轨、冶炼金属

    🔵 电解Al₂O₃（冰晶石助熔）：
       2Al₂O₃ =电解= 4Al + 3O₂↑
       工业制铝
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Al与碱反应是Al被碱氧化 | Al还原的是**水中的H⁺**，碱提供OH⁻促进Al与H₂O反应：2Al + 2NaOH + 6H₂O = 2NaAl(OH)₄ + 3H₂↑ |
| **E2** | Al₂O₃只是酸性氧化物 | Al₂O₃是**两性氧化物**，既能与酸反应（生成Al³⁺）又能与碱反应（生成AlO₂⁻）|
| **E3** | 铝在空气中会完全氧化 | 铝表面生成致密Al₂O₃氧化膜**阻止**进一步氧化，因此铝耐腐蚀 |
| **E4** | Al与浓H₂SO₄反应生成SO₂ | 冷浓H₂SO₄中Al**钝化**，不反应；加热反应可能生成SO₂，但Al钝化膜会溶解 |
| **E5** | Al(OH)₃加热分解直接得到Al | Al(OH)₃煅烧得到Al₂O₃，不是Al；Al需电解Al₂O₃得到 |

## 维度6：链（Link）—— 跨元素关联网

| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Mg** | 同周期左邻 | 金属性Mg > Al，但Mg无两性 |
| **Si** | 同周期右邻 | Si是非金属；Al是金属，有两性 |
| **Fe** | 联合出题 | 铝热反应：2Al + Fe₂O₃ =点燃= 2Fe + Al₂O₃ |
| **Na** | 联合出题 | NaAlO₂（偏铝酸钠）是Al与NaOH反应的产物 |
| **Cu** | 合金应用 | 铜铝合金（青铜是Cu-Sn，铝青铜是Cu-Al） |
| **Ga** | 同族下 | Ga熔点很低（29.8℃），比Al特殊 |
