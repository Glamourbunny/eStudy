## 三、三合一转化网 🔵🟢

### 铜的转化关系图
```
                    氧气/加热
           Cu ──────────→ CuO（黑色）
              │              ↑
              │         稀酸溶解
              ↓              │
         Cu₂O（红色）  ←───────┘
              │              │
              │ H⁺/氧化剂    │ H⁺/稀酸
              ↓              ↓
         Cu⁺ ⇌ Cu²⁺      Cu → Cu²⁺ + 2e⁻
           歧化反应        (氧化)
              │              │
              │         氨水滴加
              │              ↓
              │    [Cu(NH₃)₄]²⁺（深蓝色）
              │              │
              └──────┬────────┘
                     │ NaOH
                     ↓
              Cu(OH)₂↓（蓝色）
                     │ 加热
                     ↓
              CuO（黑色）
```

### 核心反应方程式
| 反应 | 条件 | 方程式 | 现象 |
|------|------|--------|------|
| Cu与O₂ | 🔵加热 | 2Cu + O₂ → 2CuO | 变黑 |
| Cu与浓硫酸 | 加热 | Cu + 2H₂SO₄ → CuSO₄ + SO₂↑ + 2H₂O | 刺激性气体 |
| Cu与稀硝酸 | 加热 | 3Cu + 8HNO₃ → 3Cu(NO₃)₂ + 2NO↑ + 4H₂O | 无色气体变红棕 |
| Cu与浓硝酸 | 常温 | Cu + 4HNO₃ → Cu(NO₃)₂ + 2NO₂↑ + 2H₂O | 红棕色气体 |
| Cu₂O与酸 | — | Cu₂O + H₂SO₄ → Cu + CuSO₄ + H₂O | 歧化反应 |
| Cu(OH)₂与氨水 | — | Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄]²⁺ + 2OH⁻ | 深蓝色溶液 |

### 🟢特征检验
```
铜离子的检验方法：
├─ 碱液检验：加NaOH生成蓝色Cu(OH)₂↓，加热变黑（CuO）
├─ 氨水检验：滴加氨水先蓝色沉淀，后溶解形成深蓝色[Cu(NH₃)₄]²⁺
├─ 焰色反应：绿色（不是铜离子的特征）
└─ 碘化物检验：Cu²⁺ + 4I⁻ → [CuI₄]²⁻（黄色/绿色）
```

## 四、高频陷阱 E1-E5

### 🔴 E1：Cu与稀硫酸不反应
> **陷阱**：认为铜可以与稀硫酸反应放出氢气
>
> **正确**：铜的金属活动性顺序
> - Cu在金属活动性顺序中位于H之后
> - Cu + H₂SO₄（稀）→ 不反应
> - Cu + H₂SO₄（浓，加热）→ 反应（体现酸性+强氧化性）
> - Cu只能溶于氧化性酸（浓硫酸、硝酸）

### 🔴 E2：Cu⁺的歧化反应条件
> **陷阱**：认为Cu⁺在任何条件下都稳定
>
> **正确**：Cu⁺的歧化反应
> - 热力学不稳定：2Cu⁺ → Cu²⁺ + Cu（ΔH < 0）
> - 在水溶液中：易发生歧化
> - 稳定条件：在难溶性盐（Cu₂O、CuCl）中稳定
> - 转化为稳定：在配位化合物中Cu⁺可以稳定存在

### 🔴 E3：铜绿成分的错误认识
> **陷阱**：认为铜绿（铜锈）是氧化铜
>
> **正确**：铜绿的化学成分
> - 主要成分：Cu₂(OH)₂CO₃（碱式碳酸铜）
> - 形成条件：铜与潮湿空气中的CO₂、H₂O反应
> - 反应：2Cu + O₂ + CO₂ + H₂O → Cu₂(OH)₂CO₃
> - 不是简单的氧化反应

### 🔴 E4：Cu²⁺与氨水反应
> **陷阱**：认为Cu(OH)₂可溶于过量氨水
>
> **正确**：铜氨络合的过程
> - 少量氨水：Cu²⁺ + 2NH₃ + 2H₂O → Cu(OH)₂↓ + 2NH₄⁺
> - 过量氨水：Cu(OH)₂ + 4NH₃ → [Cu(NH₃)₄]²⁺ + 2OH⁻
> - 现象：先蓝色沉淀，后溶解为深蓝色溶液
> - 络离子稳定常数：很大（不易解离）

### 🔴 E5：铜的导电性误区
> **陷阱**：认为铜导线纯度要求不高
>
> **正确**：导电用铜的要求
> - 电解铜：纯度99.95%以上
> - 无氧铜：纯度99.99%以上（用于真空电子器件）
> - 杂质影响：P、As、Sb、Bi等降低导电性
> - 实际应用：电线电缆用铜需要高导电性+良好延展性

## 五、跨元素关联

### 垂直关系（同族）
```
IB族铜族元素：
Cu（29）→ Ag（47）→ Au（79）

递变规律：
├─ 金属性：增强（但都很不活泼）
├─ 还原性：Cu < Ag < Au
├─ 导电性：Cu > Ag > Au（Ag最好但贵）
├─ 与酸反应：都难溶于非氧化性酸
└─ 氯化物的稳定性：
   - CuCl₂（稳定）vs AgCl（难溶）vs AuCl₃（不稳定）
```

### 水平关系（同周期）
```
第四周期过渡金属（Cu与Fe的关系）：
Sc → Ti → V → Cr → Mn → Fe → Co → Ni → **Cu** → Zn

铜的特殊性：
├─ d轨道全满（3d¹⁰），最不活泼的过渡金属
├─ 只有Cu是紫红色
├─ 与Fe、Co、Ni同周期，性质差异大
└─ 位于金属活动性顺序H之后
```

```
Cu-Ag-Au的共同特征：
├─ 位于金属活动性顺序H之后

├─ 能溶于硝酸或浓硫酸
├─ 易形成配位化合物
├─ 主要矿物：孔雀石Cu₂(OH)₂CO₃、辉铜矿Cu₂S

└─ 用途：货币、装饰、电线（Cu最大量）
```
