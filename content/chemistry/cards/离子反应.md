# 离子反应（Ionic Reaction）

> **反应本质**：有离子参加或生成的反应，本质是离子浓度的改变。包括离子交换反应和离子参加的反应。

---

## 一、反应本质与判断标志

```
判断标志：
✓ 电解质在溶液中电离出离子
✓ 反应前后离子种类或浓度发生变化
✓ 有沉淀、气体或弱电解质生成
```

### 核心通式
```
离子反应 = 离子交换反应 + 离子参加的非氧化还原反应

发生条件（复分解型）：
① 生成沉淀  ② 生成气体  ③ 生成弱电解质（水/弱酸/弱碱）
```

---

## 二、无机代表反应

### 酸碱中和（离子反应本质）
```
化学方程式：HCl + NaOH → NaCl + H₂O
离子方程式：H⁺ + OH⁻ → H₂O

本质：H⁺与OH⁻结合生成弱电解质H₂O
```

### 沉淀反应
```
化学方程式：AgNO₃ + NaCl → AgCl↓ + NaNO₃
离子方程式：Ag⁺ + Cl⁻ → AgCl↓

化学方程式：BaCl₂ + Na₂SO₄ → BaSO₄↓ + 2NaCl
离子方程式：Ba²⁺ + SO₄²⁻ → BaSO₄↓
```

### 气体生成反应
```
化学方程式：Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑
离子方程式：CO₃²⁻ + 2H⁺ → H₂O + CO₂↑

化学方程式：NH₄Cl + NaOH → NaCl + NH₃↑ + H₂O
离子方程式：NH₄⁺ + OH⁻ —△→ NH₃↑ + H₂O
```

---

## 三、有机代表反应

### 有机酸的中和
```
CH₃COOH + NaOH → CH₃COONa + H₂O

离子方程式（注意弱电解质写分子式）：
CH₃COOH + OH⁻ → CH₃COO⁻ + H₂O
（CH₃COOH是弱酸，不能拆成离子）
```

### 酚的弱电离
```
C₆H₅OH ⇌ C₆H₅O⁻ + H⁺    （微弱电离）

酚钠的生成：
C₆H₅OH + NaOH → C₆H₅ONa + H₂O
（酚有弱酸性，可与强碱反应）
```

---

## 四、规律与原理

### 离子方程式书写规则
```
写：写出化学方程式
拆：强酸、强碱、可溶性盐拆成离子
删：删去两边相同的离子（ spectator ions ）
查：检查原子守恒和电荷守恒

必须写分子式的：
• 难溶物（沉淀）
• 弱电解质（弱酸、弱碱、水）
• 单质和气体
• 氧化物
```

### 离子共存判断
```
不能大量共存的情况：
① 生成沉淀：Ag⁺与Cl⁻、Ba²⁺与SO₄²⁻、Ca²⁺与CO₃²⁻
② 生成气体：H⁺与CO₃²⁻/HCO₃⁻/S²⁻/SO₃²⁻
③ 生成弱电解质：H⁺与OH⁻、H⁺与CH₃COO⁻
④ 氧化还原：Fe³⁺与I⁻、MnO₄⁻与Fe²⁺
⑤ 水解互促：Al³⁺与CO₃²⁻、Fe³⁺与HCO₃⁻
```

### 常见沉淀记忆
```
白色沉淀：AgCl、BaSO₄、CaCO₃、Mg(OH)₂、Al(OH)₃
蓝色沉淀：Cu(OH)₂
红褐色沉淀：Fe(OH)₃
黑色沉淀：CuS、FeS、Ag₂S
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 离子反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 弱电解质拆成离子                                      │
│     错误：CH₃COOH + OH⁻ → CH₃COO⁻ + H₂O                 │
│     正确：CH₃COOH + OH⁻ → CH₃COO⁻ + H₂O ✓               │
│     （CH₃COOH不能拆，但产物CH₃COO⁻可以写离子）          │
│     常见弱电解质：CH₃COOH、H₂CO₃、NH₃·H₂O、H₂O          │
├─────────────────────────────────────────────────────────┤
│ E2. 微溶物处理不当                                        │
│     Ca(OH)₂：澄清溶液拆成离子，浊液写分子式              │
│     CaSO₄：一般视为沉淀，写分子式                         │
│     Ag₂SO₄：微溶，视题目条件而定                          │
├─────────────────────────────────────────────────────────┤
│ E3. 离子方程式电荷不守恒                                  │
│     错误：Fe + Fe³⁺ → 2Fe²⁺                             │
│     正确：Fe + 2Fe³⁺ → 3Fe²⁺                            │
│     必须检查：原子守恒 + 电荷守恒                         │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视反应顺序                                          │
│     向含H⁺和Al³⁺的溶液中滴加NaOH：                      │
│     先：H⁺ + OH⁻ → H₂O                                  │
│     后：Al³⁺ + 3OH⁻ → Al(OH)₃↓                         │
│     再：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O                   │
├─────────────────────────────────────────────────────────┤
│ E5. 离子共存忽视隐含条件                                  │
│     酸性溶液：不能大量存在OH⁻、CO₃²⁻、S²⁻等             │
│     碱性溶液：不能大量存在H⁺、NH₄⁺、Al³⁺、Fe³⁺等       │
│     无色溶液：不能存在Cu²⁺(蓝)、Fe³⁺(黄)、Fe²⁺(浅绿)   │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        离子反应
       /    |    \
      ↓     ↓     ↓
   复分解  酸碱中  水解反
   反应    和反应   应
    |       |       |
   沉淀    H⁺+OH⁻  盐的离子
   气体              与水电离
   弱电解质
   
离子反应 → 氧化还原反应（部分离子反应伴随电子转移）
离子反应 → 水解反应（盐类水解本质是离子反应）
```

## 核心记忆口诀

> **强酸强碱可溶性盐，拆成离子写方程**
> **沉淀气体弱电解质，分子形式要保留**
> **原子电荷双守恒，删去旁观得简式**
> **共存先看有没沉，再看气体弱电解，氧化还原不能忘**
