# 氯（Cl）元素卡片

> **元素档案**：第三周期，VIIA族，原子序数17，黄绿色有毒气体，密度大于空气，强氧化性

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：VIIA族（卤素）
- **原子序数**：17
- **核外电子排布**：(+17) 2)8)7

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | F（氟） | 非金属性F > Cl，氧化性F₂更强 |
| 下 | Br（溴） | 非金属性Cl > Br，氧化性Cl₂更强 |
| 左 | S（硫） | 非金属性Cl > S，得电子能力Cl更强 |
| 右 | Ar（氩） | 惰性气体，趋于稳定 |

### 推断题身份证 ⭐
- ⭐ "黄绿色有毒气体" → Cl₂
- ⭐ "使淀粉碘化钾试纸变蓝" → Cl₂（或氧化性物质）
- ⭐ "能使湿润的红色布条褪色" → Cl₂（含 HClO）
- ⭐ "与氢气混合见光爆炸" → HCl（Cl₂ + H₂）
- ⭐ "工业制盐酸" → Cl₂与H₂燃烧

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+17
- 电子层数：3层
- 最外层电子数：7e⁻（**易得1个电子**达到8电子稳定结构）

### 【第2层】电子行为
- **易得**：得到1e⁻ → Cl⁻（17+1=18电子，稳定结构）
- **可共享**：共用电子对形成共价键（如HCl、CCl₄）
- **可失**：失去电子需要极高能量（Cl的电负性3.16，仅次于F、O）

### 【第3层】化学键型
- **与金属（Na等）**：形成离子键 → NaCl（离子化合物）
- **与非金属（H、C等）**：形成极性共价键 → HCl、CCl₄
- **自身**：Cl-Cl共价键 → Cl₂（非极性分子）

### 【第4层】晶体类型
- **Cl₂分子**：分子晶体（熔点-101℃，沸点-34℃）
- **NaCl晶体**：离子晶体（熔点801℃，高）
- 解释：Cl₂是双原子分子，分子间范德华力弱→熔沸点低；NaCl由离子构成→离子键强→熔沸点高

> **结构解释性质**：Cl原子半径比F大 → Cl-Cl键能比F-F弱 → Cl₂氧化性比F₂弱（但仍为强氧化剂）

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色状态 | 黄绿色气体 | 分子晶体，双原子分子 |
| 密度 | 3.214g/L（大于空气） | 排空气法收集时向下排 |
| 溶解性 | 可溶于水（1:2）、有机溶剂 | 相似相溶 |
| 气味 | 刺激性气味，剧毒 | 吸入致死量极低 |

### 化学性质

#### 氧化性（被还原）🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与金属（Fe） | 2Fe + 3Cl₂ =点燃= 2FeCl₃ | 生成高价Fe³⁺，棕褐色烟 |
| 与非金属（H₂） | Cl₂ + H₂ =点燃/光照= 2HCl | 苍白色火焰，爆炸 |
| 与Br⁻/I⁻ | Cl₂ + 2NaBr = 2NaCl + Br₂ | 置换出Br₂（氧化性 Cl₂ > Br₂） |
| 与I⁻ | Cl₂ + 2KI = 2KCl + I₂ | 置换出I₂（使淀粉变蓝） |

🔴 **反例/边界**：Cl₂与Fe反应只生成FeCl₃（+3价），不生成FeCl₂（需还原剂）

#### 还原性（被氧化）—— 极少 🔴
- Cl₂有正价（+1、+3、+5、+7），说明Cl在特定条件下可失电子
- 例：Cl₂ + H₂O = HCl + HClO（Cl既失电子又得电子，歧化反应）

#### 歧化反应（Cl₂的特殊性）
| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与H₂O | Cl₂ + H₂O = HCl + HClO | 可逆，常温 |
| 与NaOH（冷） | Cl₂ + 2NaOH = NaCl + NaClO + H₂O | 漂白液（次氯酸盐） |
| 与NaOH（热） | 3Cl₂ + 6NaOH = 5NaCl + NaClO₃ + 3H₂O | 氯酸盐 |

🔴 **反例/边界**：Cl₂的歧化反应需要**碱性条件**，酸性条件下发生归中反应

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Cl₂ / HClO】

    Cl₂（黄绿色）
     │
     ├──[与H₂O]──→ HCl + HClO（可逆）
     │             条件：溶于水（自来水消毒）
     │             现象：溶液呈黄绿色（Cl₂未完全反应）
     │
     ├──[与NaOH（冷）]──→ NaCl + NaClO（漂白液）
     │                    条件：冷碱吸收
     │                    用途：84消毒液
     │
     ├──[与NaOH（热）]──→ NaCl + NaClO₃（氯酸盐）
     │                    条件：热碱
     │
     ├──[电解饱和NaCl]──→ NaOH + Cl₂↑ + H₂↑（氯碱工业）
     │                   条件：电解
     │                   🔵 用途：制漂白粉、氯气、盐酸
     │
     └──[与Fe/Cu等金属]──→ FeCl₃ / CuCl₂（高价氯化物）
                          条件：点燃/灼烧

                              【HCl的转化】

    HCl ──[溶于水]──→ 盐酸（强酸）
     │
     ├──[与AgNO₃]──→ AgCl↓（白色，不溶于稀硝酸）
     │              🟢 检验Cl⁻
     │
     ├──[与MnO₂]──→ Cl₂↑（实验室制Cl₂方法）
     │            MnO₂ + 4HCl(浓) =△= MnCl₂ + Cl₂↑ + 2H₂O
     │
     └──[与Fe₂O₃]──→ FeCl₃ + H₂O（酸碱中和）

                              【漂白粉的制备】

    Ca(OH)₂ + Cl₂ = CaCl₂ + Ca(ClO)₂ + H₂O
     │
     └── 有效成分：Ca(ClO)₂（次氯酸钙）
          │
          └──[与CO₂/H₂O]──→ HClO（漂白发原理）
                             Ca(ClO)₂ + CO₂ + H₂O = CaCO₃↓ + 2HClO
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Cl₂的漂白性直接来自Cl₂ | Cl₂本身无漂白性，是Cl₂与H₂O反应生成的**HClO**有漂白性 |
| **E2** | HClO是强酸 | HClO是**弱酸**，酸性弱于碳酸；但氧化性强，能漂白 |
| **E3** | 实验室用稀H₂SO₄与大理石制CO₂会有HCl杂质 | 如用大理石（含CaCl₂杂质），会引入Cl⁻，但这不是主要干扰 |
| **E4** | Cl₂和HCl的收集方法相同 | Cl₂（密度>空气）用**向上排空气法**；HCl（极易溶于水）用**排饱和食盐水法** |
| **E5** | 漂白粉失效是 HClO 分解 | 漂白粉失效是Ca(ClO)₂与空气中CO₂生成HClO，HClO见光分解：2HClO =光= 2HCl + O₂↑ |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **F** | 同族 | F₂氧化性更强，非金属性F最强；Cl₂能氧化Br⁻/I⁻，不能氧化F⁻ |
| **Br** | 同族 | Cl₂ > Br₂氧化性，可发生置换：Cl₂ + 2Br⁻ = 2Cl⁻ + Br₂ |
| **I** | 同族 | Cl₂ > I₂氧化性；Cl₂ + 2I⁻ = 2Cl⁻ + I₂（使淀粉变蓝） |
| **Fe** | 反应伴侣 | 2Fe + 3Cl₂ = 2FeCl₃（棕褐色烟）；FeCl₃ + 3OH⁻ = Fe(OH)₃↓ |
| **S** | 联合推断 | S与Cl₂都是强氧化剂，可联合出题（如检验SO₂用Cl₂水） |
| **N** | 工业联合 | 合成氨工业 + 氯碱工业联合（制NH₄Cl等） |

### 联合推断规律
- Cl₂ ──氧化性对比──→ Br₂/I₂（卤素氧化性顺序：F₂ > Cl₂ > Br₂ > I₂）
- Cl₂ ──联合推断──→ Fe（FeCl₃是铁盐检验的中间产物）
- Cl₂ ──工业流程──→ Na（氯碱工业，产NaOH、Cl₂、H₂）
- Cl₂ ──实验探究──→ H（盐酸的制备与检验）
