# 铁（Fe）元素卡片

> **元素档案**：第四周期，VIII族，原子序数26，银白色金属，过渡金属的代表，变价元素（Fe²⁺/Fe³⁺），地壳中含量第四

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第4周期
- **族**：VIII族（过渡金属）
- **原子序数**：26
- **核外电子排布**：(+26) 2)8)14)2

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Fe（无上方同族元素，第3周期无VIII族） | — |
| 下 | Ru（钌） | 金属性更弱 |
| 左 | Mn（锰） | 原子序数更小，性质相近 |
| 右 | Co（钴） | 都是变价金属 |
| 对角线 | 与Co、Ni同族 | 同为铁系元素 |

### 推断题身份证 ⭐
- ⭐ "浅绿色溶液" → Fe²⁺（如FeSO₄溶液）
- ⭐ "黄色溶液" → Fe³⁺（如FeCl₃溶液）
- ⭐ "加入KSCN显血红色" → Fe³⁺（检验Fe³⁺的特征反应）
- ⭐ "加入NaOH生成白色沉淀→灰绿色→红褐色" → Fe²⁺ → Fe(OH)₃
- ⭐ "能被磁铁吸引" → Fe（或Fe₃O₄）
- ⭐ "生成黑色粉末" → Fe₃O₄（Fe在O₂中燃烧）

---

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+26
- 电子层数：4层
- 最外层电子数：2e⁻（可失去）

### 【第2层】电子行为
- **可变价**：最外层2e⁻ + 次外层部分电子可参与反应
- **Fe²⁺**：失去2e⁻ → (+26) 2)8)14)2 → Fe²⁺（亚铁离子）
- **Fe³⁺**：再失去1个电子 → (+26) 2)8)13)2 → Fe³⁺（铁离子）
- Fe³⁺更稳定（半满18电子构型）

### 【第3层】化学键型
- **与强氧化剂**：形成Fe³⁺化合物（如FeCl₃）
- **与弱氧化剂**：形成Fe²⁺化合物（如FeSO₄）
- **与S**：形成FeS（黑色）（Fe + S = FeS）
- **与O**：形成Fe₂O₃（红褐色）或Fe₃O₄（黑色）
- Fe是金属晶体，金属键较强（熔点1538℃）

### 【第4层】晶体类型
- **Fe单质**：金属晶体
- **Fe²O₃/Fe₃O₄**：离子晶体（氧化物）
- 解释：Fe的金属键强 → 熔点高、硬度大；Fe₃O₄有磁性（磁铁矿）

> **结构解释性质**：Fe原子有未完全充满的d轨道 → 可形成配位化合物（如[Fe(SCN)]²⁺血红色）

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色 | 银白色，有金属光泽 | 金属晶体 |
| 磁性 | 能被磁铁吸引 | Fe、Fe₃O₄有磁性 |
| 密度 | 7.86g/cm³ | 重金属 |
| 熔点 | 1538℃（高） | 金属键强 |

### 化学性质

#### Fe的氧化（被氧化）🔴
| 氧化剂 | 产物 | 方程式 | 条件 |
|--------|------|--------|------|
| O₂ | Fe₂O₃ | 4Fe + 3O₂ = 2Fe₂O₃ | 潮湿空气中生锈（红褐色） |
| O₂（点燃） | Fe₃O₄ | 3Fe + 2O₂ =点燃= Fe₃O₄ | 火星四射，黑色 |
| Cl₂ | FeCl₃ | 2Fe + 3Cl₂ = 2FeCl₃ | 棕褐色烟 |
| S | FeS | Fe + S = FeS | 黑色固体 |
| HCl/H₂SO₄ | Fe²⁺ + H₂ | Fe + 2HCl = FeCl₂ + H₂↑ | 有气泡，溶液变绿 |

🔴 **反例/边界**：Fe在Cl₂中燃烧生成FeCl₃，不是FeCl₂（与HCl反应不同）

#### Fe²⁺的氧化（被氧化）🔴
| 氧化剂 | 产物 | 方程式 |
|--------|------|--------|
| Cl₂ | Fe³⁺ | 2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻ |
| O₂/H₂O | Fe³⁺（缓慢） | 4Fe²⁺ + O₂ + 4H⁺ = 4Fe³⁺ + 2H₂O |
| HNO₃ | Fe³⁺ | 3Fe²⁺ + NO₃⁻ + 4H⁺ = 3Fe³⁺ + NO↑ + 2H₂O |

🔴 **反例/边界**：Fe²⁺在空气中易被氧化为Fe³⁺，保存时需加铁钉或稀酸

#### Fe³⁺的还原（被还原）🔴
| 还原剂 | 产物 | 方程式 |
|--------|------|--------|
| Fe | Fe²⁺ | Fe + 2Fe³⁺ = 3Fe²⁺ |
| Cu | Fe²⁺ + Cu²⁺ | Cu + 2Fe³⁺ = Cu²⁺ + 2Fe²⁺ |
| KI | Fe²⁺ + I₂ | 2Fe³⁺ + 2I⁻ = 2Fe²⁺ + I₂（使淀粉变蓝） |

🔴 **反例/边界**：Fe³⁺与I⁻反应是可逆的，Fe³⁺过量时I₂会被重新氧化

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：Fe ↔ Fe²⁺ ↔ Fe³⁺】

                              【Fe的腐蚀与防锈】

    Fe ──[O₂+H₂O]──→ Fe₂O₃·nH₂O（铁锈，红褐色）
                      条件：潮湿空气
                      现象：生锈

    Fe ──[涂油漆/镀锌]──→ 防护（阻止腐蚀）
                      防锈方法：涂保护层、烤蓝、电镀

                              【Fe与酸反应】

    Fe + 2HCl = FeCl₂ + H₂↑
     │
     └── 溶液由无色→浅绿色（Fe²⁺）

    Fe + 4HNO₃(稀) = Fe(NO₃)₃ + NO↑ + 2H₂O
     │                 （过量Fe，或Fe不足生成Fe²⁺）
     └── 铁过量：2Fe + 6HNO₃(浓) = Fe₂O₃ + 6NO₂↑ + 3H₂O（钝化）

                              【Fe²⁺ ↔ Fe³⁺ 转化】

    Fe²⁺（浅绿色）
     │
     ├──[氧化剂：Cl₂/O₂/HNO₃]──→ Fe³⁺（黄色）
     │       2Fe²⁺ + Cl₂ = 2Fe³⁺ + 2Cl⁻
     │
     ├──[还原剂：Fe/Cu/I⁻]──← Fe³⁺（黄色）
     │       Fe + 2Fe³⁺ = 3Fe²⁺
     │
     └──[加碱]──→ Fe(OH)₂↓（白色）──[氧化]──→ Fe(OH)₃↓（红褐色）
                    4Fe(OH)₂ + O₂ + 2H₂O = 4Fe(OH)₃

                              【Fe的工业制备】

    🔵 炼铁（高炉）：
       Fe₂O₃ + 3CO =高温= 2Fe + 3CO₂
       原料：赤铁矿（Fe₂O₃）、焦炭、石灰石
       原理：CO还原Fe₂O₃

    🔵 炼钢（转炉/电炉）：
       降C含量、调Si/Mn/P/S
       2Fe + O₂ = 2FeO（氧化除杂）
       FeO + C = Fe + CO（还原）

                              【Fe³⁺的检验与除去】

    🟢 检验Fe³⁺：
       Fe³⁺ + 3SCN⁻ = Fe(SCN)₃（血红色）
       现象：溶液变红

    🟢 除去Fe³⁺：
       加Fe粉：2Fe³⁺ + Fe = 3Fe²⁺
       加Cu粉：2Fe³⁺ + Cu = 2Fe²⁺ + Cu²⁺
       加碱：Fe³⁺ + 3OH⁻ = Fe(OH)₃↓（红褐色沉淀）
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | Fe与Cl₂反应生成FeCl₂ | Fe与强氧化剂Cl₂反应生成**FeCl₃**（+3价），棕褐色烟；Fe与弱氧化剂HCl生成FeCl₂（+2价）|
| **E2** | FeCl₃溶液蒸干得到FeCl₃固体 | FeCl₃水解：FeCl₃ + 3H₂O = Fe(OH)₃ + 3HCl，蒸干后得到**Fe₂O₃**（红褐色），不是FeCl₃ |
| **E3** | Fe²⁺加NaOH只生成Fe(OH)₂ | 生成白色Fe(OH)₂，但Fe(OH)₂在空气中**迅速变灰绿→红褐**，生成Fe(OH)₃ |
| **E4** | 铁与浓H₂SO₄/HNO₃反应生成Fe²⁺ | 铁在冷浓H₂SO₄/HNO₃中**钝化**（表面氧化），加热才反应；稀HNO₃与铁生成Fe³⁺或Fe²⁺取决于量 |
| **E5** | Fe³⁺与I⁻反应可完全进行 | Fe³⁺ + 2I⁻ ⇌ Fe²⁺ + I₂（可逆），当Fe³⁺过量或I⁻浓度高时，反应方向不同；加入淀粉可检验I₂生成 |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **Cu** | 同周期邻居 | Fe > Cu（还原性），Fe + Cu²⁺ = Fe²⁺ + Cu（湿法炼铜） |
| **S** | 反应伴侣 | Fe + S = FeS（黑色）；Fe₂O₃ + 3CO = 2Fe + 3CO₂ |
| **C** | 工业联合 | 炼铁：Fe₂O₃ + 3CO = 2Fe + 3CO₂（高炉） |
| **Cl** | 反应伴侣 | 2Fe + 3Cl₂ = 2FeCl₃（棕褐色烟） |
| **O** | 反应伴侣 | 4Fe + 3O₂ = 2Fe₂O₃（铁锈）；3Fe + 2O₂ = Fe₃O₄（黑色） |
| **Mn** | 同周期 | 炼钢中调节Mn（脱氧）：Mn + FeO = MnO + Fe |

### 联合推断规律
- Fe ──联合推断──→ 与Cu/S/Cl联合（变价特征是推断题突破口）
- Fe²⁺/Fe³⁺ ──检验题──→ 与KSCN联合（血红色 = Fe³⁺）
- Fe ──工业流程──→ 与C/CO联合（高炉炼铁）
- Fe ──实验探究──→ 与S/O/Cl联合（燃烧产物判断）
