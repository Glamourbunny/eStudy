# 离子键 vs 共价键

> **底线地位**：判断标准不是"金属/非金属"，而是电负性差与电子偏移程度。概念不清会导致化合物类型判断全盘错误。

---

## 一、概念定义对比

| 维度 | 离子键 | 共价键 |
|------|--------|--------|
| **本质** | 阴、阳离子间的静电作用 | 原子间共用电子对 |
| **形成条件** | 电负性差 > 1.7（通常） | 电负性差 < 1.7（通常） |
| **电子行为** | 电子完全转移（得失） | 电子共用（偏移或不偏移） |
| **方向性** | 无方向性 | 有方向性和饱和性 |
| **存在形式** | 离子晶体（无独立分子） | 分子/原子晶体/网络结构 |
| **熔融状态** | 导电（离子自由移动） | 不导电（无自由离子） |

---

## 二、核心区别：电子偏移程度

```
电负性差值判断法：

Δχ > 1.7  ──→  离子键（电子几乎完全转移）
Δχ ≈ 1.7  ──→  过渡型（如AlCl₃偏共价）
Δχ < 1.7  ──→  共价键（电子共用，偏移程度不同）

例：
NaCl: χ(Na)=0.93, χ(Cl)=3.16, Δχ=2.23 → 离子键
HCl: χ(H)=2.20, χ(Cl)=3.16, Δχ=0.96 → 共价键
AlCl₃: χ(Al)=1.61, χ(Cl)=3.16, Δχ=1.55 → 共价键（特例！）
```

---

## 三、判断方法

```
第一步：查电负性差（高中常用简化版）

活泼金属 + 活泼非金属 → 离子键
  例：NaCl、CaO、MgF₂

非金属 + 非金属 → 共价键
  例：HCl、CO₂、H₂O

第二步：注意特例
• AlCl₃、FeCl₃：金属+非金属，但共价键（Δχ较小）
• NH₄Cl：含NH₄⁺，内部N-H是共价键，NH₄⁺与Cl⁻是离子键
• 含氧酸盐（Na₂SO₄）：Na⁺与SO₄²⁻间是离子键，SO₄²⁻内部是共价键
```

## 四、典型例子

### 例1：含金属一定是离子键？
```
错误认知：有金属就是离子化合物
反例：AlCl₃ ──→ 共价化合物（熔点低，易升华）
      FeCl₃ ──→ 共价性较强
      BeCl₂ ──→ 共价化合物

判断依据：看熔沸点和导电性
  离子化合物：高熔点，熔融导电
  共价化合物：低熔点，熔融不导电
```

### 例2：化合物中的"混合键型"
```
NaOH: Na⁺与OH⁻之间是离子键，O-H之间是共价键
Na₂O₂: Na⁺与O₂²⁻之间是离子键，O-O之间是非极性共价键
NH₄NO₃: NH₄⁺与NO₃⁻之间是离子键，内部都是共价键

结论：很多化合物同时含离子键和共价键
```

### 例3：共价键的极性细分
```
非极性共价键：H-H、Cl-Cl、N≡N（同种原子）
极性共价键：H-Cl、C-O、N-H（不同原子，电负性有差异）

键的极性顺序：
H-F > H-O > H-Cl > H-Br > H-I > C-H
（Δχ越大，极性越强）
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 键型判断常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 只看元素类别判断键型                                  │
│     错误：有金属就是离子键，没金属就是共价键              │
│     正确：看电负性差 + 实际性质（熔沸点、导电性）         │
│     反例：AlCl₃是共价化合物                              │
├─────────────────────────────────────────────────────────┤
│ E2. 认为离子化合物中不能有共价键                          │
│     错误：NaOH中只有离子键                                │
│     正确：NaOH中Na⁺与OH⁻是离子键，O-H是共价键           │
│     常见：含氧酸根、OH⁻、NH₄⁺内部都是共价键              │
├─────────────────────────────────────────────────────────┤
│ E3. 混淆"键的极性"与"分子极性"                          │
│     键有极性 ≠ 分子有极性                                 │
│     例：CO₂中C=O是极性键，但CO₂是非极性分子（对称）      │
│     例：H₂O中O-H是极性键，H₂O是极性分子（V形不对称）     │
├─────────────────────────────────────────────────────────┤
│ E4. 认为离子键比共价键"强"                              │
│     不一定！键的强度要具体情况具体分析                    │
│     比较要在同一维度：离子键强度 vs 离子键强度            │
│     或：共价键键能 vs 共价键键能                          │
└─────────────────────────────────────────────────────────┘
```
