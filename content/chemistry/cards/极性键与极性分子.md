# 极性键 vs 非极性键 & 极性分子 vs 非极性分子

> **底线地位**：这是高中化学最容易混淆的"四极"辨析。键的极性看原子是否相同，分子的极性看空间结构是否对称。

---

## 一、概念定义对比

### 1. 键的极性（微观，看原子）

| 类型 | 定义 | 判断标准 | 例子 |
|------|------|----------|------|
| **非极性键** | 电子对不偏移 | 成键原子相同 | H-H、Cl-Cl、O=O、N≡N |
| **极性键** | 电子对偏向一方 | 成键原子不同 | H-Cl、C-O、O-H、C=O |

### 2. 分子的极性（宏观，看结构）

| 类型 | 定义 | 判断标准 | 例子 |
|------|------|----------|------|
| **非极性分子** | 正负电荷中心重合 | 结构对称 | CO₂、CH₄、CCl₄、BF₃ |
| **极性分子** | 正负电荷中心不重合 | 结构不对称 | H₂O、NH₃、HCl、SO₂ |

---

## 二、核心区别：判断层级完全不同

```
键的极性：看"成键原子是否相同"
    ↓
分子的极性：看"空间结构是否对称"（键的极性向量和是否为0）

关键逻辑：
• 非极性键 → 一定形成非极性分子（同种原子双原子分子）
• 极性键 → 可能形成极性分子，也可能形成非极性分子
  （取决于分子结构是否对称抵消）
```

### 经典组合表

| 键型 | 分子结构 | 分子极性 | 例子 |
|------|----------|----------|------|
| 非极性键 | 双原子 | 非极性分子 | H₂、O₂、N₂、Cl₂ |
| 极性键 | 对称结构 | 非极性分子 | CO₂（直线）、CH₄（正四面体）、CCl₄ |
| 极性键 | 不对称结构 | 极性分子 | H₂O（V形）、NH₃（三角锥）、SO₂ |

---

## 三、判断方法

### 第一步：判断键的极性
```
成键原子相同 → 非极性键
成键原子不同 → 极性键（Δχ越大，极性越强）
```

### 第二步：判断分子极性（对于含极性键的分子）
```
看空间结构是否对称：

对称结构（非极性分子）：
  • 直线形（AB₂）：CO₂（O=C=O）
  • 正三角形（AB₃）：BF₃、SO₃（平面）
  • 正四面体（AB₄）：CH₄、CCl₄、SiH₄
  • 正八面体（AB₆）：SF₆

不对称结构（极性分子）：
  • V形（角形）：H₂O、SO₂、H₂S
  • 三角锥形：NH₃、PH₃
  • 四面体（非正四面体）：CH₃Cl
```

## 四、典型例子

### 例1：CO₂ vs H₂O
```
CO₂: O=C=O
  键：C=O是极性键
  结构：直线形，对称
  分子：非极性分子（两个C=O极性抵消）

H₂O:     O
        / \
       H   H
  键：O-H是极性键
  结构：V形（键角约104.5°），不对称
  分子：极性分子（极性不能抵消）
```

### 例2：CH₄ vs CH₃Cl
```
CH₄: 正四面体，4个C-H极性键对称分布 → 非极性分子
CH₃Cl: 四面体，但C-Cl与C-H键极性不同 → 极性分子
```

### 例3：BF₃ vs NH₃
```
BF₃: 平面三角形，3个B-F键120°对称 → 非极性分子
NH₃: 三角锥形，3个N-H键不对称 → 极性分子
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 "四极"辨析常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 认为"极性键一定形成极性分子"                        │
│     错误：CO₂是极性分子（因为有极性键C=O）               │
│     正确：CO₂是非极性分子（直线对称，极性抵消）          │
│     常见误判：CH₄、CCl₄、BF₃                            │
├─────────────────────────────────────────────────────────┤
│ E2. 认为"非极性分子一定只含非极性键"                    │
│     错误：CO₂含极性键但非极性分子                        │
│     正确：非极性分子可以含极性键，只要结构对称           │
├─────────────────────────────────────────────────────────┤
│ E3. 认为"双原子分子都是非极性的"                        │
│     错误：HCl、CO、NO都是极性分子                        │
│     正确：只有同种原子形成的双原子分子才是非极性         │
├─────────────────────────────────────────────────────────┤
│ E4. 用"是否含金属"判断分子极性                          │
│     无关！分子极性与金属无关                              │
│     判断只看：键的极性 + 空间结构对称性                  │
├─────────────────────────────────────────────────────────┤
│ E5. 混淆"极性键强弱"与"分子极性大小"                    │
│     键极性强弱：由电负性差决定                            │
│     分子极性大小：由键极性和结构共同决定                  │
│     例：H-F键极性 > H-Cl，HF分子极性 > HCl分子极性     │
└─────────────────────────────────────────────────────────┘
```

## 六、关联概念网络

```
键的极性          分子的极性
   │                  │
   ↓                  ↓
原子是否相同     结构是否对称
（电负性差）    （极性向量和）
   │                  │
   └──→ 共同决定 ←───┘
         溶解性
      （相似相溶）
```
