# 氮（N）元素卡片

> **元素档案**：第二周期VA族，原子序数7，空气中78%，-3~+5价，生命元素

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第2周期
- **族**：VA族（氮族）
- **原子序数**：7
- **核外电子排布**：(+7) 2)5

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | C（碳） | 原子半径N < C，非金属性N > C |
| 下 | P（磷） | 原子半径N < P，非金属性N > P |
| 左 | C（碳） | 得电子能力N > C |
| 右 | O（氧） | 非金属性O > N，氧化性O₂更强 |

### 推断题身份证 ⭐
- ⭐ "空气中78%" → N₂
- ⭐ "可合成氨气" → N₂ + 3H₂ ⇌ 2NH₃（工业合成氨）
- ⭐ "雷雨发庄稼" → N₂ + O₂ =闪电= 2NO（自然固氮）
- ⭐ "可制硝酸" → NO/NO₂ → HNO₃
- ⭐ "不支持燃烧但能助燃" → N₂（惰性但能参与反应）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+7
- 电子层数：2层
- 最外层电子数：5e⁻（**需得3e⁻或失5e⁻达到稳定**）

### 【第2层】电子行为
- **易得**：得3e⁻ → N³⁻（10电子稳定结构）
- **易失**：失5e⁻ → N⁵⁺（难，需高能量）
- **可共享**：形成三键（N≡N三键键能很大，942 kJ/mol）

### 【第3层】化学键型
- **与金属（Mg等）**：形成离子键 → Mg₃N₂（氮化物）
- **与非金属（H）**：形成共价键 → NH₃（极性分子）
- **自身**：N≡N三键 → N₂（氮气分子，非极性）

### 【第4层】晶体类型
- **N₂分子**：分子晶体（熔点-210℃，沸点-196℃）
- **NH₃分子**：分子晶体（氢键导致沸点较高-33℃）
- **Mg₃N₂**：离子晶体（熔点高，与水反应）
- 解释：N₂三键键能大→解离能高→化学性质稳定；NH₃有氢键→沸点反常高

> **结构解释性质**：N原子半径小→氮气中N≡N三键极难断裂→常温稳定；需高温/催化剂才能反应

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色状态 | 无色无味气体 | 双原子分子 |
| 密度 | 1.251 g/L（略轻于空气） | 收集方法：向下排空气法 |
| 溶解性 | 难溶于水（1:0.02） | 非极性分子，相似相溶 |
| 状态 | 沸点-196℃，液氮可作制冷剂 | 分子晶体 |

### 化学性质

#### 还原性（被氧化）—— 主要性质 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂ | N₂ + O₂ =闪电= 2NO | 自然界雷雨固氮 |
| 与H₂ | N₂ + 3H₂ ⇌ 2NH₃ | 🔵工业合成氨（高温高压催化剂） |
| 与Mg | N₂ + 3Mg =点燃= Mg₃N₂ | 氮化物（Mg₃N₂ + 6H₂O → 2NH₃ + 3Mg(OH)₂） |

🔴 **反例/边界**：N₂与H₂反应**可逆**，需不断分离NH₃提高产率

#### 氧化性（被还原）—— 体现氮的负价
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与金属 | N₂ + 3Mg =点燃= Mg₃N₂ | N从0→-3价，被还原（得电子） |

🔴 **反例/边界**：N₂的氧化性很弱，只有与极活泼金属（如Mg）才能体现

#### 氮的化合物价态变化网
```
        -3 NH₃/NH₄⁺
         ↑
    0  N₂ ←→ NO（+2）
         ↓
    NO₂（+4）→ HNO₃（+5）
```

## 维度4：用（Application）—— 三合一转化网

```
                        【核心枢纽：N₂ / 氨】

    N₂（无色气体）
     │
     ├──[与H₂]──→ NH₃（可逆，高温高压）
     │           条件：500℃、20-50MPa、Fe催化剂
     │           🔵工业合成氨
     │           现象：混合气通过浓硫酸洗气，NH₃被吸收
     │
     ├──[与O₂（闪电）]──→ NO → NO₂ → HNO₃
     │                    条件：闪电/高温
     │                    现象：棕红色气体
     │
     └──[与Mg]──→ Mg₃N₂（氮化物）
                  条件：点燃
                  现象：白光，生成灰绿色固体

                        【NH₃的转化】

    NH₃
     │
     ├──[与H₂O]──→ NH₃·H₂O（氨水，一水合氨）
     │             条件：溶于水（可逆）
     │             现象：刺激性气味，溶液呈碱性
     │
     ├──[与酸]──→ NH₄⁺（铵盐）
     │            例：NH₃ + HCl = NH₄Cl（白烟）
     │            🟢检验：加NaOH加热，产生使湿润红色石蕊试纸变蓝的气体
     │
     ├──[与O₂（催化氧化）]──→ NO → NO₂ → HNO₃
     │                        条件：Pt-Rh催化剂、加热
     │                        🔵工业制硝酸
     │
     └──[与CuO]──→ N₂ + Cu + H₂O（实验室检测氨气）
                   条件：高温
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | N₂化学性质稳定因为氮气是单质 | N₂稳定是因为N≡N三键键能大（942 kJ/mol），不易断裂 |
| **E2** | 氨气溶于水就是氨水，氨水就是NH₃ | 氨水是NH₃·H₂O和NH₃的混合物，主要以NH₃·H₂O（一水合氨）形式存在，是弱碱 |
| **E3** | 氨气检验用湿润蓝色石蕊试纸 | 氨气使**湿润红色**石蕊试纸变蓝（碱性），不能用蓝色试纸 |
| **E4** | 氨催化氧化直接生成了NO₂ | 氨催化氧化：4NH₃ + 5O₂ = 4NO + 6H₂O，NO再被氧化为NO₂ |
| **E5** | 合成氨反应达到平衡后增大压强，NH₃百分含量一定增大 | 增大压强平衡右移，但体积缩小，NH₃百分含量增大；减小压强则反之 |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **H** | 工业联合 | N₂ + 3H₂ ⇌ 2NH₃（合成氨工业） |
| **O** | 化合物链 | N₂ → NO → NO₂ → HNO₃（硝酸工业） |
| **P** | 同族 | N与P同为VA族，氢化物分别为NH₃和PH₃（磷化氢） |
| **S** | 酸雨关联 | NO₂ + SO₂ → HNO₃ + H₂SO₄（酸雨形成） |
| **K** | 化肥关联 | 氨态氮肥：NH₄Cl、(NH₄)₂SO₄；硝态氮肥：KNO₃ |
| **Mg** | 反应伴侣 | 3Mg + N₂ = Mg₃N₂（验证N₂的氧化性） |

### 联合推断规律
- N₂ ──氧化性对比──→ O₂（O₂氧化性更强，雷雨固氮需闪电提供能量）
- N₂ ──工业流程──→ H（合成氨是Haber-Bosch法）
- N₂ ──实验探究──→ P（氨气检验：P₂O₅作干燥剂）
- NH₃ ──酸碱反应──→ N（铵盐检验：加热分解出NH₃）
