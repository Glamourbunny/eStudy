# 还原反应（Reduction）

> **反应本质**：得电子、去氧或加氢的过程。化合价降低，氧化剂被还原。

---

## 一、反应本质与判断标志

```
判断标志（满足其一即可）：
✓ 元素化合价降低
✓ 去氧原子（被夺去氧）
✓ 加氢原子（加氢）
✓ 得电子
```

### 氧化数变化通式
```
氧化剂 + ne⁻ → 还原产物
（化合价降低，被还原）
```

---

## 二、无机代表反应

| 反应 | 方程式 | 还原特征 | 化合价变化 |
|------|--------|----------|-----------|
| **H₂还原CuO** | `H₂ + CuO → Cu + H₂O` | CuO被还原 | Cu: +2→0 |
| **CO还原Fe₂O₃** | `3CO + Fe₂O₃ → 2Fe + 3CO₂` | Fe₂O₃被还原 | Fe: +3→0 |
| **C还原CuO** | `C + 2CuO → 2Cu + CO₂↑` | CuO被还原 | Cu: +2→0 |
| **Al热反应** | `2Al + Fe₂O₃ → 2Fe + Al₂O₃` | Fe₂O₃被还原 | Fe: +3→0 |
| **金属冶炼** | `2NaCl(熔融) —电解→ 2Na + Cl₂↑` | Na⁺被还原 | Na: +1→0 |

---

## 三、有机代表反应

### 催化加氢（烯烃/炔烃/醛酮还原）
```
CH₂=CH₂ + H₂ —Ni/△→ CH₃CH₃          （烯烃→烷烃）
CH≡CH + 2H₂ —Ni/△→ CH₃CH₃          （炔烃→烷烃）
CH₃CHO + H₂ —Ni/△→ CH₃CH₂OH        （醛→伯醇）
CH₃COCH₃ + H₂ —Ni/△→ CH₃CHOHCH₃   （酮→仲醇）
```

### 硝基化合物还原
```
C₆H₅NO₂ + 3Fe + 6HCl → C₆H₅NH₂ + 3FeCl₂ + 2H₂O
（硝基苯 → 苯胺，重要工业反应）
```

### 油脂氢化（硬化）
```
不饱和油脂 + H₂ —Ni/△→ 饱和油脂（人造奶油）
```

## 四、规律与原理

### 常见还原剂强度顺序
```
强还原剂：Al > Mg > Zn > Fe
中强还原剂：C、CO、H₂
弱还原剂：Fe²⁺、SO₂、Sn²⁺、I⁻
有机还原剂：醛、醇、不饱和烃
```

### 金属冶炼方法选择
```
K Ca Na Mg Al ──→ 电解法（太活泼，化合物稳定）
Zn Fe Sn Pb Cu ──→ 热还原法（用C/CO/H₂还原）
Hg Ag ──→ 热分解法（加热分解氧化物）
Pt Au ──→ 物理富集法（自然界以单质存在）
```

### 有机还原的结构变化
```
加氢还原的本质：
C=C + H₂ → C—C        （不饱和键→饱和键）
C=O + H₂ → CH—OH      （羰基→羟基）
—NO₂ + 6[H] → —NH₂    （硝基→氨基）
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 还原反应常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆还原反应中的"还原剂"与"被还原的物质"              │
│     还原剂：失电子，化合价升高，被氧化                     │
│     氧化剂：得电子，化合价降低，被还原                     │
│     例：H₂ + CuO → Cu + H₂O                              │
│         H₂是还原剂（被氧化），CuO是氧化剂（被还原）       │
├─────────────────────────────────────────────────────────┤
│ E2. 冶炼方法选择错误                                      │
│     NaCl不能用热还原法制Na（太活泼），必须电解熔融NaCl    │
│     Fe₂O₃不能用热分解法，必须用CO/H₂/C还原               │
├─────────────────────────────────────────────────────────┤
│ E3. 催化加氢条件遗漏                                      │
│     必须写：催化剂（Ni/Pt/Pd）+ 加热                      │
│     不能只写H₂或只写加热                                  │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视CO还原的尾气处理                                  │
│     CO有毒，未反应的CO需要点燃或收集处理                  │
│     实验装置尾部要有处理装置                              │
├─────────────────────────────────────────────────────────┤
│ E5. 羧酸不能被催化加氢还原                                │
│     RCOOH + H₂ → 不反应（催化加氢条件下）                │
│     羧酸还原需用LiAlH₄等强还原剂                          │
└─────────────────────────────────────────────────────────┘
```

## 七、关联反应类型

```
        还原反应
       /    |    \
      ↓     ↓     ↓
   加氢还原  去氧还原  电解还原
    |        |        |
   催化     热还原    熔融电解
   
还原反应 ↔ 氧化反应（同时发生，电子守恒）
还原反应（加氢）↔ 加成反应（催化加氢）
```
