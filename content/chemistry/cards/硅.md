```
├── 价电子排布：3s² 3p²
```

### 第二层：电子层与成键
```

├── 成键电子对：4个
├── 孤电子对：0个
└── 特征：
    - 3p轨道参与成键
```

| 键型 | 存在形式 | 特征 |

| Si=O双键 | 极不稳定 | 自然界少见 |

### 核心反应方程式
| 反应 | 条件 | 方程式 | 现象 |
|------|------|--------|------|
| Si与NaOH | 加热 | Si + 2NaOH + H₂O → Na₂SiO₃ + 2H₂↑ | 产生气泡 |
| Si与HF酸 | 常温 | Si + 4HF → SiF₄↑ + 2H₂↑ | 刺激性气体 |
| SiO₂与NaOH | 熔融 | SiO₂ + 2NaOH → Na₂SiO₃ + H₂O | — |
| SiO₂与HF酸 | 常温 | SiO₂ + 4HF → SiF₄↑ + 2H₂O | 溶解 |
| SiO₂与CaO | 高温 | SiO₂ + CaO → CaSiO₃ | — |
| 粗硅精制 | 🔵950℃ | Si + 3HCl → SiHCl₃ + H₂ | 挥发性氯化物 |

### 🟢特征检验
```
硅酸盐的检验方法：
├─ 加酸：生成白色胶状沉淀（H₂SiO₃）
│   Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl
├─ 灼烧：灼烧后不分解（耐高温）
└─ 与HF酸反应：产生SiF₄气体（刺激性）
```

## 四、高频陷阱 E1-E5

### 🔴 E1：硅与碳的相似性误解
> **陷阱**：认为硅和碳的化学性质完全相似
>
> **正确**：硅与碳的差异
> - Si-Si键弱于C-C键，硅链不稳定（碳链稳定）
> - 硅不能形成双键或三键（碳可以形成）
> - 硅与水不反应（碳在高温下与水反应）
> - 硅与碱反应（碳与碱不反应）

### 🔴 E2：SiO₂的两性误区
> **陷阱**：认为SiO₂是酸性氧化物，只能与碱反应
>
> **正确**：SiO₂是酸性氧化物，但能特殊反应
> - 与碱反应：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O
> - 与HF酸反应：SiO₂ + 4HF → SiF₄↑ + 2H₂O（唯一能溶解SiO₂的酸）
> - 与碳酸盐反应：SiO₂ + Na₂CO₃ → Na₂SiO₃ + CO₂↑（高温）

### 🔴 E3：硅烷的稳定性
> **陷阱**：认为硅也能形成稳定的有机硅化合物
>
> **正确**：硅烷与烷烃的对比
> - Si-Si键键能226 kJ/mol < C-C键348 kJ/mol
> - 硅烷稳定性差，易被氧化
> - 有机硅化合物稳定性来自Si-O键而非Si-C键

### 🔴 E4：硅的工业制备条件
> **陷阱**：认为可以用碳直接还原SiO₂制备纯硅
>
> **正确**：碳热还原SiO₂的问题
> - 反应：SiO₂ + 2C → Si + 2CO（高温）
> - 生成物是粗硅（含杂质）
> - 需要后续精制（氯化法：Si + 3HCl → SiHCl₃，提纯后还原）

### 🔴 E5：半导体硅的纯度要求
> **陷阱**：认为普通硅可以直接用于半导体
>
> **正确**：半导体级硅纯度要求
> - 太阳能电池用硅：纯度99.9999%以上
> - 半导体芯片用硅：纯度99.999999999%以上（11个9）
> - 需要区熔法或Cz法制备单晶硅

## 五、跨元素关联

### 垂直关系（同族）
```
IVA族碳族元素：
C（6）→ Si（14）→ Ge（32）→ Sn（50）→ Pb（82）

递变规律：
├─ 原子半径：增大
├─ 非金属性：减弱，金属性：增强
├─ C/Si：非金属，Ge：准金属，Sn/Pb：金属
└─ 氢化物稳定性：
   - CH₄（很稳定）> SiH₄（不稳定）> GeH₄（易分解）
```

### 水平关系（同周期）
```
第三周期元素：
Na → Mg → Al → Si → P → S → Cl → Ar

硅的位置特征：
├─ 与Al同周期，与P相邻
├─ 是地壳中含量第二的元素（27.7%）
├─ 良好的半导体材料
└─ 与磷同周期，价电子数相同
```

### 对角线关系
```
与硼的对角线相似性：
├─ 硼（第二周期IVA）和硅（第三周期IVA）
├─ 两者都是准金属
├─ 氧化物都是酸性偏两性
├─ 氢化物都不稳定
└─ 都能与氟形成稳定化合物
```
