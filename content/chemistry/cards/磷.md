# 磷（P）元素卡片

> **元素档案**：第三周期VA族，原子序数15，-3/+3/+5价，白磷红磷，着火点低

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第3周期
- **族**：VA族（氮族）
- **原子序数**：15
- **核外电子排布**：(+15) 2)8)5

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | N（氮） | 原子半径P > N，非金属性P < N |
| 下 | As（砷） | 非金属性P > As |
| 左 | Si（硅） | 非金属性P > Si |
| 右 | S（硫） | 非金属性S > P，得电子能力S更强 |

### 推断题身份证 ⭐
- ⭐ "白磷保存在水中" → P₄（隔绝空气，防止自燃）
- ⭐ "红磷用于火柴盒" → P（红磷，着火点260℃）
- ⭐ "白磷可制高纯磷酸" → P₄O₁₀/P₄O₆ → H₃PO₄
- ⭐ "鬼火（磷火）" → P + O₂ → P₂O₅/P₂O₃（氧化发光）
- ⭐ "磷酸盐沉淀" → PO₄³⁻（与Ca²⁺、Fe³⁺等形成沉淀）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+15
- 电子层数：3层
- 最外层电子数：5e⁻（**需得3e⁻或失5e⁻达到稳定**）

### 【第2层】电子行为
- **易失**：失3e⁻ → P³⁺（如PCl₃）
- **易失**：失5e⁻ → P⁵⁺（如P₂O₅/H₃PO₄）
- **可共享**：共用电子对形成共价键（P-O键较强）

### 【第3层】化学键型
- **与金属（Ca等）**：形成离子键 → Ca₃P₂（磷化物）
- **与非金属（O、Cl）**：形成共价键 → P₂O₅、PCl₅
- **白磷自身**：P-P键 → P₄正四面体结构（分子晶体）
- **红磷自身**：P-P-P长链结构（分子晶体）

### 【第4层】晶体类型
- **白磷（P₄）**：分子晶体（熔点44.1℃，沸点280℃）
- **红磷**：分子晶体（熔点260℃，不溶于水）
- 解释：白磷P₄正四面体，分子间作用力弱→熔点低；红磷聚合结构更稳定→熔点高

> **结构解释性质**：白磷P₄正四面体，键角60°，张力大→化学性质活泼（易自燃）；红磷结构更稳定→无此性质

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色状态 | 白磷：白色固体；红磷：红棕色粉末 | 同素异形体 |
| 溶解性 | 白磷不溶于水，溶于CS₂；红磷不溶 | 白磷非极性，相似相溶 |
| 毒性 | 白磷剧毒；红磷低毒 | 白磷蒸气吸入致死 |
| 保存 | 白磷保存在冷水中（隔绝空气） | 防止自燃 |

### 化学性质

#### 还原性（被氧化）—— 主要性质 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与O₂ | P₄ + 5O₂ =点燃= P₄O₁₀ | 白磷燃烧，产生大量白烟 |
| 与O₂（不足） | P₄ + 3O₂ =点燃= P₄O₆ | 生成P₂O₃（亚磷酸酐） |
| 与Cl₂（少量） | 2P + 3Cl₂ =点燃= 2PCl₃ | 生成三氯化磷（无色液体） |
| 与Cl₂（过量） | 2P + 5Cl₂ =点燃= 2PCl₅ | 生成五氯化磷（白色固体） |

🔴 **反例/边界**：P与Cl₂反应产物的价态取决于Cl₂用量——氧化性氛围强则P⁵⁺

#### 氧化性（被还原）—— 体现磷的负价
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与金属 | 3Ca + 2P =点燃= Ca₃P₂ | P从0→-3价（体现氧化性） |
| 与H₂ | P + H₂ ≠（难直接化合） | 需通过磷化物间接制PH₃ |

🔴 **反例/边界**：磷与氢气直接化合很难，通常用磷化物水解制PH₃

#### 磷酸的性质 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与碱 | PO₄³⁻ + 3OH⁻ → PO₄³⁻（磷酸盐水解） | 磷酸盐水解呈碱性 |
| 与金属 | H₃PO₄ + Ca(OH)₂ → CaHPO₄ + 2H₂O | 中和反应 |

## 维度4：用（Application）—— 三合一转化网

```
                        【核心枢纽：P / H₃PO₄】

    白磷 P₄（正四面体）
     │
     ├──[与O₂（足量）]──→ P₄O₁₀（五氧化二磷）
     │                   条件：点燃
     │                   现象：白色浓烟
     │
     ├──[与O₂（不足）]──→ P₄O₆（三氧化二磷）
     │                   条件：控制燃烧
     │
     └──[与H₂O（歧化）]──→ H₃PO₃ + PH₃（很少见）
                          P₄ + 6H₂O = 300℃ = 2H₃PO₃ + 2PH₃

                        【磷的氧化物 → 磷酸】

    P₄O₁₀ + 6H₂O → 4H₃PO₄
     │
     └──[与碱中和]──→ 磷酸盐（Na₃PO₄、Ca₃(PO₄)₂等）
                       条件：中和反应

                        【红磷的转化】

    红磷 P（稳定）
     │
     ├──[与O₂]──→ P₄O₁₀ → H₃PO₄（间接）
     │             条件：点燃（需先转化为白磷蒸气）
     │
     └──[与HNO₃]──→ H₃PO₄ + NO↑（氧化还原）
                     条件：浓硝酸

                        【磷酸盐的转化】

    PO₄³⁻
     │
     ├──[与Ca²⁺]──→ Ca₃(PO₄)₂↓（白色，骨骼成分）
     │             条件：碱性环境
     │             🟢检验磷酸根：加AgNO₃→黄色沉淀
     │
     ├──[与Fe³⁺]──→ FePO₄↓（淡黄色）
     │
     └──[与NH₄⁺]──→ NH₄H₂PO₄（磷酸二氢铵，肥料）

                        【磷酸的工业制备】

    Ca₃(PO₄)₂ + 3H₂SO₄ = 2H₃PO₄ + 3CaSO₄↓
                          🔵湿法磷酸（副产石膏）

    P₄O₁₀ + 6H₂O = 4H₃PO₄
                          🔵热法磷酸（高纯度）
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | 白磷和红磷是同一种物质的不同颜色 | 互为同素异形体，结构不同（白磷P₄正四面体，红磷链状结构），性质差异大 |
| **E2** | 白磷保存在水中因为它溶于水 | 白磷**不溶于水**，保存在水中是为了**隔绝空气防止自燃**（着火点40℃） |
| **E3** | 磷酸是三元酸，所以能形成三种盐 | 磷酸是三元中强酸，可形成磷酸二氢盐、磷酸氢盐、磷酸盐（取决于pH） |
| **E4** | P₂O₅是干燥剂，只能干燥酸性气体 | P₂O₅可干燥酸性气体（HCl、CO₂等），但**不能干燥碱性气体（NH₃）** |
| **E5** | 磷火（鬼火）是因为磷燃烧 | 磷火是磷化氢（PH₃）自燃，不是单质磷燃烧；PH₃由有机物腐烂产生 |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **N** | 同族 | N与P同为VA族，NH₃ vs PH₃（磷化氢），氮比磷非金属性更强 |
| **O** | 氧化产物 | P → P₂O₅/P₂O₃ → H₃PO₄/H₃PO₃ |
| **Cl** | 卤化物 | PCl₃（+3）与PCl₅（+5），取决于Cl₂用量 |
| **Ca** | 骨骼关联 | Ca₃(PO₄)₂（骨骼牙齿的主要无机成分） |
| **S** | 联合推断 | 磷矿与硫铁矿联合：3Ca₃(PO₄)₂ + 2SiO₂ → 高温 → P |
| **Fe** | 肥料关联 | 磷酸铁可用作磷肥（缓释） |

### 联合推断规律
- P ──同素异形体──→ S（硫也有多种同素异形体：斜方硫、单斜硫、弹性硫）
- P ──氧化还原──→ N（磷肥与氮肥常联合出题）
- P ──工业流程──→ H（磷矿石用硫酸处理制磷酸）
- H₃PO₄ ──中和反应──→ Ca（磷酸二氢钙是可溶于水的磷肥）
