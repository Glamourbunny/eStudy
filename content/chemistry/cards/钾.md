# 钾（K）元素卡片

> **元素档案**：第四周期，IA族，原子序数19，银白色金属，密度0.862g/cm³，比水轻，化学性质极活泼

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第4周期
- **族**：IA族（碱金属）
- **原子序数**：19
- **核外电子排布**：(+19) 2)8)8)1

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | Na（钠） | 金属性K > Na（原子半径增大） |
| 下 | Rb（铷） | 金属性Rb > K（更活泼） |
| 左 | Ar（氩） | 惰性气体 |
| 右 | Ca（钙） | 金属性K > Ca（K更活泼） |

### 推断题身份证 ⭐
- ⭐ "保存在煤油中的银白色金属，比钠更活泼" → K
- ⭐ "投入水中浮在水面，熔成小球，立即燃烧" → K（比Na更剧烈）
- ⭐ "焰色反应呈紫色（透过蓝色钴玻璃）" → K
- ⭐ "与水反应生成无色碱性溶液" → KOH溶液

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+19
- 电子层数：4层
- 最外层电子数：1e⁻（极易失去）

### 【第2层】电子行为
- **极易失**：失去最外层1e⁻ → K⁺（18电子稳定结构，类似Ar）
- K的失电子能力极强（电离能最低之一）

### 【第3层】化学键型
- **与氧**：形成K₂O（离子化合物）；也可形成K₂O₂（过氧化钾）
- **与卤素**：形成KX（KCl、KBr等）
- **与水/酸**：剧烈反应，放出H₂

### 【第4层】晶体类型
- **K单质**：金属晶体
- **K₂O/K₂O₂**：离子晶体
- 解释：K⁺半径大，离子键相对较弱→K₂O/K₂O₂不如对应钠化合物稳定

> **结构解释性质**：K原子半径最大（IA族同周期）→最外层电子最易失去→K是最活泼的金属之一

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 外观 | 银白色（暴露空气变暗） | 与O₂反应 |
| 密度 | 0.862g/cm³（比水轻，比Na大） | 投入水中浮在水面 |
| 熔点 | 63.5℃（低） | 比Na（97.8℃）更低 |
| 硬度 | 极软（可用刀切） | 金属键弱 |

### 化学性质

#### 还原性（极强）🔴
| 反应 | 方程式 | 现象 |
|------|--------|------|
| 与O₂（常温） | 4K + O₂ = 2K₂O | 立刻氧化，表面变暗 |
| 与O₂（点燃） | K + O₂ =点燃= KO₂ | 超氧化钾KO₂ |
| 与H₂O | 2K + 2H₂O = 2KOH + H₂↑ | 立刻燃烧，紫色火焰，产生KOH |
| 与酸 | 2K + 2HCl = 2KCl + H₂↑ | 爆炸 |

🔴 **反例/边界**：K与H₂O反应比Na**更剧烈**（Na可能只是熔化，K会立刻燃烧）

#### 超氧化钾KO₂的特殊性 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与CO₂ | 4KO₂ + 2CO₂ = 2K₂CO₃ + 3O₂ | 供氧反应 |
| 与H₂O | 2KO₂ + 2H₂O = 2KOH + H₂O₂ + O₂↑ | 产生O₂ |

🔴 **反例/边界**：K能形成超氧化物KO₂（O₂⁻），Na主要形成过氧化物Na₂O₂

## 维度4：用（Application）—— 三合一转化网

```
                              【核心枢纽：K₂O / KOH】

    K（银白色金属）
     │
     ├──[与O₂]──→ K₂O（氧化钾）
     │    4K + O₂ = 2K₂O
     │    现象：表面立刻变暗
     │
     ├──[与O₂（过量）]──→ KO₂（超氧化钾，黄色）
     │    K + O₂ =点燃= KO₂
     │    🟢 供氧剂，用于呼吸面具
     │
     ├──[与H₂O]──→ KOH + H₂↑
     │    2K + 2H₂O = 2KOH + H₂↑
     │    现象：立刻燃烧，紫色火焰
     │
     └──[与Cl₂]──→ KCl（白烟）
         2K + Cl₂ =点燃= 2KCl

                              【K₂O的转化】

    K₂O + H₂O = 2KOH（剧烈放热）
     │
    K₂O + CO₂ = K₂CO₃

                              【KOH的转化】

    KOH + HCl = KCl + H₂O
     │
    2KOH + CO₂ = K₂CO₃ + H₂O
     │
    KOH + CO₂（过量）= KHCO₃

                              【重要化合物】

    🔵 KCl（氯化钾）：
       钾肥（K⁺是植物必需元素）
       配制生理盐水（部分）

    🔵 KNO₃（硝酸钾）：
       复合肥料
       的黑火药成分

    🔵 KO₂（超氧化钾）：
       供氧剂：4KO₂ + 2CO₂ = 2K₂CO₃ + 3O₂
       呼吸面具、防火服

    🔵 焰色反应：
       K⁺火焰呈紫色（透过蓝色钴玻璃观察）
       🟢 用于检验K⁺的存在
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | K比Na更活泼是因为K原子更小 | 事实相反：K原子半径**更大**，最外层电子离核更远，更易失去电子 |
| **E2** | K与O₂反应只生成K₂O | K能形成多种氧化物：K₂O（正常）、K₂O₂（过氧）、KO₂（超氧，K过量时） |
| **E3** | K与Na化学性质完全相同 | 有相似也有差异：K能形成超氧化物KO₂，Na主要形成Na₂O₂；K与水反应更剧烈 |
| **E4** | K保存方法与Na相同 | 都是保存在煤油中，但K密度更小（0.862），需确保煤油量足够没过K |
| **E5** | 焰色反应K呈黄色 | K焰色反应呈**紫色**（不是黄色，黄色是Na；透过蓝色钴玻璃消除Na干扰） |
