## 二、四层递进结构

### 第一层：原子结构
```
原子序数 16 → 核电荷数 +16
质子数 = 16，中子数 = 16（常见同位素 32S）
最外层电子数 = 6 → 易得2电子或失6电子
```

### 第二层：电子层排布
```
K层：2电子（饱和）
L层：8电子（饱和）
M层：6电子（**未饱和，可扩展至8**）
```

### 第三层：化学键类型
```
📌 主要成键方式：
• 共价键：S-S（单键）、S=O（双键，如SO₂）
• 配位键：SO₄²⁻中的d-pπ反馈键
📌 氧化数规律：
  -2价：H₂S、Na₂S（最低价态，强还原性）
   0价：S（单质）
  +4价：SO₂、H₂SO₃（中间价态，两性）
  +6价：SO₃、H₂SO₄（最高价态，强氧化性）
```

### 第四层：晶体类型
```
📍 S单质晶体类型：**分子晶体**（S₈环状分子）
📍 分子间作用力：**范德华力**（较弱，熔沸点低）
📍 硬度小，易粉碎，**不导电**
```

## 三、三合一转化网

### 3.1 硫的化合物转化关系

```
            燃烧 ☀️
      S ──────────→ SO₂
       ↘︎          ↓⇌
        H₂S   ←── SO₃
        ↑            ↓ 🔵
      稀盐酸    H₂SO₄
```

### 3.2 重要反应方程式

| 反应类型 | 条件 | 化学方程式 | 现象 |
|---------|------|-----------|------|
| **硫燃烧** | 点燃 | `S + O₂ → SO₂` | 蓝紫色火焰，产生刺激性气体 |
| **硫与金属** | 加热 | `Fe + S → FeS` | 黑色固体生成 |
| **硫与氢气** | 加热 | `H₂ + S ⇌ H₂S` | 气体具有臭鸡蛋味 |
| **二氧化硫氧化** | 催化剂/加热 | `2SO₂ + O₂ ⇌ 2SO₃` | 可逆反应 |
| **三氧化硫水解** | 遇水 | `SO₃ + H₂O → H₂SO₄` | 放热，大量白雾 |
| **硫化氢受热分解** | 加热 | `H₂S → H₂ + S` | 分解为H₂和S |
| **硫酸盐检验** 🟢 | — | `Ba²⁺ + SO₄²⁻ → BaSO₄↓` | **白色沉淀不溶于酸** |

### 3.3 硫酸的工业制备（接触法）🔵

```
步骤1：硫燃烧 → 二氧化硫
       S + O₂ → SO₂

步骤2：二氧化硫催化氧化 → 三氧化硫
       2SO₂ + O₂ ⇌ 2SO₃（V₂O₅催化，ΔH<0）

步骤3：三氧化硫吸收 → 硫酸
       SO₃ + H₂O → H₂SO₄
       ⚠️ 不是直接用水吸收SO₃，而是用98.3%浓硫酸吸收
```

## 四、🔴 红框反例边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 高频失分陷阱 E1-E5                                    │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆SO₂和SO₃的水溶性                                  │
│     SO₂易溶水（1:40），SO₃遇水反应放热                    │
│     ⚠️ 不能用排水法收集SO₂，应用向上排空气法               │
├─────────────────────────────────────────────────────────┤
│ E2. 认为SO₂只体现还原性                                   │
│     实际上SO₂处于中间价态(+4)，**既有氧化性又有还原性**     │
│     与H₂S反应：SO₂ + 2H₂S → 3S↓ + 2H₂O（氧化性）          │
│     与Cl₂反应：SO₂ + Cl₂ + 2H₂O → H₂SO₄ + 2HCl（还原性）  │
├─────────────────────────────────────────────────────────┤
│ E3. 误认为浓硫酸有氧化性，稀硫酸无氧化性                   │
│     稀硫酸也有氧化性（H⁺得电子），但**主要体现酸性**        │
│     浓硫酸的氧化性来源：S(+6) → S(+4)，强氧化性           │
│     稀硫酸的氧化性来源：H⁺ → H₂，氧化性弱                 │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视可逆反应条件                                      │
│     2SO₂ + O₂ ⇌ 2SO₃，需要持续通入O₂提高转化率           │
│     ⚠️ 压强增大、温度降低有利于向正反应方向移动            │
├─────────────────────────────────────────────────────────┤
│ E5. 硫酸浓度与氧化性关系混淆                              │
│     浓硫酸：强氧化性、脱水 性、吸水性                      │
│     稀硫酸：酸 性、弱氧化性                               │
│     ⚠️ 铁/铝遇浓硫酸钝化，但稀硫酸会反应                   │
└─────────────────────────────────────────────────────────┘
```

| 性质 | O | S | Se | Te |

| 单质状态 | 无色气体 | 淡黄色固体 | 灰色固体 | 银白色固体 |
| 常见价态 | -2 | -2,+4,+6 | -2,+4,+6 | -2,+4,+6 |
| 氢化物 | H₂O | H₂S | H₂Se | H₂Te |
| 氢化物沸点 | 高 | 低 | 更低 | 最低 |

### 5.2 硫-卤素反应

```
S + 2Cl₂ → SCl₄（四氯化硫）
S + Br₂ → S₂Br₂（溴化硫）
```
