# 原电池正负极

> **底线地位**：电化学的基础。正负极判断错误，电极反应式和电子流向全部错误。

---

## 一、概念定义对比

| 维度 | 负极 | 正极 |
|------|------|------|
| **电子流向** | 电子流出 | 电子流入 |
| **氧化还原** | 失电子，被氧化 | 得电子，被还原 |
| **电极性质** | 还原剂 | 氧化剂 |
| **质量变化** | 通常溶解/消耗 | 通常增重/析出 |
| **离子移动** | 阴离子向负极移动 | 阳离子向正极移动 |

---

## 二、核心区别：电子流向

```
电子流向：负极 → 外电路 → 正极
电流方向：正极 → 外电路 → 负极（与电子流向相反！）

电极反应：
负极：还原剂 — ne⁻ → 氧化产物（氧化反应）
正极：氧化剂 + ne⁻ → 还原产物（还原反应）
```

### 记忆口诀
```
"负氧正还"：
负极 → 氧化反应（失电子）
正极 → 还原反应（得电子）

"电子出负入正"：
电子从负极流出，从正极流入
```

---

## 三、判断方法

### 方法一：根据电极材料判断（一般规律）
```
活泼金属作负极，不活泼金属/非金属作正极

例：Cu-Zn原电池
  Zn比Cu活泼 → Zn是负极，Cu是正极
  负极(Zn)：Zn — 2e⁻ → Zn²⁺
  正极(Cu)：2H⁺ + 2e⁻ → H₂↑

例外：Al-Mg-NaOH原电池
  通常Mg比Al活泼，但Al能与NaOH反应
  → Al是负极，Mg是正极
  （能反应的一极是负极）
```

### 方法二：根据反应类型判断
```
失电子（氧化）→ 负极
得电子（还原）→ 正极
```

### 方法三：根据现象判断
```
溶解的一极 → 负极（金属被氧化溶解）
有气泡/析出的一极 → 正极（离子被还原）
```

## 四、典型例子

### 例1：Cu-Zn稀硫酸原电池
```
负极(Zn)：Zn — 2e⁻ → Zn²⁺      （氧化，溶解）
正极(Cu)：2H⁺ + 2e⁻ → H₂↑      （还原，气泡）

总反应：Zn + 2H⁺ → Zn²⁺ + H₂↑

离子移动：
  H⁺、Zn²⁺向正极移动
  SO₄²⁻向负极移动
```

### 例2：Fe-C-NaCl原电池（吸氧腐蚀）
```
负极(Fe)：2Fe — 4e⁻ → 2Fe²⁺     （氧化）
正极(C)：O₂ + 2H₂O + 4e⁻ → 4OH⁻  （还原）

总反应：2Fe + O₂ + 2H₂O → 2Fe(OH)₂
后续：Fe(OH)₂ → Fe(OH)₃ → Fe₂O₃·nH₂O（铁锈）
```

### 例3：燃料电池（H₂-O₂）
```
酸性介质：
负极：2H₂ — 4e⁻ → 4H⁺
正极：O₂ + 4H⁺ + 4e⁻ → 2H₂O

碱性介质：
负极：2H₂ + 4OH⁻ — 4e⁻ → 4H₂O
正极：O₂ + 2H₂O + 4e⁻ → 4OH⁻

关键：同一燃料电池，介质不同，电极反应不同！
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 正负极常见陷阱                                         │
├─────────────────────────────────────────────────────────┤
│ E1. 电子流向与电流方向混淆                                │
│     电子：负极 → 正极（外电路）                          │
│     电流：正极 → 负极（外电路）                          │
│     电子带负电，电流方向定义为正电荷流动方向               │
├─────────────────────────────────────────────────────────┤
│ E2. 用"活泼性"判断正负极时忽视特殊情况                    │
│     反例1：Al-Mg-NaOH电池，Al是负极（Al与NaOH反应）      │
│     反例2：Fe-Cu-浓HNO₃，Cu是负极（Fe钝化）              │
│     原则：能发生氧化反应的一极是负极                     │
├─────────────────────────────────────────────────────────┤
│ E3. 电极反应式环境（酸/碱）写错                           │
│     酸性：用H⁺和H₂O配平                                  │
│     碱性：用OH⁻和H₂O配平                                 │
│     错误：碱性介质中出现H⁺                               │
├─────────────────────────────────────────────────────────┤
│ E4. 离子移动方向记反                                      │
│     阳离子向正极移动（正电荷被正极吸引）                 │
│     阴离子向负极移动（负电荷被负极吸引）                 │
│     记忆："同性相斥，异性相吸"→ 阳去正极，阴去负极      │
├─────────────────────────────────────────────────────────┤
│ E5. 燃料电池中忽视介质对电极反应的影响                    │
│     H₂-O₂燃料电池：                                      │
│     酸性：负极产生H⁺，正极消耗H⁺                         │
│     碱性：负极消耗OH⁻，正极产生OH⁻                       │
│     绝对不能在碱性介质中写H⁺！                           │
└─────────────────────────────────────────────────────────┘
```

## 六、关联概念网络

```
        原电池
       /      \
      ↓        ↓
    负极      正极
   失电子    得电子
   氧化反应  还原反应
   电子流出  电子流入
      \      /
       ↓    ↓
      电子守恒
      （负极失电子=正极得电子）
```
