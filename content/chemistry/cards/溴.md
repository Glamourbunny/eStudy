## 二、四层递进结构

### 第一层：原子结构
```
原子序数 35 → 核电荷数 +35
质子数 = 35，中子数 = 45（常见同位素 79Br、81Br）
最外层电子数 = 7 → **极易得1电子**，氧化性强于碘
```

### 第二层：电子层排布
```
K层：2电子（饱和）
L层：8电子（饱和）
M层：18电子（饱和）
N层：7电子（**未饱和，得1电子达饱和**）
```

### 第三层：化学键类型
```
📌 主要成键方式：
• 离子键：NaBr、KBr（与金属形成）
• 共价键：Br-Cl、Br₂（单质）
• 配位键：Br⁻作为配体
📌 价态规律：
  -1价：Br⁻（最稳定，还原性弱于Cl⁻、I⁻）
   0价：Br₂（单质）
  +1价：BrO⁻（次溴酸根）
  +3价：BrO₂⁻（亚溴酸根）
  +5价：BrO₃⁻（溴酸根）
```

### 第四层：晶体类型
```
📍 单质溴：分子晶体（Br₂分子）
📍 溴化物：离子晶体（如NaBr晶体）
📍 分子间作用力：范德华力
📍 特性：**液溴可微溶水（溴水），易溶于有机溶剂**
```

## 三、三合一转化网

### 3.1 溴的化合物转化关系

```
            氧化剂（如Cl₂）
      Br⁻ ⇌────────────────→ Br₂
       ↑                      ↓
       ←──── 还原剂（如I₂） ←─┘
       
            碱性条件/歧化
      Br₂ ⇌───────────────→ Br⁻ + BrO⁻
```

### 3.2 重要反应方程式

| 反应类型 | 条件 | 化学方程式 | 现象 |
|---------|------|-----------|------|
| **溴与氢气** | 光照/加热 | `Br₂ + H₂ → 2HBr` | 反应较氯气慢 |
| **溴与金属** | 室温/加热 | `2Na + Br₂ → 2NaBr` | 白色晶体生成 |
| **溴与水反应** | 常温 | `Br₂ + H₂O ⇌ HBr + HBrO` | 溶液呈橙红色 |
| **溴的歧化** | 碱液 | `Br₂ + 2NaOH → NaBr + NaBrO + H₂O` | 褪色 |
| **溴与碘离子** | — | `Br₂ + 2KI → 2KBr + I₂` | 溶液变蓝（淀粉） |
| **溴与二氧化硫** | — | `Br₂ + SO₂ + 2H₂O → H₂SO₄ + 2HBr` | 溴水褪色 |
| **溴的检验** 🟢 | 淀粉溶液 | `I₂ + 淀粉 → 蓝色` | **蓝色络合物** |

### 3.3 溴的提取（海水提溴）🔵

```
步骤1：海水酸化
       母液（晒盐后的卤水）用H₂SO₄酸化

步骤2：氯气氧化
       2Br⁻ + Cl₂ → Br₂ + 2Cl⁻

步骤3：空气吹出
       用空气将Br₂吹出，被Na₂CO₃溶液吸收
       3Br₂ + 3Na₂CO₃ → 5NaBr + NaBrO₃ + 3CO₂↑

步骤4：酸化浓缩
       5HBr + HBrO₃ → 3Br₂ + 3H₂O
```

## 四、🔴 红框反例边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 高频失分陷阱 E1-E5                                    │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆液溴和溴水的性质                                  │
│     液溴：纯Br₂，深红棕色液体，**强腐蚀性**               │
│     溴水：Br₂的水溶液，橙黄色，**氧化性较弱**             │
│     ⚠️ 液溴保存在磨口玻璃瓶，水封防挥发                   │
├─────────────────────────────────────────────────────────┤
│ E2. 认为溴氧化性强于氯                                    │
│     实际上：**氧化性Cl₂ > Br₂ > I₂**                       │
│     氯气可氧化Br⁻：Cl₂ + 2Br⁻ → Br₂ + 2Cl⁻              │
│     溴氧化性强度的顺序：Cl > Br > I                        │
├─────────────────────────────────────────────────────────┤
│ E3. 忽视溴的歧化反应条件                                  │
│     Br₂在碱液中歧化：Br₂ + 2OH⁻ → Br⁻ + BrO⁻             │
│     酸性条件下逆向：Br⁻ + BrO⁻ + H⁺ → Br₂ + H₂O          │
│     ⚠️ 海水提溴就是利用这个原理                           │
├─────────────────────────────────────────────────────────┤
│ E4. 错误理解溴与淀粉显色反应                              │
│     **只有I₂与淀粉显蓝色，Br₂不与淀粉显色**               │
│     检验I₂的正确方法：淀粉溶液 → 蓝色                     │
│     ⚠️ 这是区分I₂和Br₂的关键实验                          │
├─────────────────────────────────────────────────────────┤
│ E5. 混淆溴的氧化性和溴离子的还原性                        │
│     Br₂（0价）：强氧化性                                  │
│     Br⁻（-1价）：弱还原性（还原性 Cl⁻ < Br⁻ < I⁻）        │
│     ⚠️ Br⁻可被Cl₂氧化为Br₂，但难被Br₂氧化                 │
└─────────────────────────────────────────────────────────┘
```

## 五、跨元素关联

### 5.1 卤素单质物理性质递变（VIIA族）

| 性质 | F₂ | Cl₂ | Br₂ | I₂ |
|------|-----|------|------|-----|
| 状态 | 浅黄绿气体 | 黄绿色气体 | **红棕色液体** | 紫黑色固体 |
| 颜色 | 淡黄 | 黄绿 | **红棕** | 紫黑 |
| 熔沸点 | 最低 | 较低 | 较高 | 最高 |
| 溶解性 | 与水反应 | 微溶 | **微溶** | 难溶 |
| 特性 | 最活泼 | 活泼 | 易挥发 | **升华** |

### 5.2 卤素氧化性顺序

```
氧化性：F₂ > Cl₂ > Br₂ > I₂
还原性：F⁻ < Cl⁻ < Br⁻ < I⁻（递增）

应用：
• Cl₂可氧化Br⁻ → Br₂（海水提溴原理）
• Br₂可氧化I⁻ → I₂（检验I⁻）
• I₂可被Br₂氧化（逆向不行）
```

### 5.3 溴的含氧酸

| 化合物 | 化学式 | 名称 | 稳定性 |
|--------|--------|------|--------|
| HBrO | Br-OH | 次溴酸 | **不稳定，易分解** |
| HBrO₂ | O=Br-OH | 亚溴酸 | 不稳定 |
| HBrO₃ | O=Br(=O)₂ | 溴酸 | 较稳定 |
| HBrO₄ | O=Br(=O)₃ | 高溴酸 | 稳定 |

### 5.4 关联元素网络

```
        F（卤素最强氧化性）
       /
  Cl ←  Br → I
   \    |    /
    卤化物/互化物
```
