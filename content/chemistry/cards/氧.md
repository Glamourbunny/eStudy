# 氧（O）元素卡片

> **元素档案**：第二周期VIA族，原子序数8，-2价，生命元素，地壳中含量最高（46.6%）

---

## 维度1：位（Position）

### 周期表坐标
- **周期**：第2周期
- **族**：VIA族（氧族）
- **原子序数**：8
- **核外电子排布**：(+8) 2)6

### 相邻元素递变规律
| 方向 | 元素 | 递变规律 |
|------|------|---------|
| 上 | 无（第一周期无对应） | — |
| 下 | S（硫） | 原子半径O < S，非金属性O > S |
| 左 | N（氮） | 非金属性O > N，得电子能力O更强 |
| 右 | F（氟） | 非金属性F > O，氧化性F₂更强 |

### 推断题身份证 ⭐
- ⭐ "生命元素、空气中21%" → O₂
- ⭐ "使带火星木条复燃" → O₂（助燃性）
- ⭐ "能使燃烧更旺" → O₂（助燃剂）
- ⭐ "铁锈的主要成分" → Fe₂O₃（氧腐蚀）
- ⭐ "可形成氢键" → H₂O（氧的特殊性：电负性大、原子半径小）

## 维度2：构（Structure）—— 四层递进

### 【第1层】原子结构
- 核电荷数：+8
- 电子层数：2层
- 最外层电子数：6e⁻（**需得2e⁻达到8电子稳定结构**）

### 【第2层】电子行为
- **易得**：得到2e⁻ → O²⁻（8+2=10电子，稳定结构）
- **可共享**：共用电子对形成共价键（如H₂O、CO₂）
- **难以失电子**：电负性极大（3.44），仅小于F

### 【第3层】化学键型
- **与金属（Na、Mg等）**：形成离子键 → Na₂O、MgO（离子化合物）
- **与非金属（H、C等）**：形成极性共价键 → H₂O、CO₂
- **自身**：O=O双键 → O₂（氧气，顺磁双原子分子）
- **臭氧**：O=O→O（三个氧原子，弯线型结构）

### 【第4层】晶体类型
- **O₂**：分子晶体（熔点-218℃，沸点-183℃）
- **O₃（臭氧）**：分子晶体（熔点-193℃，沸点-111℃）
- **H₂O**：分子晶体（熔点0℃，沸点100℃，反常高——氢键）
- **Na₂O/MgO**：离子晶体（熔点高，熔融导电）
- 解释：O₂和O₃是同素异形体；H₂O因氢键导致熔沸点反常高

> **结构解释性质**：O原子半径小、电负性大→O-H键极性强→水分子间氢键强→沸点高（100℃）；臭氧O₃分子极性更强，更不稳定

## 维度3：性（Property）+ 反例边界

### 物理性质
| 性质 | 描述 | 解释 |
|------|------|------|
| 颜色状态 | 无色无味气体（O₂）；淡蓝色气体（O₃） | O₃呈特征淡蓝色 |
| 密度 | O₂：1.429 g/L（大于空气） | 排空气法收集，向上排 |
| 溶解性 | O₂不易溶于水（1:0.03）；O₃可溶 | 养鱼用增氧 |
| 气味 | O₃有特殊性腥味（臭氧味） | O₃是淡蓝色有腥味的气体 |

### 化学性质

#### 氧化性（被还原）—— 主要性质 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 与金属（Mg） | 2Mg + O₂ =点燃= 2MgO | 耀眼白光，生成白色固体 |
| 与非金属（C） | C + O₂ =点燃= CO₂ | 完全燃烧 |
| 与非金属（S） | S + O₂ =点燃= SO₂ | 蓝紫色火焰 |
| 与有机物 | CH₄ + 2O₂ =点燃= CO₂ + 2H₂O | 甲烷燃烧 |

🔴 **反例/边界**：O₂氧化性比Cl₂弱（Cl₂能氧化I⁻为I₂，O₂不能直接氧化I⁻）

#### 还原性（被氧化）—— 极少体现 🔴
- O在化合物中一般为-2价，只有与F化合时显正价
- 例：O₂ + F₂ → 2OF₂（O为+2价，氟化氧）

🔴 **反例/边界**：O₂与F₂反应生成OF₂，说明F₂氧化性更强，氧在此化合物中显+2价

#### 臭氧（O₃）的特殊性 🔴
| 反应 | 方程式 | 说明 |
|------|--------|------|
| 分解 | 2O₃ = 3O₂ | O₃不稳定，易分解为O₂ |
| 强氧化性 | O₃ + 2H⁺ + 2I⁻ → I₂ + O₂ + H₂O | 🟢臭氧检验：使碘化钾淀粉试纸变蓝 |
| 漂白性 | O₃有强漂白性（比Cl₂更强） | 用于消毒杀菌 |

## 维度4：用（Application）—— 三合一转化网

```
                        【核心枢纽：O₂ / H₂O】

    O₂（无色气体）
     │
     ├──[与金属/非金属燃烧]──→ 氧化物
     │                       条件：点燃
     │                       现象：燃烧发光，放热
     │
     ├──[与H₂（爆鸣气）]──→ 2H₂O
     │                      条件：点燃（2:1体积比）
     │                      现象：爆鸣声
     │
     ├──[与CH₄]──→ CO₂ + 2H₂O
     │              条件：点燃
     │              现象：蓝色火焰
     │
     └──[呼吸/代谢]──→ CO₂ + H₂O
                       条件：生物氧化

                        【H₂O的转化】

    H₂O
     │
     ├──[电解]──→ 2H₂↑ + O₂↑
     │            条件：电解（直流电）
     │            现象：阳极产生O₂，阴极产生H₂
     │
     ├──[与Na₂O]──→ 2NaOH
     │              条件：常温
     │
     ├──[与CaO]──→ Ca(OH)₂（石灰消化）
     │             条件：常温
     │
     └──[与金属（Fe）]──→ Fe₂O₃ + H₂（氧化还原）
                           条件：高温（水蒸气）

                        【O₃的转化】

    O₃（淡蓝色气体）
     │
     ├──[加热]──→ 2O₃ = 3O₂
     │            条件：加热或紫外光
     │
     ├──[与I⁻]──→ I₂ + O₂
     │            🟢臭氧检验：碘化钾淀粉试纸变蓝
     │
     └──[消毒杀菌]──→ O₃ → O₂ + [O]
                       强氧化剂，漂白消毒

                        【工业制备】

    H₂O =电解= 2H₂↑ + O₂↑
     │
     ├──[空气分馏]──→ O₂↑（-183℃）
     │                🔵工业制氧（低温蒸馏法）
     │
     └──[过氧化氢分解]──→ 2H₂O₂ = MnO₂ = 2H₂O + O₂↑
                           🔵实验室制氧方法之一
```

## 维度5：错（Error）—— 高频陷阱区

| 编号 | 陷阱描述 | 正确认知 |
|------|---------|---------|
| **E1** | O₂支持燃烧，所以O₂是燃料 | O₂是**助燃剂**（氧化剂），不是燃料；燃烧是物质与O₂剧烈反应放热 |
| **E2** | 臭氧（O₃）是O₂的另一种形态，性质相同 | O₃是O₂的同素异形体，O₃氧化性更强、能漂白消毒，O₂无此性质 |
| **E3** | 水电解生成H₂和O₂，所以H₂O中O是-2价 | 水电解是外加电流强迫发生反应，O₂中O为0价，H₂中H为0价；化合物中O一般为-2价 |
| **E4** | 燃烧都需要O₂ | 燃烧是物质与氧化剂剧烈反应，**不一定需要O₂**（如H₂在Cl₂中燃烧：H₂ + Cl₂ = 2HCl） |
| **E5** | 氧元素在所有化合物中都显-2价 | 氧在**过氧化物（Na₂O₂、H₂O₂）中显-1价**，在**OF₂中显+2价** |

## 维度6：链（Link）—— 跨元素关联网

### 核心关联
| 元素 | 关联类型 | 关联内容 |
|------|---------|---------|
| **H** | 生命基础 | H₂O是生命之源，氧占水质量的89% |
| **S** | 工业联合 | S + O₂ → SO₂ → SO₃ → H₂SO₄（硫酸工业） |
| **C** | 碳氧循环 | CO₂ ↔ O₂（光合作用与呼吸） |
| **Fe** | 腐蚀关联 | Fe + O₂ + H₂O → Fe₂O₃·nH₂O（铁锈） |
| **N** | 氮氧平衡 | NO/NO₂ + O₃ → NO₂ + O₂（光化学烟雾） |
| **Na** | 氧化物链 | Na₂O → Na₂O₂ → NaO₂（过氧、超氧） |

### 联合推断规律
- O₂ ──氧化性对比──→ Cl₂（Cl₂氧化性更强，能氧化I⁻）
- O₂ ──生命循环──→ C（光合作用：6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂）
- H₂O ──分解反应──→ H（电解水制氢燃料）
- O₃ ──检验方法──→ I（臭氧使KI淀粉变蓝，与Cl₂区分）
