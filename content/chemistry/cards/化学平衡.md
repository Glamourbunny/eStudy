# 化学平衡 vs 非平衡

> **底线地位**：可逆反应的核心概念。平衡标志判断错误，会导致平衡移动方向和转化率计算全部错误。

---

## 一、概念定义对比

| 维度 | 化学平衡 | 非平衡状态 |
|------|----------|-----------|
| **反应状态** | 可逆反应达到动态平衡 | 反应正在进行中 |
| **正逆反应速率** | v正 = v逆 ≠ 0 | v正 ≠ v逆 |
| **各组分浓度** | 保持不变（恒定） | 正在变化 |
| **宏观现象** | 反应"似乎停止" | 明显观察到变化 |
| **本质** | 动态平衡（分子层面仍在反应） | 单向进行 |

---

## 二、核心特征：动态平衡

```
化学平衡的本质：

      反应物 ⇌ 产物
         
v正 = v逆 ≠ 0
（正逆反应速率相等，但都不为零）

宏观：各组分浓度不变
微观：正逆反应仍在进行，只是速率相等

"动态"的含义：
• 反应没有停止
• 分子仍在不断转化
• 只是正逆转化的速率相等
• 所以浓度看起来不变
```

---

## 三、平衡标志的判断方法

### 根本标志（必须满足）
```
v正 = v逆
（同一物质的生成速率 = 消耗速率）
```

### 衍生标志（间接判断）
```
① 各组分浓度不再变化
② 各组分的百分含量不变
③ 对于有颜色的物质，颜色不再变化
④ 对于气体反应，压强不再变化（Δn≠0时）
⑤ 对于气体反应，平均相对分子质量不变（特定条件）
⑥ 温度不变（绝热体系中）
```

### ⚠️ 不能作为平衡标志的情况

| 情况 | 为什么不能作为平衡标志 |
|------|----------------------|
| 各物质浓度相等 | 平衡时浓度不一定相等 |
| 分子数之比等于系数比 | 任意时刻都可能满足 |
| 速率之比等于系数比 | 任意时刻都满足 |
| 对于Δn=0的反应，压强不变 | 压强始终不变 |
| 对于全气体反应且Δn=0，密度不变 | 密度始终不变 |

---

## 四、典型例子

### 例1：N₂ + 3H₂ ⇌ 2NH₃
```
平衡标志（正确）：
✓ v生成(NH₃) = v消耗(NH₃)
✓ N₂浓度不再变化
✓ NH₃的体积分数不变
✓ 容器内压强不变（Δn≠0）
✓ 平均相对分子质量不变

非平衡标志（错误）：
✗ N₂:H₂:NH₃ = 1:3:2
✗ 每消耗1mol N₂同时消耗3mol H₂（这是任意时刻都成立的）
```

### 例2：H₂ + I₂ ⇌ 2HI
```
注意：此反应Δn = 0

平衡标志：
✓ HI浓度不变
✓ 颜色不再变化（I₂蒸气有颜色）

不能作为平衡标志：
✗ 压强不变（因为Δn=0，压强始终不变）
✗ 密度不变（全气体，恒容时密度始终不变）
```

## 五、勒夏特列原理（平衡移动）

```
如果改变影响平衡的一个条件（浓度、温度、压强），
平衡将向着减弱这种改变的方向移动。

浓度：增加反应物 → 向正反应方向移动
温度：升高温度 → 向吸热方向移动
压强：增大压强 → 向气体分子数减少的方向移动
催化剂：同等程度改变正逆反应速率，不改变平衡位置
```

### 平衡移动 vs 平衡标志
```
平衡标志：判断是否"达到"平衡
平衡移动：判断平衡"向哪"移动

两者是不同的概念！
```

## 六、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 化学平衡常见陷阱                                       │
├─────────────────────────────────────────────────────────┤
│ E1. 认为平衡时反应停止了                                  │
│     错误：达到平衡后反应不再进行                         │
│     正确：v正 = v逆 ≠ 0，反应仍在进行                    │
│     只是正逆反应速率相等，宏观上浓度不变                 │
├─────────────────────────────────────────────────────────┤
│ E2. 用"浓度相等"作为平衡标志                              │
│     错误：各物质浓度相等说明达到平衡                     │
│     正确：平衡时浓度"不变"，不一定"相等"                 │
│     浓度是否相等取决于起始投料和转化率                   │
├─────────────────────────────────────────────────────────┤
│ E3. 用"速率比等于系数比"作为平衡标志                      │
│     错误：v(N₂):v(H₂):v(NH₃)=1:3:2说明平衡              │
│     正确：速率比等于系数比在任意时刻都成立               │
│     平衡标志必须是"正逆反应速率相等"                     │
├─────────────────────────────────────────────────────────┤
│ E4. 对Δn=0的反应用压强判断平衡                            │
│     错误：H₂+I₂⇌2HI中，压强不变说明平衡                 │
│     正确：此反应Δn=0，压强始终不变，不能作为平衡标志     │
├─────────────────────────────────────────────────────────┤
│ E5. 混淆"平衡移动"与"平衡标志"                          │
│     平衡移动：条件改变后，平衡位置发生移动               │
│     平衡标志：判断当前是否处于平衡状态                   │
│     催化剂能加快达到平衡，但不改变平衡位置               │
└─────────────────────────────────────────────────────────┘
```

## 七、关联概念网络

```
      可逆反应
         │
         ↓
    化学平衡状态
    /     |     \
   ↓      ↓      ↓
v正=v逆  浓度    条件改变
不变    不变     ↓
         \      /
          ↓    ↓
       勒夏特列原理
       （平衡移动）
```

---

## 核心记忆口诀

> **平衡不等停，v正等于v逆**
> **浓度不变是标志，不是浓度要相等**
> **正逆速率要同物，系数之比任意时**
> **勒夏特列减弱变，温度压强浓度移**
