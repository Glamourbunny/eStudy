# 官能团 vs 基团

> **底线地位**：官能团决定有机物的化学性质，是有机化学的核心概念。分不清两者会导致有机反应类型判断和方程式书写错误。

---

## 一、概念定义对比

| 维度 | 官能团 | 基团 |
|------|--------|------|
| **定义** | 决定有机化合物化学性质的原子或原子团 | 有机物分子中失去一个或多个氢原子后剩余的部分 |
| **范围** | 是基团的子集 | 包含官能团 |
| **功能** | **决定化学性质** | 可以是结构片段，不一定决定性质 |
| **例子** | —OH、—COOH、—CHO、C=C | —CH₃、—C₂H₅、—OH、—NO₂ |

---

## 二、核心区别：是否决定化学性质

```
官能团 ⊂ 基团

所有官能团都是基团
但不是所有基团都是官能团

关键区别：
• 官能团 → 决定一类化合物的特征化学性质
• 基团 → 更广义，可能只是结构片段

例：
—OH（羟基）→ 既是基团，也是官能团（决定醇/酚的性质）
—CH₃（甲基）→ 是基团，但不是官能团（不决定特征反应）
```

## 三、常见官能团速查表

| 官能团 | 名称 | 典型化合物 | 特征反应 |
|--------|------|-----------|----------|
| C=C | 碳碳双键 | 乙烯 | 加成、加聚、氧化 |
| C≡C | 碳碳三键 | 乙炔 | 加成、氧化 |
| —OH | 羟基 | 乙醇/苯酚 | 取代、氧化、酯化 |
| —CHO | 醛基 | 乙醛 | 银镜反应、斐林反应 |
| —COOH | 羧基 | 乙酸 | 酯化、酸性 |
| —COO— | 酯基 | 乙酸乙酯 | 水解 |
| —NH₂ | 氨基 | 苯胺 | 碱性 |
| —NO₂ | 硝基 | 硝基苯 | 还原为氨基 |
| —X | 卤素原子 | 氯乙烷 | 水解、消去 |
| —C≡N | 氰基 | 乙腈 | 水解 |

### 常见非官能团的基团

| 基团 | 名称 | 性质 |
|------|------|------|
| —CH₃ | 甲基 | 结构片段，不决定特征反应 |
| —C₂H₅ | 乙基 | 结构片段 |
| —C₆H₅ | 苯基 | 结构片段 |
| —R | 烃基 | 泛指烷基 |

## 四、典型例子

### 例1：羟基（—OH）在不同环境下的性质
```
醇羟基（R—OH）：
  与Na反应放出H₂
  可发生取代、消去、氧化
  不与NaOH反应

酚羟基（Ar—OH）：
  与Na反应放出H₂
  有弱酸性，与NaOH反应
  与FeCl₃显色
  可发生取代（邻对位定位）

羧基中的—OH（R—COOH）：
  有酸性（比碳酸强）
  可酯化

结论：同一个官能团，连接环境不同，性质不同！
```

### 例2：含多个官能团的化合物
```
CH₂=CH—COOH（丙烯酸）：
  含碳碳双键 → 加成、加聚反应
  含羧基 → 酸性、酯化反应

HO—CH₂—CHO（羟基乙醛）：
  含羟基 → 氧化、酯化
  含醛基 → 银镜反应

注意：多官能团化合物可同时发生多种反应
```

## 五、🔴 高频错误边界

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 官能团辨析常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 将烃基当作官能团                                      │
│     错误：甲基—CH₃是官能团                              │
│     正确：甲基是基团，不是官能团                         │
│     官能团必须是能决定特征化学性质的原子或原子团         │
├─────────────────────────────────────────────────────────┤
│ E2. 混淆醇羟基与酚羟基                                    │
│     醇—OH连在饱和碳上，酚—OH直接连在苯环上               │
│     性质差异：酚有酸性，醇无酸性                         │
│     酚与FeCl₃显色，醇不显色                              │
├─────────────────────────────────────────────────────────┤
│ E3. 醛基与酮羰基混淆                                      │
│     醛基：—CHO（碳原子一端连H，另一端连R）               │
│     酮基：R—CO—R'（碳原子两端都连R基）                  │
│     性质差异：醛能发生银镜反应，酮不能                   │
├─────────────────────────────────────────────────────────┤
│ E4. 酯基与羧基混淆                                        │
│     羧基：—COOH（有酸性，可电离出H⁺）                   │
│     酯基：—COO—（无酸性，不能电离）                      │
│     酯基可水解，羧基不能水解                             │
├─────────────────────────────────────────────────────────┤
│ E5. 多官能团反应时遗漏部分官能团                          │
│     例：HOOC—CH=CH—COOH与Na反应                         │
│     只考虑羧基？不对！要考虑所有活泼H                     │
│     2个—COOH都能与Na反应，共放出1mol H₂（每2个活泼H）   │
└─────────────────────────────────────────────────────────┘
```
