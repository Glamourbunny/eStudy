## 二、论证结构模式（输入端→本论→输出端）

### 模式一：总分式
```
引论（提出论点）
  ↓
本论（分论点1 + 论据）→（分论点2 + 论据）→（分论点3 + 论据）
  ↓
结论（总结/升华）

识别标志：首段亮明观点，后文分层论证
```

### 模式二：递进式
```
是什么 → 为什么 → 怎么样
  ↓        ↓        ↓
定义    原因分析   对策/影响

识别标志：层层深入，不可调换顺序
```

### 模式三：对照式
```
正面论述 + 反面论证 → 总结观点

识别标志："然而""相反""与之对比"等转折词
```

## 三、论证方法速查表

| 方法 | 特征标志词 | 答题模板 |
|------|-----------|---------|
| **举例论证** | "例如""比如""如" | 列举……事例，具体有力地论证了…… |
| **引用论证** | "某某说""古人云" | 引用……名言，增强说服力/权威性 |
| **对比论证** | "与之相比""相反" | 将……与……对比，突出…… |
| **比喻论证** | "就像""仿佛" | 将……比作……，生动形象地阐明…… |
| **因果论证** | "因此""因为" | 分析……原因，揭示……内在逻辑 |
| **归谬论证** | "假设……那么" | 先假设对方观点成立，推出荒谬结论 |

## 四、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 论述类文本常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆"论证结构"与"论证方法"                          │
│     结构：总分/递进/对照（整体框架）                     │
│     方法：举例/引用/对比（具体手法）                     │
├─────────────────────────────────────────────────────────┤
│ E2. 认为"引用"都是为了"增强说服力"                      │
│     有时引用是为了：引出话题、作为批驳靶子、增强文采     │
│     需结合上下文判断引用的具体作用                       │
├─────────────────────────────────────────────────────────┤
│ E3. 选项设置常见偷换概念                                 │
│     原文："部分学者认为"                                 │
│     选项："学界普遍认为"（扩大范围）                     │
│     策略：逐字比对，警惕绝对化表述                       │
├─────────────────────────────────────────────────────────┤
│ E4. 忽视论证语言的限定词                                 │
│     "往往""通常""在一定程度上"≠"一定""总是""完全"     │
│     去掉限定词 = 改变原意 = 错误选项                     │
├─────────────────────────────────────────────────────────┤
│ E5. 逻辑关系颠倒                                         │
│     原文：A导致B                                         │
│     选项：B导致A（因果倒置）                             │
│     原文：A是B的充分条件                                 │
│     选项：A是B的必要条件（条件混淆）                     │
└─────────────────────────────────────────────────────────┘
```
