# 议论文「五段三分」基础范式图

> **模块定位**：高考作文（60分）的保底结构。时间紧张时，用五段三分确保文章有完整框架，不偏题、不混乱。

---

## 一、核心结构图

```
议论文五段三分结构

┌─────────────────────────────────────────────────────────┐
│  第一段：引论（80-100字）                                │
│  • 材料引入（一句话概括材料）                            │
│  • 观点提出（旗帜鲜明，一句话）                          │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  第二段：本论1（150-180字）                              │
│  • 分论点1（一句话，独立成段或段首）                     │
│  • 论据1（事例/道理/引用）                               │
│  • 分析（为什么这个论据支持分论点）                      │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  第三段：本论2（150-180字）                              │
│  • 分论点2（与分论点1并列或递进）                        │
│  • 论据2（不同类型，避免重复）                           │
│  • 分析（回扣分论点）                                    │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  第四段：本论3（150-180字）                              │
│  • 分论点3（可从"怎么办""反面论证"角度）                │
│  • 论据3                                               │
│  • 分析                                                │
└─────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────┐
│  第五段：结论（80-100字）                                │
│  • 总结三个分论点                                        │
│  • 升华主题（联系时代/青年/国家）                        │
│  • 呼告/倡议（"让我们……""愿我们……"）                  │
└─────────────────────────────────────────────────────────┘
```

## 二、分论点设计模板

### 并列式（三个角度平行展开）
```
中心论点：青年应当有担当精神

分论点1：担当是对个人责任的坚守（个人层面）
分论点2：担当是对家庭义务的承担（家庭层面）
分论点3：担当是对国家使命的践行（国家层面）

关键词：个人→家庭→国家（由小到大）
```

### 递进式（层层深入）
```
中心论点：读书使人进步

分论点1：读书可以增长知识（是什么）
分论点2：读书可以开阔视野（为什么）
分论点3：读书应当学以致用（怎么做）

关键词：是什么→为什么→怎么做
```

### 对照式（正反对比）
```
中心论点：诚信是立身之本

分论点1：诚信者，人恒敬之（正面论证）
分论点2：失信者，寸步难行（反面论证）
分论点3：当代青年更应以诚信为立身之本（怎么办）
```

## 三、论据选择原则

| 类型 | 举例 | 使用注意 |
|------|------|----------|
| **事例论据** | 袁隆平、钟南山、航天团队 | 一例一析，避免堆砌 |
| **引用论据** | 名言、古诗文、领导人讲话 | 准确无误，不张冠李戴 |
| **道理论据** | 哲学原理、科学规律 | 通俗易懂，不故弄玄虚 |
| **对比论据** | 古今对比、中外对比 | 对比点要明确 |

**论据搭配建议**：
```
每段一个核心论据 + 简要分析
三段论据尽量不重复同类型
（例：一段人物事例 + 一段名言引用 + 一段反面论证）
```

## 四、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 议论文写作常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 分论点之间交叉重叠                                   │
│     错误：分论点1谈"坚持需要毅力"，分论点2谈"坚持需要   │
│     毅力支撑"（内容重复）                                │
│     正确：三个分论点互相独立，覆盖不同角度               │
├─────────────────────────────────────────────────────────┤
│ E2. 只有论据没有分析                                     │
│     错误：举完例子就结束，不解释例子如何支持论点         │
│     正确：每个论据后必须有"正是因为……所以……"的分析   │
├─────────────────────────────────────────────────────────┤
│ E3. 观点在材料中兜圈子，不展开                           │
│     错误：全文反复重述材料，没有自己的论证               │
│     正确：用材料引入，迅速提出观点，然后用独立论据论证   │
├─────────────────────────────────────────────────────────┤
│ E4. 分论点表述模糊，缺乏判断词                           │
│     错误："关于诚信的看法"（不是观点，是话题）           │
│     正确："诚信是立身之本"（有判断，有态度）            │
├─────────────────────────────────────────────────────────┤
│ E5. 结尾空泛喊口号，没有回扣分论点                       │
│     错误："让我们共同努力吧！"（无具体内容）             │
│     正确："以坚守为基，以创新为翼，以担当为责……"       │
│     （回扣三个分论点的关键词）                           │
└─────────────────────────────────────────────────────────┘
```

## 六、自检清单

- [ ] 10分钟内能列出一个作文题的五段三分提纲
- [ ] 三个分论点互相独立、不交叉
- [ ] 每个分论点是一个完整判断句（有态度）
- [ ] 每个论据后都有分析句
- [ ] 结尾回扣三个分论点关键词，不空泛
