## 三、宝黛钗三角关系与主题（输出端）

```
         「木石前盟」          「金玉良缘」
         贾宝玉 + 林黛玉      贾宝玉 + 薛宝钗
              ↓                    ↓
         精神契合              世俗匹配
         真情                  礼教
              ↓                    ↓
         悲剧（黛玉之死）     悲剧（宝玉出家）
              \                  /
               ↓                ↓
            共同主题：封建礼教对真情的压制
```

### 核心主题归纳
```
主题一：反封建
  • 宝玉拒绝走仕途经济之路
  • 黛玉不劝宝玉读书科举
  • 对科举制度、等级制度的批判

主题二：女性命运
  • "千红一哭，万艳同悲"
  • 金陵十二钗各有悲剧
  • 有才情有志向的女性无法自主命运

主题三：理想与现实的冲突
  • 大观园是理想世界
  • 大观园最终被抄、人散
  • 理想世界在现实中的必然破灭

主题四：盛衰无常
  • 贾府从烈火烹油到忽喇喇大厦倾
  • "好一似食尽鸟投林，落了片白茫茫大地真干净"
```

## 四、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 红楼梦答题常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 将宝钗简单定义为"反面人物"                           │
│     错误：宝钗是坏人，黛玉是好人                         │
│     正确：宝钗是封建秩序的维护者，有其合理性；           │
│           作者对她既有同情也有批判                       │
│     人物是复杂的，避免二元对立                           │
├─────────────────────────────────────────────────────────┤
│ E2. 认为宝玉是"花花公子"而忽视其叛逆性                  │
│     正确理解：宝玉的"多情"是对女儿的尊重与同情，        │
│               其核心是反封建、反科举                     │
├─────────────────────────────────────────────────────────┤
│ E3. 混淆人物关系（尤其是姻亲关系）                       │
│     王熙凤是贾琏的妻子（王夫人的侄女）                   │
│     薛宝钗是薛姨妈的女儿（王夫人的外甥女）               │
│     两人都与王家有亲，但身份不同                         │
├─────────────────────────────────────────────────────────┤
│ E4. 答题时只复述情节不分析主题                           │
│     错误："林黛玉葬花了，她很伤心"                      │
│     正确："葬花象征着黛玉对自身命运的预感，             │
│           花的凋零暗喻美好事物在封建礼教下的毁灭"       │
├─────────────────────────────────────────────────────────┤
│ E5. 忽视判词和伏笔的作用                                 │
│     每回判词都预示人物命运                               │
│     答题时引用判词可增强说服力                           │
│     例：黛玉判词"玉带林中挂"暗示其悲剧结局              │
└─────────────────────────────────────────────────────────┘
```
