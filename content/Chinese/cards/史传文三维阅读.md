# 史传文「三维阅读」结构图

> **模块定位**：高考文言文阅读核心方法。史传文有固定叙事模式，三维法可快速定位信息，避免在陌生文本中迷失。

---

## 一、核心结构图

```
史传文阅读
    │
    ├── 时间轴 ─────→ 事件先后顺序
    │                  起因 → 经过 → 结果
    │
    ├── 人物关系网 ──→ 冲突 / 合作 / 从属
    │                  谁与谁？什么关系？
    │
    └── 史家评价 ────→ 作者态度与立场
                       如何评价？有何用意？
```

---

## 二、三维操作步骤

### 第一步：画时间轴（输入端）
```
阅读时标记时间词：
• 年号词："初""顷之""久之""后"
• 年龄词："年方十五""既冠"
• 事件词："及""会""时"

快速理清：这个人一生经历了哪几个阶段？
每个阶段发生了什么事？
```

### 第二步：画人物关系网（工具箱）
```
核心问题：谁与谁有什么关系？

常见关系：
┌──────────┬─────────────────────────────┐
│ 关系类型 │ 标志词/常见表述               │
├──────────┼─────────────────────────────┤
│ 君-臣    │ 上、帝、朝廷 / 拜、除、迁     │
│ 父-子    │ 父、祖、子、孙 / 嗣、袭       │
│ 师-生    │ 师、从、受业、门下           │
│ 友-敌    │ 与……善、交好 / 忌、恶、谮    │
│ 同僚     │ 同、俱、并、共              │
└──────────┴─────────────────────────────┘
```

### 第三步：抓史家评价（输出端）
```
史家评价常见位置：
• 文末"论曰""赞曰""太史公曰"
• 传末"有司奏"
• 文中褒贬用词（春秋笔法）

评价角度：
• 品德：忠/奸、仁/暴、廉/贪
• 才能：贤/愚、能/庸、智/拙
• 功过：功/过、是/非、当/不当
```

## 三、史传文必记要素

| 要素 | 内容 | 答题作用 |
|------|------|----------|
| **叙事手法** | 顺叙/倒叙/插叙 | 分析结构特点 |
| **人物刻画** | 语言/动作/细节/侧面烘托 | 分析人物形象 |
| **史家笔法** | 春秋笔法、寓论断于叙事 | 理解作者立场 |
| **文化常识** | 官职、礼制、地理、科举 | 准确理解文意 |

## 四、高频陷阱（E1-E5）

```
┌─────────────────────────────────────────────────────────┐
│ 🔴 史传文阅读常见陷阱                                     │
├─────────────────────────────────────────────────────────┤
│ E1. 混淆传主与次要人物的行为                             │
│     选项把别人的事安到传主头上                           │
│     策略：每遇到人物行为，问"这是谁做的？"               │
├─────────────────────────────────────────────────────────┤
│ E2. 忽视时间顺序导致事件混淆                             │
│     先发生的事被选项说成后发生的结果                     │
│     策略：边读边标时间词，画出简易时间轴                 │
├─────────────────────────────────────────────────────────┤
│ E3. 把"皇帝/他人评价"当作"史实"                        │
│     原文："帝曰：卿真忠臣也"                            │
│     这是皇帝的评价，不等同于传主确实忠诚                 │
│     需区分：事实陈述 vs 评价性语言                       │
├─────────────────────────────────────────────────────────┤
│ E4. 官职变动词理解错误                                   │
│     迁：调动（可升可降）≠ 一定是升迁                     │
│     除：授官 ≠ 除去官职                                  │
│     谪：贬官 ≠ 任命                                      │
│     致仕：退休 ≠ 被罢免                                  │
├─────────────────────────────────────────────────────────┤
│ E5. 忽视"省略主语"导致理解偏差                          │
│     史传文大量省略主语，需根据上下文补出                 │
│     策略：每读一句，先问"谁？做了什么？"                 │
└─────────────────────────────────────────────────────────┘
```
