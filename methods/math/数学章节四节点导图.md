# 数学章节四节点导图

用四节点强制结构梳理每个章节，确保"输入-工具-输出-易错"四个维度无死角。

---

## 一、为什么需要四节点结构

普通导图的问题：
- **节点堆砌** → 罗列大量知识点，看不出重点
- **缺乏结构** → 知识点之间关系混乱
- **无迭代方向** → 不知道学完这章能干什么

四节点导图的核心价值：

| 节点 | 回答的问题 | 价值 |
| --- | --- | --- |
| **输入端** | 这章研究什么核心问题？ | 明确学习目标 |
| **工具箱** | 这章提供了什么定理/公式/方法？ | 积累解题武器 |
| **输出端** | 这章能解决什么问题？接口在哪？ | 建立知识迁移 |
| **易错点** | 最易混淆的3个概念/3类计算陷阱 | 精准避坑 |

---

## 二、四节点模板

```markdown
# 第X章 · [章节名称]

## 输入端：本章研究的核心问题

【核心问题】（用一句话描述）
例：平面向量 → 平面中既有大小又有方向的量，如何进行运算？

【问题背景】
- 从哪里来？（前置知识）
- 往哪里去？（为哪类问题服务）

---

## 工具箱：本章提供的工具

### 定理/公式

| 名称 | 表达式 | 适用条件 |
| --- | --- | --- |
| 例：向量数量积 | a·b = |a||b|cosθ | 两向量夹角已知 |
| 例：余弦定理 | c² = a² + b² - 2ab·cosC | 已知两边及夹角 |

### 方法清单

【方法1】向量法解几何问题
- 识别信号：________________
- 操作步骤：________________

【方法2】坐标化法
- 识别信号：________________
- 操作步骤：________________

---

## 输出端：本章能解决的问题

### 典型问题类型

1. **问题类型1**：________________
   - 解题路径：________________

2. **问题类型2**：________________
   - 解题路径：________________

### 跨章节接口

| 对接章节 | 接口点 |
| --- | --- |
| 例：解三角形 | 边长求角度 |
| 例：解析几何 | 向量坐标化 |

---

## 易错点：本章最容易犯的错

### 概念混淆（3个）

| 易混淆概念 | 正确区分 |
| --- | --- |
| 例：向量平行 vs 直线平行 | 直线平行无方向，向量平行同向或反向 |
| 例：数量积 vs 叉积 | 数量积结果为数，叉积结果为向量 |

### 计算陷阱（3类）

| 陷阱类型 | 避坑方法 |
| --- | --- |
| 例：忽视零向量 | 讨论时单独考虑零向量情况 |
| 例：夹角范围遗漏 | 明确两向量夹角范围是 [0°, 180°] |

## 三、执行标准

### 最低要求（30分钟/章节）

- 输入端：1句话描述核心问题
- 工具箱：列举3-5个核心公式/定理
- 输出端：列出2-3个典型应用
- 易错点：写出2个最常见错误

### 高质量版本（60分钟/章节）

- 输入端：追溯问题起源，说明与前后章节的关系
- 工具箱：每个工具配1道典型例题
- 输出端：绘制与其他章节的接口图
- 易错点：每个易错点配1道易错题

## 四、四节点导图示例

以"平面向量"章节为例：

```markdown
# 第五章 · 平面向量

## 输入端：本章研究的核心问题

【核心问题】平面中的有向线段（向量），如何进行加减运算、数乘运算、数量积运算？

【问题背景】
- 来：从物理中的力、速度抽象而来
- 去：统一几何与代数，为解析几何、力学奠基

---

## 工具箱：本章提供的工具

### 核心公式

| 名称 | 表达式 | 适用条件 |
| --- | --- | --- |
| 向量加法 | a + b = (a₁+b₁, a₂+b₂) | 所有向量 |
| 数量积 | a·b = a₁b₁ + a₂b₂ | 两向量坐标已知 |
| 夹角公式 | cosθ = a·b/(|a||b|) | 求两向量夹角 |

### 方法清单

【方法1】向量法证明几何关系
- 识别信号：证明平行、垂直、等长
- 操作：建立向量，用运算律证明

【方法2】坐标化法求最值
- 识别信号：几何问题含参数
- 操作：建系 → 坐标化 → 运算求解

---

## 输出端：本章能解决的问题

### 典型问题

1. 证明菱形对角线互相垂直
   - 路径：建系 → 设坐标 → 计算点积 → 判定垂直

2. 求解三角形面积最值问题
   - 路径：坐标化 → 用数量积求边长 → 面积公式

### 跨章节接口

| 对接章节 | 接口点 |
| --- | --- |
| 三角函数 | 向量与三角函数结合 |
| 解析几何 | 向量作为解析几何的工具 |
| 立体几何 | 空间向量是平面向量推广 |

---

## 易错点：本章最容易犯的错

### 概念混淆（3个）

| 易混淆概念 | 正确区分 |
| --- | --- |
| 向量平行 vs 直线平行 | 向量平行考虑方向（共线），直线平行不考虑方向 |
| 数量积 vs 模 | 数量积结果为数，模是向量的大小 |
| 向量方程 vs 直线方程 | 向量方程描述点的轨迹，直线方程描述坐标关系 |

### 计算陷阱（3类）

| 陷阱类型 | 避坑方法 |
| --- | --- |
| 零向量未单独讨论 | 讨论垂直/平行时，先排除零向量 |
| 夹角余弦值正负号 | 根据向量方向判断夹角范围 |
| 坐标运算符号错误 | 强调：b₁-a₁，不是a₁-b₁ |
```

## 五、常见问题

**Q: 什么时候做导图最合适？**
A: 学完每章后24小时内完成，趁记忆鲜明时梳理结构。

**Q: 四节点导图和普通导图有什么区别？**
A: 普通导图是发散的信息罗列；四节点导图是收敛的结构化输出，每个节点回答一个明确问题。

**Q: 时间紧张时，哪些节点可以简化？**
A: 优先保证"工具箱"和"易错点"，这两项对解题最有直接价值。
