# 「母题-变式-错因」三维错题网络

放弃按"日期"或"章节"线性整理错题，改为网络式归档，实现从"错一道题"到"堵一类漏洞"的跃迁。

---

## 一、为什么要网络式归档

传统错题本的致命缺陷：
- **线性存储** → 按时间排列，三个月后翻不到
- **单题孤立** → 不标注与其他题的关系，同源变式散落各处
- **归因粗糙** → 写"粗心""不会"等宽泛标签，无指导意义

三维错题网络的核心改变：
- **按错因聚类** → 发现系统性漏洞，而非零散失误
- **标注知识坐标** → 任意维度检索，跨章节追踪
- **链接母题变式** → 形成题族，触类旁通

## 二、错题卡片模板

每道错题归档为一张标准化卡片：

```markdown
---
name: 错题卡片
id: [母题编号]
date: [归档日期]
---

## 基础信息

【知识坐标】
- X轴（对象）：[数/式/函数/方程/不等式/几何体/随机变量]
- Y轴（结构）：[概念/定理/运算/关系/思想方法]
- Z轴（能力）：[直观感知/逻辑推理/建模/运算/分析]

【错因分类】
- [ ] 概念误用
- [ ] 公式记错
- [ ] 计算失误
- [ ] 思路盲区
- [ ] 审题偏差
- [ ] 其他：________

## 题目信息

【题干】（精简到50字内，保留核心结构）

【关键条件】（导致错误/解题的关键条件，标注出来）

## 错因分析

【错因细化】（具体到哪一步、哪个符号、哪个隐含条件）
例：错误地认为"两向量平行则点积为零"，混淆了点积与叉积的判定条件。

【正确思路】（用流程图或关键节点表示，非完整抄写）
例：判定向量平行 → 存在 λ 使得 a = λb → 交叉相乘相等（或利用坐标）

## 关联信息

【变式联想】
- 与哪道已做题同源？
- 可推广到什么情形？
- 能否出一道反向题？

【回归教材】
对应教材：必修一 · 第__页 · 例__ / 习题__
对应三层卡片：概念层-第__节 / 命题层-第__节

【下次复查】
复查日期：___________
是否已堵漏：□是 □否 堵漏方式：___________
```

## 三、每周执行动作

### 周五：归档

整理本周错题，填写标准化卡片，重点：
1. 精炼题干（50字内，保留核心结构）
2. 精准定位错因（具体到哪一步，不写"粗心"）
3. 标注知识坐标（X/Y/Z三轴）

### 周日：横向扫描

将本周错题按"错因"聚类，发现系统性漏洞：

**扫描清单：**
- 哪类错因出现频率最高？（如：计算失误频发 → 指向运算规则不熟）
- 哪个Y轴结构出错最多？（如：定理适用边界混淆）
- 是否存在跨章节同源题？（如：三角函数求最值连续出错，可能影响向量/立体几何）

**行动输出：**
对高频错因，回溯到对应章节的**概念层卡片**，在"概念辨析"处补充常见误区别。

### 月末：纵向穿透

对整月错题进行深度复盘：
1. 统计错因分布柱状图
2. 标注"已堵漏"与"待解决"
3. 对反复出错的知识点，单独制作专项训练

## 四、常见问题

**Q: 错题太多来不及整理怎么办？**
A: 优先处理高频考点错题（函数/导数/三角/向量/圆锥曲线/概率）。其他错题先拍照存档，周末批量处理。

**Q: 怎么判断一道题要不要归档？**
A: 问自己："这道题下次出现，我能保证做对吗？" 不能 → 归档。不能确定 → 先标星号，复习后再决定。

**Q: 错因"思路盲区"具体指什么？**
A: 拿到题不知道往哪个方向想。典型表现：解题思路与标准答案完全不同，或完全无法下手。修复方式：回到对应章节的方法层卡片，强化识别信号。
