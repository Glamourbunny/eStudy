# 「母题-变式-错因」三维错题网络

语文错题往往不是"知识点错误"，而是"思维路径偏差"。按"母题-变式-错因"三维归档错题，建立网络式定位。

## 定位

解决"同一道题换了个问法就不会"、"答题答不到点上"的问题。中策帮助学生发现思维路径的断点，而非泛泛归因于"没看懂文章"或"粗心"。

## 核心框架

三维错题网络：

| 维度 | 内容 | 解决的问题 |
|------|------|------------|
| 母题 | 这道题属于什么题型？ | 识别问题本质 |
| 变式 | 这个问题有哪些问法？ | 识别同源变式 |
| 错因 | 思维断点在哪里？ | 精准定位问题 |

## 具体操作步骤

### 第一步：建立错题卡片

每道错题归档为标准化卡片：

```markdown
【文本坐标】
- X（文本类型）：□论述类 □实用类 □小说 □散文 □戏剧 □史传文 □古诗词
- Y（能力维度）：□识记 □理解 □分析 □鉴赏 □评价 □表达
- Z（素养维度）：□语言建构 □思维发展 □审美鉴赏 □文化传承

【错因分类】
□ 文本误读（把作者反对的观点当成支持的观点）
□ 论证识别错误（分不清"举例论证"和"类比论证"）
□ 逻辑跳跃（把必要条件当成充分条件）
□ 鉴赏术语混淆（把"借景抒情"写成"托物言志"）
□ 答题结构缺失（只写手法不写效果）
□ 文言实词迁移错误（用现代汉语意思套古文）

【正解思维链】
用"因为…所以…"写出推理步骤

【回归教材】
该题型对应教材哪篇课文？
如：论述类论证分析 → 《反对党八股》

【素材迁移】
该文本可用于作文的哪个主题？
```

### 第二步：识别母题与变式

**常见母题类型**：

| 母题 | 典型问法 |
|------|----------|
| 论证分析 | "论证特点"/"论证方法"/"论证思路" |
| 人物形象 | "人物特点"/"人物刻画方法"/"人物作用" |
| 景物描写 | "景物特点"/"描写手法"/"意境" |
| 词句赏析 | "表达效果"/"词语妙处"/"句子含义" |
| 文言实词 | "实词释义"/"词类活用"/"古今异义" |

**变式识别**：
- "用了什么手法" → 可能问比喻、对比、象征等
- "有什么作用" → 可能问结构作用、内容作用、表达效果
- "如何理解这句话" → 可能问句子含义、人物形象、主题思想

### 第三步：定位错因

**常见思维断点**：

| 错因类型 | 具体表现 | 修复方法 |
|----------|----------|----------|
| 文本误读 | 曲解作者意图 | 抓首段主题句+末尾总结句 |
| 论证识别错误 | 混淆论证方法 | 背诵每种论证方法的定义和特征 |
| 逻辑跳跃 | 答题缺漏环节 | 建立"答此类题的标准结构" |
| 鉴赏术语混淆 | 术语使用不当 | 整理易混术语对比表 |
| 答题结构缺失 | 点不全/层次乱 | 按分值估算答题点数 |

### 第四步：回归教材，溯源理解

每次错题后追问：
- 这道题在教材中有没有原型？
- 教材的课后习题是否出现过类似问法？
- 教师的课堂分析有没有涉及这个角度？

## 执行频率建议

- **周五**：归档本周阅读错题和作文批改反馈，重点标注"思维断点"
- **周日**：横向比较——把"论述类文本论证题"和"文言文信息筛选题"对比，发现共同方法论
- **月末**：对高频错因（如"论证方法判断错误"），专项重读教材经典篇目，标注所有论证手法

## 与其他学科的迁移

- **英语**：阅读理解的"定位-同义替换-判断"与语文的"文本-解读-答题"流程类似
- **物理**：受力分析的"漏力/多力"与语文的"遗漏要点"类似
- **数学**：解题步骤的"关键节点"与语文的"答题要点"类似

## 检验标准

- 错题本中每道题都能找到明确的"思维断点"
- 能区分"同一道母题"的不同变式问法
- 同一母题的变式题正确率达到80%以上
