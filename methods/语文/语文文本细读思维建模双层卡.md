# 「文本细读+思维建模」双层卡片法

对每一篇精读课文或每一道典型阅读题，建立"文本细读层+思维建模层"双层卡片，实现从"读懂文章"到"会用文章"的核心跃迁。

## 定位

解决"读了很多文章但不会做阅读题"、"作文没有思路"的问题。上策帮助学生建立文本分析与思维建模的双轨能力，适合必修下册的史传散文、诸子论说、戏剧小说等复杂文本。

## 核心框架

| 层次 | 核心任务 | 回答的问题 |
|------|----------|------------|
| 文本细读层 | 输入端 | 这篇文章写了什么？写得怎么样？ |
| 思维建模层 | 输出端 | 这篇文章考什么？怎么答？怎么写？ |

## 具体操作步骤

### 第一层：文本细读层（输入端）

**1. 文本类型标签**：
□论述类 □实用类 □小说 □散文 □戏剧 □史传文 □论说文 □古诗词

**2. 核心问题意识**：
- 作者要解决什么问题？
- 核心论点/情感/冲突是什么？

**3. 结构解剖图**：

*论述类文本*：
```
论点句位置 → 论据类型（事实/道理/比喻/对比）
    ↓
论证方法（举例/引用/对比/因果/归谬）
```

*史传文*：
```
事件链（起因→经过→结果）
    ↓
人物刻画手法（语言/动作/细节/侧面）
    ↓
史家评价（论赞）
```

*小说*：
```
情节链（开端→发展→高潮→结局）
    ↓
人物弧光（变化/成长/堕落）
    ↓
环境象征意义
```

**4. 语言炼金术**：
- 标记3处最具表现力的词句
- 写出"如果换成另一种表达，效果损失在哪里"

### 第二层：思维建模层（输出端）

**1. 题型映射**：
该文本对应高考哪类题型？
- 内容理解 / 论证分析 / 鉴赏评价 / 探究应用

**2. 答题范式**：
该题型的标准思考路径是什么？

例：论证特点题 = 论证结构 + 论证方法 + 语言风格

**3. 写作迁移**：
- 该文本的哪个论点可以用于作文？
- 哪个素材可以迁移？
- 哪种论证方式可以借鉴？

### 高一下重点卡片示例（史传文）

```
【卡片：《鸿门宴》《烛之武退秦师》《谏太宗十思疏》】

文本类型：史传文/论说文

核心问题：如何在权力博弈中说服对方？
- 烛之武：利益诱导
- 魏征：正反对比+归谬

结构解剖：
- 叙事线：背景→发展→高潮（说辞三层）→结局
- 人物论：烛之武vs子犯vs郑伯
- 史家笔法：以"亦去之"暗示晋侯无信

语言炼金术：
- "越国以鄙远"——委婉否定
- "若不阙秦"——归谬法

思维建模：
- 题型：史传文"人物评价题" = 事件筛选 + 言行分析 + 史家态度 + 历史影响
- 写作迁移：烛之武→"小人物大担当"/"语言的力量"/"理性博弈"
```

## 执行频率建议

- 每学完一篇精读课文制作一张完整卡片
- 考前复习：对着卡片回忆文章的论证结构和写作手法
- 每周至少完成1张新卡片

## 与其他学科的迁移

- **英语**：议论文的论点-论据-论证结构与语文相通
- **历史**：史传文的史料分析与历史学科的历史叙事能力相关
- **政治**：议论文的逻辑论证与政治学科的辩证思维能力相通

## 检验标准

- 给一篇陌生文本，能快速判断其类型并说出阅读策略
- 能完整分析某篇文章的论证结构或叙事结构
- 能将精读课文的素材迁移到作文写作中
