# 「词汇网络+语块提取」双轨卡片法

对每一个核心词汇，建立"语义网络+语用场景"双轨卡片，实现从"背单词"到"用词汇网络"的核心跃迁。

## 定位

解决"背了单词却不会用"、"熟词生义完全不认识"的困境。上策帮助学生建立词汇的立体网络，而非线性列表。适合高一3500词扩充期的核心方法。

## 核心框架

| 轨道 | 内容 | 解决的问题 |
|------|------|------------|
| 词汇网络轨 | 一词多义、搭配、派生词 | 这个词有哪些意思？ |
| 语块提取轨 | 语块搭配、语境功能、输出复现 | 这个词怎么用在句子里？ |

## 具体操作步骤

### 第一轨：形式-语义网络

**核心词**：选定课标高频词或常见熟词生义词

**语义网构建**：
```
record（n./v.）
├── n. 记录 → keep a record / medical records
├── n. 唱片 → play a record
├── n. 履历 → school record
├── v. 记录 → record data
├── v. 录音 → record a song
└── v. 显示/标明 → The thermometer recorded 30℃
```

**搭配链**：
- medical records（病历）
- break a record（打破纪录）
- record high/low（创新高/新低）
- off the record（非正式地）

**派生族**：
- recorder（n. 录音机；记录员）
- recording（n. 录音）
- recordable（adj. 可记录的）

### 第二轨：语块与语境功能

**语块提取**（从课文/真题中摘取，而非孤立记词）：

```
"It is recorded that…"（据记载——用于说明文/议论文引入事实）
"set/put the record straight"（澄清事实——用于辩论/应用文）
"on record"（记录在案的）
```

**近义辨析网**：

| 词 | 语境 | 例句 |
|-----|------|------|
| record | 记录数据/事件 | The police recorded the accident. |
| document | 正式文件记录 | The contract was documented. |
| register | 登记/注册 | Please register for the course. |
| file | 归档/存档 | File these documents. |

**输出激活**：
- 用该词的不同义项各造一个句子
- 确保至少一个用于写作、一个用于阅读辨识
- 将词汇归入高考高频话题库（校园/科技/文化/社会）

## 执行频率建议

- 高一下学期完成必修一至必修三核心词汇的全部双轨卡片（约800-1000词）
- 每天学习5个新词，每个词花5分钟建立双轨网络
- 每周复习上周词汇，检测语义网络是否清晰

## 与其他学科的迁移

- **语文**：文言文的一词多义、古今异义与英语的熟词生义思维类似
- **生物**：DNA碱基配对（A-T/G-C）与英语的搭配关系（collocation）概念相通
- **历史**：历史事件的因果网络与词汇的语义网络构建方式类似

## 检验标准

- 给一个核心词，能在2分钟内画出完整的语义网络图
- 阅读中遇到熟词生义，能快速激活正确义项
- 写作中能主动使用课标词汇，而非只会用初中词汇
