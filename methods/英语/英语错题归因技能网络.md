# 「错题归因」技能网络法

英语错题的本质往往不是"知识点不会"，而是"微技能缺失"。按"语言微技能"归因，建立网络式错题本。

## 定位

解决"做了很多阅读还是错很多"、"语法题反复错同一类型"的问题。中策帮助学生精准定位能力漏洞，而非泛泛归因于"粗心"或"不会"。

## 核心框架

六维错因归类体系：

| 维度 | 典型问题 |
|------|----------|
| 词汇层 | 熟词生义误判、搭配错误 |
| 语法层 | 长难句结构拆解错误、主干/从句边界混淆 |
| 语篇层 | 信息定位偏差、被干扰项吸引 |
| 逻辑层 | 推理过度/不足、事实与观点混淆 |
| 文化层 | 背景知识缺失（西方教育/社会常识） |
| 输出层 | 中式英语、搭配错误、逻辑连接词误用 |

## 具体操作步骤

### 第一步：建立错题卡片

每道错题归档为标准化卡片：

```markdown
【文本坐标】
- X（文本类型）：□说明文 □议论文 □记叙文 □应用文 □七选五
- Y（技能维度）：□细节理解 □推断 □词义猜测 □主旨 □写作意图
- Z（能力层级）：□识别 □理解 □应用 □分析 □评价

【错因分类】
□ 词汇层：熟词生义误判（具体：_____）
□ 语法层：长难句结构拆解错误（主干/从句边界/修饰对象混淆）
□ 语篇层：信息定位偏差（被干扰项的局部同义吸引，忽略全局否定/转折）
□ 逻辑层：推理过度/不足（把文中事实当成作者观点，或反之）
□ 文化层：背景知识缺失（西方教育体制/社会习俗）
□ 输出层：中式英语/搭配错误/逻辑连接词误用

【正解思维链】
写出从题干到原文定位再到选项判断的完整路径

【语块收获】
从本题文本中提取3个可用于写作或再认的语块

【回归教材】
该语法点对应必修几第几单元？课文原句是什么？
```

### 第二步：六维标签标注

对每道错题按六维标注，重点识别高频错因：

```
如果"词汇层"错题占比 > 30% → 专项复习熟词生义
如果"语法层"错题占比 > 30% → 每日一句长难句分析
如果"语篇层"错题占比 > 30% → 专项训练信息定位
```

### 第三步：横向扫描与专项突破

**每周扫描清单**：
- 哪类错因出现频率最高？
- 哪个维度最弱？
- 是否存在跨题型的同类错误？

**专项训练对应**：
- 长难句拆解错误 → 每日一句分析（找主干→标从句→译语义）
- 信息定位偏差 → 首段抓主题+题干关键词回文定位
- 推理过度/不足 → 建立"事实vs观点"区分意识

### 第四步：回归语言能力本质

每次错题归档后，问自己：
- 这道题暴露了哪个语言微技能的缺陷？
- 这个技能是否在其他题型中也存在问题？
- 需要回归教材的哪一部分来修复？

## 执行频率建议

- **周五**：归档本周错题，按六维贴标签
- **周日**：横向扫描——发现高频错因，下周专项突破
- **月末**：统计错因分布柱状图，标注"已解决"与"待解决"

## 与其他学科的迁移

- **语文**：阅读理解的"句子作用分析"与英语的长难句分析相通
- **物理**：受力分析的"漏力/多力"与英语的"遗漏信息/过度推理"类似
- **数学**：解题步骤的"关键节点"与英语阅读的"定位-同义替换-判断"流程类似

## 检验标准

- 错题本中每道题都能定位到具体哪个语言微技能
- 每周的错因统计能反映出明确的进步或退步趋势
- 同类错误连续三周不重复出现
