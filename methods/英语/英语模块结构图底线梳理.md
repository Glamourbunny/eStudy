# 「模块结构图」底线梳理

若时间有限，必须手绘并熟记4张核心结构图。这些是英语考试的高频考点框架。

## 定位

时间有限时的底线策略。不是"差的方法"，而是"有总比没有好"的保底选择。适合考前突击、漏洞扫描。

## 核心框架

4张必修结构图：

| 序号 | 结构图名称 | 核心用途 |
|------|----------|----------|
| 1 | 三大从句功能-连接词-辨析决策树 | 从句选择与辨别 |
| 2 | 阅读理解体裁-结构-策略映射表 | 不同文体的阅读方法 |
| 3 | 七选五语篇衔接信号词库 | 七选五解题 |
| 4 | 应用文写作体裁-格式-语块速查表 | 应用文写作 |

## 具体操作步骤

### 第一张：三大从句「功能-连接词-辨析」决策树

**定语从句**（修饰名词，缺主/宾/定/状）：
```
先看位置 → 名词后
  ↓
判断缺什么成分
  ↓
缺主/宾 → who/whom/that/which
  ↓
缺状语 → when/where/why/介词+which
  ↓
先行词是人还是物？ → 人：who/whom/that；物：which/that
  ↓
是否非限制性？ → 用which，不用that
```

**名词性从句**（主/宾/表/同位语，缺成分/不缺成分）：
```
缺成分 → what/who/which/wherever（在句中担当成分）
不缺成分 → that/whether/if（只起连接作用）
that无义不成分；what缺成分且有意义
```

**状语从句**（修饰动词/句子，表时间/条件/原因/让步/目的/结果）：
```
时间：when/while/as/since/before/after
条件：if/unless/as long as
原因：because/since/as
让步：although/though/even though
目的：so that/in order that
结果：so...that/such...that
```

### 第二张：阅读理解「体裁-结构-策略」映射表

| 体裁 | 核心结构 | 阅读策略 | 作者观点位置 |
|------|----------|----------|--------------|
| 记叙文 | 情感变化线（开端-发展-高潮-结局-感悟） | 抓情感变化线 | 结尾感悟段 |
| 说明文 | 说明对象+层次（现象→原因→影响→对策/时间顺序/空间顺序） | 抓说明对象+层次 | 首段或尾段 |
| 议论文 | 论点-论据-论证（反驳-立论） | 抓论点-论据-论证 | 首段末或转折后 |
| 应用文 | 信息点（广告→优惠/时间/地点；指南→步骤/注意事项） | 抓信息点 | 直接获取 |

### 第三张：七选五「语篇衔接」信号词库

**指代衔接**：
- it/they/this/these/such（指代前文内容）
- one/another/others（指代前文事物）

**逻辑衔接**：
- however/therefore/besides/moreover/furthermore（递进）
- for example/in fact/actually（举例/强调）
- but/however/although/on the contrary（转折）

**词汇衔接**：
- 同义替换：important → significant → crucial
- 反义：like → unlike / include → exclude
- 上下义：animal → dog / fruit → apple

**结构衔接**：
- 段首总起：First of all / To begin with / The first point is...
- 段中过渡：Furthermore / Moreover / In addition
- 段尾总结：Therefore / In conclusion / To sum up

### 第四张：应用文写作「体裁-格式-语块」速查表

**书信类**（建议/申请/邀请/道歉/感谢）：

| 部分 | 内容 | 语块 |
|------|------|------|
| 开头 | 称呼+写信目的 | I am writing to... / I would like to... |
| 主体 | 说明情况/提出请求 | As for... / It is suggested that... |
| 结尾 | 期待回复/感谢 | I am looking forward to your reply / I would appreciate it if... |
| 落款 | Yours sincerely, Li Hua | 注意逗号与换行 |

**通知**：
- 时间/地点/事件/要求/联系方式
- 语块：Notice / All the students are required to... / be held at / contact... at

**演讲稿**：
- 称呼：Ladies and gentlemen / Dear judges and fellow students
- 引入：I'm honored to... / Today I want to talk about...
- 号召：Let's... / Together we can...

## 执行频率建议

- **每章语法学完后**：对照决策树检查是否理解
- **每次阅读前**：回忆不同体裁的阅读策略
- **每次写作前**：参照语块速查表

## 与其他学科的迁移

- **语文**：议论文结构（论点-论据-论证）与英语议论文类似
- **数学**：分类讨论的决策树与英语的从句决策树思维类似
- **历史**：时间线梳理与记叙文的叙事结构类似

## 检验标准

- 能完整默写三大从句的决策树
- 给一篇阅读，能快速判断体裁并说出相应阅读策略
- 给一个应用文题目，能在5分钟内列出写作提纲
