# 「遗传推理」题型建模法

遗传题的本质是"在特定约束条件下进行概率推演"。建立标准决策树模型，实现从"凭感觉猜"到"按模型推"的跃迁。

## 定位

解决遗传题无从下手、概率计算频繁出错的问题。中策提供可操作的步骤流程，适合必修二遗传与进化章节的核心难点突破。

## 核心框架

遗传推理四步决策树：

| 步骤 | 任务 | 关键判断 |
|------|------|----------|
| 第一步 | 判断遗传方式 | 系谱图/杂交实验→显隐/常染色体/伴性 |
| 第二步 | 确定个体基因型 | 隐性纯合突破法、概率空间标注 |
| 第三步 | 计算目标概率 | 分支法、配子法、条件概率 |
| 第四步 | 验证与回归 | 生物意义检验、测交验证 |

## 具体操作步骤

### 第一步：判断遗传方式（模型识别）

拿到系谱图或杂交实验结果，按以下顺序判断：

```
第一步：看表现型比例
  ↓
无中生有 → 隐性
有中生无 → 显性
  ↓
第二步：看是否与性别相关
  ↓
伴性遗传特征：交叉遗传/隔代遗传/性别比例异常
  ↓
第三步：找特殊标记
  ↓
致死效应：比例偏离（如2:1而非3:1）
```

### 第二步：确定个体基因型（逻辑推演）

从已知表现型个体入手：

1. **隐性纯合突破法**：
   - 隐性个体一定是隐性纯合（aa）
   - 显性个体的父母若有一个是隐性，则该显性个体一定是杂合（Aa）

2. **概率空间标注**：
   ```
   显性个体可能是：AA（1/3）或 Aa（2/3）
   不要假设所有显性个体都是纯合！
   ```

### 第三步：计算目标概率（数学工具）

根据题目选择合适的计算方法：

**分支法（逐对分析再组合）**：
- 适用于多对等位基因独立遗传
- 每对单独分析，最后概率相乘

**配子法**：
- 先求亲本配子类型及比例
- 用棋盘格或树状图组合配子

**条件概率**：
- 在"已知某种表现型"条件下修正概率
- 例：在显性后代中，求杂合子概率
  - P(杂合 | 显性) = P(杂合且显性) / P(显性)

### 第四步：验证与回归（生物意义）

- 结果是否符合生物学事实？（概率不可能大于1或小于0）
- 能否用测交实验验证？
- 是否回归到核心概念？（分离定律/自由组合/伴性遗传）

## 执行频率建议

- **每道遗传题**：严格按四步流程操作，即使做对了也要完整走一遍
- **每周**：整理本周遇到的遗传题，按"遗传方式→基因型推导→概率计算"归档
- **月末**：建立"9:3:3:1变式"专题，整理各种比例偏离情况

## 与其他学科的迁移

- **数学**：概率论（加法原理、乘法原理、条件概率）与生物遗传概率直接对应
- **化学**：用数学方法解决分子层面问题的思路与生物遗传推理类似
- **物理**：守恒思想（原子守恒、能量守恒）与遗传中的基因守恒相通

## 检验标准

- 给定任意系谱图，能在3分钟内判断遗传方式并写出相关个体的基因型
- 能区分"男孩患病概率"与"患病男孩概率"的不同含义
- 遗传概率计算题正确率达到90%以上
