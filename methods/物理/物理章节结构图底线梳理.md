# 「章节结构图」底线梳理

若时间有限，每章结束时手绘一张A4结构图，强制包含四个核心节点。这是时间紧张时的保底方法。

## 定位

时间有限时的底线策略。不能保证深入掌握，但能确保对全章有基本框架感。下策不是"差的方法"，而是"有总比没有好"的选择。

## 核心框架

每张结构图必须回答四个问题：

| 节点 | 问题 |
|------|------|
| 输入端 | 这一章研究什么问题？ |
| 工具箱 | 本章提供了哪些核心工具？ |
| 输出端 | 这些工具能解决什么类型的问题？ |
| 易错点 | 本章最容易混淆的3个概念/计算陷阱 |

## 具体操作步骤

### 第一步：确定输入端（一句话）

精简到一句话回答"这一章研究什么问题"。

示例：
- 曲线运动 → "如何用代数方法描述曲线运动"
- 万有引力 → "天体运动的规律是什么"
- 机械能 → "功与能的关系"

追问：
- 这一章的"研究对象"是什么？
- 它和前一章的核心区别是什么？

### 第二步：列出工具箱（定理/公式清单）

列出本章所有核心定理、公式、结论。

示例（平面向量）：
- 向量加法/数乘运算律
- 数量积公式：a·b = |a||b|cosθ
- 坐标表示与坐标运算
- 共线向量定理
- 正弦定理/余弦定理

追问：
- 这些工具在其他章节能否迁移使用？
- 工具之间的关联是什么？

### 第三步：明确输出端（问题类型）

列出本章工具能解决的问题类型，与其他章节的接口。

示例（数列）：
- 已知递推关系求通项 → 接口：函数思想
- 求前n项和 → 接口：裂项技巧
- 证明单调性/有界性 → 接口：导数、不等式

### 第四步：标注易错点（3个坑）

写出本章最容易踩的3个坑：

示例（导数）：
1. 导数存在 ⇏ 原函数连续
2. f'(x₀)=0 ⟹ x₀是极值点（假命题！如f(x)=x³）
3. 可导必连续，但连续不一定可导（如y=|x|）

## 执行频率建议

- **每章结束时**：生成完整版四节点图
- **期中/期末考前**：复习性梳理，对着模板回顾本章结构
- **寒暑假**：跨章节整合，绘制"接口地图"

## 与其他学科的迁移

- **数学**：章节结构图的绘制方法可直接迁移
- **化学**：每章的"反应类型-规律"结构与物理的"模型-规律"思维同构
- **生物**："输入-工具-输出-易错"四节点与生物的"结构-功能-过程-陷阱"相通

## 检验标准

- 不翻书，能在10分钟内画出某一章的四节点结构图
- 能向同学讲解这一章的核心框架，且不出现知识性错误
- 遇到该章的综合题，能快速定位属于哪个节点
