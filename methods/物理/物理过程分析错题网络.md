# 「过程分析」错题网络

物理错题的核心不是"答案错了"，而是"过程分析断了"。按"状态-过程-状态"链条归档错题，建立网络式定位。

## 定位

当学生有错题但不知道根因在哪里时适用。中策比上策门槛低，适合错题积累初期。重点在于发现"过程分析断了"的具体位置，而非泛泛归因于"粗心"。

## 核心框架

错题卡片包含三个核心维度：

| 维度 | 内容 |
|------|------|
| 知识坐标 | X（对象）、Y（规律）、Z（方法）三维定位 |
| 过程断点 | 初态→过程→中间态→过程→末态的链条标注 |
| 错因细化 | 具体到哪一步、哪个符号、哪个隐含条件 |

## 具体操作步骤

### 步骤1：建立错题卡片

每道错题归档为标准化卡片：

```markdown
【知识坐标】
- X轴（对象）：[质点/刚体/点电荷/理想气体]
- Y轴（规律）：[牛顿定律/动能定理/动量守恒/机械能守恒]
- Z轴（方法）：[受力分析/过程分析/守恒判断/图像法]

【情境标签】
□ 斜面问题 □ 竖直平面圆周 □ 天体运动 □ 碰撞 □ 电磁场

【错因分类】
□ 受力分析漏力/多力
□ 运动分解方向错误（速度/位移混淆）
□ 规律选择错误（该用能量法却用了牛顿定律）
□ 过程分界点判断错误
□ 矢量方向/正负号处理错误
□ 守恒条件判断错误

【正解关键节点】
初态 → [过程1：_____] → 中间态 → [过程2：_____] → 末态
```

### 步骤2：标注过程链条

用"状态-过程-状态"链条表示解题关键节点：

```
初态（已知量）→ [过程1：受重力+支持力] → 中间态A（某速度）→ [过程2：只有重力做功] → 末态（某高度）
```

在每个箭头处标注使用的物理公式和成立条件。

### 步骤3：守恒检查

每个物理问题必须回答：
- 机械能是否守恒？
- 动量是否守恒？
- 是否需要验证守恒条件？

### 步骤4：回归教材

- 对应教材哪一节？
- 课后哪道习题是母题？

## 执行频率建议

- **周五**：归档本周错题，重点标注"过程分界点"和"规律选择理由"
- **周日**：横向比较——把"圆周运动"和"天体运动"的错题对比，发现模型同构性
- **月末**：对"规律选择错误"高频项，专门训练"同一题用三种方法解"

## 与其他学科的迁移

- **数学**：方程思想与物理的过程分析思维相通
- **化学**：反应过程（反应物→过渡态→产物）与物理的过程分析类似
- **生物**：生理过程（如信号传导）也可做类似的"状态-过程-状态"分析

## 检验标准

- 错题本中每道题都能找到明确的"过程断点"
- 同类错因不再重复出现（如：受力分析漏力的问题已解决）
- 能向他人讲解这道题的完整分析链条
