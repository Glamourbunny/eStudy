# 「反应类型」网络归档法

放弃按"章节"记忆方程式，改为按"反应本质类型"建立网络，实现从"记方程式"到"理解反应规律"的跃迁。

## 定位

当学生会背方程式但不会写新反应时适用。中策帮助学生看清无机反应和有机反应共享的底层逻辑——化学键的断裂与形成。网络归档比按章节记忆更灵活，迁移性更强。

## 核心框架

核心反应类型矩阵：

| 反应本质 | 无机代表 | 有机代表 | 规律/原理 |
|----------|----------|----------|------------|
| 取代反应 | 甲烷氯代（光） | 烷烃卤代、酯化反应 | 旧键断+新键成，饱和键特征 |
| 加成反应 | （无机较少） | 乙烯+Br₂/HCl/H₂O | 不饱和键打开 |
| 氧化反应 | 金属/非金属与O₂ | 乙醇→乙醛→乙酸 | 失电子/加氧/去氢 |
| 还原反应 | 金属氧化物被还原 | （高一下较少） | 得电子/去氧/加氢 |
| 聚合反应 | （无机较少） | 乙烯→聚乙烯 | 小分子→高分子 |
| 离子反应 | 酸碱中和/盐类反应 | 有机酸/酚弱电离 | 离子交换或生成沉淀/气体/弱电解质 |
| 氧化还原 | 置换/化合/分解 | 燃烧/催化氧化 | 电子转移；化合价变化 |

## 具体操作步骤

### 步骤1：判断反应类型归属

每学一个新反应，先判断它属于哪种反应本质类型：

```
看到这个反应 → 找不饱和键 → 判断加成or取代？
               → 找化合价变化 → 判断氧化还原or非氧化还原？
               → 找离子 → 判断离子反应or分子反应？
```

### 步骤2：横向比较无机与有机的机理差异

| 类型 | 无机视角 | 有机视角 |
|------|----------|----------|
| 氧化反应 | 看化合价升降 | 看加氧/去氢 |
| 还原反应 | 看化合价降低 | 看加氢/去氧 |
| 取代反应 | 酸碱中和、复分解 | 有机物原子被替代 |

### 步骤3：纵向贯通——该反应在后续如何扩展

- 取代反应 → 卤代烃水解/酯的水解/酯化反应逆过程
- 加成反应 → 烯烃加聚/炔烃加成
- 氧化反应 → 醇的催化氧化/醛的银镜反应/新制Cu(OH)₂反应

### 步骤4：建立反应网络图

以一个元素或官能团为中心，辐射所有相关反应：

```
CH₃CH₂OH
    ↓ 氧化
CH₃CHO
    ↓ 氧化
CH₃COOH
    ↓ 酯化
CH₃COOCH₂CH₃
```

每个箭头标注反应类型和条件。

## 执行频率建议

- **每周五**：整理本周学到的反应，分类归档到对应类型
- **每周日**：横向扫描——本周学到的反应中，哪种类型出现最多？
- **月末**：以碳为中心绘制有机反应网络图

## 与其他学科的迁移

- **生物**：有机反应类型（酯化、水解）与生物大分子消化类似
- **物理**：反应热与化学键键能相关（断键吸热、成键放热）
- **数学**：氧化还原反应的电子转移可以用守恒法计算

## 检验标准

- 能准确判断任意一个学过的反应属于哪种反应类型
- 能解释"为什么取代反应不需要条件，而加成反应需要不饱和键"
- 给定一个有机物，能写出它能发生的所有反应类型
