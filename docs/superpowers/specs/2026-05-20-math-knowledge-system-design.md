# 高中数学知识体系 - 三层卡片法

## 概念与目标

基于五大知识主线，构建「**概念-命题-方法**」三层知识卡片体系，服务于个人学习与知识分享。

## 设计原则

1. **知识模块化** - 按五大主线组织，每条主线包含若干知识节点
2. **三层递进** - 每节点分概念/命题/方法三层，层层递进
3. **交互友好** - 支持展开/折叠、搜索、切换主线
4. **视觉清晰** - 深色主题，卡片式布局，层次分明

## 五大主线结构

### 主线① 函数与分析链
```
函数概念 → 幂指对函数 → 三角函数 → 数列 → 导数
```

### 主线② 几何与度量链
```
平面向量 → 复数 → 空间向量 → 立体几何 → 直线与圆 → 圆锥曲线
```

### 主线③ 代数与结构链
```
集合 → 常用逻辑 → 不等式 → 计数原理 → 二项式定理
```

### 主线④ 概率与统计链
```
统计 → 概率 → 随机变量 → 成对数据分析
```

### 主线⑤ 数学思想链
```
函数与方程 · 数形结合 · 分类讨论 · 转化与化归 · 特殊与一般
```

## 三层卡片结构

### 第一层：概念层（是什么）

**内容要素：**
- 原始定义（一字不差）
- 定义的三要素（满足什么条件？有哪些等价表述？）
- 与邻近概念的区别（如：向量平行 vs 直线平行）

**呈现方式：**
- 左侧橙色边框
- 标签：`概念`
- 内容为引述样式（斜体或特殊背景）

### 第二层：命题层（为什么）

**内容要素：**
- 核心定理的逻辑链条（定理 ← 引理 ← 公理）
- 定理的适用边界（什么条件下成立？逆命题是否成立？）
- 教材中隐藏步骤补全

**呈现方式：**
- 左侧蓝色边框
- 标签：`命题`
- 内容为定理/公式样式

### 第三层：方法层（怎么用）

**内容要素：**
- 典型问题的识别信号（看到什么条件→想到什么方法）
- 方法间的决策树（如：求最值 → 函数法？不等式法？几何法？）
- 跨章节方法迁移（如：向量法解三角函数问题）

**呈现方式：**
- 左侧绿色边框
- 标签：`方法`
- 内容为列表/流程样式

## 交互设计

1. **默认全部折叠** - 点击知识节点展开三层卡片
2. **层级可独立折叠** - 每个概念/命题/方法层可独立展开折叠
3. **全局搜索** - 支持关键词搜索高亮
4. **主线切换** - 侧边栏或顶部标签切换主线视图
5. **响应式** - 适配桌面和移动端

## 技术实现

- 纯 HTML + CSS + JavaScript（无框架依赖）
- 单文件部署，便于分享
- 使用 CSS Grid/Flexbox 布局
- 使用 CSS Variables 便于主题切换
