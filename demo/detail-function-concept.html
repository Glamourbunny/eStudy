原始定义（教材原文）

设 A、B 是非空的数集，如果按照某种确定的对应关系 f，使对于集合 A 中的任意一个数 x，在集合 B 中都有唯一确定的数 f(x) 和它对应，那么就称 f：A → B 为从集合 A 到集合 B 的函数，记作 y = f(x)。

定义的三要素

要素含义核心问题
定义域自变量 x 的取值范围"谁可以输入？"
值域因变量 y 的取值范围"输出什么？"
对应法则 fx → y 的对应规则"怎么算？"
💡 两函数相等 ⟺ 定义域相同 且 对应法则相同

易混淆概念辨析

函数 f(x)

y 随 x 变化的过程，强调动态关系

方程 F(x,y)=0

x、y 满足的约束条件，强调静态关系

函数 f(x)

定义域为连续区间（或区间并）

数列 aₙ

定义域为正整数集 {1,2,3,...} 的特殊函数

函数与映射的关系

函数是映射的特殊情况：

  • 映射：集合 A（定义域）→ 集合 B（陪域），A 中元素必须都有唯一对应
  • 函数：A、B 都是数集的特殊映射
📌 函数 ⊂ 映射

函数相等定理

两个函数 f(x) 和 g(x) 相等的充要条件：

① 定义域相同
② 对应法则相同

⚠️ 判断函数相等时，定义域优先

单调性定理

定义（增量形式）：设 f(x) 在区间 D 上有定义，若对任意 x₁, x₂ ∈ D：

  • 当 x₁ < x₂ 时，若 f(x₁) < f(x₂) ⇒ f(x) 在 D 上递增
  • 当 x₁ < x₂ 时，若 f(x₁) > f(x₂) ⇒ f(x) 在 D 上递减

图像判定法：从左向右看，图像上升则递增，下降则递减。

📌 单调性是区间性质，说"函数在区间 D 上单调"

奇偶性定理

奇函数

f(−x) = −f(x)

图像关于原点对称

偶函数

f(−x) = f(x)

图像关于 y 轴对称

⚠️ 定义域必须关于原点对称 📌 f(0) = 0 是奇函数的必要条件（非充分）

最值存在定理

若 f(x) 在闭区间 [a, b] 上连续，则：

① f(x) 必有最大值和最小值
② 最值在端点或极值点处取得

📌 开区间上连续 ≠ 有最值（如 y = x 在 (−1,1) 上无最值）

+

求定义域的决策树

📋 求定义域 — 先列出解析式中的限制条件
分式函数 → 分母 ≠ 0
偶次根式 → 被开方数 ≥ 0
对数函数 logax → 真数 x > 0
复合函数 f(g(x)) → 先求 g(x) 的值域，再将其作为 f 的定义域
多个函数和差积商 → 分别求定义域，再取交集

单调性判断 — 方法选择

表达式可因式分解 差值比较可行 → 定义法：f(x₁) - f(x₂)
二次函数 图像对称轴明确 → 配方法 + 对称轴
可导函数 高阶函数 → 求导法：f'(x) > 0?
抽象函数 赋值法 → 特殊值法 + 单调性定义

奇偶性判定流程

🔍 判断奇偶性 — 三步走
Step 1: 先看定义域是否关于原点对称
不对称 → 非奇非偶（直接结束）
Step 2: 求 f(−x)，与 f(x) 比较
Step 3: 结论
f(−x) = f(x) → 偶函数
f(−x) = −f(x) → 奇函数
都不成立 → 非奇非偶

跨章节方法迁移

  • 向量法判断单调性：将函数值视为向量点积，利用向量数量积的几何意义辅助理解
  • 导数法研究函数：导数是研究函数单调性、极值的通用工具，后续贯穿整个函数体系
  • 数形结合：函数图像与代数表达式结合，可快速判断奇偶性、零点、最值
  • 换元法：复合函数与数列求通项中使用，转化复杂形式为简单形式
