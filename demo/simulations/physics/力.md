# 力（性质力 vs 效果力）

## 概念层

### 直觉与动态仿真锚
**生活现实直觉：**
- 推箱子时感到费力——这就是力的大小感受
- 手被锤子打到会痛——压力的直觉
- 磁铁吸引铁钉——不接触也有力（场力）

**静态与动态图像：**
- 力是矢量，箭头表示方向，长度表示大小
- 弹簧被拉长/压缩——弹力的直观图景
- 用VPython仿真：展示多个力合成平移变化

### 定量与代数工具锚
**实验测量：**
- 弹簧测力计：利用弹力与形变量正比关系
- 压力传感器：应变片形变→电信号
- DIS实验：力传感器实时采集数据

**严格形式化公式：**
- **比值定义式：** $F = ma$ （牛顿第二定律，加速度的定义式/决定式）
- **性质力分类：**
  - 重力 $G = mg$ （地球吸引）
  - 弹力 $F = -kx$ （弹簧胡克定律）
  - 摩擦力 $f = \mu N$ （滑动/静摩擦）
  - 万有引力 $F = G\frac{Mm}{r^2}$
  - 电场力 $F = qE$
  - 洛伦兹力 $F = qvB\sin\theta$

**代数空间解耦：**
- 正交分解：将斜面上的力分解为垂直和平行于斜面的分力
- 坐标系的选取决定解题难度

## 命题层

### 对偶与网络错觉锚
**经典对偶配对：**
| 性质力 | 效果力 |
|--------|--------|
| 重力、弹力、摩擦力... | 向心力、回复力、下滑力... |
| 按本质来源分类 | 按作用效果命名 |
| 不能从受力图上消失 | 可以分解或合成得到 |

**击碎考场错觉：**
- ❌ 误区：向心力是一种新的力
- ✓ 真相：向心力是效果力，是其他力的合力或分力，指向圆心
- ❌ 误区：下滑力是一种新的力
- ✓ 真相：下滑力是重力沿斜面分力，属于重力的效果

### 边界与考场临界锚
**失效边界：**
- 摩擦力：当接触面光滑时，$\mu = 0$
- 弹簧弹力：超过弹性限度，胡克定律失效，$F = -kx$ 变为非线性

**临界状态识别：**
- 静摩擦力刚被推动瞬间：$f_{静} \to f_{最大}$，此时 $a = 0$
- 分离临界：两物体刚好分离时，弹力 $N = 0$，但加速度相同
- 绳模型 vs 轻杆模型：力只能拉不能推

## 方法层

### 跨学科网络同构锚
**前置概念：** 加速度、质量
**后继概念：** 牛顿第二定律、动能定理、动量守恒、功能关系
**公式网络：**
- $F_{合} = ma$ （核心方程）
- $F_{合} = \frac{dp}{dt}$ （动量定理的微分形式）
- $W = F \cdot x \cos\theta$ （功的定义）
- $P = F \cdot v$ （功率）

**物理内同构：**
- 重力场与电场高度相似：$g$ 类似于 $qE/m$
- 引力场与库仑场：$F = G\frac{Mm}{r^2}$ vs $F = k\frac{q_1q_2}{r^2}$

**物化生联动：**
- 化学：化学键本质是电磁力
- 生物：肌肉收缩是拉力（弹力），骨骼支撑是压力（弹力）
- 医学：血压对血管壁的切应力

## 计算机演示代码

```python
# 力的合成与分解仿真
import numpy as np
import matplotlib.pyplot as plt

def simulate_force_resolution():
    """模拟斜面上力的分解"""
    fig, axes = plt.subplots(1, 2, figsize=(14, 6))
    
    # 参数设置
    m = 5  # kg
    g = 9.8  # m/s²
    theta = 30  # 度
    
    # 重力
    G = m * g
    
    # 分解
    G_parallel = G * np.sin(np.radians(theta))  # 沿斜面向下
    G_perpendicular = G * np.cos(np.radians(theta))  # 垂直斜面
    
    # 画图：斜面分解
    ax1 = axes[0]
    ax1.set_xlim(-1, 10)
    ax1.set_ylim(-1, 10)
    ax1.set_aspect('equal')
    ax1.axis('off')
    
    # 绘制斜面
    from matplotlib.patches import Polygon
    slope = np.tan(np.radians(theta))
    x_slope = np.linspace(0, 8, 100)
    y_slope = slope * x_slope
    ax1.fill_between(x_slope, 0, y_slope, alpha=0.3, color='gray')
    
    # 物块位置
    block_x, block_y = 4, slope * 4
    ax1.plot(block_x, block_y, 'ko', markersize=20, fillstyle='none')
    
    # 重力 G（竖直向下）
    ax1.annotate('', xy=(block_x, block_y - 3), xytext=(block_x, block_y),
                 arrowprops=dict(arrowstyle='->', color='red', lw=2))
    ax1.text(block_x + 0.3, block_y - 1.5, f'G = {G:.1f}N', color='red')
    
    # 分解力：沿斜面 G_parallel（蓝色）
    direction = np.array([np.cos(np.radians(theta)), np.sin(np.radians(theta))])
    ax1.annotate('', xy=(block_x + 2.5*direction[0], block_y + 2.5*direction[1]),
                 xytext=(block_x, block_y),
                 arrowprops=dict(arrowstyle='->', color='blue', lw=2))
    ax1.text(block_x + 1.5*direction[0], block_y + 1.5*direction[1] + 0.3, 
             f'G₁ = {G_parallel:.1f}N', color='blue')
    
    # 分解力：垂直斜面 G_perpendicular（绿色）
    perp_direction = np.array([-np.sin(np.radians(theta)), np.cos(np.radians(theta))])
    ax1.annotate('', xy=(block_x + 2*perp_direction[0], block_y + 2*perp_direction[1]),
                 xytext=(block_x, block_y),
                 arrowprops=dict(arrowstyle='->', color='green', lw=2))
    ax1.text(block_x + 0.5*perp_direction[0], block_y + 0.5*perp_direction[1], 
             f'G₂ = {G_perpendicular:.1f}N', color='green')
    
    ax1.set_title('斜面上力的分解：G₁ = Gsinθ, G₂ = Gcosθ')
    
    # 力的合成图
    ax2 = axes[1]
    ax2.set_xlim(-1, 10)
    ax2.set_ylim(-1, 10)
    ax2.set_aspect('equal')
    ax2.axis('off')
    
    # 两个分力
    F1 = 4
    F2 = 3
    angle1 = 0  # 水平向右
    angle2 = 60  # 60度向上
    
    # 画分力
    ax2.annotate('', xy=(F1, 0), xytext=(0, 0),
                 arrowprops=dict(arrowstyle='->', color='blue', lw=2))
    ax2.text(F1/2, -0.5, 'F₁ = 4N', color='blue')
    
    # 画F2
    ax2.annotate('', 
                 xy=(F2 * np.cos(np.radians(angle2)), F2 * np.sin(np.radians(angle2))),
                 xytext=(0, 0),
                 arrowprops=dict(arrowstyle='->', color='green', lw=2))
    ax2.text(F2 * np.cos(np.radians(angle2)) + 0.3, F2 * np.sin(np.radians(angle2)) + 0.3, 
             'F₂ = 3N', color='green')
    
    # 合力（平行四边形法则）
    F1_vec = np.array([F1, 0])
    F2_vec = np.array([F2 * np.cos(np.radians(angle2)), F2 * np.sin(np.radians(angle2))])
    F_result = F1_vec + F2_vec
    
    ax2.annotate('', xy=(F_result[0], F_result[1]), xytext=(0, 0),
                 arrowprops=dict(arrowstyle='->', color='red', lw=3))
    ax2.text(F_result[0]/2 + 0.5, F_result[1]/2, f'F合 = {np.linalg.norm(F_result):.1f}N', color='red')
    
    ax2.set_title('力的合成：平行四边形法则')
    
    plt.tight_layout()
    plt.savefig('force_resolution.png', dpi=150)
    plt.show()
    
    print(f"G = {G}N, G₁ = {G_parallel:.1f}N (沿斜面), G₂ = {G_perpendicular:.1f}N (垂直斜面)")
    print(f"合力F = {np.linalg.norm(F_result):.1f}N")

simulate_force_resolution()
```
