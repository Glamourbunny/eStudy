指数函数原始定义

一般地，函数 y = aˣ（a > 0 且 a ≠ 1）叫做指数函数，其中 x 是自变量，定义域为 R。

📌 底数 a 必须满足：a > 0 且 a ≠ 1 📌 指数 x 是自变量，范围为全体实数 R

对数函数原始定义

一般地，函数 y = logₐx（a > 0 且 a ≠ 1）叫做对数函数，其中 x 是自变量，定义域为 (0, +∞)。

与指数函数的关系：

  • 对数函数是指数函数的反函数
  • y = logₐx ⟺ x = aʸ
  • 图像关于直线 y = x 对称

幂函数原始定义

一般地，函数 y = x^α 叫做幂函数，其中 x 是自变量，α 是常数。

📌 幂函数的指数 α 可以是任意实数 📌 定义域随 α 的取值而变化

指数与对数的运算要素

运算指数律对数律
积aᵐ · aⁿ = aᵐ⁺ⁿlogₐ(MN) = logₐM + logₐN
商aᵐ ÷ aⁿ = aᵐ⁻ⁿlogₐ(M/N) = logₐM - logₐN
幂(aᵐ)ⁿ = aᵐⁿlogₐMⁿ = n · logₐM
换底—logₐM = logᵦM / logᵦa

当 0 < a < 1 时，y = logₐx 在 (0, +∞) 上单调递减

反函数存在定理

指数函数 y = aˣ 和对数函数 y = logₐx 互为反函数。

即：f⁻¹(x) = logₐx 当且仅当 f(x) = aˣ

f⁻¹(f(x)) = x

比较大小 — 题型识别

📋 比较大小 — 常见题型与解法
底数相同，指数不同
利用指数函数单调性：aˣ₁ vs aˣ₂ → 比较 x₁, x₂

构造幂函数或利用图像：xᵃ 形式比较底数

对数比较大小
logₐx 与 logₐy：a>1 时看 x,y；0

解指数/对数不等式

📋 解不等式 — 标准化 + 单调性
指数不等式 aᶠ⁽ˣ⁾ > aᵍ⁽ˣ⁾
a > 1：等价于 f(x) > g(x)
0 < a < 1：等价于 f(x) < g(x)（单调性相反）
对数不等式 logₐf(x) > logₐg(x)
先保证定义域：f(x) > 0 且 g(x) > 0
a > 1：等价于 f(x) > g(x)
0 < a < 1：等价于 f(x) < g(x)
⚠️ 定义域优先！先写出隐含条件 ⚠️ 对数函数真数必须 > 0

复合函数定义域

📋 求复合函数定义域 — 层层约束
已知 f(x) 定义域为 D
求 f(g(x)) 定义域：解 g(x) ∈ D
f(x) = aˣ⁺¹ + logₐ(2x-1)
两个约束：x+1 ∈ R（恒成立）且 2x-1 > 0
取交集：x > 1/2

图像识别与应用

  • 指数函数图像：过 (0,1)，在 x 轴上方，a>1 时上升，0
  • 对数函数图像：过 (1,0)，在 y 轴右侧，a>1 时上升，0
  • 幂函数图像：α>0 时过 (0,0) 和 (1,1)；α<0 时不过 (0,0)
  • 图像交点：方程 aˣ = logₐx 的解 = 图像交点个数

跨章节方法迁移

  • 导数法判断单调性：(aˣ)' = aˣ · ln a，可快速判断指数函数增长速度
  • 对数运算综合题：结合函数零点，logₐf(x)=0 ⟺ f(x)=1（定义域内）
  • 数列中的指数运算：等比数列通项 aₙ = a₁ · qⁿ⁻¹，指数运算律同样适用
  • 换元法：令 t = logₐx 换元，解二次不等式
💡 核心思想：指数与对数互为逆运算，转化是解题关键
