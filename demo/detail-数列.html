数列的定义

数列是定义域为正整数集 N*（或它的有限子集 {1, 2, 3, ..., n}）的离散函数。

数列可以看作是一列按照一定顺序排列的数：a₁, a₂, a₃, ..., aₙ, ...

有穷数列

项数有限，有最后一项

无穷数列

项数无限，没有最后一项

数列的表示方法

  • 通项公式：aₙ = f(n)，用 n 表示 aₙ
  • 递推公式：用 aₙ₋₁、aₙ₋₂ 等表示 aₙ
  • 前 n 项和公式：Sₙ = a₁ + a₂ + ... + aₙ
📌 通项公式是显式表示，递推公式是隐式表示 📌 Sₙ 与 aₙ 的关系：aₙ = Sₙ - Sₙ₋₁ (n ≥ 2)

等差数列定义

如果一个数列从第 2 项起，每一项与它的前一项的差都等于同一个常数，那么这个数列就叫做等差数列。

定义式：aₙ₊₁ - aₙ = d（常数 d 叫做公差）

通项公式：aₙ = a₁ + (n-1)d

📌 d > 0 递增，d < 0 递减，d = 0 常数列

等比数列定义

如果一个数列从第 2 项起，每一项与它的前一项的比都等于同一个常数，那么这个数列就叫做等比数列。

定义式：aₙ₊₁ / aₙ = q（常数 q 叫做公比，q ≠ 0）

通项公式：aₙ = a₁ · qⁿ⁻¹

⚠️ a₁ ≠ 0，q ≠ 0 是等比数列的基本要求 📌 |q| > 1 指数增长，|q| < 1 趋于 0，q < 0 振荡

等差与等比数列概念辨析

特征等差数列等比数列
基本运算加/减（差值相等）乘/除（比值相等）
公差/公比d = aₙ₊₁ - aₙq = aₙ₊₁ / aₙ
通项结构一次函数型指数函数型
图像特征点在直线上点在指数曲线上
中项公式2b = a + cb² = ac

等差数列命题

前 n 项和公式：

Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2

等差中项

若 a, b, c 成等差，则 2b = a + c

b 叫做 a 和 c 的等差中项

重要性质

若 m + n = p + q，则 aₘ + aₙ = aₚ + aᵧ

特别地，aₘ + aₙ₊₁₋ₘ = a₁ + aₙ

📌 Sₙ 是关于 n 的二次函数（d ≠ 0 时） 📌 Sₙ, S₂ₙ-Sₙ, S₃ₙ-S₂ₙ 仍成等差数列

等比数列命题

前 n 项和公式：

当 q ≠ 1 时：Sₙ = a₁(1 - qⁿ)/(1 - q) = a₁(qⁿ - 1)/(q - 1)

当 q = 1 时：Sₙ = na₁

等比中项

若 a, b, c 成等比，则 b² = ac

b 叫做 a 和 c 的等比中项

重要性质

若 m + n = p + q，则 aₘ · aₙ = aₚ · aᵧ

特别地，aₘ · aₙ₊₁₋ₘ = a₁ · aₙ

⚠️ q = 1 时是常系数列，q = -1 时是交替数列 📌 |q| < 1 时，qⁿ → 0（当 n → ∞），Sₙ 有极限

数列单调性与有界性定理

单调性判别：

等差数列：d > 0 则递增，d < 0 则递减

等比数列：a₁ > 0 且 q > 1 时递增；a₁ > 0 且 0 < q < 1 时递减

有界性

数列有上界：存在 M 使 aₙ ≤ M 恒成立

数列有下界：存在 m 使 aₙ ≥ m 恒成立

收敛数列

单调有界数列必有极限

等比数列 |q| < 1 时收敛于 0

求通项公式的方法

📋 已知条件 → 通项公式
已知 a₁ 和 d/q
直接套用等差/等比通项公式
已知 Sₙ 表达式
用 aₙ = Sₙ - Sₙ₋₁（n ≥ 2），再验证 a₁
已知递推关系
累加法（等差型）或累乘法（等比型）
已知 aₘ, aₙ, m, n
等差：d = (aₙ-aₘ)/(n-m)；等比：qⁿ⁻ᵐ = aₙ/aₘ

数列求和的方法

📋 数列求和 — 方法选择
等差数列
公式法：Sₙ = n(a₁+aₙ)/2
等比数列（q ≠ 1）
公式法：Sₙ = a₁(1-qⁿ)/(1-q)
等差×等比（错位相减）
乘以 q 相减，大量项相互抵消
分式型 1/[n(n+k)]（裂项）
裂项：1/[n(n+1)] = 1/n - 1/(n+1)
可分组求和
拆分成等差+等比分别求和

递推数列的解法

📋 由递推式求通项
aₙ₊₁ = aₙ + d（等差递推）
→ 等差数列，aₙ = a₁ + (n-1)d
aₙ₊₁ = q · aₙ（等比递推）
→ 等比数列，aₙ = a₁ · qⁿ⁻¹
aₙ₊₁ = p·aₙ + q（标准线性递推）
→ 设 aₙ₊₁ - A = p(aₙ - A)，求 A = q/(1-p)
aₙ₊₁ = p·aₙ + q·n + r（高阶递推）
→ 待定系数法，构造新等比数列
💡 一阶线性递推：aₙ₊₁ = p·aₙ + q 的通解 = 特解 + 齐次解

数列与函数、不等式综合

  • 数列的函数观点：数列是定义域为 N* 的函数，可用函数思想研究单调性、极值
  • 最值问题：等差数列前 n 项和 Sₙ = (d/2)n² + (a₁-d/2)n，是二次函数在正整数上的最值
  • 不等式证明：放缩法、裂项相消后的放缩、数学归纳法
  • 实际应用题：等差→等差数列求和；等比→等比数列求和；复利问题

跨章节方法迁移

  • 导数法判断单调性：将 aₙ 视为离散函数，可用差分 Δaₙ = aₙ₊₁ - aₙ 判断单调性
  • 向量法：等差数列前 n 项和可类比梯形面积公式
  • 复数法：等比数列求和公式的复数形式应用
💡 核心思想：数列是特殊的函数，离散与连续的桥梁
