导数的定义（增量形式）

设函数 y = f(x) 在点 x₀ 附近有定义，当自变量在 x₀ 处取得增量 Δx 时，函数相应取得增量 Δy = f(x₀ + Δx) - f(x₀)。如果当 Δx → 0 时，Δy/Δx 的极限存在，则称这个极限为函数 y = f(x) 在 x₀ 处的导数。

定义式：f'(x₀) = lim(Δx→0) [f(x₀+Δx) - f(x₀)] / Δx

也可写成：f'(x₀) = lim(h→0) [f(x₀+h) - f(x₀)] / h

📌 导数反映的是函数在某一点的"瞬间变化率" 📌 导数是函数值的增量与自变量增量之比当自变量增量趋于0时的极限

导数的几何意义

函数 y = f(x) 在 x₀ 处的导数 f'(x₀)，等于曲线 y = f(x) 在点 (x₀, f(x₀)) 处的切线斜率。

切线方程

y - f(x₀) = f'(x₀)(x - x₀)

法线方程

y - f(x₀) = -1/f'(x₀) · (x - x₀)

（f'(x₀) ≠ 0 时）

📌 导数 > 0：切线与 x 轴正方向夹角为锐角，函数递增 📌 导数 < 0：切线与 x 轴正方向夹角为钝角，函数递减

导数与连续的关系

如果函数 y = f(x) 在 x₀ 处可导，则它一定在 x₀ 处连续。

可导 ⟹ 连续（但连续不一定可导）

可导的充要条件

左导数 lim(Δx→0⁻) Δy/Δx 存在

右导数 lim(Δx→0⁺) Δy/Δx 存在

且左右导数相等

连续不可导的例子

y = |x| 在 x = 0 处连续

但左导数 = -1，右导数 = 1

故不可导

导数运算法则

加法/减法：(u ± v)' = u' ± v'

乘法：(u · v)' = u'v + uv'

除法：(u/v)' = (u'v - uv')/v²（v ≠ 0）

💡 特别地，(C·u)' = C·u'（常数可提出）

复合函数求导法则（链式法则）

如果 y = f(g(x))，即 y = f(u)，u = g(x)，则：

y' = f'(u) · g'(x) 或 dy/dx = (dy/du) · (du/dx)

例：y = sin(2x + 1)

令 u = 2x + 1，则 y = sin u

y' = cos u · 2 = 2cos(2x + 1)

例：y = eˣ²

令 u = x²，则 y = eᵘ

y' = eᵘ · 2x = 2xeˣ²

⚠️ 链式法则是导数计算的核心，必须熟练掌握

导数与单调性定理

设函数 f(x) 在区间 (a, b) 上可导：

① f'(x) > 0 ⟹ f(x) 在 (a, b) 上单调递增

② f'(x) < 0 ⟹ f(x) 在 (a, b) 上单调递减

③ f'(x) = 0 ⟹ 常数函数（导数恒为零）

📌 这是用导数判断函数单调性的核心依据 📌 注意：区间内导数 > 0 是严格递增的充分条件

极值与最值定理

极值点定义：

若存在 x₀ 的某邻域，使得对所有 x ≠ x₀ 有 f(x) ≥ f(x₀)，则 x₀ 为极小值点

若存在 x₀ 的某邻域，使得对所有 x ≠ x₀ 有 f(x) ≤ f(x₀)，则 x₀ 为极大值点

极值必要条件

若 x₀ 是极值点，且 f'(x₀) 存在

则 f'(x₀) = 0

（即极值点处导数为 0）

极值充分条件（第一充分条件）

若 f'(x₀) = 0

且 x₀ 左侧 f' > 0，右侧 f' < 0

→ x₀ 为极大值点

📌 f'(x₀) = 0 是极值点的必要条件，不是充分条件 📌 导数为零不一定是极值点（如 y = x³ 在 x = 0 处）

闭区间上最值定理

若 f(x) 在闭区间 [a, b] 上连续，则 f(x) 在 [a, b] 上必有最大值和最小值。

求法：

① 求极值点（f'(x) = 0 的点）

② 求端点值 f(a), f(b)

③ 比较所有极值点和端点值的大小

求导计算流程

逐层求导，注意别漏层

利用导数研究函数单调性

📋 已知函数解析式，判断/证明单调性
Step 1: 求定义域
先确定 x 的取值范围
Step 2: 求导 f'(x)
正确求导，注意复合函数链式法则
Step 3: 解不等式 f'(x) > 0 或 < 0
在定义域内解导数不等式
Step 4: 写出单调区间
根据导数符号写出递增/递减区间

求极值与最值的流程

📋 求函数极值/最值
求极值
f'(x) = 0 → 检验左右符号 → 确定极大/极小
求最值（闭区间）
f'(x) = 0 的点 + 端点 → 比较大小
求最值（开区间）
考虑极限：lim f(x) 当 x → 边界

导数与恒成立问题

📋 不等式恒成立问题的解法
f(x) ≥ 0 恒成立
→ f(x)ₘᵢₙ ≥ 0（求最小值）
f(x) ≤ 0 恒成立
→ f(x)ₘₐₓ ≤ 0（求最大值）
参数分离可行
→ 转化为 g(x) ≥/≤ a 的形式
参数分离不可行
→ 构造函数 F(x)，研究 F(x) 的最值
⚠️ 恒成立问题的核心是转化为求函数的最值

导数应用：优化问题

📋 用导数解决实际问题（最优化）
Step 1: 建立函数关系
根据题意，用一个变量表示要优化的量
Step 2: 确定定义域
实际问题中变量的取值范围
Step 3: 求导求最值
f'(x) = 0，找出极值点，判断最值
Step 4: 验证符合实际
检查是否符合实际问题的约束条件
